## 📚 核心内容

### 范畴论的核心思想

范畴论通过"对象-态射"抽象统一数学结构，不关心"物体是什么"，只关心"如何转化"。

### 跨学科实例

| 领域 | 对象 | 态射 | 函子 |
|------|------|------|------|
| 数据库 | 表 | 外键 | 查询 |
| 编程语言 | 类型 | 函数 | 类型构造子 |
| 量子物理 | 希尔伯特空间 | 算符 | 量子态演化 |
| 供应链 | 工厂 | 物流 | 流程映射 |
| 认知 | 概念 | 联想 | 推理链 |

### 数据库设计：Schema作为范畴

- **对象** = 表（Table）
- **态射** = 外键约束（FK）
- **交换图** = 外键传递性
- **查询** = 函子 Q: ℬ → Set
- **JOIN** = 拉回（Pullback）
- **规范化** = 伴随函子

### 编程语言：类型系统作为范畴

- **对象** = 类型（Type）
- **态射** = 函数（Function）
- **笛卡尔闭范畴** = 函数式编程
- **Monad** = 副作用管理
- **Curry-Howard同构** = 逻辑-类型对应
