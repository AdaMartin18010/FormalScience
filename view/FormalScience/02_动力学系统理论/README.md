## 📚 核心内容

### 第一性原理

动力学系统的三个要素：

1. **状态空间** X：所有可能状态的集合
2. **时间演化算子** φ(t)：描述状态如何随时间变化
3. **演化规则**：微分/差分方程、迭代映射或随机过程

### 跨学科映射表

| 领域 | 状态变量 | 演化规则 | 典型方程 |
|------|----------|----------|----------|
| 物理 | 位置、动量 | 牛顿定律 | F=ma |
| 生物 | 种群数量 | 增长-竞争 | Logistic方程 |
| 经济 | 价格、产量 | 供需平衡 | 市场动力学 |
| 社会 | 观点分布 | 传播-衰减 | SIR模型 |
| 认知 | 信念强度 | 证据累积 | 漂移-扩散 |

### 统一数学语言

- **相空间分析**：不动点、雅可比矩阵、稳定性
- **李雅普诺夫函数**：能量类比
- **通用仿真范式**：状态更新循环
