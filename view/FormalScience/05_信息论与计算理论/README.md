## 📚 核心内容

### 信息论的核心概念

1. **信息熵**：衡量信息的不确定性
2. **信道容量**：通信信道的最大传输速率
3. **信息增益**：信息量的增加
4. **互信息**：两个变量之间的信息共享

### 计算理论的核心问题

1. **可计算性**：什么是可计算的？
2. **复杂度**：计算的效率如何？
3. **P vs NP**：是否存在多项式时间算法？
4. **停机问题**：程序是否会终止？
