## 概述

深度学习调度是GPU调度在AI领域的重要应用，涉及大规模分布式训练、高效推理服务和资源优化管理。

### 形式化定义 16.2.1 (深度学习调度系统)

**定义**：深度学习调度系统 $\mathcal{D}$ 是一个八元组：

$$\mathcal{D} = (\mathcal{J}, \mathcal{N}, \mathcal{G}, \mathcal{S}, \mathcal{M}, \mathcal{C}, \mathcal{P}, \mathcal{O})$$

其中：

- $\mathcal{J} = \{j_1, j_2, ..., j_m\}$：作业集合
- $\mathcal{N} = \{n_1, n_2, ..., n_k\}$：计算节点集合
- $\mathcal{G} = \{g_1, g_2, ..., g_p\}$：GPU设备集合
- $\mathcal{S}$：调度策略函数
- $\mathcal{M}$：内存需求函数
- $\mathcal{C}$：计算需求函数
- $\mathcal{P}$：并行策略函数
- $\mathcal{O}$：优化目标函数

## 训练任务调度

### 16.2.1 数据并行（Data Parallelism, DP）

#### 形式化定义 16.2.2 (数据并行)

**定义**：数据并行 $\mathcal{P}_{DP}$ 将批次数据划分到 $n$ 个GPU：

$$\mathcal{P}_{DP}: \mathcal{D}_{batch} \rightarrow \prod_{i=1}^{n} \mathcal{D}_{batch}^{(i)}$$

**All-Reduce通信**：
$$\nabla_{global} = \frac{1}{n} \sum_{i=1}^{n} \nabla_i$$

#### 数据并行通信定理 (Theorem 16.2.1)

**定理**：数据并行的通信开销 $T_{comm}$ 满足：

$$T_{comm} = 2(n-1) \cdot \frac{M_{grad}}{n \cdot B} \cdot \alpha + M_{grad} \cdot \beta$$

其中 $n$ 为GPU数量，$M_{grad}$ 为梯度大小，$B$ 为带宽，$\alpha$ 为启动延迟，$\beta$ 为传输延迟系数。

**证明**：数据并行采用Ring All-Reduce算法，分为Scatter-Reduce和All-Gather两个阶段，各需 $n-1$ 步，每步通信量为 $M_{grad}/n$。总通信量为 $2(n-1) \cdot M_{grad}/n$，代入 $\alpha$-$\beta$ 模型即得证。□

#### PyTorch DDP实现

```python
import torch
import torch.distributed as dist
from torch.nn.parallel import DistributedDataParallel as DDP

class DataParallelTrainer:
    def __init__(self, model, local_rank, world_size):
        self.local_rank = local_rank
        self.world_size = world_size

        dist.init_process_group(
            backend='nccl',
            init_method='env://',
            world_size=world_size,
            rank=local_rank
        )

        torch.cuda.set_device(local_rank)
        self.device = torch.device(f'cuda:{local_rank}')

        self.model = DDP(
            model.to(self.device),
            device_ids=[local_rank],
            find_unused_parameters=False,
            gradient_as_bucket_view=True
        )

        self.optimizer = torch.optim.AdamW(
            self.model.parameters(), lr=1e-3
        )
        self.scaler = torch.cuda.amp.GradScaler()

    def train_step(self, batch):
        self.model.train()
        self.optimizer.zero_grad()

        inputs, labels = batch
        inputs = inputs.to(self.device)
        labels = labels.to(self.device)

        with torch.cuda.amp.autocast():
            outputs = self.model(inputs)
            loss = torch.nn.functional.cross_entropy(outputs, labels)
            loss = loss / self.world_size

        self.scaler.scale(loss).backward()
        self.scaler.unscale_(self.optimizer)
        torch.nn.utils.clip_grad_norm_(self.model.parameters(), max_norm=1.0)
        self.scaler.step(self.optimizer)
        self.scaler.update()

        return loss.item()
```

### 16.2.2 模型并行与张量并行

#### 形式化定义 16.2.3 (张量并行)

**定义**：对于线性层 $y = Wx + b$，张量并行将权重矩阵按列/行划分：

**列并行**：$W = [W_1 | W_2 | ... | W_n]$，$y = [W_1x | W_2x | ... | W_nx] + b$

**行并行**：$y = \sum_{i=1}^{n} W_i x_i + b$，需要All-Reduce

#### Megatron-LM实现

```python
import torch
import torch.nn as nn
import torch.distributed as dist

class ColumnParallelLinear(nn.Module):
    def __init__(self, input_size, output_size, world_size):
        super().__init__()
        self.output_size_per_partition = output_size // world_size

        self.weight = nn.Parameter(
            torch.empty(self.output_size_per_partition, input_size)
        )
        self.bias = nn.Parameter(torch.empty(self.output_size_per_partition))
        nn.init.kaiming_uniform_(self.weight)

    def forward(self, input):
        output_parallel = torch.nn.functional.linear(input, self.weight, self.bias)
        output_list = [torch.empty_like(output_parallel) for _ in range(dist.get_world_size())]
        dist.all_gather(output_list, output_parallel)
        return torch.cat(output_list, dim=-1)

class RowParallelLinear(nn.Module):
    def __init__(self, input_size, output_size):
        super().__init__()
        self.input_size_per_partition = input_size // dist.get_world_size()

        self.weight = nn.Parameter(
            torch.empty(output_size, self.input_size_per_partition)
        )
        self.bias = nn.Parameter(torch.empty(output_size))

    def forward(self, input):
        input_parallel = torch.chunk(input, dist.get_world_size(), dim=-1)[dist.get_rank()]
        output_parallel = torch.nn.functional.linear(input_parallel, self.weight, None)
        dist.all_reduce(output_parallel)
        return output_parallel + self.bias
```

### 16.2.3 流水线并行

#### 形式化定义 16.2.4 (流水线并行)

**定义**：流水线并行将模型按层划分为 $n$ 个阶段：$\mathcal{M} = \mathcal{L}_1 \rightarrow \mathcal{L}_2 \rightarrow ... \rightarrow \mathcal{L}_n$

**GPipe气泡比例**：
$$\text{Bubble} = \frac{n-1}{m + n - 1}$$

其中 $m$ 为微批次数量，$n$ 为流水线深度。

#### 1F1B调度实现

```python
class PipelineStage(nn.Module):
    def __init__(self, module, stage_id):
        super().__init__()
        self.module = module
        self.stage_id = stage_id
        self.input_cache = []
        self.output_cache = []

    def forward(self, input_tensor):
        output = self.module(input_tensor)
        if self.training:
            self.input_cache.append(input_tensor)
            self.output_cache.append(output)
        return output

    def backward(self, grad_output):
        input_tensor = self.input_cache.pop(0)
        output_tensor = self.output_cache.pop(0)
        torch.autograd.backward(output_tensor, grad_output)
        return input_tensor.grad if input_tensor.requires_grad else None

class PipelineScheduler:
    def __init__(self, stages, num_micro_batches):
        self.stages = stages
        self.num_micro_batches = num_micro_batches

    def run_forward_backward(self, micro_batches):
        # Warmup
        for i in range(len(self.stages) - 1):
            if i < len(micro_batches):
                self._forward(i, micro_batches[i])

        # 1F1B阶段
        for i in range(len(self.stages) - 1, len(micro_batches)):
            self._forward(i, micro_batches[i])
            self._backward(i - len(self.stages) + 1)

        # Flush
        for i in range(len(self.stages) - 1):
            self._backward(len(micro_batches) - len(self.stages) + 1 + i)

    def _forward(self, idx, batch):
        x = batch
        for stage in self.stages:
            x = stage.forward(x)

    def _backward(self, idx):
        grad = None
        for stage in reversed(self.stages):
            grad = stage.backward(grad)
```

### 16.2.4 连续批处理

#### 形式化定义 16.2.5 (推理批处理)

**定理16.2.2**：在延迟约束 $L_{max}$ 下，最大批处理大小：
$$B_{max} = \max \{ b \mid T_{prefill}(b) + T_{decode}(b) \leq L_{max} \}$$

#### 连续批处理实现

```python
from dataclasses import dataclass
import time

@dataclass
class InferenceRequest:
    request_id: int
    input_ids: list
    max_new_tokens: int
    arrival_time: float
    generated_tokens: list = None
    is_finished: bool = False

    def __post_init__(self):
        if self.generated_tokens is None:
            self.generated_tokens = []

class ContinuousBatchingScheduler:
    def __init__(self, model, max_batch_size=8):
        self.model = model
        self.max_batch_size = max_batch_size
        self.waiting_queue = []
        self.active_batch = []

    def add_request(self, request):
        self.waiting_queue.append(request)

    def schedule(self):
        self.active_batch = [r for r in self.active_batch if not r.is_finished]
        available = self.max_batch_size - len(self.active_batch)

        for req in sorted(self.waiting_queue,
                         key=lambda r: time.time() - r.arrival_time)[:available]:
            self.active_batch.append(req)
            self.waiting_queue.remove(req)

        return self.active_batch

    def run_step(self):
        batch = self.schedule()
        if not batch:
            return []

        inputs = [r.input_ids + r.generated_tokens for r in batch]
        outputs = self.model.generate(inputs, max_new_tokens=1)

        for req, out in zip(batch, outputs):
            req.generated_tokens.append(out[-1])
            if len(req.generated_tokens) >= req.max_new_tokens:
                req.is_finished = True

        return [r for r in batch if r.is_finished]
```

### 16.2.5 PagedAttention

```python
class BlockManager:
    def __init__(self, num_blocks, block_size):
        self.num_blocks = num_blocks
        self.block_size = block_size
        self.free_blocks = list(range(num_blocks))
        self.req_to_blocks = {}
        self.block_ref_count = {}

    def allocate(self, request_id, num_tokens):
        num_blocks = (num_tokens + self.block_size - 1) // self.block_size
        if len(self.free_blocks) < num_blocks:
            return False

        self.req_to_blocks[request_id] = []
        for _ in range(num_blocks):
            block = self.free_blocks.pop(0)
            self.req_to_blocks[request_id].append(block)
            self.block_ref_count[block] = self.block_ref_count.get(block, 0) + 1
        return True

    def free(self, request_id):
        for block in self.req_to_blocks.get(request_id, []):
            self.block_ref_count[block] -= 1
            if self.block_ref_count[block] == 0:
                self.free_blocks.append(block)
        del self.req_to_blocks[request_id]
```

### 16.2.8 Gang Scheduling

#### 形式化定义 16.2.6 (Gang Scheduling)

**定义**：Gang Scheduling确保作业的所有任务同时获得资源（All-or-Nothing）。

**定理16.2.3**：在集群利用率 $\rho$ 下，Gang Scheduling的作业等待时间：
$$\mathbb{E}[W] \leq \frac{\lambda \mathbb{E}[S^2]}{2(1 - \rho)} + \frac{n - 1}{\mu}$$

## 调度模拟器代码

### 16.2.14 完整模拟器

```python
#!/usr/bin/env python3
import heapq
import random
import numpy as np
from dataclasses import dataclass, field
from typing import List, Dict, Optional
from enum import Enum

class JobStatus(Enum):
    PENDING = "pending"
    RUNNING = "running"
    COMPLETED = "completed"

@dataclass
class DLJob:
    job_id: int
    arrival_time: float
    duration: float
    gpu_demand: int
    memory_demand: float
    priority: int = 0
    status: JobStatus = JobStatus.PENDING
    start_time: Optional[float] = None
    end_time: Optional[float] = None
    assigned_gpus: List[int] = field(default_factory=list)

    @property
    def wait_time(self):
        return (self.start_time or 0) - self.arrival_time

    @property
    def turnaround_time(self):
        return (self.end_time or 0) - self.arrival_time

@dataclass
class GPU:
    gpu_id: int
    memory_capacity: float
    running_job: Optional[int] = None

    @property
    def is_free(self):
        return self.running_job is None

class GangScheduler:
    def __init__(self, num_gpus: int, gpu_memory: float = 80.0):
        self.num_gpus = num_gpus
        self.gpus = [GPU(i, gpu_memory) for i in range(num_gpus)]
        self.pending_queue: List[DLJob] = []
        self.running_jobs: List[DLJob] = []
        self.completed_jobs: List[DLJob] = []
        self.current_time = 0.0
        self.gpu_utilization_log = []

    def submit_job(self, job: DLJob):
        self.pending_queue.append(job)
        self.pending_queue.sort(key=lambda j: (-j.priority, j.arrival_time))

    def get_available_gpus(self, count: int) -> List[int]:
        available = [g.gpu_id for g in self.gpus if g.is_free]
        return available[:count] if len(available) >= count else []

    def allocate_job(self, job: DLJob, gpu_ids: List[int]):
        job.assigned_gpus = gpu_ids
        job.status = JobStatus.RUNNING
        job.start_time = self.current_time
        for gid in gpu_ids:
            self.gpus[gid].running_job = job.job_id

    def release_job(self, job: DLJob):
        job.status = JobStatus.COMPLETED
        job.end_time = self.current_time
        for gid in job.assigned_gpus:
            self.gpus[gid].running_job = None
        self.completed_jobs.append(job)

    def try_schedule(self, job: DLJob) -> bool:
        gpu_ids = self.get_available_gpus(job.gpu_demand)
        if len(gpu_ids) == job.gpu_demand:
            self.allocate_job(job, gpu_ids)
            self.running_jobs.append(job)
            self.pending_queue.remove(job)
            return True
        return False

    def run_simulation(self, duration: float):
        event_queue = []
        for job in list(self.pending_queue):
            heapq.heappush(event_queue, (job.arrival_time, 'arrival', job))
        self.pending_queue = []

        while event_queue and self.current_time < duration:
            time, event_type, job = heapq.heappop(event_queue)
            self.current_time = time

            if event_type == 'arrival':
                if not self.try_schedule(job):
                    self.pending_queue.append(job)
                for pending in list(self.pending_queue):
                    self.try_schedule(pending)
            elif event_type == 'completion':
                self.release_job(job)
                self.running_jobs.remove(job)
                for pending in list(self.pending_queue):
                    self.try_schedule(pending)

            busy = sum(1 for g in self.gpus if not g.is_free)
            self.gpu_utilization_log.append((self.current_time, busy / self.num_gpus))

            for running in self.running_jobs:
                if running.start_time == self.current_time:
                    completion_time = self.current_time + running.duration
                    heapq.heappush(event_queue, (completion_time, 'completion', running))

    def get_statistics(self) -> Dict:
        if not self.completed_jobs:
            return {}
        wait_times = [j.wait_time for j in self.completed_jobs]
        turnaround_times = [j.turnaround_time for j in self.completed_jobs]
        return {
            'completed_jobs': len(self.completed_jobs),
            'avg_wait_time': np.mean(wait_times),
            'avg_turnaround_time': np.mean(turnaround_times),
            'avg_gpu_utilization': np.mean([u for _, u in self.gpu_utilization_log]),
            'throughput': len(self.completed_jobs) / max(1, self.current_time)
        }

def generate_workload(num_jobs: int, arrival_rate: float, seed: int = 42) -> List[DLJob]:
    random.seed(seed)
    np.random.seed(seed)
    jobs = []
    current_time = 0.0

    for i in range(num_jobs):
        current_time += np.random.exponential(1.0 / arrival_rate)
        duration = np.random.lognormal(2, 0.5) * 60
        gpu_demand = random.choice([1, 2, 4, 8])
        memory = np.random.uniform(16, 64)
        priority = random.randint(0, 5)

        jobs.append(DLJob(
            job_id=i,
            arrival_time=current_time,
            duration=duration,
            gpu_demand=gpu_demand,
            memory_demand=memory,
            priority=priority
        ))
    return jobs

def run_comparison():
    NUM_GPUS = 32
    NUM_JOBS = 100
    ARRIVAL_RATE = 0.1
    SIMULATION_TIME = 10000

    workload = generate_workload(NUM_JOBS, ARRIVAL_RATE)
    scheduler = GangScheduler(NUM_GPUS)

    for job in workload:
        scheduler.submit_job(job)
```

```python
import heapq
from dataclasses import dataclass, field

@dataclass
class PriorityJob:
    job_id: str
    priority: int  # 0 = highest
    gpu_demand: int
    duration: float
    preemptible: bool = False

    def __lt__(self, other):
        return self.priority < other.priority

class PreemptiveScheduler:
    """
    优先级抢占调度器
    支持高优先级作业抢占低优先级作业
    """

    def __init__(self, num_gpus: int):
        self.num_gpus = num_gpus
        self.gpu_status = [None] * num_gpus  # job_id or None

        # 队列
        self.high_priority_queue = []  # 不可抢占
        self.normal_queue = []         # 可抢占
        self.preempted_jobs = []       # 被抢占的作业

        # 运行中的作业
        self.running_jobs: Dict[str, Set[int]] = {}  # job_id -> gpu_set

    def submit_job(self, job: PriorityJob):
        """提交作业"""
        if job.priority == 0:
            heapq.heappush(self.high_priority_queue, job)
        else:
            heapq.heappush(self.normal_queue, job)

    def schedule(self):
        """
        执行调度
        1. 优先调度高优先级作业
        2. 必要时抢占低优先级作业
        """
        # 首先调度高优先级作业
        while self.high_priority_queue:
            job = self.high_priority_queue[0]

            # 尝试找到空闲GPU
            free_gpus = self._find_free_gpus(job.gpu_demand)

            if len(free_gpus) >= job.gpu_demand:
                # 可以直接调度
                self._allocate_job(job, free_gpus[:job.gpu_demand])
                heapq.heappop(self.high_priority_queue)
            else:
                # 需要抢占
                needed = job.gpu_demand - len(free_gpus)
                preempted = self._preempt_jobs(needed)

                if len(preempted) + len(free_gpus) >= job.gpu_demand:
                    all_gpus = free_gpus + preempted
                    self._allocate_job(job, all_gpus[:job.gpu_demand])
                    heapq.heappop(self.high_priority_queue)
                else:
                    break  # 无法调度，等待

        # 调度普通优先级作业（使用剩余资源）
        self._schedule_normal_jobs()

    def _preempt_jobs(self, num_gpus_needed: int) -> List[int]:
        """抢占低优先级作业释放GPU"""
        preempted_gpus = []

        # 按优先级排序运行中的作业（抢占最低优先级的）
        sorted_jobs = sorted(
            self.running_jobs.items(),
            key=lambda x: self._get_job_priority(x[0]),
            reverse=True
        )

        for job_id, gpus in sorted_jobs:
            if len(preempted_gpus) >= num_gpus_needed:
                break

            # 只抢占可抢占的作业
            if self._is_preemptible(job_id):
                # 保存检查点
                self._checkpoint_job(job_id)

                # 释放GPU
                for gpu_id in gpus:
                    self.gpu_status[gpu_id] = None
                    preempted_gpus.append(gpu_id)

                # 加入被抢占队列
                self.preempted_jobs.append(job_id)
                del self.running_jobs[job_id]

        return preempted_gpus

    def _find_free_gpus(self, count: int) -> List[int]:
        """查找空闲GPU"""
        free = [i for i, status in enumerate(self.gpu_status) if status is None]
        return free

    def _allocate_job(self, job: PriorityJob, gpu_ids: List[int]):
        """分配GPU给作业"""
        for gpu_id in gpu_ids:
            self.gpu_status[gpu_id] = job.job_id

        self.running_jobs[job_id] = set(gpu_ids)

    def _checkpoint_job(self, job_id: str):
        """保存作业检查点"""
        # 实际实现：保存模型状态、优化器状态、训练进度
        pass

    def _is_preemptible(self, job_id: str) -> bool:
        """检查作业是否可抢占"""
        # 查询作业配置
        return True  # 简化实现

    def _get_job_priority(self, job_id: str) -> int:
        """获取作业优先级"""
        return 1  # 简化实现

    def _schedule_normal_jobs(self):
        """调度普通优先级作业"""
        while self.normal_queue:
            job = self.normal_queue[0]
            free_gpus = self._find_free_gpus(job.gpu_demand)

            if len(free_gpus) >= job.gpu_demand:
                self._allocate_job(job, free_gpus[:job.gpu_demand])
                heapq.heappop(self.normal_queue)
```

### 16.2.19 性能监控与调优

```python
import time
from collections import defaultdict
import torch

class GPUPerformanceMonitor:
    """
    GPU性能监控器
    """

    def __init__(self):
        self.metrics = defaultdict(list)
        self.start_times = {}

    def record_gpu_utilization(self):
        """记录GPU利用率"""
        for i in range(torch.cuda.device_count()):
            util = torch.cuda.utilization(i)
            self.metrics[f'gpu_{i}_util'].append(util)

    def record_memory(self):
        """记录内存使用情况"""
        for i in range(torch.cuda.device_count()):
            allocated = torch.cuda.memory_allocated(i) / 1e9
            reserved = torch.cuda.memory_reserved(i) / 1e9

            self.metrics[f'gpu_{i}_allocated_gb'].append(allocated)
            self.metrics[f'gpu_{i}_reserved_gb'].append(reserved)

    def start_timer(self, name: str):
        """开始计时"""
        self.start_times[name] = time.time()

    def end_timer(self, name: str):
        """结束计时"""
        if name in self.start_times:
            elapsed = time.time() - self.start_times[name]
            self.metrics[f'{name}_time'].append(elapsed)
            del self.start_times[name]

    def get_summary(self) -> Dict:
        """获取性能摘要"""
        summary = {}
        for key, values in self.metrics.items():
            if values:
                summary[key] = {
                    'mean': sum(values) / len(values),
                    'max': max(values),
                    'min': min(values),
                    'count': len(values)
                }
        return summary

    def print_summary(self):
        """打印性能摘要"""
        summary = self.get_summary()
        print("=" * 60)
        print("GPU Performance Summary")
        print("=" * 60)
        for key, stats in summary.items():
            print(f"{key:30s}: mean={stats['mean']:.3f}, max={stats['max']:.3f}")


def profile_cuda_kernel(kernel_fn, *args, num_warmup=10, num_iters=100):
    """
    分析CUDA内核性能
    """
    # 预热
    for _ in range(num_warmup):
        kernel_fn(*args)
    torch.cuda.synchronize()

    # 计时
    start = torch.cuda.Event(enable_timing=True)
    end = torch.cuda.Event(enable_timing=True)

    start.record()
    for _ in range(num_iters):
        kernel_fn(*args)
    end.record()
    torch.cuda.synchronize()

    elapsed_ms = start.elapsed_time(end) / num_iters
    return elapsed_ms
```
