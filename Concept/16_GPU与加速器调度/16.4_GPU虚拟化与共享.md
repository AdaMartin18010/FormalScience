## 概述

GPU虚拟化技术使得多个用户或应用能够共享物理GPU资源，提高硬件利用率并降低成本。本章深入探讨GPU虚拟化的技术原理、调度策略和实现机制。

### 形式化定义 16.4.1 (GPU虚拟化系统)

**定义**：GPU虚拟化系统 $\mathcal{V}$ 是一个八元组：

$$\mathcal{V} = (\mathcal{P}, \mathcal{V}_{guest}, \mathcal{H}, \mathcal{S}, \mathcal{I}, \mathcal{Q}, \mathcal{M}, \mathcal{O})$$

其中：

- $\mathcal{P}$：物理GPU资源集合
- $\mathcal{V}_{guest} = \{v_1, v_2, ..., v_n\}$：虚拟GPU实例集合
- $\mathcal{H}$：Hypervisor/虚拟化层
- $\mathcal{S}: \mathcal{V}_{guest} \times \mathcal{T} \rightarrow \mathcal{P}$：调度映射
- $\mathcal{I}$：隔离机制
- $\mathcal{Q}$：QoS保证策略
- $\mathcal{M}$：内存虚拟化
- $\mathcal{O}$：开销模型

## GPU虚拟化技术

### 16.4.1 虚拟化技术分类

#### 形式化定义 16.4.2 (虚拟化模式)

**定义**：GPU虚拟化模式 $\mathcal{M}_{virt}$ 可以是：

1. **API重定向（Pass-through）**：
   $$\mathcal{H}_{pass} = \{(host \rightarrow guest) | \text{direct hardware access}\}$$

2. **设备模拟（Device Emulation）**：
   $$\mathcal{H}_{emu} = \{(guest \rightarrow host \rightarrow hardware)\}$$

3. **硬件辅助虚拟化（SR-IOV）**：
   $$\mathcal{H}_{sriov} = \{VF_i | VF_i \subset PF, \bigcup VF_i = PF\}$$

4. **时间分片（Time-slicing）**：
   $$\mathcal{H}_{time} = \{(v_i, [t_i, t_i + \delta_i])\}$$

5. **空间分片（MIG）**：
   $$\mathcal{H}_{mig} = \{(GI_i | GI_i \subset GPU, GI_i \cap GI_j = \emptyset)\}$$

#### 技术对比

| 技术 | 性能 | 隔离性 | 密度 | 适用场景 |
|------|------|--------|------|----------|
| Pass-through | 100% | 高 | 低 | 高性能计算 |
| SR-IOV | 95%+ | 高 | 中 | 云计算 |
| vGPU | 90%+ | 中 | 高 | VDI、AI推理 |
| MIG | 98%+ | 高 | 高 | AI训练/推理 |
| Time-slicing | 85%+ | 低 | 极高 | 开发测试 |

```python
class VGPUManager:
    """
    NVIDIA vGPU管理器概念模型
    """

    def __init__(self, physical_gpu):
        self.physical_gpu = physical_gpu
        self.vgpus = {}
        self.profiles = {
            'M60-8Q': {'vram': 8192, 'heads': 4, 'display_res': '4096x2160'},
            'M60-4Q': {'vram': 4096, 'heads': 4, 'display_res': '2560x1600'},
            'M60-2Q': {'vram': 2048, 'heads': 4, 'display_res': '1920x1200'},
            'M60-1Q': {'vram': 1024, 'heads': 2, 'display_res': '1280x1024'},
            'M60-0Q': {'vram': 512, 'heads': 2, 'display_res': '1024x768'},
        }

    def create_vgpu(self, vm_id, profile):
        """创建vGPU实例"""
        if profile not in self.profiles:
            raise ValueError(f"Unknown profile: {profile}")

        vgpu_config = self.profiles[profile]

        # 分配资源
        vgpu = {
            'id': f"vgpu-{vm_id}",
            'vm_id': vm_id,
            'profile': profile,
            'vram': vgpu_config['vram'],
            'frame_buffer': None,  # 分配显存
            'command_queue': [],
        }

        self.vgpus[vm_id] = vgpu
        return vgpu

    def destroy_vgpu(self, vm_id):
        """销毁vGPU实例"""
        if vm_id in self.vgpus:
            # 清理资源
            del self.vgpus[vm_id]

    def schedule_commands(self, vm_id, commands):
        """调度命令到物理GPU"""
        vgpu = self.vgpus.get(vm_id)
        if not vgpu:
            raise RuntimeError(f"vGPU not found for VM {vm_id}")

        # 添加到命令队列
        vgpu['command_queue'].extend(commands)

        # 调度到物理GPU执行
        self._dispatch_to_hardware(vgpu)
```

## 时间片调度

### 16.4.3 时间片轮转调度

#### 形式化定义 16.4.3 (时间片调度)

**定义**：时间片调度 $\mathcal{S}_{TS}$ 将时间划分为固定或可变长度的时间片：

$$\mathcal{S}_{TS} = \{(v_i, \tau_i, [s_i, e_i]) | i = 1, 2, ..., n\}$$

其中：

- $v_i$：第 $i$ 个虚拟GPU
- $\tau_i$：时间片长度
- $[s_i, e_i]$：调度时间窗口

#### 时间片调度定理 (Theorem 16.4.1)

**定理**：对于 $n$ 个虚拟GPU，时间片长度 $\tau$，上下文切换开销 $c$，总响应时间 $R$ 满足：

$$R \leq \frac{n \cdot \tau}{2} + n \cdot c$$

**证明**：

考虑最坏情况：一个请求在时间片开始时到达。

1. **等待当前时间片完成**：最多 $\tau$
2. **等待其他 $n-1$ 个vGPU执行**：$(n-1) \cdot (\tau + c)$
3. **自身执行**：$\tau$

平均等待时间：
$$W_{avg} = \frac{1}{n} \sum_{i=0}^{n-1} i \cdot (\tau + c) = \frac{(n-1)(\tau + c)}{2}$$

总响应时间：
$$R = W_{avg} + \tau = \frac{(n-1)(\tau + c)}{2} + \tau$$

$$
\leq \frac{n \cdot \tau}{2} + n \cdot c
$$

**证毕** □

```cpp
# include <queue>
# include <vector>
# include <chrono>
# include <thread>
# include <mutex>
# include <condition_variable>

struct VGPUContext {
    int vgpu_id;
    int vm_id;

    // GPU状态
    void* register_state;
    void* memory_context;
    void* command_buffer;

    // 调度统计
    uint64_t total_time_slices;
    uint64_t total_execution_time_us;
    uint64_t wait_time_us;

    // QoS参数
    int priority;
    uint64_t time_slice_quota_us;
};

class TimeSliceScheduler {
private:
    std::queue<VGPUContext*> ready_queue;
    std::vector<VGPUContext*> all_vgpus;

    VGPUContext* current_vgpu;
    std::mutex queue_mutex;
    std::condition_variable cv;

    // 配置参数
    uint64_t default_time_slice_us;
    uint64_t context_switch_overhead_us;
    bool running;

    std::thread scheduler_thread;

public:
    TimeSliceScheduler(uint64_t time_slice_us = 2000,  // 2ms default
                      uint64_t ctx_switch_us = 20)      // 20us overhead
        : default_time_slice_us(time_slice_us),
          context_switch_overhead_us(ctx_switch_us),
          current_vgpu(nullptr),
          running(false) {}

    void register_vgpu(VGPUContext* vgpu) {
        std::lock_guard<std::mutex> lock(queue_mutex);
        all_vgpus.push_back(vgpu);
        ready_queue.push(vgpu);
    }

    void start() {
        running = true;
        scheduler_thread = std::thread(&TimeSliceScheduler::run, this);
    }

    void stop() {
        running = false;
        cv.notify_all();
        if (scheduler_thread.joinable()) {
            scheduler_thread.join();
        }
    }

private:
    void run() {
        while (running) {
            VGPUContext* next_vgpu = nullptr;

            {
                std::unique_lock<std::mutex> lock(queue_mutex);
                cv.wait(lock, [this] { return !ready_queue.empty() || !running; });

                if (!running) break;

                next_vgpu = ready_queue.front();
                ready_queue.pop();
            }

            if (next_vgpu) {
                schedule_vgpu(next_vgpu);
            }
        }
    }

    void schedule_vgpu(VGPUContext* vgpu) {
        auto start_time = std::chrono::high_resolution_clock::now();

        // 上下文切换：保存当前
        if (current_vgpu) {
            save_context(current_vgpu);
        }

        // 上下文切换：恢复新vGPU
        restore_context(vgpu);
        current_vgpu = vgpu;

        auto switch_end = std::chrono::high_resolution_clock::now();
        uint64_t switch_time = std::chrono::duration_cast<std::chrono::microseconds>(
            switch_end - start_time).count();

        // 执行时间片
        uint64_t time_slice = vgpu->time_slice_quota_us > 0 ?
                             vgpu->time_slice_quota_us : default_time_slice_us;

        execute_time_slice(vgpu, time_slice);

        auto exec_end = std::chrono::high_resolution_clock::now();
        uint64_t exec_time = std::chrono::duration_cast<std::chrono::microseconds>(
            exec_end - switch_end).count();

        // 更新统计
        vgpu->total_time_slices++;
        vgpu->total_execution_time_us += exec_time;

        // 重新加入队列
        {
            std::lock_guard<std::mutex> lock(queue_mutex);
            ready_queue.push(vgpu);
        }
    }

    void save_context(VGPUContext* vgpu) {
        // 保存寄存器状态
        // save_registers(vgpu->register_state);

        // 刷新缓存
        // flush_cache();

        // 模拟开销
        std::this_thread::sleep_for(
            std::chrono::microseconds(context_switch_overhead_us / 2));
    }

    void restore_context(VGPUContext* vgpu) {
        // 恢复寄存器状态
        // restore_registers(vgpu->register_state);

        // 恢复内存上下文
        // switch_memory_context(vgpu->memory_context);

        // 模拟开销
        std::this_thread::sleep_for(
            std::chrono::microseconds(context_switch_overhead_us / 2));
    }

    void execute_time_slice(VGPUContext* vgpu, uint64_t time_slice_us) {
        // 处理vGPU的命令队列
        // process_commands(vgpu->command_buffer, time_slice_us);

        // 模拟执行
        std::this_thread::sleep_for(std::chrono::microseconds(time_slice_us));
    }
};
```

## 空间共享

### 16.4.5 MIG技术详解

#### 形式化定义 16.4.4 (MIG实例)

**定义**：MIG将物理GPU划分为独立的GPU实例：

$$GPU = \{GI_1, GI_2, ..., GI_n\}$$

其中：

- 每个 $GI_i = (SM_i, Memory_i, Cache_i, CE_i)$
- $GI_i \cap GI_j = \emptyset$ for $i \neq j$

**A100 MIG配置文件**：

| 配置文件 | 内存 | SM | 计算单元 | 适用场景 |
|----------|------|-----|----------|----------|
| 1g.5gb | 5GB | 14 | 1/7 | 推理 |
| 2g.10gb | 10GB | 28 | 2/7 | 推理 |
| 3g.20gb | 20GB | 42 | 3/7 | 训练 |
| 4g.20gb | 20GB | 56 | 4/7 | 训练 |
| 7g.40gb | 40GB | 98 | 7/7 | 大模型 |

#### MIG调度定理 (Theorem 16.4.2)

**定理**：MIG实例的隔离性保证：

$$\forall i \neq j: \text{Performance}(GI_i | GI_j \text{ loaded}) = \text{Performance}(GI_i | GI_j \text{ idle}) \pm \epsilon$$

其中 $\epsilon < 3\%$ 对于计算密集型工作负载。

**证明概要**：

MIG通过以下机制保证隔离：

1. **SM分区**：每个GI独占分配的SM
2. **内存分区**：硬件内存保护隔离
3. **缓存分区**：独立的L2缓存分区
4. **计算引擎**：独立的Copy Engine (CE)

由于硬件资源完全分区，不存在资源竞争，因此性能隔离性接近100%。

跨分区干扰仅来自：

- 共享的电源管理（可忽略）
- 全局内存控制器（<3%）

**证毕** □

### 16.4.6 MIG配置与管理

```python
# !/usr/bin/env python3
"""
MIG配置管理器
自动化MIG实例创建和管理
"""

import subprocess
import json
from typing import List, Dict, Optional
from dataclasses import dataclass

@dataclass
class MIGProfile:
    name: str
    profile_id: int
    memory_gb: int
    num_sm: int

@dataclass
class GPUInstance:
    gi_id: int
    profile: MIGProfile
    placement: Dict
    compute_instances: List[int]

class MIGManager:
    """
    MIG (Multi-Instance GPU) 管理器
    """

    # A100 MIG配置文件
    PROFILES = {
        '1g.5gb': MIGProfile('1g.5gb', 0, 5, 14),
        '2g.10gb': MIGProfile('2g.10gb', 1, 10, 28),
        '3g.20gb': MIGProfile('3g.20gb', 2, 20, 42),
        '4g.20gb': MIGProfile('4g.20gb', 3, 20, 56),
        '7g.40gb': MIGProfile('7g.40gb', 4, 40, 98),
    }

    def __init__(self, gpu_id: int = 0):
        self.gpu_id = gpu_id
        self.instances: Dict[int, GPUInstance] = {}

    def enable_mig(self) -> bool:
        """启用MIG模式"""
        try:
            result = subprocess.run(
                ['nvidia-smi', '-i', str(self.gpu_id), '-mig', '1'],
                capture_output=True, text=True
            )
            return result.returncode == 0
        except Exception as e:
            print(f"Failed to enable MIG: {e}")
            return False

    def disable_mig(self) -> bool:
        """禁用MIG模式"""
        try:
            result = subprocess.run(
                ['nvidia-smi', '-i', str(self.gpu_id), '-mig', '0'],
                capture_output=True, text=True
            )
            return result.returncode == 0
        except Exception as e:
            print(f"Failed to disable MIG: {e}")
            return False

    def list_profiles(self) -> List[MIGProfile]:
        """列出可用的MIG配置文件"""
        try:
            result = subprocess.run(
                ['nvidia-smi', 'mig', '-i', str(self.gpu_id), '-lgip'],
                capture_output=True, text=True
            )
            # 解析输出
            profiles = []
            for name, profile in self.PROFILES.items():
                profiles.append(profile)
            return profiles
        except Exception as e:
            print(f"Failed to list profiles: {e}")
            return []

    def create_instance(self, profile_name: str) -> Optional[GPUInstance]:
        """创建GPU实例"""
        if profile_name not in self.PROFILES:
            raise ValueError(f"Unknown profile: {profile_name}")

        profile = self.PROFILES[profile_name]

        try:
            # 创建GPU实例
            result = subprocess.run(
                ['nvidia-smi', 'mig', '-i', str(self.gpu_id),
                 '-cgi', str(profile.profile_id)],
                capture_output=True, text=True
            )

            if result.returncode != 0:
                print(f"Failed to create instance: {result.stderr}")
                return None

            # 解析实例ID
            # 简化处理，实际应解析输出
            gi_id = len(self.instances)

            instance = GPUInstance(
                gi_id=gi_id,
                profile=profile,
                placement={'start_sm': 0, 'end_sm': profile.num_sm},
                compute_instances=[]
            )

            self.instances[gi_id] = instance
            return instance

        except Exception as e:
            print(f"Failed to create instance: {e}")
            return None

    def create_compute_instance(self, gi_id: int) -> bool:
        """为GPU实例创建计算实例"""
        try:
            result = subprocess.run(
                ['nvidia-smi', 'mig', '-i', str(self.gpu_id),
                 '-gi', str(gi_id), '-C'],
                capture_output=True, text=True
            )
            return result.returncode == 0
        except Exception as e:
            print(f"Failed to create compute instance: {e}")
            return False

    def destroy_instance(self, gi_id: int) -> bool:
        """销毁GPU实例"""
        try:
            result = subprocess.run(
                ['nvidia-smi', 'mig', '-i', str(self.gpu_id),
                 '-gi', str(gi_id), '-dgi'],
                capture_output=True, text=True
            )

            if result.returncode == 0 and gi_id in self.instances:
                del self.instances[gi_id]

            return result.returncode == 0
        except Exception as e:
            print(f"Failed to destroy instance: {e}")
            return False

    def list_instances(self) -> List[GPUInstance]:
        """列出所有MIG实例"""
        try:
            result = subprocess.run(
                ['nvidia-smi', 'mig', '-i', str(self.gpu_id), '-lgi'],
                capture_output=True, text=True
            )

            # 解析输出并更新实例列表
            return list(self.instances.values())
        except Exception as e:
            print(f"Failed to list instances: {e}")
            return []

    def get_optimal_placement(self, workloads: List[Dict]) -> List[MIGProfile]:
        """
        计算最优的MIG配置组合

        workloads: [{'memory_gb': 8, 'compute_intensity': 'high'}, ...]
        """
        # A100最大配置：7个1g.5gb 或 3个2g.10gb + 1个1g.5gb 等

        # 贪心算法：优先匹配最大的配置文件
        placements = []
        remaining_capacity = 7  # A100有7个SM分区

        sorted_workloads = sorted(
            workloads,
            key=lambda w: w['memory_gb'],
            reverse=True
        )

        for workload in sorted_workloads:
            memory_needed = workload['memory_gb']

            # 找到满足内存要求的最小配置文件
            for name, profile in sorted(
                self.PROFILES.items(),
                key=lambda x: x[1].memory_gb
            ):
                if profile.memory_gb >= memory_needed:
                    partitions_needed = profile.num_sm // 14  # 14 SM per partition
                    if partitions_needed <= remaining_capacity:
                        placements.append(profile)
                        remaining_capacity -= partitions_needed
                        break

        return placements
```

### 16.4.10 隔离机制

#### 形式化定义 16.4.5 (隔离性)

**定义**：隔离性 $\mathcal{I}$ 定义为租户间的干扰程度：

$$\mathcal{I}(i, j) = 1 - \frac{\text{Perf}_i(\text{with } j)}{\text{Perf}_i(\text{without } j)}$$

理想情况下，$\mathcal{I}(i, j) \approx 0$。

```python
        """
        QoS感知调度
        优先级：
        1. 满足SLA的最高优先级租户
        2. 根据历史性能动态调整资源

            self.tenants.values(),
            key=lambda t: (self._sla_violation_risk(t), t.qos.priority)

                gpu_id = self._allocate_gpu(tenant.tenant_id)

                if gpu_id is not None:
                    self._execute_request(gpu_id, request, tenant)
                else:
                    # 无可用GPU，放回队列
                    queue.appendleft(request)
                    break

        """计算SLA违反风险（0-1）"""
        if not self.latency_history[tenant.tenant_id]:
            return 0.0

        recent_latencies = self.latency_history[tenant.tenant_id][-10:]
        avg_latency = sum(recent_latencies) / len(recent_latencies)

        # 超过SLA的比例
        violation_ratio = avg_latency / tenant.qos.max_latency_ms
        return min(1.0, max(0.0, violation_ratio - 0.8))

    def _qos_at_risk(self, tenant: Tenant) -> bool:
        """检查QoS是否处于风险中"""
        return self._sla_violation_risk(tenant) > 0.5

    def _allocate_gpu(self, tenant_id: str) -> int:

        # 优先分配已持有的GPU
        for gpu_id, allocated_tenant in self.gpu_allocations.items():

                return gpu_id

        # 分配空闲GPU
        for gpu_id in range(self.total_gpus):

                self.gpu_allocations[gpu_id] = tenant_id
                return gpu_id

        return None  # 无可用GPU
```

## 调度开销分析

### 16.4.12 开销模型

#### 形式化定义 16.4.7 (调度开销)

**定义**：调度开销 $\mathcal{O}$ 包括：

$$\mathcal{O} = \mathcal{O}_{context} + \mathcal{O}_{memory} + \mathcal{O}_{virtualization}$$

其中：

- $\mathcal{O}_{context}$：上下文切换开销
- $\mathcal{O}_{memory}$：内存虚拟化开销
- $\mathcal{O}_{virtualization}$：Hypervisor开销

#### 开销分析定理 (Theorem 16.4.3)

**定理**：在虚拟化环境下，有效GPU利用率为：

$$\eta_{eff} = \eta_{raw} \cdot \prod_{i} (1 - \frac{\mathcal{O}_i}{T_{frame}})$$

其中 $T_{frame}$ 是时间帧长度。

**证明**：

考虑每个时间帧内的开销：

**上下文切换开销**：
$$\mathcal{O}_{ctx} = n \cdot c$$
其中 $n$ 是vGPU数量，$c$ 是单次切换开销。

**内存虚拟化开销**：
$$\mathcal{O}_{mem} = T_{frame} \cdot r_{page\_fault} \cdot t_{fault}$$

**Hypervisor开销**：
$$\mathcal{O}_{hyp} = T_{frame} \cdot \alpha$$

**有效执行时间**：
$$T_{eff} = T_{frame} - \mathcal{O}_{ctx} - \mathcal{O}_{mem} - \mathcal{O}_{hyp}$$

**有效利用率**：
$$\eta_{eff} = \frac{T_{eff}}{T_{frame}} = 1 - \frac{\sum \mathcal{O}_i}{T_{frame}}$$

对于多个虚拟化层：
$$\eta_{eff} = \prod_{i} (1 - \frac{\mathcal{O}_i}{T_{frame}})$$

**证毕** □

```c
    int vm_id;

    // 资源配额

    // 调度状态

    struct CommandQueue *queue;

    // 上下文
    struct GPUContext context;

// 调度循环
void vgpu_scheduler_loop(struct vGPUScheduler *sched) {
    while (sched->running) {
        // 获取下一个vGPU
        struct vGPU *next = select_next_vgpu(sched);

        if (!next) {
            udelay(10);
            continue;
        }

        // 上下文切换
        uint64_t switch_start = get_time_us();
        context_switch(sched->current_vgpu, next);
        uint64_t switch_time = get_time_us() - switch_start;

        // 更新时间片
        uint64_t time_slice = calculate_time_slice(next);

        // 执行vGPU
        execute_vgpu(next, time_slice);

        // 更新统计
        next->accumulated_runtime_us += time_slice;
        sched->current_vgpu = next;
    }
}
```

```python
# !/usr/bin/env python3
"""
GPU虚拟化调度模拟器
模拟vGPU和MIG的调度行为
"""

import heapq
import random
import numpy as np
from dataclasses import dataclass, field
from typing import List, Dict, Optional, Tuple
from enum import Enum
import matplotlib.pyplot as plt

class VirtualizationType(Enum):
    TIME_SLICING = "time_slicing"
    MIG = "mig"
    MPS = "mps"
    PASSTHROUGH = "passthrough"

@dataclass
class vGPU:
    """虚拟GPU"""
    vgpu_id: int
    vm_id: int
    memory_mb: int
    sm_count: int

    # 调度状态
    status: str = "idle"  # idle, running, waiting
    accumulated_time: float = 0.0
    wait_time: float = 0.0

    # 性能
    compute_demand: float = 1.0  # 0-1

@dataclass
class PhysicalGPU:
    """物理GPU"""
    pgpu_id: int
    total_memory_mb: int = 81920  # 80GB A100
    total_sm: int = 108

    # 虚拟化配置
    vgpus: List[vGPU] = field(default_factory=list)
    virtualization_type: VirtualizationType = VirtualizationType.TIME_SLICING

    # 时间分片配置
    time_slice_ms: float = 2.0
    context_switch_ms: float = 0.02

    # MIG配置
    mig_instances: Dict[int, Dict] = field(default_factory=dict)

    def can_add_vgpu(self, vgpu: vGPU) -> bool:
        """检查是否可以添加vGPU"""
        if self.virtualization_type == VirtualizationType.MIG:
            return self._can_create_mig_instance(vgpu)
        else:
            return len(self.vgpus) < 32  # 最大vGPU数

    def _can_create_mig_instance(self, vgpu: vGPU) -> bool:
        """检查MIG配置可行性"""
        total_sm_used = sum(v.sm_count for v in self.vgpus)
        return total_sm_used + vgpu.sm_count <= self.total_sm

class VirtualizationScheduler:
    """
    GPU虚拟化调度器
    支持多种虚拟化技术
    """

    def __init__(self, num_pgpus: int = 1):
        self.pgpus: List[PhysicalGPU] = [
            PhysicalGPU(pgpu_id=i) for i in range(num_pgpus)
        ]
        self.current_time: float = 0.0
        self.event_queue: List[Tuple[float, str, int]] = []

        # 统计
        self.schedule_log: List[Dict] = []
        self.utilization_log: List[Tuple[float, float]] = []

    def create_vgpu(self, pgpu_id: int, vm_id: int,
                   memory_mb: int, sm_count: int) -> Optional[vGPU]:
        """创建vGPU"""
        pgpu = self.pgpus[pgpu_id]

        vgpu = vGPU(
            vgpu_id=len(pgpu.vgpus),
            vm_id=vm_id,
            memory_mb=memory_mb,
            sm_count=sm_count
        )

        if pgpu.can_add_vgpu(vgpu):
            pgpu.vgpus.append(vgpu)
            return vgpu
        return None

    def simulate_time_slicing(self, duration_ms: float):
        """模拟时间分片调度"""
        pgpu = self.pgpus[0]
        pgpu.virtualization_type = VirtualizationType.TIME_SLICING

        num_vgpus = len(pgpu.vgpus)
        if num_vgpus == 0:
            return

        time_slice = pgpu.time_slice_ms
        ctx_switch = pgpu.context_switch_ms

        current_time = 0.0
        vgpu_idx = 0

        while current_time < duration_ms:
            vgpu = pgpu.vgpus[vgpu_idx]

            # 上下文切换开销
            current_time += ctx_switch

            # 执行时间片
            vgpu.status = "running"
            exec_time = time_slice * vgpu.compute_demand
            vgpu.accumulated_time += exec_time

            self.schedule_log.append({
                'time': current_time,
                'vgpu_id': vgpu.vgpu_id,
                'action': 'execute',
                'duration': exec_time
            })

            current_time += time_slice
            vgpu.status = "waiting"

            # 更新其他vGPU的等待时间
            for v in pgpu.vgpus:
                if v.vgpu_id != vgpu.vgpu_id:
                    v.wait_time += time_slice + ctx_switch

            # 下一个vGPU
            vgpu_idx = (vgpu_idx + 1) % num_vgpus

            # 记录利用率
            utilization = self._calculate_utilization()
            self.utilization_log.append((current_time, utilization))

        self.current_time = current_time

    def simulate_mig(self, duration_ms: float):
        """模拟MIG调度（无干扰）"""
        pgpu = self.pgpus[0]
        pgpu.virtualization_type = VirtualizationType.MIG

        # MIG实例并行执行，无上下文切换
        for vgpu in pgpu.vgpus:
            vgpu.status = "running"
            vgpu.accumulated_time = duration_ms * vgpu.compute_demand

            self.schedule_log.append({
                'time': 0,
                'vgpu_id': vgpu.vgpu_id,
                'action': 'execute',
                'duration': duration_ms
            })

        # 利用率始终保持100%
        for t in np.arange(0, duration_ms, 10):
            self.utilization_log.append((t, 1.0))

        self.current_time = duration_ms

    def _calculate_utilization(self) -> float:
        """计算当前GPU利用率"""
        pgpu = self.pgpus[0]
        if not pgpu.vgpus:
            return 0.0

        running_count = sum(1 for v in pgpu.vgpus if v.status == "running")
        return running_count / len(pgpu.vgpus)

    def get_fairness_index(self) -> float:
        """计算Jain公平性指数"""
        pgpu = self.pgpus[0]
        if not pgpu.vgpus:
            return 1.0

        times = [v.accumulated_time for v in pgpu.vgpus]
        if sum(times) == 0:
            return 1.0

        n = len(times)
        sum_times = sum(times)
        sum_squares = sum(t * t for t in times)

        return (sum_times * sum_times) / (n * sum_squares)

    def get_statistics(self) -> Dict:
        """获取模拟统计"""
        pgpu = self.pgpus[0]

        stats = {
            'virtualization_type': pgpu.virtualization_type.value,
            'num_vgpus': len(pgpu.vgpus),
            'total_time': self.current_time,
            'fairness_index': self.get_fairness_index(),
            'avg_utilization': np.mean([u for _, u in self.utilization_log]),
        }

        if pgpu.vgpus:
            stats['avg_wait_time'] = np.mean([v.wait_time for v in pgpu.vgpus])
            stats['avg_accumulated_time'] = np.mean([v.accumulated_time for v in pgpu.vgpus])

        return stats

def run_comparison():
    """运行不同虚拟化技术的对比"""
    print("=" * 60)
    print("GPU Virtualization Comparison")
    print("=" * 60)

    # 创建4个vGPU
    results = {}

    for vt in [VirtualizationType.TIME_SLICING, VirtualizationType.MIG]:
        scheduler = VirtualizationScheduler(num_pgpus=1)
        pgpu = scheduler.pgpus[0]
        pgpu.virtualization_type = vt

        # 创建4个vGPU，每个1/4资源
        for i in range(4):
            scheduler.create_vgpu(
                pgpu_id=0,
                vm_id=i,
                memory_mb=20480,  # 20GB
                sm_count=27       # 27 SM (1/4 of A100)
            )

        # 设置不同的计算需求
        for i, vgpu in enumerate(pgpu.vgpus):
            vgpu.compute_demand = 0.5 + i * 0.15  # 0.5, 0.65, 0.8, 0.95

        # 运行模拟
        duration = 100  # ms
        if vt == VirtualizationType.TIME_SLICING:
            scheduler.simulate_time_slicing(duration)
        else:
            scheduler.simulate_mig(duration)

        stats = scheduler.get_statistics()
        results[vt.value] = stats

        print(f"\n{vt.value.upper()}:")
        print(f"  Number of vGPUs: {stats['num_vgpus']}")
        print(f"  Fairness Index: {stats['fairness_index']:.4f}")
        print(f"  Avg Utilization: {stats['avg_utilization']*100:.1f}%")
        if 'avg_wait_time' in stats:
            print(f"  Avg Wait Time: {stats['avg_wait_time']:.2f} ms")

    # 可视化
    plot_comparison(results)

    return results
```
