### 5个基础假设（Foundation）

```text
【H₀.1 核心同构假设】━━━━━━━━━━━━━━━━━━━━━┓
│  人脑 ≃ 世界 ≃ 形式语言               │
│  • 认知可建模                          │
│  • 世界可形式化                        │
│  • 存在同构映射                        │
└───────────────────────────────────────┘
                ↓
【H₀.2 可形式化假设】━━━━━━━━━━━━━━━━━━━━┓
│  一切计算现象都可被形式化               │
│  • 语法可定义                          │
│  • 语义可映射                          │
│  • 推理可验证                         │
└──────────────────────────────────────┘
                ↓
【H₀.3 反身性驱动假设】━━━━━━━━━━━━━━━━━━━━━┓
│  系统演化由自我指涉驱动                   │
│  • 自指产生涌现                          │
│  • 递归产生复杂性                        │
│  • 反思产生智能                          │
└─────────────────────────────────────────┘
                ↓
【H₀.4 计算主义假设】━━━━━━━━━━━━━━━━━━━━━━━┓
│  认知本质上是计算过程                     │
│  • 思维是符号操作                         │
│  • 大脑是计算机器                         │
│  • 智能可人工实现                         │
└─────────────────────────────────────────┘
                ↓
【H₀.5 语义客观性假设】━━━━━━━━━━━━━━━━━━━━┓
│  语义在实践共同体中可达成间主观性         │
│  • 意义可共享                           │
│  • 理解可对齐                           │
│  • 沟通可实现                           │
└────────────────────────────────────────┘
```

**假设间关系**：

- H₀.1 是本体论基础（存在论）
- H₀.2 是认识论基础（可知论）
- H₀.3 是演化论基础（发展论）
- H₀.4 是心智哲学基础（功能论）
- H₀.5 是语言哲学基础（实用论）
