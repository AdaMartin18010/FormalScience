## 核心公式

> 📝 **使用说明**: 每个公式都有唯一编号，可在其他文档中引用
> 格式：`[分类-编号]` 如 `[INFO-01]` 表示信息论第1个公式

### 信息论

**[INFO-01] Shannon熵**:

```text
H(X) = -Σ p(x) log p(x)
H(X|Y) = -ΣΣ p(x,y) log p(x|y)
```

**[INFO-02] 互信息**:

```text
I(X;Y) = H(X) + H(Y) - H(X,Y)
       = H(X) - H(X|Y)
       = H(Y) - H(Y|X)
       = D_KL(P(X,Y) || P(X)P(Y))
```

**[INFO-03] 条件互信息**:

```text
I(X;Y|Z) = H(X|Z) - H(X|Y,Z)
```

**[INFO-04] 数据处理不等式**:

```text
X → Y → Z  ⟹  I(X;Z) ≤ I(X;Y)
```

**[INFO-05] Fano不等式**:

```text
H(X|Y) ≤ H(P_e) + P_e log(|𝒳| - 1)
```

**[INFO-06] Shannon信道容量**:

```text
C = max I(X;Y)
    P(X)

AWGN信道: C = (1/2) log(1 + SNR)
```

**[INFO-07] 率失真函数**:

```text
R(D) = min I(X;X̂)
      E[d(X,X̂)]≤D
```

**[INFO-08] Kolmogorov复杂度**:

```text
K(x) = min{|p| : U(p) = x}
K(x|y) = min{|p| : U(p,y) = x}
K(x,y) ≤ K(x) + K(y) + O(1)
```

**[PHYS-01] Landauer极限**:

```text
E_min = kT ln 2  ≈ 2.9 × 10^(-21) J @ 300K
```

### 学习理论

**[LEARN-01] VC维与样本复杂度**:

```text
m ≥ O((d/ε²) log(1/δ))   # d = VC维
```

**[LEARN-02] 泛化误差界（PAC）**:

```text
P(|R(h) - R̂(h)| ≤ ε) ≥ 1 - δ

ε = O(√((d log(m/d) + log(1/δ)) / m))
```

**[LEARN-03] Rademacher复杂度**:

```text
R_m(ℋ) = E_σ[sup_{h∈ℋ} (1/m) Σᵢ σᵢ h(xᵢ)]

R(h) ≤ R̂(h) + 2R_m(ℋ) + √(log(1/δ) / 2m)
```

**[LEARN-04] No Free Lunch定理**:

```text
E_f[error(A₁, f)] = E_f[error(A₂, f)]  # 对所有学习算法
```

**[LEARN-05] Gold可学习性**:

```text
可从正例学习 ⟺ 有限弹性 ⟺ 有限告知数
```

### 控制论

**[CTRL-01] Ashby必要多样性定律**:

```text
V_controller ≥ V_system

其中 V = log |S| (系统状态空间大小)
```

**[CTRL-02] Data Rate定理**:

```text
R_control ≥ H(系统状态)
```

**[CTRL-03] Lyapunov稳定性**:

```text
V(x) > 0, V(0) = 0
V̇(x) < 0  ⟹  系统稳定
```

**[CTRL-04] 反馈控制误差界**:

```text
|e(t)| ≤ |e(0)| e^(-λt)  # λ = 控制增益
```

**[CTRL-05] 系统响应时间**:

```text
T_response ≈ 1/λ  # λ = 系统带宽
```

### 计算复杂性

**[COMP-01] 时间复杂性**:

```text
TIME(f(n)) = {L : L可在O(f(n))时间内判定}
P = ∪_{k≥0} TIME(n^k)
```

**[COMP-02] 空间复杂性**:

```text
SPACE(f(n)) = {L : L可在O(f(n))空间内判定}
PSPACE = ∪_{k≥0} SPACE(n^k)
```

**[COMP-03] Savitch定理**:

```text
NSPACE(f(n)) ⊆ SPACE(f²(n))
```

**[COMP-04] 时间-空间权衡**:

```text
TIME(f(n)) ⊆ SPACE(f(n))
SPACE(f(n)) ⊆ TIME(2^O(f(n)))
```

### 分布式系统

**[DIST-01] BFT阈值**:

```text
n ≥ 3f + 1  # n = 总节点数, f = 故障节点数
```

**[DIST-02] 最终一致性**:

```text
∀ t > T : |v_i(t) - v_j(t)| < ε  # 所有节点i,j
```

**[DIST-03] CAP定理**:

```text
一致性(C) ∧ 可用性(A) ∧ 分区容错(P)  ⟹  不可能同时满足
```

### 编程算法设计

**[PROG-01] UH-Cost统一元模型**:

```text
Cost(程序) = Σ (操作数 × 操作成本)
```

**[PROG-02] 形式语义复杂度**:

```text
复杂度(语义) = |语法| × |语义映射|
```

**[PROG-03] 算法复杂度界**:

```text
T(n) = O(f(n))  ⟹  存在常数c, n₀使得
∀ n > n₀ : T(n) ≤ c·f(n)
```

**[PROG-04] 空间复杂度界**:

```text
S(n) = O(f(n))  ⟹  存在常数c, n₀使得
∀ n > n₀ : S(n) ≤ c·f(n)
```

**[PROG-05] 复杂度权衡**:

```text
时间 × 空间 ≥ 问题规模
```

## 重要定理

### 不可判定性/不可能性

1. **Church-Turing论题**: 可计算函数 = 图灵可计算函数
2. **停机问题**: 不存在算法判定任意程序是否停机
3. **Rice定理**: 程序的所有非平凡性质都不可判定
4. **CAP定理**: 分布式系统无法同时满足一致性、可用性、分区容错
5. **Arrow不可能定理**: 不存在满足所有合理条件的投票系统

### 可学习性

1. **PAC学习**: 概率近似正确学习框架
2. **VC维理论**: 学习能力的度量
3. **No Free Lunch**: 不存在对所有问题都最优的学习算法
4. **Gold可学习性**: 从正例学习的充要条件

### 系统与控制

1. **Ashby定律**: 控制器多样性必须≥系统多样性
2. **Lyapunov稳定性**: 系统稳定的充分条件
3. **反馈控制**: 误差指数衰减

## 关键常数

### 物理常数

- **Landauer极限**: E_min ≈ 2.9 × 10⁻²¹ J @ 300K
- **光速**: c = 2.998 × 10⁸ m/s
- **普朗克常数**: h = 6.626 × 10⁻³⁴ J·s

### 信息理论界限

- **Shannon熵上界**: H(X) ≤ log |𝒳|
- **互信息上界**: I(X;Y) ≤ min(H(X), H(Y))
- **信道容量**: C ≤ log(1 + SNR) (AWGN信道)

### 计算极限

- **图灵机状态数**: 有限但可任意大
- **可计算数**: 可数无穷
- **不可计算数**: 不可数无穷

## 不可能三角

### CAP三角

```text
       一致性 (C)
         /\
        /  \
       /    \
可用性(A) ── 分区容错(P)
```

**选择**:

- **CA系统**: 传统数据库（单机）
- **CP系统**: 分布式数据库（强一致性）
- **AP系统**: 分布式缓存（高可用）

### 虚拟化三角（冯·诺依曼）

```text
      性能
       /\
      /  \
     /    \
隔离 ── 可移植性
```
