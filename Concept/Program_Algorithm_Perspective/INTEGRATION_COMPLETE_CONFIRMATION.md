```text
UH-Cost = ⟨Σ, ⟶, κ, Φ⟩

Σ  : 系统状态空间
⟶  : 转换关系
κ  : 代价函数
Φ  : 属性规范
```

**地位**：编程算法设计的"大统一理论"

### 三元视角

```text
System = ⟨Control, Execution, Data⟩
```

**地位**：与七视角形成正交分析维度
