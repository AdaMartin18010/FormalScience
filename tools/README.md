# Tools - 项目工具

> **目录**: `tools/`  
> **用途**: 文档处理、格式检查和项目维护工具  
> **最后更新**: 2026-04-12

---

## 📁 工具分类

### 文档处理工具
| 工具 | 用途 |
|------|------|
| `batch_add_three_modules.py` | 批量添加三个模块 |
| `batch_fix_markdown.py` | 批量修复 Markdown 格式 |
| `batch_fix_toc.py` | 批量修复目录 |
| `batch_normalize_documents.py` | 批量规范化文档 |
| `comprehensive_fix_all_markdown.py` | 全面修复 Markdown |

### 链接和格式工具
| 工具 | 用途 |
|------|------|
| `link_validator.py` | 链接验证器 |
| `scan_broken_links.py` | 扫描断链 |
| `fix_links_comprehensive.py` | 全面修复链接 |
| `format_unifier.py` | 格式统一器 |

### 内容分析工具
| 工具 | 用途 |
|------|------|
| `content_analysis_tool.py` | 内容分析 |
| `document_linter.py` | 文档检查 |
| `document_quality_checker.py` | 质量检查 |
| `structure_checker.py` | 结构检查 |

---

## 🚀 使用示例

### 扫描断链

```bash
python tools/scan_broken_links.py
```

### 修复 Markdown 格式

```bash
python tools/batch_fix_markdown.py
```
