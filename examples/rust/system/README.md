## 📁 示例文件

| 文件 | 描述 |
|------|------|
| `bpf_program.rs` | BPF 程序示例 |
| `memory_management.rs` | 内存管理 |

## 🚀 运行示例

```bash
# 运行特定示例
cargo run --example bpf_program
cargo run --example memory_management
cargo run --example syscalls
```

部分示例需要特定权限或环境：

- BPF 示例需要 root 权限和 BPF 支持
- 系统调用示例可能需要特定操作系统
