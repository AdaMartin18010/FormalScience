#### 2.2 基础验证演示 (3:00-3:45)

**画面**: 代码编辑器

**操作演示**:

1. 输入简单逻辑表达式

   ```python

       return (not p) or q

   # 验证: p → q ≡ ¬p ∨ q
   verify(implies, equivalence="¬p ∨ q")
   ```

2. 点击验证按钮
3. 展示验证结果

**旁白**:
> "比如这里，我们定义了一个蕴含函数。验证引擎会检查这个实现是否与标准的逻辑等价。"

**字幕**: 基础逻辑验证

**操作演示**:

1. 展示更复杂的验证场景

           post="is_sorted(result) ∧ permutation(result, input)")

**旁白**:
