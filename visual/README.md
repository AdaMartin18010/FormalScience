## 🎨 图表类型

### 架构图 (architecture/)
系统整体架构和组件关系图

### 类图 (class_diagrams/)
面向对象设计的类关系图

### 流程图 (flowcharts/)
算法和业务流程图

### 时序图 (sequence_diagrams/)
系统交互和调用序列图

### 状态图 (state_diagrams/)
状态机和状态转换图

---

## 📝 图表规范

1. 使用 Mermaid 语法或标准图形格式
2. 文件名应描述图表内容
3. 在相关文档中引用图表
