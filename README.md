## 📖 项目简介

FormalScience 是一个致力于形式科学内容整理与形式化的开源项目。项目涵盖数学、逻辑学、计算机科学、信息论等形式科学领域的核心知识体系，通过严格的数学规范和形式化表达，构建可追溯、可验证的知识网络。

### 核心主题

- **Mass (质量)** - 物理与数学基础
- **Energy (能量)** - 动力系统与变换
- **Information (信息)** - 信息论与计算理论
- **View (视角)** - 认知框架与表征系统

## 🚀 快速开始

### 环境要求

- [Rust](https://www.rust-lang.org/) (>= 1.70) - 用于形式化验证
- [Python](https://www.python.org/) (>= 3.9) - 用于工具脚本
- [Lean](https://leanprover.github.io/) - 用于定理证明
- [VSCode](https://code.visualstudio.com/) 或 [Cursor](https://cursor.sh/) - 推荐的编辑器

### 安装与使用

```bash
# 克隆仓库
git clone https://github.com/username/FormalScience.git
cd FormalScience

# 安装 Python 依赖（可选）
pip install -r requirements.txt

# 运行代码检查
cd tools && python check_all.py
```

### 阅读指南

1. **入门推荐**: 从 `docs/Refactor/` 目录开始，包含重构后的结构化文档
2. **主题浏览**: 查看 `Concept/` 目录了解核心概念体系
3. **形式化代码**: 查看 `FormalRE/` 和 `*.lean` 文件了解严格证明
4. **可视化**: 查看 `view/` 目录的 SVG 图表
