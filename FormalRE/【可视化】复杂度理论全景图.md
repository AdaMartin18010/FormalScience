## 2. P vs NP 关系决策树

```text
问题: P = NP 吗？
    |
    ├─ 情况1: P = NP (概率<1%)
    │   ├─ 后果

    │   │   ├─ NP完全可多项式解
    │   │   └─ 计算机科学革命
    │   │
    │   ├─ 证明路径
    │   │   ├─ 构造多项式SAT算法
    │   │   ├─ 证明P中间类不存在
    │   │   └─ (至今无人成功)
    │   │
    │   └─ 哲学含义
    │       └─ "搜索 = 验证"
    │

    │   ├─ 后果
    │   │   ├─ 密码学安全 ✓
    │   │   ├─ 某些问题本质困难
    │   │   └─ 当前世界观正确
    │   │
    │   ├─ 证明路径
    │   │   ├─ 对角化？(被相对化阻碍)
    │   │   ├─ 电路复杂度下界
    │   │   ├─ 自然证明障碍(Razborov)
    │   │   └─ 代数化障碍
```

## 3. 算术谱系 vs 多项式层级对比

### 对比矩阵

| 层级 | 算术谱系 | 多项式层级 | 关系 | 典型问题 |
|------|---------|-----------|------|---------|
| **Σ₀⁰ = Π₀⁰** | 递归集 | - | = 可判定 | 2+2=4? |
| **Σ₁⁰** | RE (∃) | - | 半可判定 | 停机问题 |
| **Π₁⁰** | co-RE (∀) | - | 半可判定补 | 非停机 |
| **Δ₂⁰** | 递归于停机 | - | - | - |
| **Σ₂⁰** | ∃∀ | - | - | 全称停机 |
| - | - | **P** | ⊆ Σ₀⁰ | 排序 |
| - | - | **NP** | ⊆ Σ₀⁰ | SAT |
| - | - | **Σ₂ᵖ** | ⊆ Σ₀⁰ | Σ₂-SAT |
| - | - | **PH** | ⊆ Σ₀⁰ | 多项式谱系 |
| - | - | **PSPACE** | ⊆ Σ₀⁰ | QSAT |

**关键区别**:

- 算术谱系: 基于**逻辑量词**复杂度
- 多项式层级: 基于**时间复杂度** + 量词

---

### 层次关系图

```text
不可判定层 (超出RE)
    ↑
━━━━━━━━━━━━━━━━━━━ RE边界 ━━━━━━━━━━━━━━━━━━
    ↑
算术谱系顶层 (Σₙ⁰, Πₙ⁰ → ∞)
    ↑
Σ₂⁰ (∃∀) : 全称停机问题
    ↑
Π₁⁰ (co-RE) : 非停机集
    ↑
Σ₁⁰ (RE) : 停机集
    ↑
━━━━━━━━━━━━━━━━━━━ 可判定边界 ━━━━━━━━━━━━━━━━━
    ↑
Σ₀⁰ = Π₀⁰ (递归) : 包含下面所有
    ↑
EXP (2ⁿ时间) : 指数时间
    ↑
PSPACE (多项式空间) : QSAT
    ↑
PH (多项式层级) : Σₖᵖ, Πₖᵖ
    ↑
Σ₂ᵖ : ∃∀多项式
    ↑
coNP : NP的补
    ↑
NP : 非确定多项式
    ↑
P : 确定多项式
    ↑
NL : 非确定对数空间
    ↑
L : 对数空间
```

## 4. 归约技术矩阵

| 归约类型 | 定义 | 用途 | 复杂度类 | 示例 |
|---------|------|------|---------|------|
| **多项式时间归约** | A ≤ₚ B | NP完全性 | NP, coNP | SAT ≤ₚ 3-SAT |
| **对数空间归约** | A ≤ₗ B | P完全性 | P | 电路求值 |
| **图灵归约** | A ≤ₜ B | 计算能力 | RE | 停机≤ₜ全称停机 |
| **多对一归约** | A ≤ₘ B | 不可判定性 | RE | 停机≤ₘ空带 |
| **真值表归约** | A ≤ₜₜ B | 平行归约 | 递归 | - |

**归约决策树**:

```text
我要证明什么？
    |
    ├─ NP完全性
    │   └─ 使用: 多项式时间归约 (≤ₚ)
    │       └─ 示例: 已知NP完全 ≤ₚ 目标问题
    │
    ├─ P完全性
    │   └─ 使用: 对数空间归约 (≤ₗ)
    │       └─ 示例: CVP ≤ₗ 目标问题
    │
    ├─ 不可判定性
    │   └─ 使用: 多对一归约 (≤ₘ)
    │       └─ 示例: 停机 ≤ₘ 目标问题
    │
    └─ 相对可计算性
        └─ 使用: 图灵归约 (≤ₜ)
            └─ 示例: A ≤ₜ B意味着Oracle B可解A
```

```text
      |          |

                 |

            |    |    |
          PH    QBF  IP
            |         |
        ┌───┴───┐     |
        |       |     |
      coNP     NP ────┘
        |       |
        └───┬───┘

✓ L ⊊ PSPACE

? NP ≠ coNP
```

## 6. 实践应用决策树

```text
我有一个问题，该用什么算法？
    |
    ├─ 问题在P中？
    │   ├─ 是 → 找多项式算法
    │   │   ├─ 排序: O(n log n)
    │   │   ├─ 图最短路: Dijkstra
    │   │   └─ 线性规划: 内点法
    │   │
    │   └─ 否 → 继续判断
    │
    ├─ 问题在NP中？
    │   ├─ 是NP完全
    │   │   ├─ 精确解: 指数算法
    │   │   ├─ 近似解: 近似算法
    │   │   ├─ 启发式: 遗传/模拟退火
    │   │   └─ 特殊实例: SAT求解器
    │   │
    │   └─ 在NP但不知道完全性
    │       └─ 如: 因数分解 (NP∩coNP)
    │           └─ 量子算法: Shor
    │
    ├─ 问题在PSPACE中？
    │   └─ 如: 博弈树搜索
    │       ├─ Alpha-Beta剪枝
    │       └─ MCTS
    │
    └─ 问题不可判定？
        └─ 半判定 + 实践启发式
            └─ 如: 停机问题的实践版本

关键判断:
✓ 是否验证比求解容易? → NP
✓ 是否需要对抗搜索? → PSPACE
✓ 是否有随机性? → BPP/RP
✓ 是否量子可解? → BQP
```
