### 1 1 定义 {#1-定义} {#1-定义-1-定义}

- _范畴_* C 包含:

1. **对象集** Ob(C)
2. **态射集** Hom(A,B) （对任意对象A,B）
3. **复合**: f:A→B, g:B→C ⇒ g∘f:A→C
4. **恒等态射**: id_A: A→A

- _公理_*:

```text
1. 结合律: h∘(g∘f) = (h∘g)∘f
2. 单位律: f∘id_A = f = id_B∘f
```

### 1 2 直觉理解 {#2-直觉理解} {#2-直觉理解-2-直觉理解}

- _范畴 = 对象 + 箭头 + 复合规则_*

- _图示_*:

```text
A ──f──→ B ──g──→ C
 ↘      ↗
   g∘f
```

- _与集合论对比_*:

- 集合论: 关心对象**内部**（元素）
- 范畴论: 关心对象**之间**（态射）

- _态射_*: 函数 f: A→B

- _对象_*: 群 (G, ·, e)

- _对象_*: 偏序集元素
- _态射_*: a→b 当且仅当 a≤b
- _复合_*: 传递性
- _恒等_*: 自反性

- _深刻_*: 偏序 = 特殊的范畴！

- _复合_*: 程序复合

## 3. 函子 {#函子} {#函子-函子}

### 3 1 定义 {#1-定义} {#1-定义-1-定义}

- _函子_* F: C → D:

- F: Ob(C) → Ob(D)
- F: Hom_C(A,B) → Hom_D(F(A), F(B))

- _保持结构_*:

```text
F(id_A) = id_{F(A)}
F(g∘f) = F(g)∘F(f)
```

### 3 2 例子 {#2-例子} {#2-例子-2-例子}

- _幂集函子_* P: Set → Set:

```text
P(A) = {S | S⊆A}
P(f: A→B)(S) = {f(a) | a∈S}
```

- _遗忘函子_* U: Grp → Set:

```text
U(G) = 底层集合
U(φ) = 底层函数
```

- _列表函子_* List: Type → Type:

```haskell
List :: * -> *
List A = [A]
fmap f xs = map f xs
```

## 4. 自然变换 {#自然变换} {#自然变换-自然变换}

### 4 1 定义 {#1-定义} {#1-定义-1-定义}

- _自然变换_* η: F ⇒ G:

对每个对象A，给出 η_A: F(A) → G(A)

- _自然性_*: 对任意 f: A→B，交换图成立:

```text
F(A) ─F(f)─→ F(B)
 │            │
η_A           η_B
 │            │
 ↓            ↓
G(A) ─G(f)─→ G(B)
```

### 4 2 例子 {#2-例子} {#2-例子-2-例子}

- _reverse: List ⇒ List_*:

```haskell
reverse :: [a] -> [a]
reverse [1,2,3] = [3,2,1]

- - 自然性:
map f (reverse xs) = reverse (map f xs)
```

- _length: List ⇒ Const Nat_*:

```haskell
length :: [a] -> Nat
- - 虽然类型不同，仍是自然变换
```

## 5. 极限与余极限 {#极限与余极限} {#极限与余极限-极限与余极限}

### 5 1 极限 (Limit) {#1-极限-limit} {#1-极限-limit-1-极限-limit}

- _直觉_*: 所有对象的"公共部分"

- _例子_*:

1. **积** A×B:

   ```text
   ∀C, f:C→A, g:C→B
   ∃!h:C→A×B
   ```

2. **等化子**: f,g: A⇉B

   ```text
   Eq(f,g) = {a∈A | f(a)=g(a)}
   ```

### 5 2 余极限 (Colimit) {#2-余极限-colimit} {#2-余极限-colimit-2-余极限-colimit}

- _直觉_*: 所有对象的"粘合"

- _例子_*:

1. **余积** A+B:

   ```text
   和类型 Either A B
   ```

2. **余等化子**: f,g: A⇉B

   ```text
   B / ~ （商）
   ```

### 5 3 与不动点的联系 {#3-与不动点的联系} {#3-与不动点的联系-3-与不动点的联系}

- _定理_*: 在有限极限范畴中，

```text
初始代数 μF = colim_{n} Fⁿ(0)
           = F(0) + F²(0) + F³(0) + ...
```

- _应用_*: RE语言

```text
L(G) = colim_{n} L_n
     = L₀ ∪ L₁ ∪ L₂ ∪ ...
```

## 6. 为什么范畴论重要 {#为什么范畴论重要} {#为什么范畴论重要-为什么范畴论重要}

### 6 1 统一数学 {#1-统一数学} {#1-统一数学-1-统一数学}

- _范畴论提供_*:

- 统一语言（对象、态射）

- _例子_*: 同构的统一定义

```text
在Set: 双射
在Grp: 群同构
在Top: 同胚
在范畴论: 可逆态射

统一定义: ∃g: g∘f=id, f∘g=id
```

### 6 2 Curry-Howard-Lambek {#2-curry-howard-lambek} {#2-curry-howard-lambek-2-curry-}

- _三位一体_*:

```text
逻辑 ≅ 类型论 ≅ 范畴论

命题 ≅ 类型 ≅ 对象
证明 ≅ 项 ≅ 态射
```

- _范畴论视角_*: 都是CCC (笛卡尔闭范畴)

### 6 3 程序设计 {#3-程序设计} {#3-程序设计-3-程序设计}

- _Haskell_*:

```haskell
class Functor f where
  fmap :: (a -> b) -> f a -> f b

class Monad m where
  return :: a -> m a
  (>>=) :: m a -> (a -> m b) -> m b
```

- _范畴论_*:

- Functor = 函子
- Monad = 幺半范畴中的幺半对象

### 15.2 核心代码示例

```lean4
-- 范畴的公理
variable {C : Type*} [Category C]

-- 恒等态射的单位律
theorem id_comp' {X Y : C} (f : X ⟶ Y) : 𝟙 X ≫ f = f := by
  rw [Category.id_comp]

-- 同构的定义
structure Iso' {C : Type*} [Category C] (X Y : C) where
  hom : X ⟶ Y
  inv : Y ⟶ X
  hom_inv_id : hom ≫ inv = 𝟙 X
  inv_hom_id : inv ≫ hom = 𝟙 Y

-- 函子的定义
def Functor' (C D : Type*) [Category C] [Category D] := C ⥤ D

-- 自然变换的定义
structure NatTrans' {C D : Type*} [Category C] [Category D]
    (F G : C ⥤ D) where
  app : ∀ X, F.obj X ⟶ G.obj X
  naturality : ∀ {X Y} (f : X ⟶ Y),
    F.map f ≫ app Y = app X ≫ G.map f
```

### 15.3 Curry-Howard-Lambek对应

```lean4
-- 类型范畴是笛卡尔闭的
def TypeCCC : CartesianClosed where
  terminal := PUnit
  prod := fun A B => {
    prod := A × B
    π₁ := fun p => p.1
    π₂ := fun p => p.2
    universal := fun f g =>
      ⟨fun x => (f x, g x), by simp, fun h hh => by funext x; simp [hh]⟩
  }
  exp := fun A B => A → B
  eval := fun A B p => p.1 p.2
  curry := fun f => fun x y => f (x, y)
```
