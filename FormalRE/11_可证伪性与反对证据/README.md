**波普尔的核心洞察**:

> "一个理论如果不能被证伪，就不是科学"

**应用于递归范式**:

```text
问题:
- 如何证明"递归范式"是错误的？
- 存在哪些现象递归无法解释？
- 有哪些替代范式表现更好？

如果回答不出 → 递归是教条，非科学
如果能回答 → 递归是可批判的科学理论 ✓
```

---

**11.2 反对递归范式的系统证据** ⭐⭐⭐⭐⭐

```text
五大反对证据:

1. 意识的困难问题 (Chalmers)
   - "感受质"(Qualia) 无法递归还原
   - "僵尸论证"

2. 量子纠缠的非局域性 (Bell)
   - 瞬时关联
   - 超越递归的局域性假设

3. 创造性的非算法性 (Penrose)
   - 数学直觉 vs 递归证明
   - 哥德尔论证

4. 涌现的不可还原性 (Anderson)
   - "More is Different"
   - 高层规律 ≠ 低层递归

5. 混沌的不可预测性 (Lorenz)
   - 确定性系统
   - 但递归无法预测
```

**11.5 如何证伪递归范式** ⭐⭐⭐⭐⭐

```text
可证伪预测:

P1: 如果发现不可递归定义的有效计算
   → 递归范式被证伪

P2: 如果AI实现超出图灵机能力
   → Church-Turing论题被证伪

P3: 如果意识被证明非算法
   → 递归心智理论被证伪

P4: 如果涌现被证明不可还原
   → 递归还原论被证伪

关键实验/观察:
- 量子计算突破
- AGI的创造性
- 神经科学新发现
```
