## 1. 波普尔的划界问题

### 1.1 科学 vs 伪科学

**划界问题 (Demarcation Problem)**:

```text
什么是科学？什么是伪科学？

逻辑实证主义 (1920s):
"可验证的才是科学"
→ 问题: 普遍命题无法完全验证

波普尔 (1934):
"可证伪的才是科学"
→ 即使未被证伪，也要能够被证伪
```

**经典案例**:

| 理论 | 可证伪性 | 波普尔判断 |
|------|----------|-----------|
| 牛顿力学 | 高 (可预测) | 科学 ✓ |
| 相对论 | 高 (日食观测) | 科学 ✓ |
| 达尔文进化论 | 中 (化石证据) | 科学 ✓ |
| 精神分析 (弗洛伊德) | 低 (总能解释) | 伪科学 ✗ |
| 马克思主义 (教条版) | 低 (总能解释) | 伪科学 ✗ |
| 占星术 | 无 (模糊预测) | 伪科学 ✗ |

---

### 1.2 可证伪性原则

**核心逻辑**:

```text
科学理论 T:

1. T 做出明确预测 P
2. P 可以被观察/实验检验
3. 如果观察 ¬P，则 T 被证伪
4. 如果 T 无法被任何可能观察证伪 → 非科学

关键:
- 不是要求"已被证伪"
- 而是"能够被证伪"
- 可证伪性 = 冒风险
```

**例子**:

```text
牛顿: "F = ma"
预测: 苹果以a = g = 9.8m/s²下落
可证伪: 测量 a ≠ 9.8 → 牛顿错
结果: 在地球表面，测量 a ≈ 9.8 ✓

精神分析: "俄狄浦斯情结"
预测: ??? (模糊)
可证伪: 任何行为都可解释为"情结"
结果: 无法证伪 → 伪科学
```

---

### 1.3 历史案例

**爱因斯坦 vs 牛顿**:

```text
1919年日食观测:
- 爱因斯坦预测: 光线弯曲1.7角秒
- 牛顿预测: 0.87角秒
- 观测结果: ~1.7角秒

→ 牛顿被证伪 (在强引力场)
→ 爱因斯坦通过检验 (暂时)

波普尔评价:
"这是科学的典范时刻"
```

## 2. 递归范式的可证伪性检验

### 2.1 Church-Turing论题

**是否可证伪？**

```text
论题: "所有有效计算 = 图灵可计算"

预测:
P1: 不存在超图灵计算
P2: 人脑可计算性不超过图灵机
P3: 物理宇宙可模拟为图灵机

可证伪条件:
F1: 发现物理过程超出图灵机
F2: 证明人脑可解决停机问题
F3: 量子计算超出递归可枚举

当前状态:
- F1: 未发现 ✓
- F2: 未证明 ✓
- F3: 量子仍在RE内 ✓

→ Church-Turing是可证伪的科学命题
→ 至今未被证伪
```

Tier 1 (数学/计算机):

- 可以构造反例

| **逻辑矛盾** | 递归定义自相矛盾 | 未发现 ✓ | <0.1% | Tier 1全部 |
| **量子超越RE** | BQP ⊄ RE 或 真随机 | 未证明 | 5% | Tier 1部分 |

**形式化证伪条件**:

```coq
(* Coq形式化 *)
Definition falsifiable_CT : Prop :=
  (* 存在可计算函数不是递归可枚举 *)
  exists f : nat -> nat,
    computable(f) /\ ~ recursively_enumerable(f).

Theorem CT_safe : ~ falsifiable_CT.
Proof.
  (* 至今无人证明这个存在性 *)
Admitted.
```

## 5. 拉卡托斯的科学研究纲领

### 5.1 硬核 vs 保护带

**拉卡托斯修正波普尔**:

```text
波普尔: 一次证伪就推翻理论
拉卡托斯: 理论有"硬核"和"保护带"

硬核 (Hard Core):
- 不可修改的核心假设
- 被保护带保护

保护带 (Protective Belt):
- 可修改的辅助假设
- 吸收反常，保护硬核

例子: 牛顿纲领
硬核: F=ma, 万有引力
保护带: 天王星轨道偏差 → 发现海王星
→ 保护带成功，硬核保留
```

---

### 5.2 递归范式的硬核

**递归研究纲领**:

```text
硬核 (不可动摇):
1. Church-Turing论题
2. 可计算 = 递归可枚举
3. 算法的存在性

保护带 (可修改):
1. 具体形式系统 (Petri/Lambda/类型论等)
2. 复杂度类 (P/NP/BQP等)
3. 跨学科应用 (DNA/市场/意识等)

反常处理:
- 量子计算 → 扩展为BQP (保护带调整)
- 混沌系统 → 承认不可预测性 (保护带调整)
- 意识困难 → 承认未解 (保护带调整)

→ 硬核至今未动摇
```

---

### 5.3 进步性 vs 退化性

**拉卡托斯判断标准**:

```text
进步性研究纲领:
- 新理论预测新现象
- 部分预测被证实
- 解决旧问题 + 提出新问题

退化性研究纲领:
- 只解释已知现象
- 不预测新现象
- ad hoc修改
```

**递归范式评估**:

```text
进步性证据:
✓ 1936: 预测不可判定问题
✓ 1970s: 预测P vs NP
✓ 1990s: 预测BQP
✓ 2010s: 预测AI可计算性边界

→ 递归范式是进步性的 ✓

但警告:
⚠️ 意识/宇宙等跨学科扩展
   可能是退化性的 (无新预测)
```

## 6. 费耶阿本德的挑战

**反对方法论 (Against Method)**:

```text
费耶阿本德更激进:
"科学无统一方法"
"怎么都行" (Anything Goes)

对递归范式:
- 为何一定要"递归"？
- 为何不能用其他框架？
- 范式多元主义
```

**我们的回应**:

```text
✓ 承认范式多元性
✓ 研究替代框架 (11.4)
✓ 不主张递归垄断

但坚持:
- 当前递归范式最成功
- 这是经验事实，非教条
```
