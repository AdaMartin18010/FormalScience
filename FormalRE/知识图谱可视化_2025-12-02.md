### 递归构造的统一模式

```text
┌─────────────────────────────────────────────────┐
│  所有系统共享: μX.F(X) = LUB{Fⁿ(⊥)}              │
├─────────────────────────────────────────────────┤
│                                                 │
│  文法:    L(G) = μX. (T ∪ derive(X))            │
│  Petri:   Reach = μX. (M₀ ∪ next(X))           │
│  λ演算:   Eval = μX. ({M} ∪ reduce(X))          │
│  集合论:  V = ⋃_{α} V_α                         │
│  范畴论:  μF = 初始F-代数                        │
│                                                 │
└─────────────────────────────────────────────────┘
```

```text
       逻辑                类型论              范畴论

    命题 A ←─────────────→ 类型 A ←──────────→ 对象 A
```
