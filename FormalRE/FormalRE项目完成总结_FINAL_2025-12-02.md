```text
逻辑 (命题、证明)
  ≅

  ≅

→ 三者同构，不只是类比

```

```text
自指 + 对角化
  → 停机、Russell、Gödel
  → Rice定理、死锁、CH独立
  → 理论限制 → 工程应对
  → Chaos Engineering, 测试, 监控
```

### 洞察4: 表达力与安全的根本权衡

```text
简单类型 → 依赖类型 → 无类型
可判定   →   需证明   → 不可判定
弱表达   →    强大    →   最强

→ 无完美方案，只有权衡选择
→ Dhall, Rust, Coq, Python 各有定位
```
