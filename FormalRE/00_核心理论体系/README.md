## 🔑 核心概念速查

### 递归可枚举 (RE)

一个语言L是递归可枚举的，当且仅当存在图灵机M，使得：

- w ∈ L ⟹ M接受w
- w ∉ L ⟹ M可能拒绝w或永不停机

**关键特征**: 可半判定（semi-decidable）

### 递归集 (Recursive)

一个语言L是递归的，当且仅当L和其补集L̄都是RE的。

**关键特征**: 可判定（decidable）

### Church-Turing论题

所有"有效可计算"的函数都可以被图灵机计算。

**强形式**: 物理可计算 = 图灵可计算

### Curry-Howard对应

| 逻辑 | 类型论 | 范畴论 |
|------|--------|--------|
| 命题A | 类型A | 对象A |
| 证明p:A | 项t:A | 态射f:1→A |
| A→B | A→B | Aᴮ (指数对象) |
| A∧B | A×B | 积A×B |
| A∨B | A+B | 余积A+B |

## 🔬 关键定理列表

### T0.1: Church-Turing论题（非形式化）

所有直觉上"有效可计算"的函数都图灵可计算。

### T0.2: 图灵机与λ演算等价

**定理**: 图灵可计算函数类 = λ可定义函数类

**证明思路**:

- 图灵机 → λ演算: 编码磁带为列表，用Y组合子模拟循环
- λ演算 → 图灵机: 将β规约编码为图灵机状态转移

### T0.3: Curry-Howard同构

**定理**: 简单类型λ演算的类型 ≅ 直觉主义命题逻辑的命题

**证明**: 通过自然演绎系统的引入/消去规则与λ项的构造/解构一一对应

### T0.4: Knaster-Tarski不动点定理

**定理**: 设(L,≤)是完备格，f:L→L单调，则f有最小不动点μf = ⋂{x∈L | f(x)≤x}

**应用**: 所有递归定义都可以理解为不动点

### T0.5: Scott不动点定理

**定理**: 设D是CPO，f:D→D连续，则f有最小不动点lfp(f) = ⋃ₙfⁿ(⊥)

**证明**: 利用CPO的完备性和连续函数保持极限

## 💡 深刻洞察

### 洞察1: 递归的本质是不动点

所有"递归可枚举"的构造，本质上都在寻找函数的最小不动点：

```text
L(G) = μX. (终结符集 ∪ 由X产生的符号串)
Reach(N) = μX. (初始标识 ∪ X的一步后继)
Eval(M) = μX. (M ∪ X的β归约)
```

### 洞察2: 计算即证明

Curry-Howard对应揭示：

- 写程序 = 构造证明
- 类型检查 = 证明验证
- 程序运行 = 证明标准化（归约到范式）

这不是类比，而是**同构**！

### 洞察3: 不可判定性的统一根源

所有不可判定问题（停机、可达性、语言空性）都源于同一根源：
**自指（Self-reference）+ 对角化（Diagonalization）**

这是哥德尔不完备定理、停机问题、Russell悖论的共同本质。
