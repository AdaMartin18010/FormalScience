## 🎯 核心立场声明

**FormalRE的三层哲学框架**:

```text
1️⃣ 本体论: 互动涌现论
   递归 = 认知-世界协同进化的产物

2️⃣ 认识论: 批判实在论 + 分层客观性
   递归 = 有效接口 (Tier 1⭐⭐⭐⭐⭐ → Tier 4⭐⭐)

3️⃣ 范式论: 库恩历史主义
   递归 = 当前范式核心 (1936-2025, 未来可替代)
```

**一句话总结**:

> 递归是**有效的、有限的、历史的、可替代的**

---

## ⚠️ 阅读前须知

**我们不主张**:

```text
✗ 递归是世界的"本质"或"真理"
✗ 递归是永恒不变的
✗ 一切现象都可递归解释
✗ 递归范式不可被证伪
```

**我们主张**:

```text
✓ 递归是当前最成功的科学范式 (1936-2025)
✓ 在Tier 1-2有强客观性支持
✓ 在Tier 3-4是有效解释框架
✓ 存在反常现象与替代范式
✓ 未来可能被新范式取代
✓ 科学诚实 > 范式忠诚
```
