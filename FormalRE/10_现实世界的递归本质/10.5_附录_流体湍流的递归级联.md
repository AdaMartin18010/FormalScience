## 1. Navier-Stokes方程

**流体运动的基本定律**:

```text
∂v/∂t + (v·∇)v = -∇p/ρ + ν∇²v + f

v: 速度场
p: 压力
ρ: 密度
ν: 粘度

特点:
- 完全确定性
- 递归时间演化
- 但湍流极其复杂
```

## 2. Richardson能量级联

**"大涡吃小涡"** (1922):

```text
湍流过程:
1. 大尺度涡旋 (L ~ 1m)
2. 失稳 → 分裂为小涡 (L ~ 0.1m)
3. 小涡再分裂 (L ~ 0.01m)
4. ...递归下去...
5. 直到粘性尺度 (Kolmogorov长度)

→ 能量从大尺度递归传递到小尺度
→ 最终耗散为热
```

**递归形式**:

```text
E(L) → E(L/2) + E(L/4) + E(L/8) + ...

自相似性:
不同尺度的湍流结构相似

→ 分形维度 ≈ 5/3 (Kolmogorov)
```

## 3. Kolmogorov湍流理论

**K41理论** (1941):

```text
假设:
- 统计各向同性
- 局域性
- 标度不变性

预测:
E(k) ∼ k^{-5/3}  (能谱)

实验:
✓ 大量验证

→ 湍流的递归统计规律
```

## 4. 递归结构

**湍流的多层递归**:

```text
1. 几何递归:
   涡旋 → 子涡旋 → 孙涡旋

2. 能量递归:
   E_n → E_{n+1} (级联)

3. 尺度递归:
   L → L/2 → L/4 → ...

4. 动力学递归:
   方程在不同尺度自相似
```
