## 1. 免疫的递归挑战

**问题**:

```text
病原体: 10²⁰+ 可能抗原
人类基因: ~2万个

矛盾:
如何用有限基因编码无限多样性？

解决: 递归生成算法 ✓
```

---

## 2. V(D)J重组：组合递归

**机制** (Tonegawa, 1987诺贝尔奖):

```text
抗体基因片段:
V段: ~40个选择
D段: ~25个选择
J段: ~6个选择

组合:
Antibody = V_i + D_j + J_k
可能性: 40 × 25 × 6 = 6,000

重链+轻链:
6,000 × 6,000 = 3600万种

→ 递归组合生成
```

## 3. 体细胞超变：进化递归

**机制**:

```text
B细胞激活后:
1. 抗体基因高速突变
2. 亲和力选择
3. 存活B细胞继续突变
4. 循环迭代

数学:
Affinity_t+1 = Select(Mutate(Affinity_t))

→ 递归进化算法
→ 类似遗传算法 (GA)
```

---

## 4. 克隆选择：递归放大

**Burnet理论 (1957)**:

```text
选择机制:
1. 抗原结合B细胞
2. 该B细胞增殖 (2^n倍)
3. 后代继续突变
4. 再选择

递归:
Population_t+1 = Expand(Select(Population_t))

指数增长:
1个B细胞 → 10⁶个后代 (7天)
```

## 5. 免疫记忆：递归存储

**长期免疫**:

```text
记忆B细胞:
- 首次感染后保留
- 可存活数十年
- 再次激活更快

递归性:
Memory = f(Previous_exposure, Current_antigen)

→ 历史依赖的递归响应
```
