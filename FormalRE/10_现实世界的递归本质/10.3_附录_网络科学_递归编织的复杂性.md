## 1. 网络的递归生成

### 1.1 优先连接 (Preferential Attachment)

**"富者愈富"机制**:

```text
网络增长规则:
1. t=0: 初始m₀个节点
2. t=t+1: 添加新节点
3. 连接概率: P(i) ∝ kᵢ (度数)

→ 递归过程:
Gₜ₊₁ = Gₜ + new_node(选择∝度)

结果: 无标度 (Power Law)
P(k) ∼ k^{-γ}
```

---

### 1.2 BA模型

**Barabási-Albert (1999)**:

```text
参数: m (每次添加m条边)

递归生成:
t=0: m₀个全连接节点
t→t+1:
  新节点v
  选择m个旧节点 {u₁,...,uₘ}
  P(uᵢ) = kᵤᵢ / Σⱼkⱼ
  连接v到{u₁,...,uₘ}

分析解:
P(k) ∼ k^{-3} (γ=3)
```

---

### 1.3 无标度网络

**普遍性**:

```text
发现于:
- 互联网 (AS级别)
- WWW (超链接)
- 社交网络 (Facebook)
- 引用网络 (论文)
- 蛋白质互作网络

→ 递归生成机制的普遍性
```

## 2. 小世界现象

### 2.1 六度分隔

**Milgram实验 (1967)**:

```text
实验: 信件传递
- 从Kansas到Boston
- 只能发给认识的人

结果: 平均6跳

数学:
L ∼ log N (网络直径)

递归搜索:
BFS(起点, 目标, depth=6)
→ 多数可达
```

---

### 2.2 WS模型

**Watts-Strogatz (1998)**:

```text
小世界网络:
- 高聚类系数 (局部密集)
- 短路径长度 (全局连接)

构造:
1. 规则环 (邻居连接)
2. 随机重连 (概率p)

递归性:
聚类 = 局部递归结构
```

## 3. 网络上的递归动力学

### 3.1 传播过程

**SIS模型 (流行病)**:

```text
状态: S (易感) / I (感染)
规则:
  S + I →^β I + I (感染)
  I →^γ S (恢复)

网络上:
dIᵢ/dt = β Σⱼ Aᵢⱼ Iⱼ - γIᵢ

→ 耦合递归方程
```

---

### 3.2 同步现象

**Kuramoto模型**:

```text
振子: dθᵢ/dt = ωᵢ + K Σⱼ Aᵢⱼ sin(θⱼ-θᵢ)

网络结构 → 同步能力
无标度网络: 更易同步

→ 递归相互作用
```

## 5. 与递归范式的关系

**网络 vs 递归**:

```text
网络科学强调:
- 结构 (拓扑)
- 涌现 (整体性质)
- 非线性 (相变)

递归范式强调:
- 过程 (算法)
- 计算 (可枚举)
- 确定性

互补:
网络 = 递归过程的涌现结构
递归 = 网络动力学的生成机制
```

**整合**:

```text
递归生成 → 网络结构 → 涌现性质

例子:
BA递归规则 → 无标度网络 → 鲁棒性+脆弱性

→ 递归在meta-层级
```
