## 1. 传统递归计算模型

**大脑 = 计算机隐喻**:

```text
1960s-2000s主导:
- 神经元 = 逻辑门
- 突触 = 权重
- 激活 = 递归传播
- 学习 = 递归更新权重

例子:
- 反向传播 (1986)
- 递归神经网络 (RNN)
- 深度学习 (2012-)

→ 完全递归框架
```

## 2. Friston自由能原理

### 2.1 核心思想

**革命性视角**:

```text
大脑 ≠ 计算机
大脑 = 贝叶斯推理机

目标: 最小化惊奇
方法: 预测 + 误差修正

关键:
- 不是"计算"感官输入
- 而是"预测"感官输入
- 预测误差 → 更新模型
```

---

### 2.2 数学形式化

**自由能**:

```text
F = E_q[log q(x) - log p(x,y)]
  = D_KL(q(x)||p(x|y)) + log p(y)

最小化F:
- 感知: 更新q(x) (推理)
- 行动: 改变y (采样)

→ 统一感知与行动
```

---

### 2.3 预测编码实现

**层级结构**:

```text
L3: 高层预测
  ↓ 预测
L2: 中层预测 + 误差
  ↓ 预测
L1: 低层预测 + 误差
  ↓ 预测
感官输入

误差向上传播
预测向下传播

→ 双向递归？
→ 但本质是概率推理
```

## 3. 两种范式的根本差异

| 维度 | 递归计算 | 预测编码 |
|------|----------|----------|
| 本质 | 确定性变换 | 概率推理 |
| 方向 | 自下而上 | 双向 (预测+误差) |
| 目标 | 计算输出 | 最小化自由能 |
| 学习 | 梯度下降 | 贝叶斯更新 |
| 神经对应 | 激活传播 | 预测误差信号 |

**哲学差异**:

```text
递归: 世界→感官→计算→输出
预测编码: 内部模型→预测→误差→修正

→ 主动 vs 被动
→ 生成 vs 计算
```

## 4. 实验证据

**支持预测编码**:

```text
1. 惊奇反应 (MMN, Mismatch Negativity)
   - ERP脑电 100-250ms
   - 预测违反时出现

2. 注意力 (Precision weighting)
   - 高精度预测误差优先处理
   - 符合预测编码

3. 错觉 (视错觉)
   - 预测压倒输入
   - 难以用递归解释

4. 主动推理 (运动控制)
   - 运动 = 实现感官预测
   - 统一感知与行动
```

## 5. 对递归范式的挑战

**深层矛盾**:

```text
递归强调:
- 计算 (确定性)
- 算法 (步骤)
- 自下而上

预测编码强调:
- 推理 (概率性)
- 优化 (全局)
- 双向交互

能否调和？
```

## 6. 可能的调和

**整合尝试**:

```text
观点1: 预测编码可递归实现
- 贝叶斯更新 = 递归算法
- 自由能 = 递归优化
→ 仍在递归范式内

观点2: 预测编码是新范式
- 概率 ≠ 确定
- 推理 ≠ 计算
→ 超越递归

当前: 无共识
```
