## 1. 技术路线

### 1.1 硬件平台

**多路线竞争**:

```text
超导 (IBM, Google):
✓ 成熟度最高
✓ 100+ qubit
⚠️ 极低温 (~10mK)
⚠️ 相干时间短 (~100μs)

离子阱 (IonQ, Honeywell):
✓ 高保真度 (99.9%+)
✓ 长相干 (~秒)
⚠️ 规模化难
⚠️ 速度慢

光学 (Xanadu):
✓ 室温运行
✓ 网络友好
⚠️ 测量困难
⚠️ 非通用 (目前)

拓扑 (Microsoft):
✓ 本征容错 (理论)
⚠️ 实验极难
✗ 0个qubit (2024)

中性原子 (QuEra):
✓ 可扩展性好
✓ 100+ qubit
⚠️ 新兴技术

递归:
✓ 技术递归竞争
✓ 路线递归收敛/分化
```
