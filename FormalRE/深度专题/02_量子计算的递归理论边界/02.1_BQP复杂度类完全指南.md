## 1. BQP精确定义

```text
BQP (Bounded-error Quantum Polynomial time):

定义:
L ∈ BQP ⟺ ∃量子算法Q, poly时间
  x ∈ L → P(Q接受) ≥ 2/3
  x ∉ L → P(Q接受) ≤ 1/3

关键参数:
- 误差: ε = 1/3 (可指数降低)
- 时间: 多项式
- 空间: 多项式qubits

误差放大:
运行k次取多数 → ε' = e^{-Ω(k)}
```

## 4. BQP vs NP关系

```text
问题: BQP vs NP?

可能情况:
1. BQP ⊂ NP (量子≤非确定)
   概率: 20%

2. NP ⊂ BQP (非确定≤量子)
   概率: 10%

3. BQP ∦ NP ∧ NP ∦ BQP (不可比)
   概率: 70% ✓ (主流)

Oracle分离:
∃Oracle A: NP^A ⊄ BQP^A
∃Oracle B: BQP^B ⊄ NP^B
→ 相对化意义下不可比

实践含义:
✓ 量子≠万能
✓ NP完全仍困难 (可能)
```
