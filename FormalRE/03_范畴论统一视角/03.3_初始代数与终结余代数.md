## 1. F-代数 {#f-代数} {#f-代数-f-代数}

### 1 1 定义 {#1-定义} {#1-定义-1-定义}

- _函子_*: $F: \mathcal{C} \to \mathcal{C}$

- _F-代数_*: 对 $(A, \alpha)$，其中：

- $A$: 对象（载体 carrier）
- $\alpha: F(A) \to A$（结构态射 structure map）

- _形式化_*:

$$\text{F-Alg}(F) = \{(A, \alpha) \mid A \in \mathcal{C}, \alpha: F(A) \to A\}$$

  ↓            ↓

## 2. 初始代数 {#初始代数} {#初始代数-初始代数}

### 2 1 定义 {#1-定义} {#1-定义-1-定义}

- _(μF, in)_* 是**初始F-代数**，如果：

对任意 F-代数 (A, α)，存在唯一态射 fold_α: μF → A 使得：

```text
F(μF) ─^F(fold)→ F(A)
   │               │
  in│               │α
   ↓               ↓
  μF ─────^fold──→ A
```

### 2 2 例子: 自然数 {#2-例子-自然数} {#2-例子-自然数-2-例子-自然数}

- _F(X) = 1 + X_*

- _初始代数_*: (ℕ, [zero, succ])

```text
in: 1 + ℕ → ℕ
in(inl(*)) = 0
in(inr(n)) = n+1
```

- *fold (归约)**: 给定 (A, α)，fold: ℕ → A

```text
fold(0) = α(inl(*))  (零的情况)
fold(n+1) = α(inr(fold(n)))  (归纳步)
```

- _例子_*: 加法

```text
plus_m: ℕ → ℕ
plus_m = fold_{(ℕ, [m, succ])}

plus_m(0) = m
plus_m(n+1) = succ(plus_m(n)) = plus_m(n) + 1
```

## 3. Lambek引理 {#lambek引理} {#lambek引理-lambek引理}

### 3 1 定理陈述 {#1-定理陈述} {#1-定理陈述-1-定理陈述}

- *定理3.1 (Lambek)**:

如果 (μF, in) 是初始F-代数，则：

```text
in: F(μF) ≅ μF  (同构!)
```

- _含义_*: 初始代数与其"展开"**同构**

### 3 2 证明 {#2-证明} {#2-证明-2-证明}

- _构造逆_*: out = fold_{(F(μF), id)} : μF → F(μF)

- _验证_* in ∘ out = id:

```text
F(μF) ─^F(out)→ F(F(μF))
   │               │
  in│               │F(in)
   ↓               ↓
  μF ─────^out──→ F(μF)
   ↓               ↓
   in ∘ out       F(in) ∘ F(out) = F(in ∘ out)

由初始性: in ∘ out = id  □
```

### 3 3 深刻含义 {#3-深刻含义} {#3-深刻含义-3-深刻含义}

- _递归类型 = 不动点_*:

```text
List A = 1 + (A × List A)
       = F(List A)  其中 F(X) = 1 + A×X
```

- _Lambek_*: List A ≅ F(List A)

- 列表同构于其"展开"

## 4. 归纳类型 {#归纳类型} {#归纳类型-归纳类型}

### 4 1 归纳类型的范畴论定义 {#1-归纳类型的范畴论定义} {#1-归纳类型的范畴论定义-1-归纳类型的范畴论定义}

- _归纳类型_* = 初始代数 μF

- _例子_*:

| 类型 | 函子F | 初始代数 |
|------|-------|----------|
| ℕ | 1 + X | (ℕ, [0, succ]) |
| List A | 1 + A×X | ([A], [[], cons]) |
| Tree A | A + X×X | (Tree, [leaf, node]) |

### 4 2 递归函数 = fold {#2-递归函数--fold} {#2-递归函数--fold-2-递归函数--fold}

- _原理_*: 归纳类型上的函数 = F-代数态射

```haskell
- - Haskell的foldr就是范畴论的fold!
foldr :: (a -> b -> b) -> b -> [a] -> b
foldr f z = fold_{(b, [z, f])}
```

## 5. 余归纳类型 {#余归纳类型} {#余归纳类型-余归纳类型}

### 5 1 终结余代数 {#1-终结余代数} {#1-终结余代数-1-终结余代数}

- _对偶_*: (νF, out) 是**终结F-余代数**

对任意余代数 (A, α: A → F(A))，存在唯一 unfold: A → νF

- _Lambek对偶_*: out: νF ≅ F(νF)

### 5 2 无限数据结构 {#2-无限数据结构} {#2-无限数据结构-2-无限数据结构}

- _Stream = 余归纳类型_*:

```text
F(X) = A × X  (总有头和尾)
νF = Stream A = A × A × A × ... (无限流)
```

- *unfold (对偶fold)**:

```haskell
unfold :: (b -> (a, b)) -> b -> Stream a
unfold f seed = out(seed) : unfold f (snd (f seed))
  where out = fst ∘ f
```
