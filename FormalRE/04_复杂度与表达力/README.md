## 🔑 关键定理列表

### T4.1: Chomsky层级严格性

```text
正则 ⊊ 上下文无关 ⊊ 上下文有关 ⊊ RE ⊊ 所有语言
```

**证明方法**:

- 泵引理
- 对角化

### T4.2: Post定理

```text
A ∈ Σn ⟺ ∃ Σn-1 公式 φ(x) 定义A
A ∈ Πn ⟺ ∀ Πn-1 公式 φ(x) 定义A
```

### T4.3: Rice定理

```text
设 P 是 RE 语言的非平凡性质（语义性质）
则 {⟨M⟩ | L(M) ∈ P} 不可判定
```

### T4.4: 停机问题不可判定

```text
HALT = {⟨M,w⟩ | M 在 w 上停机} ∉ R
但 HALT ∈ RE
```

## ⚠️ 学习注意事项

### 常见误解

❌ **误解1**: "不可判定 = 不能求解"

- **纠正**: 不可判定 = 无通用算法，特定实例可能可解

❌ **误解2**: "RE = 可计算"

- **纠正**: RE = 半可判定，递归集 = 可判定

❌ **误解3**: "Rice定理说所有性质不可判定"

- **纠正**: 只针对**语义**性质，语法性质可能可判定

### 学习建议

✅ **建议1**: 动手构造证明

- 尝试自己证明泵引理
- 理解对角化技巧

✅ **建议2**: 关注边界

- 什么可判定 vs 不可判定
- 为什么差一点就不行

✅ **建议3**: 连接实践

- 思考编译器、静态分析的限制
- 理解测试的必要性
