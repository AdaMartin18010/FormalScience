## 2. 第一个证明 {#第一个证明}

### 3.1 Coq示例 {#coq示例}

```coq
(* first_proof.v *)

(* 导入库 *)
Require Import Coq.Arith.Arith.

(* 第一个定理: 0 + n = n *)
Theorem plus_0_n : forall n : nat,
  0 + n = n.
Proof.
  intro n.      (* 引入n *)
  simpl.        (* 化简 *)
  reflexivity.  (* 反射性 *)
Qed.

(* 归纳证明: n + 0 = n *)
Theorem plus_n_0 : forall n : nat,
  n + 0 = n.
Proof.
  induction n as [| n' IHn'].
  - (* n = 0 *)
    reflexivity.
  - (* n = S n' *)
    simpl.
    rewrite IHn'.
    reflexivity.
Qed.
```

### 3.2 Lean 4示例 {#lean-4示例}

```lean
- - First.lean

- - 第一个定理
theorem zero_add (n : Nat) : 0 + n = n := by
  rfl

- - 归纳证明
theorem add_zero (n : Nat) : n + 0 = n := by
  induction n with
  | zero => rfl
  | succ n ih =>
    simp [Nat.add_succ]
    exact ih
```

## 3. 常用策略 {#常用策略}

### 4.1 Coq策略 {#coq策略}

```coq
intro / intros    (* 引入假设 *)
split             (* 分解合取 *)
left / right      (* 选择析取 *)
exists x          (* 提供存在见证 *)
destruct H        (* 分情况 *)
induction n       (* 归纳 *)
rewrite H         (* 重写 *)
reflexivity       (* 相等 *)
apply H           (* 应用定理 *)
simpl             (* 化简 *)
auto / trivial    (* 自动 *)
```

### 4.2 Lean 4策略 {#lean-4策略}

```lean
intro / intros    -- 引入
constructor       -- 构造
cases h           -- 分情况
induction h       -- 归纳
rw [h]            -- 重写
rfl               -- 反射
exact h           -- 精确提供
simp              -- 化简
assumption        -- 使用假设
```
