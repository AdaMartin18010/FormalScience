## 1. 定理陈述 {#定理陈述}

- _Rice定理_* (1951/1953):

```text
设P是RE语言的性质
若P非平凡 (有语言满足,有语言不满足)
则P不可判定

即: 不存在算法判定"程序是否满足P"
```

### 2 2 非平凡性 {#2-非平凡性}

```lean
- - 非平凡 = 存在满足和不满足的
def NonTrivial (P : SemanticProperty) : Prop :=
  (∃ f, P f) ∧ (∃ g, ¬ P g)

- - 例: AlwaysHalts是非平凡的
theorem always_halts_nontrivial : NonTrivial AlwaysHalts := by
  constructor
  · -- 存在总停机的函数
    use (fun n => some 0)
    intro n
    use 0
    rfl
  · -- 存在不总停机的函数
    use (fun _ => none)
    intro h
    specialize h 0
    cases h with
    | intro m hm => contradiction
```

## 3. 证明策略 {#证明策略}

- _归约步骤_*:

```text
1. 假设存在P的判定器 D
2. 取f₀满足P, f₁不满足P (非平凡性)
3. 对任意程序M和输入x, 构造M':

   M'(n) =
     if M(x)停机
     then f₀(n)  (满足P)
     else f₁(n)  (不满足P)

4. 则: M(x)停机 ⟺ D(M')=true
5. → 停机判定器 (矛盾!)
```

- _Lean实现_*:

```lean

import Mathlib.Computability.Halting

namespace RiceTheorem

- - 程序与语义 (见上)

- - 停机问题不可判定 (引用)
axiom halting_undecidable :
  ¬ ∃ dec : Program → Nat → Bool,
    ∀ M x, dec M x = true ↔ halts M x

- - Rice定理主证明
theorem rice
    (P : SemanticProperty)
    (hNT : NonTrivial P) :
    ¬ ∃ decide : Program → Bool,
      ∀ p, decide p = true ↔ P (semantics p) := by
  intro ⟨decide, h⟩

  - - 构造停机判定器
  have halt_dec := rice_reduces_halting P hNT decide h

  - - 矛盾
  exact halting_undecidable halt_dec
```
