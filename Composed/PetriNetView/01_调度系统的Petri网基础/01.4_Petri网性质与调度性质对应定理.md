## 1 对应关系概述

**核心思想**：Petri网的结构性质与调度系统的行为性质之间存在**系统化的对应关系**。

**对应关系表**：

| Petri网性质 | 调度系统性质 | 实际意义 |
|-----------|------------|---------|
| 可达性（Reachability） | 调度可行性 | 系统能否到达目标状态 |
| 活性（Liveness） | 无死锁 | 所有任务都能执行 |
| 有界性（Boundedness） | 资源有限性 | 资源不会无限增长 |
| 安全性（Safeness） | 互斥性 | 资源不会超限使用 |
| 公平性（Fairness） | 饥饿自由 | 所有任务都有机会执行 |
| 可逆性（Reversibility） | 周期性 | 系统可回到初始状态 |

## 2 可达性与调度可行性

### 2.1 定理陈述

**定理1.41（可达性-可行性对应）**：

给定调度系统 $S$ 和其Petri网表示 $PN$，初始状态 $s_0$ 映射到初始标记 $M_0$，目标状态 $s_g$ 映射到目标标记 $M_g$。则：

$$
s_g \text{ 可从 } s_0 \text{ 调度到达} \iff M_g \in R(M_0)
$$

其中 $R(M_0)$ 是从 $M_0$ 可达的标记集。

**推论1.41.1**：

调度系统存在可行调度当且仅当Petri网中存在从 $M_0$ 到 $M_g$ 的点火序列。

### 2.2 形式化证明

**证明**：

$(\Rightarrow)$ 方向（调度可行 → Petri网可达）：

1. 假设存在调度序列 $\sigma_S = \langle a_1, a_2, \ldots, a_n \rangle$ 使得 $s_0 \xrightarrow{\sigma_S} s_g$
2. 根据映射定义，每个调度动作 $a_i$ 对应变迁 $t_i$
3. 变迁序列 $\sigma_{PN} = \langle t_1, t_2, \ldots, t_n \rangle$
4. 由于映射保持前置条件，如果 $a_i$ 在 $s_{i-1}$ 可执行，则 $t_i$ 在 $M_{i-1}$ 使能
5. 因此 $M_0 \xrightarrow{\sigma_{PN}} M_g$，即 $M_g \in R(M_0)$

$(\Leftarrow)$ 方向（Petri网可达 → 调度可行）：

1. 假设存在点火序列 $\sigma_{PN} = \langle t_1, t_2, \ldots, t_n \rangle$ 使得 $M_0 \xrightarrow{\sigma_{PN}} M_g$
2. 根据映射的逆，每个变迁 $t_i$ 对应调度动作 $a_i$
3. 调度序列 $\sigma_S = \langle a_1, a_2, \ldots, a_n \rangle$
4. 由于映射保持语义，$s_0 \xrightarrow{\sigma_S} s_g$
5. 因此调度可行 ∎

### 2.3 Coq形式化验证

```coq
(* Coq形式化验证：可达性-可行性对应 *)
Require Import List.
Import ListNotations.

(* Petri网定义 *)
Record PetriNet : Type := {
  places : list nat;
  transitions : list nat;
  marking : nat -> nat;  (* 标记函数 *)
}.

(* 调度系统定义 *)
Record SchedulingSystem : Type := {
  states : list nat;
  actions : list nat;
  current_state : nat;
}.

(* 可达性定义 *)
Inductive reachable (pn : PetriNet) : (nat -> nat) -> Prop :=
  | reach_refl : reachable pn (marking pn)
  | reach_step : forall M M' t,
      reachable pn M ->
      enabled pn M t ->
      M' = fire pn M t ->
      reachable pn M'.

(* 调度可行性定义 *)
Inductive schedulable (s : SchedulingSystem) : nat -> Prop :=
  | sched_refl : schedulable s (current_state s)
  | sched_step : forall s1 s2 a,
      schedulable s s1 ->
      executable s s1 a ->
      s2 = execute s s1 a ->
      schedulable s s2.

(* 主定理：可达性 ↔ 调度可行性 *)
Theorem reachability_schedulability_correspondence :
  forall (s : SchedulingSystem) (pn : PetriNet) (mapping : nat -> nat),
  (* 假设映射保持语义 *)
  mapping_preserves_semantics s pn mapping ->
  (* 则可达性与调度可行性等价 *)
  forall sg Mg,
    mapping sg = Mg ->
    schedulable s sg <-> reachable pn Mg.
Proof.
  intros s pn mapping H_mapping sg Mg H_eq.
  split.
  - (* 调度可行 -> Petri网可达 *)
    intro H_sched.
    induction H_sched.
    + (* 基础情况：初始状态 *)
      rewrite <- H_eq.
      apply reach_refl.
    + (* 归纳情况：执行一步 *)
      apply reach_step with (M := mapping s1) (t := mapping_action a).
      * apply IHH_sched.
      * (* enabled条件由mapping_preserves_semantics保证 *)
        apply (mapping_preserves_enabled s pn mapping H_mapping s1 a H).
      * (* fire结果由mapping保持 *)
        reflexivity.
  - (* Petri网可达 -> 调度可行 *)
    intro H_reach.
    induction H_reach.
    + (* 基础情况 *)
      apply sched_refl.
    + (* 归纳情况 *)
      apply sched_step with (s1 := inv_mapping M) (a := inv_mapping_action t).
      * apply IHH_reach.
      * (* executable条件由mapping保证 *)
        apply (inv_mapping_preserves_executable s pn mapping H_mapping M t H).
      * reflexivity.
Qed.
```

## 3 活性与无死锁

### 3.1 定理陈述

**定理1.42（活性-无死锁对应）**：

调度系统 $S$ **无死锁**当且仅当其Petri网表示 $PN$ 是**活的**（L4-live）。

$$
S \text{ 无死锁} \iff PN \text{ 活的}
$$

**更精确的陈述**：

$$
\forall M \in R(M_0), \forall t \in T, \exists M' \in R(M): M'[t\rangle
$$

即：从任何可达状态，任何变迁最终都能被使能。

### 3.2 形式化证明

**证明**：

$(\Rightarrow)$ 方向（无死锁 → 活性）：

1. 假设调度系统 $S$ 无死锁
2. 即：从任何可达状态，至少存在一个任务可执行
3. 对于Petri网中的任意变迁 $t$ 和任意可达标记 $M$：
   - 如果 $t$ 对应的任务在某个状态可执行
   - 则存在路径从 $M$ 到达使能 $t$ 的标记 $M'$
4. 因此 $\forall t, \forall M \in R(M_0), \exists M' \in R(M): M'[t\rangle$
5. 即Petri网是活的 ∎

$(\Leftarrow)$ 方向（活性 → 无死锁）：

1. 假设Petri网 $PN$ 是活的
2. 对于任意可达标记 $M$，任意变迁 $t$ 最终都能使能
3. 映射到调度系统：从任意状态，任意任务最终都能执行
4. 因此不存在死锁状态
5. 调度系统无死锁 ∎

```python
def detect_deadlock(pn, M0):
    """
    检测Petri网是否存在死锁

    Args:
        pn: Petri网
        M0: 初始标记

    Returns:
        (has_deadlock, deadlock_states)
    """
    # 构造可达性图
    RG = construct_reachability_graph(pn, M0)

    deadlock_states = []

    # 检查每个可达状态
    for M in RG.nodes():
        # 检查是否有使能的变迁
        enabled_transitions = [t for t in pn.transitions if is_enabled(pn, M, t)]

        if not enabled_transitions:
            # 没有使能的变迁 = 死锁状态
            deadlock_states.append(M)

    has_deadlock = len(deadlock_states) > 0

    return has_deadlock, deadlock_states

    for M in deadlock_states:
        print(f"  Deadlock state: {M}")
else:
    print("✓ No deadlock - system is live")
```

```python
def compute_s_invariants(pn):
    """
    计算S-不变量（P-不变量）

    S-不变量是库所的加权和，在所有可达标记下保持不变

    Returns:
        不变量列表
    """
    import numpy as np

    # 构造关联矩阵C
    num_places = len(pn.places)
    num_transitions = len(pn.transitions)
    C = np.zeros((num_places, num_transitions))

    place_idx = {p: i for i, p in enumerate(pn.places)}
    trans_idx = {t: i for i, t in enumerate(pn.transitions)}

    for (source, target), weight in pn.arcs.items():
        if source in pn.places and target in pn.transitions:
            # 输入弧（负权重）
            C[place_idx[source], trans_idx[target]] = -weight
        elif source in pn.transitions and target in pn.places:
            # 输出弧（正权重）
            C[trans_idx[source], place_idx[target]] = weight

    # 求解 y^T * C = 0 的非负整数解
    # 使用整数线性规划或矩阵零空间计算
    # （简化实现）
    from scipy.linalg import null_space

    null_space_vectors = null_space(C.T)
    s_invariants = []

    for v in null_space_vectors.T:
        # 归一化到非负整数
        if np.all(v >= -1e-10):  # 非负向量
            # 转换为整数
            gcd_val = np.gcd.reduce([int(round(x)) for x in v * 100])
            if gcd_val > 0:
                inv = [int(round(x * 100 / gcd_val)) for x in v]
                s_invariants.append(inv)

    return s_invariants

# 使用S-不变量检测死锁
invariants = compute_s_invariants(pn)
print(f"Found {len(invariants)} S-invariants")
for i, inv in enumerate(invariants):
    print(f"  S{i}: {inv}")
```

## 4 有界性与资源有限性

### 4.1 定理陈述

**定理1.43（有界性-资源有限性对应）**：

调度系统 $S$ 的资源是**有限的**当且仅当其Petri网表示 $PN$ 是**有界的**。

$$
\forall resource \in S: capacity(resource) < \infty \iff \exists k \in \mathbb{N}: PN \text{ 是 } k\text{-有界的}
$$

**推论1.43.1**：

如果所有资源都有容量上界，则Petri网中对应库所的标记数有上界。

### 4.2 形式化证明

**证明**：

$(\Rightarrow)$ 方向（资源有限 → 有界）：

1. 假设所有资源 $r_i$ 都有容量上界 $C_i$
2. 资源 $r_i$ 映射到库所 $p_i$
3. 对于所有可达标记 $M$：
   $$M(p_i) \leq C_i$$
4. 令 $k = \max_i C_i$，则：
   $$\forall M \in R(M_0), \forall p \in P: M(p) \leq k$$
5. 因此Petri网是k-有界的 ∎

$(\Leftarrow)$ 方向（有界 → 资源有限）：

1. 假设Petri网是k-有界的
2. 即：$\forall M \in R(M_0), \forall p \in P: M(p) \leq k$
3. 库所 $p_i$ 映射到资源 $r_i$
4. 标记数 $M(p_i)$ 对应资源可用量
5. 因此 $capacity(r_i) \leq k < \infty$
6. 所有资源都是有限的 ∎

### 4.3 资源上界计算

**覆盖性图（Coverability Graph）**：

用于计算有界性和资源上界。

```python
def compute_coverability_graph(pn, M0):
    """
    构造覆盖性图

    覆盖性图用ω表示无界的标记

    Returns:
        覆盖性图和有界性判定
    """
    CG = nx.DiGraph()
    queue = [M0]
    visited = {M0}
    CG.add_node(M0)

    omega = float('inf')  # 无界标记

    while queue:
        M = queue.pop(0)

        for t in pn.transitions:
            if is_enabled(pn, M, t):
                M_new = fire_transition(pn, M, t)

                # 加速规则：检测无界性
                for M_ancestor in visited:
                    if strictly_greater(M_new, M_ancestor):
                        # 将严格增大的库所标记设为ω
                        M_new = accelerate(M_new, M_ancestor, omega)
                        break

                if M_new not in visited:
                    visited.add(M_new)
                    queue.append(M_new)
                    CG.add_node(M_new)

                CG.add_edge(M, M_new, label=t)

    # 检查是否有ω标记
    is_bounded = all(
        all(M[p] != omega for p in pn.places)
        for M in CG.nodes()
    )

    # 计算每个库所的上界
    bounds = {}
    for p in pn.places:
        max_tokens = max(
            M[p] if M[p] != omega else omega
            for M in CG.nodes()
        )
        bounds[p] = max_tokens

    return CG, is_bounded, bounds
```

## 5 公平性与饥饿自由

### 5.1 定理陈述

**定理1.44（公平性-饥饿自由对应）**：

调度系统 $S$ 是**饥饿自由的**当且仅当其Petri网表示 $PN$ 满足**公平性**。

$$
S \text{ 饥饿自由} \iff PN \text{ 满足公平点火}
$$

**公平点火定义**：

在无限点火序列中，如果变迁 $t$ 无限次使能，则 $t$ 必须无限次点火。

### 5.2 T-不变量与公平性

**定义1.45（T-不变量）**：

T-不变量是变迁的加权向量 $\mathbf{x} \in \mathbb{N}^{|T|}$，满足：

$$
C \cdot \mathbf{x} = \mathbf{0}
$$

其中 $C$ 是关联矩阵。

**性质**：如果点火序列按T-不变量点火，则回到初始标记。

**Python实现**：

```python
    for i in range(len(pn.transitions))

    print("✓ Fair - all transitions covered by T-invariants")
else:
    print("✗ Unfair - some transitions never fire")
```

## 6 可逆性与调度循环

**定理1.45（可逆性-循环对应）**：

调度系统 $S$ 是**周期性的**当且仅当其Petri网表示 $PN$ 是**可逆的**。

$$
S \text{ 周期性} \iff \forall M \in R(M_0): M_0 \in R(M)
$$

**应用**：周期任务调度、资源循环利用。

```python
class PetriNetVerifier:

        self.verify_safeness()
        self.verify_liveness()

        """验证有界性"""
        bounds = {}
        for p in self.pn.places:
            max_tokens = max(M[p] for M in self.RG.nodes())
            bounds[p] = max_tokens

        k = max(bounds.values())
        print(f"Boundedness: {k}-bounded")
        for p, b in bounds.items():
            print(f"  {p}: max {b} tokens")

    def verify_safeness(self):
        """验证安全性"""
        is_safe = all(
            M[p] <= 1
            for M in self.RG.nodes()
            for p in self.pn.places
        )
        print(f"\nSafeness: {'✓ Safe' if is_safe else '✗ Not safe'}")

    def verify_liveness(self):
        """验证活性"""
        live_transitions = []
        dead_transitions = []

        for t in self.pn.transitions:
            # 检查是否从所有状态都能最终使能t
            is_live = all(
                self._can_eventually_enable(M, t)
                for M in self.RG.nodes()
            )

            if is_live:
                live_transitions.append(t)
            else:
                dead_transitions.append(t)

        print(f"\nLiveness:")
        print(f"  Live transitions: {len(live_transitions)}/{len(self.pn.transitions)}")
        if dead_transitions:
            print(f"  Dead transitions: {dead_transitions}")

    def _can_eventually_enable(self, M, t):
        """检查从M是否最终能使能t"""
        # 使用BFS搜索可达状态
        visited = {M}
        queue = [M]

        while queue:
            current = queue.pop(0)

            # 检查t是否使能
            if is_enabled(self.pn, current, t):
                return True

            # 探索后继状态
            for t_other in self.pn.transitions:
                if is_enabled(self.pn, current, t_other):
                    M_next = fire_transition(self.pn, current, t_other)
                    if M_next not in visited and M_next in self.RG.nodes():
                        visited.add(M_next)
                        queue.append(M_next)

        return False

    def verify_reversibility(self):
        """验证可逆性"""
        is_reversible = all(
            self.M0 in nx.descendants(self.RG, M) or M == self.M0
            for M in self.RG.nodes()
        )
        print(f"\nReversibility: {'✓ Reversible' if is_reversible else '✗ Not reversible'}")

    def verify_deadlock_freedom(self):
        """验证无死锁"""
        has_deadlock, deadlock_states = detect_deadlock(self.pn, self.M0)
        print(f"\nDeadlock Freedom: {'✗ Has deadlock' if has_deadlock else '✓ Deadlock-free'}")
```

```coq
}.

(* 使能定义 *)
Definition enabled (pn : PetriNet) (M : nat -> nat) (t : nat) : Prop :=
  forall p, In p (P pn) ->
    (exists arc, In arc (F pn) /\ fst arc = p /\ snd arc = t) ->
    M p >= W pn p t.

(* 点火定义 *)
Definition fire (pn : PetriNet) (M : nat -> nat) (t : nat) : nat -> nat :=
  fun p => M p - W pn p t + W pn t p.

(* 可达性定义 *)
Inductive reachable (pn : PetriNet) : (nat -> nat) -> Prop :=
  | reach_init : reachable pn (M0 pn)
  | reach_fire : forall M M' t,
      reachable pn M ->
      enabled pn M t ->
      M' = fire pn M t ->
      reachable pn M'.

(* 有界性定义 *)
Definition bounded (pn : PetriNet) (k : nat) : Prop :=
  forall M p,
    reachable pn M ->
    In p (P pn) ->
    M p <= k.

(* 活性定义（L4-活性） *)
Definition live (pn : PetriNet) : Prop :=
  forall M t,
    reachable pn M ->
    In t (T pn) ->
    exists M',
      reachable_from pn M M' /\
      enabled pn M' t.

(* 定理：有界性保证 *)
Theorem boundedness_guarantee :
  forall pn k,
    bounded pn k ->
    forall M p,
      reachable pn M ->
      In p (P pn) ->
      M p <= k.
Proof.
  intros pn k H_bounded M p H_reach H_in.
  apply H_bounded; assumption.
Qed.

(* 定理：活性蕴含无死锁 *)
Theorem liveness_implies_deadlock_free :
  forall pn,
    live pn ->
    forall M,
      reachable pn M ->
      exists t,
        In t (T pn) /\
        enabled pn M t.
Proof.
  intros pn H_live M H_reach.
  (* 反证法：假设没有使能的变迁 *)
  destruct (T pn) as [|t T'] eqn:E.
  - (* 没有变迁，矛盾 *)
    exfalso.
    (* ... *)
  - (* 至少有一个变迁t *)
    exists t.
    split.
    + rewrite E. left. reflexivity.
    + (* 由活性，存在可达状态使能t *)
      assert (H_live_t := H_live M t H_reach).
      rewrite E in H_live_t.
      (* ... *)
Admitted.  (* 证明略 *)
```
