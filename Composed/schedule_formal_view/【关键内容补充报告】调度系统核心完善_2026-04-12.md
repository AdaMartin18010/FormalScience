## 五、核心文档内容摘要

### 5.1 进程调度（01.4_进程调度.md）

**主要贡献**：

- 完整覆盖经典调度算法（FCFS、SJF、RR、多级反馈队列）
- Linux调度器演进分析（O(1) → CFS）
- 实时调度（RMS、EDF）
- 可运行C代码和Python模拟

**关键公式**：

- LRU竞争比 = k（页帧数）
- CFS vruntime计算

### 5.2 内存调度理论（内存调度理论.md）

**主要贡献**：

- 页面置换算法详解（OPT、FIFO、LRU、Clock）
- 工作集模型与抖动分析
- Linux内存管理实现
- 老化算法、PFF算法

**关键公式**：

- 工作集定义：$W(t, \Delta) = \{p \mid p \text{ 在}[t-\Delta, t]\text{被引用}\}$
- Belady异常分析

### 5.3 IO调度理论（IO调度理论.md）

**主要贡献**：

- 磁盘调度算法（SSTF、SCAN、C-SCAN、LOOK）
- Linux I/O调度器对比（NOOP、Deadline、CFQ、BFQ）
- SSD与HDD调度差异
- RAID调度优化

**关键数据**：

- 磁盘访问时间 = 寻道 + 旋转 + 传输
- 各算法寻道距离对比

### 5.4 GPU调度模型（GPU调度模型.md）

**主要贡献**：

- GPU架构详解（CUDA、RDNA/CDNA）
- Warp/Block/Grid调度机制
- 内存层次优化
- 深度学习调度（vLLM、PagedAttention）

**关键数据**：

- H100性能：67 TFLOPS FP32
- 合并访问效率：90-100%

### 5.5 存储调度 - 内存调度（14.1_内存调度.md）

**主要贡献**：

- DRAM命令序列分析
- FR-FCFS调度算法
- HBM与DDR对比
- 多通道/Bank交织优化

**关键数据**：

- DDR5-6400延迟：75ns
- HBM3带宽：819 GB/s

### 5.6 网络调度 - SDN（15.4_软件定义网络调度.md）

**主要贡献**：

- SDN三层架构详解
- OpenFlow流表模型
- 流量工程与负载均衡
- P4可编程调度

**关键数据**：

- ONOS吞吐量：1M+ flows/s
- OpenFlow延迟：<10μs

### 5.7 实时调度核心理论（实时调度核心理论.md）

**主要贡献**：

- RMS与EDF算法详解
- 可调度性分析（利用率边界、响应时间）
- 优先级反转与继承
- 多处理器实时调度

**关键公式**：

- RMS边界：$U \leq n(2^{1/n} - 1)$
- EDF最优性条件：$U \leq 1$

### 5.8 调度复杂度理论（调度复杂度理论.md）

**主要贡献**：

- Graham三参数表示法
- 经典问题复杂度分类
- 近似算法理论（LPT、PTAS）
- 在线调度竞争分析

**关键数据**：

- List Scheduling近似比：$2 - \frac{1}{m}$
- LPT近似比：$\frac{4}{3} - \frac{1}{3m}$
