# 15.4 软件定义网络调度

## 1. 概述

软件定义网络（Software-Defined Networking, SDN）是一种革命性的网络架构，通过将控制平面与数据平面分离，实现网络的集中化控制和灵活调度。SDN调度器是SDN架构的核心组件，负责流量工程、负载均衡和资源优化。

### 1.1 SDN架构

```
┌─────────────────────────────────────────────────────────┐
│                   应用层 (Application Layer)              │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐   │
```

## 2. SDN调度模型

### 2.1 网络抽象

**拓扑抽象**：
$$G = (V, E, C, L)$$
其中：

- $V$：交换机/路由器节点集合
- $E$：链路集合
- $C$：链路容量
- $L$：链路延迟

**流量矩阵**：
$$T = [t_{ij}]_{n \times n}$$
其中$t_{ij}$表示从节点i到节点j的流量需求。

## 3. 流量工程调度

### 3.1 路径计算算法

**最短路径（Dijkstra）**：
$$\min \sum_{e \in P} w(e)$$

**约束最短路径**：
$$\min \sum_{e \in P} delay(e) \quad \text{s.t.} \quad \sum_{e \in P} cost(e) \leq B$$

**widest path（最大带宽路径）**：
$$\max \min_{e \in P} available\_bandwidth(e)$$

### 3.2 负载均衡调度

**ECMP（等价多路径）**：

```
                  ┌──────┐
             ┌───→│ SW 2 │───┐
             │    └──────┘   │
┌──────┐     │               ↓    ┌──────┐
│ SW 1 │─────┤               ├────│ SW 4 │
└──────┘     │    ┌──────┐   │    └──────┘
             └───→│ SW 3 │───┘
                  └──────┘

哈希分流：
Flow1 (hash=1) ──→ SW2
Flow2 (hash=2) ──→ SW3
Flow3 (hash=1) ──→ SW2
```

**动态负载均衡**：

- 实时监控链路利用率
- 动态调整流量分配
- 拥塞时重路由

### 3.3 拥塞控制

**流量调度策略**：

| 策略 | 机制 | 适用场景 |
|------|------|----------|
| 流量工程 | 显式路径配置 | 可预测流量 |
| 拥塞避免 | 早期标记/丢弃 | 突发流量 |
| 准入控制 | 拒绝新连接 | 资源受限 |
| 速率限制 | 流量整形 | 带宽保证 |

## 4. QoS调度

### 4.1 差异化服务

**IEEE 802.1p优先级**：

| 优先级 | 类别 | 应用 |
|--------|------|------|
| 7 | 网络控制 | 路由协议 |
| 6 | 语音 | VoIP |
| 5 | 视频 | 视频会议 |
| 4 | 实时应用 | 流媒体 |
| 3 | 优秀 effort | 关键业务 |
| 2 | 标准 effort | 一般业务 |
| 1 | 背景 | 文件传输 |
| 0 | 尽力而为 | 默认 |

### 4.2 队列调度算法

**严格优先级（SP）**：

```
┌──────────┐
│ 高优先级  │ → 优先发送
├──────────┤
│ 中优先级  │ → 高队列为空时发送
├──────────┤
│ 低优先级  │ → 其他队列为空时发送
└──────────┘
```

**加权公平队列（WFQ）**：
$$Service_i = \frac{w_i}{\sum w_j} \times TotalBandwidth$$

**赤字轮询（DRR）**：

- 每个队列分配配额
- 发送不超过配额的包
- 剩余配额累积

## 6. P4可编程调度

### 6.1 P4语言

**特点**：

- 协议无关的数据包处理

- 高性能转发

**P4程序结构**：

```p4
// 头部定义
header ethernet_t {
    bit<48> dstAddr;
    bit<48> srcAddr;
    bit<16> etherType;
}

// 解析器
parser MyParser(packet_in packet, out headers hdr) {
    state start {
        packet.extract(hdr.ethernet);
        transition accept;
    }
}

// 控制流
control MyIngress(inout headers hdr, inout metadata meta) {
    action forward(bit<9> port) {
        standard_metadata.egress_spec = port;
    }

    table forwarding {
        key = { hdr.ethernet.dstAddr: exact; }
        actions = { forward; drop; }
    }

    apply {
        forwarding.apply();
    }
}
```

## 7. 调度算法实现

### 7.1 最短路径路由

```python
import heapq
from typing import Dict, List, Tuple

class SDNRouting:
    def __init__(self):
        self.graph = {}  # 网络拓扑
        self.flow_table = {}  # 流表

    def add_link(self, node1: str, node2: str, weight: float):
        """添加链路"""
        if node1 not in self.graph:
            self.graph[node1] = {}
        if node2 not in self.graph:
            self.graph[node2] = {}
        self.graph[node1][node2] = weight
        self.graph[node2][node1] = weight

    def dijkstra(self, src: str, dst: str) -> List[str]:
        """Dijkstra最短路径"""
        dist = {node: float('inf') for node in self.graph}
        prev = {node: None for node in self.graph}
        dist[src] = 0

        pq = [(0, src)]

        while pq:
            d, u = heapq.heappop(pq)
            if d > dist[u]:
                continue
            for v, w in self.graph[u].items():
                if dist[u] + w < dist[v]:
                    dist[v] = dist[u] + w
                    prev[v] = u
                    heapq.heappush(pq, (dist[v], v))

        # 重建路径
        path = []
        curr = dst
        while curr is not None:
            path.append(curr)
            curr = prev[curr]
        return path[::-1] if path[-1] == src else []

    def install_flow(self, src_ip: str, dst_ip: str, path: List[str]):
        """安装流表项"""
        for i in range(len(path) - 1):
            switch = path[i]
            next_hop = path[i + 1]

            if switch not in self.flow_table:
                self.flow_table[switch] = []

            self.flow_table[switch].append({
                'match': {'nw_src': src_ip, 'nw_dst': dst_ip},
                'action': {'output': self.get_port(switch, next_hop)}
            })

    def get_port(self, node1: str, node2: str) -> int:
        """获取连接端口（简化实现）"""
        return hash(node2) % 48 + 1

# 使用示例
sdn = SDNRouting()
sdn.add_link('S1', 'S2', 1)
sdn.add_link('S2', 'S3', 1)
sdn.add_link('S1', 'S3', 3)

path = sdn.dijkstra('S1', 'S3')
print(f"最短路径: {' -> '.join(path)}")

sdn.install_flow('10.0.0.1', '10.0.0.2', path)
```
