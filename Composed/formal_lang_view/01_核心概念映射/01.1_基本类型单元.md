## 🔑 核心概念

### 编程语言中的基本类型

- `int`, `string`, `bool` 等不可再分的原子类型
- 这些类型是类型系统的基础构建块

### 容器化中的对应

- **OCI镜像层**是只读的原子单元
- 运行时实例化为**容器**（类似类型实例化）
- **关键对应**：镜像层哈希 ≈ 类型签名，不可变性 ≈ 类型安全

---

## 📊 对应关系表

| 编程语言 | 容器化 | 关键特性 |
|---------|--------|---------|
| `int` | OCI镜像层 | 原子性、不可变性 |
| 类型签名 | 镜像层哈希 | 唯一标识 |
| 类型实例化 | 容器运行 | 运行时创建 |
| 类型安全 | 不可变性 | 保证一致性 |

## 🔍 技术细节

### OCI镜像层结构

```text
镜像层 (Layer)
├── 内容哈希 (Content Hash)
├── 元数据 (Metadata)
└── 文件系统变更 (Filesystem Changes)
```

### 类型签名对应

```haskell
-- 类型系统视角
type ImageLayer = Layer { hash :: SHA256, content :: ByteString }

-- 基础设施视角
type Container = ImageLayer -> RuntimeState
```

---

## 💡 实际应用

### Docker镜像构建

```dockerfile
FROM ubuntu:22.04  # 基础层（基类型）
RUN apt-get update # 新层（类型扩展）
COPY app /app      # 新层（类型组合）
```

### 类型安全保证

- **不可变性**: 镜像层一旦创建不可修改
- **内容寻址**: 通过哈希值唯一标识
- **可组合性**: 多个层可以组合成完整镜像
