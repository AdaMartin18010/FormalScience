## 2 引用格式规范

### 2.1 基本格式

```markdown
[显示文本](相对路径) - 简短描述
```

### 2.2 路径规范

| 引用类型 | 路径格式 | 示例 |
|---------|---------|------|
| **同目录** | `./文件名.md` | `./README.md` |
| **子目录** | `./子目录/文件名.md` | `./01_核心概念映射/01.1_基本类型单元.md` |
| **父目录** | `../文件名.md` | `../README.md` |
| **跨视角** | `../../另一视角/路径` | `../../schedule_formal_view/README.md` |

### 2.3 锚点格式

```markdown
[章节名](#小写-连字符-分隔)
[文件#章节](./文件.md#章节锚点)
```

## 4 主题内引用模板

### 4.1 同主题子文档引用

```markdown
### 相关子主题

- [25.1 Transformer推理调度](./25.1_Transformer推理调度.md) - 注意力机制调度
- [25.2 KV-Cache调度](./25.2_KV-Cache调度.md) - 内存管理
```

每个文档末尾应包含：

```markdown

```
