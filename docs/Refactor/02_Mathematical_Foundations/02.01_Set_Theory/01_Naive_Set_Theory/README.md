# 02.01.1 朴素集合论

## 模块概述

朴素集合论是集合论的基础部分，研究集合的基本概念、性质和运算。它为公理集合论提供了直观的基础，是理解现代数学的重要起点。

- **集合表示**：列举法、描述法、构造法
