## 📋 内容概述

本模块涵盖深度形式化理论，为系统规约、验证和分析提供严格的数学工具。
内容包括时序逻辑、Petri网理论、控制论和计算理论四大领域，这些理论在形式化验证、系统设计和分析中具有核心作用。

### 核心主题

- **时序逻辑**: LTL、CTL、实时时序逻辑及其应用
- **Petri网理论**: 网结构、分析方法、高级Petri网
- **控制论**: 系统动力学、反馈控制、离散事件系统
- **计算理论**: 自动机、可计算性、计算复杂性
