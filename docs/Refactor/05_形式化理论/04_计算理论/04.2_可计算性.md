可计算性理论（递归论）研究什么是可计算的、什么是不可计算的，以及可计算性之间的归约关系。

## 2. 递归函数

### 2.1 原始递归函数

**定义 2.1** (基本函数)。原始递归函数从以下基本函数构造：

1. **零函数**: $Z(x) = 0$
2. **后继函数**: $S(x) = x + 1$
3. **投影函数**: $P_i^n(x_1, \ldots, x_n) = x_i$

**定义 2.2** (构造规则)。

1. **复合**：若 $g, h_1, \ldots, h_m$ 是原始递归的，则：
   $$f(x) = g(h_1(x), \ldots, h_m(x))$$
   也是原始递归的。

2. **原始递归**：若 $g, h$ 是原始递归的，则：
   $$f(0, x) = g(x)$$
   $$f(n+1, x) = h(n, x, f(n, x))$$
   也是原始递归的。

**例 2.1** (基本算术函数)。

```
加法: add(0, y) = y
      add(x+1, y) = S(add(x, y))

乘法: mult(0, y) = 0
      mult(x+1, y) = add(mult(x, y), y)

指数: exp(0, y) = 1
      exp(x+1, y) = mult(exp(x, y), y)
```

### 2.2 μ-递归函数

**定义 2.3** (μ-算子/极小化)。

$$\mu y. R(x, y) = \text{满足 } R(x, y) \text{ 的最小 } y$$

若不存在这样的 $y$，则无定义。

**定义 2.4** (部分递归函数)。由基本函数通过复合、原始递归和 μ-算子构造的函数。

**定义 2.5** (全递归函数)。处处有定义的部分递归函数。

### 2.3 丘奇论题

**论题 2.1** (丘奇-图灵)。部分递归函数类恰好是图灵可计算函数类。

这与自动机理论中的丘奇-图灵论题相呼应。

## 3. 可判定性与可识别性

### 3.1 语言分类

**定义 3.1** (语言分类)。

| 类别 | 定义 | 特征 |
|-----|------|-----|
| **递归 (R)** | 存在判定图灵机 | 总能停机并给出答案 |
| **递归可枚举 (RE)** | 存在识别图灵机 | 对"是"实例必停机 |
| **共递归可枚举 (co-RE)** | 补集是 RE | 对"否"实例必停机 |

**关系**：$R = RE \cap co\text{-}RE$，且 $R \subsetneq RE \subsetneq \text{所有语言}$

### 3.2 通用图灵机

**定理 3.1** (通用图灵机)。存在图灵机 $U$，对于任意图灵机 $M$ 和输入 $w$：

$$U(\langle M \rangle, w) \text{ 模拟 } M(w)$$

**意义**：通用图灵机证明了通用计算的可能性，是计算机的理论基础。

**算法 3.1** (通用图灵机模拟)。

```python
class UniversalTM:
    """
    通用图灵机模拟器
    """
    def __init__(self):
        self.tape = []
        self.head = 0
        self.state = 'q0'

    def simulate(self, M_description, w):
        """
        模拟图灵机 M 在输入 w 上的运行
        M_description: M 的编码 (转移函数表)
        """
        # 在带上构造配置: <M> # w
        self.tape = self.encode_TM(M_description) + ['#'] + list(w)

        while self.state not in ['accept', 'reject']:
            # 解析当前配置
            current_symbol = self.tape[self.head]

            # 查找转移函数
            transition = self.lookup_transition(
                M_description, self.state, current_symbol
            )

            if transition is None:
                self.state = 'reject'
                break

            # 执行转移
            new_state, write_symbol, direction = transition
            self.tape[self.head] = write_symbol
            self.state = new_state

            # 移动读写头
            if direction == 'R':
                self.head += 1
            elif direction == 'L':
                self.head -= 1

        return self.state == 'accept'
```

## 4. 不可判定性

### 4.1 对角化方法

**定理 4.1** (停机问题不可判定)。语言 $A_{TM} = \{\langle M, w \rangle \mid M \text{ 接受 } w\}$ 是不可判定的。

_证明_：假设存在判定器 $H$ 判定 $A_{TM}$。构造图灵机 $D$：

```
D(⟨M⟩):
  模拟 H(⟨M⟩, ⟨M⟩)
  若 H 接受, 则拒绝
  若 H 拒绝, 则接受
```

考虑 $D(\langle D \rangle)$：

- 若 $D$ 接受 $\langle D \rangle$，则 $H$ 应拒绝 $\langle D, D \rangle$，但 $H$ 判定器会接受...
- 矛盾！∎

### 4.2 Rice 定理

**定理 4.2** (Rice)。任何关于程序行为的非平凡性质都是不可判定的。

形式化：设 $P$ 是图灵机接受语言的非平凡性质，则语言

$$L_P = \{\langle M \rangle \mid L(M) \text{ 满足 } P\}$$

是不可判定的。

**例 4.1** (Rice 定理的应用)。以下问题都不可判定：

- $M$ 是否接受任何输入？
- $M$ 是否接受正则语言？
- $M$ 的语言是否等于某个固定语言？
- $M$ 的语言是否无限？

### 4.3 其他不可判定问题

| 问题 | 描述 | 归约来源 |
|-----|------|---------|
| $E_{TM}$ | 图灵机语言是否为空 | $A_{TM}$ |
| $EQ_{TM}$ | 两图灵机是否等价 | $E_{TM}$ |
| $HALT_{TM}$ | 图灵机是否停机 | $A_{TM}$ |
| Post 对应问题 | 拼接匹配问题 | $A_{TM}$ |
| 希尔伯特第10问题 | 多项式方程整数解 | (复杂归约) |

## 5. 归约理论

### 5.1 映射归约

**定义 5.1** (映射归约)。$A \leq_m B$ 若存在可计算函数 $f$ 使得：

$$w \in A \Leftrightarrow f(w) \in B$$

**定理 5.1** (归约的传递性)。若 $A \leq_m B$ 且 $B \leq_m C$，则 $A \leq_m C$。

**定理 5.2** (归约与可判定性)。

- 若 $A \leq_m B$ 且 $B$ 可判定，则 $A$ 可判定
- 若 $A \leq_m B$ 且 $A$ 不可判定，则 $B$ 不可判定

### 5.2 典型归约

**算法 5.1** ($A_{TM} \leq_m HALT_{TM}$)。

```python
def reduce_A_TM_to_HALT(M, w):
    """
    将 A_TM 归约到 HALT_TM
    构造 M' 使得 M 接受 w 当且仅当 M' 停机
    """
    def M_prime(x):
        # M' 忽略输入 x，直接模拟 M(w)
        result = simulate(M, w)
        if result == 'accept':
            return 'halt'
        else:
            # 进入无限循环
            while True:
                pass

    return encode_TM(M_prime), ''
```

### 5.3 完备性

**定义 5.2** (RE-完备)。语言 $B$ 是 RE-完备的，若：

1. $B \in RE$
2. $\forall A \in RE: A \leq_m B$

**定理 5.3** ($A_{TM}$ 是 RE-完备的)。

**证明**：$A_{TM} \in RE$（通用图灵机可识别）。对于任意 $A \in RE$，设 $M_A$ 识别 $A$，则：

$$w \in A \Leftrightarrow M_A \text{ 接受 } w \Leftrightarrow \langle M_A, w \rangle \in A_{TM}$$

故 $A \leq_m A_{TM}$。 ∎

## 6. Lean 形式化

### 6.1 原始递归函数

```lean4
import Mathlib

-- 原始递归函数定义
inductive PrimRec : (n : ℕ) → (Fin n → ℕ) → ℕ → Type
  -- 零函数
  | zero : PrimRec 0 (λ _ => 0) 0
  -- 后继函数
  | succ : PrimRec 1 (λ x => x 0 + 1) 1
  -- 投影函数
  | proj {n} (i : Fin n) : PrimRec n (λ x => x i) 1
  -- 复合
  | comp {m n g hs}
      (hg : PrimRec m g n)
      (hhs : ∀ i, PrimRec n (hs i) (g i)) :
      PrimRec m (λ x => g (λ i => hs i x)) (n + 1)
  -- 原始递归
  | prec {n f g}
      (hf : PrimRec n f 1)
      (hg : PrimRec (n+2) g 1) :
      PrimRec (n+1) (λ x =>
        Nat.rec (f (λ i => x (Fin.succ i)))
                (λ y ih => g (λ i => match i with
                  | 0 => y
                  | 1 => ih
                  | Fin.succ (Fin.succ j) => x (Fin.succ j)))
                (x 0)) (n + 2)

-- 加法定义示例
def primRecAdd : PrimRec 2 (λ x => x 0 + x 1) 3 :=
  PrimRec.prec (PrimRec.proj 0)
    (PrimRec.comp PrimRec.succ
      (λ _ => PrimRec.proj 1))
```

### 6.2 部分递归函数

```lean4
-- 部分递归函数 (包含μ-算子)
inductive PartRec : (n : ℕ) → (Fin n → ℕ) → Part ℕ → Type
  -- 所有原始递归函数都是部分递归的
  | ofPrim {n f} (pf : PrimRec n f) : PartRec n (λ x => f x) (f x)
  -- μ-算子: 极小化
  | mu {n f} (pf : PartRec (n+1) f) :
      PartRec n (λ x => Part.assert (∃ y, f (y :: x) = 0)
        (λ h => Nat.find (λ y => f (y :: x) = 0) h))

-- 部分递归函数可计算性
def PartRec.computable {n} {f : Fin n → ℕ → Part ℕ}
    (pf : ∀ x, PartRec n (f x)) : Prop :=
  ∃ M : TM, ∀ x, M.computes (f x)
```

### 6.3 可判定性定义

```lean4
-- 语言可判定
def DecidableLanguage {Sigma} [Fintype Sigma]
    (L : Set (List Sigma)) : Prop :=
  ∃ M : TM, ∀ w,
    (w ∈ L → M.accepts w) ∧
    (w ∉ L → M.rejects w)

-- 语言递归可枚举
def RecursivelyEnumerable {Sigma} [Fintype Sigma]
    (L : Set (List Sigma)) : Prop :=
  ∃ M : TM, ∀ w, w ∈ L → M.accepts w

-- 可识别与可判定的关系
theorem decidable_iff_RE_and_coRE {Sigma} [Fintype Sigma]
    (L : Set (List Sigma)) :
    DecidableLanguage L ↔
    (RecursivelyEnumerable L ∧ RecursivelyEnumerable Lᶜ) := by
  constructor
  · -- 可判定蕴含RE且co-RE
    intro h
    constructor
    · rcases h with ⟨M, hM⟩
      use M
      intro w hw
      exact (hM w).left hw
    · rcases h with ⟨M, hM⟩
      use M.swap_accept_reject  -- 交换接受/拒绝
      intro w hw
      exact (hM w).right hw
  · -- RE且co-RE蕴含可判定
    sorry  -- 需构造并行模拟器
```

### 6.4 停机问题

```lean4
-- 停机问题语言
def HaltProblem (Sigma) [Fintype Sigma] : Set (List Sigma) :=
  { ⟨M, w⟩ | M.accepts w }

-- 停机问题不可判定
theorem halt_problem_undecidable {Sigma} [Fintype Sigma] :
    ¬DecidableLanguage (HaltProblem Sigma) := by
  intro h_decidable
  -- 假设存在判定器H
  rcases h_decidable with ⟨H, hH⟩

  -- 构造对角化图灵机D
  let D : TM := {
    run := λ ⟨M⟩ =>
      if H.run (⟨M⟩, ⟨M⟩) = 'accept' then
        loop_forever
      else
        'accept',
    -- ...
  }

  -- 矛盾: D(⟨D⟩)的行为
  have contradiction := hH ⟨D, ⟨D⟩⟩
  sorry  -- 形式化对角论证
```
