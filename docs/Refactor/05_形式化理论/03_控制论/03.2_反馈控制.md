### 1.1 反馈控制原理

反馈是控制论的核心机制。通过测量输出并与参考值比较，控制器调整输入以减小误差。

## 2. PID 控制

### 2.1 PID 控制器结构

**定义 2.1** (PID 控制器)。比例-积分-微分控制器：

$$u(t) = K_p e(t) + K_i \int_0^t e(\tau)d\tau + K_d \frac{de(t)}{dt}$$

其中 $e(t) = r(t) - y(t)$ 是误差信号。

**传递函数形式**：

$$C(s) = K_p + \frac{K_i}{s} + K_d s = K_p\left(1 + \frac{1}{T_i s} + T_d s\right)$$

### 2.2 各分量作用

| 分量 | 作用 | 效果 | 风险 |
|-----|------|-----|-----|
| P (比例) | 即时响应误差 | 快速但有余差 | 过冲 |
| I (积分) | 消除稳态误差 | 无差调节 | 积分饱和 |
| D (微分) | 预测误差趋势 | 阻尼振荡 | 噪声放大 |

### 2.3 参数整定方法

**算法 2.1** (Ziegler-Nichols 方法)。

```python
def ziegler_nichols_tuning(Ku, Tu):
    """
    Ziegler-Nichols临界比例度法
    Ku: 临界增益
    Tu: 临界周期
    """
    # PID参数
    Kp = 0.6 * Ku
    Ti = Tu / 2
    Td = Tu / 8

    Ki = Kp / Ti
    Kd = Kp * Td

    return Kp, Ki, Kd

def relay_feedback_tuning(system, dt=0.01, T_sim=100):
    """
    继电器反馈自动整定 (Aström-Hägglund)
    """
    t = np.arange(0, T_sim, dt)
    y = np.zeros_like(t)
    u = np.zeros_like(t)

    d = 1.0  # 继电器幅度

    # 检测极限环
    crossings = []
    for i in range(1, len(t)):
        u[i] = d if y[i-1] < 0 else -d
        y[i] = system.step(u[i], dt)  # 简化表示

        if y[i-1] * y[i] < 0:  # 过零检测
            crossings.append(t[i])

    Tu = np.mean(np.diff(crossings[1:]))  # 平均周期
    a = np.max(np.abs(y))  # 振幅
    Ku = 4 * d / (np.pi * a)

    return ziegler_nichols_tuning(Ku, Tu)
```

## 3. 状态反馈控制

### 3.1 极点配置

**定义 3.1** (状态反馈)。控制律 $u = -Kx + r$，闭环系统：

$$\dot{x} = (A - BK)x + Br$$

**定理 3.1** (极点配置定理)。若 $(A, B)$ 能控，则对于任意期望特征值集合，存在状态反馈增益 $K$ 使得 $A-BK$ 具有这些特征值。

**算法 3.1** (Ackermann 公式)。

```python
def ackermann_pole_placement(A, B, desired_poles):
    """
    使用Ackermann公式计算状态反馈增益
    适用于SISO系统
    """
    n = A.shape[0]

    # 计算期望特征多项式
    desired_char_poly = np.poly(desired_poles)

    # 计算能控性矩阵
    C = ctrb(A, B)

    if np.linalg.matrix_rank(C) < n:
        raise ValueError("系统不可控")

    # Ackermann公式: K = [0 ... 0 1] C^-1 φ(A)
    phi_A = np.zeros((n, n))
    for i, coeff in enumerate(desired_char_poly):
        phi_A += coeff * np.linalg.matrix_power(A, n - i)

    K = np.zeros(n)
    K[-1] = 1
    K = K @ np.linalg.inv(C) @ phi_A

    return K

def place_poles(A, B, poles):
    """
    使用SciPy的极点配置
    """
    from scipy.signal import place_poles
    result = place_poles(A, B, poles)
    return result.gain_matrix
```

### 3.2 最优控制：LQR

**定义 3.2** (LQR 问题)。线性二次调节器：最小化代价函数

$$J = \int_0^{\infty} (x^T Q x + u^T R u) dt$$

**定理 3.2** (LQR 解)。最优控制律 $u = -Kx$，其中：

$$K = R^{-1}B^TP$$

$P$ 满足代数 Riccati 方程 (ARE)：

$$A^TP + PA - PBR^{-1}B^TP + Q = 0$$

**算法 3.2** (求解 ARE)。

```python
from scipy.linalg import solve_continuous_are

def lqr_design(A, B, Q, R):
    """
    LQR控制器设计
    """
    # 解代数Riccati方程
    P = solve_continuous_are(A, B, Q, R)

    # 计算最优增益
    K = np.linalg.inv(R) @ B.T @ P

    # 计算闭环极点
    closed_loop_poles = np.linalg.eigvals(A - B @ K)

    return K, P, closed_loop_poles

# 示例: 倒立摆LQR控制
def inverted_pendulum_lqr():
    # 线性化模型 (小角度近似)
    M, m, l, g = 1.0, 0.1, 0.5, 9.81

    A = np.array([[0, 1, 0, 0],
                  [0, 0, -m*g/M, 0],
                  [0, 0, 0, 1],
                  [0, 0, (M+m)*g/(M*l), 0]])

    B = np.array([[0], [1/M], [0], [-1/(M*l)]])

    # 权重矩阵
    Q = np.diag([10, 1, 10, 1])  # 强调位置和角度
    R = np.array([[0.1]])  # 控制代价

    K, P, poles = lqr_design(A, B, Q, R)
    return K
```

### 3.3 观测器设计

**定义 3.3** (Luenberger 观测器)。

$$\dot{\hat{x}} = A\hat{x} + Bu + L(y - C\hat{x})$$

其中 $L$ 是观测器增益。

**定理 3.3** (分离原理)。状态反馈和观测器可分别设计：闭环极点 = 控制器极点 ∪ 观测器极点。

**算法 3.3** (观测器设计)。

```python
def observer_design(A, C, desired_observer_poles):
    """
    设计Luenberger观测器
    使用对偶原理
    """
    # 观测器设计等价于对偶系统的状态反馈
    A_dual = A.T
    C_dual = C.T

    # 计算 L^T (对偶系统的反馈增益)
    L_T = place_poles(A_dual, C_dual, desired_observer_poles)

    L = L_T.T
    return L

def kalman_filter_design(A, B, C, Q_w, R_v):
    """
    稳态Kalman滤波器 (随机最优观测器)
    Q_w: 过程噪声协方差
    R_v: 测量噪声协方差
    """
    # 解滤波器Riccati方程
    P = solve_continuous_are(A.T, C.T, Q_w, R_v)

    # Kalman增益
    L = P @ C.T @ np.linalg.inv(R_v)

    return L, P
```

## 4. 鲁棒控制

### 4.1 灵敏度分析

**定义 4.1** (灵敏度函数)。对于单位反馈系统：

$$S(s) = \frac{1}{1 + L(s)}, \quad T(s) = \frac{L(s)}{1 + L(s)}$$

其中 $L(s) = C(s)G(s)$ 是开环传递函数。

**设计目标**：

- 低频：$|S(j\omega)| \ll 1$（良好跟踪）
- 高频：$|T(j\omega)| \ll 1$（噪声抑制）

### 4.2 H∞ 控制

**定义 4.2** (H∞ 范数)。

$$\|G\|_\infty = \sup_{\omega} \sigma_{max}(G(j\omega))$$

**定义 4.3** (H∞ 控制问题)。设计控制器 $K$ 使得闭环稳定且：

$$\|T_{zw}\|_\infty < \gamma$$

其中 $T_{zw}$ 是从扰动 $w$ 到性能输出 $z$ 的传递函数。

**定理 4.1** (标准 H∞ 问题)。存在控制器当且仅当两个 Riccati 方程有正定解且谱半径条件满足。

## 5. 数字控制

### 5.1 离散化方法

**定义 5.1** (系统离散化)。连续系统离散化：

$$A_d = e^{AT_s}, \quad B_d = \int_0^{T_s} e^{A\tau}Bd\tau$$

**算法 5.1** (控制器离散化)。

```python
def c2d_zero_order_hold(A, B, Ts):
    """
    零阶保持离散化
    """
    n = A.shape[0]

    # 矩阵指数方法
    M = np.block([
        [A, B],
        [np.zeros((B.shape[1], n)), np.zeros((B.shape[1], B.shape[1]))]
    ])

    exp_M = expm(M * Ts)

    Ad = exp_M[:n, :n]
    Bd = exp_M[:n, n:]

    return Ad, Bd

def pid_discretize(Kp, Ki, Kd, Ts, method='tustin'):
    """
    PID控制器离散化
    """
    if method == 'tustin':
        # 双线性变换
        Ki_d = Ki * Ts / 2
        Kd_d = 2 * Kd / Ts

        # 差分方程系数
        a = [1, -1]
        b_p = [Kp, 0]
        b_i = [Ki_d, Ki_d]
        b_d = [Kd_d, -Kd_d]

    elif method == 'backward':
        # 后向差分
        Ki_d = Ki * Ts
        Kd_d = Kd / Ts

        a = [1, -1]
        b_p = [Kp, 0]
        b_i = [Ki_d, 0]
        b_d = [Kd_d, -Kd_d]

    return a, b_p, b_i, b_d
```

### 5.2 数字 PID 实现

```python
class DigitalPID:
    def __init__(self, Kp, Ki, Kd, Ts, windup_limit=10):
        self.Kp = Kp
        self.Ki = Ki
        self.Kd = Kd
        self.Ts = Ts
        self.windup_limit = windup_limit

        # 状态
        self.integral = 0
        self.prev_error = 0
        self.prev_measurement = 0

    def update(self, setpoint, measurement):
        error = setpoint - measurement

        # 积分项 (带抗饱和)
        self.integral += error * self.Ts
        self.integral = np.clip(self.integral,
                                -self.windup_limit,
                                self.windup_limit)

        # 微分项 (对测量值微分，避免设定值突变)
        derivative = (measurement - self.prev_measurement) / self.Ts

        # PID输出
        output = (self.Kp * error +
                 self.Ki * self.integral -
                 self.Kd * derivative)

        self.prev_error = error
        self.prev_measurement = measurement

        return output
```

## 7. 应用实例

### 7.1 无人机姿态控制

```python
def quadcopter_attitude_control():
    """
    四旋翼姿态LQR控制
    状态: [roll, pitch, yaw, roll_rate, pitch_rate, yaw_rate]
    """
    # 线性化模型 (悬停附近)
    I_xx, I_yy, I_zz = 0.01, 0.01, 0.02

    A = np.block([
        [np.zeros((3, 3)), np.eye(3)],
        [np.zeros((3, 3)), np.zeros((3, 3))]
    ])

    B = np.block([
        [np.zeros((3, 3))],
        [np.diag([1/I_xx, 1/I_yy, 1/I_zz])]
    ])

    Q = np.diag([10, 10, 5, 1, 1, 0.5])
    R = np.eye(3) * 0.1

    K, _, _ = lqr_design(A, B, Q, R)
    return K
```
