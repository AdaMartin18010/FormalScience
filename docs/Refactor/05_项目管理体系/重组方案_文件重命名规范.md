# Refactor文件夹重组方案与文件重命名规范

## 重组目标

1. **统一命名规范**：建立一致的命名体系
2. **层次化组织**：按照理论层次和主题分类
3. **清理冗余**：合并重复内容，删除过时文件
4. **建立索引**：创建清晰的导航体系

## 命名规范

### 目录命名规范

```text
XX_主题名称_英文描述/
```

### 文件命名规范

```text
XX.XX_中文标题_英文标题.md
```

### 编号体系

- **01-15**: 基础理论体系
- **16-25**: 应用理论体系  
- **26-35**: 前沿理论体系
- **36-50**: 高级理论体系
- **51-99**: 项目管理与工具
