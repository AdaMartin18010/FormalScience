- **核心内容**:
  - 数学对象存在性问题
  - 主要哲学立场（柏拉图主义、形式主义、直觉主义、结构主义、虚构主义）
  - 本体论比较分析

**主要立场**:

- **直觉主义**：数学对象是心智构造
- **结构主义**：数学对象是结构中的位置
- **虚构主义**：数学对象是虚构的，但有用

**形式化分析**:

- 本体论承诺：$\text{Ont}(T) = \{x \mid \exists y (T \vdash \exists x \phi(x) \land \phi(y))\}$
- 存在性量词的不同解释
- 数学真理的不同理论
