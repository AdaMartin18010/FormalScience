## 1. 控制系统基础

### 1.1 系统定义

#### 1.1.1 动态系统

**定义 1.1.1 (动态系统)**
动态系统是一个五元组 $\Sigma = (X, U, Y, f, h)$，其中：

- $X \subseteq \mathbb{R}^n$ 是状态空间
- $U \subseteq \mathbb{R}^m$ 是输入空间
- $Y \subseteq \mathbb{R}^p$ 是输出空间
- $f : X \times U \rightarrow X$ 是状态转移函数
- $h : X \rightarrow Y$ 是输出函数

**定义 1.1.2 (连续时间系统)**
连续时间系统的状态方程：
$$\dot{x}(t) = f(x(t), u(t))$$
$$y(t) = h(x(t))$$

**定义 1.1.3 (离散时间系统)**
离散时间系统的状态方程：
$$x(k+1) = f(x(k), u(k))$$
$$y(k) = h(x(k))$$

#### 1.1.2 系统状态

**定义 1.1.4 (系统状态)**
系统状态是描述系统内部状况的最小信息集合，满足：

1. **马尔可夫性**：未来状态只依赖于当前状态和输入
2. **完整性**：状态包含预测未来行为的所有必要信息
3. **最小性**：状态是满足上述条件的最小集合

**定理 1.1.1 (状态唯一性)**
对于给定的初始状态和输入，系统状态是唯一的。

**证明：** 通过状态方程的解的唯一性：

1. **局部唯一性**：在局部区域内，状态方程满足利普希茨条件
2. **全局唯一性**：通过延拓定理扩展到全局
3. **时间一致性**：状态在时间上连续且唯一

### 1.2 线性系统理论

#### 1.2.1 线性系统定义

**定义 1.2.1 (线性系统)**
线性系统满足叠加原理：
$$f(\alpha x_1 + \beta x_2, \alpha u_1 + \beta u_2) = \alpha f(x_1, u_1) + \beta f(x_2, u_2)$$

**定义 1.2.2 (线性时不变系统)**
线性时不变系统的状态空间表示：
$$\dot{x}(t) = Ax(t) + Bu(t)$$
$$y(t) = Cx(t) + Du(t)$$

其中：

- $A \in \mathbb{R}^{n \times n}$ 是系统矩阵
- $B \in \mathbb{R}^{n \times m}$ 是输入矩阵
- $C \in \mathbb{R}^{p \times n}$ 是输出矩阵
- $D \in \mathbb{R}^{p \times m}$ 是直接传递矩阵

#### 1.2.2 线性系统解

**定理 1.2.1 (线性系统解)**
线性时不变系统的解为：
$$x(t) = e^{At}x(0) + \int_0^t e^{A(t-\tau)}Bu(\tau)d\tau$$

**证明：** 通过状态方程的积分：

1. **齐次解**：$\dot{x} = Ax$ 的解为 $x_h(t) = e^{At}x(0)$
2. **特解**：通过变分常数法求解非齐次方程
3. **完整解**：$x(t) = x_h(t) + x_p(t)$

**证明细节：**

```haskell
-- 线性系统解的计算
data LinearSystem = LinearSystem
  { a :: Matrix Double  -- 系统矩阵
  , b :: Matrix Double  -- 输入矩阵
  , c :: Matrix Double  -- 输出矩阵
  , d :: Matrix Double  -- 直接传递矩阵
  }

-- 状态转移矩阵
stateTransitionMatrix :: Matrix Double -> Double -> Matrix Double
stateTransitionMatrix a t = expm (a * scalar t)

-- 系统响应
systemResponse :: LinearSystem -> Vector Double -> Vector Double -> Double -> Vector Double
systemResponse sys x0 u t = 
  let phi = stateTransitionMatrix (a sys) t
      integral = convolutionIntegral sys u t
  in phi * x0 + integral

-- 卷积积分
convolutionIntegral :: LinearSystem -> Vector Double -> Double -> Vector Double
convolutionIntegral sys u t = 
  let integrand tau = stateTransitionMatrix (a sys) (t - tau) * (b sys) * u
  in integrate 0 t integrand
```

#### 1.2.3 传递函数

**定义 1.2.3 (传递函数)**
线性系统的传递函数：
$$G(s) = C(sI - A)^{-1}B + D$$

**定理 1.2.2 (传递函数性质)**
传递函数满足：

1. **有理性**：$G(s)$ 是有理函数
2. **真性**：分子次数不超过分母次数
3. **因果性**：系统是因果的

### 1.3 系统分类

#### 1.3.1 按时间特性分类

**定义 1.3.1 (时变系统)**
时变系统的状态方程：
$$\dot{x}(t) = f(x(t), u(t), t)$$

**定义 1.3.2 (时不变系统)**
时不变系统的状态方程：
$$\dot{x}(t) = f(x(t), u(t))$$

#### 1.3.2 按线性特性分类

**定义 1.3.3 (非线性系统)**
不满足叠加原理的系统。

**定义 1.3.4 (仿射系统)**
仿射系统的状态方程：
$$\dot{x}(t) = f(x(t)) + g(x(t))u(t)$$

## 2. 稳定性分析

### 2.1 李雅普诺夫稳定性

#### 2.1.1 稳定性定义

**定义 2.1.1 (平衡点)**
状态 $x_e \in X$ 是平衡点，如果 $f(x_e, 0) = 0$。

**定义 2.1.2 (李雅普诺夫稳定性)**
平衡点 $x_e$ 是李雅普诺夫稳定的，如果对于任意 $\epsilon > 0$，存在 $\delta > 0$ 使得：
$$\|x(0) - x_e\| < \delta \Rightarrow \|x(t) - x_e\| < \epsilon \text{ for all } t \geq 0$$

**定义 2.1.3 (渐近稳定性)**
平衡点 $x_e$ 是渐近稳定的，如果它是李雅普诺夫稳定的且：
$$\lim_{t \rightarrow \infty} x(t) = x_e$$

**定义 2.1.4 (指数稳定性)**
平衡点 $x_e$ 是指数稳定的，如果存在常数 $M > 0$ 和 $\alpha > 0$ 使得：
$$\|x(t) - x_e\| \leq M\|x(0) - x_e\|e^{-\alpha t}$$

#### 2.1.2 李雅普诺夫直接法

**定理 2.1.1 (李雅普诺夫直接法)**
如果存在连续可微函数 $V : X \rightarrow \mathbb{R}$ 满足：

1. $V(x_e) = 0$
2. $V(x) > 0$ 对于 $x \neq x_e$
3. $\dot{V}(x) \leq 0$ 对于 $x \neq x_e$

则平衡点 $x_e$ 是李雅普诺夫稳定的。

**证明：** 通过李雅普诺夫函数的单调性：

1. **正定性**：$V(x) > 0$ 确保函数在平衡点附近有下界
2. **单调性**：$\dot{V}(x) \leq 0$ 确保 $V(x)$ 不增加
3. **有界性**：状态轨迹保持在平衡点附近

**定理 2.1.2 (渐近稳定性判据)**
如果存在连续可微函数 $V : X \rightarrow \mathbb{R}$ 满足：

1. $V(x_e) = 0$
2. $V(x) > 0$ 对于 $x \neq x_e$
3. $\dot{V}(x) < 0$ 对于 $x \neq x_e$

则平衡点 $x_e$ 是渐近稳定的。

**证明：** 通过李雅普诺夫函数的严格单调性：

1. **严格单调性**：$\dot{V}(x) < 0$ 确保 $V(x)$ 严格递减
2. **收敛性**：$V(x)$ 收敛到最小值 $V(x_e) = 0$
3. **状态收敛**：状态收敛到平衡点

#### 2.1.3 李雅普诺夫函数构造

**算法 2.1.1 (李雅普诺夫函数构造)**

```haskell
-- 李雅普诺夫函数构造
data LyapunovFunction = LyapunovFunction
  { function :: Vector Double -> Double
  , derivative :: Vector Double -> Double
  , positiveDefinite :: Bool
  , negativeSemiDefinite :: Bool
  }

-- 构造二次型李雅普诺夫函数
constructQuadraticLyapunov :: Matrix Double -> LyapunovFunction
constructQuadraticLyapunov p = 
  let v x = x' * p * x
      vDot x = x' * (p * a + a' * p) * x
      isPositiveDefinite = all (> 0) (eigenvalues p)
      isNegativeSemiDefinite = all (<= 0) (eigenvalues (p * a + a' * p))
  in LyapunovFunction v vDot isPositiveDefinite isNegativeSemiDefinite

-- 李雅普诺夫方程求解
solveLyapunovEquation :: Matrix Double -> Matrix Double -> Matrix Double
solveLyapunovEquation a q = 
  let n = rows a
      m = kron (eye n) a' + kron a' (eye n)
      b = vectorize q
      p = reshape n (m <\> b)
  in p
```

### 2.2 线性系统稳定性

#### 2.2.1 特征值判据

**定理 2.2.1 (线性系统稳定性)**
线性系统 $\dot{x} = Ax$ 的零平衡点是渐近稳定的当且仅当 $A$ 的所有特征值都有负实部。

**证明：** 通过特征值分解：

1. **特征值分解**：$A = T\Lambda T^{-1}$，其中 $\Lambda = \text{diag}(\lambda_1, \ldots, \lambda_n)$
2. **状态转移矩阵**：$e^{At} = Te^{\Lambda t}T^{-1}$
3. **稳定性条件**：$e^{\lambda_i t} \rightarrow 0$ 当且仅当 $\text{Re}(\lambda_i) < 0$

**定理 2.2.2 (临界稳定性)**
线性系统 $\dot{x} = Ax$ 的零平衡点是李雅普诺夫稳定的当且仅当 $A$ 的所有特征值都有非正实部，且零实部特征值的代数重数等于几何重数。

#### 2.2.2 稳定性判据

**定义 2.2.1 (赫尔维茨判据)**
多项式 $p(s) = a_n s^n + a_{n-1} s^{n-1} + \cdots + a_0$ 是赫尔维茨的，如果所有根都有负实部。

**算法 2.2.1 (赫尔维茨判据)**

```haskell
-- 赫尔维茨判据
hurwitzCriterion :: [Double] -> Bool
hurwitzCriterion coeffs = 
  let n = length coeffs - 1
      hurwitzMatrix = buildHurwitzMatrix coeffs
      minors = [determinant (submatrix hurwitzMatrix i) | i <- [1..n]]
  in all (> 0) minors

-- 构建赫尔维茨矩阵
buildHurwitzMatrix :: [Double] -> Matrix Double
buildHurwitzMatrix coeffs = 
  let n = length coeffs - 1
      matrix = matrix n n (\(i,j) -> 
        if j <= i && i - j < length coeffs 
        then coeffs !! (i - j) 
        else 0)
  in matrix

-- 劳斯判据
routhCriterion :: [Double] -> Bool
routhCriterion coeffs = 
  let routhArray = buildRouthArray coeffs
      firstColumn = head <$> routhArray
  in all (> 0) firstColumn
```

### 2.3 稳定性判据

#### 2.3.1 频域判据

**定理 2.3.1 (奈奎斯特判据)**
闭环系统稳定的充分必要条件是奈奎斯特图不包围点 $(-1, 0)$。

**证明：** 通过幅角原理：

1. **幅角原理**：$\frac{1}{2\pi i} \oint_C \frac{f'(z)}{f(z)} dz = N - P$
2. **稳定性条件**：$N = P$，其中 $N$ 是包围 $(-1, 0)$ 的次数，$P$ 是开环不稳定极点个数

#### 2.3.2 时域判据

**定理 2.3.2 (根轨迹判据)**
闭环系统稳定的充分必要条件是所有闭环极点都在左半平面。

## 3. 可控性和可观性

### 3.1 可控性理论

#### 3.1.1 可控性定义

**定义 3.1.1 (可控性)**
系统 $\Sigma$ 在时间 $T$ 内可控，如果对于任意初始状态 $x_0$ 和目标状态 $x_f$，存在输入 $u(t)$ 使得 $x(T) = x_f$。

**定义 3.1.2 (可控性矩阵)**
线性系统的可控性矩阵：
$$\mathcal{C} = [B \quad AB \quad A^2B \quad \cdots \quad A^{n-1}B]$$

**定理 3.1.1 (可控性判据)**
线性系统完全可控当且仅当可控性矩阵 $\mathcal{C}$ 满秩。

**证明：** 通过凯莱-哈密顿定理：

1. **可达性**：可控性矩阵的列空间包含可达状态空间
2. **满秩性**：满秩确保可达整个状态空间
3. **有限性**：凯莱-哈密顿定理限制矩阵幂的线性相关性

**证明细节：**

```haskell
-- 可控性分析
data Controllability = Controllability
  { controllabilityMatrix :: Matrix Double
  , rank :: Int
  , isControllable :: Bool
  }

-- 计算可控性矩阵
controllabilityMatrix :: Matrix Double -> Matrix Double -> Matrix Double
controllabilityMatrix a b = 
  let n = rows a
      powers = [a^i | i <- [0..n-1]]
      matrices = [power * b | power <- powers]
  in hconcat matrices

-- 可控性判据
isControllable :: Matrix Double -> Matrix Double -> Bool
isControllable a b = 
  let c = controllabilityMatrix a b
      r = rank c
      n = rows a
  in r == n

-- 可控性分解
controllabilityDecomposition :: Matrix Double -> Matrix Double -> (Matrix Double, Matrix Double, Matrix Double)
controllabilityDecomposition a b = 
  let c = controllabilityMatrix a b
      (u, s, v) = svd c
      rank = rank c
      controllable = u ^*^ diag (take rank (toList s))
      uncontrollable = u ^*^ diag (drop rank (toList s))
  in (controllable, uncontrollable, v)
```

#### 3.1.2 可控性标准形

**定义 3.1.3 (可控性标准形)**
可控系统的可控性标准形：
$$\dot{x} = \begin{bmatrix} 0 & 1 & 0 & \cdots & 0 \\ 0 & 0 & 1 & \cdots & 0 \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ -a_0 & -a_1 & -a_2 & \cdots & -a_{n-1} \end{bmatrix} x + \begin{bmatrix} 0 \\ 0 \\ \vdots \\ 1 \end{bmatrix} u$$

**定理 3.1.2 (可控性标准形存在性)**
如果系统 $(A, B)$ 可控，则存在非奇异变换 $T$ 使得 $(T^{-1}AT, T^{-1}B)$ 是可控性标准形。

### 3.2 可观性理论

#### 3.2.1 可观性定义

**定义 3.2.1 (可观性)**
系统 $\Sigma$ 可观，如果任意初始状态 $x_0$ 都可以通过输出 $y(t)$ 唯一确定。

**定义 3.2.2 (可观性矩阵)**
线性系统的可观性矩阵：
$$\mathcal{O} = \begin{bmatrix} C \\ CA \\ CA^2 \\ \vdots \\ CA^{n-1} \end{bmatrix}$$

**定理 3.2.1 (可观性判据)**
线性系统完全可观当且仅当可观性矩阵 $\mathcal{O}$ 满秩。

**证明：** 通过输出方程：

1. **可观测性**：可观性矩阵的行空间包含可观测状态空间
2. **满秩性**：满秩确保状态唯一确定
3. **信息充分性**：输出序列包含足够信息重构状态

#### 3.2.2 可观性标准形

**定义 3.2.3 (可观性标准形)**
可观系统的可观性标准形：
$$\dot{x} = \begin{bmatrix} 0 & 0 & \cdots & 0 & -a_0 \\ 1 & 0 & \cdots & 0 & -a_1 \\ 0 & 1 & \cdots & 0 & -a_2 \\ \vdots & \vdots & \ddots & \vdots & \vdots \\ 0 & 0 & \cdots & 1 & -a_{n-1} \end{bmatrix} x$$
$$y = [0 \quad 0 \quad \cdots \quad 0 \quad 1] x$$

### 3.3 对偶性原理

#### 3.3.1 对偶系统

**定义 3.3.1 (对偶系统)**
系统 $\Sigma = (A, B, C, D)$ 的对偶系统为 $\Sigma^T = (A^T, C^T, B^T, D^T)$。

**定理 3.3.1 (对偶性原理)**
系统 $\Sigma$ 可控当且仅当对偶系统 $\Sigma^T$ 可观。

**证明：** 通过可控性和可观性矩阵的关系：

1. **矩阵关系**：$\mathcal{C}_\Sigma = \mathcal{O}_{\Sigma^T}^T$
2. **秩关系**：$\text{rank}(\mathcal{C}_\Sigma) = \text{rank}(\mathcal{O}_{\Sigma^T})$
3. **等价性**：可控性等价于对偶可观性

## 4. 反馈控制

### 4.1 状态反馈

#### 4.1.1 状态反馈定义

**定义 4.1.1 (状态反馈)**
状态反馈控制律：
$$u(t) = -Kx(t) + r(t)$$

其中 $K \in \mathbb{R}^{m \times n}$ 是反馈增益矩阵，$r(t)$ 是参考输入。

**定义 4.1.2 (闭环系统)**
闭环系统状态方程：
$$\dot{x}(t) = (A - BK)x(t) + Br(t)$$

#### 4.1.2 极点配置

**定理 4.1.1 (极点配置)**
如果系统 $(A, B)$ 可控，则可以通过状态反馈任意配置闭环极点。

**证明：** 通过可控性标准形：

1. **标准形变换**：可控系统可以变换为标准形
2. **极点配置**：标准形下极点配置直接可得
3. **逆变换**：变换回原坐标系得到反馈增益

**算法 4.1.1 (极点配置)**

```haskell
-- 极点配置算法
polePlacement :: Matrix Double -> Matrix Double -> [Complex Double] -> Matrix Double
polePlacement a b desiredPoles = 
  let controllableForm = toControllableForm a b
      kStandard = placePoles controllableForm desiredPoles
      transformation = getTransformation a b
  in kStandard * transformation

-- 转换为可控性标准形
toControllableForm :: Matrix Double -> Matrix Double -> (Matrix Double, Matrix Double)
toControllableForm a b = 
  let c = controllabilityMatrix a b
      t = inverse c
      aBar = t * a * inverse t
      bBar = t * b
  in (aBar, bBar)

-- 在标准形下配置极点
placePoles :: (Matrix Double, Matrix Double) -> [Complex Double] -> Matrix Double
placePoles (aBar, bBar) poles = 
  let characteristicPoly = product [s - pole | pole <- poles]
      coeffs = polynomialCoefficients characteristicPoly
      kStandard = fromList [coeffs !! i | i <- [0..n-1]]
  in kStandard
```

#### 4.1.3 最优状态反馈

**定理 4.1.2 (线性二次调节器)**
对于性能指标：
$$J = \int_0^\infty (x^T Q x + u^T R u) dt$$

最优反馈增益为：
$$K = R^{-1} B^T P$$

其中 $P$ 是代数黎卡提方程的解：
$$A^T P + PA - PBR^{-1}B^T P + Q = 0$$

### 4.2 输出反馈

#### 4.2.1 输出反馈定义

**定义 4.2.1 (输出反馈)**
输出反馈控制律：
$$u(t) = -Ky(t) + r(t)$$

**定义 4.2.2 (输出反馈闭环系统)**
闭环系统状态方程：
$$\dot{x}(t) = (A - BKC)x(t) + Br(t)$$

#### 4.2.2 输出反馈限制

**定理 4.2.1 (输出反馈限制)**
输出反馈不能任意配置极点，只能配置可观部分的极点。

**证明：** 通过可观性分解：

1. **系统分解**：系统可以分解为可观和不可观部分
2. **反馈影响**：输出反馈只能影响可观部分
3. **极点限制**：不可观部分的极点无法通过输出反馈改变

### 4.3 观测器设计

#### 4.3.1 全维观测器

**定义 4.3.1 (全维观测器)**
全维观测器：
$$\dot{\hat{x}}(t) = A\hat{x}(t) + Bu(t) + L(y(t) - C\hat{x}(t))$$

其中 $L \in \mathbb{R}^{n \times p}$ 是观测器增益矩阵。

**定义 4.3.2 (观测误差)**
观测误差：
$$e(t) = x(t) - \hat{x}(t)$$

**定理 4.3.1 (观测误差动态)**
观测误差满足：
$$\dot{e}(t) = (A - LC)e(t)$$

#### 4.3.2 观测器极点配置

**定理 4.3.2 (观测器极点配置)**
如果系统 $(A, C)$ 可观，则可以通过选择 $L$ 任意配置观测器极点。

**证明：** 通过可观性对偶性：

1. **对偶性**：$(A, C)$ 可观等价于 $(A^T, C^T)$ 可控
2. **极点配置**：对偶系统的极点配置问题
3. **增益计算**：$L = K^T$，其中 $K$ 是对偶系统的反馈增益

#### 4.3.3 降维观测器

**定义 4.3.3 (降维观测器)**
降维观测器只估计不可观部分的状态。

**定理 4.3.3 (降维观测器存在性)**
如果系统 $(A, C)$ 可观，则存在降维观测器。

## 5. 最优控制

### 5.1 变分法

#### 5.1.1 变分问题

**定义 5.1.1 (变分问题)**
变分问题：
$$\min_{u(t)} J = \int_{t_0}^{t_f} L(x(t), u(t), t) dt$$

约束条件：
$$\dot{x}(t) = f(x(t), u(t), t)$$

#### 5.1.2 欧拉-拉格朗日方程

**定理 5.1.1 (欧拉-拉格朗日方程)**
最优解满足欧拉-拉格朗日方程：
$$\frac{d}{dt} \frac{\partial L}{\partial \dot{x}} - \frac{\partial L}{\partial x} = 0$$

### 5.2 动态规划

#### 5.2.1 贝尔曼方程

**定义 5.2.1 (值函数)**
值函数：
$$V(x, t) = \min_{u(\tau)} \int_t^{t_f} L(x(\tau), u(\tau), \tau) d\tau$$

**定理 5.2.1 (贝尔曼方程)**
值函数满足贝尔曼方程：
$$-\frac{\partial V}{\partial t} = \min_u \left[ L(x, u, t) + \frac{\partial V}{\partial x} f(x, u, t) \right]$$

### 5.3 线性二次调节器

#### 5.3.1 LQR问题

**定义 5.3.1 (LQR问题)**
线性二次调节器问题：
$$\min_{u(t)} J = \int_0^\infty (x^T Q x + u^T R u) dt$$

约束条件：
$$\dot{x}(t) = Ax(t) + Bu(t)$$

#### 5.3.2 最优解

**定理 5.3.1 (LQR最优解)**
LQR问题的最优解为：
$$u^*(t) = -Kx(t)$$

其中：
$$K = R^{-1} B^T P$$

$P$ 是代数黎卡提方程的解：
$$A^T P + PA - PBR^{-1}B^T P + Q = 0$$

## 6. 鲁棒控制

### 6.1 不确定性建模

#### 6.1.1 不确定性类型

**定义 6.1.1 (参数不确定性)**
参数不确定性：
$$G(s) = G_0(s)(1 + \Delta(s)W(s))$$

其中 $|\Delta(j\omega)| \leq 1$。

**定义 6.1.2 (非结构化不确定性)**
非结构化不确定性：
$$G(s) = G_0(s) + \Delta(s)W(s)$$

### 6.2 H∞控制

#### 6.2.1 H∞范数

**定义 6.2.1 (H∞范数)**
传递函数 $G(s)$ 的H∞范数：
$$\|G\|_\infty = \sup_{\omega} \sigma_{\max}(G(j\omega))$$

#### 6.2.2 H∞控制问题

**定义 6.2.2 (H∞控制问题)**
H∞控制问题：
$$\min_K \|T_{zw}\|_\infty$$

其中 $T_{zw}$ 是从干扰 $w$ 到性能输出 $z$ 的闭环传递函数。

### 6.3 μ综合

#### 6.3.1 结构奇异值

**定义 6.3.1 (结构奇异值)**
结构奇异值：
$$\mu_\Delta(M) = \frac{1}{\min\{\sigma_{\max}(\Delta) : \Delta \in \Delta, \det(I - M\Delta) = 0\}}$$

#### 6.3.2 μ综合算法

**算法 6.3.1 (μ综合)**
μ综合算法：

1. **D-K迭代**：交替优化 $D$ 和 $K$
2. **μ分析**：计算结构奇异值
3. **控制器设计**：基于μ分析结果设计控制器

## 7. 形式化证明

### 7.1 存在性证明

#### 7.1.1 解的存在性

**定理 7.1.1 (解的存在性)**
如果 $f(x, u, t)$ 连续且满足利普希茨条件，则状态方程存在唯一解。

**证明：** 通过皮卡-林德洛夫定理：

1. **连续性**：$f$ 连续确保局部存在性
2. **利普希茨条件**：确保唯一性
3. **延拓定理**：扩展到全局存在性

### 7.2 唯一性证明

#### 7.2.1 最优解唯一性

**定理 7.2.1 (最优解唯一性)**
如果性能指标严格凸，则最优控制解唯一。

**证明：** 通过凸优化理论：

1. **凸性**：严格凸函数有唯一最小值
2. **约束凸性**：凸约束保持凸性
3. **最优性**：最优解唯一

### 7.3 收敛性证明

#### 7.3.1 观测器收敛性

**定理 7.3.1 (观测器收敛性)**
如果观测器极点都在左半平面，则观测误差指数收敛到零。

**证明：** 通过观测误差动态：

1. **稳定性**：观测器极点决定收敛性
2. **指数收敛**：负实部特征值对应指数衰减
3. **全局收敛**：线性系统全局收敛
