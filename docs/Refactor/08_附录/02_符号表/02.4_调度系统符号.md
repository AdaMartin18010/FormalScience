# 调度系统符号表

## 任务模型符号

### 基本任务参数

| 符号 | LaTeX | 含义 | 单位/说明 |
|------|-------|------|-----------|
| $\tau$ | `\tau` | 任务集 | $\tau = \{\tau_1, \tau_2, ..., \tau_n\}$ |
| $\tau_i$ | `\tau_i` | 第i个任务 | 任务标识 |
| $C_i$ | `C_i` | 最坏执行时间 | WCET, 时间单位 |
| $T_i$ | `T_i` | 任务周期 | 时间单位 |
| $D_i$ | `D_i` | 相对截止时间 | 时间单位 |
| $R_i$ | `R_i` | 响应时间 | 实际完成时间 |
| $P_i$ | `P_i` | 任务优先级 | 数值越小优先级越高 |
| $O_i$ | `O_i` | 任务偏移 | 首次释放时间 |
| $J_i$ | `J_i` | 抖动 | 释放时间变化 |

### 派生参数

| 符号 | LaTeX | 含义 | 公式 |
|------|-------|------|------|
| $U_i$ | `U_i` | 任务利用率 | $U_i = C_i / T_i$ |
| $U$ | `U` | 总利用率 | $U = \sum U_i$ |
| $U_{\text{max}}$ | `U_{\text{max}}` | 最大利用率 | $\max_i U_i$ |
| $L_i$ | `L_i` | 任务松弛时间 | $L_i = D_i - C_i$ |
| $S_i$ | `S_i` | 任务跨度 | 关键路径长度 |
| $SL_i$ | `SL_i` | 任务速度级别 | 动态电压调节 |

## 调度算法符号

### 优先级调度

| 符号 | LaTeX | 含义 | 说明 |
|------|-------|------|------|
| $\pi(t)$ | `\pi(t)` | 时刻t的优先级 | 动态优先级 |
| $\pi_i$ | `\pi_i` | 任务i的优先级 | 静态优先级 |
| $h(t)$ | `h(t)` | 时刻t的最高优先级任务 | 调度选择 |
| $hp(i)$ | `hp(i)` | 高于i的优先级任务集 | $hp(i) = \{j \mid P_j < P_i\}$ |
| $hep(i)$ | `hep(i)` | 高于或等于i的任务集 | $hep(i) = \{j \mid P_j \leq P_i\}$ |

### 调度策略

| 符号 | LaTeX | 含义 | 算法 |
|------|-------|------|------|
| $\text{RMS}$ | `\text{RMS}` | 速率单调调度 | 周期短优先级高 |
| $\text{DMS}$ | `\text{DMS}` | 截止期单调调度 | 截止期短优先级高 |
| $\text{EDF}$ | `\text{EDF}` | 最早截止期优先 | 动态优先级 |
| $\text{LLF}$ | `\text{LLF}` | 最小松弛度优先 | $L_i(t) = D_i - t - c_i(t)$ |
| $\text{FIFO}$ | `\text{FIFO}` | 先进先出 | 队列顺序 |
| $\text{SJF}$ | `\text{SJF}` | 最短作业优先 | 最小$C_i$ |
| $\text{RR}_q$ | `\text{RR}_q` | 时间片轮转 | 时间片q |

## 速查索引

### 按类别索引

| 类别 | 关键符号 |
|------|----------|
| 任务参数 | $C_i$, $T_i$, $D_i$, $R_i$, $U_i$, $U$ |
| 调度算法 | RMS, DMS, EDF, LLF, RR, FIFO |
| 处理器 | $m$, $\mathcal{P}$, $s_j$, hom/het |
| 资源 | $\mathcal{R}$, $N_{i,k}$, $CS_{i,k}$ |
| 时间 | $t$, $r_{i,j}$, $d_{i,j}$, $H$, $B$ |
| 可调度性 | DBF, SBF, sched($\tau$), feas($\tau$) |
| 响应时间 | $R_i^{(k)}$, $I_i$, $B_i$ |
| 能耗 | $E$, $P_{dyn}$, $P_{static}$, $s$ |
| 网络 | $B$, $L$, $Q$, WFQ, DRR |
| DAG | $G=(V,E)$, pred, succ, $C_{max}$ |

### 常用公式汇总

| 名称 | 公式 |
|------|------|
| 利用率 | $U_i = C_i / T_i$, $U = \sum_{i=1}^n U_i$ |
| 超周期 | $H = \text{lcm}(T_1, T_2, ..., T_n)$ |
| RMS可调度条件 | $U \leq n(2^{1/n} - 1)$ |
| EDF可调度条件 | $U \leq 1$ |
| 响应时间迭代 | $R_i^{(k+1)} = C_i + \sum_{j \in hp(i)} \lceil R_i^{(k)}/T_j \rceil \cdot C_j$ |
| 动态功率 | $P_{dyn} = C_{eff} \cdot V^2 \cdot f$ |
