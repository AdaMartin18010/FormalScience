## 基本概念

### 1. 集合 (Set)

**定义 4.1** (集合)
集合是一些对象的汇集，这些对象称为集合的元素。

**形式化表示**:

```latex
x \in A \text{ 表示 } x \text{ 是集合 } A \text{ 的元素}
```

### 2. 元素 (Element)

**定义 4.2** (元素)
元素是集合的组成部分，可以是任何对象。

**形式化表示**:

```latex
\text{Element}(x, A) \equiv x \in A
```

### 3. 集合相等

**定义 4.3** (集合相等)
两个集合相等当且仅当它们包含相同的元素。

**形式化表示**:

```latex
A = B \equiv \forall x(x \in A \leftrightarrow x \in B)
```

### 4. 子集

**定义 4.4** (子集)
集合 $A$ 是集合 $B$ 的子集，当且仅当 $A$ 的每个元素都是 $B$ 的元素。

**形式化表示**:

```latex
A \subseteq B \equiv \forall x(x \in A \rightarrow x \in B)
```

## 公理体系

### 公理 1: 外延公理

```latex
\forall A \forall B(\forall x(x \in A \leftrightarrow x \in B) \rightarrow A = B)
```

**解释**: 两个集合相等当且仅当它们包含相同的元素。

### 公理 2: 空集公理

```latex
\exists A \forall x(x \notin A)
```

**解释**: 存在一个不包含任何元素的集合。

### 公理 3: 配对公理

```latex
\forall x \forall y \exists A \forall z(z \in A \leftrightarrow z = x \lor z = y)
```

**解释**: 对于任意两个对象，存在包含它们的集合。

### 公理 4: 并集公理

```latex
\forall F \exists A \forall x(x \in A \leftrightarrow \exists B(B \in F \land x \in B))
```

**解释**: 对于任意集合族，存在包含所有成员元素的集合。

### 公理 5: 幂集公理

```latex
\forall A \exists P \forall B(B \in P \leftrightarrow B \subseteq A)
```

**解释**: 对于任意集合，存在包含其所有子集的集合。

### 公理 6: 分离公理

```latex
\forall A \forall \phi \exists B \forall x(x \in B \leftrightarrow x \in A \land \phi(x))
```

**解释**: 对于任意集合和性质，存在满足该性质的子集。

### 公理 7: 无穷公理

```latex
\exists A(\emptyset \in A \land \forall x(x \in A \rightarrow x \cup \{x\} \in A))
```

**解释**: 存在包含无穷多个元素的集合。

### 公理 8: 选择公理

```latex
\forall F(\emptyset \notin F \land \forall A \forall B(A \in F \land B \in F \land A \neq B \rightarrow A \cap B = \emptyset) \rightarrow \exists C \forall A(A \in F \rightarrow |A \cap C| = 1))
```

**解释**: 对于任意非空不交集合族，存在选择函数。

## 核心定理

### 定理 1: 空集唯一性

**陈述**: 空集是唯一的。

**形式化**:

```latex
\forall A \forall B((\forall x(x \notin A) \land \forall x(x \notin B)) \rightarrow A = B)
```

**证明**:

1. 假设 $\forall x(x \notin A)$ 和 $\forall x(x \notin B)$
2. 对于任意 $x$，$x \in A \leftrightarrow x \in B$ 都成立（因为两边都为假）
3. 根据外延公理，$A = B$

### 定理 2: 子集传递性

**陈述**: 如果 $A \subseteq B$ 且 $B \subseteq C$，那么 $A \subseteq C$。

**形式化**:

```latex
\forall A \forall B \forall C((A \subseteq B \land B \subseteq C) \rightarrow A \subseteq C)
```

**证明**:

1. 假设 $A \subseteq B \land B \subseteq C$
2. 对于任意 $x$，如果 $x \in A$，那么 $x \in B$（因为 $A \subseteq B$）
3. 如果 $x \in B$，那么 $x \in C$（因为 $B \subseteq C$）
4. 因此，如果 $x \in A$，那么 $x \in C$
5. 所以 $A \subseteq C$

### 定理 3: 德摩根律

**陈述**: $(A \cup B)^c = A^c \cap B^c$ 和 $(A \cap B)^c = A^c \cup B^c$。

**形式化**:

```latex
\forall A \forall B((A \cup B)^c = A^c \cap B^c \land (A \cap B)^c = A^c \cup B^c)
```

**证明**:

1. 对于第一个等式：
   - $x \in (A \cup B)^c \leftrightarrow x \notin A \cup B$
   - $\leftrightarrow \neg(x \in A \lor x \in B)$
   - $\leftrightarrow x \notin A \land x \notin B$
   - $\leftrightarrow x \in A^c \land x \in B^c$
   - $\leftrightarrow x \in A^c \cap B^c$

2. 对于第二个等式：
   - $x \in (A \cap B)^c \leftrightarrow x \notin A \cap B$
   - $\leftrightarrow \neg(x \in A \land x \in B)$
   - $\leftrightarrow x \notin A \lor x \notin B$
   - $\leftrightarrow x \in A^c \lor x \in B^c$
   - $\leftrightarrow x \in A^c \cup B^c$

## 证明过程

### 构造性证明方法

```latex
\text{集合构造}: \text{通过公理构造具体集合}
\text{反证法}: \text{通过假设集合不存在导出矛盾}
\text{归纳法}: \text{通过归纳推理证明一般性结论}
```

### 证明示例

**示例 1**: 证明空集是任何集合的子集

```latex
\text{证明}: \forall A(\emptyset \subseteq A)
\text{方法}: \text{反证法}
\text{假设}: \exists A(\emptyset \not\subseteq A)
\text{导出}: \exists x(x \in \emptyset \land x \notin A)
\text{矛盾}: x \in \emptyset \text{ 与空集定义矛盾}
\text{结论}: \forall A(\emptyset \subseteq A)
```

**示例 2**: 证明集合运算的结合律

```latex
\text{证明}: A \cup (B \cup C) = (A \cup B) \cup C
\text{方法}: \text{外延公理}
\text{验证}: \forall x(x \in A \cup (B \cup C) \leftrightarrow x \in (A \cup B) \cup C)
\text{结论}: A \cup (B \cup C) = (A \cup B) \cup C
```

## 集合运算

### 1. 基本运算

```latex
\text{并集}: A \cup B = \{x : x \in A \lor x \in B\}
\text{交集}: A \cap B = \{x : x \in A \land x \in B\}
\text{差集}: A \setminus B = \{x : x \in A \land x \notin B\}
\text{对称差}: A \triangle B = (A \setminus B) \cup (B \setminus A)
```

### 2. 运算性质

```latex
\text{交换律}: A \cup B = B \cup A, A \cap B = B \cap A
\text{结合律}: A \cup (B \cup C) = (A \cup B) \cup C, A \cap (B \cap C) = (A \cap B) \cap C
\text{分配律}: A \cup (B \cap C) = (A \cup B) \cap (A \cup C), A \cap (B \cup C) = (A \cap B) \cup (A \cap C)
\text{德摩根律}: (A \cup B)^c = A^c \cap B^c, (A \cap B)^c = A^c \cup B^c
```

### 3. 幂集运算

```latex
\text{幂集}: \mathcal{P}(A) = \{B : B \subseteq A\}
\text{幂集性质}: |\mathcal{P}(A)| = 2^{|A|}
```

## 集合关系

### 1. 包含关系

```latex
\text{子集}: A \subseteq B \equiv \forall x(x \in A \rightarrow x \in B)
\text{真子集}: A \subset B \equiv A \subseteq B \land A \neq B
\text{超集}: A \supseteq B \equiv B \subseteq A
\text{真超集}: A \supset B \equiv B \subset A
```

### 2. 相等关系

```latex
\text{相等}: A = B \equiv \forall x(x \in A \leftrightarrow x \in B)
\text{不相等}: A \neq B \equiv \neg(A = B)
```

### 3. 集合族关系

```latex
\text{集合族}: \mathcal{F} = \{A_i : i \in I\}
\text{广义并}: \bigcup \mathcal{F} = \bigcup_{i \in I} A_i = \{x : \exists i \in I(x \in A_i)\}
\text{广义交}: \bigcap \mathcal{F} = \bigcap_{i \in I} A_i = \{x : \forall i \in I(x \in A_i)\}
```

## 特殊集合

### 1. 空集

```latex
\text{空集}: \emptyset = \{x : x \neq x\}
\text{空集性质}: \forall A(\emptyset \subseteq A)
\text{空集唯一性}: \text{空集是唯一的}
```

### 2. 单元素集

```latex
\text{单元素集}: \{a\} = \{x : x = a\}
\text{单元素集性质}: a \in \{a\}
```

### 3. 有序对

```latex
\text{有序对}: (a, b) = \{\{a\}, \{a, b\}\}
\text{有序对性质}: (a, b) = (c, d) \leftrightarrow a = c \land b = d
```

### 4. 笛卡尔积

```latex
\text{笛卡尔积}: A \times B = \{(a, b) : a \in A \land b \in B\}
\text{笛卡尔积性质}: |A \times B| = |A| \times |B|
```
