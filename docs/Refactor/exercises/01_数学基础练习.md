## 一、集合论（5题）

### 第1题 ⭐ 集合的基本运算

**题目**：设 $A = \{1, 2, 3, 4\}$，$B = \{3, 4, 5, 6\}$，$C = \{2, 4, 6\}$，求：

1. $A \cup B$
2. $A \cap B$
3. $A \setminus C$
4. $(A \cap B) \setminus C$

<details>
<summary>💡 提示</summary>

回顾并集、交集、差集的定义：

- $A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$
- $A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$
- $A \setminus C = \{x \mid x \in A \text{ 且 } x \notin C\}$

</details>

<details>
<summary>✅ 解答</summary>

1. $A \cup B = \{1, 2, 3, 4, 5, 6\}$
2. $A \cap B = \{3, 4\}$
3. $A \setminus C = \{1, 3\}$（在A中但不在C中的元素）
4. $(A \cap B) \setminus C = \{3, 4\} \setminus \{2, 4, 6\} = \{3\}$

</details>

### 第2题 ⭐⭐ 幂集与基数

**题目**：

1. 求集合 $\{a, b\}$ 的幂集 $\mathcal{P}(\{a, b\})$
2. 若 $|A| = n$，证明 $|\mathcal{P}(A)| = 2^n$

<details>
<summary>💡 提示</summary>

幂集是原集合所有子集构成的集合。对于证明，考虑每个元素在子集中有"出现"或"不出现"两种选择。
</details>

<details>
<summary>✅ 解答</summary>

**第1问**：
$$\mathcal{P}(\{a, b\}) = \{\emptyset, \{a\}, \{b\}, \{a, b\}\}$$

**第2问证明**：

设 $A = \{a_1, a_2, \ldots, a_n\}$，对于任意子集 $S \subseteq A$：

- 每个元素 $a_i$ 有2种选择：属于 $S$ 或不属于 $S$
- 由乘法原理，总子集数为 $2 \times 2 \times \cdots \times 2 = 2^n$

因此 $|\mathcal{P}(A)| = 2^n$。∎
</details>

### 第3题 ⭐⭐ 关系与函数

**题目**：判断下列关系中哪些是函数？如果是函数，判断是否为单射、满射或双射。

1. $f: \mathbb{Z} \to \mathbb{Z}$，$f(x) = x^2$
2. $g: \mathbb{R} \to \mathbb{R}$，$g(x) = 2x + 1$
3. $h: \{1, 2, 3\} \to \{a, b, c, d\}$，$h = \{(1,a), (2,b), (3,c)\}$

<details>
<summary>💡 提示</summary>

函数要求每个定义域元素有唯一像。单射：不同元素映射到不同像；满射：陪域每个元素都有原像。
</details>

<details>
<summary>✅ 解答</summary>

**1. $f(x) = x^2$**

- ✅ 是函数（每个整数有唯一平方）
- ❌ 不是单射：$f(2) = f(-2) = 4$
- ❌ 不是满射：负数（如 -1）没有整数平方根

**2. $g(x) = 2x + 1$**

- ✅ 是函数
- ✅ 单射：若 $g(x_1) = g(x_2)$，则 $2x_1 + 1 = 2x_2 + 1 \Rightarrow x_1 = x_2$
- ✅ 满射：对任意 $y \in \mathbb{R}$，取 $x = (y-1)/2$，则 $g(x) = y$
- ✅ **双射**

**3. $h$**

- ✅ 是函数（每个输入有唯一输出）
- ✅ 单射：不同输入映射到不同输出
- ❌ 不是满射：$d$ 没有原像

</details>

### 第4题 ⭐⭐⭐ 等价关系

**题目**：在整数集 $\mathbb{Z}$ 上定义关系 $R$：$(a, b) \in R$ 当且仅当 $a - b$ 能被3整除。

1. 证明 $R$ 是等价关系
2. 求等价类 $[0]$、$[1]$、$[2]$
3. 描述商集 $\mathbb{Z}/R$

<details>
<summary>💡 提示</summary>

等价关系需满足自反性、对称性、传递性。等价类 $[a]$ 包含所有与 $a$ 等价的元素。
</details>

<details>
<summary>✅ 解答</summary>

**1. 证明 $R$ 是等价关系**

记 $a \equiv b \pmod{3}$ 表示 $3 \mid (a - b)$

- **自反性**：$\forall a \in \mathbb{Z}$，$a - a = 0 = 3 \times 0$，故 $(a, a) \in R$ ✓
- **对称性**：若 $(a, b) \in R$，则 $a - b = 3k$，于是 $b - a = -3k = 3(-k)$，故 $(b, a) \in R$ ✓
- **传递性**：若 $(a, b) \in R$ 且 $(b, c) \in R$，则 $a - b = 3k_1$，$b - c = 3k_2$，于是 $a - c = 3(k_1 + k_2)$，故 $(a, c) \in R$ ✓

**2. 等价类**

- $[0] = \{3k \mid k \in \mathbb{Z}\} = \{\ldots, -6, -3, 0, 3, 6, \ldots\}$
- $[1] = \{3k + 1 \mid k \in \mathbb{Z}\} = \{\ldots, -5, -2, 1, 4, 7, \ldots\}$
- $[2] = \{3k + 2 \mid k \in \mathbb{Z}\} = \{\ldots, -4, -1, 2, 5, 8, \ldots\}$

**3. 商集**

$$\mathbb{Z}/R = \{[0], [1], [2]\} \cong \mathbb{Z}_3$$

这正是模3的剩余类，与整数模3同构。
</details>

### 第5题 ⭐⭐⭐ 无限集的基数

**题目**：证明自然数集 $\mathbb{N}$ 与有理数集 $\mathbb{Q}$ 等势（即存在双射）。

<details>
<summary>💡 提示</summary>

使用Cantor-Bernstein定理：若存在单射 $f: A \to B$ 和 $g: B \to A$，则 $|A| = |B|$。或者直接构造从 $\mathbb{N}$ 到 $\mathbb{Q}^+$ 的枚举。
</details>

<details>
<summary>✅ 解答</summary>

**方法：Cantor对角线枚举**

首先，$\mathbb{Q} = \mathbb{Q}^- \cup \{0\} \cup \mathbb{Q}^+$，只需证明 $|\mathbb{N}| = |\mathbb{Q}^+|$。

将有理数按以下方式排列（Cantor矩阵）：

```
    1   2   3   4   5  ...
1  1/1 1/2 1/3 1/4 1/5
2  2/1 2/2 2/3 2/4  ...
3  3/1 3/2 3/3  ...  ...
4  4/1 4/2  ...  ...  ...
5  5/1  ...  ...  ...  ...
```

按对角线方向枚举（跳过已出现的等价分数）：
$$1/1, 1/2, 2/1, 1/3, 3/1, 1/4, 2/3, 3/2, 4/1, \ldots$$

这给出了 $\mathbb{N} \to \mathbb{Q}^+$ 的双射。

**结论**：$|\mathbb{N}| = |\mathbb{Q}| = \aleph_0$（可数无穷）。∎
</details>

## 二、代数基础（5题）

### 第6题 ⭐ 群的判定

**题目**：判断下列代数系统是否为群：

1. $(\mathbb{Z}, +)$
2. $(\mathbb{Z}, \times)$（普通乘法）
3. $(\{0, 1, 2\}, +_3)$（模3加法）

<details>
<summary>💡 提示</summary>

群需满足：封闭性、结合律、单位元、逆元。逐一验证。
</details>

<details>
<summary>✅ 解答</summary>

**1. $(\mathbb{Z}, +)$**

- ✅ 封闭性：整数加整数仍为整数
- ✅ 结合律：$(a+b)+c = a+(b+c)$
- ✅ 单位元：$0$，因为 $a + 0 = a$
- ✅ 逆元：$a$ 的逆元是 $-a$

**是群**（实际上是阿贝尔群）

**2. $(\mathbb{Z}, \times)$**

- ✅ 封闭性、结合律、单位元（1）
- ❌ 逆元：除 $\pm 1$ 外，整数在乘法下无逆元（$1/2 \notin \mathbb{Z}$）

**不是群**

**3. $(\{0, 1, 2\}, +_3)$**

- ✅ 封闭性：模3加法结果在 $\{0,1,2\}$
- ✅ 结合律：模运算保持结合律
- ✅ 单位元：$0$
- ✅ 逆元：$0^{-1}=0$，$1^{-1}=2$，$2^{-1}=1$

**是群**（有限循环群 $\mathbb{Z}_3$）
</details>

### 第7题 ⭐⭐ 子群与陪集

**题目**：设 $G = \mathbb{Z}_6 = \{0, 1, 2, 3, 4, 5\}$（模6加法群），$H = \{0, 3\}$。

1. 证明 $H$ 是 $G$ 的子群
2. 求 $H$ 在 $G$ 中的所有左陪集

<details>
<summary>💡 提示</summary>

子群判定：非空子集，对运算和逆元封闭。陪集形如 $gH = \{gh \mid h \in H\}$。
</details>

<details>
<summary>✅ 解答</summary>

**1. 证明 $H$ 是子群**

- $H \subseteq G$ 且 $H \neq \emptyset$ ✓
- 运算封闭：$0 +_3 0 = 0$，$0 +_3 3 = 3$，$3 +_3 3 = 0$，都在 $H$ 中 ✓
- 逆元：$0^{-1} = 0 \in H$，$3^{-1} = 3 \in H$ ✓

因此 $H \leq G$。

**2. 左陪集**

- $0 + H = \{0, 3\}$
- $1 + H = \{1, 4\}$
- $2 + H = \{2, 5\}$
- $3 + H = \{3, 0\} = 0 + H$
- $4 + H = \{4, 1\} = 1 + H$
- $5 + H = \{5, 2\} = 2 + H$

不同的左陪集有3个：$\{0, 3\}$，$\{1, 4\}$，$\{2, 5\}$

注意：$[G : H] = |G|/|H| = 6/2 = 3$，符合Lagrange定理。
</details>

### 第8题 ⭐⭐ 同态与同构

**题目**：定义映射 $\phi: (\mathbb{R}, +) \to (\mathbb{R}^+, \times)$，$\phi(x) = e^x$。

1. 证明 $\phi$ 是群同态
2. 证明 $\phi$ 是群同构

<details>
<summary>💡 提示</summary>

同态：$\phi(a + b) = \phi(a) \times \phi(b)$。同构：双射的同态。
</details>

<details>
<summary>✅ 解答</summary>

**1. 证明是同态**

对任意 $a, b \in \mathbb{R}$：
$$\phi(a + b) = e^{a+b} = e^a \times e^b = \phi(a) \times \phi(b)$$

满足同态条件。

**2. 证明是同构**

需证 $\phi$ 是双射：

- **单射**：若 $\phi(a) = \phi(b)$，则 $e^a = e^b$，取对数得 $a = b$ ✓
- **满射**：对任意 $y \in \mathbb{R}^+$，取 $x = \ln y \in \mathbb{R}$，则 $\phi(x) = e^{\ln y} = y$ ✓

因此 $\phi$ 是同构，$(\mathbb{R}, +) \cong (\mathbb{R}^+, \times)$。

> 这说明：加法群和乘法群本质上是相同的代数结构！
</details>

### 第9题 ⭐⭐⭐ 环与理想

**题目**：

1. 证明 $(\mathbb{Z}_6, +_6, \times_6)$ 是交换环
2. 求它的所有理想
3. 判断是否为整环和域

<details>
<summary>💡 提示</summary>

环需满足：加法群、乘法半群、分配律。整环要求无零因子，域要求非零元都有逆元。
</details>

<details>
<summary>✅ 解答</summary>

**1. 证明是交换环**

- $(\mathbb{Z}_6, +_6)$ 是阿贝尔群（模6加法群）
- $(\mathbb{Z}_6, \times_6)$ 是交换幺半群（乘法封闭、结合、交换，单位元1）
- 分配律：$a \times_6 (b +_6 c) = (a \times_6 b) +_6 (a \times_6 c)$（模运算保持分配律）

因此是交换环。

**2. 所有理想**

$\mathbb{Z}_n$ 的理想对应于 $n$ 的因数：

- $I_1 = \{0\}$
- $I_2 = \{0, 2, 4\}$（由2生成）
- $I_3 = \{0, 3\}$（由3生成）
- $I_6 = \mathbb{Z}_6$（由1生成）

**3. 判断**

- **不是整环**：$2 \times_6 3 = 0$，但 $2 \neq 0$，$3 \neq 0$，存在零因子
- **不是域**：2没有乘法逆元（$2 \times_6 x = 1$ 无解）

> 一般结论：$\mathbb{Z}_n$ 是域 ⟺ $n$ 是素数
</details>

### 第10题 ⭐⭐⭐ 格与布尔代数

**题目**：设 $B = \mathcal{P}(\{a, b\})$，运算为集合交 $\cap$ 和并 $\cup$。

1. 证明 $(B, \cap, \cup)$ 构成格
2. 证明它是分配格
3. 证明它是布尔代数（求补运算）

<details>
<summary>💡 提示</summary>

格：偏序集且任意两元素有最大下界和最小上界。布尔代数：有补的分配格。
</details>

<details>
<summary>✅ 解答</summary>

**1. 证明是格**

$B = \{\emptyset, \{a\}, \{b\}, \{a,b\}\}$

对任意 $X, Y \in B$：

- 最大下界：$X \cap Y \in B$
- 最小上界：$X \cup Y \in B$

因此 $(B, \cap, \cup)$ 是格。

**2. 证明是分配格**

需验证分配律：$X \cap (Y \cup Z) = (X \cap Y) \cup (X \cap Z)$

这是集合运算的基本性质，对所有集合成立，故是分配格。

**3. 证明是布尔代数**

- 最小元：$\emptyset$，最大元：$\{a,b\}$
- 补运算：$X^c = \{a,b\} \setminus X$
  - $\emptyset^c = \{a,b\}$
  - $\{a\}^c = \{b\}$
  - $\{b\}^c = \{a\}$
  - $\{a,b\}^c = \emptyset$
- 验证：$X \cap X^c = \emptyset$，$X \cup X^c = \{a,b\}$

因此 $(B, \cap, \cup, {}^c, \emptyset, \{a,b\})$ 是布尔代数。

> 一般结论：$\mathcal{P}(S)$ 总是布尔代数，同构于 $\{0,1\}^{|S|}$
</details>

### 第12题 ⭐⭐ 真值表与等价

**题目**：用真值表证明德摩根定律：$\neg(p \wedge q) \equiv \neg p \vee \neg q$

<details>
<summary>💡 提示</summary>

构造包含 $p$、$q$、$p \wedge q$、$\neg(p \wedge q)$、$\neg p$、$\neg q$、$\neg p \vee \neg q$ 的真值表，比较最后两列。
</details>

<details>
<summary>✅ 解答</summary>

| $p$ | $q$ | $p \wedge q$ | $\neg(p \wedge q)$ | $\neg p$ | $\neg q$ | $\neg p \vee \neg q$ |
|:---:|:---:|:------------:|:------------------:|:--------:|:--------:|:--------------------:|
| T | T | T | F | F | F | F |
| T | F | F | T | F | T | T |
| F | T | F | T | T | F | T |
| F | F | F | T | T | T | T |

比较第4列和第7列，完全一致。因此：
$$\neg(p \wedge q) \equiv \neg p \vee \neg q$$ ∎

同理可证：$\neg(p \vee q) \equiv \neg p \wedge \neg q$
</details>

### 第13题 ⭐⭐ 范式转换

**题目**：将公式 $(p \rightarrow q) \wedge r$ 转换为：

1. 合取范式（CNF）
2. 析取范式（DNF）

<details>
<summary>💡 提示</summary>

先用 $p \rightarrow q \equiv \neg p \vee q$ 消去蕴含，然后用分配律展开。
</details>

<details>
<summary>✅ 解答</summary>

**原式**：$(p \rightarrow q) \wedge r = (\neg p \vee q) \wedge r$

**1. CNF（已经是合取范式）**
$$(\neg p \vee q) \wedge r$$

这是两个子句的合取，满足CNF形式。

**2. DNF（用分配律展开）**
$$(\neg p \vee q) \wedge r = (\neg p \wedge r) \vee (q \wedge r)$$

这是两个合取式的析取，满足DNF形式。
</details>

### 第14题 ⭐⭐ 谓词逻辑

**题目**：将下列语句符号化（选择适当的论域）：

1. "所有人都是会死的"
2. "存在一个偶素数"
3. "每个实数都有唯一的加法逆元"

<details>
<summary>💡 提示</summary>

使用全称量词 $\forall$、存在量词 $\exists$、唯一量词 $\exists!$。注意论域的选择影响谓词的定义。
</details>

<details>
<summary>✅ 解答</summary>

**1. "所有人都是会死的"**

论域：所有人

- $M(x)$：$x$ 是会死的
$$\forall x \cdot M(x)$$

或论域：所有生物

- $H(x)$：$x$ 是人
$$\forall x \cdot (H(x) \rightarrow M(x))$$

**2. "存在一个偶素数"**

论域：自然数

- $E(x)$：$x$ 是偶数，$P(x)$：$x$ 是素数
$$\exists x \cdot (E(x) \wedge P(x))$$

> 实际上这个数是2

**3. "每个实数都有唯一的加法逆元"**

论域：实数

- $plus(x, y, z)$：$x + y = z$，即 $z$ 是 $x$ 和 $y$ 的和
$$\forall x \cdot \exists! y \cdot plus(x, y, 0)$$

或更直观地：
$$\forall x \cdot \exists! y \cdot (x + y = 0)$$

> 显然 $y = -x$ 是唯一满足条件的实数
</details>

### 第15题 ⭐⭐⭐ 归结原理

**题目**：用归结原理证明：从前提 $\{p \vee q, \neg p \vee r, \neg q \vee r, \neg r\}$ 可推出矛盾。

<details>
<summary>💡 提示</summary>

归结规则：从 $A \vee B$ 和 $\neg B \vee C$ 推出 $A \vee C$。不断归结直到得到空子句（矛盾）。
</details>

<details>
<summary>✅ 解答</summary>

**前提子句**：

1. $p \vee q$
2. $\neg p \vee r$
3. $\neg q \vee r$
4. $\neg r$

**归结过程**：

从 (2) 和 (4)：$\neg p \vee r$ 与 $\neg r$ 归结得
5. $\neg p$

从 (3) 和 (4)：$\neg q \vee r$ 与 $\neg r$ 归结得
6. $\neg q$

从 (1) 和 (5)：$p \vee q$ 与 $\neg p$ 归结得
7. $q$

从 (6) 和 (7)：$\neg q$ 与 $q$ 归结得
8. $\square$（空子句，矛盾）

因此前提集是不可满足的。∎

> 归结原理是自动定理证明和逻辑编程（如Prolog）的基础
</details>

## 四、证明技术（5题）

### 第16题 ⭐ 直接证明

**题目**：证明：若 $n$ 是偶数，则 $n^2$ 是偶数。

<details>
<summary>💡 提示</summary>

直接证明：假设前提成立，通过逻辑推导得出结论。
</details>

<details>
<summary>✅ 解答</summary>

**证明**：

设 $n$ 是偶数，则存在整数 $k$ 使得 $n = 2k$。

于是：
$$n^2 = (2k)^2 = 4k^2 = 2(2k^2)$$

令 $m = 2k^2$，则 $n^2 = 2m$，其中 $m$ 是整数。

因此 $n^2$ 是偶数。∎
</details>

### 第17题 ⭐ 反证法

**题目**：用反证法证明：$\sqrt{2}$ 是无理数。

<details>
<summary>💡 提示</summary>

假设结论不成立（$\sqrt{2}$ 是有理数），推导出矛盾。
</details>

<details>
<summary>✅ 解答</summary>

**证明**：

假设 $\sqrt{2}$ 是有理数，则可表示为 $\sqrt{2} = p/q$，其中 $p, q \in \mathbb{Z}$，$q \neq 0$，且 $\gcd(p,q) = 1$（最简形式）。

两边平方：
$$2 = \frac{p^2}{q^2} \Rightarrow p^2 = 2q^2$$

因此 $p^2$ 是偶数，由第16题知 $p$ 是偶数。

设 $p = 2k$，代入得：
$$(2k)^2 = 2q^2 \Rightarrow 4k^2 = 2q^2 \Rightarrow q^2 = 2k^2$$

因此 $q^2$ 是偶数，$q$ 也是偶数。

但 $p$ 和 $q$ 都是偶数，与 $\gcd(p,q) = 1$ 矛盾！

因此假设不成立，$\sqrt{2}$ 是无理数。∎

> 这是反证法的经典例子，由古希腊人发现
</details>

### 第18题 ⭐⭐ 数学归纳法

**题目**：用数学归纳法证明：对所有 $n \geq 1$：
$$1 + 2 + 3 + \cdots + n = \frac{n(n+1)}{2}$$

<details>
<summary>💡 提示</summary>

归纳法三步：基础情况（$n=1$）、归纳假设（假设对 $n=k$ 成立）、归纳步骤（证明对 $n=k+1$ 成立）。
</details>

<details>
<summary>✅ 解答</summary>

**证明**：

**基础情况**（$n = 1$）：
左边 $= 1$，右边 $= \frac{1 \times 2}{2} = 1$ ✓

**归纳假设**：假设对 $n = k$ 成立，即：
$$1 + 2 + \cdots + k = \frac{k(k+1)}{2}$$

**归纳步骤**（$n = k + 1$）：
$$\begin{aligned}
1 + 2 + \cdots + k + (k+1) &= \frac{k(k+1)}{2} + (k+1) \\
&= \frac{k(k+1) + 2(k+1)}{2} \\
&= \frac{(k+1)(k+2)}{2}
\end{aligned}$$

这正是 $n = k+1$ 时的公式。

由数学归纳法，公式对所有 $n \geq 1$ 成立。∎

> 传说高斯小学时用这个技巧快速计算了 $1+2+\cdots+100$
</details>

### 第19题 ⭐⭐ 结构归纳法

**题目**：设二叉树的定义：空树是二叉树；若 $T_1$、$T_2$ 是二叉树，则 $\text{node}(v, T_1, T_2)$ 也是二叉树。

证明：任何有 $n$ 个节点的二叉树都有 $n-1$ 条边。

<details>
<summary>💡 提示</summary>

结构归纳法：对递归结构的构造进行归纳。基础：空树；归纳：非空树的左右子树。
</details>

<details>
<summary>✅ 解答</summary>

**证明**：

对二叉树结构进行归纳。

**基础情况**：空树有0个节点，0条边。公式 $0 - 1 = -1$？修正：非空树。

重新表述：**非空**二叉树有 $n$ 个节点，则有 $n-1$ 条边。

**基础情况**：只有一个节点的树，有0条边，$1-1=0$ ✓

**归纳假设**：假设对节点数少于 $n$ 的所有二叉树成立。

**归纳步骤**：设 $T = \text{node}(v, T_1, T_2)$ 有 $n$ 个节点。

- 设 $T_1$ 有 $n_1$ 个节点，$T_2$ 有 $n_2$ 个节点
- 则 $n = 1 + n_1 + n_2$

由归纳假设：
- $T_1$ 有 $n_1 - 1$ 条边
- $T_2$ 有 $n_2 - 1$ 条边

$T$ 的边数 = $T_1$ 的边 + $T_2$ 的边 + 2（连接根到左右子树的边）
$$= (n_1 - 1) + (n_2 - 1) + 2 = n_1 + n_2 = n - 1$$

因此命题对所有二叉树成立。∎
</details>

### 第20题 ⭐⭐⭐ 构造性证明与存在性证明

**题目**：

1. 用构造性证明：存在无理数 $a, b$ 使得 $a^b$ 是有理数
2. 比较构造性证明与非构造性证明的区别

<details>
<summary>💡 提示</summary>

考虑 $\sqrt{2}^{\sqrt{2}}$。分两种情况讨论：它是有理数或无理数。
</details>

<details>
<summary>✅ 解答</summary>

**1. 构造性证明**

**情况1**：若 $\sqrt{2}^{\sqrt{2}}$ 是有理数

则取 $a = \sqrt{2}$，$b = \sqrt{2}$，都是无理数，而 $a^b$ 是有理数。✓

**情况2**：若 $\sqrt{2}^{\sqrt{2}}$ 是无理数

则取 $a = \sqrt{2}^{\sqrt{2}}$，$b = \sqrt{2}$。

$$a^b = (\sqrt{2}^{\sqrt{2}})^{\sqrt{2}} = \sqrt{2}^{\sqrt{2} \cdot \sqrt{2}} = \sqrt{2}^2 = 2$$

2是有理数。✓

无论哪种情况，都存在满足条件的无理数 $a, b$。

> 实际上已知 $\sqrt{2}^{\sqrt{2}}$ 是无理数（Gelfond-Schneider定理），但上述证明无需知道这一点。

**2. 比较**

| 特征 | 构造性证明 | 非构造性证明（经典） |
|------|-----------|---------------------|
| 核心 | 显式给出实例 | 证明存在性而不给出实例 |
| 方法 | 直接构造 | 反证法、鸽巢原理等 |
| 信息 | 提供具体的解 | 只说明解存在 |
| 计算 | 通常可计算 | 可能不可计算 |

**例子对比**：
- 构造性："$\sqrt{2}$ 和 $\log_2 9$ 使 $(\sqrt{2})^{\log_2 9} = 3$"
- 非构造性："考虑 $\sqrt{2}^{\sqrt{2}}$，若无理则情况1，若有理则情况2"

构造性证明在计算机科学中尤为重要，因为它提供了算法可计算的解。
</details>
