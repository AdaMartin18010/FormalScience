# 01 入门指南系列 - 视频教程大纲

> 形式化方法入门：从零开始理解证明与验证

---

## 系列概览

| 属性 | 内容 |
|------|------|
| **总集数** | 10集 |
| **总时长** | 约5小时（每集25-35分钟） |
| **难度** | ⭐ 初级 |
| **目标受众** | 编程初学者、计算机科学学生、对形式化方法感兴趣的开发者 |
| **前置知识** | 基础编程概念（变量、函数、条件语句） |

## 学习目标

完成本系列后，学员将能够：

- 理解形式化方法的核心概念与应用场景
- 掌握基本的逻辑推理方法
- 了解定理证明的基本流程
- 为后续Lean学习打下坚实基础

---

## 第1集：什么是形式化方法？

**时长**: 30分钟
**难度**: ⭐

### 内容大纲

1. **开场介绍** (3分钟)
   - 讲师自我介绍
   - 系列课程概览
   - 学习路径说明

2. **形式化方法定义** (8分钟)
   - 什么是形式化方法
   - 与传统软件开发方法的区别
   - 为什么需要形式化方法

3. **实际案例展示** (12分钟)
   - Ariane 5 火箭爆炸事故分析
   - Intel Pentium FDIV 缺陷
   - Therac-25 医疗事故
   - 形式化方法如何避免这些错误

4. **形式化方法的应用领域** (5分钟)
   - 航空航天
   - 医疗设备
   - 金融系统
   - 操作系统内核

5. **总结与下集预告** (2分钟)

### 学习目标

- 理解形式化方法的基本概念
- 认识到形式化方法的重要性
- 了解形式化方法的实际应用场景

### 配套练习

1. 研究一个软件故障案例，分析形式化方法如何帮助避免
2. 列出你所在领域中可能受益于形式化方法的应用场景

### 前置知识

- 无特殊要求

## 第2集：数学证明基础

**时长**: 35分钟
**难度**: ⭐

### 内容大纲

1. **什么是数学证明** (5分钟)
   - 定义与概念
   - 证明 vs 验证
   - 公理系统简介

2. **直接证明法** (8分钟)
   - 基本概念
   - 示例：证明偶数之和为偶数
   - 实战演练

3. **反证法** (8分钟)
   - 原理与逻辑
   - 经典示例：√2是无理数
   - 何时使用反证法

4. **数学归纳法** (10分钟)
   - 归纳原理
   - 基础案例与归纳步骤
   - 示例：证明1+2+...+n = n(n+1)/2
   - 强归纳法简介

5. **证明写作规范** (4分钟)

### 学习目标

- 掌握三种基本证明方法
- 能够阅读和理解简单的数学证明
- 学会规范地书写证明过程

### 配套练习

1. 用直接证明法证明：两个奇数的乘积是奇数
2. 用反证法证明：质数有无穷多个
3. 用数学归纳法证明：2^n > n 对所有正整数n成立

### 前置知识

- 第1集：什么是形式化方法？

## 第3集：命题逻辑入门

**时长**: 30分钟
**难度**: ⭐⭐

### 内容大纲

1. **命题与真值** (5分钟)
   - 什么是命题
   - 真值表基础
   - 原子命题与复合命题

2. **逻辑连接词** (12分钟)
   - 合取（AND）：∧
   - 析取（OR）：∨
   - 蕴含（IMPLIES）：→
   - 等价（IFF）：↔
   - 否定（NOT）：¬
   - 每个连接词的真值表与示例

3. **逻辑等价** (8分钟)
   - 德摩根定律
   - 分配律
   - 结合律与交换律
   - 证明逻辑等价

4. **实际应用** (5分钟)
   - 程序中的逻辑表达式
   - 规范说明中的使用

### 学习目标

- 理解命题逻辑的基本概念
- 掌握五个基本逻辑连接词
- 能够构建和解读真值表
- 理解常见的逻辑等价关系

### 配套练习

1. 为表达式 (p ∧ q) → r 构建完整的真值表
2. 证明 ¬(p ∧ q) 等价于 ¬p ∨ ¬q
3. 将自然语言语句转换为逻辑表达式

### 前置知识

- 第2集：数学证明基础

---

## 第4集：一阶逻辑

**时长**: 35分钟
**难度**: ⭐⭐

### 内容大纲

1. **从命题逻辑到谓词逻辑** (5分钟)
   - 命题逻辑的局限性
   - 引入量词的必要性

2. **全称量词 ∀** (10分钟)
   - 定义与含义
   - 示例：∀x (Human(x) → Mortal(x))
   - 证明全称命题
   - 反例的概念

3. **存在量词 ∃** (10分钟)
   - 定义与含义
   - 示例：∃x (Prime(x) ∧ x > 100)
   - 证明存在命题
   - 构造性证明 vs 非构造性证明

4. **量词的嵌套与顺序** (6分钟)
   - ∀x ∃y 与 ∃y ∀x 的区别
   - 多个量词的组合
   - 经典示例分析

5. **量词否定规则** (4分钟)
   - ¬∀x P(x) ≡ ∃x ¬P(x)
   - ¬∃x P(x) ≡ ∀x ¬P(x)

### 学习目标

- 理解全称量词和存在量词
- 能够阅读和书写一阶逻辑公式
- 理解量词顺序的重要性
- 掌握量词否定规则

### 配套练习

1. 将"每个人都有一位朋友"形式化为逻辑公式
2. 将"存在一个数大于所有其他数"形式化，并分析其真假
3. 练习量词否定转换

### 前置知识

- 第3集：命题逻辑入门
