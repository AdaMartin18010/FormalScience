# 04 Rust形式化系列 - 视频教程大纲

> 用形式化方法验证Rust程序：所有权、类型与内存安全

---

## 系列概览

| 属性 | 内容 |
|------|------|
| **总集数** | 10集 |
| **总时长** | 约6小时（每集30-40分钟） |
| **难度** | ⭐⭐⭐⭐ 中高级 |
| **目标受众** | 熟悉Rust和Lean，希望进行形式化验证的开发者 |
| **前置知识** | [02 Lean基础](02_lean_basics_outline.md)、[03 类型论](03_type_theory_outline.md) 系列 |

## 学习目标

完成本系列后，学员将能够：

- 理解Rust形式化验证的理论基础
- 掌握所有权系统的形式化模型
- 使用Lean验证Rust程序的关键性质
- 理解Rust类型系统的安全保证
- 实现简单的Rust子集形式化验证

## 第1集：Rust形式化概述

**时长**: 35分钟
**难度**: ⭐⭐⭐

### 内容大纲

1. **为什么验证Rust** (8分钟)

2. **Rust验证方法概览** (12分钟)
   - 类型系统验证
   - 分离逻辑（Viper, Prusti）
   - 模型检测
   - 定理证明（RustBelt, Iris）
   - 各方法的优缺点

3. **RustBelt与Iris** (8分钟)
   - RustBelt项目介绍
   - Iris框架概述
   - 逻辑关系方法
   - 验证案例

4. **本系列路线图** (7分钟)
   - 学习目标
   - 系列结构
   - 实践项目预览

### 学习目标

- 了解Rust验证的不同方法
- 理解RustBelt的贡献
- 明确学习路径

### 配套练习

1. 研究一个Rust unsafe代码的bug案例
2. 比较不同的Rust验证工具
3. 阅读RustBelt论文摘要

### 前置知识

- Rust编程基础
- Lean证明经验
