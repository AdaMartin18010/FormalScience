# 02 Lean基础系列 - 视频教程大纲

> 掌握Lean 4证明助手：从安装到独立证明

---

## 系列概览

| 属性 | 内容 |
|------|------|
| **总集数** | 15集 |
| **总时长** | 约8小时（每集30-35分钟） |
| **难度** | ⭐⭐ 初中级 |
| **目标受众** | 有一定编程经验，希望学习定理证明的开发者 |
| **前置知识** | [01 入门指南](01_getting_started_outline.md) 系列 |

## 学习目标

完成本系列后，学员将能够：

- 熟练安装和配置Lean 4开发环境
- 理解Lean的类型系统和核心概念
- 使用tactic编写清晰的证明
- 定义归纳类型和递归函数
- 独立完成中等难度的数学证明
- 阅读和理解Mathlib中的证明
