# 03 类型论系列 - 视频教程大纲

> 从简单类型到依赖类型：类型系统的理论基础与实践

---

## 系列概览

| 属性 | 内容 |
|------|------|
| **总集数** | 12集 |
| **总时长** | 约7小时（每集30-40分钟） |
| **难度** | ⭐⭐⭐ 中级 |
| **目标受众** | 已掌握Lean基础，希望深入理解类型理论的学员 |
| **前置知识** | [02 Lean基础](02_lean_basics_outline.md) 系列 |

## 学习目标

完成本系列后，学员将能够：

- 理解类型论的核心概念与理论框架
- 掌握简单类型λ演算（STLC）
- 理解多态类型系统（System F）
- 深入理解依赖类型及其应用
- 掌握Curry-Howard对应关系
- 能够设计和分析类型系统

---

## 第1集：类型论概述与历史

**时长**: 35分钟
**难度**: ⭐⭐

### 内容大纲

1. **类型论的历史** (10分钟)
   - Russell的悖论与类型论起源
   - Church的λ演算
   - Curry与函数式编程
   - Martin-Löf类型论
   - 现代发展（CoC, CIC）

2. **类型论 vs 集合论** (10分钟)
   - 构造性 vs 经典数学
   - 类型作为规范 vs 类型作为集合
   - 计算内容

3. **类型论的家族** (10分钟)
   - 简单类型λ演算
   - 多态类型
   - 依赖类型
   - 线性类型
   - 效应系统

4. **应用领域** (5分钟)
   - 编程语言设计
   - 定理证明
   - 程序验证

### 学习目标

- 了解类型论的历史发展
- 理解类型论与集合论的区别
- 认识类型论的各个分支

### 配套练习

1. 研究Russell悖论与类型论的关联
2. 对比类型论和集合论处理函数的方式
3. 列出3个使用依赖类型的实际系统

### 前置知识

- [02系列](02_lean_basics_outline.md) 完成

## 第2集：简单类型λ演算（STLC）

**时长**: 40分钟
**难度**: ⭐⭐⭐

### 内容大纲

1. **λ演算回顾** (8分钟)
   - 语法：变量、抽象、应用
   - α-等价
   - β-归约
   - η-等价

2. **简单类型** (10分钟)
   - 基本类型
   - 函数类型 A → B
   - 类型语境
   - 类型判断 Γ ⊢ t : A

3. **类型规则** (12分钟)
   - 变量规则
   - 抽象规则（→I）
   - 应用规则（→E）
   - 类型推导示例
   - 类型唯一性

4. **类型安全** (10分钟)
   - 进展性（Progress）
   - 保持性（Preservation）
   - 规范化（Normalization）
   - 强规范化证明概览

### 学习目标

- 掌握STLC的语法和类型规则
- 能够进行类型推导
- 理解类型安全定理

### 配套练习

1. 为给定的λ项推导类型
2. 证明类型唯一性定理
3. 实现一个简单类型检查器

### 前置知识

- 第1集：类型论概述与历史
