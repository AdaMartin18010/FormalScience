# 05 调度器设计系列 - 视频教程大纲

> 从理论到实现：形式化方法指导的调度器设计

---

## 系列概览

| 属性 | 内容 |
|------|------|
| **总集数** | 8集 |
| **总时长** | 约5小时（每集35-40分钟） |
| **难度** | ⭐⭐⭐⭐⭐ 高级 |
| **目标受众** | 系统程序员，希望用形式化方法设计调度系统 |
| **前置知识** | [02 Lean基础](02_lean_basics_outline.md)、[03 类型论](03_type_theory_outline.md) 系列 |

## 学习目标

完成本系列后，学员将能够：

- 理解调度算法的理论基础
- 使用形式化方法指定和验证调度器
- 设计和实现高效的任务调度系统
- 分析调度器的实时性能和公平性
- 掌握调度系统的测试和验证方法

---

## 第1集：调度理论基础

**时长**: 40分钟
**难度**: ⭐⭐⭐⭐

### 内容大纲

1. **调度问题定义** (8分钟)
   - 什么是调度
   - 调度问题的形式化
   - 优化目标
   - 约束条件

2. **调度算法分类** (12分钟)
   - 抢占式 vs 非抢占式
   - 在线 vs 离线
   - 单处理器 vs 多处理器
   - 实时 vs 尽力而为

3. **经典算法回顾** (12分钟)
   - FCFS（先来先服务）
   - SJF（最短作业优先）
   - Round Robin（时间片轮转）
   - Priority Scheduling
   - 多级反馈队列

4. **性能指标** (8分钟)
   - 周转时间
   - 响应时间
   - 等待时间
   - 吞吐量
   - 公平性度量

### 学习目标

- 理解调度问题的数学定义
- 掌握经典调度算法
- 理解不同性能指标

### 配套练习

1. 计算不同调度算法的性能指标
2. 分析实际系统中的调度策略
3. 设计针对特定场景的调度算法

### 前置知识

- 基础算法知识
- 概率论基础

## 第2集：实时调度理论

**时长**: 40分钟
**难度**: ⭐⭐⭐⭐

   - 算法描述
   - 最优性证明
   - 利用率界限
   - 与RM的比较

4. **形式化规范** (8分钟)
   - 任务集定义
   - 调度可行性
   - 在Lean中定义

### 学习目标

- 掌握实时调度理论
- 理解RM和EDF算法
- 能够进行可调度性分析

### 配套练习

1. 判断任务集是否可调
2. 实现RM和EDF调度器
3. 证明简单任务集的可调度性

### 前置知识

- 第1集：调度理论基础
