## 📚 教程总览

本教程系列涵盖形式化科学的五大核心领域，从入门概念到高级主题，帮助你系统掌握形式化方法的理论与实践。

| 系列 | 集数 | 难度 | 预估总时长 | 状态 |
|------|------|------|------------|------|
| [01 入门指南](01_getting_started_outline.md) | 10集 | ⭐ 初级 | 5小时 | 计划中 |
| [02 Lean基础](02_lean_basics_outline.md) | 15集 | ⭐⭐ 初中级 | 8小时 | 计划中 |
| [03 类型论](03_type_theory_outline.md) | 12集 | ⭐⭐⭐ 中级 | 7小时 | 计划中 |
| [04 Rust形式化](04_rust_formal_outline.md) | 10集 | ⭐⭐⭐⭐ 中高级 | 6小时 | 计划中 |
| [05 调度器设计](05_scheduler_design_outline.md) | 8集 | ⭐⭐⭐⭐⭐ 高级 | 5小时 | 计划中 |

**总计：55集视频，约31小时学习内容**

## 📖 推荐学习路径

### 🌱 路径一：形式化方法入门（推荐）

适合：所有想要系统学习形式化方法的学员

```
01 入门指南 (10集)
    ↓
02 Lean基础 (15集)
    ↓
03 类型论 (12集)
    ↓
04 Rust形式化 或 05 调度器设计（根据兴趣选择）
```

**预计学习时间**：4-6周（每天1-2小时）

### 🎯 路径二：Lean专项精通

适合：希望深入掌握Lean证明助手的学员

```

    ↓
```

**预计学习时间**：3-4周（每天1-2小时）

### 🔄 路径四：调度器设计与实现

适合：系统程序员，关注并发与调度

```
01 入门指南 - 快速浏览(1-5集)
    ↓
02 Lean基础 (15集)
    ↓
03 类型论 - 重点章节(3-6集)
    ↓
05 调度器设计 (8集)
```

**预计学习时间**：2-3周（每天1-2小时）

## 📋 学习建议

### 🎯 高效学习策略

1. **循序渐进**
   - 不要跳过基础系列，即使你有相关经验
   - 每个系列建议按顺序观看

2. **动手实践**
   - 每集都配有练习题，务必完成
   - 建立自己的"证明笔记本"

3. **社区参与**
   - 加入 [Lean Zulip](https://leanprover.zulipchat.com/)
   - 参与 [Rust Formal Methods](https://rust-formal-methods.github.io/) 讨论

4. **定期复习**
   - 类型论概念需要反复理解
   - 建议制作个人知识卡片

### ⚠️ 常见误区

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| 只看视频不动手 | 边看边写代码 |
| 跳过练习题 | 完成所有配套练习 |
| 追求速度 | 理解深度优先 |
| 孤立学习 | 参与社区讨论 |
| 死记硬背 | 理解核心概念 |

## 🎓 进阶方向

完成本系列教程后，你可以向以下方向深入学习：

### 数学证明

- 使用Lean证明数学定理
- 参与Mathlib项目贡献
- 探索代数、分析、拓扑的形式化

### 程序验证

- 形式化验证实际Rust项目
- 学习分离逻辑（Separation Logic）
- 研究模型检测方法

### 类型系统研究

- 设计新的类型系统
- 实现类型检查器
- 研究线性类型、效应系统

### 系统验证

- 验证操作系统内核
- 形式化验证编译器
- 验证网络协议
