## 使用指南

### 添加新任务

1. 获取新任务 ID（当前最大 + 1）
2. 在"任务记录"部分添加新条目
3. 填写所有必要字段
4. 在"变更历史"中记录文件变更
5. 更新"日志概览"统计

### 更新任务状态

1. 找到对应任务条目
2. 更新"状态"字段
3. 如完成，填写"完成时间"
4. 在"变更历史"中记录

### 记录文件变更

对于每次文件变更，记录：
- 操作类型（创建/修改/删除/重命名）
- 文件路径
- 变更简要说明
