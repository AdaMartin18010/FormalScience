# 04_软件工程

本目录包含软件工程主题的形式化文档，涵盖设计模式、微服务架构、工作流系统和分布式系统。

## 目录结构

```
04_软件工程/
├── 01_设计模式/
│   ├── 01.1_创建型模式.md    # 工厂、构建器、单例、原型
│   ├── 01.2_结构型模式.md    # 适配器、装饰器、代理、组合
│   ├── 01.3_行为型模式.md    # 策略、观察者、命令、状态
│   ├── 01.4_并发模式.md      # 线程池、生产者-消费者、读写锁
│   └── 01.5_分布式模式.md    # 断路器、重试、幂等、Saga
├── 02_微服务架构/
│   ├── 02.1_微服务设计原则.md # 单一职责、松耦合、高内聚
│   ├── 02.2_服务发现与注册.md # Consul、Eureka、服务网格
│   ├── 02.3_API网关.md       # 路由、限流、认证、熔断
│   └── 02.4_可观测性.md      # 日志、指标、追踪、OpenTelemetry
├── 03_工作流系统/
│   ├── 03.1_工作流基础.md    # BPMN、状态机、Petri网
│   ├── 03.2_编排与编排.md    # Orchestration vs Choreography
│   ├── 03.3_工作流引擎.md    # Cadence、Temporal、Airflow
│   └── 03.4_长时间运行流程.md # Saga模式、补偿事务
└── 04_分布式系统/
    ├── 04.1_分布式基础.md    # CAP、一致性模型、时钟
    ├── 04.2_共识算法.md      # Paxos、Raft、PBFT
    ├── 04.3_分布式事务.md    # 2PC、3PC、TCC、Saga
    └── 04.4_数据分区.md      # 分片、复制、一致性哈希
```

## 文档特点

1. **严格序号目录结构**：所有文件采用数字序号前缀，便于排序和引用
2. **形式化定义 + LaTeX**：提供数学形式化定义，便于精确理解
3. **代码示例**：优先使用 Rust/Go 语言，兼顾现代系统编程实践
4. **Mermaid 架构图**：使用 Mermaid 语法绘制架构图，便于版本控制
5. **文档间交叉引用**：提供相关文档链接，形成知识网络
