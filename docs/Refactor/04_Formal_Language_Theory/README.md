## 理论框架

### 乔姆斯基层次结构

**定义 4.1（乔姆斯基层次）**：
形式语言按照生成文法的复杂度分为四个层次：

1. **Type 0 (无限制文法)** → 递归可枚举语言
2. **Type 1 (上下文相关文法)** → 上下文相关语言  
3. **Type 2 (上下文无关文法)** → 上下文无关语言
4. **Type 3 (正则文法)** → 正则语言

**包含关系**：
> **REG ⊂ CFL ⊂ CSL ⊂ REL**
>
> 其中：
>
> - REG: 正则语言类
> - CFL: 上下文无关语言类
> - CSL: 上下文相关语言类
> - REL: 递归可枚举语言类

### 自动机等价性

**定理 4.1（自动机-文法等价性）**：
每种语言类都有对应的识别自动机：

- 正则语言 ↔ 有限自动机
- 上下文无关语言 ↔ 下推自动机
- 上下文相关语言 ↔ 线性有界自动机
- 递归可枚举语言 ↔ 图灵机

### 计算复杂性

**定义 4.2（语言的复杂性）**：
语言的复杂性由识别该语言所需的计算资源决定：

- **时间复杂性**: 识别字符串所需的步数
- **空间复杂性**: 识别字符串所需的存储空间
- **描述复杂性**: 描述语言所需的文法大小

## 核心概念

### 形式语言

**定义 4.3（形式语言）**：
设Σ是有限字母表，则Σ上的形式语言L是Σ*的子集。

**基本运算**：

- **并集**: L₁ ∪ L₂
- **连接**: L₁ · L₂ = {xy | x ∈ L₁, y ∈ L₂}
- **克林闭包**: L* = ∪ᵢ≥₀ Lⁱ

### 正则表达式

**定义 4.4（正则表达式）**：
Σ上的正则表达式递归定义为：

1. ∅、ε、a (a ∈ Σ) 是正则表达式
2. 若r、s是正则表达式，则(r|s)、(rs)、r*也是正则表达式

**Kleene定理**：正则表达式与有限自动机等价。

### 上下文无关文法

**定义 4.5（上下文无关文法）**：
上下文无关文法G = (V, Σ, P, S)，其中：

- V: 非终结符集合
- Σ: 终结符集合
- P: 产生式规则集合 (A → α形式)
- S: 开始符号

**推导关系**：

- **直接推导**: α ⇒ β
- **推导**: α ⇒* β
- **最左推导**: α ⇒ₗₘ* β
- **最右推导**: α ⇒ᵣₘ* β
