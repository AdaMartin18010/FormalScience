## 1. 定义与背景

上下文无关语言（CFL）是下推自动机（PDA）能够识别的语言类，广泛应用于编译原理、语法分析等领域。

---

## 2. 批判性分析

- 能表达嵌套结构，但无法描述所有上下文相关依赖
- 理论基础扎实，工程实现复杂度适中
