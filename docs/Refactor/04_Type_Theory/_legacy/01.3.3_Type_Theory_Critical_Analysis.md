## 哲学维度

- 本体论：类型对象的存在条件与空间化（HoTT）
- 认识论：证明即程序的可执行性与知识的可检验性
- 价值论：形式化的社会价值与可能的技术异化

## 方法论维度

- 等式观与工程后果：内涵/外延/高维路径  
- 资源与规范性：线性/仿射的实践边界  
- 可判定性与自动化：内核可信基与工具治理

## 社会技术维度

- 门槛与不平等：学习曲线、英语/工具偏见  
- 公共基建：标准/库/基准与可持续维护  
- 伦理与合规：安全关键、隐私、公平与问责
