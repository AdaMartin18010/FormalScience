## 类型理论形式化语义与多表征方式

### 简单类型理论（Simple Type Theory）

**形式化语义：**

- 以类型环境Γ和类型判断Γ ⊢ e : τ形式化类型系统
- 基本类型：Bool, Int, Float, String, Unit
- 构造类型：函数类型τ₁ → τ₂、积类型τ₁ × τ₂、和类型τ₁ + τ₂

**多表征方式：**

- 类型推导树
- 类型规则表
- 类型检查算法伪代码
- 类型系统可视化图

### 依赖类型理论（Dependent Type Theory）

**形式化语义：**

- 以Π类型Π(x:A).B和Σ类型Σ(x:A).B形式化依赖类型
- 类型可依赖于值，类型系统表达更丰富的性质
- 形式化为λΠ演算、Martin-Löf类型理论等

**多表征方式：**

- 依赖类型表达式
- 类型检查推理树
- 证明助理交互示例
- 依赖类型可视化

### 线性类型理论（Linear Type Theory）

**形式化语义：**

- 以线性类型!A、⊗、⊸等形式化资源管理
- 线性逻辑与类型系统的结合
- 形式化为线性λ演算、线性逻辑等

**多表征方式：**

- 线性类型规则表
- 资源使用图
- 线性逻辑证明树
- 线性类型系统可视化

### 同伦类型论（Homotopy Type Theory）

**形式化语义：**

- 以路径类型Path A a b、同伦类型等表达拓扑性质
- 类型作为空间，项作为点，类型相等作为路径
- 形式化为同伦理论、高阶范畴论等

**多表征方式：**

- 同伦类型表达式
- 路径空间可视化
- 同伦证明树
- 拓扑结构图

### Curry-Howard对应（Curry-Howard Correspondence）

**形式化语义：**

- 以类型作为命题，项作为证明
- 函数类型对应蕴含，积类型对应合取，和类型对应析取
- 形式化为证明论、直觉主义逻辑等

**多表征方式：**

- 逻辑-类型对应表
- 证明-程序对应图
- 逻辑推理树
- 类型推导示例
