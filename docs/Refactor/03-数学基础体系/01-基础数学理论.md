## 2. 集合论基础

### 2.1 基本概念

**定义 2.1.1 (集合)**
集合是满足外延公理的对象：

$$\text{Set}(A) \equiv \forall x, y (x \in A \land y \in A \land x = y \rightarrow x = y)$$

**公理 2.1.1 (外延公理)**
$$\forall A, B (\forall x (x \in A \leftrightarrow x \in B) \rightarrow A = B)$$

**公理 2.1.2 (空集公理)**
$$\exists A \forall x (x \notin A)$$

**定义 2.1.2 (空集)**
空集 $\emptyset$ 是唯一不包含任何元素的集合：

$$\emptyset = \{x \mid x \neq x\}$$

**公理 2.1.3 (配对公理)**
$$\forall a, b \exists A \forall x (x \in A \leftrightarrow x = a \lor x = b)$$

**定义 2.1.3 (无序对)**
集合 $\{a, b\}$ 定义为包含 $a$ 和 $b$ 的集合。

### 2.2 集合运算

**定义 2.2.1 (并集)**
$$A \cup B = \{x \mid x \in A \lor x \in B\}$$

**定义 2.2.2 (交集)**
$$A \cap B = \{x \mid x \in A \land x \in B\}$$

**定义 2.2.3 (差集)**
$$A \setminus B = \{x \mid x \in A \land x \notin B\}$$

**定义 2.2.4 (补集)**
$$A^c = \{x \mid x \notin A\}$$

**定理 2.2.1 (德摩根律)**
$$(A \cup B)^c = A^c \cap B^c$$
$$(A \cap B)^c = A^c \cup B^c$$

**证明**: 通过集合的定义和逻辑等价性证明。

### 2.3 幂集与笛卡尔积

**公理 2.3.1 (幂集公理)**
$$\forall A \exists P \forall B (B \in P \leftrightarrow B \subseteq A)$$

**定义 2.3.1 (幂集)**
集合 $A$ 的幂集 $\mathcal{P}(A)$ 定义为：

$$\mathcal{P}(A) = \{B \mid B \subseteq A\}$$

**定义 2.3.2 (笛卡尔积)**
集合 $A$ 和 $B$ 的笛卡尔积 $A \times B$ 定义为：

$$A \times B = \{(a, b) \mid a \in A \land b \in B\}$$

**定理 2.3.1 (幂集基数)**
如果 $|A| = n$，则 $|\mathcal{P}(A)| = 2^n$。

**证明**: 通过数学归纳法证明。

## 3. 数系理论

### 3.1 自然数

**公理 3.1.1 (皮亚诺公理)**
自然数集 $\mathbb{N}$ 满足以下公理：

1. **零公理**: $0 \in \mathbb{N}$
2. **后继公理**: $\forall n \in \mathbb{N} (S(n) \in \mathbb{N})$
3. **零不是后继**: $\forall n \in \mathbb{N} (S(n) \neq 0)$
4. **后继唯一性**: $\forall m, n \in \mathbb{N} (S(m) = S(n) \rightarrow m = n)$
5. **数学归纳**: $\forall P (P(0) \land \forall n (P(n) \rightarrow P(S(n))) \rightarrow \forall n P(n))$

-**定义 3.1.1 (自然数运算)**

- **加法**: $m + 0 = m$, $m + S(n) = S(m + n)$
- **乘法**: $m \cdot 0 = 0$, $m \cdot S(n) = m \cdot n + m$

-**定理 3.1.1 (自然数性质)**

1. 加法结合律: $(a + b) + c = a + (b + c)$
2. 加法交换律: $a + b = b + a$
3. 乘法结合律: $(a \cdot b) \cdot c = a \cdot (b \cdot c)$
4. 乘法交换律: $a \cdot b = b \cdot a$
5. 分配律: $a \cdot (b + c) = a \cdot b + a \cdot c$

### 3.2 整数

**定义 3.2.1 (整数)**
整数集 $\mathbb{Z}$ 定义为：

$$\mathbb{Z} = \mathbb{N} \times \mathbb{N} / \sim$$

其中等价关系 $\sim$ 定义为：

$$(a, b) \sim (c, d) \leftrightarrow a + d = b + c$$

-**定义 3.2.2 (整数运算)**

- **加法**: $[(a, b)] + [(c, d)] = [(a + c, b + d)]$
- **乘法**: $[(a, b)] \cdot [(c, d)] = [(a \cdot c + b \cdot d, a \cdot d + b \cdot c)]$

**定理 3.2.1 (整数环)**
$(\mathbb{Z}, +, \cdot)$ 构成一个环。

### 3.3 有理数

**定义 3.3.1 (有理数)**
有理数集 $\mathbb{Q}$ 定义为：

$$\mathbb{Q} = \mathbb{Z} \times (\mathbb{Z} \setminus \{0\}) / \sim$$

其中等价关系 $\sim$ 定义为：

$$(a, b) \sim (c, d) \leftrightarrow a \cdot d = b \cdot c$$

-**定义 3.3.2 (有理数运算)**

- **加法**: $\frac{a}{b} + \frac{c}{d} = \frac{a \cdot d + b \cdot c}{b \cdot d}$
- **乘法**: $\frac{a}{b} \cdot \frac{c}{d} = \frac{a \cdot c}{b \cdot d}$

**定理 3.3.1 (有理数域)**
$(\mathbb{Q}, +, \cdot)$ 构成一个域。

### 3.4 实数

**定义 3.4.1 (戴德金分割)**
实数通过戴德金分割定义：

$$\mathbb{R} = \{(A, B) \mid A, B \subseteq \mathbb{Q}, A \cup B = \mathbb{Q}, A \cap B = \emptyset, \forall a \in A, b \in B (a < b)\}$$

-**定义 3.4.2 (实数运算)**

- **加法**: $(A, B) + (C, D) = (\{a + c \mid a \in A, c \in C\}, \mathbb{Q} \setminus \{a + c \mid a \in A, c \in C\})$
- **乘法**: 类似定义

**定理 3.4.1 (实数完备性)**
实数集是完备的，即每个有上界的非空子集都有最小上界。

### 3.5 复数

**定义 3.5.1 (复数)**
复数集 $\mathbb{C}$ 定义为：

$$\mathbb{C} = \mathbb{R} \times \mathbb{R}$$

-**定义 3.5.2 (复数运算)**

- **加法**: $(a, b) + (c, d) = (a + c, b + d)$
- **乘法**: $(a, b) \cdot (c, d) = (a \cdot c - b \cdot d, a \cdot d + b \cdot c)$

**定理 3.5.1 (代数基本定理)**
每个非常数复系数多项式都有复根。

## 4. 关系理论

### 4.1 基本概念

**定义 4.1.1 (二元关系)**
集合 $A$ 和 $B$ 之间的二元关系 $R$ 是 $A \times B$ 的子集：

$$R \subseteq A \times B$$

**定义 4.1.2 (关系性质)**
关系 $R \subseteq A \times A$ 的性质：

- **自反性**: $\forall x \in A (x R x)$
- **对称性**: $\forall x, y \in A (x R y \rightarrow y R x)$
- **传递性**: $\forall x, y, z \in A (x R y \land y R z \rightarrow x R z)$
- **反对称性**: $\forall x, y \in A (x R y \land y R x \rightarrow x = y)$

### 4.2 等价关系

**定义 4.2.1 (等价关系)**
等价关系是自反、对称、传递的关系：

$$\text{EquivalenceRelation}(R) \equiv \text{Reflexive}(R) \land \text{Symmetric}(R) \land \text{Transitive}(R)$$

**定义 4.2.2 (等价类)**
元素 $a$ 的等价类 $[a]_R$ 定义为：

$$[a]_R = \{x \in A \mid x R a\}$$

**定义 4.2.3 (商集)**
关系 $R$ 的商集 $A/R$ 定义为：

$$A/R = \{[a]_R \mid a \in A\}$$

-**定理 4.2.1 (等价类性质)**

1. $a \in [a]_R$
2. $[a]_R = [b]_R \leftrightarrow a R b$
3. $[a]_R \cap [b]_R = \emptyset \lor [a]_R = [b]_R$

### 4.3 序关系

**定义 4.3.1 (偏序)**
偏序是自反、反对称、传递的关系：

$$\text{PartialOrder}(R) \equiv \text{Reflexive}(R) \land \text{Antisymmetric}(R) \land \text{Transitive}(R)$$

**定义 4.3.2 (全序)**
全序是偏序且满足完全性：

$$\text{TotalOrder}(R) \equiv \text{PartialOrder}(R) \land \forall x, y (x R y \lor y R x)$$

**定义 4.3.3 (良序)**
良序是全序且每个非空子集都有最小元素：

$$\text{WellOrder}(R) \equiv \text{TotalOrder}(R) \land \forall S \subseteq A (S \neq \emptyset \rightarrow \exists m \in S \forall s \in S (m R s))$$

## 5. 函数理论

### 5.1 基本概念

**定义 5.1.1 (函数)**
函数 $f: A \rightarrow B$ 是满足以下条件的二元关系：

$$\forall x \in A \exists! y \in B ((x, y) \in f)$$

**定义 5.1.2 (函数性质)**
函数 $f: A \rightarrow B$ 的性质：

- **单射**: $\forall x_1, x_2 \in A (f(x_1) = f(x_2) \rightarrow x_1 = x_2)$
- **满射**: $\forall y \in B \exists x \in A (f(x) = y)$
- **双射**: 单射且满射

**定义 5.1.3 (函数运算)**

- **复合**: $(g \circ f)(x) = g(f(x))$
- **逆函数**: $f^{-1}(y) = x \leftrightarrow f(x) = y$

### 5.2 函数类型

**定义 5.2.1 (常函数)**
常函数 $c_a: A \rightarrow B$ 定义为：

$$c_a(x) = a$$

**定义 5.2.2 (恒等函数)**
恒等函数 $\text{id}_A: A \rightarrow A$ 定义为：

$$\text{id}_A(x) = x$$

**定义 5.2.3 (特征函数)**
集合 $A$ 的特征函数 $\chi_A: X \rightarrow \{0, 1\}$ 定义为：

$$\chi_A(x) = \begin{cases} 1 & \text{if } x \in A \\ 0 & \text{if } x \notin A \end{cases}$$

### 5.3 函数空间

**定义 5.3.1 (函数空间)**
从 $A$ 到 $B$ 的函数空间 $B^A$ 定义为：

$$B^A = \{f \mid f: A \rightarrow B\}$$

**定理 5.3.1 (函数空间基数)**
如果 $|A| = m$ 且 $|B| = n$，则 $|B^A| = n^m$。

**证明**: 通过乘法原理证明。

## 6. 基数与序数

### 6.1 基数

**定义 6.1.1 (基数)**
集合 $A$ 的基数 $|A|$ 是 $A$ 的等势类的代表。

**定义 6.1.2 (基数比较)**
$|A| \leq |B|$ 当且仅当存在单射 $f: A \rightarrow B$。

-**定义 6.1.3 (基数运算)**

- **加法**: $|A| + |B| = |A \sqcup B|$
- **乘法**: $|A| \cdot |B| = |A \times B|$
- **幂**: $|A|^{|B|} = |A^B|$

**定理 6.1.1 (康托尔定理)**
对于任何集合 $A$，$|A| < |\mathcal{P}(A)|$。

**证明**: 通过对角线论证证明。

### 6.2 序数

**定义 6.2.1 (序数)**
序数是传递的良序集合：

$$\text{Ordinal}(\alpha) \equiv \text{Transitive}(\alpha) \land \text{WellOrdered}(\alpha, \in)$$

**定义 6.2.2 (序数运算)**

- **后继**: $S(\alpha) = \alpha \cup \{\alpha\}$
- **极限**: $\sup A = \bigcup A$

**定理 6.2.1 (序数性质)**

1. 每个良序集都同构于唯一序数
2. 序数类是真类
3. 序数在 $\in$ 下良序

## 7. 公理化系统

### 7.1 策梅洛-弗兰克尔集合论

**公理 7.1.1 (ZFC公理)**

1. **外延公理**: $\forall A, B (\forall x (x \in A \leftrightarrow x \in B) \rightarrow A = B)$
2. **空集公理**: $\exists A \forall x (x \notin A)$
3. **配对公理**: $\forall a, b \exists A \forall x (x \in A \leftrightarrow x = a \lor x = b)$
4. **并集公理**: $\forall A \exists B \forall x (x \in B \leftrightarrow \exists C (C \in A \land x \in C))$
5. **幂集公理**: $\forall A \exists P \forall B (B \in P \leftrightarrow B \subseteq A)$
6. **无穷公理**: $\exists A (\emptyset \in A \land \forall x (x \in A \rightarrow x \cup \{x\} \in A))$
7. **替换公理**: $\forall A \forall F (\text{Function}(F) \rightarrow \exists B \forall y (y \in B \leftrightarrow \exists x (x \in A \land y = F(x))))$
8. **正则公理**: $\forall A (A \neq \emptyset \rightarrow \exists x (x \in A \land x \cap A = \emptyset))$
9. **选择公理**: $\forall A (\emptyset \notin A \rightarrow \exists f \forall B (B \in A \rightarrow f(B) \in B))$

### 7.2 范畴论公理

**定义 7.2.1 (范畴)**
范畴 $\mathcal{C}$ 包含：

1. **对象类**: $\text{Ob}(\mathcal{C})$
2. **态射类**: $\text{Mor}(\mathcal{C})$
3. **复合运算**: $\circ: \text{Mor}(B, C) \times \text{Mor}(A, B) \rightarrow \text{Mor}(A, C)$
4. **恒等态射**: $\text{id}_A: A \rightarrow A$

**公理 7.2.1 (范畴公理)**

1. **结合律**: $(f \circ g) \circ h = f \circ (g \circ h)$
2. **单位律**: $f \circ \text{id}_A = f = \text{id}_B \circ f$

## 8. 构造性数学

### 8.1 直觉主义逻辑

**公理 8.1.1 (直觉主义公理)**

1. $\phi \rightarrow (\psi \rightarrow \phi)$
2. $(\phi \rightarrow (\psi \rightarrow \chi)) \rightarrow ((\phi \rightarrow \psi) \rightarrow (\phi \rightarrow \chi))$
3. $\phi \land \psi \rightarrow \phi$
4. $\phi \land \psi \rightarrow \psi$
5. $\phi \rightarrow (\psi \rightarrow \phi \land \psi)$
6. $\phi \rightarrow \phi \lor \psi$
7. $\psi \rightarrow \phi \lor \psi$
8. $(\phi \rightarrow \chi) \rightarrow ((\psi \rightarrow \chi) \rightarrow (\phi \lor \psi \rightarrow \chi))$
9. $\bot \rightarrow \phi$

### 8.2 构造性集合论

**公理 8.2.1 (构造性公理)**

## 9. 应用与扩展

### 9.1 计算机科学应用

**应用 9.1.1 (类型理论)**
使用集合论构建类型理论：

```typescript
// 类型定义
interface Type {
  kind: 'unit' | 'bool' | 'nat' | 'function' | 'product' | 'sum';
  domain?: Type;
  codomain?: Type;
}

// 类型检查
function typeCheck(expr: Expr, context: Context): Type {
  switch (expr.kind) {
    case 'lambda':
      return {
        kind: 'function',
        domain: expr.paramType,
        codomain: typeCheck(expr.body, context.extend(expr.param, expr.paramType))
      };
    case 'application':
      const funcType = typeCheck(expr.function, context);
      const argType = typeCheck(expr.argument, context);
      if (funcType.domain && typesEqual(funcType.domain, argType)) {
        return funcType.codomain;
      }
      throw new TypeError('Type mismatch');
  }
}
```

```coq

Inductive nat : Type :=
  | O : nat
  | S : nat -> nat.

(* 加法定义 *)
Fixpoint plus (n m : nat) : nat :=
  match n with
  | O => m
  | S p => S (plus p m)
  end.

(* 加法结合律 *)
Theorem plus_assoc : forall n m p : nat,
  plus n (plus m p) = plus (plus n m) p.
Proof.
  intros n m p.
  induction n as [| n IHn].
  - reflexivity.
  - simpl. rewrite IHn. reflexivity.
Qed.
```
