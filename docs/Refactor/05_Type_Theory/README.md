## 主题分类与说明

-   **简单类型论**: 类型论的基础，引入了类型作为对λ演算项的分类。
-   **依赖类型论**: 类型可以依赖于值，极大地增强了表达能力，是现代证明助理的基础。
-   **同伦类型论 (HoTT)**: 将类型论与同伦论联系起来，为数学基础提供了新的视角。
-   **线性类型论**: 精确控制资源的使用，不允许随意复制或丢弃变量。
-   **Curry-Howard同构**: 在逻辑和计算之间建立了深刻的对应关系，即"命题即类型，证明即程序"。
