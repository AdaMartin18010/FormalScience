## 如何成为支持者

### 资金支持

您可以通过以下方式提供资金支持：

1. **GitHub Sponsors** - 通过 GitHub 平台赞助
2. **Open Collective** - 通过 Open Collective 平台支持
3. **直接捐赠** - 通过其他方式联系项目团队

### 非资金支持

除了资金，您还可以通过以下方式支持项目：

1. **贡献代码** - 提交 Pull Request
2. **完善文档** - 帮助改进文档
3. **推广项目** - 在社交媒体和技术社区分享
4. **提供反馈** - 报告 bug 或提出建议
5. **参与测试** - 帮助测试新版本
