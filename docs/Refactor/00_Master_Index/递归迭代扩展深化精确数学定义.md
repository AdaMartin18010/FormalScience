```text

A1.1.2: 传递性 - ∀s₁,s₂,s₃ ∈ S. s₁ ≤ s₂ ∧ s₂ ≤ s₃ → s₁ ≤ s₃
A1.1.3: 反对称性 - ∀s₁,s₂ ∈ S. s₁ ≤ s₂ ∧ s₂ ≤ s₁ → s₁ = s₂
A1.1.4: 组合结合律 - ∀s₁,s₂,s₃ ∈ S. (s₁ ⊗ s₂) ⊗ s₃ = s₁ ⊗ (s₂ ⊗ s₃)

A1.1.10: 分配律 - ∀s₁,s₂,s₃ ∈ S. s₁ ⊗ (s₂ ⊕ s₃) = (s₁ ⊗ s₂) ⊕ (s₁ ⊗ s₃)
```
