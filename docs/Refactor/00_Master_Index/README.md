## 重构原则

1. **严格序号树形结构** - 所有目录和文件按序号组织
2. **内容相关性** - 按主题相关性组织内容
3. **本地跳转** - 建立完整的本地引用体系
4. **形式化规范** - 符合数学和学术规范
5. **多表征方式** - 包含图表、数学符号、代码示例
6. **一致性保证** - 内容、证明、相关性、语义一致性
