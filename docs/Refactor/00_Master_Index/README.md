## 📝 文档规范

### 1. 文件命名规范
- 使用英文命名，采用下划线分隔
- 格式：`序号_主题_子主题.md`
- 示例：`01_Set_Basics.md`, `02_Set_Operations.md`

### 2. 内容结构规范
每个文档必须包含：
- 文档标题和概述
- 严格的序号目录
- 形式化定义
- 定理和证明
- 代码示例（Rust/Haskell）
- 应用实例
- 参考文献

### 3. 数学表示规范
- 使用LaTeX数学公式
- 定理编号：`定理 章节.序号`
- 定义编号：`定义 章节.序号`
- 算法编号：`算法 章节.序号`

### 4. 代码规范
- 优先使用Rust，其次Haskell
- 包含完整的类型定义
- 提供详细的注释
- 包含测试用例
