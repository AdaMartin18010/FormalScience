**基本概念**:

- **集合** - 基本数学对象
- **元素** - 集合的成员
- **子集** - 集合的包含关系
- **幂集** - 所有子集的集合

**集合运算**:

- **并集** - $A \cup B = \{x : x \in A \text{ or } x \in B\}$
- **交集** - $A \cap B = \{x : x \in A \text{ and } x \in B\}$
- **差集** - $A \setminus B = \{x : x \in A \text{ and } x \notin B\}$
- **补集** - $A^c = \{x : x \notin A\}$

**关系与函数**:

- **关系** - 集合间的对应关系
- **函数** - 单值对应关系
