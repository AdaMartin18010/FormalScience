## 主题说明

本模块探讨将高级语言编写的源代码，转换为计算机可以执行的机器代码或等价表示的过程。这主要通过两种方式实现：编译和解释。本模块将深入探讨这两种技术的核心理论、权衡以及支撑它们运行的关键技术，如虚拟机和内存管理。
