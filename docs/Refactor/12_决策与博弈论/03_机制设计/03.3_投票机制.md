# 12.3.3 投票机制

## 12.3.3.1 引言

**社会选择理论**研究如何从个体偏好聚合为集体决策。**投票机制**（voting mechanisms）是社会选择的核心工具，用于在多个候选方案或候选人中做出集体选择。本章介绍投票规则的基本形式、策略证明性（strategy-proofness）以及不可能性结果。

---

## 12.3.3.2 偏好与投票规则

### 12.3.3.2.1 偏好表示

**定义 12.3.3.1**（偏好排序）

设 $A = \{a_1, ..., a_m\}$ 为候选方案集。参与人 $i$ 的**偏好**是 $A$ 上的完备、传递、反对称的二元关系 $\succ_i$。

**定义 12.3.3.2**（偏好剖面）

**偏好剖面**（preference profile）$\succ = (\succ_1, ..., \succ_n)$ 是所有参与人的偏好组合。

### 12.3.3.2.2 投票规则

**定义 12.3.3.3**（投票规则）

**投票规则**（或社会选择函数）$f: \mathcal{L}^n \to A$ 将偏好剖面映射到获胜方案，其中 $\mathcal{L}$ 是 $A$ 上的线性序集合。

**常见投票规则**：

| 规则 | 描述 |
|------|------|
| **多数制** | 得票最多的候选人获胜（可能需要多轮） |
| **Borda 计数** | 排名为 $k$ 的候选人得 $m-k$ 分，总分最高者获胜 |
| **Condorcet 方法** | 若存在与其他所有候选人一对一都获胜的候选人，则该人获胜 |
| **instant-runoff** | 淘汰得票最少者，重新分配选票，直至有人获多数 |

---

## 12.3.3.3 Condorcet 悖论

### 12.3.3.3.1 Condorcet 赢家

**定义 12.3.3.4**（Condorcet 赢家）

候选人 $a$ 是 **Condorcet 赢家**，如果在所有两两比较中 $a$ 都获得多数支持：

$$|\{i: a \succ_i b\}| > \frac{n}{2}, \quad \forall b \neq a$$

### 12.3.3.3.2 Condorcet 循环

**Condorcet 悖论**：即使个体偏好传递，多数投票可能导致循环社会偏好。

**示例**（3 人 3 方案）：

| 参与人 | 偏好 |
|--------|------|
| 1 | $a \succ b \succ c$ |
| 2 | $b \succ c \succ a$ |
| 3 | $c \succ a \succ b$ |

两两比较：

- $a$ vs $b$: $a$ 胜（参与人 1, 3）
- $b$ vs $c$: $b$ 胜（参与人 1, 2）
- $c$ vs $a$: $c$ 胜（参与人 2, 3）

**循环**：$a \succ b \succ c \succ a$

---

## 12.3.3.4 策略证明性

### 12.3.3.4.1 定义

**定义 12.3.3.5**（策略证明性）

投票规则 $f$ 是**策略证明的**（strategy-proof），如果真实报告偏好是占优策略：

$$f(\succ_i, \succ_{-i}) \succ_i f(\succ_i', \succ_{-i}), \quad \forall \succ_i', \succ_{-i}$$

等价表述：没有参与人可以通过虚报偏好获得更优结果。

### 12.3.3.4.2 策略证明性的检验

**定理 12.3.3.1**

多数规则不是策略证明的。

**证明示例**：

设偏好剖面：

- 参与人 1：$a \succ b \succ c$
- 参与人 2：$b \succ a \succ c$
- 参与人 3：$c \succ b \succ a$

若参与人 1 真实投票，$b$ 可能获胜。但若参与人 1 虚报 $c \succ b \succ a$，可能改变结果使其更偏好。

---

## 12.3.3.5 Gibbard-Satterthwaite 定理

### 12.3.3.5.1 定理陈述

**定理 12.3.3.2**（Gibbard, 1973; Satterthwaite, 1975）

设 $|A| \geq 3$，$f$ 是满射（onto）且非独裁的投票规则。则 $f$ 不是策略证明的。

**等价表述**：在至少 3 个候选方案时，任何非独裁的选择规则都容易受到策略操纵。

### 12.3.3.5.2 证明概要

**证明思路**（基于 Arrow 不可能定理）：

1. 策略证明性蕴含单调性（monotonicity）
2. 单调性 + 满射 $\Rightarrow$ Pareto 效率
3. 构造社会福利函数满足 Arrow 条件
4. 由 Arrow 定理，必为独裁
5. 因此原投票规则必为独裁

## 12.3.3.6 单峰偏好与 Median 投票者定理

### 12.3.3.6.1 单峰偏好

**定义 12.3.3.6**（单峰偏好）

偏好 $\succ_i$ 在候选方案线性序 $<$ 上是**单峰的**（single-peaked），如果存在最偏好点 $p_i$（峰值），满足：

$$a < b < p_i \text{ 或 } p_i < b < a \Rightarrow b \succ_i a$$

**解释**：偏好沿序"山峰状"，远离峰值效用递减。

### 12.3.3.6.2 Median 投票者定理

**定理 12.3.3.3**（Median Voter Theorem, Black, 1948）

若所有偏好都是单峰的，则 Condorcet 赢家存在且等于 median 峰值。

**证明**：

设 $p^*$ 是 median 峰值（半数峰值在其左，半数在其右）。

对任意其他候选 $a < p^*$：

- 峰值 $\geq p^*$ 的参与人（至少半数）偏好 $p^* \succ a$
- 因此 $p^*$ 在两两比较中击败 $a$

同理对 $a > p^*$。∎

## 12.3.3.7 Python 实现：投票机制分析

```python
"""
投票机制与社会选择理论实现
包含投票规则、Condorcet 悖论、策略证明性分析
基于 Gibbard (1973) 和 Satterthwaite (1975)
"""

from typing import List, Dict, Tuple, Callable, Optional
from dataclasses import dataclass
from collections import defaultdict
import numpy as np
from itertools import combinations, permutations


@dataclass
class Preference:
    """偏好排序"""
    ranking: List[str]  # 从最优到最差

    def prefers(self, a: str, b: str) -> bool:
        """检查是否偏好 a 胜过 b"""
        return self.ranking.index(a) < self.ranking.index(b)

    def rank_of(self, a: str) -> int:
        """获取 a 的排名（0 = 最优）"""
        return self.ranking.index(a)

    def top_choice(self) -> str:
        """最优选择"""
        return self.ranking[0]


class VotingRule:
    """投票规则基类"""

    def __init__(self, candidates: List[str]):
        self.candidates = candidates
        self.m = len(candidates)

    def winner(self, preferences: List[Preference]) -> str:
        """给定偏好剖面，返回获胜者"""
        raise NotImplementedError

    def is_strategy_proof(self, n_voters: int, test_cases: int = 100) -> Tuple[bool, str]:
        """
        检验策略证明性
        通过随机测试检查是否存在操纵可能
        """
        for _ in range(test_cases):
            # 生成随机偏好剖面
            prefs = []
            for _ in range(n_voters):
                ranking = np.random.permutation(self.candidates).tolist()
                prefs.append(Preference(ranking))

            # 检查每个参与人是否有利可图的偏离
            for i in range(n_voters):
                true_pref = prefs[i]
                true_winner = self.winner(prefs)

                # 生成所有可能的虚报偏好
                for fake_ranking in permutations(self.candidates):
                    if list(fake_ranking) == true_pref.ranking:
                        continue

                    fake_prefs = prefs.copy()
                    fake_prefs[i] = Preference(list(fake_ranking))
                    fake_winner = self.winner(fake_prefs)

                    # 检查是否获得更优结果
                    if true_pref.prefers(fake_winner, true_winner):
                        return False, f"参与人 {i} 可以操纵: {true_winner} -> {fake_winner}"

        return True, "未发现操纵可能（在测试样本中）"


class PluralityVoting(VotingRule):
    """
    多数制投票（Plurality）
    每人投一票给最偏好候选人，得票最多者获胜
    """

    def winner(self, preferences: List[Preference]) -> str:
        votes = defaultdict(int)
        for pref in preferences:
            votes[pref.top_choice()] += 1

        return max(votes.items(), key=lambda x: x[1])[0]


class BordaCount(VotingRule):
    """
    Borda 计数
    排名第 k 的候选人得 (m-k) 分
    """

    def winner(self, preferences: List[Preference]) -> str:
        scores = defaultdict(int)

        for pref in preferences:
            for candidate in self.candidates:
                rank = pref.rank_of(candidate)
                scores[candidate] += self.m - 1 - rank

        return max(scores.items(), key=lambda x: x[1])[0]


class CondorcetMethod(VotingRule):
    """
    Condorcet 方法
    若存在 Condorcet 赢家，返回该候选人
    否则返回 None（或按其他规则打破平局）
    """

    def pairwise_comparison(self, preferences: List[Preference]) -> Dict[Tuple[str, str], int]:
        """
        两两比较
        返回: {(a, b): 偏好 a 胜过 b 的票数}
        """
        wins = defaultdict(int)

        for a, b in combinations(self.candidates, 2):
            for pref in preferences:
                if pref.prefers(a, b):
                    wins[(a, b)] += 1
                else:
                    wins[(b, a)] += 1

        return wins

    def find_condorcet_winner(self, preferences: List[Preference]) -> Optional[str]:
        """寻找 Condorcet 赢家"""
        n = len(preferences)
        wins = self.pairwise_comparison(preferences)

        for a in self.candidates:
            is_condorcet = True
            for b in self.candidates:
                if a == b:
                    continue
                if wins[(a, b)] <= n / 2:
                    is_condorcet = False
                    break

            if is_condorcet:
                return a

        return None

    def winner(self, preferences: List[Preference]) -> str:
        cw = self.find_condorcet_winner(preferences)
        if cw:
            return cw

        # 没有 Condorcet 赢家时，返回 Borda 赢家作为备选
        borda = BordaCount(self.candidates)
        return borda.winner(preferences)


class InstantRunoff(VotingRule):
    """
    Instant-Runoff Voting (IRV) / 排序复选制
    淘汰得票最少者，重新分配选票，直至有人获多数
    """

    def winner(self, preferences: List[Preference]) -> str:
        remaining = set(self.candidates)
        prefs = [Preference(pref.ranking.copy()) for pref in preferences]

        while len(remaining) > 1:
            # 计算当前轮次的首选项票数
            first_choice_votes = defaultdict(int)
            for pref in prefs:
                for candidate in pref.ranking:
                    if candidate in remaining:
                        first_choice_votes[candidate] += 1
                        break

            # 检查是否有候选人获多数
            total_votes = sum(first_choice_votes.values())
            for candidate, votes in first_choice_votes.items():
                if votes > total_votes / 2:
                    return candidate

            # 淘汰得票最少者
            loser = min(first_choice_votes.items(), key=lambda x: x[1])[0]
            remaining.remove(loser)

        return remaining.pop()


class SinglePeakedPreference:
    """单峰偏好相关工具"""

    @staticmethod
    def generate_single_peaked(candidates: List[str], peak: str) -> Preference:
        """
        生成以 peak 为峰值的单峰偏好
        假设候选人按原始列表顺序排列在一条线上
        """
        peak_idx = candidates.index(peak)

        # 按距离峰值的远近排序
        ranking = sorted(candidates, key=lambda c: abs(candidates.index(c) - peak_idx))

        return Preference(ranking)

    @staticmethod
    def median_voter_winner(preferences: List[Preference],
                            candidate_order: List[str]) -> str:
        """
        Median 投票者定理
        返回 median 峰值
        """
        peaks = []
        for pref in preferences:
            # 找到峰值（最偏好）
            peaks.append(pref.top_choice())

        # 按候选人在线上的位置排序峰值
        peak_positions = [(p, candidate_order.index(p)) for p in peaks]
        peak_positions.sort(key=lambda x: x[1])

        # 返回 median
        median_idx = len(peak_positions) // 2
        return peak_positions[median_idx][0]


def demonstrate_condorcet_paradox():
    """演示 Condorcet 悖论"""

    print("=" * 70)
    print("Condorcet 悖论演示")
    print("=" * 70)

    candidates = ["a", "b", "c"]

    # 产生循环的偏好剖面
    preferences = [
        Preference(["a", "b", "c"]),  # 参与人 1
        Preference(["b", "c", "a"]),  # 参与人 2
        Preference(["c", "a", "b"]),  # 参与人 3
    ]

    print("\n偏好剖面:")
    for i, pref in enumerate(preferences):
        print(f"  参与人 {i+1}: {' > '.join(pref.ranking)}")

    # 两两比较
    condorcet = CondorcetMethod(candidates)
    wins = condorcet.pairwise_comparison(preferences)

    print("\n两两比较结果:")
    n = len(preferences)
    for a, b in combinations(candidates, 2):
        a_wins = wins[(a, b)]
        b_wins = wins[(b, a)]
        winner = a if a_wins > b_wins else b
        print(f"  {a} vs {b}: {winner} 胜 ({max(a_wins, b_wins)}/{n})")

    print("\n结果: a > b > c > a (循环！)")
    print(f"Condorcet 赢家存在? {condorcet.find_condorcet_winner(preferences) is not None}")


def demonstrate_voting_rules():
    """演示不同投票规则"""

    print("\n" + "=" * 70)
    print("不同投票规则比较")
    print("=" * 70)

    candidates = ["A", "B", "C"]

    # 偏好剖面
    preferences = [
        Preference(["A", "B", "C"]),
        Preference(["A", "B", "C"]),
        Preference(["B", "A", "C"]),
        Preference(["B", "C", "A"]),
        Preference(["C", "B", "A"]),
    ]

    print("\n偏好剖面 (5 位参与人):")
    for i, pref in enumerate(preferences):
        print(f"  {i+1}: {' > '.join(pref.ranking)}")

    # 各规则结果
    rules = [
        ("多数制 (Plurality)", PluralityVoting(candidates)),
        ("Borda 计数", BordaCount(candidates)),
        ("Condorcet 方法", CondorcetMethod(candidates)),
        ("Instant-Runoff", InstantRunoff(candidates)),
    ]

    print("\n各规则的获胜者:")
    for name, rule in rules:
        winner = rule.winner(preferences)
        print(f"  {name:20s}: {winner}")

    # 策略证明性检验
    print("\n策略证明性检验 (随机测试):")
    for name, rule in rules:
        is_sp, msg = rule.is_strategy_proof(n_voters=5, test_cases=200)
        status = "✓ 策略证明" if is_sp else "✗ 可操纵"
        print(f"  {name:20s}: {status}")
        if not is_sp:
            print(f"    {msg}")


def demonstrate_median_voter_theorem():
    """演示 Median 投票者定理"""

    print("\n" + "=" * 70)
    print("Median 投票者定理演示")
    print("=" * 70)

    # 候选人按政治光谱排列
    candidates = ["极左", "左派", "中间", "右派", "极右"]

    # 生成单峰偏好
    peaks = ["左派", "极左", "中间", "右派", "左派", "中间"]
    preferences = [
        SinglePeakedPreference.generate_single_peaked(candidates, peak)
        for peak in peaks
    ]

    print(f"\n候选人: {' - '.join(candidates)}")
    print("\n参与人偏好 (单峰):")
    for i, (pref, peak) in enumerate(zip(preferences, peaks)):
        print(f"  {i+1} (峰值: {peak}): {' > '.join(pref.ranking[:3])} > ...")

    # Median 投票者定理预测
    median_winner = SinglePeakedPreference.median_voter_winner(preferences, candidates)
    print(f"\nMedian 峰值: {median_winner}")

    # 验证 Condorcet 性质
    condorcet = CondorcetMethod(candidates)
    cw = condorcet.find_condorcet_winner(preferences)
    print(f"Condorcet 赢家: {cw}")
    print(f"定理验证: {'✓ 通过' if median_winner == cw else '✗ 失败'}")


if __name__ == "__main__":
    demonstrate_condorcet_paradox()
    demonstrate_voting_rules()
    demonstrate_median_voter_theorem()
```
