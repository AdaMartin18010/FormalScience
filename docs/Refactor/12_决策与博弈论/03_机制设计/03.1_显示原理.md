# 12.3.1 显示原理

## 12.3.1.1 引言

**机制设计**（mechanism design）是博弈论的"逆向工程"：给定目标社会选择函数，设计博弈规则（机制）使得均衡结果实现该目标。**显示原理**（Revelation Principle）是机制设计理论的基石，由 Gibbard (1973)、Green & Laffont (1977) 和 Myerson (1979) 独立提出。它表明任何机制的均衡都可由直接机制实现，极大简化了机制设计问题。

---

## 12.3.1.2 社会选择函数

### 12.3.1.2.1 定义

**定义 12.3.1.1**（社会选择函数）

设：

- $N = \{1, ..., n\}$：参与人集合
- $\Theta_i$：参与人 $i$ 的**类型空间**（type space）
- $X$：**结果集**（alternatives）

**社会选择函数**（SCF）$f: \Theta \to X$ 将类型组合映射到社会结果：

$$f(\theta_1, ..., \theta_n) \in X$$

### 12.3.1.2.2 环境示例

| 环境 | 类型 $\theta_i$ | 结果 $X$ | SCF 目标 |
|------|-----------------|----------|----------|
| 拍卖 | 估值 | 分配+支付 | 效率/收益最大化 |
| 投票 | 偏好排序 | 获胜者 |  Condorcet 一致性 |
| 公共品 | 支付意愿 | 供给水平 | 效率 |

---

## 12.3.1.3 机制与实施

### 12.3.1.3.1 机制的定义

**定义 12.3.1.2**（机制）

**机制** $\mathcal{M} = \langle M, g \rangle$，其中：

- $M = M_1 \times ... \times M_n$：**消息空间**（message space）
- $g: M \to X$：**结果函数**（outcome function）

参与人选择消息 $m_i \in M_i$，结果 $g(m_1, ..., m_n)$ 被实施。

### 12.3.1.3.2 实施概念

**定义 12.3.1.3**（占优策略实施）

机制 $\mathcal{M}$ **占优策略实施** SCF $f$，如果存在策略 $s_i^*: \Theta_i \to M_i$ 使得：

1. **均衡**：$s^*$ 是占优策略均衡
2. **实施**：$g(s_1^*(\theta_1), ..., s_n^*(\theta_n)) = f(\theta)$ 对所有 $\theta$

**定义 12.3.1.4**（贝叶斯纳什实施）

类似定义，但要求 $s^*$ 是贝叶斯纳什均衡。

## 12.3.1.4 显示原理

### 12.3.1.4.1 直接机制

**定义 12.3.1.5**（直接机制）

**直接机制**是消息空间等于类型空间的机制：$M_i = \Theta_i$。

在直接机制中，参与人报告类型 $\hat{\theta}_i$，结果 $f(\hat{\theta})$ 被实施。

### 12.3.1.4.2 显示原理的陈述

**定理 12.3.1.1**（显示原理，Revelation Principle）

设 SCF $f$ 可由某机制 $\mathcal{M}$ 在占优策略（或贝叶斯纳什）均衡下实施。则 $f$ 可由**直接机制**在**真实报告**为占优策略（或贝叶斯纳什均衡）下实施。

### 12.3.1.4.3 证明

**证明**（占优策略情形）：

设 $\mathcal{M} = \langle M, g \rangle$ 在策略 $s^*$ 下实施 $f$。

**构造直接机制**：定义 $\tilde{g}(\hat{\theta}) = g(s_1^*(\hat{\theta}_1), ..., s_n^*(\hat{\theta}_n))$。

由实施定义，$\tilde{g}(\theta) = f(\theta)$。

**证明真实报告是均衡**：

假设类型 $\theta_i$ 的参与人考虑报告 $\hat{\theta}_i$。

在直接机制中，偏离到 $\hat{\theta}_i$ 的收益：
$$u_i(\tilde{g}(\hat{\theta}_i, \theta_{-i}), \theta_i) = u_i(g(s_i^*(\hat{\theta}_i), s_{-i}^*(\theta_{-i})), \theta_i)$$

由于 $s^*$ 是占优策略均衡，对任意 $m_i$：
$$u_i(g(s_i^*(\theta_i), s_{-i}^*(\theta_{-i})), \theta_i) \geq u_i(g(m_i, s_{-i}^*(\theta_{-i})), \theta_i)$$

特别地，取 $m_i = s_i^*(\hat{\theta}_i)$：
$$u_i(f(\theta_i, \theta_{-i}), \theta_i) \geq u_i(\tilde{g}(\hat{\theta}_i, \theta_{-i}), \theta_i)$$

因此真实报告是最优的。∎

---

## 12.3.1.5 激励相容

### 12.3.1.5.1 占优策略激励相容

**定义 12.3.1.6**（DSIC）

SCF $f$ 是**占优策略激励相容**（Dominant Strategy Incentive Compatible），如果对任意 $i$，$\theta_i, \hat{\theta}_i$，和 $\theta_{-i}$：

$$u_i(f(\theta_i, \theta_{-i}), \theta_i) \geq u_i(f(\hat{\theta}_i, \theta_{-i}), \theta_i)$$

**推论**（显示原理的重新表述）：

SCF $f$ 可占优策略实施当且仅当 $f$ 是 DSIC。

### 12.3.1.5.2 贝叶斯激励相容

**定义 12.3.1.7**（BIC）

设类型分布为 $F(\theta)$，条件分布为 $F_{-i}(\theta_{-i}|\theta_i)$。SCF $f$ 是**贝叶斯激励相容**，如果对任意 $i$ 和 $\theta_i, \hat{\theta}_i$：

$$\mathbb{E}_{\theta_{-i}}[u_i(f(\theta_i, \theta_{-i}), \theta_i) | \theta_i] \geq \mathbb{E}_{\theta_{-i}}[u_i(f(\hat{\theta}_i, \theta_{-i}), \theta_i) | \theta_i]$$

---

## 12.3.1.6 显示原理的应用

### 12.3.1.6.1 机制设计问题的简化

**原问题**：在所有可能的机制 $\mathcal{M}$ 中寻找实施 $f$ 的机制

**简化后**：在直接机制中寻找使真实报告为均衡的机制

### 12.3.1.6.2 不可能性定理的推论

**Gibbard-Satterthwaite 定理**：在非限制域上，非独裁的 DSIC SCF 只能是独裁。

**推论**：一般社会选择问题无法在占优策略下实施非平凡目标。

## 12.3.1.7 Python 实现：显示原理与激励相容

```python
"""
显示原理 (Revelation Principle) 实现
包含直接机制、激励相容检验
基于 Myerson (1979) 和 Green & Laffont (1977)
"""

from typing import List, Callable, Dict, Tuple, TypeVar, Generic
import numpy as np
from dataclasses import dataclass
from itertools import product

T = TypeVar('T')  # 类型
X = TypeVar('X')  # 结果


@dataclass
class Agent:
    """参与人"""
    id: int
    type_space: List[T]

    def utility(self, outcome: X, agent_type: T) -> float:
        """效用函数，由子类实现"""
        raise NotImplementedError


class SocialChoiceFunction:
    """社会选择函数 f: Θ → X"""

    def __init__(self,
                 agents: List[Agent],
                 outcome_space: List[X],
                 scf_map: Dict[Tuple, X]):
        """
        参数:
            agents: 参与人列表
            outcome_space: 结果空间
            scf_map: {(θ₁, θ₂, ...): outcome} 社会选择函数的映射
        """
        self.agents = agents
        self.outcome_space = outcome_space
        self.scf_map = scf_map
        self.n_agents = len(agents)

    def evaluate(self, types: Tuple) -> X:
        """计算 SCF 在给定类型组合下的结果"""
        return self.scf_map.get(types)

    def is_dsic(self, type_profiles: List[Tuple] = None) -> Tuple[bool, List[str]]:
        """
        检验是否为占优策略激励相容 (DSIC)
        返回: (是否 DSIC, 违反列表)
        """
        violations = []

        if type_profiles is None:
            # 生成所有类型组合
            type_profiles = list(product(*[a.type_space for a in self.agents]))

        for true_profile in type_profiles:
            outcome = self.evaluate(true_profile)

            for i, agent in enumerate(self.agents):
                true_type = true_profile[i]
                true_utility = agent.utility(outcome, true_type)

                # 检查所有可能的虚假报告
                for reported_type in agent.type_space:
                    if reported_type == true_type:
                        continue

                    reported_profile = list(true_profile)
                    reported_profile[i] = reported_type
                    reported_profile = tuple(reported_profile)

                    reported_outcome = self.evaluate(reported_profile)
                    reported_utility = agent.utility(reported_outcome, true_type)

                    if reported_utility > true_utility + 1e-10:
                        violations.append(
                            f"Agent {i}: 真实类型 {true_type}, 报告 {reported_type} "
                            f"更有利 (效用: {reported_utility:.2f} > {true_utility:.2f})"
                        )

        return len(violations) == 0, violations

    def is_bic(self,
               type_distribution: Dict[Tuple, float],
               conditional_dist: Callable[[int, T], Dict[Tuple, float]]) -> Tuple[bool, List[str]]:
        """
        检验是否为贝叶斯激励相容 (BIC)

        参数:
            type_distribution: 类型联合分布
            conditional_dist: 条件分布函数 (i, θ_i) -> {θ_{-i}: prob}
        """
        violations = []

        for i, agent in enumerate(self.agents):
            for true_type in agent.type_space:
                # 真实报告下的期望效用
                cond_dist = conditional_dist(i, true_type)

                expected_truthful = sum(
                    prob * agent.utility(
                        self.evaluate(tuple(list(true_profile[:i]) + [true_type] + list(true_profile[i:]))),
                        true_type
                    )
                    for true_profile, prob in cond_dist.items()
                )

                # 检查所有可能的虚假报告
                for reported_type in agent.type_space:
                    if reported_type == true_type:
                        continue

                    expected_reported = sum(
                        prob * agent.utility(
                            self.evaluate(tuple(list(true_profile[:i]) + [reported_type] + list(true_profile[i:]))),
                            true_type
                        )
                        for true_profile, prob in cond_dist.items()
                    )

                    if expected_reported > expected_truthful + 1e-10:
                        violations.append(
                            f"Agent {i}, 类型 {true_type}: 报告 {reported_type} "
                            f"期望效用更高 ({expected_reported:.2f} > {expected_truthful:.2f})"
                        )

        return len(violations) == 0, violations


class DirectMechanism:
    """
    直接机制
    消息空间 = 类型空间
    """

    def __init__(self, scf: SocialChoiceFunction):
        self.scf = scf

    def simulate_report(self, reported_types: Tuple) -> X:
        """模拟参与人报告类型后的结果"""
        return self.scf.evaluate(reported_types)

    def find_profitable_deviation(self,
                                   true_types: Tuple,
                                   agent_id: int) -> Tuple[bool, T, float]:
        """
        检查某参与人是否有利可图的偏离
        返回: (是否有偏离, 最优偏离类型, 效用增益)
        """
        agent = self.scf.agents[agent_id]
        true_type = true_types[agent_id]

        # 真实报告的结果
        true_outcome = self.simulate_report(true_types)
        true_utility = agent.utility(true_outcome, true_type)

        best_deviation = None
        best_gain = 0

        for reported_type in agent.type_space:
            if reported_type == true_type:
                continue

            reported_types = list(true_types)
            reported_types[agent_id] = reported_type
            reported_types = tuple(reported_types)

            reported_outcome = self.simulate_report(reported_types)
            reported_utility = agent.utility(reported_outcome, true_type)

            gain = reported_utility - true_utility
            if gain > best_gain:
                best_gain = gain
                best_deviation = reported_type

        return best_gain > 1e-10, best_deviation, best_gain


# 示例：单物品拍卖环境
class AuctionAgent(Agent):
    """拍卖中的竞标者"""

    def __init__(self, id: int, valuations: List[float]):
        super().__init__(id, valuations)
        self.valuations = valuations

    def utility(self, outcome: Tuple[int, float], valuation: float) -> float:
        """
        结果: (winner_id, payment)
        效用: 如果是获胜者，则 v - payment，否则 0
        """
        winner, payment = outcome
        if winner == self.id:
            return valuation - payment
        return 0


class SecondPriceAuction(SocialChoiceFunction):
    """
    第二价格拍卖（VCG 机制）
    结果: (winner, second_highest_bid)
    """

    def __init__(self, agents: List[AuctionAgent]):
        # 构建 SCF 映射
        scf_map = {}
        type_profiles = list(product(*[a.type_space for a in agents]))

        for profile in type_profiles:
            # 最高估值者获胜
            bids = [(i, profile[i]) for i in range(len(agents))]
            bids.sort(key=lambda x: x[1], reverse=True)

            winner = bids[0][0]
            second_price = bids[1][1] if len(bids) > 1 else 0

            scf_map[profile] = (winner, second_price)

        super().__init__(agents, [(i, p) for i in range(len(agents)) for p in range(100)], scf_map)

    @staticmethod
    def create_example():
        """创建示例拍卖"""
        valuations = [10, 20, 30, 40, 50]
        agents = [AuctionAgent(i, valuations) for i in range(3)]
        return SecondPriceAuction(agents)


def demo_revelation_principle():
    """显示原理演示"""

    print("=" * 70)
    print("显示原理 (Revelation Principle) 演示")
    print("=" * 70)

    # 示例：第二价格拍卖
    print("\n1. 第二价格拍卖 (VCG 机制)")
    print("-" * 50)

    auction = SecondPriceAuction.create_example()

    print("参与人:")
    for agent in auction.agents:
        print(f"  Agent {agent.id}: 估值空间 {agent.type_space}")

    print("\nSCF 规则:")
    print("  - 最高估值者获胜")
    print("  - 支付第二高估值（VCG 支付）")

    # 检验 DSIC
    print("\n检验占优策略激励相容 (DSIC):")
    is_dsic, violations = auction.is_dsic()

    if is_dsic:
        print("  ✓ 第二价格拍卖满足 DSIC")
        print("  真实报价是占优策略")
    else:
        print("  ✗ 发现违反:")
        for v in violations:
            print(f"    {v}")

    # 直接机制模拟
    print("\n" + "-" * 50)
    print("2. 直接机制模拟")

    dm = DirectMechanism(auction)

    # 测试真实报告
    true_types = (30, 40, 20)
    outcome = dm.simulate_report(true_types)
    print(f"\n  真实类型: {true_types}")
    print(f"  SCF 结果: 获胜者 = Agent {outcome[0]}, 支付 = {outcome[1]}")

    # 检查偏离动机
    for i in range(len(auction.agents)):
        has_deviation, dev_type, gain = dm.find_profitable_deviation(true_types, i)
        if has_deviation:
            print(f"  Agent {i} 可以偏离到 {dev_type}，增益 {gain:.2f}")
        else:
            print(f"  Agent {i}: 真实报告是最优的")

    # 显示原理的意义
    print("\n" + "=" * 70)
    print("3. 显示原理的意义")
    print("-" * 50)
    print("""
显示原理表明：
1. 任何机制的均衡结果都可以由直接机制实现
2. 只需考虑"真实报告为均衡"的直接机制
3. 极大简化了机制设计问题

应用：
- 拍卖设计：只需考虑激励相容的直接机制
- 投票规则：只需考虑真实偏好表达的规则
- 契约理论：只需考虑真实信息揭示的契约
    """)


if __name__ == "__main__":
    demo_revelation_principle()
```
