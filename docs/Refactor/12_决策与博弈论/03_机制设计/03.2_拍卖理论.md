# 12.3.2 拍卖理论

## 12.3.2.1 引言

**拍卖**是资源分配和价格发现的核心机制。
拍卖理论由 Vickrey (1961)、Myerson (1981) 和 Riley & Samuelson (1981) 奠基，结合博弈论和机制设计，为拍卖规则的设计和分析提供了严谨框架。
本章介绍标准拍卖形式、最优拍卖设计和收益等价定理。

---

## 12.3.2.2 标准拍卖形式

### 12.3.2.2.1 四种基本拍卖

设有 $n$ 个竞标者，物品对竞标者 $i$ 的价值为 $v_i$（私有价值）：

| 拍卖形式 | 规则 |
|----------|------|
| **一级密封价格** | 最高报价者获胜，支付自己的报价 |
| **二级密封价格** | 最高报价者获胜，支付第二高报价（Vickrey） |
| **英式拍卖** | 公开 ascending，最后退出者获胜，支付最终价格 |
| **荷兰式拍卖** | 公开 descending，最先接受者获胜，支付当前价格 |

### 12.3.2.2.2 英式与第二价格的等价

**定理 12.3.2.1**（英式拍卖的占优策略）

在英式拍卖中，竞标者的占优策略是：

- 当价格低于估值时保持参与
- 当价格达到估值时退出

**推论**：英式拍卖的均衡结果等价于第二价格密封拍卖。

---

## 12.3.2.3 收益等价定理

### 12.3.2.3.1 定理陈述

**定理 12.3.2.2**（收益等价定理，Myerson, 1981; Riley & Samuelson, 1981）

假设：

1. 竞标者风险中性
2. 估值独立同分布 $F(v)$，支撑 $[\underline{v}, \bar{v}]$
3. 竞标者对称
4. 零估值竞标者期望支付为零

则所有**标准拍卖**（分配效率相同、零类型支付为零的机制）产生相同的期望收益。

### 12.3.2.3.2 证明

**证明**：

**步骤 1**：竞标者的期望效用

设竞标者 $i$ 的期望效用为 $U_i(v_i)$。由包络定理：

$$U_i(v_i) = U_i(\underline{v}) + \int_{\underline{v}}^{v_i} Q_i(x) dx$$

其中 $Q_i(v_i)$ 是类型 $v_i$ 获胜的概率。

**步骤 2**：期望支付

竞标者的期望支付：

$$P_i(v_i) = v_i Q_i(v_i) - U_i(v_i) = v_i Q_i(v_i) - \int_{\underline{v}}^{v_i} Q_i(x) dx$$

（假设 $U_i(\underline{v}) = 0$）

**步骤 3**：卖方的期望收益

$$\mathbb{E}[R] = \sum_{i=1}^n \mathbb{E}[P_i(v_i)] = \sum_{i=1}^n \int_{\underline{v}}^{\bar{v}} P_i(v) dF(v)$$

$$= \sum_{i=1}^n \int_{\underline{v}}^{\bar{v}} \left[v Q_i(v) - \int_{\underline{v}}^{v} Q_i(x) dx\right] dF(v)$$

**步骤 4**：分部积分

$$\int_{\underline{v}}^{\bar{v}} \int_{\underline{v}}^{v} Q_i(x) dx \, dF(v) = \int_{\underline{v}}^{\bar{v}} Q_i(v) (1 - F(v)) dv$$

（交换积分顺序）

**步骤 5**：整理

$$\mathbb{E}[R] = \sum_{i=1}^n \int_{\underline{v}}^{\bar{v}} Q_i(v) \left[v - \frac{1-F(v)}{f(v)}\right] dF(v)$$

$$= \sum_{i=1}^n \mathbb{E}\left[Q_i(v_i) \cdot \psi(v_i)\right]$$

其中 $\psi(v) = v - \frac{1-F(v)}{f(v)}$ 是**虚拟估值**（virtual valuation）。

**步骤 6**：标准拍卖的比较

标准拍卖都满足：

1. 分配效率：最高估值者获胜，$Q_i(v) = F(v)^{n-1}$
2. 零类型支付为零：$P_i(\underline{v}) = 0$

因此期望收益相同。∎

## 12.3.2.4 最优拍卖设计

### 12.3.2.4.1 虚拟估值

**定义 12.3.2.1**（虚拟估值）

竞标者的**虚拟估值**：

$$\psi(v) = v - \frac{1 - F(v)}{f(v)}$$

**正则分布**：$\psi(v)$ 严格递增。

### 12.3.2.4.2 最优拍卖

**定理 12.3.2.3**（Myerson 最优拍卖）

最优拍卖的规则：

1. **分配**：将物品分配给虚拟估值最高（且为正）的竞标者
2. **支付**：按虚拟估值第二高价支付（调整为真实报价空间）

**保留价**：设置保留价 $r^*$ 满足 $\psi(r^*) = 0$。

---

## 12.3.2.5 多物品拍卖

### 12.3.2.5.1 VCG 机制

**定义 12.3.2.2**（VCG 机制）

对多物品分配问题，VCG 机制：

1. **分配规则**：选择社会剩余最大化的分配 $x^*$
   $$x^* \in \arg\max_x \sum_i v_i(x)$$

2. **支付规则**：每个参与人支付外部性
   $$p_i = \max_x \sum_{j \neq i} v_j(x) - \sum_{j \neq i} v_j(x^*)$$

**定理 12.3.2.4**（VCG 的性质）

VCG 机制是：

- 效率的（分配最大化社会剩余）
- 占优策略激励相容的
- 个体理性的（支付不超过收益）

## 12.3.2.6 Python 实现：拍卖模拟与分析

```python
"""
拍卖理论实现
包含标准拍卖形式、收益等价定理、最优拍卖设计
基于 Vickrey (1961), Myerson (1981)
"""

from typing import List, Tuple, Callable, Optional
import numpy as np
from scipy import stats
from scipy.optimize import minimize_scalar
import matplotlib.pyplot as plt


class Bidder:
    """竞标者"""

    def __init__(self, id: int, valuation: float):
        self.id = id
        self.v = valuation  # 真实估值
        self.bid = None     # 报价

    def utility(self, win: bool, payment: float) -> float:
        """计算效用"""
        if win:
            return self.v - payment
        return 0


class Auction:
    """拍卖基类"""

    def __init__(self, bidders: List[Bidder]):
        self.bidders = bidders
        self.n = len(bidders)
        self.winner = None
        self.payment = None
        self.revenue = None

    def run(self) -> Tuple[int, float]:
        """运行拍卖，返回 (获胜者ID, 支付)"""
        raise NotImplementedError

    def get_revenue(self) -> float:
        """获取拍卖收益"""
        return self.revenue if self.revenue is not None else 0


class FirstPriceAuction(Auction):
    """一级密封价格拍卖"""

    def __init__(self, bidders: List[Bidder], equilibrium_bidding: bool = True):
        super().__init__(bidders)
        self.equilibrium_bidding = equilibrium_bidding

    def run(self) -> Tuple[int, float]:
        """
        运行一级价格拍卖
        均衡策略: b(v) = E[v_{(2)} | v_{(1)} = v]（对称均匀分布时: (n-1)/n * v）
        """
        if self.equilibrium_bidding:
            # 对称均匀分布 [0,1] 的均衡策略
            for bidder in self.bidders:
                n = self.n
                bidder.bid = (n - 1) / n * bidder.v
        else:
            # 真实报价（非均衡）
            for bidder in self.bidders:
                bidder.bid = bidder.v

        # 最高报价者获胜，支付自己的报价
        winner = max(self.bidders, key=lambda b: b.bid)
        self.winner = winner.id
        self.payment = winner.bid
        self.revenue = self.payment

        return self.winner, self.payment


class SecondPriceAuction(Auction):
    """二级密封价格拍卖 (Vickrey)"""

    def run(self) -> Tuple[int, float]:
        """
        运行二级价格拍卖
        占优策略: 真实报价 b = v
        """
        # 真实报价
        for bidder in self.bidders:
            bidder.bid = bidder.v

        # 排序
        sorted_bidders = sorted(self.bidders, key=lambda b: b.bid, reverse=True)

        # 最高报价者获胜，支付第二高报价
        winner = sorted_bidders[0]
        second_price = sorted_bidders[1].bid if len(sorted_bidders) > 1 else 0

        self.winner = winner.id
        self.payment = second_price
        self.revenue = self.payment

        return self.winner, self.payment


class DutchAuction(Auction):
    """荷兰式拍卖（降价拍卖）"""

    def __init__(self, bidders: List[Bidder], initial_price: float = 1.0,
                 decrement: float = 0.01):
        super().__init__(bidders)
        self.initial_price = initial_price
        self.decrement = decrement

    def run(self) -> Tuple[int, float]:
        """
        运行荷兰式拍卖
        策略等价于一级价格拍卖
        """
        # 在对称均匀分布下，均衡策略是当价格降到 (n-1)/n * v 时接受
        threshold_prices = [(n - 1) / self.n * b.v for b in self.bidders]

        # 找到最先接受的竞标者（最低阈值）
        winner_idx = np.argmin(threshold_prices)
        winner = self.bidders[winner_idx]

        self.winner = winner.id
        self.payment = threshold_prices[winner_idx]
        self.revenue = self.payment

        return self.winner, self.payment


class EnglishAuction(Auction):
    """英式拍卖（升价拍卖）"""

    def run(self) -> Tuple[int, float]:
        """
        运行英式拍卖
        占优策略: 当价格达到 v 时退出
        结果等价于二级价格拍卖
        """
        # 按估值排序
        sorted_bidders = sorted(self.bidders, key=lambda b: b.v, reverse=True)

        # 最高估值者获胜，支付第二高估值
        winner = sorted_bidders[0]
        second_highest = sorted_bidders[1].v if len(sorted_bidders) > 1 else 0

        self.winner = winner.id
        self.payment = second_highest
        self.revenue = self.payment

        return self.winner, self.payment


class RevenueEquivalenceAnalyzer:
    """收益等价定理分析器"""

    def __init__(self, n_bidders: int, distribution: Callable[[], float],
                 n_simulations: int = 10000):
        self.n = n_bidders
        self.dist = distribution
        self.n_sim = n_simulations

    def simulate_auctions(self) -> Dict[str, float]:
        """模拟不同拍卖形式的收益"""
        revenues = {
            'First Price': [],
            'Second Price': [],
            'Dutch': [],
            'English': []
        }

        for _ in range(self.n_sim):
            # 生成估值
            valuations = [self.dist() for _ in range(self.n)]
            bidders = [Bidder(i, v) for i, v in enumerate(valuations)]

            # 一级价格拍卖
            fp = FirstPriceAuction(bidders.copy())
            fp.run()
            revenues['First Price'].append(fp.get_revenue())

            # 二级价格拍卖
            sp = SecondPriceAuction(bidders.copy())
            sp.run()
            revenues['Second Price'].append(sp.get_revenue())

            # 荷兰式拍卖
            dutch = DutchAuction(bidders.copy())
            dutch.run()
            revenues['Dutch'].append(dutch.get_revenue())

            # 英式拍卖
            eng = EnglishAuction(bidders.copy())
            eng.run()
            revenues['English'].append(eng.get_revenue())

        # 计算平均收益
        return {k: np.mean(v) for k, v in revenues.items()}

    def plot_revenue_distribution(self):
        """绘制收益分布"""
        fig, axes = plt.subplots(2, 2, figsize=(12, 10))
        axes = axes.flatten()

        auction_classes = [
            ('First Price', FirstPriceAuction),
            ('Second Price', SecondPriceAuction),
            ('Dutch', DutchAuction),
            ('English', EnglishAuction)
        ]

        for idx, (name, AuctionClass) in enumerate(auction_classes):
            revenues = []
            for _ in range(self.n_sim):
                valuations = [self.dist() for _ in range(self.n)]
                bidders = [Bidder(i, v) for i, v in enumerate(valuations)]
                auction = AuctionClass(bidders)
                auction.run()
                revenues.append(auction.get_revenue())

            axes[idx].hist(revenues, bins=50, alpha=0.7, edgecolor='black')
            axes[idx].axvline(np.mean(revenues), color='red', linestyle='--',
                            label=f'Mean: {np.mean(revenues):.3f}')
            axes[idx].set_title(f'{name} Auction')
            axes[idx].set_xlabel('Revenue')
            axes[idx].set_ylabel('Frequency')
            axes[idx].legend()

        plt.tight_layout()
        return fig


class OptimalAuction:
    """
    Myerson 最优拍卖
    考虑虚拟估值和保留价
    """

    def __init__(self, distribution: stats.rv_continuous):
        """
        参数:
            distribution: 估值分布
        """
        self.dist = distribution

    def virtual_valuation(self, v: float) -> float:
        """
        计算虚拟估值 ψ(v) = v - (1-F(v))/f(v)
        """
        F = self.dist.cdf(v)
        f = self.dist.pdf(v)

        if f < 1e-10:
            return v

        return v - (1 - F) / f

    def optimal_reserve_price(self, bracket: Tuple[float, float] = (0, 10)) -> float:
        """
        计算最优保留价 r* = ψ^{-1}(0)
        """
        result = minimize_scalar(
            lambda r: abs(self.virtual_valuation(r)),
            bounds=bracket,
            method='bounded'
        )
        return result.x

    def run_optimal_auction(self, valuations: List[float]) -> Tuple[Optional[int], float]:
        """
        运行最优拍卖
        分配给虚拟估值最高且为正的竞标者
        """
        virtual_vals = [self.virtual_valuation(v) for v in valuations]

        # 找到最高虚拟估值
        max_vv = max(virtual_vals)

        if max_vv <= 0:
            # 不分配
            return None, 0

        winner = virtual_vals.index(max_vv)

        # 计算支付（第二高的虚拟估值对应的真实报价）
        # 简化为第二高报价（对对称分布）
        sorted_vals = sorted(valuations, reverse=True)
        payment = sorted_vals[1] if len(sorted_vals) > 1 else self.optimal_reserve_price()

        return winner, payment


def demo_auction_theory():
    """拍卖理论演示"""

    print("=" * 70)
    print("拍卖理论演示")
    print("=" * 70)

    # 收益等价定理演示
    print("\n1. 收益等价定理")
    print("-" * 50)

    # 均匀分布 [0,1]
    np.random.seed(42)
    analyzer = RevenueEquivalenceAnalyzer(
        n_bidders=5,
        distribution=lambda: np.random.uniform(0, 1),
        n_simulations=5000
    )

    print("模拟参数:")
    print("  - 竞标者数量: 5")
    print("  - 估值分布: Uniform[0,1]")
    print("  - 模拟次数: 5000")

    revenues = analyzer.simulate_auctions()

    print("\n各拍卖形式的期望收益:")
    for name, rev in revenues.items():
        print(f"  {name:15s}: {rev:.4f}")

    print("\n理论预期收益 (n=5, Uniform[0,1]):")
    print(f"  E[Revenue] = (n-1)/(n+1) = 4/6 ≈ 0.6667")

    # 最优拍卖
    print("\n" + "=" * 70)
    print("2. 最优拍卖设计")
    print("-" * 50)

    # 均匀分布的最优拍卖
    uniform_dist = stats.uniform(loc=0, scale=1)
    opt_auction = OptimalAuction(uniform_dist)

    r_star = opt_auction.optimal_reserve_price(bracket=(0, 1))
    print(f"均匀分布 [0,1] 的最优保留价: r* = {r_star:.4f}")
    print(f"理论值: r* = 0.5")

    # 模拟
    print("\n最优拍卖 vs 标准拍卖 (带保留价):")
    n_sim = 10000

    revenues_optimal = []
    revenues_sp = []

    for _ in range(n_sim):
        valuations = np.random.uniform(0, 1, 5)

        # 最优拍卖
        _, payment_opt = opt_auction.run_optimal_auction(list(valuations))
        revenues_optimal.append(payment_opt)

        # 标准二级价格拍卖（无保留价）
        bidders = [Bidder(i, v) for i, v in enumerate(valuations)]
        sp = SecondPriceAuction(bidders)
        sp.run()
        revenues_sp.append(sp.get_revenue())

    print(f"  最优拍卖期望收益: {np.mean(revenues_optimal):.4f}")
    print(f"  标准二级价格期望收益: {np.mean(revenues_sp):.4f}")
    print(f"  收益提升: {(np.mean(revenues_optimal)/np.mean(revenues_sp)-1)*100:.1f}%")


if __name__ == "__main__":
    demo_auction_theory()
```
