## 12.3.4.1 引言

**匹配理论**（matching theory）研究如何将对立的双方（如工人与职位、学生与学校）配对。Gale & Shapley (1962) 的开创性论文提出了**延迟接受算法**（deferred acceptance algorithm），证明了稳定匹配的存在性，并引发了匹配市场设计的广泛研究。2012 年诺贝尔经济学奖授予 Alvin Roth 和 Lloyd Shapley，表彰他们在稳定匹配理论和市场设计实践中的贡献。

---

## 12.3.4.2 匹配问题的基本形式

### 12.3.4.2.1 双边匹配

**定义 12.3.4.1**（双边匹配问题）

设 $M$ 为**男性集合**，$W$ 为**女性集合**。一个**匹配** $\mu$ 是从 $M \cup W$ 到 $M \cup W$ 的双射，满足：

1. $\mu(m) \in W \cup \{m\}$（男性匹配到女性或保持单身）
2. $\mu(w) \in M \cup \{w\}$（女性匹配到男性或保持单身）
3. $\mu(m) = w \iff \mu(w) = m$

### 12.3.4.2.2 偏好与稳定性

**定义 12.3.4.2**（稳定匹配）

匹配 $\mu$ 是**稳定的**，如果不存在**阻止对**（blocking pair）$(m, w)$ 使得：

1. $w \succ_m \mu(m)$（$m$ 偏好 $w$ 胜于当前匹配）
2. $m \succ_w \mu(w)$（$w$ 偏好 $m$ 胜于当前匹配）

## 12.3.4.3 Gale-Shapley 算法

### 12.3.4.3.1 算法描述

**算法 12.3.4.1**（男性提议的延迟接受算法）

```
初始化：所有男性和女性都未婚

当存在未匹配的男性 m 且 m 还有未求婚的女性时：
    m 向他偏好列表中排名最高的未求婚女性 w 求婚

    如果 w 未婚：
        (m, w) 暂时匹配
    否则如果 w 偏好 m 胜过当前匹配 m'：
        (m, w) 匹配，m' 变为未婚
    否则：
        w 拒绝 m

返回最终匹配
```

### 12.3.4.3.2 存在性与唯一性

**定理 12.3.4.1**（Gale & Shapley, 1962）

对任何双边匹配问题，稳定匹配总是存在。

**证明**（算法终止性）：

1. **终止性**：每个男性最多向每个女性求婚一次，有限步后终止
2. **完备性**：算法结束时，没有未匹配男性还有未求婚女性
3. **稳定性**：
   - 不存在男性有更偏好的未求婚女性（已求婚都被拒或当前匹配更偏好）
   - 不存在女性有更偏好的男性未向她求婚（如存在，该男性应已求婚）

因此无阻止对，匹配稳定。∎

### 12.3.4.3.3 最优性

**定理 12.3.4.2**（男性最优稳定匹配）

男性提议的 Gale-Shapley 算法产生**男性最优稳定匹配**：在所有稳定匹配中，每个男性都获得最偏好的女性。

**定理 12.3.4.3**（女性最劣稳定匹配）

男性提议的算法产生女性最劣稳定匹配。

## 12.3.4.4 匹配的性质

### 12.3.4.4.1 策略证明性

**定理 12.3.4.4**（Dubins & Freedman, 1981; Roth, 1982）

在男性提议的 Gale-Shapley 算法中，男性没有虚报偏好的动机（真实偏好是占优策略），但女性可能有操纵动机。

### 12.3.4.4.2  lattice 结构

**定理 12.3.4.5**（Knuth, 1976）

稳定匹配集合构成 distributive lattice：

- join：男性视角的最优匹配
- meet：女性视角的最优匹配

## 12.3.4.5 单边匹配：宿舍分配问题

### 12.3.4.5.1 问题描述

**宿舍分配问题**：将学生分配到宿舍房间，只有学生对房间的偏好，房间无偏好。

### 12.3.4.5.2 序列独裁机制

**机制 12.3.4.1**（序列独裁）

1. 按某种优先级排序学生
2. 按顺序，每个学生选择最喜欢的可用房间

**性质**：策略证明且 Pareto 有效。

### 12.3.4.5.3 Top Trading Cycles

**算法 12.3.4.2**（TTC, Shapley & Scarf, 1974）

```
初始化：所有房间可用

重复直到所有学生分配：
    每个学生指向最偏好的可用房间
    每个房间指向当前占用者
    识别所有循环
    循环中的学生获得指向的房间并退出
```

**定理 12.3.4.6**（TTC 的性质）

TTC 机制是：

- 策略证明的
- Pareto 有效的
- 个体理性的

## 12.3.4.6 市场设计应用

### 12.3.4.6.1 医学住院医师匹配

**美国 NRMP**（National Resident Matching Program）：

- 医院向医学生提供职位
- 使用医院提议的延迟接受算法
- 解决了" scrambling" 问题

### 12.3.4.6.2 学校选择

**波士顿机制 vs 延迟接受**：

- **波士顿机制**：优先处理第一志愿，竞争激烈
- **延迟接受**：更安全，策略性较弱

## 12.3.4.7 Python 实现：匹配算法

```python
"""
匹配理论实现
包含 Gale-Shapley 算法、Top Trading Cycles
基于 Gale & Shapley (1962) 和 Shapley & Scarf (1974)
"""

from typing import List, Dict, Optional, Tuple, Set
from dataclasses import dataclass, field
from collections import deque
import copy


@dataclass
class Agent:
    """匹配参与人"""
    id: str
    preferences: List[str]  # 从最优到最差的偏好列表

    def preference_rank(self, other_id: str) -> int:
        """返回 other_id 在偏好列表中的排名（0 = 最优）"""
        if other_id not in self.preferences:
            return float('inf')
        return self.preferences.index(other_id)

    def prefers(self, a: str, b: str) -> bool:
        """检查是否偏好 a 胜过 b"""
        return self.preference_rank(a) < self.preference_rank(b)


class GaleShapley:
    """
    Gale-Shapley 延迟接受算法
    支持男性提议和女性提议两种模式
    """

    def __init__(self, men: List[Agent], women: List[Agent]):
        self.men = {m.id: m for m in men}
        self.women = {w.id: w for w in women}
        self.men_list = men
        self.women_list = women

    def solve(self, proposer_side: str = "men") -> Dict[str, str]:
        """
        求解稳定匹配

        参数:
            proposer_side: "men" 或 "women"，决定哪一方主动提议

        返回:
            匹配字典 {提议方ID: 被提议方ID}
        """
        if proposer_side == "men":
            proposers = self.men_list
            receivers = self.women
            proposer_dict = self.men
        else:
            proposers = self.women_list
            receivers = self.men
            proposer_dict = self.women

        # 初始化
        matches = {}  # {接收者ID: 提议者ID}
        next_proposal = {p.id: 0 for p in proposers}  # 下一个要提议的索引
        unmatched = deque([p.id for p in proposers])  # 未匹配的提议者队列

        while unmatched:
            proposer_id = unmatched.popleft()
            proposer = proposer_dict[proposer_id]

            # 检查是否还有可提议的对象
            if next_proposal[proposer_id] >= len(proposer.preferences):
                continue  # 保持单身

            # 向下一个最偏好的对象提议
            receiver_id = proposer.preferences[next_proposal[proposer_id]]
            next_proposal[proposer_id] += 1

            if receiver_id not in receivers:
                # 提议对象不存在（可能在测试中）
                unmatched.append(proposer_id)
                continue

            receiver = receivers[receiver_id]

            if receiver_id not in matches:
                # 接收者未婚
                matches[receiver_id] = proposer_id
            else:
                # 接收者已匹配，比较
                current_match = matches[receiver_id]

                if receiver.prefers(proposer_id, current_match):
                    # 接受新提议
                    matches[receiver_id] = proposer_id
                    unmatched.append(current_match)  # 旧匹配变为未婚
                else:
                    # 拒绝新提议
                    unmatched.append(proposer_id)

        return matches

    def get_full_matching(self, proposer_side: str = "men") -> Dict[str, str]:
        """
        获取完整的双向匹配
        返回: {参与人ID: 匹配对象ID}
        """
        matches = self.solve(proposer_side)

        # 转换为双向匹配
        full_matching = {}
        for receiver, proposer in matches.items():
            full_matching[receiver] = proposer
            full_matching[proposer] = receiver

        # 添加未匹配的单身者
        if proposer_side == "men":
            for m in self.men:
                if m.id not in full_matching:
                    full_matching[m.id] = None
            for w in self.women:
                if w.id not in full_matching:
                    full_matching[w.id] = None
        else:
            for w in self.women:
                if w.id not in full_matching:
                    full_matching[w.id] = None
            for m in self.men:
                if m.id not in full_matching:
                    full_matching[m.id] = None

        return full_matching

    def is_stable(self, matching: Dict[str, str]) -> Tuple[bool, Optional[Tuple]]:
        """
        检验匹配是否稳定
        返回: (是否稳定, 阻止对 或 None)
        """
        # 构造反向匹配
        reverse_match = {v: k for k, v in matching.items() if v is not None}

        # 检查所有可能的阻止对
        for m in self.men.values():
            current_match = matching.get(m.id)

            for w_id in m.preferences:
                if w_id == current_match:
                    break  # 只检查比当前匹配更偏好的

                if w_id not in self.women:
                    continue

                w = self.women[w_id]
                w_current = matching.get(w_id)

                # 检查 w 是否偏好 m
                if w_current is None or w.prefers(m.id, w_current):
                    return False, (m.id, w_id)

        return True, None

    def compare_matchings(self, matching1: Dict[str, str],
                         matching2: Dict[str, str],
                         side: str = "men") -> Dict[str, str]:
        """
        比较两个匹配对某一方的好坏
        返回: {参与人ID: "better", "worse", "same"}
        """
        comparison = {}
        agents = self.men if side == "men" else self.women

        for agent in agents.values():
            m1 = matching1.get(agent.id)
            m2 = matching2.get(agent.id)

            if m1 == m2:
                comparison[agent.id] = "same"
            elif m1 is None:
                comparison[agent.id] = "worse"
            elif m2 is None:
                comparison[agent.id] = "better"
            elif agent.prefers(m1, m2):
                comparison[agent.id] = "better"
            else:
                comparison[agent.id] = "worse"

        return comparison


class TopTradingCycles:
    """
    Top Trading Cycles (TTC) 算法
    用于单边匹配问题（如宿舍分配）
    """

    def __init__(self, agents: List[Agent], initial_endowment: Dict[str, str]):
        """
        参数:
            agents: 参与人列表
            initial_endowment: {参与人ID: 初始分配物品ID}
        """
        self.agents = {a.id: a for a in agents}
        self.initial = initial_endowment
        self.items = set(initial_endowment.values())

    def solve(self) -> Dict[str, str]:
        """
        求解 TTC 分配
        返回: {参与人ID: 最终分配物品ID}
        """
        allocation = self.initial.copy()
        unassigned = set(self.agents.keys())
        available_items = self.items.copy()

        while unassigned:
            # 构造指向图
            # 每个人指向最偏好的可用物品
            points_to = {}

            for agent_id in unassigned:
                agent = self.agents[agent_id]
                # 找到最偏好的可用物品
                for item in agent.preferences:
                    if item in available_items:
                        points_to[agent_id] = item
                        break

            # 物品指向当前占用者
            item_held_by = {v: k for k, v in allocation.items() if k in unassigned}

            # 寻找循环
            visited = set()
            cycles = []

            for start in unassigned:
                if start in visited:
                    continue

                path = []
                current = start

                while current not in visited:
                    visited.add(current)
                    path.append(current)

                    if current in points_to:
                        item = points_to[current]
                        current = item_held_by.get(item)
                        if current is None:
                            break
                    else:
                        break

                    if current in path:
                        # 找到循环
                        cycle_start = path.index(current)
                        cycle = path[cycle_start:]
                        cycles.append(cycle)
                        break

                # 如果当前路径没有形成循环且已终止，继续下一个起点

            if not cycles:
                break  # 没有循环，算法结束

            # 执行循环中的交易
            for cycle in cycles:
                # 循环中的每个人获得指向的物品
                new_allocations = {}
                for i, agent_id in enumerate(cycle):
                    item = points_to[agent_id]
                    new_allocations[agent_id] = item

                # 更新分配
                for agent_id, item in new_allocations.items():
                    allocation[agent_id] = item
                    unassigned.remove(agent_id)

        return allocation


def demo_gale_shapley():
    """演示 Gale-Shapley 算法"""

    print("=" * 70)
    print("Gale-Shapley 延迟接受算法演示")
    print("=" * 70)

    # 定义偏好
    men = [
        Agent("m1", ["w1", "w2", "w3", "w4"]),
        Agent("m2", ["w2", "w1", "w4", "w3"]),
        Agent("m3", ["w3", "w4", "w1", "w2"]),
        Agent("m4", ["w4", "w3", "w2", "w1"]),
    ]

    women = [
        Agent("w1", ["m2", "m1", "m4", "m3"]),
        Agent("w2", ["m1", "m2", "m3", "m4"]),
        Agent("w3", ["m4", "m3", "m2", "m1"]),
        Agent("w4", ["m3", "m4", "m1", "m2"]),
    ]

    gs = GaleShapley(men, women)

    # 男性提议
    print("\n1. 男性提议的算法:")
    men_optimal = gs.get_full_matching("men")
    print("   匹配结果:")
    for m in men:
        match = men_optimal[m.id]
        print(f"     {m.id} <-> {match if match else '单身'}")

    is_stable, blocking = gs.is_stable(men_optimal)
    print(f"   稳定性: {'✓ 稳定' if is_stable else '✗ 不稳定'}")

    # 女性提议
    print("\n2. 女性提议的算法:")
    women_optimal = gs.get_full_matching("women")
    print("   匹配结果:")
    for m in men:
        match = women_optimal[m.id]
        print(f"     {m.id} <-> {match if match else '单身'}")

    is_stable, blocking = gs.is_stable(women_optimal)
    print(f"   稳定性: {'✓ 稳定' if is_stable else '✗ 不稳定'}")

    # 比较两个匹配
    print("\n3. 两个匹配的比较:")
    comparison = gs.compare_matchings(men_optimal, women_optimal, "men")
    men_better = sum(1 for v in comparison.values() if v == "better")
    women_better = sum(1 for v in comparison.values() if v == "worse")
    print(f"   男性视角: {men_better}/{len(men)} 在男性最优匹配中更好")
    print(f"   女性视角: {women_better}/{len(women)} 在女性最优匹配中更好")
    print("   （男性提议的算法产生男性最优稳定匹配）")


def demo_ttc():
    """演示 Top Trading Cycles 算法"""

    print("\n" + "=" * 70)
    print("Top Trading Cycles (TTC) 算法演示")
    print("=" * 70)

    # 宿舍分配问题
    # 4 个学生，每人有一个初始宿舍，但想换更好的
    agents = [
        Agent("s1", ["d1", "d2", "d3", "d4"]),  # 已在 d1，但最喜欢 d1
        Agent("s2", ["d1", "d2", "d3", "d4"]),  # 已在 d2，但最喜欢 d1
        Agent("s3", ["d2", "d3", "d1", "d4"]),  # 已在 d3，但最喜欢 d2
        Agent("s4", ["d3", "d4", "d1", "d2"]),  # 已在 d4，但最喜欢 d3
    ]

    # 初始分配
    initial = {
        "s1": "d1",
        "s2": "d2",
        "s3": "d3",
        "s4": "d4",
    }

    print("\n初始分配:")
    for s, d in initial.items():
        print(f"   {s} -> {d}")

    ttc = TopTradingCycles(agents, initial)
    final = ttc.solve()

    print("\nTTC 最终分配:")
    for s, d in final.items():
        print(f"   {s} -> {d}")

    print("\n交换分析:")
    for s, d in final.items():
        init_d = initial[s]
        if d != init_d:
            print(f"   {s}: {init_d} -> {d} (交易)")
        else:
            print(f"   {s}: 保持 {d}")


if __name__ == "__main__":
    demo_gale_shapley()
    demo_ttc()
```
