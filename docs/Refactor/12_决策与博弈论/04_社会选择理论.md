## 4.1 社会福利函数

### 4.1.1 偏好聚合问题

**问题设定**: 如何将多个个体的偏好聚合为集体决策？

**基本要素**:

- N = {1,...,n}：投票者集合
- X = {x,y,z,...}：备选方案（|X| = m >= 3）
- ≽ᵢ：个体i的偏好关系

**定义 4.1.1 (社会福利函数 SWF)**: F: Rⁿ → R，将个体偏好映射到社会偏好

### 4.1.2 社会偏好的公理

| 公理 | 名称 | 内容 |
|:-----|:-----|:-----|
| U | 无限制域 | F定义在所有理性偏好组合上 |
| P | 帕累托效率 | 若所有人偏好x≻y，则社会偏好x≻ₛy |
| I | 独立性(IIA) | 社会对{x,y}的排序只依赖个体对{x,y}的排序 |
| D | 非独裁性 | 不存在i使得F总是复制i的偏好 |

### 4.1.3 社会福利函数形式

**功利主义**: W(x) = Σᵢ uᵢ(x)
**罗尔斯主义**: W(x) = minᵢ uᵢ(x)
**纳什社会福利**: W(x) = Πᵢ uᵢ(x)

## 4.2 投票规则

### 4.2.1 多数投票

**孔多塞胜者**: 在成对比较中击败所有其他选项的候选者

**孔多塞悖论**（循环偏好）:

```
投票者1: A ≻ B ≻ C
投票者2: B ≻ C ≻ A
投票者3: C ≻ A ≻ B

A≻B, B≻C, C≻A → 循环！
```

### 4.2.2 计分规则

**波达计数**: 排名k得(m-k)分

### 4.2.3 投票规则对比矩阵

| 规则 | 孔多塞一致 | 单调性 | IIA | 计算 |
|:-----|:----------:|:------:|:---:|:----:|
| 多数投票 | ✗ | ✓ | ✗ | 易 |
| 波达计数 | ✗ | ✓ | ✗ | 易 |
| IRV | ✗ | ✗ | ✗ | 中 |
| Schulze | ✓ | ✓ | ✗ | 中 |

## 4.3 阿罗不可能定理

### 4.3.1 定理陈述

**定理 4.3.1 (Arrow, 1951)**: 若|X| ≥ 3，不存在同时满足U、P、I、D、T的社会福利函数。

### 4.3.2 证明概要

1. 定义决定性集合J ⊆ N
2. 证明最小决定性集合存在且为单元素
3. 单元素决定性集合即独裁者
4. 矛盾！

### 4.3.3 变体定理

**Gibbard-Satterthwaite定理**: 策略证明的SCF必是独裁的
**Sen不可能定理**: 自由主义与帕累托效率冲突

## 4.4 不可能定理的规避

### 4.4.1 域限制：单峰偏好

**定理 4.4.1 (中位选民定理)**: 单峰偏好下，多数投票产生传递排序，中位理想点获胜。

### 4.4.2 其他规避策略

- 放松IIA → 波达计数
- 放松帕累托 → 公平性优先
- 随机化 → 随机独裁制
