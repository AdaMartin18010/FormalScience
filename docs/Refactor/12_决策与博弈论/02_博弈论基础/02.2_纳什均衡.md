## 12.2.2.1 引言

**纳什均衡**（Nash Equilibrium, NE）是博弈论最核心的解概念，由 John Nash (1950, 1951) 提出并证明存在性。
纳什均衡刻画了理性参与人在相互最优反应下的稳定状态：给定他人的策略选择，没有任何参与人可以通过单方面偏离来改善自己的支付。

---

## 12.2.2.2 最优反应与纳什均衡

### 12.2.2.2.1 最优反应

**定义 12.2.2.1**（最优反应对应）

参与人 $i$ 的**最优反应对应** $BR_i: \Delta(A_{-i}) \to 2^{\Delta(A_i)}$ 定义为：

$$BR_i(\sigma_{-i}) = \arg\max_{\sigma_i \in \Delta(A_i)} U_i(\sigma_i, \sigma_{-i})$$

**引理 12.2.2.1**（最优反应的特征）

纯策略 $a_i$ 是对 $\sigma_{-i}$ 的最优反应当且仅当：

$$a_i \in \arg\max_{a_i' \in A_i} U_i(a_i', \sigma_{-i})$$

### 12.2.2.2.2 纳什均衡的定义

**定义 12.2.2.2**（纳什均衡）

策略组合 $\sigma^* = (\sigma_1^*, ..., \sigma_n^*)$ 是**纳什均衡**，如果对所有 $i \in N$：

$$\sigma_i^* \in BR_i(\sigma_{-i}^*)$$

等价表述：对所有 $i$ 和所有 $a_i \in A_i$：

$$U_i(\sigma_i^*, \sigma_{-i}^*) \geq U_i(a_i, \sigma_{-i}^*)$$

### 12.2.2.2.3 最优反应函数

定义**最优反应函数** $br: \Delta(A) \to \Delta(A)$：

$$br(\sigma) = BR_1(\sigma_{-1}) \times ... \times BR_n(\sigma_{-n})$$

纳什均衡是最优反应函数的**不动点**：$\sigma^* \in br(\sigma^*)$。

---

## 12.2.2.3 纯策略纳什均衡

### 12.2.2.3.1 有限博弈的 NE

**囚徒困境**（回顾）：

|  | C | D |
|--|---|---|
| **C** | 3, 3 | 0, 5 |
| **D** | 5, 0 | 1, 1 |

- 给定 Player 2 选 C，Player 1 的最优反应是 D
- 给定 Player 2 选 D，Player 1 的最优反应是 D
- (D, D) 是纯策略纳什均衡

### 12.2.2.3.2 协调性博弈

**性别战**（Battle of the Sexes）：

|  | B | O |
|--|---|---|
| **B** | 2, 1 | 0, 0 |
| **O** | 0, 0 | 1, 2 |

- 两个纯策略 NE：(B, B) 和 (O, O)
- 多重均衡问题：参与人如何协调？

---

## 12.2.2.4 混合策略纳什均衡

### 12.2.2.4.1 无纯策略 NE 的博弈

**匹配硬币**（Matching Pennies）：

|  | H | T |
|--|---|---|
| **H** | 1, -1 | -1, 1 |
| **T** | -1, 1 | 1, -1 |

- 无纯策略 NE
- 混合策略 NE：双方以 1/2 概率随机化

### 12.2.2.4.2 混合策略均衡的求解

**定理 12.2.2.1**（无差异原理）

设 $\sigma^*$ 是混合策略纳什均衡，$a_i, a_i' \in \text{supp}(\sigma_i^*)$。则：

$$U_i(a_i, \sigma_{-i}^*) = U_i(a_i', \sigma_{-i}^*) = U_i(\sigma^*)$$

**证明**：若 $U_i(a_i, \sigma_{-i}^*) > U_i(a_i', \sigma_{-i}^*)$，则参与人 $i$ 会将概率从 $a_i'$ 转移到 $a_i$，偏离 $\sigma_i^*$，矛盾。∎

**求解匹配硬币**：

设 Player 1 以概率 $p$ 选 H，Player 2 以概率 $q$ 选 H。

Player 2 无差异条件：
$$U_2(H; p) = p( -1) + (1-p)(1) = 1 - 2p$$
$$U_2(T; p) = p(1) + (1-p)(-1) = 2p - 1$$

无差异要求 $1 - 2p = 2p - 1 \Rightarrow p = 1/2$。

同理 $q = 1/2$。混合策略 NE：$((1/2, 1/2), (1/2, 1/2))$。

---

## 12.2.2.5 纳什均衡存在性定理

### 12.2.2.5.1 Nash 定理

**定理 12.2.2.2**（Nash, 1950）

每个有限博弈（有限参与人，有限行动集）至少存在一个纳什均衡（可能是混合策略）。

### 12.2.2.5.2 证明

**证明**（基于 Kakutani 不动点定理）：

**步骤 1**：策略空间

$\Sigma = \prod_{i \in N} \Delta(A_i)$ 是非空、紧、凸欧氏空间子集。

**步骤 2**：最优反应对应

$br: \Sigma \to 2^\Sigma$ 定义为 $br(\sigma) = \prod_i BR_i(\sigma_{-i})$。

**步骤 3**：验证 Kakutani 条件

1. **非空性**：$BR_i(\sigma_{-i})$ 非空（支付函数连续，策略空间紧）
2. **凸性**：$BR_i(\sigma_{-i})$ 是凸集（支付函数对 $\sigma_i$ 线性）
3. **上半连续性**：支付函数连续 $\Rightarrow$ 最优反应对应上半连续

**步骤 4**：应用 Kakutani 不动点定理

存在 $\sigma^* \in br(\sigma^*)$，即纳什均衡。∎

---

## 12.2.2.6 均衡精炼

### 12.2.2.6.1 帕累托最优均衡

**定义 12.2.2.3**（帕累托最优）

纳什均衡 $\sigma$ **帕累托占优** $\sigma'$，如果对所有 $i$，$U_i(\sigma) \geq U_i(\sigma')$ 且至少有一个严格不等式。

### 12.2.2.6.2 风险占优

**定义 12.2.2.4**（风险占优，Harsanyi & Selten, 1988）

在 2×2 协调博弈中，均衡 $a$ **风险占优** $b$，如果 $&\u0026(a) &\u0026(b) > &\u0026(b) &\u0026(a)$，其中 $&\u0026$ 是偏离损失乘积。

## 12.2.2.7 Python 实现：纳什均衡计算

```python
"""
纳什均衡计算模块
支持纯策略和混合策略纳什均衡的求解
基于 Nash (1950, 1951)
"""

from typing import List, Tuple, Optional, Dict
import numpy as np
from scipy.optimize import minimize, differential_evolution
from itertools import product
import warnings
warnings.filterwarnings('ignore')


class NashEquilibriumSolver:
    """
    纳什均衡求解器
    支持二人博弈的均衡计算
    """

    def __init__(self, payoff_matrix: np.ndarray):
        """
        参数:
            payoff_matrix: shape (n_actions1, n_actions2, 2)
                          payoff_matrix[i,j] = (u1, u2)
        """
        self.payoff = payoff_matrix
        self.n1, self.n2 = payoff_matrix.shape[:2]

    def find_pure_strategy_ne(self) -> List[Tuple[int, int]]:
        """
        寻找所有纯策略纳什均衡
        返回: [(行动1, 行动2), ...]
        """
        equilibria = []

        for i in range(self.n1):
            for j in range(self.n2):
                # Player 1 的收益
                u1_ij = self.payoff[i, j, 0]
                # Player 2 的收益
                u2_ij = self.payoff[i, j, 1]

                # 检查 Player 1 是否最优反应
                is_best_response_1 = True
                for i_prime in range(self.n1):
                    if self.payoff[i_prime, j, 0] > u1_ij + 1e-10:
                        is_best_response_1 = False
                        break

                # 检查 Player 2 是否最优反应
                is_best_response_2 = True
                for j_prime in range(self.n2):
                    if self.payoff[i, j_prime, 1] > u2_ij + 1e-10:
                        is_best_response_2 = False
                        break

                if is_best_response_1 and is_best_response_2:
                    equilibria.append((i, j))

        return equilibria

    def expected_payoff(self, sigma1: np.ndarray, sigma2: np.ndarray, player: int) -> float:
        """
        计算期望支付
        player: 0 for Player 1, 1 for Player 2
        """
        if player == 0:
            # U1 = Σ_i Σ_j σ1[i] * σ2[j] * u1[i,j]
            return np.sum(sigma1[:, None] * sigma2[None, :] * self.payoff[:, :, 0])
        else:
            return np.sum(sigma1[:, None] * sigma2[None, :] * self.payoff[:, :, 1])

    def find_mixed_strategy_ne(self, method: str = 'replicator') -> List[Tuple[np.ndarray, np.ndarray]]:
        """
        寻找混合策略纳什均衡

        方法:
            'replicator': 复制者动态
            'optimization': 全局优化
            'support': 支持集枚举（小博弈）
        """
        if method == 'support':
            return self._support_enumeration()
        elif method == 'replicator':
            return self._replicator_dynamics()
        else:
            return self._optimization_method()

    def _support_enumeration(self) -> List[Tuple[np.ndarray, np.ndarray]]:
        """
        支持集枚举法
        遍历所有可能的支持集组合
        """
        equilibria = []

        # 遍历所有非空支持集
        for supp1 in self._all_non_empty_subsets(self.n1):
            for supp2 in self._all_non_empty_subsets(self.n2):
                # 尝试求解该支持集下的均衡
                eq = self._solve_for_support(supp1, supp2)
                if eq is not None:
                    sigma1, sigma2 = eq
                    # 验证是否为均衡
                    if self._verify_ne(sigma1, sigma2):
                        equilibria.append((sigma1, sigma2))

        # 去重
        unique_eq = []
        for eq in equilibria:
            is_duplicate = False
            for existing in unique_eq:
                if (np.allclose(eq[0], existing[0]) and
                    np.allclose(eq[1], existing[1])):
                    is_duplicate = True
                    break
            if not is_duplicate:
                unique_eq.append(eq)

        return unique_eq

    def _all_non_empty_subsets(self, n: int) -> List[Tuple[int, ...]]:
        """生成所有非空子集"""
        from itertools import combinations
        subsets = []
        for r in range(1, n + 1):
            subsets.extend(combinations(range(n), r))
        return subsets

    def _solve_for_support(self, supp1: Tuple[int, ...], supp2: Tuple[int, ...]) -> Optional[Tuple[np.ndarray, np.ndarray]]:
        """
        为给定支持集求解混合策略
        使用无差异原理
        """
        import numpy.linalg as la

        # 构造无差异方程组
        # Player 1: 支持集中所有行动对σ2无差异
        # Player 2: 支持集中所有行动对σ1无差异

        m, n = len(supp1), len(supp2)

        # Player 2 的策略 σ2 使 Player 1 在 supp1 上无差异
        if m >= 2:
            # 构造方程: u1[i, :] · σ2 = u1[i', :] · σ2 对所有 i, i' ∈ supp1
            A1 = np.zeros((m - 1, n))
            b1 = np.zeros(m - 1)
            for k in range(m - 1):
                i = supp1[k]
                i_next = supp1[k + 1]
                A1[k] = self.payoff[i, list(supp2), 0] - self.payoff[i_next, list(supp2), 0]
            # 添加概率和为1的约束
            A1_full = np.vstack([A1, np.ones(n)])
            b1_full = np.append(b1, 1)

            try:
                sigma2_support = la.solve(A1_full, b1_full)
                if np.any(sigma2_support < -1e-10):
                    return None
                sigma2_support = np.maximum(sigma2_support, 0)
                sigma2_support /= sigma2_support.sum()
            except la.LinAlgError:
                return None
        else:
            sigma2_support = np.ones(1)

        # Player 1 的策略 σ1 使 Player 2 在 supp2 上无差异
        if n >= 2:
            A2 = np.zeros((n - 1, m))
            b2 = np.zeros(n - 1)
            for k in range(n - 1):
                j = supp2[k]
                j_next = supp2[k + 1]
                A2[k] = self.payoff[list(supp1), j, 1] - self.payoff[list(supp1), j_next, 1]
            A2_full = np.vstack([A2, np.ones(m)])
            b2_full = np.append(b2, 1)

            try:
                sigma1_support = la.solve(A2_full, b2_full)
                if np.any(sigma1_support < -1e-10):
                    return None
                sigma1_support = np.maximum(sigma1_support, 0)
                sigma1_support /= sigma1_support.sum()
            except la.LinAlgError:
                return None
        else:
            sigma1_support = np.ones(1)

        # 构造完整策略向量
        sigma1 = np.zeros(self.n1)
        sigma2 = np.zeros(self.n2)
        for idx, i in enumerate(supp1):
            sigma1[i] = sigma1_support[idx]
        for idx, j in enumerate(supp2):
            sigma2[j] = sigma2_support[idx]

        return sigma1, sigma2

    def _verify_ne(self, sigma1: np.ndarray, sigma2: np.ndarray, tol: float = 1e-6) -> bool:
        """验证是否为纳什均衡"""
        # Player 1 的最优性
        u1_current = self.expected_payoff(sigma1, sigma2, 0)
        for i in range(self.n1):
            u1_deviate = self.expected_payoff(np.eye(self.n1)[i], sigma2, 0)
            if u1_deviate > u1_current + tol:
                return False

        # Player 2 的最优性
        u2_current = self.expected_payoff(sigma1, sigma2, 1)
        for j in range(self.n2):
            u2_deviate = self.expected_payoff(sigma1, np.eye(self.n2)[j], 1)
            if u2_deviate > u2_current + tol:
                return False

        return True

    def _replicator_dynamics(self, n_iter: int = 1000, n_runs: int = 10) -> List[Tuple[np.ndarray, np.ndarray]]:
        """
        复制者动态寻找均衡
        """
        equilibria = []

        for _ in range(n_runs):
            # 随机初始策略
            sigma1 = np.random.dirichlet(np.ones(self.n1))
            sigma2 = np.random.dirichlet(np.ones(self.n2))

            for _ in range(n_iter):
                # 计算期望支付
                u1 = np.array([self.expected_payoff(np.eye(self.n1)[i], sigma2, 0)
                              for i in range(self.n1)])
                u2 = np.array([self.expected_payoff(sigma1, np.eye(self.n2)[j], 1)
                              for j in range(self.n2)])

                # 复制者动态更新
                avg_u1 = np.dot(sigma1, u1)
                avg_u2 = np.dot(sigma2, u2)

                sigma1 = sigma1 * u1 / (avg_u1 + 1e-10)
                sigma2 = sigma2 * u2 / (avg_u2 + 1e-10)

                sigma1 /= sigma1.sum()
                sigma2 /= sigma2.sum()

            # 四舍五入到合理精度
            sigma1 = np.round(sigma1, 4)
            sigma2 = np.round(sigma2, 4)
            sigma1 /= sigma1.sum()
            sigma2 /= sigma2.sum()

            # 验证
            if self._verify_ne(sigma1, sigma2):
                # 检查是否已有相同均衡
                is_new = True
                for eq in equilibria:
                    if (np.allclose(sigma1, eq[0], atol=1e-3) and
                        np.allclose(sigma2, eq[1], atol=1e-3)):
                        is_new = False
                        break
                if is_new:
                    equilibria.append((sigma1, sigma2))

        return equilibria

    def _optimization_method(self) -> List[Tuple[np.ndarray, np.ndarray]]:
        """
        全局优化方法（适用于小博弈）
        """
        # 使用罚函数法
        def objective(x):
            """最小化偏离激励"""
            sigma1 = x[:self.n1]
            sigma2 = x[self.n1:]

            # 归一化
            sigma1 = np.maximum(sigma1, 0)
            sigma2 = np.maximum(sigma2, 0)
            sigma1 /= sigma1.sum() + 1e-10
            sigma2 /= sigma2.sum() + 1e-10

            # 计算最大偏离收益
            u1_current = self.expected_payoff(sigma1, sigma2, 0)
            u2_current = self.expected_payoff(sigma1, sigma2, 1)

            max_gain1 = max(self.expected_payoff(np.eye(self.n1)[i], sigma2, 0) - u1_current
                           for i in range(self.n1))
            max_gain2 = max(self.expected_payoff(sigma1, np.eye(self.n2)[j], 1) - u2_current
                           for j in range(self.n2))

            return max_gain1**2 + max_gain2**2

        bounds = [(0, 1)] * (self.n1 + self.n2)
        result = differential_evolution(objective, bounds, seed=42, maxiter=100)

        if result.fun < 1e-6:
            sigma1 = result.x[:self.n1]
            sigma2 = result.x[self.n1:]
            sigma1 /= sigma1.sum()
            sigma2 /= sigma2.sum()
            return [(sigma1, sigma2)]

        return []


# 经典博弈示例
def matching_pennies() -> np.ndarray:
    """匹配硬币博弈"""
    return np.array([
        [[1, -1], [-1, 1]],
        [[-1, 1], [1, -1]]
    ])


def battle_of_sexes() -> np.ndarray:
    """性别战博弈"""
    return np.array([
        [[2, 1], [0, 0]],
        [[0, 0], [1, 2]]
    ])


def prisoners_dilemma() -> np.ndarray:
    """囚徒困境"""
    return np.array([
        [[3, 3], [0, 5]],
        [[5, 0], [1, 1]]
    ])


def chicken_game() -> np.ndarray:
    """胆小鬼博弈"""
    return np.array([
        [[0, 0], [-1, 1]],
        [[1, -1], [-10, -10]]
    ])


def demo():
    """演示纳什均衡求解"""

    print("=" * 70)
    print("纳什均衡求解演示")
    print("=" * 70)

    games = [
        ("囚徒困境", prisoners_dilemma(), ["合作", "背叛"]),
        ("性别战", battle_of_sexes(), ["拳击", "歌剧"]),
        ("匹配硬币", matching_pennies(), ["正面", "反面"]),
        ("胆小鬼博弈", chicken_game(), ["直行", "避让"]),
    ]

    for name, payoff, actions in games:
        print(f"\n{name}")
        print("-" * 50)

        solver = NashEquilibriumSolver(payoff)

        # 纯策略 NE
        pure_ne = solver.find_pure_strategy_ne()
        print(f"纯策略纳什均衡: {len(pure_ne)} 个")
        for i, j in pure_ne:
            print(f"  ({actions[i]}, {actions[j]}): ({payoff[i,j,0]}, {payoff[i,j,1]})")

        # 混合策略 NE
        mixed_ne = solver.find_mixed_strategy_ne(method='support')
        print(f"\n混合策略纳什均衡: {len(mixed_ne)} 个")
        for sigma1, sigma2 in mixed_ne:
            s1_str = ", ".join([f"{a}:{p:.3f}" for a, p in zip(actions, sigma1)])
            s2_str = ", ".join([f"{a}:{p:.3f}" for a, p in zip(actions, sigma2)])
            u1 = solver.expected_payoff(sigma1, sigma2, 0)
            u2 = solver.expected_payoff(sigma1, sigma2, 1)
            print(f"  Player 1: [{s1_str}]")
            print(f"  Player 2: [{s2_str}]")
            print(f"  期望支付: ({u1:.3f}, {u2:.3f})")


if __name__ == "__main__":
    demo()
```
