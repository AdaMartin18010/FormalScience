## 12.2.1.1 引言

博弈论研究多个理性决策者（**参与人**）在策略互动情境中的选择问题。von Neumann 和 Morgenstern (1944) 奠定了博弈论的数学基础。博弈可以用三种基本形式表示：**标准式**（normal form）、**扩展式**（extensive form）和**特征函数式**（coalitional form）。

---

## 12.2.1.2 标准式博弈

### 12.2.1.2.1 定义

**定义 12.2.1.1**（标准式博弈）

一个**标准式博弈**（或策略式博弈）是三元组 $G = \langle N, (A_i)_{i \in N}, (u_i)_{i \in N} \rangle$，其中：

- $N = \{1, 2, ..., n\}$：**参与人集合**
- $A_i$：参与人 $i$ 的**行动集**（action set）
- $A = \prod_{i \in N} A_i$：**行动组合空间**
- $u_i: A \to \mathbb{R}$：参与人 $i$ 的**支付函数**（payoff function）

**纯策略**就是选择行动 $a_i \in A_i$。

### 12.2.1.2.2 博弈矩阵表示

**二人有限博弈**可用**支付矩阵**表示：

**囚徒困境**（Prisoner's Dilemma）：

|         | 合作 (C) | 背叛 (D) |
|---------|----------|----------|
| **合作 (C)** | 3, 3     | 0, 5     |
| **背叛 (D)** | 5, 0     | 1, 1     |

行参与人（Player 1）的收益在前，列参与人（Player 2）的收益在后。

### 12.2.1.2.3 占优策略

**定义 12.2.1.2**（严格占优）

行动 $a_i^*$ **严格占优** $a_i'$，如果对所有 $a_{-i} \in A_{-i}$：

$$u_i(a_i^*, a_{-i}) > u_i(a_i', a_{-i})$$

**定义 12.2.1.3**（占优策略均衡）

行动组合 $a^* = (a_1^*, ..., a_n^*)$ 是**占优策略均衡**，如果每个 $a_i^*$ 都是参与人 $i$ 的占优策略。

---

## 12.2.1.3 扩展式博弈

### 12.2.1.3.1 博弈树结构

**定义 12.2.1.4**（扩展式博弈）

一个**扩展式博弈**是七元组 $\Gamma = \langle N, H, P, (A_i), (u_i), \rho, I \rangle$：

- $N$：参与人集合
- $H$：**历史**（histories）集合，即有穷行动序列，包括空历史 $\emptyset$
- $P: H \setminus Z \to N \cup \{c\}$：**参与人函数**，$Z$ 为终历史集合，$c$ 表示自然（机会）
- $A(h) = \{a: (h, a) \in H\}$：历史 $h$ 后的可行行动集
- $u_i: Z \to \mathbb{R}$：参与人 $i$ 的支付函数
- $\rho$：自然的历史依赖概率分布
- $I = (I_i)_{i \in N}$：**信息划分**，$I_i$ 是参与人 $i$ 的信息集族

### 12.2.1.3.2 信息集

**定义 12.2.1.5**（信息集）

**信息集** $I \in \mathcal{I}_i$ 是参与人 $i$ 无法区分的历史集合。形式化要求：

- 对所有 $h, h' \in I$，$A(h) = A(h')$（相同可行行动）
- $i$ 在 $I$ 中所有历史上行动

**完美信息**：所有信息集都是单元素集。

**完美回忆**：参与人不会忘记自己已知的信息。

### 12.2.1.3.3 扩展式示例

**进入博弈**（Entry Game）：

```
            [1: 潜在进入者]
           /           \
     进入 /             \ 不进入
         /               \
    [2: 在位者]        (0, 2)
   /          \
对抗/          \容纳
   |             |
(-1, -1)      (1, 1)
```

## 12.2.1.4 混合策略

### 12.2.1.4.1 定义

**定义 12.2.1.6**（混合策略）

参与人 $i$ 的**混合策略**是 $A_i$ 上的概率分布，记作 $\sigma_i \in \Delta(A_i)$。

**混合策略组合** $\sigma = (\sigma_1, ..., \sigma_n)$ 给出行动组合上的乘积分布。

**期望支付**：

$$U_i(\sigma) = \sum_{a \in A} \left(\prod_{j \in N} \sigma_j(a_j)\right) u_i(a)$$

### 12.2.1.4.2 支持集

**定义 12.2.1.7**（支持集）

混合策略 $\sigma_i$ 的**支持集**：

$$\text{supp}(\sigma_i) = \{a_i \in A_i: \sigma_i(a_i) > 0\}$$

## 12.2.1.5 特征函数式（合作博弈）

### 12.2.1.5.1 定义

**定义 12.2.1.8**（特征函数式博弈）

一个**特征函数式博弈**（合作博弈）是二元组 $\langle N, v \rangle$，其中：

- $N = \{1, ..., n\}$：参与人集合
- $v: 2^N \to \mathbb{R}$：**特征函数**，满足 $v(\emptyset) = 0$

$v(S)$ 表示联盟 $S \subseteq N$ 能保证获得的最大总支付。

### 12.2.1.5.2 超可加性

**定义 12.2.1.9**（超可加性）

特征函数 $v$ 是**超可加的**（superadditive），如果对任意不相交的 $S, T \subseteq N$：

$$v(S \cup T) \geq v(S) + v(T)$$

**解释**：合并的联盟至少能获得分开时的总和。

### 12.2.1.5.3 博弈类型

| 类型 | 定义 | 示例 |
|------|------|------|
| **本质博弈** | $v(N) > \sum_{i \in N} v(\{i\})$ | 有合作剩余 |
| **常和博弈** | $v(S) + v(N \setminus S) = v(N)$ | 零和博弈 |
| **凸博弈** | $v(S) + v(T) \leq v(S \cup T) + v(S \cap T)$ | 联盟边际收益递增 |
| **简单博弈** | $v(S) \in \{0, 1\}$ | 投票博弈 |

---

## 12.2.1.6 博弈分类

### 12.2.1.6.1 按信息结构

| 类型 | 特征 |
|------|------|
| **完全信息** | 所有参与人知道博弈结构 |
| **不完全信息** | 某些参与人不知道某些特征（Harsanyi 转换） |
| **完美信息** | 所有信息集单元素（见扩展式） |
| **不完美信息** | 存在非单元素信息集 |

### 12.2.1.6.2 按时间结构

| 类型 | 特征 |
|------|------|
| **静态博弈** | 所有参与人同时行动 |
| **动态博弈** | 参与人按顺序行动 |
| **重复博弈** | 阶段博弈重复进行 |

## 12.2.1.7 Python 实现：博弈表示与转换

```python
"""
博弈论表示模块
实现标准式、扩展式、特征函数式的表示与转换
基于 Osborne & Rubinstein (1994) 框架
"""

from typing import List, Dict, Set, Tuple, Optional, Callable
from dataclasses import dataclass, field
from enum import Enum
import numpy as np
from itertools import product


# ==================== 标准式博弈 ====================

@dataclass
class NormalFormGame:
    """
    标准式博弈 G = <N, (A_i), (u_i)>
    """
    players: List[str]                      # 参与人名称
    action_sets: Dict[str, List]            # 各参与人的行动集
    payoff_matrix: Dict[Tuple, Tuple]       # {(a1, a2, ...): (u1, u2, ...)}

    def __post_init__(self):
        self.n_players = len(self.players)
        # 验证支付矩阵完整性
        expected_size = 1
        for p in self.players:
            expected_size *= len(self.action_sets[p])
        assert len(self.payoff_matrix) == expected_size, "支付矩阵不完整"

    def get_payoff(self, action_profile: Tuple, player: str) -> float:
        """获取特定行动组合下某参与人的支付"""
        idx = self.players.index(player)
        return self.payoff_matrix[action_profile][idx]

    def get_action_profiles(self) -> List[Tuple]:
        """获取所有行动组合"""
        action_lists = [self.action_sets[p] for p in self.players]
        return list(product(*action_lists))

    def is_strictly_dominated(self, player: str, action) -> Tuple[bool, Optional]:
        """
        检查某行动是否被严格占优
        返回: (是否被占优, 占优行动)
        """
        player_idx = self.players.index(player)
        other_players = [p for p in self.players if p != player]

        for candidate in self.action_sets[player]:
            if candidate == action:
                continue

            is_dominated = True
            for profile in self.get_action_profiles():
                if profile[player_idx] != action:
                    continue

                # 构造对比组合
                alt_profile = list(profile)
                alt_profile[player_idx] = candidate
                alt_profile = tuple(alt_profile)

                # 检查是否对所有他人行动都更优
                if self.payoff_matrix[profile][player_idx] >= \
                   self.payoff_matrix[alt_profile][player_idx]:
                    is_dominated = False
                    break

            if is_dominated:
                return True, candidate

        return False, None

    def find_dominant_strategy_equilibrium(self) -> Optional[Tuple]:
        """寻找占优策略均衡（如果存在）"""
        dominant_actions = []

        for player in self.players:
            found = False
            for action in self.action_sets[player]:
                # 检查是否占优所有其他行动
                is_dominant = True
                for other_action in self.action_sets[player]:
                    if other_action == action:
                        continue

                    # 检查是否对所有对手行动都更好
                    for profile in self.get_action_profiles():
                        p_idx = self.players.index(player)
                        if profile[p_idx] != action:
                            continue

                        alt_profile = list(profile)
                        alt_profile[p_idx] = other_action
                        alt_profile = tuple(alt_profile)

                        if self.payoff_matrix[profile][p_idx] <= \
                           self.payoff_matrix[alt_profile][p_idx]:
                            is_dominant = False
                            break

                    if not is_dominant:
                        break

                if is_dominant:
                    dominant_actions.append(action)
                    found = True
                    break

            if not found:
                return None

        return tuple(dominant_actions)

    def to_matrix_form(self) -> np.ndarray:
        """
        转换为矩阵形式（仅适用于二人博弈）
        返回: (n_actions1, n_actions2, 2) 的数组
        """
        assert self.n_players == 2, "仅支持二人博弈"

        n1 = len(self.action_sets[self.players[0]])
        n2 = len(self.action_sets[self.players[1]])
        matrix = np.zeros((n1, n2, 2))

        for i, a1 in enumerate(self.action_sets[self.players[0]]):
            for j, a2 in enumerate(self.action_sets[self.players[1]]):
                matrix[i, j] = self.payoff_matrix[(a1, a2)]

        return matrix


# ==================== 扩展式博弈 ====================

class NodeType(Enum):
    DECISION = "decision"
    CHANCE = "chance"
    TERMINAL = "terminal"


@dataclass
class ExtensiveFormNode:
    """扩展式博弈的节点"""
    id: str
    node_type: NodeType
    player: Optional[str] = None  # 决策节点：哪个参与人行动

    # 子节点: {行动: 子节点ID}
    children: Dict = field(default_factory=dict)

    # 机会节点的概率分布: {子节点ID: 概率}
    probabilities: Dict[str, float] = field(default_factory=dict)

    # 终节点支付: {参与人: 支付}
    payoffs: Dict[str, float] = field(default_factory=dict)

    # 信息集
    information_set: Optional[str] = None


@dataclass
class ExtensiveFormGame:
    """
    扩展式博弈
    """
    players: List[str]
    root: ExtensiveFormNode
    nodes: Dict[str, ExtensiveFormNode] = field(default_factory=dict)

    def __post_init__(self):
        self._build_node_dict(self.root)

    def _build_node_dict(self, node: ExtensiveFormNode):
        self.nodes[node.id] = node
        for child in node.children.values():
            if isinstance(child, ExtensiveFormNode) and child.id not in self.nodes:
                self._build_node_dict(child)

    def is_perfect_information(self) -> bool:
        """检查是否为完美信息博弈"""
        for node in self.nodes.values():
            if node.node_type == NodeType.DECISION:
                # 检查信息集（简化：假设单节点信息集为完美信息）
                if node.information_set is not None:
                    return False
        return True

    def get_strategies(self, player: str) -> List[Dict]:
        """
        获取参与人的纯策略
        策略是从信息集到行动的映射
        """
        # 找到该参与人的所有信息集
        info_sets = {}
        for node in self.nodes.values():
            if node.node_type == NodeType.DECISION and node.player == player:
                info_set = node.information_set or node.id
                if info_set not in info_sets:
                    info_sets[info_set] = []
                info_sets[info_set].append(node)

        # 每个信息集的可行行动
        actions_per_info = {}
        for info_id, nodes in info_sets.items():
            actions = set()
            for node in nodes:
                actions.update(node.children.keys())
            actions_per_info[info_id] = list(actions)

        # 生成所有策略组合
        info_ids = list(actions_per_info.keys())
        if not info_ids:
            return [{}]

        action_combinations = product(*[actions_per_info[i] for i in info_ids])
        strategies = []
        for combo in action_combinations:
            strategy = dict(zip(info_ids, combo))
            strategies.append(strategy)

        return strategies


# ==================== 特征函数式博弈 ====================

@dataclass
class CoalitionalGame:
    """
    特征函数式博弈（合作博弈）
    <N, v> 其中 v: 2^N -> R
    """
    players: List[str]

    # 特征函数: {frozenset(联盟): 价值}
    characteristic_function: Dict[frozenset, float]

    def __post_init__(self):
        # 验证 v(empty) = 0
        empty_set = frozenset()
        if empty_set in self.characteristic_function:
            assert self.characteristic_function[empty_set] == 0, "v(∅) 必须等于 0"

    def get_value(self, coalition: Set[str]) -> float:
        """获取联盟的价值"""
        return self.characteristic_function.get(frozenset(coalition), 0)

    def is_superadditive(self) -> bool:
        """检查超可加性"""
        from itertools import combinations

        all_players = set(self.players)

        for r in range(1, len(self.players)):
            for S in combinations(self.players, r):
                S = set(S)
                remaining = all_players - S

                for T in combinations(remaining, len(remaining)):
                    T = set(T)

                    v_S = self.get_value(S)
                    v_T = self.get_value(T)
                    v_union = self.get_value(S | T)

                    if v_union < v_S + v_T - 1e-10:
                        return False

        return True

    def is_convex(self) -> bool:
        """检查凸性: v(S) + v(T) <= v(S∪T) + v(S∩T)"""
        from itertools import combinations

        for r1 in range(len(self.players) + 1):
            for S in combinations(self.players, r1):
                S = set(S)
                for r2 in range(len(self.players) + 1):
                    for T in combinations(self.players, r2):
                        T = set(T)

                        v_S = self.get_value(S)
                        v_T = self.get_value(T)
                        v_union = self.get_value(S | T)
                        v_inter = self.get_value(S & T)

                        if v_S + v_T > v_union + v_inter + 1e-10:
                            return False

        return True

    def is_essential(self) -> bool:
        """检查是否为本质博弈"""
        grand_coalition = set(self.players)
        v_N = self.get_value(grand_coalition)
        v_individual = sum(self.get_value({p}) for p in self.players)
        return v_N > v_individual

    def get_all_coalitions(self) -> List[Set[str]]:
        """获取所有可能的联盟"""
        from itertools import combinations

        coalitions = []
        for r in range(len(self.players) + 1):
            for combo in combinations(self.players, r):
                coalitions.append(set(combo))
        return coalitions


# ==================== 辅助函数 ====================

def prisoners_dilemma() -> NormalFormGame:
    """创建囚徒困境博弈"""
    players = ["Player 1", "Player 2"]
    actions = {
        "Player 1": ["Cooperate", "Defect"],
        "Player 2": ["Cooperate", "Defect"]
    }
    payoffs = {
        ("Cooperate", "Cooperate"): (3, 3),
        ("Cooperate", "Defect"): (0, 5),
        ("Defect", "Cooperate"): (5, 0),
        ("Defect", "Defect"): (1, 1)
    }
    return NormalFormGame(players, actions, payoffs)


def battle_of_sexes() -> NormalFormGame:
    """创建性别战博弈"""
    players = ["Husband", "Wife"]
    actions = {
        "Husband": ["Boxing", "Opera"],
        "Wife": ["Boxing", "Opera"]
    }
    payoffs = {
        ("Boxing", "Boxing"): (2, 1),
        ("Boxing", "Opera"): (0, 0),
        ("Opera", "Boxing"): (0, 0),
        ("Opera", "Opera"): (1, 2)
    }
    return NormalFormGame(players, actions, payoffs)


def glove_game() -> CoalitionalGame:
    """
    手套博弈（经典合作博弈）
    三个参与人：两人有左手套，一人有右手套
    一对配套手套价值 1
    """
    players = ["Player 1 (Left)", "Player 2 (Left)", "Player 3 (Right)"]

    def v(coalition: Set[str]) -> float:
        has_left = any("Left" in p for p in coalition)
        has_right = any("Right" in p for p in coalition)
        if has_left and has_right:
            return 1.0
        return 0.0

    char_func = {}
    from itertools import combinations
    for r in range(len(players) + 1):
        for combo in combinations(players, r):
            char_func[frozenset(combo)] = v(set(combo))

    return CoalitionalGame(players, char_func)


# 示例演示
def demo():
    """演示博弈表示"""

    print("=" * 60)
    print("博弈表示演示")
    print("=" * 60)

    # 囚徒困境
    print("\n1. 囚徒困境（标准式）")
    print("-" * 40)
    pd = prisoners_dilemma()
    print(f"参与人: {pd.players}")
    print(f"行动集: {pd.action_sets}")

    print("\n支付矩阵:")
    matrix = pd.to_matrix_form()
    for i, a1 in enumerate(pd.action_sets["Player 1"]):
        for j, a2 in enumerate(pd.action_sets["Player 2"]):
            print(f"  ({a1}, {a2}): ({matrix[i,j,0]}, {matrix[i,j,1]})")

    # 占优检验
    dominated, dominator = pd.is_strictly_dominated("Player 1", "Cooperate")
    if dominated:
        print(f"\nPlayer 1 的 'Cooperate' 被 '{dominator}' 严格占优")

    # 性别战
    print("\n2. 性别战博弈")
    print("-" * 40)
    bos = battle_of_sexes()
    print("支付矩阵 (Husband, Wife):")
    for profile in bos.get_action_profiles():
        print(f"  {profile}: {bos.payoff_matrix[profile]}")

    # 检查占优策略均衡
    dse = bos.find_dominant_strategy_equilibrium()
    print(f"\n占优策略均衡: {dse}")

    # 手套博弈
    print("\n3. 手套博弈（合作博弈）")
    print("-" * 40)
    glove = glove_game()
    print(f"参与人: {glove.players}")

    print("\n特征函数值:")
    for coalition in glove.get_all_coalitions():
        if coalition:
            members = ", ".join(sorted(coalition))
            print(f"  v({{{members}}}) = {glove.get_value(coalition)}")

    print(f"\n超可加性: {glove.is_superadditive()}")
    print(f"凸性: {glove.is_convex()}")
    print(f"本质博弈: {glove.is_essential()}")


if __name__ == "__main__":
    demo()
```
