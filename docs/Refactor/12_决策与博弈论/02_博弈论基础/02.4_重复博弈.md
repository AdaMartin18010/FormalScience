## 12.2.4.1 引言

**重复博弈**（repeated games）研究阶段博弈（stage game）在时间上重复进行的情形。
重复互动为合作提供了可能性，即使阶段博弈的唯一均衡是非合作的。
本章讨论有限和无限重复博弈的均衡结构，以及**民间定理**（Folk Theorem）刻画的可行均衡支付集。

---

## 12.2.4.2 重复博弈的基本概念

### 12.2.4.2.1 阶段博弈

**定义 12.2.4.1**（阶段博弈）

**阶段博弈** $G = \langle N, (A_i), (u_i) \rangle$ 是在每期重复的标准式博弈。

### 12.2.4.2.2 重复博弈的表示

**定义 12.2.4.2**（$T$ 期重复博弈）

$T$ 期重复博弈 $G(T)$ 由以下要素构成：

- **历史**：$h^t = (a^1, ..., a^{t-1})$，前 $t-1$ 期的行动组合序列
- **策略**：$s_i: H \to \Delta(A_i)$，从所有可能历史到混合行动的映射
- **支付**：$U_i = \sum_{t=1}^T \delta^{t-1} u_i(a^t)$（有限情形 $\delta = 1$）

---

## 12.2.4.3 有限重复博弈

### 12.2.4.3.1 唯一均衡阶段博弈

**定理 12.2.4.1**（有限重复的唯一性）

若阶段博弈 $G$ 有唯一纳什均衡 $a^*$，则对任意有限 $T$，$G(T)$ 有唯一子博弈精炼均衡：每期都玩 $a^*$。

**证明**（逆向归纳）：

- 第 $T$ 期：唯一 NE 是 $a^*$
- 第 $T-1$ 期：最后一期必定是 $a^*$，因此当前期也是 $a^*$
- 归纳得证 ∎

### 12.2.4.3.2 多重均衡阶段博弈

若 $G$ 有多个 NE，则可能通过**触发策略**维持非均衡路径上的合作。

**示例**：阶段博弈有两个 NE

- $NE_1$：高效支付 (3, 3)
- $NE_2$：惩罚支付 (0, 0)

**触发策略**：

- 正常路径：每期都选 (C, C)，获得 (4, 4)
- 偏离惩罚：若有人偏离，永久切换到 $NE_2$

---

## 12.2.4.4 无限重复博弈

### 12.2.4.4.1 贴现与平均支付

**定义 12.2.4.3**（贴现支付）

对贴现因子 $\delta \in (0,1)$，支付流 $(u_i^1, u_i^2, ...)$ 的**贴现和**为：

$$U_i = (1-\delta) \sum_{t=1}^\infty \delta^{t-1} u_i^t$$

标准化因子 $(1-\delta)$ 使得支付与阶段博弈可比。

### 12.2.4.4.2 无名氏定理（Folk Theorem）

**定义 12.2.4.4**（可行支付集）

**可行支付集** $V$ 是阶段博弈支付向量的凸包：

$$V = \text{co}\{u(a): a \in A\}$$

**定义 12.2.4.5**（最小最大支付）

参与人 $i$ 的**最小最大支付**（minmax value）：

$$\underline{v}_i = \min_{\sigma_{-i}} \max_{\sigma_i} u_i(\sigma_i, \sigma_{-i})$$

**定理 12.2.4.2**（纳什民间定理）

设 $v \in V$ 且 $v_i > \underline{v}_i$ 对所有 $i$。则存在 $\underline{\delta} < 1$，对所有 $\delta > \underline{\delta}$，存在子博弈精炼均衡实现支付 $v$。

**证明思路**：

- **策略构造**：
  - 正常路径：按实现 $v$ 的行动组合行动
  - 惩罚阶段：若 $i$ 偏离，其他参与人最小最大化 $i$（持续有限期）
  - 恢复阶段：惩罚后回到正常路径

- **激励相容**：
  偏离的短期收益 < 惩罚的长期损失

  当 $\delta$ 足够接近 1 时成立 ∎

### 12.2.4.4.3 子博弈精炼民间定理

**定理 12.2.4.3**（Fudenberg & Maskin, 1986）

对两人博弈，纳什民间定理的支付向量可通过 SPNE 实现（可能需使用更复杂的惩罚策略）。

---

## 12.2.4.5 触发策略

### 12.2.4.5.1 冷酷触发策略

**定义 12.2.4.6**（冷酷触发，Grim Trigger）

- 开始：每期都选合作行动 $a^C$
- 触发：若任何参与人曾偏离，永久切换到最小最大策略

**定理 12.2.4.4**（冷酷触发的 SPNE 条件）

冷酷触发构成 SPNE 当且仅当：

$$u_i^D - u_i^C \leq \frac{\delta}{1-\delta}(u_i^C - \underline{v}_i)$$

其中 $u_i^D$ 是最优偏离支付，$u_i^C$ 是合作支付。

### 12.2.4.5.2 以牙还牙策略

**定义 12.2.4.7**（Tit-for-Tat）

- 第 1 期：合作
- 第 $t$ 期：选择对手在第 $t-1$ 期的行动

**特征**：

- 宽容：对手重新合作后恢复合作
- 激励：偏离一期后对手在下一期惩罚

---

## 12.2.4.6 囚徒困境的重复博弈

### 12.2.4.6.1 阶段博弈

|  | C | D |
|--|---|---|
| **C** | 3, 3 | 0, 5 |
| **D** | 5, 0 | 1, 1 |

- 唯一 NE：(D, D)，支付 (1, 1)
- 合作最优：(C, C)，支付 (3, 3)
- 最小最大支付：$\underline{v}_i = 1$

### 12.2.4.6.2 合作的条件

**冷酷触发维持 (C, C) 的条件**：

偏离（一期得 5，之后永远得 1）vs 合作（永远得 3）：

$$(1-\delta) \cdot 5 + \delta \cdot 1 \leq 3$$

$$5 - 5\delta + \delta \leq 3$$

$$2 \leq 4\delta$$

$$\delta \geq \frac{1}{2}$$

## 12.2.4.7 Python 实现：重复博弈分析

```python
"""
重复博弈分析模块
实现有限/无限重复博弈、触发策略、民间定理
基于 Friedman (1971) 和 Fudenberg & Maskin (1986)
"""

from typing import List, Tuple, Dict, Callable
import numpy as np
from scipy.optimize import linprog
import matplotlib.pyplot as plt
from itertools import product


class StageGame:
    """阶段博弈"""

    def __init__(self, payoff_matrix: np.ndarray):
        """
        payoff_matrix: (n_actions1, n_actions2, 2)
        """
        self.payoff = payoff_matrix
        self.n1, self.n2 = payoff_matrix.shape[:2]
        self.n_players = 2

    def get_payoff(self, action1: int, action2: int, player: int) -> float:
        return self.payoff[action1, action2, player]

    def find_nash_equilibria(self) -> List[Tuple[int, int]]:
        """寻找纯策略纳什均衡"""
        ne = []
        for i in range(self.n1):
            for j in range(self.n2):
                u1 = self.payoff[i, j, 0]
                u2 = self.payoff[i, j, 1]

                # Player 1 的最优反应
                is_br1 = all(self.payoff[k, j, 0] <= u1 for k in range(self.n1))
                # Player 2 的最优反应
                is_br2 = all(self.payoff[i, k, 1] <= u2 for k in range(self.n2))

                if is_br1 and is_br2:
                    ne.append((i, j))
        return ne

    def compute_minmax(self, player: int) -> Tuple[float, np.ndarray, np.ndarray]:
        """
        计算参与人的最小最大支付
        返回: (minmax_value, opponent_strategy, player_best_response)
        """
        # 其他参与人试图最小化该参与人的最大支付
        opponent = 1 - player

        # 对每个对手策略，计算参与人的最大支付
        if player == 0:
            # 对手选择列策略
            min_val = float('inf')
            best_opp_strat = None
            best_player_br = None

            for j in range(self.n2):
                max_payoff_for_i = max(self.payoff[i, j, 0] for i in range(self.n1))
                if max_payoff_for_i < min_val:
                    min_val = max_payoff_for_i
                    best_opp_strat = np.eye(self.n2)[j]
                    best_player_br = np.eye(self.n1)[np.argmax([self.payoff[i, j, 0] for i in range(self.n1)])]
        else:
            min_val = float('inf')
            best_opp_strat = None
            best_player_br = None

            for i in range(self.n1):
                max_payoff_for_j = max(self.payoff[i, j, 1] for j in range(self.n2))
                if max_payoff_for_j < min_val:
                    min_val = max_payoff_for_j
                    best_opp_strat = np.eye(self.n1)[i]
                    best_player_br = np.eye(self.n2)[np.argmax([self.payoff[i, j, 1] for j in range(self.n2)])]

        return min_val, best_opp_strat, best_player_br


class RepeatedGame:
    """重复博弈"""

    def __init__(self, stage_game: StageGame, delta: float):
        """
        参数:
            stage_game: 阶段博弈
            delta: 贴现因子
        """
        self.stage = stage_game
        self.delta = delta

    def grim_trigger_sustainable(self,
                                  coop_payoff: Tuple[float, float],
                                  deviate_payoff: Tuple[float, float],
                                  minmax_payoff: Tuple[float, float]) -> bool:
        """
        检验冷酷触发策略是否可持续
        条件: (1-δ)*u_d + δ*u_m <= u_c
        """
        for i in range(2):
            lhs = (1 - self.delta) * deviate_payoff[i] + self.delta * minmax_payoff[i]
            rhs = coop_payoff[i]

            if lhs > rhs + 1e-10:
                return False

        return True

    def compute_critical_delta(self,
                                coop_payoff: Tuple[float, float],
                                deviate_payoff: Tuple[float, float],
                                minmax_payoff: Tuple[float, float]) -> float:
        """
        计算维持合作所需的最小贴现因子
        """
        critical_deltas = []

        for i in range(2):
            # (1-δ)*u_d + δ*u_m <= u_c
            # u_d - δ*u_d + δ*u_m <= u_c
            # u_d - u_c <= δ*(u_d - u_m)

            if deviate_payoff[i] - minmax_payoff[i] > 1e-10:
                delta_min = (deviate_payoff[i] - coop_payoff[i]) / (deviate_payoff[i] - minmax_payoff[i])
                critical_deltas.append(max(0, min(1, delta_min)))

        return max(critical_deltas) if critical_deltas else 0.0

    def feasible_payoff_set(self, n_samples: int = 100) -> np.ndarray:
        """
        计算可行支付集的近似
        阶段博弈支付的凸包
        """
        payoffs = []
        for i in range(self.stage.n1):
            for j in range(self.stage.n2):
                payoffs.append(self.stage.payoff[i, j])

        payoffs = np.array(payoffs)

        # 生成凸组合
        convex_combinations = []
        for _ in range(n_samples):
            weights = np.random.dirichlet(np.ones(len(payoffs)))
            convex_combinations.append(np.dot(weights, payoffs))

        return np.array(convex_combinations)


class PrisonersDilemma(StageGame):
    """囚徒困境"""

    def __init__(self):
        # C=0, D=1
        # (C,C)=3, (C,D)=0, (D,C)=5, (D,D)=1
        payoff = np.array([
            [[3, 3], [0, 5]],   # C
            [[5, 0], [1, 1]]    # D
        ])
        super().__init__(payoff)
        self.action_names = ["Cooperate", "Defect"]

    def get_analysis(self) -> Dict:
        """完整的博弈分析"""
        analysis = {
            'stage_game_ne': self.find_nash_equilibria(),
            'cooperative_outcome': (0, 0),  # (C, C)
            'cooperative_payoff': (3, 3),
            'ne_payoff': (1, 1)
        }

        # 最小最大支付
        mm0, _, _ = self.compute_minmax(0)
        mm1, _, _ = self.compute_minmax(1)
        analysis['minmax'] = (mm0, mm1)

        # 最优偏离
        analysis['deviate_from_coop'] = (5, 5)  # 从(C,C)偏离到D得5

        return analysis


class FolkTheoremAnalyzer:
    """民间定理分析器"""

    def __init__(self, stage_game: StageGame):
        self.stage = stage_game
        self.minmax = [stage_game.compute_minmax(i)[0] for i in range(2)]

    def is_individually_rational(self, payoff: Tuple[float, float]) -> bool:
        """检查支付是否个体理性"""
        return all(payoff[i] >= self.minmax[i] - 1e-10 for i in range(2))

    def get_folk_theorem_region(self, n_points: int = 1000) -> np.ndarray:
        """
        获取民间定理可行区域
        可行支付集 ∩ 个体理性支付集
        """
        # 计算可行支付集的边界
        payoffs = self.stage.payoff.reshape(-1, 2)

        from scipy.spatial import ConvexHull
        hull = ConvexHull(payoffs)

        # 生成区域内的点
        min_u1, max_u1 = payoffs[:, 0].min(), payoffs[:, 0].max()
        min_u2, max_u2 = payoffs[:, 1].min(), payoffs[:, 1].max()

        feasible_points = []
        for _ in range(n_points):
            u1 = np.random.uniform(min_u1, max_u1)
            u2 = np.random.uniform(min_u2, max_u2)

            # 检查是否在凸包内且个体理性
            if self.is_individually_rational((u1, u2)):
                # 简化的凸包测试（对囚徒困境有效）
                feasible_points.append([u1, u2])

        return np.array(feasible_points)

    def plot_folk_theorem(self, save_path: str = None):
        """可视化民间定理区域"""
        fig, ax = plt.subplots(figsize=(10, 8))

        # 绘制阶段博弈支付点
        payoffs = self.stage.payoff.reshape(-1, 2)
        ax.scatter(payoffs[:, 0], payoffs[:, 1], s=200, c='red',
                  marker='o', label='Stage game outcomes', zorder=5)

        # 标记点
        for i, (u1, u2) in enumerate(payoffs):
            ax.annotate(f'({i//self.stage.n2},{i%self.stage.n2})',
                       (u1, u2), textcoords="offset points",
                       xytext=(5, 5), fontsize=9)

        # 绘制个体理性支付边界
        ax.axvline(x=self.minmax[0], color='green', linestyle='--',
                  label=f'Minmax player 1 = {self.minmax[0]:.1f}')
        ax.axhline(y=self.minmax[1], color='blue', linestyle='--',
                  label=f'Minmax player 2 = {self.minmax[1]:.1f}')

        # 绘制 NE 支付
        nes = self.stage.find_nash_equilibria()
        for ne in nes:
            u = self.stage.payoff[ne[0], ne[1]]
            ax.scatter(u[0], u[1], s=300, c='yellow', marker='*',
                      edgecolors='black', linewidths=2,
                      label='Nash Equilibrium', zorder=6)

        ax.set_xlabel('Player 1 Payoff', fontsize=12)
        ax.set_ylabel('Player 2 Payoff', fontsize=12)
        ax.set_title('Folk Theorem: Feasible and Individually Rational Payoffs', fontsize=14)
        ax.legend(loc='best')
        ax.grid(True, alpha=0.3)

        # 设置合理的边界
        ax.set_xlim(payoffs[:, 0].min() - 0.5, payoffs[:, 0].max() + 0.5)
        ax.set_ylim(payoffs[:, 1].min() - 0.5, payoffs[:, 1].max() + 0.5)

        if save_path:
            plt.savefig(save_path, dpi=150, bbox_inches='tight')

        return fig


def demo_prisoners_dilemma():
    """囚徒困境重复博弈演示"""

    print("=" * 70)
    print("重复囚徒困境分析")
    print("=" * 70)

    pd = PrisonersDilemma()
    analysis = pd.get_analysis()

    print("\n阶段博弈分析:")
    print(f"  支付矩阵 (C=合作, D=背叛):")
    print(f"         C        D")
    print(f"  C   (3,3)    (0,5)")
    print(f"  D   (5,0)    (1,1)")

    print(f"\n  纳什均衡: {analysis['stage_game_ne']}")
    print(f"  合作结果: {analysis['cooperative_outcome']}")
    print(f"  合作支付: {analysis['cooperative_payoff']}")
    print(f"  最小最大支付: {analysis['minmax']}")

    # 冷酷触发分析
    print("\n" + "-" * 50)
    print("冷酷触发策略分析:")

    coop = analysis['cooperative_payoff']
    deviate = analysis['deviate_from_coop']
    minmax = analysis['minmax']

    print(f"  合作支付: {coop}")
    print(f"  偏离支付（一期）: {deviate}")
    print(f"  惩罚支付: {minmax}")

    # 计算临界贴现因子
    repeated = RepeatedGame(pd, delta=0.9)
    critical_delta = repeated.compute_critical_delta(coop, deviate, minmax)

    print(f"\n  维持合作的临界贴现因子: δ ≥ {critical_delta:.3f}")

    # 检验不同贴现因子
    print("\n  不同贴现因子下的合作可持续性:")
    for delta in [0.3, 0.5, 0.7, 0.9]:
        rg = RepeatedGame(pd, delta)
        sustainable = rg.grim_trigger_sustainable(coop, deviate, minmax)
        status = "✓ 可持续" if sustainable else "✗ 不可持续"
        print(f"    δ = {delta:.1f}: {status}")

    print("\n" + "=" * 70)

    print("\n可行支付集 = 阶段博弈支付的凸包")
    print("民间定理可行区域 = 可行支付集 ∩ 个体理性支付集")
```
