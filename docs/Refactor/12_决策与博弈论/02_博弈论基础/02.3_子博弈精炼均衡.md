# 12.2.3 子博弈精炼均衡

## 12.2.3.1 引言

在动态博弈中，**纳什均衡**可能存在**不可置信威胁**（non-credible threats）：参与人威胁采取对自己不利的行动，但在实际行动时会选择背叛。Selten (1965) 提出**子博弈精炼均衡**（Subgame Perfect Nash Equilibrium, SPNE），通过要求均衡在每个子博弈中都构成纳什均衡，排除了不可置信威胁。

---

## 12.2.3.2 子博弈

### 12.2.3.2.1 定义

**定义 12.2.3.1**（子博弈）

在扩展式博弈 $\Gamma$ 中，**子博弈**是满足以下条件的历史 $h$ 及其后续博弈：

1. $h$ 是单节点信息集（$h \in I_i$ 且 $I_i = \{h\}$）
2. 对任何 $h'$ 是 $h$ 的前缀，$I(h')$ 是单节点信息集

**等价表述**：子博弈从某个决策节点开始，且不切割任何信息集。

### 12.2.3.2.2 子博弈的示例

**进入博弈**：

```
           [1: 进入者]
          /           \
     进入/             \不进入
        /               \
   [2: 在位者]        (0, 2)
   /          \
对抗/            \容纳
   |               |
(-1, -1)      (1, 1)
```

子博弈：

- 整个博弈
- 从在位者决策节点开始的子博弈

---

## 12.2.3.3 子博弈精炼均衡

### 12.2.3.3.1 定义

**定义 12.2.3.2**（子博弈精炼均衡）

策略组合 $s^*$ 是**子博弈精炼均衡**，如果 $s^*$ 在每个子博弈上都诱导出纳什均衡。

**形式化**：设 $\Gamma(h)$ 为从 $h$ 开始的子博弈，$s^*|_h$ 为 $s^*$ 在 $\Gamma(h)$ 上的限制。则对所有子博弈 $\Gamma(h)$：

$$s^*|_h \in NE(\Gamma(h))$$

### 12.2.3.3.2 进入博弈分析

**策略空间**：

- 进入者：{进入, 不进入}
- 在位者：{对抗, 容纳} × {如果进入则对抗, 如果进入则容纳}

在位者的策略实际上是映射：{进入} → {对抗, 容纳}

**纳什均衡分析**：

1. **(进入, 容纳)**：支付 (1, 1) —— SPNE
2. **(不进入, 对抗)**：支付 (0, 2) —— NE 但非 SPNE（在位者的威胁不可信）

**子博弈精炼**：从在位者节点开始的子博弈中，唯一 NE 是 (容纳)。因此整个博弈的唯一 SPNE 是 (进入, 容纳)。

---

## 12.2.3.4 逆向归纳与 SPNE

### 12.2.3.4.1 有限博弈的 SPNE 求解

**定理 12.2.3.1**（逆向归纳与 SPNE）

对于有限完美信息扩展式博弈：

1. 逆向归纳产生唯一的 SPNE
2. 该 SPNE 是唯一的子博弈精炼均衡

**证明**（归纳法）：

对博弈高度 $h$ 归纳：

- **基例**（$h = 1$）：最后行动者选择最优行动，这是唯一的 NE，也是 SPNE
- **归纳步骤**：假设对高度 $< h$ 成立。在高度 $h$ 的博弈中，最后行动者在每个终子博弈中选择最优。这些选择构成高度 $< h$ 的简化博弈的 SPNE。∎

### 12.2.3.4.2 逆向归纳算法

```
function BackwardInduction(Γ):
    for 每个终节点 z:
        V(z) = u(z)

    for h 按逆序:
        if P(h) = i (参与人 i 决策):
            a* = argmax_{a∈A(h)} V((h,a))
            s_i*(h) = a*
            V(h) = V((h, a*))
        else if P(h) = c (机会节点):
            V(h) = Σ_a p(a|h) · V((h,a))

    return s*
```

## 12.2.3.5 斯塔克尔伯格博弈

### 12.2.3.5.1 领导者-跟随者模型

**斯塔克尔伯格双寡头**：

- 企业 1（领导者）先选择产量 $q_1$
- 企业 2（跟随者）观察到 $q_1$ 后选择 $q_2$
- 市场价格 $P(Q) = a - Q$，$Q = q_1 + q_2$
- 成本 $c$ 相同

### 12.2.3.5.2 SPNE 求解

**跟随者的最优反应**：

$$\max_{q_2} (a - q_1 - q_2 - c) q_2$$

一阶条件：$a - q_1 - 2q_2 - c = 0$

$$q_2^*(q_1) = \frac{a - c - q_1}{2}$$

**领导者的优化**：

$$\max_{q_1} (a - q_1 - q_2^*(q_1) - c) q_1 = \max_{q_1} \frac{(a - c - q_1) q_1}{2}$$

$$q_1^* = \frac{a - c}{2}, \quad q_2^* = \frac{a - c}{4}$$

**先动优势**：领导者产量高于古诺均衡，利润更高。

## 12.2.3.6 重复博弈中的 SPNE

### 12.2.3.6.1 有限重复博弈

**定理 12.2.3.2**（有限重复的唯一 SPNE）

设阶段博弈有唯一纳什均衡 $a^*$。则 $T$ 期重复博弈的唯一 SPNE 是在每期都玩 $a^*$。

**证明**（逆向归纳）：

- 第 $T$ 期：唯一 NE 是 $a^*$
- 第 $T-1$ 期：无论历史如何，最后一期都是 $a^*$，因此当前期最优是 $a^*$
- 归纳可得全部期都玩 $a^*$

### 12.2.3.6.2 多重均衡阶段博弈

若阶段博弈有多个 NE，则重复博弈可能有非平凡 SPNE。

**示例**：阶段博弈有两个 NE：$a^*$（高效）和 $b^*$（低效）

- 早期采用 $a^*$
- 若偏离则惩罚（切换到 $b^*$）

## 12.2.3.7 Python 实现：子博弈精炼均衡

```python
"""
子博弈精炼均衡 (SPNE) 求解
支持完美信息扩展式博弈的逆向归纳
基于 Selten (1965) 和 Osborne & Rubinstein (1994)
"""

from typing import List, Dict, Optional, Tuple, Callable
from dataclasses import dataclass, field
from enum import Enum
import numpy as np
from copy import deepcopy


class NodeType(Enum):
    DECISION = "decision"
    CHANCE = "chance"
    TERMINAL = "terminal"


@dataclass
class ExtensiveFormNode:
    """扩展式博弈节点"""
    id: str
    node_type: NodeType
    player: Optional[int] = None  # 参与人编号 (0, 1, ...)

    children: Dict[str, 'ExtensiveFormNode'] = field(default_factory=dict)
    probabilities: Dict[str, float] = field(default_factory=dict)
    payoffs: Optional[np.ndarray] = None  # 终节点支付

    # 求解结果
    value: Optional[np.ndarray] = None
    optimal_action: Optional[str] = None


class ExtensiveFormGame:
    """扩展式博弈"""

    def __init__(self, root: ExtensiveFormNode, n_players: int):
        self.root = root
        self.n_players = n_players
        self.nodes: Dict[str, ExtensiveFormNode] = {}
        self._build_node_dict(root)

    def _build_node_dict(self, node: ExtensiveFormNode):
        self.nodes[node.id] = node
        for child in node.children.values():
            if child.id not in self.nodes:
                self._build_node_dict(child)

    def backward_induction(self) -> Dict[int, Dict[str, str]]:
        """
        逆向归纳求解 SPNE
        返回: {player_id: {node_id: action}}
        """
        self._solve_node(self.root)

        # 收集策略
        strategies = {i: {} for i in range(self.n_players)}
        self._collect_strategy(self.root, strategies)

        return strategies

    def _solve_node(self, node: ExtensiveFormNode) -> np.ndarray:
        """
        递归求解节点值
        返回: 支付向量 (对每个参与人)
        """
        if node.node_type == NodeType.TERMINAL:
            node.value = node.payoffs
            return node.payoffs

        elif node.node_type == NodeType.CHANCE:
            # 期望值
            expected_value = np.zeros(self.n_players)
            for action, prob in node.probabilities.items():
                child = node.children[action]
                child_value = self._solve_node(child)
                expected_value += prob * child_value

            node.value = expected_value
            return expected_value

        elif node.node_type == NodeType.DECISION:
            player = node.player
            best_value = None
            best_action = None

            for action, child in node.children.items():
                child_value = self._solve_node(child)

                if best_value is None or child_value[player] > best_value[player] + 1e-10:
                    best_value = child_value
                    best_action = action

            node.value = best_value
            node.optimal_action = best_action
            return best_value

    def _collect_strategy(self, node: ExtensiveFormNode,
                          strategies: Dict[int, Dict[str, str]]):
        """递归收集最优策略"""
        if node.node_type == NodeType.DECISION:
            strategies[node.player][node.id] = node.optimal_action
            # 只沿着最优路径继续收集（SPNE 策略）
            if node.optimal_action in node.children:
                self._collect_strategy(node.children[node.optimal_action], strategies)
        elif node.node_type == NodeType.CHANCE:
            for child in node.children.values():
                self._collect_strategy(child, strategies)

    def get_subgames(self) -> List[ExtensiveFormNode]:
        """
        识别所有子博弈的起始节点
        完美信息博弈中：每个决策节点都启动一个子博弈
        """
        subgames = []
        self._find_subgames(self.root, subgames)
        return subgames

    def _find_subgames(self, node: ExtensiveFormNode, subgames: List):
        if node.node_type == NodeType.DECISION:
            subgames.append(node)
            for child in node.children.values():
                self._find_subgames(child, subgames)
        elif node.node_type == NodeType.CHANCE:
            for child in node.children.values():
                self._find_subgames(child, subgames)

    def verify_spne(self, strategies: Dict[int, Dict[str, str]]) -> bool:
        """
        验证策略组合是否为 SPNE
        （检查每个子博弈是否都是 NE）
        """
        subgames = self.get_subgames()

        for subgame_root in subgames:
            if not self._verify_ne_in_subgame(subgame_root, strategies):
                return False

        return True

    def _verify_ne_in_subgame(self, subgame_root: ExtensiveFormNode,
                               strategies: Dict[int, Dict[str, str]]) -> bool:
        """验证子博弈是否为 NE"""
        # 简化的验证：检查在子博弈根节点是否有偏离激励
        if subgame_root.node_type != NodeType.DECISION:
            return True

        player = subgame_root.player
        prescribed_action = strategies[player].get(subgame_root.id)

        if prescribed_action is None:
            return False

        # 计算规定行动的支付
        self._solve_node(subgame_root.children[prescribed_action])
        prescribed_payoff = subgame_root.children[prescribed_action].value[player]

        # 检查所有偏离
        for action, child in subgame_root.children.items():
            self._solve_node(child)
            if child.value[player] > prescribed_payoff + 1e-10:
                return False

        return True


class StackelbergGame:
    """
    斯塔克尔伯格双寡头博弈
    领导者-跟随者模型
    """

    def __init__(self, a: float = 100, c: float = 10):
        """
        参数:
            a: 需求截距
            c: 边际成本
        """
        self.a = a
        self.c = c

    def follower_best_response(self, q1: float) -> float:
        """
        跟随者的最优反应函数
        q2*(q1) = (a - c - q1) / 2
        """
        return max(0, (self.a - self.c - q1) / 2)

    def leader_profit(self, q1: float) -> float:
        """
        领导者的利润（考虑跟随者的最优反应）
        """
        q2 = self.follower_best_response(q1)
        Q = q1 + q2
        P = self.a - Q
        return (P - self.c) * q1

    def solve_spne(self) -> Tuple[float, float, float, float]:
        """
        求解 SPNE
        返回: (q1*, q2*, π1*, π2*)
        """
        # 领导者优化
        q1_opt = (self.a - self.c) / 2

        # 跟随者反应
        q2_opt = self.follower_best_response(q1_opt)

        # 计算利润
        Q = q1_opt + q2_opt
        P = self.a - Q

        pi1 = (P - self.c) * q1_opt
        pi2 = (P - self.c) * q2_opt

        return q1_opt, q2_opt, pi1, pi2

    def compare_with_cournot(self) -> Dict:
        """
        比较斯塔克尔伯格均衡与古诺均衡
        """
        # 斯塔克尔伯格
        q1_s, q2_s, pi1_s, pi2_s = self.solve_spne()
        Q_s = q1_s + q2_s
        P_s = self.a - Q_s

        # 古诺均衡
        q_c = (self.a - self.c) / 3
        Q_c = 2 * q_c
        P_c = self.a - Q_c
        pi_c = (P_c - self.c) * q_c

        return {
            'stackelberg': {
                'q1': q1_s, 'q2': q2_s, 'Q': Q_s, 'P': P_s,
                'pi1': pi1_s, 'pi2': pi2_s
            },
            'cournot': {
                'q1': q_c, 'q2': q_c, 'Q': Q_c, 'P': P_c,
                'pi1': pi_c, 'pi2': pi_c
            },
            'first_mover_advantage': pi1_s - pi_c
        }


def create_entry_game() -> ExtensiveFormGame:
    """
    创建进入博弈

    结构:
        1 (进入者): 进入 / 不进入
        如果进入:
            2 (在位者): 对抗 / 容纳
    """
    # 终节点支付 (进入者, 在位者)
    t_enter_fight = ExtensiveFormNode(
        "t_ef", NodeType.TERMINAL,
        payoffs=np.array([-1, -1])
    )
    t_enter_accommodate = ExtensiveFormNode(
        "t_ea", NodeType.TERMINAL,
        payoffs=np.array([1, 1])
    )
    t_stay_out = ExtensiveFormNode(
        "t_so", NodeType.TERMINAL,
        payoffs=np.array([0, 2])
    )

    # 在位者决策节点
    d_incumbent = ExtensiveFormNode(
        "d_i", NodeType.DECISION, player=1,
        children={"fight": t_enter_fight, "accommodate": t_enter_accommodate}
    )

    # 进入者决策节点
    d_entrant = ExtensiveFormNode(
        "d_e", NodeType.DECISION, player=0,
        children={"enter": d_incumbent, "stay_out": t_stay_out}
    )

    return ExtensiveFormGame(d_entrant, n_players=2)


def create_chain_store_game(n_markets: int = 5) -> ExtensiveFormGame:
    """
    连锁店悖论（Kreps & Wilson, 1982）
    简化版：n个市场的进入博弈序列
    """
    # 这里创建简化版本，每个市场一个进入决策
    # 实际模型包含不完全信息（强/弱在位者）

    def build_subgame(market: int) -> ExtensiveFormNode:
        if market > n_markets:
            return ExtensiveFormNode(
                f"t_end", NodeType.TERMINAL,
                payoffs=np.array([0, 0])
            )

        # 当前市场
        t_out = ExtensiveFormNode(
            f"t_out_{market}", NodeType.TERMINAL,
            payoffs=np.array([0, 2])
        )
        t_fight = ExtensiveFormNode(
            f"t_fight_{market}", NodeType.TERMINAL,
            payoffs=np.array([-1, -1])
        )
        t_acc = ExtensiveFormNode(
            f"t_acc_{market}", NodeType.TERMINAL,
            payoffs=np.array([1, 1])
        )

        d_inc = ExtensiveFormNode(
            f"d_i_{market}", NodeType.DECISION, player=1,
            children={"fight": t_fight, "accommodate": t_acc}
        )

        d_ent = ExtensiveFormNode(
            f"d_e_{market}", NodeType.DECISION, player=0,
            children={"enter": d_inc, "stay_out": t_out}
        )

        return d_ent

    # 简化为单市场版本用于演示
    return create_entry_game()


def demo():
    """演示 SPNE 求解"""

    print("=" * 70)
    print("子博弈精炼均衡 (SPNE) 求解演示")
    print("=" * 70)

    # 示例 1: 进入博弈
    print("\n1. 进入博弈 (Entry Game)")
    print("-" * 50)

    game = create_entry_game()
    strategies = game.backward_induction()

    print("博弈结构:")
    print("  参与人 1 (进入者): 进入 / 不进入")
    print("  参与人 2 (在位者): 对抗 / 容纳")
    print("  支付: (进入者, 在位者)")
    print("    进入+对抗: (-1, -1)")
    print("    进入+容纳: (1, 1)")
    print("    不进入: (0, 2)")

    print("\n逆向归纳求解:")
    print(f"  子博弈 (在位者决策): 容纳得 1 > 对抗得 -1")
    print(f"  主博弈: 进入得 1 > 不进入得 0")

    print("\nSPNE 策略:")
    for player, strat in strategies.items():
        player_name = "进入者" if player == 0 else "在位者"
        for node, action in strat.items():
            print(f"  {player_name} 在节点 {node} 选择: {action}")

    print("\n均衡结果: (进入, 容纳) → 支付 (1, 1)")
    print("注意: (不进入, 对抗) 是 NE 但不是 SPNE（对抗威胁不可信）")

    # 示例 2: 斯塔克尔伯格博弈
    print("\n" + "=" * 70)
    print("2. 斯塔克尔伯格双寡头博弈")
    print("-" * 50)

    stackelberg = StackelbergGame(a=100, c=10)
    q1, q2, pi1, pi2 = stackelberg.solve_spne()
    comparison = stackelberg.compare_with_cournot()

    print("模型参数: a=100, c=10")
    print(f"\nSPNE 结果:")
    print(f"  领导者产量 q1* = {q1:.2f}")
    print(f"  跟随者产量 q2* = {q2:.2f}")
    print(f"  总产量 Q = {q1+q2:.2f}")
    print(f"  价格 P = {100-q1-q2:.2f}")
    print(f"  领导者利润 π1 = {pi1:.2f}")
    print(f"  跟随者利润 π2 = {pi2:.2f}")

    print(f"\n与古诺均衡比较:")
    print(f"  古诺产量: q1 = q2 = {comparison['cournot']['q1']:.2f}")
    print(f"  古诺利润: π1 = π2 = {comparison['cournot']['pi1']:.2f}")
    print(f"  先动优势: π1(Stackelberg) - π1(Cournot) = {comparison['first_mover_advantage']:.2f}")


if __name__ == "__main__":
    demo()
```
