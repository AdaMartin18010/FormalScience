# 12.5.1 有限理性

## 12.5.1.1 引言

经典博弈论假设参与人具有**完全理性**（perfect rationality）：无限计算能力、完美信息处理和效用最大化。然而，实验证据表明，现实中的人类决策往往偏离理性假设。**有限理性**（bounded rationality）模型试图刻画认知限制下的实际决策行为，由 Herbert Simon (1955, 1956) 开创，已成为行为经济学和博弈论的核心研究领域。

---

## 12.5.1.2 完全理性的局限

### 12.5.1.2.1 认知限制的来源

| 限制类型 | 描述 | 示例 |
|----------|------|------|
| **计算限制** | 求解复杂问题的能力有限 | 无法计算混合策略均衡 |
| **信息限制** | 获取和处理信息的能力有限 | 不知道所有可行策略 |
| **注意力限制** | 无法同时关注所有相关因素 | 忽略支付矩阵的某些部分 |
| **记忆限制** | 存储和检索信息的能力有限 | 忘记对手的历史行动 |

### 12.5.1.2.2 实验证据

**Beauty Contest 博弈**（Nagel, 1995）：

- 规则：每人选 $[0, 100]$ 的数，最接近平均数 2/3 者获胜
- 纳什均衡：所有人都选 0
- 实际结果：平均约 21（一级迭代占优）

**结论**：参与人只进行有限步的迭代推理。

## 12.5.1.3 满意性原则

### 12.5.1.3.1 Simon 的满意性模型

**定义 12.5.1.1**（满意性）

决策者设定**抱负水平**（aspiration level），选择第一个超过该水平的可行方案，而非寻找全局最优。

**算法**：

```
1. 设定抱负水平 A
2. 按某种顺序检查可行方案
3. 若方案 i 的效用 u_i ≥ A，选择 i 并停止
4. 若所有方案都检查完毕，降低 A 或扩大搜索
```

### 12.5.1.3.2 搜索成本

**定义 12.5.1.2**（最优搜索）

设搜索每个方案的成本为 $c$，方案效用分布为 $F(u)$。最优停止规则是**保留效用规则**：接受第一个满足 $u \geq u^*$ 的方案，其中 $u^*$ 满足：

$$\int_{u^*}^{\infty} (u - u^*) dF(u) = c$$

## 12.5.1.4 认知层级模型

### 12.5.1.4.1 Level-k 模型

**定义 12.5.1.3**（Level-k 思考）

参与人的推理层级：

- **L0**：非策略性（随机或 naive 行为）
- **L1**：认为对手是 L0，对此最优反应
- **L2**：认为对手是 L1，对此最优反应
- ...
- **Lk**：认为对手是 L(k-1)，对此最优反应

### 12.5.1.4.2 认知层级模型

**定义 12.5.1.4**（CH 模型，Camerer et al., 2004）

设参与人分布服从泊松分布：

$$P(\text{level } k) = \frac{\tau^k e^{-\tau}}{k!}$$

其中 $\tau$ 是平均认知层级。

**预测**：

Lk 玩家的行动分布：

$$\sigma_k = BR(\sigma_{k-1})$$

总体预测：

$$\sigma = \sum_{k=0}^{\infty} P(k) \cdot \sigma_k$$

---

## 12.5.1.5 量化响应均衡

### 12.5.1.5.1 Logit QRE

**定义 12.5.1.5**（Quantal Response Equilibrium, McKelvey & Palfrey, 1995）

参与人的选择是**随机最优反应**：

$$P(a_i) = \frac{\exp(\lambda \cdot EU_i(a_i))}{\sum_{a_i'} \exp(\lambda \cdot EU_i(a_i'))}$$

其中 $\lambda$ 是精度参数：

- $\lambda = 0$：完全随机
- $\lambda \to \infty$：完全理性（纳什均衡）

**均衡条件**：

$$\sigma_i(a_i) = P(a_i | \sigma_{-i}), \quad \forall i, a_i$$

## 12.5.1.6 Python 实现：有限理性模型

```python
"""
有限理性模型实现
包含 Level-k 认知层级、Quantal Response Equilibrium
基于 Camerer et al. (2004) 和 McKelvey & Palfrey (1995)
"""

from typing import List, Dict, Tuple, Callable, Optional
import numpy as np
from scipy.optimize import fsolve
from scipy.special import softmax
import matplotlib.pyplot as plt


class LevelKModel:
    """
    Level-k 认知层级模型
    """

    def __init__(self, payoff_matrix: np.ndarray, level0_strategy: np.ndarray = None):
        """
        参数:
            payoff_matrix: (n_actions, n_actions) 行参与人的支付矩阵
            level0_strategy: L0 策略（默认为均匀分布）
        """
        self.payoff = payoff_matrix
        self.n_actions = payoff_matrix.shape[0]

        if level0_strategy is None:
            self.sigma0 = np.ones(self.n_actions) / self.n_actions
        else:
            self.sigma0 = level0_strategy

    def best_response(self, opponent_strategy: np.ndarray) -> np.ndarray:
        """计算对给定对手策略的最优反应"""
        expected_payoffs = self.payoff @ opponent_strategy

        # 所有最优行动赋予相等概率
        max_payoff = np.max(expected_payoffs)
        br = (expected_payoffs >= max_payoff - 1e-10).astype(float)
        return br / br.sum()

    def level_k_strategy(self, k: int) -> np.ndarray:
        """
        计算 Lk 策略
        Lk 对 L(k-1) 最优反应
        """
        if k == 0:
            return self.sigma0

        sigma_prev = self.level_k_strategy(k - 1)
        return self.best_response(sigma_prev)

    def cognitive_hierarchy_prediction(self, tau: float, max_k: int = 10) -> np.ndarray:
        """
        认知层级模型预测
        使用泊松分布

        参数:
            tau: 平均认知层级
            max_k: 最大层级（截断）
        """
        # 计算各层级概率（泊松分布归一化）
        level_probs = []
        for k in range(max_k + 1):
            level_probs.append((tau ** k) * np.exp(-tau) / np.math.factorial(k))

        level_probs = np.array(level_probs)
        level_probs = level_probs / level_probs.sum()  # 归一化

        # 加权平均
        prediction = np.zeros(self.n_actions)
        for k, prob in enumerate(level_probs):
            sigma_k = self.level_k_strategy(k)
            prediction += prob * sigma_k

        return prediction


class QuantalResponseEquilibrium:
    """
    量化响应均衡 (QRE)
    """

    def __init__(self, payoff_matrix: np.ndarray):
        """
        参数:
            payoff_matrix: (n_actions, n_actions) 行参与人的支付
        """
        self.payoff = payoff_matrix
        self.n_actions = payoff_matrix.shape[0]

    def logit_response(self, opponent_strategy: np.ndarray,
                       lambda_param: float) -> np.ndarray:
        """
        Logit 量化响应

        参数:
            lambda_param: 精度参数（0 = 随机，∞ = 最优反应）
        """
        expected_payoffs = self.payoff @ opponent_strategy

        # 数值稳定性处理
        exp_payoffs = np.exp(lambda_param * (expected_payoffs - np.max(expected_payoffs)))
        return exp_payoffs / exp_payoffs.sum()

    def compute_qre(self, lambda_param: float,
                   initial_guess: np.ndarray = None,
                   max_iter: int = 1000) -> np.ndarray:
        """
        计算 QRE

        使用不动点迭代
        """
        if initial_guess is None:
            sigma = np.ones(self.n_actions) / self.n_actions
        else:
            sigma = initial_guess

        for _ in range(max_iter):
            sigma_new = self.logit_response(sigma, lambda_param)

            if np.linalg.norm(sigma_new - sigma) < 1e-10:
                return sigma_new

            sigma = 0.5 * sigma + 0.5 * sigma_new  # 阻尼迭代

        return sigma

    def qre_correspondence(self, lambda_values: np.ndarray) -> np.ndarray:
        """
        计算 QRE 对应（一系列 λ 值的均衡）
        """
        correspondences = []

        for lam in lambda_values:
            qre = self.compute_qre(lam)
            correspondences.append(qre)

        return np.array(correspondences)


class SatisficingModel:
    """
    Simon 满意性模型
    """

    def __init__(self, alternatives: List[str],
                 utility_function: Callable[[str], float]):
        self.alternatives = alternatives
        self.u = utility_function

    def satisficing_choice(self, aspiration_level: float,
                          search_order: List[str] = None) -> Tuple[str, int]:
        """
        满意性选择

        返回: (选择的方案, 搜索次数)
        """
        if search_order is None:
            search_order = self.alternatives.copy()

        for i, alt in enumerate(search_order):
            if self.u(alt) >= aspiration_level:
                return alt, i + 1

        # 如果没有满意的，返回最后一个
        return search_order[-1], len(search_order)

    def optimal_aspiration(self, search_cost: float,
                          utility_distribution: Callable[[], float],
                          n_samples: int = 10000) -> float:
        """
        计算最优抱负水平

        参数:
            search_cost: 每次搜索的成本
            utility_distribution: 效用分布采样函数
        """
        # 通过模拟估计最优抱负水平
        samples = [utility_distribution() for _ in range(n_samples)]

        best_a = 0
        best_expected_utility = -float('inf')

        for a in np.linspace(min(samples), max(samples), 100):
            # 期望效用 = E[u | u >= a] * P(u >= a) - E[search_cost]
            above_a = [u for u in samples if u >= a]

            if not above_a:
                continue

            prob_above = len(above_a) / len(samples)
            expected_u_given_above = np.mean(above_a)
            expected_search = (1 / prob_above) * search_cost  # 几何分布

            expected_net_utility = expected_u_given_above * prob_above - expected_search

            if expected_net_utility > best_expected_utility:
                best_expected_utility = expected_net_utility
                best_a = a

        return best_a


def beauty_contest_game():
    """
    Beauty Contest 博弈分析
    展示 Level-k 模型的应用
    """

    print("=" * 70)
    print("Beauty Contest 博弈的有限理性分析")
    print("=" * 70)

    # 游戏设置：猜 2/3 的平均数
    print("\n游戏规则:")
    print("  - 每人选择 [0, 100] 的数字")
    print("  - 最接近 2/3 × 平均数者获胜")
    print("  - 纳什均衡：所有人都选 0")

    # L0：随机选择
    print("\n1. Level-0 (L0): 随机选择 [0, 100]")
    l0_mean = 50  # 均匀分布的平均值
    print(f"   预期平均数: {l0_mean}")
    print(f"   L0 选择的 2/3: {2/3 * l0_mean:.1f}")

    # L1：认为对手是 L0
    print("\n2. Level-1 (L1): 认为对手是 L0")
    l1_choice = 2/3 * l0_mean
    print(f"   L1 选择: {l1_choice:.1f}")
    l1_mean = l1_choice
    print(f"   预期平均数: {l1_mean:.1f}")

    # L2
    print("\n3. Level-2 (L2): 认为对手是 L1")
    l2_choice = 2/3 * l1_mean
    print(f"   L2 选择: {l2_choice:.1f}")

    # L3
    print("\n4. Level-3 (L3): 认为对手是 L2")
    l3_choice = 2/3 * l2_choice
    print(f"   L3 选择: {l3_choice:.1f}")

    # 认知层级预测
    print("\n5. 认知层级模型预测 (τ=1.5):")
    tau = 1.5

    # 计算各层级概率
    level_probs = []
    for k in range(5):
        level_probs.append((tau ** k) * np.exp(-tau) / np.math.factorial(k))
    level_probs = np.array(level_probs) / sum(level_probs)

    choices = [50, 33.3, 22.2, 14.8, 9.9]
    predicted_average = sum(p * c for p, c in zip(level_probs, choices))

    print(f"   各层级概率: {[f'{p:.3f}' for p in level_probs]}")
    print(f"   预测的群体平均选择: {predicted_average:.1f}")
    print(f"   实验观察的平均值: ~21 (接近 L1-L2)")


def demo_qre():
    """演示 QRE"""

    print("\n" + "=" * 70)
    print("量化响应均衡 (QRE) 分析")
    print("=" * 70)

    # 匹配硬币博弈
    print("\n匹配硬币博弈:")
    print("  支付矩阵（行参与人）:")
    print("         H     T")
    print("  H   ( 1,   -1)")
    print("  T   (-1,    1)")

    payoff = np.array([[1, -1], [-1, 1]])
    qre = QuantalResponseEquilibrium(payoff)

    # 不同 λ 值的 QRE
    print("\n不同精度参数 λ 的 QRE:")
    for lam in [0.1, 1.0, 5.0, 20.0]:
        equilibrium = qre.compute_qre(lam)
        print(f"  λ = {lam:5.1f}: σ(H) = {equilibrium[0]:.4f}, σ(T) = {equilibrium[1]:.4f}")

    print("\n  当 λ → ∞ 时，QRE 收敛于纳什均衡 (0.5, 0.5)")


def demo_satisficing():
    """演示满意性模型"""

    print("\n" + "=" * 70)
    print("满意性 (Satisficing) 模型")
    print("=" * 70)

    # 定义选择问题
    alternatives = ["A", "B", "C", "D", "E"]
    utilities = {"A": 8, "B": 5, "C": 9, "D": 3, "E": 7}

    model = SatisficingModel(alternatives, lambda x: utilities[x])

    print(f"\n选择问题:")
    for alt in alternatives:
        print(f"  {alt}: 效用 = {utilities[alt]}")

    # 不同抱负水平的选择
    print(f"\n不同抱负水平下的选择:")
    for A in [4, 6, 8, 10]:
        choice, n_search = model.satisficing_choice(A)
        print(f"  抱负 A = {A}: 选择 {choice} (搜索 {n_search} 个选项)")

    # 最优抱负水平
    print(f"\n最优抱负水平分析:")
    np.random.seed(42)

    # 效用分布：均匀 [0, 10]
    def utility_dist():
        return np.random.uniform(0, 10)

    for c in [0.1, 0.5, 1.0]:
        optimal_a = model.optimal_aspiration(c, utility_dist, n_samples=5000)
        print(f"  搜索成本 c = {c}: 最优抱负水平 A* = {optimal_a:.2f}")


if __name__ == "__main__":
    beauty_contest_game()
    demo_qre()
    demo_satisficing()
```
