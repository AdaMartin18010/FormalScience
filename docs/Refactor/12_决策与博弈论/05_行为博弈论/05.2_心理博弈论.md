# 12.5.2 心理博弈论

## 12.5.2.1 引言

**心理博弈论**（Psychological Game Theory）扩展了标准博弈论框架，允许参与人的效用不仅依赖于物质结果，还依赖于**信念**（beliefs）——关于他人行为、意图和信念的信念。这一框架由 Geanakoplos、Pearce 和 Stacchetti (1989) 提出，为分析**互惠**（reciprocity）、**公平**（fairness）和**内疚**（guilt）等社会偏好提供了严格的数学工具。

---

## 12.5.2.2 心理博弈的基本框架

### 12.5.2.2.1 效用依赖于信念

**定义 12.5.2.1**（心理效用函数）

在心理博弈中，参与人 $i$ 的效用函数为：

$$U_i = U_i(a, b_i, c_i)$$

其中：

- $a = (a_1, ..., a_n)$：行动组合
- $b_i = (b_{i1}, ..., b_{in})$：$i$ 对其他人行动的一阶信念
- $c_i$：$i$ 对他人信念的二阶信念

### 12.5.2.2.2 信念的一致性

**定义 12.5.2.2**（合理信念）

信念必须满足：

$$b_{ij} = a_j, \quad c_{ijk} = b_{jk}$$

即信念与实际行动一致。

---

## 12.5.2.3 意向性互惠模型

### 12.5.2.3.1 Rabin 的互惠模型

**模型设定**（Rabin, 1993）：

二人博弈中，参与人 $i$ 的效用：

$$U_i(a_i, b_j, c_i) = \pi_i(a_i, b_j) + \tilde{f}_j(b_j, c_i) \cdot [1 + f_i(a_i, b_j)]$$

其中：

- $\pi_i$：物质支付
- $f_i(a_i, b_j)$：$i$ 对 $j$ 的"友善度"（kindness）
- $\tilde{f}_j(b_j, c_i)$：$i$ 认为的 $j$ 对 $i$ 的友善度

### 12.5.2.3.2 友善度函数

**定义 12.5.2.3**（友善度）

$$f_i(a_i, b_j) = \frac{\pi_j(b_j, a_i) - \pi_j^e(b_j)}{\pi_j^{max}(b_j) - \pi_j^{min}(b_j)}$$

其中：

- $\pi_j^e(b_j)$：$j$ 的"公平"支付（纳什均衡支付）
- $\pi_j^{max}, \pi_j^{min}$：给定 $b_j$ 时 $j$ 的最大/最小可能支付

**解释**：

- $f_i > 0$：$i$ 对 $j$ 友善（给 $j$ 高于公平的支付）
- $f_i < 0$：$i$ 对 $j$ 不友善

### 12.5.2.3.3 互惠均衡

**定义 12.5.2.4**（互惠均衡）

策略组合 $a^*$ 是**互惠均衡**，如果给定合理信念，每个参与人的选择是最优的。

**关键洞察**：参与人愿意牺牲物质利益来奖励友善行为或惩罚不友善行为。

---

## 12.5.2.4 礼物交换博弈

### 12.5.2.4.1 博弈设定

**阶段 1**：雇主选择工资 $w \in [\underline{w}, \bar{w}]$

**阶段 2**：工人选择努力水平 $e \in [0, 1]$

**支付**：

- 雇主：$\pi_E = q \cdot e - w$
- 工人：$\pi_W = w - c(e)$，其中 $c(e) = \frac{1}{2}e^2$

### 12.5.2.4.2 标准博弈论预测

**子博弈精炼均衡**：

- 工人最小努力：$e^* = 0$（努力成本，无激励）
- 雇主最低工资：$w^* = \underline{w}$

### 12.5.2.4.3 心理博弈论预测

引入**公平工资**概念：

$$\bar{w}(e) = \frac{1}{2}q \cdot e$$

若 $w \geq \bar{w}(e)$，工人可能选择正努力作为回报。

**互惠均衡**：

- 高工资 $\Rightarrow$ 工人的正互惠 $\Rightarrow$ 高努力
- 解释了实验中的高工资-高努力现象

## 12.5.2.5 Python 实现：心理博弈分析

"""
心理博弈论实现
包含互惠模型、礼物交换博弈
基于 Geanakoplos et al. (1989) 和 Rabin (1993)
"""

from typing import List, Tuple, Callable
import numpy as np
from scipy.optimize import minimize
import matplotlib.pyplot as plt


class PsychologicalGame:
    """
    心理博弈基类
    """

    def __init__(self, n_players: int):
        self.n = n_players

    def material_payoff(self, actions: Tuple) -> Tuple[float, ...]:
        """物质支付"""
        raise NotImplementedError

    def kindness(self, actions: Tuple, beliefs: Tuple) -> Tuple[float, ...]:
        """计算友善度"""
        raise NotImplementedError

    def psychological_utility(self, actions: Tuple, beliefs: Tuple,
                             higher_beliefs: Tuple) -> Tuple[float, ...]:
        """心理效用"""
        raise NotImplementedError


class RabinReciprocityModel:
    """
    Rabin (1993) 互惠模型
    适用于二人博弈
    """

    def __init__(self,
                 material_payoff_i: Callable,
                 material_payoff_j: Callable,
                 action_space_i: List,
                 action_space_j: List):
        """
        参数:
            material_payoff_i, material_payoff_j: (a_i, a_j) -> payoff
            action_space_i, action_space_j: 可行行动集
        """
        self.pi_i = material_payoff_i
        self.pi_j = material_payoff_j
        self.A_i = action_space_i
        self.A_j = action_space_j

    def fairness_payoff_j(self, a_i) -> float:
        """
        计算 j 的公平支付
        取 j 在给定 a_i 下的最好和最差支付的平均
        """
        payoffs_j = [self.pi_j(a_i, a_j) for a_j in self.A_j]
        return (max(payoffs_j) + min(payoffs_j)) / 2

    def kindness_i(self, a_i, b_j) -> float:
        """
        i 对 j 的友善度
        f_i(a_i, b_j) = [π_j(b_j, a_i) - π_j^e(b_j)] / [π_j^max - π_j^min]
        """
        actual_j = self.pi_j(b_j, a_i)
        fair_j = self.fairness_payoff_j(b_j)

        payoffs_j = [self.pi_j(b_j, a_i_prime) for a_i_prime in self.A_i]
        max_j = max(payoffs_j)
        min_j = min(payoffs_j)

        if max_j - min_j < 1e-10:
            return 0

        return (actual_j - fair_j) / (max_j - min_j)

    def psychological_utility_i(self, a_i, b_j, c_i, rho: float = 0.5) -> float:
        """
        i 的心理效用

        参数:
            rho: 互惠强度参数
        """
        material = self.pi_i(a_i, b_j)
        kindness_i = self.kindness_i(a_i, b_j)

        # 假设 c_i 表示 i 认为的 j 对 i 的友善度
        perceived_kindness_j = c_i

        # 互惠效用
        reciprocal = kindness_i * (1 + perceived_kindness_j)

        return material + rho * reciprocal

    def find_fair_equilibrium(self, rho: float = 0.5) -> Tuple:
        """
        寻找互惠均衡（简化版本）
        """
        best_welfare = -float('inf')
        best_equilibrium = None

        for a_i in self.A_i:
            for a_j in self.A_j:
                # 检查是否为均衡（给定信念下的最优反应）
                # 简化：假设信念与行动一致
                b_j = a_j
                c_i = self.kindness_i(a_j, a_i)  # 对称假设

                u_i = self.psychological_utility_i(a_i, b_j, c_i, rho)

                # 检查 i 是否有偏离动机（简化）
                can_deviate = False
                for a_i_alt in self.A_i:
                    if a_i_alt != a_i:
                        u_i_alt = self.psychological_utility_i(a_i_alt, b_j, c_i, rho)
                        if u_i_alt > u_i + 1e-10:
                            can_deviate = True
                            break

                if not can_deviate:
                    welfare = self.pi_i(a_i, a_j) + self.pi_j(a_i, a_j)
                    if welfare > best_welfare:
                        best_welfare = welfare
                        best_equilibrium = (a_i, a_j)

        return best_equilibrium


class GiftExchangeGame:
    """
    礼物交换博弈（雇主-工人）
    """

    def __init__(self, q: float = 10, w_min: float = 0, w_max: float = 10):
        """
        参数:
            q: 产出系数
            w_min, w_max: 工资范围
        """
        self.q = q
        self.w_min = w_min
        self.w_max = w_max

    def employer_payoff(self, w: float, e: float) -> float:
        """雇主的物质支付"""
        return self.q * e - w

    def worker_material_payoff(self, w: float, e: float) -> float:
        """工人的物质支付"""
        return w - 0.5 * e ** 2

    def worker_psychological_payoff(self, w: float, e: float,
                                     fair_wage: float = None,
                                     reciprocity_param: float = 0.5) -> float:
        """
        工人的心理支付

        包含互惠动机：高工资 -> 高努力
        """
        material = self.worker_material_payoff(w, e)

        if fair_wage is None:
            fair_wage = 0.5 * self.q * e  # 公平工资假设

        # 互惠效用：当 w > fair_wage 时，工人获得正效用（从回报雇主中获得满足）
        reciprocal_utility = reciprocity_param * (w - fair_wage) * e

        return material + reciprocal_utility

    def worker_optimal_effort(self, w: float, reciprocity: float = 0.5) -> float:
        """
        工人的最优努力（心理效用最大化）
        """
        # 最大化：w - 0.5*e^2 + ρ*(w - 0.5*q*e)*e
        # 一阶条件：-e + ρ*(w - q*e) = 0
        # e = ρ*w / (1 + ρ*q)

        if reciprocity == 0:
            # 标准博弈论：最小努力
            return 0

        e_opt = (reciprocity * w) / (1 + reciprocity * self.q)
        return min(max(e_opt, 0), 1)  # 约束在 [0, 1]

    def employer_optimal_wage(self, reciprocity: float = 0.5) -> float:
        """
        考虑工人互惠反应后，雇主的最优工资
        """
        def employer_objective(w):
            e = self.worker_optimal_effort(w, reciprocity)
            return -(self.q * e - w)  # 最大化 -> 最小化负数

        from scipy.optimize import minimize_scalar
        result = minimize_scalar(employer_objective, bounds=(self.w_min, self.w_max), method='bounded')
        return result.x

    def analyze(self, reciprocity_levels: List[float] = None):
        """分析不同互惠水平下的均衡"""

        if reciprocity_levels is None:
            reciprocity_levels = [0, 0.2, 0.5, 1.0, 2.0]

        results = []

        for rho in reciprocity_levels:
            w_opt = self.employer_optimal_wage(rho)
            e_opt = self.worker_optimal_effort(w_opt, rho)

            pi_E = self.employer_payoff(w_opt, e_opt)
            pi_W = self.worker_material_payoff(w_opt, e_opt)

            results.append({
                'reciprocity': rho,
                'wage': w_opt,
                'effort': e_opt,
                'employer_payoff': pi_E,
                'worker_payoff': pi_W,
                'total_surplus': pi_E + pi_W
            })

        return results


def demo_gift_exchange():
    """演示礼物交换博弈"""

    print("=" * 70)
    print("礼物交换博弈的心理博弈分析")
    print("=" * 70)

    game = GiftExchangeGame(q=10, w_min=0, w_max=10)

    print("\n模型设定:")
    print("  - 雇主选择工资 w ∈ [0, 10]")
    print("  - 工人选择努力 e ∈ [0, 1]")
    print("  - 雇主支付: π_E = 10*e - w")
    print("  - 工人物质支付: π_W = w - 0.5*e²")
    print("  - 工人心理支付: π_W + ρ*(w - fair_wage)*e")

    print("\n" + "-" * 50)
    print("不同互惠强度 ρ 下的均衡:")
    print("-" * 50)

    results = game.analyze([0, 0.1, 0.3, 0.5, 1.0])

    print(f"{'ρ':>6s} {'工资':>8s} {'努力':>8s} {'雇主支付':>10s} {'工人支付':>10s} {'总剩余':>10s}")
    print("-" * 60)

    for r in results:
        print(f"{r['reciprocity']:6.1f} {r['wage']:8.2f} {r['effort']:8.2f} "
              f"{r['employer_payoff']:10.2f} {r['worker_payoff']:10.2f} {r['total_surplus']:10.2f}")

    print("\n关键发现:")
    print("  - ρ=0 (标准博弈论): 最低工资，零努力")
    print("  - ρ>0 (互惠): 正工资，正努力")
    print("  - 高互惠导致高效率和社会剩余")


def demo_reciprocity_prisoners_dilemma():
    """互惠动机下的囚徒困境"""

    print("\n" + "=" * 70)
    print("互惠动机下的囚徒困境")
    print("=" * 70)

    # 标准囚徒困境
    print("\n标准囚徒困境支付矩阵:")
    print("          C       D")
    print("  C    (3,3)   (0,5)")
    print("  D    (5,0)   (1,1)")

    # 物质支付函数
    def pi_i(a_i, a_j):
        payoffs = {('C', 'C'): 3, ('C', 'D'): 0, ('D', 'C'): 5, ('D', 'D'): 1}
        return payoffs[(a_i, a_j)]

    def pi_j(a_i, a_j):
        payoffs = {('C', 'C'): 3, ('C', 'D'): 5, ('D', 'C'): 0, ('D', 'D'): 1}
        return payoffs[(a_i, a_j)]

    model = RabinReciprocityModel(
        pi_i, pi_j,
        ['C', 'D'], ['C', 'D']
    )

    print("\n友善度计算 (f_i):")
    print("  当双方都选 C: f_i = (3 - 3) / (5 - 0) = 0 (公平)")
    print("  当 i选C, j选D: f_i = (5 - 3) / 5 = 0.4 (i对j友善)")
    print("  当 i选D, j选C: f_i = (0 - 3) / 5 = -0.6 (i对j不友善)")

    print("\n在互惠均衡中:")
    print("  - 双方预期对方合作")
    print("  - 背叛会引致内疚（负互惠效用）")
    print("  - 合作可能成为互惠均衡")


if __name__ == "__main__":
    demo_gift_exchange()
    demo_reciprocity_prisoners_dilemma()
