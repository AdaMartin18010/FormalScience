## 12.5.3.1 引言

**演化博弈论**（Evolutionary Game Theory, EGT）将博弈论与生物演化理论相结合，研究策略在种群中的动态演化。由 Maynard Smith 和 Price (1973) 奠基，EGT 引入了**演化稳定策略**（ESS）和**复制者动态**（replicator dynamics）等核心概念，为理解社会规范、习俗的形成和稳定性提供了有力工具。

---

## 12.5.3.2 演化稳定策略

### 12.5.3.2.1 定义

**定义 12.5.3.1**（演化稳定策略，ESS）

策略 $x^*$ 是**演化稳定策略**，如果对任意突变策略 $y \neq x^*$，存在 $\bar{\epsilon} > 0$ 使得对所有 $\epsilon \in (0, \bar{\epsilon})$：

$$u(x^*, \epsilon y + (1-\epsilon)x^*) > u(y, \epsilon y + (1-\epsilon)x^*)$$

**解释**：

- 种群主要由 $x^*$ 组成
- 小比例 $\epsilon$ 的突变者采用 $y$
- ESS 要求原策略比突变策略获得更高支付

### 12.5.3.2.2 等价条件

**定理 12.5.3.1**（ESS 的等价条件）

策略 $x^*$ 是 ESS 当且仅当：

1. **纳什均衡条件**：$u(x^*, x^*) \geq u(y, x^*)$，$\forall y$
2. **稳定性条件**：若 $u(x^*, x^*) = u(y, x^*)$，则 $u(x^*, y) > u(y, y)$

**证明**：

展开 ESS 定义中的支付：
$$u(x^*, (1-\epsilon)x^* + \epsilon y) = (1-\epsilon)u(x^*, x^*) + \epsilon u(x^*, y)$$

ESS 不等式：
$$(1-\epsilon)u(x^*, x^*) + \epsilon u(x^*, y) > (1-\epsilon)u(y, x^*) + \epsilon u(y, y)$$

当 $\epsilon \to 0$，得 $u(x^*, x^*) \geq u(y, x^*)$。

若等号成立，则对所有小 $\epsilon > 0$：
$$u(x^*, y) > u(y, y)$$
∎

## 12.5.3.3 复制者动态

### 12.5.3.3.1 离散时间动态

**定义 12.5.3.2**（复制者动态）

设种群有 $n$ 种策略，$x_i(t)$ 是策略 $i$ 在时期 $t$ 的频率。

**离散时间**：

$$x_i(t+1) = x_i(t) \cdot \frac{u(i, x(t))}{\bar{u}(x(t))}$$

其中 $\bar{u}(x) = \sum_j x_j u(j, x)$ 是种群平均支付。

### 12.5.3.3.2 连续时间动态

**连续时间**（Taylor & Jonker, 1978）：

$$\dot{x}_i = x_i [u(i, x) - \bar{u}(x)]$$

**性质**：

- 若 $u(i, x) > \bar{u}(x)$，则 $x_i$ 增加
- 若 $u(i, x) < \bar{u}(x)$，则 $x_i$ 减少

---

## 12.5.3.4 演化博弈的基本定理

### 12.5.3.4.1 ESS 与动态稳定性

**定理 12.5.3.2**（ESS 是渐近稳定的）

若 $x^*$ 是 ESS，则它是复制者动态的渐近稳定不动点。

**逆否命题**：若 $x^*$ 在复制者动态下不稳定，则它不是 ESS。

### 12.5.3.4.2 纳什均衡与不动点

**定理 12.5.3.3**（纳什均衡是不动点）

混合策略 $\sigma^*$ 是纳什均衡当且仅当它是复制者动态的不动点。

**证明**：

若 $\sigma^*$ 是 NE，则对所有 $i \in \text{supp}(\sigma^*)$：
$$u(i, \sigma^*) = u(\sigma^*, \sigma^*) = \bar{u}(\sigma^*)$$

因此 $\dot{x}_i = 0$ 对所有 $i$。∎

## 12.5.3.5 经典例子

### 12.5.3.5.1 鹰鸽博弈

**设定**：

- $V$：资源价值
- $C$：冲突成本
- 支付矩阵：

|  | H | D |
|--|---|---|
| **H** | $(V-C)/2$ | $V$ |
| **D** | $0$ | $V/2$ |

**ESS 分析**：

- 若 $V > C$：纯策略 H 是 ESS（总是战斗）
- 若 $V < C$：混合策略 $p^* = V/C$ 是 ESS

### 12.5.3.5.2 协调博弈

**设定**：

|  | A | B |
|--|---|---|
| **A** | $a, a$ | $0, 0$ |
| **B** | $0, 0$ | $b, b$ |

其中 $a, b > 0$。

**ESS**：两个纯策略 A 和 B 都是 ESS，取决于初始条件。

## 12.5.3.6 Python 实现：演化博弈动态

```python
"""
演化博弈论实现
包含 ESS 分析、复制者动态模拟
基于 Maynard Smith (1982) 和 Weibull (1995)
"""

from typing import List, Tuple, Callable, Optional
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt
from itertools import product


class EvolutionaryGame:
    """演化博弈"""

    def __init__(self, payoff_matrix: np.ndarray):
        """
        参数:
            payoff_matrix: (n_strategies, n_strategies)
        """
        self.A = payoff_matrix
        self.n = payoff_matrix.shape[0]

    def expected_payoff(self, strategy: np.ndarray, population: np.ndarray) -> float:
        """策略在种群中的期望支付"""
        return strategy @ self.A @ population

    def average_payoff(self, population: np.ndarray) -> float:
        """种群的平均支付"""
        return population @ self.A @ population

    def replicator_dynamics(self, x: np.ndarray, t: float = 0) -> np.ndarray:
        """
        复制者动态 (dx/dt)

        dx_i/dt = x_i * [(Ax)_i - x^T A x]
        """
        ax = self.A @ x
        avg_payoff = x @ ax

        dx = x * (ax - avg_payoff)
        return dx

    def simulate(self, initial_pop: np.ndarray,
                t_span: Tuple[float, float] = (0, 50),
                n_points: int = 1000) -> Tuple[np.ndarray, np.ndarray]:
        """
        模拟复制者动态

        返回: (时间点, 种群演化轨迹)
        """
        t = np.linspace(t_span[0], t_span[1], n_points)
        trajectory = odeint(self.replicator_dynamics, initial_pop, t)

        # 归一化（数值稳定性）
        trajectory = np.clip(trajectory, 0, 1)
        trajectory = trajectory / trajectory.sum(axis=1, keepdims=True)

        return t, trajectory

    def find_ess(self, tol: float = 1e-6) -> List[np.ndarray]:
        """
        寻找 ESS（简化版本：检查纯策略）
        """
        ess_list = []

        # 检查纯策略
        for i in range(self.n):
            x = np.eye(self.n)[i]

            # 检查 ESS 条件
            is_ess = True

            # 条件 1: u(i, i) >= u(j, i) 对所有 j
            for j in range(self.n):
                if self.A[i, i] < self.A[j, i] - tol:
                    is_ess = False
                    break

            if is_ess:
                ess_list.append(x)

        return ess_list

    def find_nash_equilibria(self) -> List[np.ndarray]:
        """
        寻找对称纳什均衡（简化版本）
        """
        # 检查纯策略 NE
        ne_list = []

        for i in range(self.n):
            x = np.eye(self.n)[i]

            is_ne = True
            for j in range(self.n):
                if self.A[i, i] < self.A[j, i] - 1e-6:
                    is_ne = False
                    break

            if is_ne:
                ne_list.append(x)

        return ne_list


class HawkDoveGame(EvolutionaryGame):
    """
    鹰鸽博弈 (Hawk-Dove Game)

    参数:
        V: 资源价值
        C: 冲突成本
    """

    def __init__(self, V: float = 2, C: float = 3):
        # 支付矩阵 (行: 自身策略, 列: 对手策略)
        # H: Hawk, D: Dove
        payoff = np.array([
            [(V - C) / 2, V],    # Hawk vs (Hawk, Dove)
            [0, V / 2]           # Dove vs (Hawk, Dove)
        ])

        super().__init__(payoff)
        self.V = V
        self.C = C

    def theoretical_ess(self) -> Optional[float]:
        """理论 ESS 概率"""
        if self.V >= self.C:
            return 1.0  # 纯 Hawk
        else:
            return self.V / self.C  # 混合 ESS

    def analyze(self):
        """完整分析"""
        print(f"鹰鸽博弈分析 (V={self.V}, C={self.C})")
        print("=" * 50)

        print("\n支付矩阵:")
        print(f"         Hawk    Dove")
        print(f"  Hawk   {(self.V-self.C)/2:.1f}     {self.V:.1f}")
        print(f"  Dove    0      {self.V/2:.1f}")

        print(f"\n理论分析:")
        if self.V >= self.C:
            print(f"  V >= C: 纯 Hawk 策略是 ESS")
        else:
            ess_prob = self.theoretical_ess()
            print(f"  V < C: 混合 ESS，Hawk 概率 = V/C = {ess_prob:.3f}")

        # 数值验证
        t, traj = self.simulate(np.array([0.5, 0.5]), t_span=(0, 100))
        final_hawk = traj[-1, 0]

        print(f"\n数值模拟:")
        print(f"  初始: 50% Hawk, 50% Dove")
        print(f"  稳态: {final_hawk*100:.1f}% Hawk")


class CoordinationGame(EvolutionaryGame):
    """
    协调博弈

    参数:
        a: 双方都选 A 的支付
        b: 双方都选 B 的支付
    """

    def __init__(self, a: float = 2, b: float = 1):
        payoff = np.array([
            [a, 0],
            [0, b]
        ])
        super().__init__(payoff)
        self.a = a
        self.b = b

    def basin_of_attraction(self) -> float:
        """A 策略的吸引域边界"""
        return self.b / (self.a + self.b)

    def analyze(self):
        """完整分析"""
        print(f"\n协调博弈分析 (a={self.a}, b={self.b})")
        print("=" * 50)

        print("\n支付矩阵:")
        print(f"      A    B")
        print(f"  A  {self.a}    0")
        print(f"  B   0   {self.b}")

        print(f"\nESS 分析:")
        print(f"  纯策略 A 是 ESS (当种群主要选 A 时，A 是最优反应)")
        print(f"  纯策略 B 是 ESS (当种群主要选 B 时，B 是最优反应)")

        boundary = self.basin_of_attraction()
        print(f"\n吸引域边界: p_A = {boundary:.3f}")
        print(f"  若初始 p_A > {boundary:.3f}，演化收敛到 A")
        print(f"  若初始 p_A < {boundary:.3f}，演化收敛到 B")


class PrisonersDilemmaEGT(EvolutionaryGame):
    """
    囚徒困境的演化博弈分析
    """

    def __init__(self, T: float = 5, R: float = 3, P: float = 1, S: float = 0):
        """
        T: Temptation (背叛诱惑)
        R: Reward (合作奖励)
        P: Punishment (惩罚)
        S: Sucker (被 sucker 支付)
        """
        assert T > R > P > S

        payoff = np.array([
            [R, S],  # C vs (C, D)
            [T, P]   # D vs (C, D)
        ])

        super().__init__(payoff)
        self.T, self.R, self.P, self.S = T, R, P, S

    def analyze(self):
        """完整分析"""
        print(f"\n囚徒困境演化分析")
        print("=" * 50)

        print(f"\n支付矩阵 (T={self.T}, R={self.R}, P={self.P}, S={self.S}):")
        print(f"       C    D")
        print(f"  C   {self.R}    {self.S}")
        print(f"  D   {self.T}    {self.P}")

        print(f"\nESS 分析:")
        print(f"  背叛 (D) 是唯一的 ESS")
        print(f"  原因: D 是对 C 的最优反应，且 D 对 D 也是最优反应")

        # 数值验证
        for init_C in [0.9, 0.5, 0.1]:
            t, traj = self.simulate(np.array([init_C, 1-init_C]), t_span=(0, 50))
            final_C = traj[-1, 0]
            print(f"  初始 C={init_C:.1f} -> 稳态 C={final_C*100:.1f}%")


def plot_phase_diagram(game: EvolutionaryGame,
                       title: str = "Phase Diagram",
                       save_path: str = None):
    """
    绘制演化相位图（适用于 2 策略）
    """
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

    # 左图：不同初始条件的演化轨迹
    initial_conditions = np.linspace(0.1, 0.9, 9)

    for init in initial_conditions:
        x0 = np.array([init, 1 - init])
        t, traj = game.simulate(x0, t_span=(0, 50))
        ax1.plot(t, traj[:, 0], alpha=0.7, label=f'init={init:.1f}' if init in [0.1, 0.5, 0.9] else '')

    ax1.set_xlabel('Time')
    ax1.set_ylabel('Frequency of Strategy 1')
    ax1.set_title(f'{title} - Evolution Trajectories')
    ax1.legend()
    ax1.grid(True, alpha=0.3)
    ax1.set_ylim(0, 1)

    # 右图：向量场（相图）
    x = np.linspace(0, 1, 20)
    dx = []

    for xi in x:
        if 0 < xi < 1:
            x_vec = np.array([xi, 1 - xi])
            dx_vec = game.replicator_dynamics(x_vec)
            dx.append(dx_vec[0])
        else:
            dx.append(0)

    ax2.plot(x, dx, 'b-', linewidth=2)
    ax2.axhline(y=0, color='k', linestyle='-', linewidth=0.5)
    ax2.set_xlabel('Frequency of Strategy 1 (x)')
    ax2.set_ylabel('dx/dt')
    ax2.set_title(f'{title} - Phase Portrait')
    ax2.grid(True, alpha=0.3)

    # 标记不动点
    for xi in x:
        if abs(game.replicator_dynamics(np.array([xi, 1-xi]))[0]) < 0.01:
            stability = 'stable' if game.replicator_dynamics(np.array([xi+0.01, 0.99-xi]))[0] < 0 else 'unstable'
            color = 'green' if stability == 'stable' else 'red'
            ax2.axvline(x=xi, color=color, linestyle='--', alpha=0.5)

    plt.tight_layout()

    if save_path:
        plt.savefig(save_path, dpi=150, bbox_inches='tight')

    return fig


def demo_evolutionary_games():
    """演示演化博弈分析"""

    print("=" * 70)
    print("演化博弈论演示")
    print("=" * 70)

    # 1. 鹰鸽博弈
    hd = HawkDoveGame(V=2, C=3)
    hd.analyze()

    # 2. 协调博弈
    coord = CoordinationGame(a=2, b=1)
    coord.analyze()

    # 3. 囚徒困境
    pd = PrisonersDilemmaEGT()
    pd.analyze()

    # 可视化
    print("\n" + "=" * 70)
    print("生成演化动态可视化...")

    fig1 = plot_phase_diagram(hd, "Hawk-Dove Game", "hawk_dove_phase.png")
    fig2 = plot_phase_diagram(coord, "Coordination Game", "coordination_phase.png")

    print("  图表已保存至 hawk_dove_phase.png 和 coordination_phase.png")


if __name__ == "__main__":
    demo_evolutionary_games()
```
