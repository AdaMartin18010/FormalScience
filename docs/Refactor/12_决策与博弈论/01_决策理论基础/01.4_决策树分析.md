## 12.1.4.1 引言

**决策树**（decision tree）是序列决策问题的图形化表示工具，广泛应用于运筹学、博弈论和人工智能领域。
决策树将复杂的决策过程分解为一系列按时间顺序发生的决策节点和机会节点，为**逆向归纳**（backward induction）求解提供了直观框架。

---

## 12.1.4.2 决策树的形式化定义

### 12.1.4.2.1 决策树结构

**定义 12.1.4.1**（决策树）

一个**决策树**是四元组 $\mathcal{T} = (N, E, n_0, \tau)$，其中：

- $N$：节点集合
- $E \subseteq N \times N$：有向边集合，形成有根树结构
- $n_0 \in N$：根节点（初始决策点）
- $\tau: N \to \{\text{D}, \text{C}, \text{T}\}$：节点类型标记函数
  - **D**（Decision）：决策节点，决策者选择行动
  - **C**（Chance）：机会节点，自然按概率选择
  - **T**（Terminal）：终节点，支付确定

**定义 12.1.4.2**（信息集）

信息集 $H \subseteq N$ 是决策者无法区分的决策节点集合。完美信息决策树中，每个信息集都是单元素集。

### 12.1.4.2.2 决策树的扩展

**定义 12.1.4.3**（策略）

决策者的**纯策略**是一个函数 $s: H \to A$，为每个信息集 $h \in H$ 指定一个可行行动 $a \in A(h)$。

---

## 12.1.4.3 逆向归纳法

### 12.1.4.3.1 逆向归纳原理

**定理 12.1.4.1**（逆向归纳的最优性）

对于有限完美信息决策树，逆向归纳产生的策略是最优的（最大化期望效用）。

**逆向归纳算法**：

1. **初始化**：设 $V(n)$ 为节点 $n$ 的值函数
2. **终节点**：对所有 $n \in T$，$V(n) = u(z(n))$，其中 $z(n)$ 是终节点支付
3. **逆向迭代**：
   - 若 $\tau(n) = \text{C}$（机会节点）：$V(n) = \sum_{m \in C(n)} p(m|n) \cdot V(m)$
   - 若 $\tau(n) = \text{D}$（决策节点）：$V(n) = \max_{m \in C(n)} V(m)$，选择达到最大值的行动
4. **终止**：$V(n_0)$ 为最优值

### 12.1.4.3.2 证明

**证明**（数学归纳法）：

对决策树高度 $h$（最长路径的边数）归纳：

- **基例**（$h = 0$）：终节点，$V(n) = u(z(n))$ 显然最优
- **归纳假设**：对所有高度 $< h$ 的子树，逆向归纳产生最优值
- **归纳步骤**（高度 $h$）：
  - 若根为机会节点：各子树高度 $< h$，由归纳假设子树值最优，加权平均即最优期望效用
  - 若根为决策节点：各子树高度 $< h$，由归纳假设已知子树最优值，取最大即最优

∎

## 12.1.4.4 序列理性

### 12.1.4.4.1 子博弈完美

在博弈论语境下，决策树对应**扩展式博弈**（见 12.2.1）。逆向归纳的均衡概念是**子博弈精炼均衡**（Subgame Perfect Nash Equilibrium, SPNE）。

**定理 12.1.4.2**（逆向归纳与 SPNE）

有限完美信息扩展式博弈中，逆向归纳策略组合构成唯一的子博弈精炼均衡。

### 12.1.4.4.2 不可置信威胁

逆向归纳排除了**不可置信威胁**（non-credible threats）：在决策节点上，承诺采取对自己不利的行动是不可置信的。

## 12.1.4.5 敏感性分析与信息价值

### 12.1.4.5.1 完全信息价值

**定义 12.1.4.4**（完全信息期望价值 EVSI）

设 $V^*$ 为最优期望效用，$V_{\text{info}}$ 为获得信息后的最优期望效用：

$$\text{EVSI} = V_{\text{info}} - V^*$$

**定理 12.1.4.3**（信息价值的非负性）

在理性决策下，$  ext{EVSI} \geq 0$。

### 12.1.4.5.2 概率敏感性

通过**龙卷风图**（tornado diagram）分析不同参数变化对最优决策的影响。

## 12.1.4.6 决策树图例

### 12.1.4.6.1 新产品投资决策

```
                    [决策节点 D1]
                   /             \
                  / 投资          \ 放弃
                 /                 \
           [机会 C1]              (0, 0)
          /     |     \
     成功/0.4  持平/0.4  失败/0.2
        /        |         \
    [决策 D2] [决策 D3]   (-50)
    /    \      /   \
扩张  维持  扩张  维持
 |      |     |     |
(100) (60)  (30)  (20)
```

**逆向归纳求解**：

- D2: max{100, 60} = 100（选择扩张）
- D3: max{30, 20} = 30（选择扩张）
- C1: 0.4×100 + 0.4×30 + 0.2×(-50) = 40 + 12 - 10 = 42
- D1: max{42, 0} = 42（选择投资）

## 12.1.4.7 Python 实现：决策树求解器

```python
"""
决策树分析与逆向归纳求解
支持决策节点、机会节点和终节点
"""

from typing import List, Dict, Optional, Tuple, Union
from dataclasses import dataclass, field
from enum import Enum
import numpy as np
from graphviz import Digraph


class NodeType(Enum):
    DECISION = "decision"      # 决策节点
    CHANCE = "chance"          # 机会节点
    TERMINAL = "terminal"      # 终节点


@dataclass
class Node:
    """决策树节点"""
    id: str
    node_type: NodeType
    name: str = ""

    # 对于决策节点：子节点 -> (行动名称, 子节点引用)
    # 对于机会节点：子节点 -> (概率, 子节点引用)
    children: List[Tuple] = field(default_factory=list)

    # 终节点支付值
    payoff: Optional[float] = None

    # 求解结果
    value: Optional[float] = None
    optimal_action: Optional[str] = None


class DecisionTree:
    """
    决策树求解器
    使用逆向归纳法求解最优策略
    """

    def __init__(self, root: Node):
        self.root = root
        self.nodes: Dict[str, Node] = {}
        self._build_node_dict(root)

    def _build_node_dict(self, node: Node):
        """构建节点字典"""
        self.nodes[node.id] = node
        for child_info in node.children:
            child_node = child_info[1] if node.node_type == NodeType.DECISION else child_info[1]
            if child_node.id not in self.nodes:
                self._build_node_dict(child_node)

    def backward_induction(self, node: Optional[Node] = None) -> float:
        """
        逆向归纳求解
        返回节点的最优值
        """
        if node is None:
            node = self.root

        if node.node_type == NodeType.TERMINAL:
            # 终节点：值等于支付
            node.value = node.payoff
            return node.value

        elif node.node_type == NodeType.CHANCE:
            # 机会节点：计算期望效用
            expected_value = 0.0
            for prob, child in node.children:
                child_value = self.backward_induction(child)
                expected_value += prob * child_value

            node.value = expected_value
            return expected_value

        elif node.node_type == NodeType.DECISION:
            # 决策节点：选择最优行动
            best_value = float('-inf')
            best_action = None

            for action, child in node.children:
                child_value = self.backward_induction(child)

                if child_value > best_value:
                    best_value = child_value
                    best_action = action

            node.value = best_value
            node.optimal_action = best_action
            return best_value

    def get_optimal_strategy(self, node: Optional[Node] = None) -> Dict[str, str]:
        """
        获取最优策略（决策节点的行动选择）
        返回: {节点ID: 最优行动}
        """
        if node is None:
            node = self.root

        strategy = {}

        if node.node_type == NodeType.DECISION:
            strategy[node.id] = node.optimal_action

            # 递归获取后续策略
            for action, child in node.children:
                if action == node.optimal_action:
                    strategy.update(self.get_optimal_strategy(child))
                # 对于非最优分支，也记录（完整策略）
                else:
                    strategy.update(self.get_optimal_strategy(child))

        elif node.node_type == NodeType.CHANCE:
            for prob, child in node.children:
                strategy.update(self.get_optimal_strategy(child))

        return strategy

    def get_optimal_path(self, node: Optional[Node] = None, path: List[str] = None) -> List[str]:
        """
        获取最优路径（仅沿着最优选择）
        """
        if node is None:
            node = self.root
            path = []

        path.append(f"{node.name} (值={node.value:.2f})")

        if node.node_type == NodeType.TERMINAL:
            return path

        elif node.node_type == NodeType.DECISION:
            # 沿着最优行动
            for action, child in node.children:
                if action == node.optimal_action:
                    path.append(f"  -> 行动: {action}")
                    return self.get_optimal_path(child, path)

        elif node.node_type == NodeType.CHANCE:
            # 对于机会节点，显示所有可能路径
            # 这里只返回主路径信息
            path.append(f"  [机会节点，期望值={node.value:.2f}]")
            return path

        return path

    def sensitivity_analysis(self, param_name: str,
                            param_range: Tuple[float, float],
                            n_points: int = 50) -> List[Tuple[float, float]]:
        """
        敏感性分析：参数变化对根节点值的影响
        需要子类化以具体实现参数修改
        """
        results = []
        # 由具体应用实现
        return results

    def visualize(self, filename: str = "decision_tree"):
        """
        使用 Graphviz 可视化决策树
        """
        dot = Digraph(comment='Decision Tree')
        dot.attr(rankdir='TB')

        # 节点样式
        self._add_nodes_to_graph(dot, self.root)

        dot.render(filename, format='png', cleanup=False)
        print(f"决策树已保存至 {filename}.png")

    def _add_nodes_to_graph(self, dot: Digraph, node: Node):
        """递归添加节点到图"""
        # 节点形状和颜色
        if node.node_type == NodeType.DECISION:
            shape = 'square'
            color = 'lightblue'
            label = f"{node.name}\n[决策]"
            if node.optimal_action:
                label += f"\n最优: {node.optimal_action}"
        elif node.node_type == NodeType.CHANCE:
            shape = 'circle'
            color = 'lightyellow'
            label = f"{node.name}\n[机会]"
        else:
            shape = 'box'
            color = 'lightgreen'
            label = f"{node.name}\n支付={node.payoff}"

        if node.value is not None:
            label += f"\nV={node.value:.2f}"

        dot.node(node.id, label, shape=shape, style='filled', fillcolor=color)

        # 添加边
        if node.node_type == NodeType.DECISION:
            for action, child in node.children:
                is_optimal = (action == node.optimal_action)
                edge_style = 'bold' if is_optimal else 'solid'
                edge_color = 'red' if is_optimal else 'black'
                dot.edge(node.id, child.id, label=action,
                        style=edge_style, color=edge_color)
                self._add_nodes_to_graph(dot, child)

        elif node.node_type == NodeType.CHANCE:
            for prob, child in node.children:
                dot.edge(node.id, child.id, label=f"{prob:.2f}")
                self._add_nodes_to_graph(dot, child)


# 示例：新产品投资决策
def new_product_investment_example():
    """新产品投资决策树示例"""

    # 构建决策树
    # 终节点
    t1 = Node("T1", NodeType.TERMINAL, "成功-扩张", payoff=100)
    t2 = Node("T2", NodeType.TERMINAL, "成功-维持", payoff=60)
    t3 = Node("T3", NodeType.TERMINAL, "持平-扩张", payoff=30)
    t4 = Node("T4", NodeType.TERMINAL, "持平-维持", payoff=20)
    t5 = Node("T5", NodeType.TERMINAL, "失败", payoff=-50)
    t6 = Node("T6", NodeType.TERMINAL, "放弃投资", payoff=0)

    # 决策节点（成功后）
    d2 = Node("D2", NodeType.DECISION, "成功后决策",
              children=[("扩张", t1), ("维持", t2)])

    # 决策节点（持平后）
    d3 = Node("D3", NodeType.DECISION, "持平后决策",
              children=[("扩张", t3), ("维持", t4)])

    # 机会节点
    c1 = Node("C1", NodeType.CHANCE, "市场反应",
              children=[(0.4, d2), (0.4, d3), (0.2, t5)])

    # 根决策节点
    root = Node("D1", NodeType.DECISION, "投资决策",
                children=[("投资", c1), ("放弃", t6)])

    # 求解
    tree = DecisionTree(root)
    optimal_value = tree.backward_induction()

    print("=" * 60)
    print("新产品投资决策分析")
    print("=" * 60)
    print(f"\n最优期望收益: {optimal_value:.2f}")
    print(f"\n最优策略:")

    strategy = tree.get_optimal_strategy()
    for node_id, action in strategy.items():
        if node_id in ["D1", "D2", "D3"]:
            print(f"  节点 {node_id}: 选择 '{action}'")

    print(f"\n最优路径:")
    path = tree.get_optimal_path()
    for step in path:
        print(f"  {step}")

    return tree


# 示例：石油钻探决策
def oil_drilling_example():
    """石油钻探决策树示例"""

    # 终节点
    t_oil_good = Node("T_OG", NodeType.TERMINAL, "油-好地质", payoff=200-50-10)
    t_oil_med = Node("T_OM", NodeType.TERMINAL, "油-中地质", payoff=200-50-10)
    t_oil_poor = Node("T_OP", NodeType.TERMINAL, "油-差地质", payoff=200-50-10)

    t_mid_good = Node("T_MG", NodeType.TERMINAL, "中-好地质", payoff=50-50-10)
    t_mid_med = Node("T_MM", NodeType.TERMINAL, "中-中地质", payoff=50-50-10)

    t_dry = Node("T_D", NodeType.TERMINAL, "干井", payoff=-50-10)
    t_no_survey = Node("T_NS", NodeType.TERMINAL, "放弃勘测", payoff=0)
    t_no_drill = Node("T_ND", NodeType.TERMINAL, "不钻探", payoff=-10)

    # 钻探结果机会节点
    c_drill_good = Node("C_DG", NodeType.CHANCE, "钻探结果(好)",
                        children=[(0.6, t_oil_good), (0.3, t_mid_good), (0.1, t_dry)])
    c_drill_med = Node("C_DM", NodeType.CHANCE, "钻探结果(中)",
                       children=[(0.3, t_oil_med), (0.4, t_mid_med), (0.3, t_dry)])
    c_drill_poor = Node("C_DP", NodeType.CHANCE, "钻探结果(差)",
                        children=[(0.1, t_oil_poor), (0.2, t_mid_med), (0.7, t_dry)])

    # 勘测后的决策节点
    d_after_good = Node("D_AG", NodeType.DECISION, "勘测后决策(好)",
                        children=[("钻探", c_drill_good), ("放弃", t_no_drill)])
    d_after_med = Node("D_AM", NodeType.DECISION, "勘测后决策(中)",
                       children=[("钻探", c_drill_med), ("放弃", t_no_drill)])
    d_after_poor = Node("D_AP", NodeType.DECISION, "勘测后决策(差)",
                        children=[("钻探", c_drill_poor), ("放弃", t_no_drill)])

    # 勘测结果机会节点
    c_survey = Node("C_S", NodeType.CHANCE, "勘测结果",
                    children=[(0.3, d_after_good), (0.4, d_after_med), (0.3, d_after_poor)])

    # 根决策节点
    root = Node("D0", NodeType.DECISION, "初始决策",
                children=[("勘测", c_survey), ("放弃", t_no_survey)])

    # 求解
    tree = DecisionTree(root)
    optimal_value = tree.backward_induction()

    print("\n" + "=" * 60)
    print("石油钻探决策分析")
    print("=" * 60)
    print(f"\n最优期望收益: {optimal_value:.2f}")
    print(f"\n最优策略:")

    strategy = tree.get_optimal_strategy()
    for node_id, action in strategy.items():
        if node_id in ["D0", "D_AG", "D_AM", "D_AP"]:
            node_name = tree.nodes[node_id].name
            print(f"  {node_name}: 选择 '{action}'")

    return tree
```
