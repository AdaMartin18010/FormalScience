## 12.1.3.1 引言

Knight (1921) 区分了**风险**（risk，概率已知）与**不确定性**（uncertainty，概率未知）。
von Neumann-Morgenstern 理论处理风险，而 Savage 理论处理主观不确定性。
本章讨论风险态度、度量方法以及行为经济学对标准理论的修正。

## 12.1.3.2 风险态度

### 12.1.3.2.1 风险态度的定义

**定义 12.1.3.1**（确定等价物与风险溢价）

设 $L$ 为一维货币彩票（结果 $x \in \mathbb{R}$），效用函数 $u: \mathbb{R} \to \mathbb{R}$ 单调递增。

- **确定等价物**（Certainty Equivalent）：$CE(L)$ 满足 $u(CE(L)) = \mathbb{E}[u(X)]$
- **风险溢价**：$\pi(L) = \mathbb{E}[X] - CE(L)$

**定义 12.1.3.2**（风险态度分类）

决策者对 Lottery $L$ 是：

- **风险厌恶**（risk averse）：若 $CE(L) \leq \mathbb{E}[X]$（即 $\pi(L) \geq 0$）
- **风险中性**（risk neutral）：若 $CE(L) = \mathbb{E}[X]$（即 $\pi(L) = 0$）
- **风险偏好**（risk seeking）：若 $CE(L) \geq \mathbb{E}[X]$（即 $\pi(L) \leq 0$）

### 12.1.3.2.2 Jensen 不等式刻画

**定理 12.1.3.1**（风险态度的效用刻画）

设效用函数 $u$ 二次可微：

- 风险厌恶 $\iff u''(x) \leq 0$（凹效用）
- 风险中性 $\iff u''(x) = 0$（线性效用）
- 风险偏好 $\iff u''(x) \geq 0$（凸效用）

**证明**（风险厌恶 $\Rightarrow$ 凹性）：

设决策者风险厌恶。对任意 $x_1, x_2$ 和 $\alpha \in [0,1]$，考虑彩票 $L = \alpha \delta_{x_1} + (1-\alpha) \delta_{x_2}$。

风险厌恶意味着 $CE(L) \leq \mathbb{E}[L] = \alpha x_1 + (1-\alpha) x_2$。

由 $u$ 单调性：
$$u(CE(L)) \leq u(\alpha x_1 + (1-\alpha) x_2)$$

由确定等价物定义：
$$u(CE(L)) = \mathbb{E}[u(X)] = \alpha u(x_1) + (1-\alpha) u(x_2)$$

因此：
$$\alpha u(x_1) + (1-\alpha) u(x_2) \leq u(\alpha x_1 + (1-\alpha) x_2)$$

这正是凹函数的定义。∎

## 12.1.3.3 风险厌恶度量

### 12.1.3.3.1 Arrow-Pratt 度量

**定义 12.1.3.3**（绝对风险厌恶系数）

Arrow (1965) 和 Pratt (1964) 提出**绝对风险厌恶系数**：

$$A(x) = -\frac{u''(x)}{u'(x)}$$

**定义 12.1.3.4**（相对风险厌恶系数）

$$R(x) = -\frac{x \cdot u''(x)}{u'(x)} = x \cdot A(x)$$

**定理 12.1.3.2**（风险溢价的近似）

对小风险（方差 $\sigma^2$ 小）：

$$\pi(L) \approx \frac{1}{2} A(\bar{x}) \cdot \sigma^2$$

其中 $\bar{x} = \mathbb{E}[X]$。

**证明**（Taylor 展开）：

在 $\bar{x}$ 处展开 $u(CE)$ 和 $\mathbb{E}[u(X)]$：

$$u(CE) \approx u(\bar{x}) + u'(\bar{x})(CE - \bar{x})$$
$$\mathbb{E}[u(X)] \approx u(\bar{x}) + \frac{1}{2}u''(\bar{x})\sigma^2$$

令 $u(CE) = \mathbb{E}[u(X)]$：

$$u'(\bar{x})(CE - \bar{x}) \approx \frac{1}{2}u''(\bar{x})\sigma^2$$

$$\pi = \bar{x} - CE \approx -\frac{1}{2}\frac{u''(\bar{x})}{u'(\bar{x})}\sigma^2 = \frac{1}{2}A(\bar{x})\sigma^2$$
∎

### 12.1.3.3.2 标准效用函数族

| 类型 | 效用函数 | $A(x)$ | $R(x)$ |
|------|----------|--------|--------|
| CARA | $u(x) = -e^{-\gamma x}/\gamma$ | $\gamma$（常数） | $\gamma x$ |
| CRRA | $u(x) = \frac{x^{1-\rho}}{1-\rho}$ | $\rho/x$ | $\rho$（常数） |
| 对数 | $u(x) = \ln(x)$ | $1/x$ | 1 |
| 二次 | $u(x) = x - \frac{b}{2}x^2$ | $\frac{b}{1-bx}$ | $\frac{bx}{1-bx}$ |

## 12.1.3.4 随机占优

### 12.1.3.4.1 一阶随机占优

**定义 12.1.3.5**（一阶随机占优 FSD）

分布 $F$ **一阶随机占优** $G$（记作 $F \succeq_{FSD} G$），如果对所有非减函数 $u$：

$$\int u(x) dF(x) \geq \int u(x) dG(x)$$

**定理 12.1.3.3**（FSD 的等价条件）

以下等价：

1. $F \succeq_{FSD} G$
2. 对所有 $x$，$F(x) \leq G(x)$（$F$ 的 CDF 在 $G$ 下方）
3. $Y \stackrel{d}{=} X + Z$，其中 $Z \geq 0$ 且 $X \sim F$

### 12.1.3.4.2 二阶随机占优

**定义 12.1.3.6**（二阶随机占优 SSD）

分布 $F$ **二阶随机占优** $G$（记作 $F \succeq_{SSD} G$），如果对所有非减凹函数 $u$：

$$\int u(x) dF(x) \geq \int u(x) dG(x)$$

**定理 12.1.3.4**（SSD 的等价条件）

对具有相同均值的分布，以下等价：

1. $F \succeq_{SSD} G$
2. $\int_{-\infty}^x F(t) dt \leq \int_{-\infty}^x G(t) dt$ 对所有 $x$
3. $G$ 是 $F$ 的均值保持展形（MPS）

## 12.1.3.5 前景理论

### 12.1.3.5.1 标准理论的挑战

Allais 悖论（1953）和 Ellsberg 悖论（1961）揭示了 EU 理论的局限：

- **确定性效应**：人们对确定收益过度加权
- **孤立效应**：决策框架影响选择
- **概率加权**：对小概率事件过度反应

### 12.1.3.5.2 Kahneman-Tversky 前景理论

**定义 12.1.3.7**（前景价值函数）

Kahneman & Tversky (1979) 提出：

$$V(L) = \sum_i \pi(p_i) v(x_i)$$

其中：

- $v(x)$：**价值函数**，定义在相对参考点 $r$ 的收益/损失上
- $\pi(p)$：**概率加权函数**

**价值函数特征**：

$$v(x) = \begin{cases} x^\alpha & x \geq 0 \\ -\lambda(-x)^\beta & x < 0 \end{cases}$$

典型参数：$\alpha = \beta = 0.88$，$\lambda = 2.25$（损失厌恶系数）

**概率加权函数**（Prelec, 1998）：

$$\pi(p) = \exp(-(-\ln p)^\gamma)$$

### 12.1.3.5.3 参考点依赖

**定义 12.1.3.8**（参考点）

前景理论的核心特征是**参考点依赖**（reference dependence）：效用评估基于相对于参考点的变化，而非最终财富水平。

**损失厌恶**：

$$v(-x) < -v(x), \quad \forall x > 0$$

## 12.1.3.6 模糊性厌恶

### 12.1.3.6.1 Ellsberg 悖论

**设定**：

- 瓮中有 30 个红球，60 个黑或黄球（比例未知）
- 赌注 1A：抽中红得 100 元；1B：抽中黑得 100 元
- 赌注 2A：抽中红或黄得 100 元；2B：抽中黑或黄得 100 元

实验显示多数人选 1A 和 2B，违反 Savage 主观概率的确定原理。

### 12.1.3.6.2 多先验模型

**定义 12.1.3.9**（Gilboa-Schmeidler 最大最小期望效用）

设 $\mathcal{P}$ 为概率测度集合，效用为：

$$U(f) = \min_{P \in \mathcal{P}} \mathbb{E}_P[u(f)]$$

## 12.1.3.7 Python 实现：风险分析与前景理论

```python
"""
风险与不确定性分析
包含风险度量、随机占优、前景理论实现
"""

from typing import Callable, List, Tuple
import numpy as np
from scipy import integrate, stats
from scipy.optimize import minimize_scalar
import matplotlib.pyplot as plt


class RiskMeasure:
    """风险态度与度量"""

    @staticmethod
    def arrow_pratt_absolute(u: Callable[[float], float],
                              x: float,
                              h: float = 1e-5) -> float:
        """
        计算 Arrow-Pratt 绝对风险厌恶系数 A(x) = -u''/u'
        """
        # 数值微分
        u_prime = (u(x + h) - u(x - h)) / (2 * h)
        u_double = (u(x + h) - 2*u(x) + u(x - h)) / (h ** 2)

        if abs(u_prime) < 1e-10:
            return float('inf')

        return -u_double / u_prime

    @staticmethod
    def arrow_pratt_relative(u: Callable[[float], float], x: float) -> float:
        """相对风险厌恶系数 R(x) = x * A(x)"""
        return x * RiskMeasure.arrow_pratt_absolute(u, x)

    @staticmethod
    def certainty_equivalent(outcomes: List[float],
                              probs: List[float],
                              utility: Callable[[float], float]) -> float:
        """
        计算确定等价物 CE
        u(CE) = E[u(X)]
        """
        # 计算期望效用
        eu = sum(p * utility(x) for x, p in zip(outcomes, probs))

        # 反解 CE
        def objective(ce):
            return (utility(ce) - eu) ** 2

        result = minimize_scalar(objective, bounds=(min(outcomes), max(outcomes)),
                                  method='bounded')
        return result.x

    @staticmethod
    def risk_premium(outcomes: List[float],
                     probs: List[float],
                     utility: Callable[[float], float]) -> float:
        """风险溢价 = E[X] - CE"""
        ev = sum(x * p for x, p in zip(outcomes, probs))
        ce = RiskMeasure.certainty_equivalent(outcomes, probs, utility)
        return ev - ce


class UtilityFunctions:
    """标准效用函数族"""

    @staticmethod
    def cara(gamma: float) -> Callable[[float], float]:
        """
        CARA: u(x) = -exp(-γx)/γ
        A(x) = γ (常数)
        """
        return lambda x: -np.exp(-gamma * x) / gamma

    @staticmethod
    def crra(rho: float) -> Callable[[float], float]:
        """
        CRRA: u(x) = x^(1-ρ)/(1-ρ) for ρ≠1, ln(x) for ρ=1
        R(x) = ρ (常数)
        """
        if abs(rho - 1.0) < 1e-10:
            return lambda x: np.log(max(x, 1e-10))
        else:
            return lambda x: (max(x, 0) ** (1 - rho)) / (1 - rho)

    @staticmethod
    def quadratic(b: float, a: float = 1.0) -> Callable[[float], float]:
        """
        二次效用: u(x) = ax - (b/2)x^2
        注意：当 x > a/b 时效用递减（不合理区域）
        """
        return lambda x: a * x - 0.5 * b * x ** 2


class ProspectTheory:
    """
    Kahneman-Tversky 前景理论实现
    """

    def __init__(self,
                 alpha: float = 0.88,      # 收益曲率
                 beta: float = 0.88,       # 损失曲率
                 lambda_: float = 2.25,     # 损失厌恶
                 gamma: float = 0.65):     # 概率加权曲率
        self.alpha = alpha
        self.beta = beta
        self.lambda_ = lambda_
        self.gamma = gamma

    def value_function(self, x: float, reference: float = 0) -> float:
        """
        价值函数 v(x)（相对于参考点）
        x: 结果值
        reference: 参考点（默认0）
        """
        deviation = x - reference

        if deviation >= 0:
            # 收益域：凹函数
            return deviation ** self.alpha
        else:
            # 损失域：凸函数，损失厌恶
            return -self.lambda_ * ((-deviation) ** self.beta)

    def probability_weight(self, p: float) -> float:
        """
        Prelec 概率加权函数
        π(p) = exp(-(-ln p)^γ)
        """
        if p <= 0:
            return 0.0
        if p >= 1:
            return 1.0

        return np.exp(-((-np.log(p)) ** self.gamma))

    def prospect_value(self,
                       outcomes: List[float],
                       probs: List[float],
                       reference: float = 0) -> float:
        """
        计算前景值 V = Σ π(p_i) * v(x_i)
        """
        total_value = 0.0

        # 按相对于参考点排序
        gains = [(x, p) for x, p in zip(outcomes, probs) if x >= reference]
        losses = [(x, p) for x, p in zip(outcomes, probs) if x < reference]

        # 累积加权（累积前景理论 CPT）
        if gains:
            gains_sorted = sorted(gains, key=lambda t: t[0])
            cum_prob = 0
            for i, (x, p) in enumerate(gains_sorted):
                new_cum = cum_prob + p
                weight = self.probability_weight(new_cum) - self.probability_weight(cum_prob)
                total_value += weight * self.value_function(x, reference)
                cum_prob = new_cum

        if losses:
            losses_sorted = sorted(losses, key=lambda t: t[0], reverse=True)
            cum_prob = 0
            for i, (x, p) in enumerate(losses_sorted):
                new_cum = cum_prob + p
                weight = self.probability_weight(new_cum) - self.probability_weight(cum_prob)
                total_value += weight * self.value_function(x, reference)
                cum_prob = new_cum

        return total_value

    def compare_with_eu(self,
                        outcomes: List[float],
                        probs: List[float],
                        eu_utility: Callable[[float], float],
                        reference: float = 0) -> Tuple[float, float]:
        """
        比较前景理论与期望效用
        返回: (PT_value, EU_value)
        """
        pt_val = self.prospect_value(outcomes, probs, reference)
        eu_val = sum(p * eu_utility(x) for x, p in zip(outcomes, probs))
        return pt_val, eu_val


class StochasticDominance:
    """随机占优检验"""

    @staticmethod
    def fsd_test(cdf1: Callable[[float], float],
                 cdf2: Callable[[float], float],
                 x_range: Tuple[float, float],
                 n_points: int = 1000) -> bool:
        """
        检验 F1 是否一阶随机占优 F2
        即是否对所有 x: F1(x) <= F2(x)
        """
        xs = np.linspace(x_range[0], x_range[1], n_points)

        for x in xs:
            if cdf1(x) > cdf2(x) + 1e-10:
                return False

        return True

    @staticmethod
    def ssd_test(cdf1: Callable[[float], float],
                 cdf2: Callable[[float], float],
                 x_range: Tuple[float, float],
                 n_points: int = 1000) -> bool:
        """
        检验 F1 是否二阶随机占优 F2
        即是否对所有 x: ∫_{-∞}^x F1(t)dt <= ∫_{-∞}^x F2(t)dt
        """
        xs = np.linspace(x_range[0], x_range[1], n_points)

        for x in xs:
            # 数值积分
            int1, _ = integrate.quad(cdf1, x_range[0], x)
            int2, _ = integrate.quad(cdf2, x_range[0], x)

            if int1 > int2 + 1e-10:
                return False

        return True


# 示例应用
def risk_analysis_demo():
    """风险分析示例"""

    print("=" * 60)
    print("风险态度分析示例")
    print("=" * 60)

    # 彩票：50% 得 100，50% 得 0
    outcomes = [100, 0]
    probs = [0.5, 0.5]
    ev = 50

    print(f"\n彩票: 50% × 100 + 50% × 0 (期望值 = {ev})")
    print("-" * 40)

    # CARA 效用
    for gamma in [0.1, 0.5, 1.0, 2.0]:
        u = UtilityFunctions.cara(gamma)
        ce = RiskMeasure.certainty_equivalent(outcomes, probs, u)
        rp = RiskMeasure.risk_premium(outcomes, probs, u)
        A = RiskMeasure.arrow_pratt_absolute(u, ev)

        print(f"CARA (γ={gamma}): CE={ce:.2f}, 风险溢价={rp:.2f}, A({ev})={A:.4f}")

    # CRRA 效用
    print("\n" + "-" * 40)
    for rho in [0.5, 1.0, 2.0, 3.0]:
        u = UtilityFunctions.crra(rho)
        ce = RiskMeasure.certainty_equivalent(outcomes, probs, u)
        rp = ev - ce
        R = RiskMeasure.arrow_pratt_relative(u, ev)

        print(f"CRRA (ρ={rho}): CE={ce:.2f}, 风险溢价={rp:.2f}, R({ev})={R:.4f}")


def prospect_theory_demo():
    """前景理论示例"""

    print("\n" + "=" * 60)
    print("前景理论示例")
    print("=" * 60)

    pt = ProspectTheory()

    # 示例1：收益框架 vs 损失框架
    print("\n框架效应演示:")
    print("-" * 40)

    # 收益框架：确定获得 3000 vs 80% 获得 4000
    outcomes_gain_a = [3000]
    probs_gain_a = [1.0]
    outcomes_gain_b = [4000, 0]
    probs_gain_b = [0.8, 0.2]

    print("收益框架:")
    print(f"  A: 确定获得 3000 → PT值 = {pt.prospect_value(outcomes_gain_a, probs_gain_a):.2f}")
    print(f"  B: 80% 获得 4000 → PT值 = {pt.prospect_value(outcomes_gain_b, probs_gain_b):.2f}")

    # 损失框架：确定损失 3000 vs 80% 损失 4000
    outcomes_loss_a = [-3000]
    probs_loss_a = [1.0]
    outcomes_loss_b = [-4000, 0]
    probs_loss_b = [0.8, 0.2]

    print("\n损失框架:")
    print(f"  A: 确定损失 3000 → PT值 = {pt.prospect_value(outcomes_loss_a, probs_loss_a):.2f}")
    print(f"  B: 80% 损失 4000 → PT值 = {pt.prospect_value(outcomes_loss_b, probs_loss_b):.2f}")

    print("\n说明：损失厌恶使得人们在收益域风险厌恶，在损失域风险偏好")


if __name__ == "__main__":
    risk_analysis_demo()
    prospect_theory_demo()
```
