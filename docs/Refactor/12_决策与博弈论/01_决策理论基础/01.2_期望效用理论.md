# 12.1.2 期望效用理论

## 12.1.2.1 引言

von Neumann 和 Morgenstern (1944) 在其开创性著作中提出了期望效用理论（Expected Utility Theory, EU），为不确定性下的理性决策提供了严格的数学基础。
EU 理论将决策者的选择对象从确定的结果扩展到**彩票**（lotteries），并建立了一组公理，使得偏好可以用期望效用函数表示。

---

## 12.1.2.2 彩票空间

### 12.1.2.2.1 简单彩票

**定义 12.1.2.1**（简单彩票）

设 $C = \{c_1, c_2, ..., c_n\}$ 为**结果集**（consequence set）。一个**简单彩票**（simple lottery）是 $C$ 上的概率分布，即向量 $L = (p_1, p_2, ..., p_n)$，其中 $p_i \geq 0$ 且 $\sum_{i=1}^n p_i = 1$。记 $\mathcal{L}$ 为所有简单彩票的集合。

**记法**：$L = (p_1: c_1, p_2: c_2, ..., p_n: c_n)$ 表示以概率 $p_i$ 获得结果 $c_i$ 的彩票。

### 12.1.2.2.2 复合彩票

**定义 12.1.2.2**（复合彩票）

**复合彩票**（compound lottery）是以彩票为结果的彩票。设 $\alpha \in [0,1]$，$L$ 和 $L'$ 是两个彩票，则 $\alpha L \oplus (1-\alpha) L'$ 表示：

- 以概率 $\alpha$ 获得彩票 $L$
- 以概率 $(1-\alpha)$ 获得彩票 $L'$

**简化公理**（Reduction Axiom）：

复合彩票 $\alpha L \oplus (1-\alpha) L'$ 等价于简单彩票 $L''$，其中：

$$L''_i = \alpha L_i + (1-\alpha) L'_i, \quad \forall i$$

这允许我们将所有彩票视为 $\mathcal{L}$ 中的元素。

---

## 12.1.2.3 von Neumann-Morgenstern 公理体系

### 12.1.2.3.1 公理陈述

设 $\succeq$ 是彩票空间 $\mathcal{L}$ 上的偏好关系：

**公理 12.1.2.1**（完备性与传递性）

$\succeq$ 是理性偏好关系（见 12.1.1）。

**公理 12.1.2.2**（连续性/Archimedean 公理）

对所有 $L, L', L'' \in \mathcal{L}$，若 $L \succ L' \succ L''$，则存在 $\alpha, \beta \in (0,1)$ 使得：

$$\alpha L + (1-\alpha) L'' \succ L' \succ \beta L + (1-\beta) L''$$

**解释**：不存在"无限好"或"无限坏"的结果，任何优劣之间都可以通过概率混合找到中间偏好。

**公理 12.1.2.3**（独立性公理）

对所有 $L, L', L'' \in \mathcal{L}$ 和 $\alpha \in (0,1]$：

$$L \succeq L' \iff \alpha L + (1-\alpha) L'' \succeq \alpha L' + (1-\alpha) L''$$

**解释**：在两个彩票之间插入相同的"噪声"（$L''$ 以相同概率）不改变偏好顺序。这是 EU 理论的核心公理。

### 12.1.2.3.2 公理的含义

**引理 12.1.2.1**（线性性）

由独立性公理可推出：

$$L \sim L' \implies \alpha L + (1-\alpha) L'' \sim \alpha L' + (1-\alpha) L''$$

## 12.1.2.4 期望效用表示定理

### 12.1.2.4.1 表示定理

**定理 12.1.2.1**（von Neumann-Morgenstern 期望效用定理）

偏好关系 $\succeq$ 在 $\mathcal{L}$ 上满足公理 12.1.2.1-12.1.2.3 当且仅当存在**期望效用函数** $U: \mathcal{L} \to \mathbb{R}$ 使得：

1. **表示性**：$L \succeq L' \iff U(L) \geq U(L')$
2. **期望形式**：存在 $u: C \to \mathbb{R}$ 使得
   $$U(L) = \sum_{i=1}^n p_i u(c_i) = \mathbb{E}_L[u(c)]$$

此外，$u$ 在正仿射变换下唯一：$v = a \cdot u + b$（$a > 0$）表示同一偏好。

### 12.1.2.4.2 证明

**证明**：(von Neumann & Morgenstern, 1944)

**步骤 1：构造基准效用**

设 $c^* \succ c_*$ 分别为最偏好和最不偏好的结果（若所有结果无差异，定理显然成立）。定义：

- $u(c^*) = 1$
- $u(c_*) = 0$

**步骤 2：定义任意结果的效用**

对任意 $c \in C$，由连续性，存在唯一的 $\alpha_c \in [0,1]$ 使得：

$$c \sim \alpha_c c^* + (1-\alpha_c) c_*$$

定义 $u(c) = \alpha_c$。

**步骤 3：扩展到彩票**

对任意彩票 $L = (p_1: c_1, ..., p_n: c_n)$，定义：

$$U(L) = \sum_{i=1}^n p_i u(c_i)$$

**步骤 4：验证表示性**

设 $L \succeq L'$。由步骤 2，每个 $c_i \sim u(c_i) c^* + (1-u(c_i)) c_*$。由独立性公理的迭代应用：

$$L \sim \left(\sum_i p_i u(c_i)\right) c^* + \left(1 - \sum_i p_i u(c_i)\right) c_*$$

同理对 $L'$。由于 $c^* \succ c_*$，比较两个复合彩票即得比较效用值。

**步骤 5：唯一性**

设 $u$ 和 $v$ 都给出 EU 表示。由 12.1.1 的基数效用唯一性定理，$v = a \cdot u + b$。∎

## 12.1.2.5 主观期望效用（Savage 理论）

### 12.1.2.5.1 问题设定

von Neumann-Morgenstern 理论假设客观概率已知。Savage (1954) 扩展至主观概率情形。

**设定**：

- $S$：状态空间（states of the world）
- $C$：结果集
- **行为**（act）：函数 $f: S \to C$

### 12.1.2.5.2 Savage 公理

**定理 12.1.2.2**（Savage 主观期望效用定理）

在适当的公理条件下，存在唯一的概率测度 $P$ 在 $S$ 上和效用函数 $u: C \to \mathbb{R}$ 使得：

$$f \succeq g \iff \int_S u(f(s)) \, dP(s) \geq \int_S u(g(s)) \, dP(s)$$

---

## 12.1.2.6 时间一致性与动态决策

### 12.1.2.6.1 指数贴现

**定义 12.1.2.3**（指数贴现效用）

对跨期消费流 $(c_0, c_1, ..., c_T)$，指数贴现效用为：

$$U(c_0, ..., c_T) = \sum_{t=0}^T \delta^t u(c_t)$$

其中 $\delta \in (0,1)$ 为贴现因子。

**定理 12.1.2.3**（时间一致性）

指数贴现偏好是时间一致的：在任何时点重新优化，计划不会改变。

## 12.1.2.7 Python 实现：期望效用计算

```python
"""
期望效用理论 (EUT) 的 Python 实现
基于 von Neumann & Morgenstern (1944)
"""

from typing import List, Dict, Tuple, Callable
from dataclasses import dataclass
import numpy as np
from scipy.optimize import minimize_scalar


@dataclass
class Outcome:
    """决策结果"""
    name: str
    value: float  # 货币值或其他度量

    def __hash__(self):
        return hash(self.name)

    def __eq__(self, other):
        return isinstance(other, Outcome) and self.name == other.name


@dataclass
class Lottery:
    """
    简单彩票
    L = {(p1, c1), (p2, c2), ..., (pn, cn)}
    """
    probabilities: Dict[Outcome, float]

    def __post_init__(self):
        # 验证概率分布
        total = sum(self.probabilities.values())
        assert abs(total - 1.0) < 1e-10, f"概率之和必须等于1，当前为{total}"
        assert all(p >= 0 for p in self.probabilities.values()), "概率必须非负"

    def expected_value(self) -> float:
        """期望值"""
        return sum(p * c.value for c, p in self.probabilities.items())

    def expected_utility(self, utility_func: Callable[[Outcome], float]) -> float:
        """
        计算期望效用
        EU(L) = Σ p_i * u(c_i)
        """
        return sum(p * utility_func(c) for c, p in self.probabilities.items())

    def mix(self, other: 'Lottery', alpha: float) -> 'Lottery':
        """
        复合彩票混合：alpha * self + (1-alpha) * other
        """
        new_probs = {}
        all_outcomes = set(self.probabilities.keys()) | set(other.probabilities.keys())

        for c in all_outcomes:
            p1 = self.probabilities.get(c, 0)
            p2 = other.probabilities.get(c, 0)
            new_probs[c] = alpha * p1 + (1 - alpha) * p2

        return Lottery(new_probs)

    def __repr__(self):
        items = [f"{p:.3f}:{c.name}" for c, p in self.probabilities.items()]
        return f"Lottery({', '.join(items)})"


class ExpectedUtility:
    """
    期望效用理论实现
    包含 von Neumann-Morgenstern 公理验证
    """

    def __init__(self, utility_func: Callable[[Outcome], float]):
        """
        utility_func: 结果 -> 效用值的映射
        """
        self.u = utility_func
        self.lotteries: List[Lottery] = []

    def add_lottery(self, lottery: Lottery):
        """添加彩票到比较集"""
        self.lotteries.append(lottery)

    def rank_lotteries(self) -> List[Tuple[Lottery, float]]:
        """
        按期望效用排序彩票
        返回: [(lottery, EU), ...] 按EU降序
        """
        ranked = [(L, L.expected_utility(self.u)) for L in self.lotteries]
        return sorted(ranked, key=lambda x: x[1], reverse=True)

    def compare(self, L1: Lottery, L2: Lottery) -> int:
        """
        比较两个彩票
        返回: 1 (L1 ≻ L2), 0 (L1 ~ L2), -1 (L2 ≻ L1)
        """
        eu1 = L1.expected_utility(self.u)
        eu2 = L2.expected_utility(self.u)

        if abs(eu1 - eu2) < 1e-10:
            return 0
        return 1 if eu1 > eu2 else -1

    def check_independence_axiom(self,
                                  L1: Lottery,
                                  L2: Lottery,
                                  L3: Lottery,
                                  alpha: float = 0.5) -> bool:
        """
        验证独立性公理：
        L1 ≽ L2  ⟺  αL1 + (1-α)L3 ≽ αL2 + (1-α)L3
        """
        left_mix = L1.mix(L3, alpha)
        right_mix = L2.mix(L3, alpha)

        pref_direct = self.compare(L1, L2)
        pref_mixed = self.compare(left_mix, right_mix)

        return (pref_direct > 0 and pref_mixed >= 0) or \
               (pref_direct < 0 and pref_mixed <= 0) or \
               (pref_direct == 0 and pref_mixed == 0)

    def certainty_equivalent(self, lottery: Lottery) -> float:
        """
        确定等价物 CE
        满足: u(CE) = EU(L)
        """
        eu = lottery.expected_utility(self.u)

        # 反解 CE（假设效用函数单调）
        from scipy.optimize import brentq

        # 寻找满足 u(Outcome("CE", x)) = eu 的 x
        def equation(x):
            return self.u(Outcome("temp", x)) - eu

        try:
            ce = brentq(equation, -1e6, 1e6)
            return ce
        except ValueError:
            return None

    def risk_premium(self, lottery: Lottery) -> float:
        """
        风险溢价 = E[L] - CE(L)
        """
        ev = lottery.expected_value()
        ce = self.certainty_equivalent(lottery)
        return ev - ce if ce is not None else None


def von_neumann_morgenstern_utility(results: List[Tuple[Outcome, float]],
                                     c_star: Outcome,
                                     c_underscore: Outcome) -> Dict[Outcome, float]:
    """
    使用 vNM 方法构造效用函数

    参数:
        results: [(outcome, probability_equivalent), ...]
        c_star: 最偏好结果，u(c*) = 1
        c_underscore: 最不偏好结果，u(c_*) = 0

    返回:
        效用函数字典 {Outcome: utility}
    """
    utilities = {c_star: 1.0, c_underscore: 0.0}

    for outcome, alpha in results:
        # c ~ alpha*c* + (1-alpha)*c_
        utilities[outcome] = alpha

    return utilities


# 示例应用：圣彼得堡悖论分析
def st_petersburg_paradox():
    """
    圣彼得堡悖论：
    掷公平硬币直到出现正面，若第n次出现，赢得 2^n 元
    期望货币值 = ∞，但人们只愿付有限金额

    用对数效用解释
    """

    # 伯努利对数效用: u(x) = ln(x)
    def log_utility(c: Outcome) -> float:
        return np.log(max(c.value, 0.01))

    eu_calc = ExpectedUtility(log_utility)

    # 构造圣彼得堡彩票（截断版）
    outcomes = {}
    for n in range(1, 21):  # 最多20次
        prob = (0.5) ** n
        value = 2 ** n
        outcomes[Outcome(f"H at {n}", value)] = prob

    # 归一化（由于截断）
    total_prob = sum(outcomes.values())
    outcomes = {k: v/total_prob for k, v in outcomes.items()}

    lottery = Lottery(outcomes)

    print("圣彼得堡悖论分析")
    print("=" * 50)
    print(f"彩票: {lottery}")
    print(f"期望货币值: ${lottery.expected_value():.2f}")
    print(f"期望效用 (对数): {lottery.expected_utility(log_utility):.4f}")

    # 确定等价物
    eu = lottery.expected_utility(log_utility)
    ce = np.exp(eu)
    print(f"确定等价物: ${ce:.2f}")
    print(f"风险溢价: ${lottery.expected_value() - ce:.2f}")


# 示例：Allais 悖论
def allais_paradox():
    """
    Allais 悖论 (1953) 挑战独立性公理
    """

    # 结果
    zero = Outcome("0", 0)
    one_m = Outcome("1M", 1_000_000)
    five_m = Outcome("5M", 5_000_000)

    # 问题 1
    L1 = Lottery({one_m: 1.0})  # 确定获得 100万
    L2 = Lottery({zero: 0.01, one_m: 0.89, five_m: 0.10})  # 10%得500万，1%得0

    # 问题 2
    L3 = Lottery({zero: 0.89, one_m: 0.11})  # 11%得100万，89%得0
    L4 = Lottery({zero: 0.90, five_m: 0.10})  # 10%得500万，90%得0

    # 线性效用
    def linear_u(c: Outcome) -> float:
        return c.value / 1_000_000

    eu = ExpectedUtility(linear_u)

    print("\nAllais 悖论分析")
    print("=" * 50)
    print(f"问题1: L1 (确定100万) vs L2 (89%*1M + 10%*5M + 1%*0)")
    print(f"  EU(L1) = {L1.expected_utility(linear_u):.4f}")
    print(f"  EU(L2) = {L2.expected_utility(linear_u):.4f}")
    print(f"  EU理论预测: {'L1 ≻ L2' if eu.compare(L1, L2) > 0 else 'L2 ≻ L1'}")

    print(f"\n问题2: L3 (11%*1M + 89%*0) vs L4 (10%*5M + 90%*0)")
    print(f"  EU(L3) = {L3.expected_utility(linear_u):.4f}")
    print(f"  EU(L4) = {L4.expected_utility(linear_u):.4f}")
    print(f"  EU理论预测: {'L3 ≻ L4' if eu.compare(L3, L4) > 0 else 'L4 ≻ L3'}")

    print("\n注: 许多人选 L1 ≻ L2 但 L4 ≻ L3，违反独立性公理")


if __name__ == "__main__":
    st_petersburg_paradox()
    allais_paradox()
```
