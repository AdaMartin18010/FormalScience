## 12.4.3.1 引言

**公平分配**（fair division）研究如何在多个参与人之间分配资源，使得分配结果满足某种公平性标准。从古老的"分蛋糕"问题到现代的经济机制设计，公平分配理论结合了博弈论、算法设计和规范性分析，为资源分配提供了严格的数学框架。

---

## 12.4.3.2 分配问题的基本设定

### 12.4.3.2.1 蛋糕分割问题

**定义 12.4.3.1**（蛋糕分割）

"蛋糕" $C$ 是区间 $[0, 1]$ 的 Borel 子集。参与人 $i$ 有**价值测度** $v_i$，满足：

- 非负性：$v_i(S) \geq 0$
- 可加性：$v_i(S \cup T) = v_i(S) + v_i(T)$（不相交集合）
- 归一化：$v_i(C) = 1$

**分配**：蛋糕分割 $P = (S_1, ..., S_n)$，其中 $S_i \cap S_j = \emptyset$（$i \neq j$）且 $\bigcup_i S_i = C$。

### 12.4.3.2.2 无嫉妒性

**定义 12.4.3.2**（无嫉妒分配，Foley, 1967）

分配 $P$ 是**无嫉妒的**（envy-free），如果对所有 $i, j$：

$$v_i(S_i) \geq v_i(S_j)$$

**解释**：没有人认为别人的份额比自己的更好。

### 12.4.3.2.3 比例公平

**定义 12.4.3.3**（比例分配）

分配 $P$ 是**比例的**（proportional），如果对所有 $i$：

$$v_i(S_i) \geq \frac{1}{n}$$

## 12.4.3.3 经典分配机制

### 12.4.3.3.1 分割-选择协议

**协议 12.4.3.1**（Cut-and-Choose，二人情形）

```
1. 参与人 1 将蛋糕分成两份 S1, S2，满足 v1(S1) = v1(S2) = 1/2
2. 参与人 2 选择一份
3. 参与人 1 获得另一份
```

**定理 12.4.3.1**

分割-选择协议产生无嫉妒且比例的分配。

### 12.4.3.3.2 Dubins-Spanier 移动刀算法

**协议 12.4.3.2**（Moving Knife，n 人情形）

```
1. 在蛋糕 [0,1] 上从左向右移动刀
2. 当 v_i([0,x]) = 1/n 时，参与人 i 喊"停"
3. 第一个喊停者获得 [0,x]，离开
4. 重复直到只剩一人，获得剩余部分
```

## 12.4.3.4 离散物品分配

### 12.4.3.4.1 无嫉妒性定义

对于**不可分割物品**，无嫉妒性需要调整：

**定义 12.4.3.4**（EF1：最多一个物品的无嫉妒）

分配 $A$ 满足 **EF1**，如果对任意 $i, j$，存在物品 $g \in A_j$ 使得：

$$v_i(A_i) \geq v_i(A_j \setminus \{g\})$$

**解释**：$i$ 不嫉妒 $j$，即使允许 $j$ 丢弃一个物品。

### 12.4.3.4.2 轮询分配

**协议 12.4.3.3**（Round-Robin）

```
1. 按某种顺序排列参与人
2. 每轮每人选择一个最喜欢的剩余物品
3. 重复直到所有物品分配
```

**定理 12.4.3.2**（Caragiannis et al., 2016）

轮询分配产生 EF1 分配。

---

## 12.4.3.5 平等主义解

### 12.4.3.5.1 Max-Min 公平

**定义 12.4.3.5**（平等主义解）

**平等主义解**（Egalitarian solution）最大化最不幸参与人的效用：

$$\max_P \min_i v_i(S_i)$$

### 12.4.3.5.2  leximin 排序

**定义 12.4.3.6**（leximin 准则）

按效用排序 $(u_{(1)}, u_{(2)}, ..., u_{(n)})$，其中 $u_{(1)} \leq u_{(2)} \leq ... \leq u_{(n)}$。

分配 $P$ leximin 优于 $P'$，如果存在 $k$ 使得 $u_{(k)} > u'_{(k)}$ 且对所有 $i < k$，$u_{(i)} = u'_{(i)}$。

## 12.4.3.6 Python 实现：公平分配算法

```python
"""
公平分配理论实现
包含蛋糕分割、无嫉妒分配、平等主义解
基于 Brams & Taylor (1996) 和 Moulin (2004)
"""

from typing import List, Dict, Tuple, Callable, Optional
from dataclasses import dataclass
import numpy as np
from copy import deepcopy


@dataclass
class Valuation:
    """参与人的价值评估"""
    values: Dict[str, float]  # 物品 -> 价值

    def value_of(self, items: List[str]) -> float:
        """计算一组物品的总价值"""
        return sum(self.values.get(item, 0) for item in items)

    def value_of_single(self, item: str) -> float:
        """单个物品的价值"""
        return self.values.get(item, 0)


class FairDivision:
    """公平分配算法"""

    @staticmethod
    def cut_and_choose(valuation1: Callable[[float], float],
                       valuation2: Callable[[float], float],
                       epsilon: float = 0.01) -> Tuple[Tuple[float, float], Tuple[float, float]]:
        """
        分割-选择协议（连续情形）

        参数:
            valuation1, valuation2: [0,1] -> [0,1] 的价值密度函数

        返回:
            ((a1, b1), (a2, b2)) 两份蛋糕的区间
        """
        # 参与人 1 找到分割点 c，使得 v1([0,c]) = v1([c,1]) = 0.5
        # 数值逼近
        def integrate(f, a, b, n=1000):
            x = np.linspace(a, b, n)
            y = [f(xi) for xi in x]
            return np.trapz(y, x)

        # 二分查找分割点
        left, right = 0.0, 1.0
        c = 0.5

        for _ in range(50):
            v_left = integrate(valuation1, 0, c)
            if abs(v_left - 0.5) < epsilon:
                break
            elif v_left < 0.5:
                left = c
                c = (c + right) / 2
            else:
                right = c
                c = (left + c) / 2

        # 参与人 2 选择
        v2_left = integrate(valuation2, 0, c)
        v2_right = integrate(valuation2, c, 1)

        if v2_left >= v2_right:
            return (0, c), (c, 1)
        else:
            return (c, 1), (0, c)

    @staticmethod
    def round_robin(agents: List[str],
                   items: List[str],
                   valuations: Dict[str, Valuation],
                   order: List[int] = None) -> Dict[str, List[str]]:
        """
        轮询分配（Round-Robin）
        产生 EF1 分配
        """
        n = len(agents)
        allocation = {agent: [] for agent in agents}
        remaining = set(items)

        if order is None:
            order = list(range(n))

        turn = 0
        while remaining:
            agent_idx = order[turn % n]
            agent = agents[agent_idx]

            # 选择最喜欢的剩余物品
            best_item = max(remaining,
                          key=lambda item: valuations[agent].value_of_single(item))

            allocation[agent].append(best_item)
            remaining.remove(best_item)

            turn += 1

        return allocation

    @staticmethod
    def is_envy_free(allocation: Dict[str, List[str]],
                    valuations: Dict[str, Valuation]) -> bool:
        """检查分配是否无嫉妒"""
        agents = list(allocation.keys())

        for i in agents:
            for j in agents:
                if i == j:
                    continue

                vi_of_i = valuations[i].value_of(allocation[i])
                vi_of_j = valuations[i].value_of(allocation[j])

                if vi_of_i < vi_of_j - 1e-10:
                    return False

        return True

    @staticmethod
    def is_ef1(allocation: Dict[str, List[str]],
               valuations: Dict[str, Valuation]) -> bool:
        """检查分配是否满足 EF1"""
        agents = list(allocation.keys())

        for i in agents:
            for j in agents:
                if i == j:
                    continue

                vi_of_i = valuations[i].value_of(allocation[i])

                # 检查是否存在一个物品 g ∈ A_j 使得 vi(A_i) >= vi(A_j \ {g})
                is_ef1_for_pair = False

                if not allocation[j]:  # j 没有物品
                    is_ef1_for_pair = True
                else:
                    for g in allocation[j]:
                        bundle_without_g = [item for item in allocation[j] if item != g]
                        vi_of_j_minus_g = valuations[i].value_of(bundle_without_g)

                        if vi_of_i >= vi_of_j_minus_g - 1e-10:
                            is_ef1_for_pair = True
                            break

                if not is_ef1_for_pair:
                    return False

        return True

    @staticmethod
    def is_proportional(allocation: Dict[str, List[str]],
                       valuations: Dict[str, Valuation],
                       all_items: List[str]) -> bool:
        """检查分配是否比例公平"""
        n = len(allocation)

        for agent, bundle in allocation.items():
            vi_bundle = valuations[agent].value_of(bundle)
            vi_total = valuations[agent].value_of(all_items)

            if vi_bundle < vi_total / n - 1e-10:
                return False

        return True

    @staticmethod
    def egalitarian_allocation(agents: List[str],
                               items: List[str],
                               valuations: Dict[str, Valignment]) -> Dict[str, List[str]]:
        """
        计算平等主义解（最大化最小效用）
        使用贪心启发式（最优解是 NP-hard）
        """
        n = len(agents)
        best_allocation = None
        best_min_utility = -float('inf')

        # 简化的启发式：尝试多种随机分配
        np.random.seed(42)

        for _ in range(1000):
            # 随机打乱物品顺序
            shuffled_items = items.copy()
            np.random.shuffle(shuffled_items)

            # 轮询分配
            allocation = {agent: [] for agent in agents}
            for i, item in enumerate(shuffled_items):
                agent = agents[i % n]
                allocation[agent].append(item)

            # 计算最小效用
            min_utility = min(
                valuations[agent].value_of(allocation[agent])
                for agent in agents
            )

            if min_utility > best_min_utility:
                best_min_utility = min_utility
                best_allocation = allocation

        return best_allocation


def demo_cake_cutting():
    """演示蛋糕分割"""

    print("=" * 70)
    print("蛋糕分割：分割-选择协议")
    print("=" * 70)

    # 参与人 1 均匀分布
    v1 = lambda x: 1.0

    # 参与人 2 偏好前半部分
    v2 = lambda x: 2.0 if x < 0.5 else 0.0

    piece1, piece2 = FairDivision.cut_and_choose(v1, v2)

    print(f"\n参与人 1 的分割: [0, {piece1[1]:.3f}] 和 [{piece1[1]:.3f}, 1]")
    print(f"参与人 2 选择: {'第一部分' if piece1[0] == 0 else '第二部分'}")
    print(f"最终分配: 参与人 1 得 [{piece2[0]:.3f}, {piece2[1]:.3f}], "
          f"参与人 2 得 [{piece1[0]:.3f}, {piece1[1]:.3f}]")

    # 验证无嫉妒性
    print("\n验证:")
    print("  参与人 1: 两部分对他价值相等（都是 0.5）→ 无嫉妒")
    print("  参与人 2: 选择对自己价值更高的部分 → 无嫉妒")


def demo_discrete_allocation():
    """演示离散物品分配"""

    print("\n" + "=" * 70)
    print("离散物品分配：轮询算法")
    print("=" * 70)

    # 设置
    agents = ["Alice", "Bob", "Charlie"]
    items = ["Item" + str(i) for i in range(1, 10)]

    # 随机生成估值
    np.random.seed(42)
    valuations = {}
    for agent in agents:
        values = {item: np.random.uniform(0, 10) for item in items}
        # 归一化
        total = sum(values.values())
        values = {k: v/total for k, v in values.items()}
        valuations[agent] = Valuation(values)

    print(f"\n物品: {', '.join(items)}")
    print(f"参与人: {', '.join(agents)}")

    # 轮询分配
    allocation = FairDivision.round_robin(agents, items, valuations)

    print("\n轮询分配结果:")
    for agent in agents:
        bundle = allocation[agent]
        value = valuations[agent].value_of(bundle)
        print(f"  {agent}: {bundle} (价值: {value:.3f})")

    # 验证性质
    print("\n公平性验证:")

    is_ef = FairDivision.is_envy_free(allocation, valuations)
    print(f"  无嫉妒 (EF): {'✓' if is_ef else '✗'}")

    is_ef1 = FairDivision.is_ef1(allocation, valuations)
    print(f"  EF1: {'✓' if is_ef1 else '✗'}")

    is_prop = FairDivision.is_proportional(allocation, valuations, items)
    print(f"  比例公平: {'✓' if is_prop else '✗'}")

    # 平等主义解
    print("\n" + "-" * 50)
    print("平等主义解（近似）:")

    egal_allocation = FairDivision.egalitarian_allocation(agents, items, valuations)

    for agent in agents:
        bundle = egal_allocation[agent]
        value = valuations[agent].value_of(bundle)
        print(f"  {agent}: {len(bundle)} 个物品, 价值: {value:.3f}")

    min_utility = min(valuations[a].value_of(egal_allocation[a]) for a in agents)
    print(f"\n  最小效用: {min_utility:.3f}")
    print(f"  这是公平性最强的分配（最大化最不幸者效用）")


if __name__ == "__main__":
    demo_cake_cutting()
    demo_discrete_allocation()
```
