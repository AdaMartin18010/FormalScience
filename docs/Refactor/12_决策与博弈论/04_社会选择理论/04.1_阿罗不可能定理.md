# 12.4.1 阿罗不可能定理

## 12.4.1.1 引言

**阿罗不可能定理**（Arrow's Impossibility Theorem）是社会选择理论中最著名的结果，由 Kenneth Arrow (1951) 在其博士论文中证明。该定理表明：在满足一系列看似合理的条件下，不存在能够将所有个体偏好聚合为社会偏好的非独裁性方法。这一结果深刻揭示了民主决策的内在困难，为后续的社会选择研究奠定了基础。

---

## 12.4.1.2 社会偏好聚合框架

### 12.4.1.2.1 基本设定

**定义 12.4.1.1**（偏好域）

- $A = \{a, b, c, ...\}$：候选方案集，$|A| = m \geq 3$
- $N = \{1, 2, ..., n\}$：参与人集合
- $\mathcal{L}$：$A$ 上的**线性序**（完备、传递、反对称）集合

**定义 12.4.1.2**（偏好剖面）

**偏好剖面** $R = (R_1, ..., R_n) \in \mathcal{L}^n$ 是所有参与人偏好的组合。

### 12.4.1.2.2 社会福利函数

**定义 12.4.1.3**（社会福利函数）

**社会福利函数**（Social Welfare Function, SWF）$F: \mathcal{L}^n \to \mathcal{L}$ 将偏好剖面映射到社会偏好排序。

**对比**：社会选择函数 $f: \mathcal{L}^n \to A$ 只选出一个获胜者。

---

## 12.4.1.3 阿罗条件

### 12.4.1.3.1 条件陈述

Arrow 提出以下四个条件：

**条件 12.4.1.1**（U：全域性 / Unrestricted Domain）

$F$ 的定义域是所有可能的偏好剖面：$\text{dom}(F) = \mathcal{L}^n$

**条件 12.4.1.2**（P：帕累托效率 / Pareto Efficiency）

如果所有参与人都偏好 $a$ 胜过 $b$，则社会也偏好 $a$ 胜过 $b$：

$$a \succ_i b, \forall i \Rightarrow a \succ_F b$$

**条件 12.4.1.3**（I：独立性 / Independence of Irrelevant Alternatives, IIA）

社会对 $a$ 和 $b$ 的排序仅依赖于个体对 $a$ 和 $b$ 的排序：

如果在两个剖面 $R$ 和 $R'$ 中，所有参与人对 $\{a, b\}$ 的排序相同，则 $F(R)$ 和 $F(R')$ 对 $\{a, b\}$ 的排序也相同。

**条件 12.4.1.4**（D：非独裁性 / Non-Dictatorship）

不存在参与人 $i$ 使得无论他人偏好如何，社会偏好总与 $i$ 的偏好一致：

$$\neg \exists i: \forall R, \forall a, b: a \succ_i b \Rightarrow a \succ_F b$$

### 12.4.1.3.2 条件的直观解释

| 条件 | 含义 | 违反后果 |
|------|------|----------|
| 全域性 | 接受任何偏好组合 | 限制偏好域可能允许可能性结果 |
| 帕累托 | 全体一致时应采纳 | 否则忽略 unanimous 决策 |
| IIA | 两两比较独立于其他选项 | 可能产生策略操纵或循环 |
| 非独裁 | 没有单一决策者 | 独裁是最简单的聚合方式 |

## 12.4.1.4 阿罗不可能定理

### 12.4.1.4.1 定理陈述

**定理 12.4.1.1**（Arrow's Impossibility Theorem, 1951）

若 $|A| \geq 3$，则不存在同时满足全域性、帕累托效率、IIA 和非独裁性的社会福利函数。

### 12.4.1.4.2 完整证明

**证明**：

**引理 12.4.1.1**（决定性集合的封闭性）

设 $J \subseteq N$ 对 $(a, b)$ 是**决定性的**（即 $a \succ_i b, \forall i \in J$ 且 $b \succ_i a, \forall i \notin J$ 蕴含 $a \succ_F b$）。则对任意 $x, y$，$J$ 对 $(x, y)$ 也是决定性的。

**证明**：设 $J$ 对 $(a, b)$ 决定性。考虑剖面：

- $i \in J$: $x \succ a \succ b \succ y$
- $i \notin J$: $a \succ b \succ y \succ x$（且 $b \succ a$）

由帕累托，$a \succ_F b$。
由 IIA（比较 $a, b$），社会偏好 $a \succ b$ 仅依赖于个体对 $\{a, b\}$ 的排序，即 $J$ 决定性 $\Rightarrow a \succ_F b$。

由传递性：$x \succ_F a \succ_F b \succ_F y \Rightarrow x \succ_F y$。
由 IIA（比较 $x, y$），$J$ 对 $(x, y)$ 决定性。∎

**引理 12.4.1.2**（决定性集合的交集）

设 $J$ 和 $K$ 对 $(a, b)$ 都是决定性的。则 $J \cap K$ 也对 $(a, b)$ 决定性。

**证明**：考虑剖面：

- $i \in J \cap K$: $a \succ b$
- $i \in J \setminus K$: $b \succ a$（但 $J$ 决定性要求...需调整）

构造：

- $i \in J \cap K$: $a \succ x \succ b$
- $i \in J \setminus K$: $x \succ a \succ b$
- $i \in K \setminus J$: $a \succ b \succ x$
- $i \notin J \cup K$: $b \succ a \succ x$

由 $J$ 对 $(a, x)$ 决定性：$a \succ_F x$
由 $K$ 对 $(x, b)$ 决定性：$x \succ_F b$
因此 $a \succ_F b$。

由 IIA，仅依赖于 $J \cap K$ 中 $a \succ b$，故 $J \cap K$ 对 $(a, b)$ 决定性。∎

**引理 12.4.1.3**（最小决定性集合）

由引理 12.4.1.2，存在唯一的最小决定性集合 $J^*$。若 $|J^*| \geq 2$，设 $i \in J^*$，则 $J^* \setminus \{i\}$ 和 $\{i\}$ 都不是决定性的。

考虑剖面：

- $i$: $b \succ a \succ c$
- $J^* \setminus \{i\}$: $a \succ c \succ b$
- $N \setminus J^*$: $c \succ b \succ a$

$J^*$ 对 $(a, c)$ 决定性 $\Rightarrow a \succ_F c$。
若 $c \succ_F b$，则传递性 $a \succ_F b$。
由 IIA，$J^*$ 中 $a \succ b$ 即可，与 $J^*$ 最小性矛盾。

因此 $b \succ_F c$。
由 IIA，仅依赖于 $i$ 的 $b \succ c$，故 $\{i\}$ 对 $(b, c)$ 决定性。
由引理 12.4.1.1，$\{i\}$ 对所有对决定性，即 $i$ 是独裁者。

**主定理**：存在独裁者，违反非独裁性条件。∎

---

## 12.4.1.5 条件的放松与可能性

### 12.4.1.5.1 限制偏好域

**单峰偏好**（Black, 1948）：若所有偏好都是单峰的，多数规则满足阿罗条件（除全域性）。

**准传递性**：社会偏好只需准传递（无循环），不要求完备。此时存在满足其他条件的 SWF。

### 12.4.1.5.2 基数效用

若允许效用的人际比较（基数效用），社会福利函数存在（如功利主义、罗尔斯主义）。

## 12.4.1.6 Python 实现：阿罗定理验证

```python
"""
阿罗不可能定理的验证与分析
演示各种投票规则如何违反阿罗条件
基于 Arrow (1951) 和 Sen (1970)
"""

from typing import List, Dict, Tuple, Optional, Callable
from dataclasses import dataclass
from itertools import combinations, permutations
import numpy as np


@dataclass
class Preference:
    """个体偏好"""
    ranking: List[str]  # 从最优到最差

    def prefers(self, a: str, b: str) -> bool:
        """检查是否偏好 a 胜过 b"""
        return self.ranking.index(a) < self.ranking.index(b)

    def pairwise_ranking(self, alternatives: List[str]) -> List[Tuple[str, str]]:
        """获取两两比较的结果列表"""
        pairs = []
        for i, a in enumerate(alternatives):
            for b in alternatives[i+1:]:
                if self.prefers(a, b):
                    pairs.append((a, b))
                else:
                    pairs.append((b, a))
        return pairs


class SocialWelfareFunction:
    """社会福利函数"""

    def __init__(self, name: str, rule: Callable):
        self.name = name
        self.rule = rule

    def aggregate(self, profile: List[Preference]) -> List[str]:
        """聚合偏好"""
        return self.rule(profile)


class ArrowsConditions:
    """阿罗条件检验器"""

    @staticmethod
    def check_unrestricted_domain(swffunc: SocialWelfareFunction,
                                   alternatives: List[str],
                                   n_voters: int) -> bool:
        """
        检验是否满足全域性
        检查是否对所有可能的偏好剖面都有定义
        """
        try:
            # 生成一些随机偏好剖面
            for _ in range(100):
                profile = []
                for _ in range(n_voters):
                    ranking = np.random.permutation(alternatives).tolist()
                    profile.append(Preference(ranking))

                result = swffunc.aggregate(profile)

                # 检查结果是否为有效排序
                if set(result) != set(alternatives):
                    return False

            return True
        except Exception:
            return False

    @staticmethod
    def check_pareto_efficiency(swffunc: SocialWelfareFunction,
                                 alternatives: List[str]) -> Tuple[bool, str]:
        """
        检验帕累托效率
        如果所有人都偏好 a 胜过 b，社会也应如此
        """
        n_voters = 3
        a, b = alternatives[0], alternatives[1]
        others = alternatives[2:]

        # 构造所有人都偏好 a > b 的剖面
        profile = []
        for _ in range(n_voters):
            ranking = [a, b] + others
            profile.append(Preference(ranking))

        social_ranking = swffunc.aggregate(profile)

        # 检查社会偏好 a > b
        if social_ranking.index(a) < social_ranking.index(b):
            return True, "通过"
        else:
            return False, f"违反: 全体偏好 {a}>{b} 但社会排序为 {social_ranking}"

    @staticmethod
    def check_iiavswffunc: SocialWelfareFunction,
                         alternatives: List[str]) -> Tuple[bool, str]:
        """
        检验独立性（IIA）
        社会对 {a, b} 的排序应仅依赖于个体对 {a, b} 的排序
        """
        if len(alternatives) < 3:
            return True, "跳过（候选不足）"

        a, b, c = alternatives[0], alternatives[1], alternatives[2]

        # 剖面 1
        profile1 = [
            Preference([a, b, c]),
            Preference([a, b, c]),
            Preference([b, a, c]),
        ]

        # 剖面 2：改变 c 的位置，但保持 a, b 的相对顺序
        profile2 = [
            Preference([a, c, b]),
            Preference([a, c, b]),
            Preference([b, c, a]),
        ]

        ranking1 = swffunc.aggregate(profile1)
        ranking2 = swffunc.aggregate(profile2)

        # 检查社会对 {a, b} 的排序是否一致
        def relative_order(ranking, x, y):
            return ranking.index(x) < ranking.index(y)

        if relative_order(ranking1, a, b) == relative_order(ranking2, a, b):
            return True, "通过"
        else:
            return False, f"违反: 剖面1社会排序{ranking1}，剖面2社会排序{ranking2}，但个体对{{{a},{b}}}排序相同"

    @staticmethod
    def check_non_dictatorship(swffunc: SocialWelfareFunction,
                                alternatives: List[str],
                                n_voters: int) -> Tuple[bool, str]:
        """
        检验非独裁性
        检查是否存在独裁者
        """
        # 检查每个可能的参与人是否为独裁者
        for potential_dictator in range(n_voters):
            is_dictator = True

            # 测试多个剖面
            for _ in range(50):
                profile = []
                for i in range(n_voters):
                    ranking = np.random.permutation(alternatives).tolist()
                    profile.append(Preference(ranking))

                social_ranking = swffunc.aggregate(profile)
                dictator_ranking = profile[potential_dictator].ranking

                # 检查社会排序是否与独裁者一致
                if social_ranking != dictator_ranking:
                    is_dictator = False
                    break

            if is_dictator:
                return False, f"参与人 {potential_dictator} 是独裁者"

        return True, "无非独裁性违反"

    @staticmethod
    def full_analysis(swffunc: SocialWelfareFunction,
                     alternatives: List[str],
                     n_voters: int = 3) -> Dict[str, Tuple[bool, str]]:
        """完整的阿罗条件检验"""
        results = {}

        results['全域性 (U)'] = (
            ArrowsConditions.check_unrestricted_domain(swffunc, alternatives, n_voters),
            "定义域检查"
        )

        results['帕累托 (P)'] = ArrowsConditions.check_pareto_efficiency(swffunc, alternatives)
        results['IIA'] = ArrowsConditions.check_iia(swffunc, alternatives)
        results['非独裁 (D)'] = ArrowsConditions.check_non_dictatorship(swffunc, alternatives, n_voters)

        return results


# 定义一些社会福利函数

def plurality_swf(profile: List[Preference]) -> List[str]:
    """
    多数制社会福利函数
    按第一选择得票数排序
    """
    votes = {}
    for pref in profile:
        top = pref.ranking[0]
        votes[top] = votes.get(top, 0) + 1

    # 按得票排序（得票相同则按名字）
    all_alternatives = profile[0].ranking
    return sorted(all_alternatives, key=lambda a: (-votes.get(a, 0), a))


def borda_swf(profile: List[Preference]) -> List[str]:
    """
    Borda 计数社会福利函数
    """
    m = len(profile[0].ranking)
    scores = {}

    for pref in profile:
        for i, alt in enumerate(pref.ranking):
            scores[alt] = scores.get(alt, 0) + (m - 1 - i)

    return sorted(scores.keys(), key=lambda a: -scores[a])


def pairwise_majority_swf(profile: List[Preference]) -> List[str]:
    """
    两两多数制（Condorcet 方法）
    按 Copeland 得分排序（胜场数 - 负场数）
    """
    alternatives = profile[0].ranking
    n = len(alternatives)

    # 计算两两比较
    wins = {a: 0 for a in alternatives}

    for a, b in combinations(alternatives, 2):
        a_votes = sum(1 for pref in profile if pref.prefers(a, b))
        b_votes = len(profile) - a_votes

        if a_votes > b_votes:
            wins[a] += 1
            wins[b] -= 1
        elif b_votes > a_votes:
            wins[b] += 1
            wins[a] -= 1

    return sorted(alternatives, key=lambda a: -wins[a])


def demo_arrows_theorem():
    """演示阿罗不可能定理"""

    print("=" * 70)
    print("阿罗不可能定理验证")
    print("=" * 70)

    alternatives = ["A", "B", "C"]
    n_voters = 3

    # 定义 SWF
    swfs = [
        SocialWelfareFunction("多数制 (Plurality)", plurality_swf),
        SocialWelfareFunction("Borda 计数", borda_swf),
        SocialWelfareFunction("两两多数 (Copeland)", pairwise_majority_swf),
    ]

    for swf in swfs:
        print(f"\n{swf.name}")
        print("-" * 50)

        results = ArrowsConditions.full_analysis(swf, alternatives, n_voters)

        for condition, (passed, msg) in results.items():
            status = "✓" if passed else "✗"
            print(f"  {status} {condition}: {msg}")

        # 统计
        n_passed = sum(1 for p, _ in results.values() if p)
        print(f"  通过 {n_passed}/{len(results)} 个条件")

    print("\n" + "=" * 70)
    print("结论")
    print("-" * 50)
    print("""
阿罗不可能定理表明：
- 当 |A| ≥ 3 时，不存在同时满足 U、P、IIA、D 的社会福利函数
- 上述投票规则都至少违反一个条件（通常是 IIA）
- 放宽某些条件（如限制偏好域为单峰）可获得可能性结果
    """)


if __name__ == "__main__":
    demo_arrows_theorem()
```
