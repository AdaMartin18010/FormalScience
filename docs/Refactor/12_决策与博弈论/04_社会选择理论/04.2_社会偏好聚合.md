# 12.4.2 社会偏好聚合

## 12.4.2.1 引言

社会选择理论的核心问题是：**如何从个体偏好导出合理的社会偏好？** Arrow 不可能定理揭示了在一般条件下这一问题的困难性。本章探讨在放宽某些条件后获得的可能性结果，包括限制偏好域、使用基数信息以及探索各类聚合规则的性质。

---

## 12.4.2.2 限制偏好域的可能性

### 12.4.2.2.1 单峰偏好域

**定义 12.4.2.1**（单峰偏好）

设 $<$ 是候选方案 $A$ 上的线性序。偏好 $\succ_i$ 是**单峰的**（single-peaked），如果存在最偏好点 $p_i \in A$（峰值），使得：

$$a < b < p_i \text{ 或 } p_i < b < a \Rightarrow b \succ_i a$$

**解释**：偏好沿 $<$ 序"山峰状"，远离峰值效用递减。

### 12.4.2.2.2 Median 投票者定理

**定理 12.4.2.1**（Median Voter Theorem, Black, 1948）

若所有偏好都是单峰的，则 Condorcet 赢家存在且等于 median 峰值。

**证明**：设 $p^*$ 是 median 峰值。对任意 $a < p^*$：

- 峰值 $\geq p^*$ 的参与人（至少半数）偏好 $p^* \succ a$
- 因此 $p^*$ 在多数投票中击败 $a$

同理对 $a > p^*$。∎

### 12.4.2.2.3 单峰域上的可能性

**定理 12.4.2.2**（Moulin, 1980）

在单峰偏好域上，以下规则满足 Arrow 条件（除全域性）：

- **中位点规则**：选择 median 峰值
- **Phragmen 规则**：广义 median 选择

## 12.4.2.3 基数效用聚合

### 12.4.2.3.1 功利主义

**定义 12.4.2.2**（功利主义 SWF）

设 $u_i(a)$ 是参与人 $i$ 对 $a$ 的效用。功利主义社会福利函数：

$$W(a) = \sum_{i=1}^n u_i(a)$$

**性质**：

- 满足帕累托效率
- 满足 IIA（基数形式）
- 需要效用的人际比较

### 12.4.2.3.2 罗尔斯主义

**定义 12.4.2.3**（最大最小 / Maximin）

罗尔斯主义（Rawlsian）SWF：

$$W(a) = \min_{i \in N} u_i(a)$$

**性质**：

- 关注最不幸的参与人
- 满足帕累托效率
- 强烈的风险厌恶

---

## 12.4.2.4 聚合规则分类

### 12.4.2.4.1 计分规则

**定义 12.4.2.4**（计分规则）

**计分规则**由向量 $(s_1, s_2, ..., s_m)$ 定义，其中 $s_1 \geq s_2 \geq ... \geq s_m$ 且 $s_1 > s_m$。

排名为 $k$ 的候选人得 $s_k$ 分，总分最高者获胜。

**特例**：

| 规则 | 计分向量 |
|------|----------|
| 多数制 | $(1, 0, ..., 0)$ |
| Borda | $(m-1, m-2, ..., 0)$ |
| 反对制 | $(0, ..., 0, -1)$ |

### 12.4.2.4.2 Condorcet 一致性

**定义 12.4.2.5**（Condorcet 一致性）

投票规则 $f$ 是**Condorcet 一致的**，如果 Condorcet 赢家存在时 $f$ 选择该赢家。

**定理 12.4.2.3**（Fishburn, 1977）

不存在同时满足 Condorcet 一致性和策略证明性的计分规则。

---

## 12.4.2.5 不可能结果的扩展

### 12.4.2.5.1 Gibbard-Satterthwaite 定理

**定理 12.4.2.4**（Gibbard, 1973; Satterthwaite, 1975）

设 $|A| \geq 3$，$f$ 是满射且非独裁的社会选择函数。则 $f$ 不是策略证明的。

**与 Arrow 定理的关系**：

- Arrow：社会福利函数的聚合问题
- Gibbard-Satterthwaite：社会选择函数的激励相容问题

### 12.4.2.5.2 Sen 不可能定理

**定理 12.4.2.5**（Sen, 1970）

不存在同时满足以下条件的社会决策函数：

1. 无限制域
2. 帕累托原则
3. 最小自由（Liberalism）：每个参与人对至少一对方案有决定权

**解释**：帕累托效率与个人自由的冲突。

## 12.4.2.6 Python 实现：偏好聚合规则

```python
"""
社会偏好聚合规则实现
包含各种投票规则、基数效用聚合
基于 Moulin (1988) 和 Gaertner (2009)
"""

from typing import List, Dict, Tuple, Callable
from dataclasses import dataclass
from collections import defaultdict
import numpy as np
from itertools import combinations


@dataclass
class Preference:
    """偏好排序"""
    ranking: List[str]
    utilities: Dict[str, float] = None  # 可选的基数效用

    def prefers(self, a: str, b: str) -> bool:
        return self.ranking.index(a) < self.ranking.index(b)

    def rank_of(self, a: str) -> int:
        return self.ranking.index(a)

    def utility(self, a: str) -> float:
        if self.utilities and a in self.utilities:
            return self.utilities[a]
        # 默认：排名越高效用越高
        return len(self.ranking) - self.rank_of(a)


class AggregationRule:
    """聚合规则基类"""

    def __init__(self, name: str):
        self.name = name

    def aggregate(self, profile: List[Preference]) -> str:
        """返回获胜者"""
        raise NotImplementedError

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        """返回完整的社会排序"""
        raise NotImplementedError


class PluralityRule(AggregationRule):
    """多数制规则"""

    def __init__(self):
        super().__init__("Plurality")

    def aggregate(self, profile: List[Preference]) -> str:
        votes = defaultdict(int)
        for pref in profile:
            votes[pref.ranking[0]] += 1
        return max(votes.items(), key=lambda x: x[1])[0]

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        votes = defaultdict(int)
        for pref in profile:
            votes[pref.ranking[0]] += 1

        alternatives = profile[0].ranking
        return sorted(alternatives, key=lambda a: -votes[a])


class BordaRule(AggregationRule):
    """Borda 计数"""

    def __init__(self):
        super().__init__("Borda Count")

    def aggregate(self, profile: List[Preference]) -> str:
        m = len(profile[0].ranking)
        scores = defaultdict(int)

        for pref in profile:
            for i, alt in enumerate(pref.ranking):
                scores[alt] += m - 1 - i

        return max(scores.items(), key=lambda x: x[1])[0]

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        m = len(profile[0].ranking)
        scores = defaultdict(int)

        for pref in profile:
            for i, alt in enumerate(pref.ranking):
                scores[alt] += m - 1 - i

        alternatives = profile[0].ranking
        return sorted(alternatives, key=lambda a: -scores[a])


class CondorcetRule(AggregationRule):
    """Condorcet 方法（Copeland 规则）"""

    def __init__(self):
        super().__init__("Condorcet (Copeland)")

    def pairwise_matrix(self, profile: List[Preference]) -> Dict[Tuple[str, str], int]:
        """计算两两比较矩阵"""
        alternatives = profile[0].ranking
        wins = defaultdict(int)

        for a, b in combinations(alternatives, 2):
            a_wins = sum(1 for pref in profile if pref.prefers(a, b))
            b_wins = len(profile) - a_wins

            if a_wins > b_wins:
                wins[(a, b)] = a_wins
                wins[(b, a)] = -a_wins
            elif b_wins > a_wins:
                wins[(b, a)] = b_wins
                wins[(a, b)] = -b_wins
            else:
                wins[(a, b)] = wins[(b, a)] = 0

        return wins

    def aggregate(self, profile: List[Preference]) -> str:
        alternatives = profile[0].ranking
        copeland_scores = {a: 0 for a in alternatives}

        for a, b in combinations(alternatives, 2):
            a_wins = sum(1 for pref in profile if pref.prefers(a, b))
            b_wins = len(profile) - a_wins

            if a_wins > b_wins:
                copeland_scores[a] += 1
                copeland_scores[b] -= 1
            elif b_wins > a_wins:
                copeland_scores[b] += 1
                copeland_scores[a] -= 1

        return max(copeland_scores.items(), key=lambda x: x[1])[0]

    def find_condorcet_winner(self, profile: List[Preference]) -> Tuple[Optional[str], bool]:
        """寻找 Condorcet 赢家（如果存在）"""
        alternatives = profile[0].ranking
        n = len(profile)

        for a in alternatives:
            is_winner = True
            for b in alternatives:
                if a == b:
                    continue
                a_wins = sum(1 for pref in profile if pref.prefers(a, b))
                if a_wins <= n / 2:
                    is_winner = False
                    break

            if is_winner:
                return a, True

        return None, False

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        alternatives = profile[0].ranking
        copeland_scores = {a: 0 for a in alternatives}

        for a, b in combinations(alternatives, 2):
            a_wins = sum(1 for pref in profile if pref.prefers(a, b))
            b_wins = len(profile) - a_wins

            if a_wins > b_wins:
                copeland_scores[a] += 1
                copeland_scores[b] -= 1
            elif b_wins > a_wins:
                copeland_scores[b] += 1
                copeland_scores[a] -= 1

        return sorted(alternatives, key=lambda a: -copeland_scores[a])


class InstantRunoffRule(AggregationRule):
    """Instant Runoff 投票"""

    def __init__(self):
        super().__init__("Instant Runoff")

    def aggregate(self, profile: List[Preference]) -> str:
        alternatives = set(profile[0].ranking)
        n = len(profile)

        while len(alternatives) > 1:
            # 计算第一选择票数
            first_choice = defaultdict(int)
            for pref in profile:
                for alt in pref.ranking:
                    if alt in alternatives:
                        first_choice[alt] += 1
                        break

            # 检查是否有 majority 赢家
            for alt, votes in first_choice.items():
                if votes > n / 2:
                    return alt

            # 淘汰得票最少者
            min_votes = min(first_choice.values())
            losers = [alt for alt, v in first_choice.items() if v == min_votes]
            alternatives -= set(losers)

        return alternatives.pop()

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        # 简化为返回 aggregate
        return [self.aggregate(profile)]


class UtilitarianRule(AggregationRule):
    """功利主义规则（需要基数效用）"""

    def __init__(self):
        super().__init__("Utilitarian")

    def aggregate(self, profile: List[Preference]) -> str:
        alternatives = profile[0].ranking
        total_utilities = {a: 0.0 for a in alternatives}

        for pref in profile:
            for alt in alternatives:
                total_utilities[alt] += pref.utility(alt)

        return max(total_utilities.items(), key=lambda x: x[1])[0]

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        alternatives = profile[0].ranking
        total_utilities = {a: 0.0 for a in alternatives}

        for pref in profile:
            for alt in alternatives:
                total_utilities[alt] += pref.utility(alt)

        return sorted(alternatives, key=lambda a: -total_utilities[a])


class RawlsianRule(AggregationRule):
    """罗尔斯主义规则（最大最小）"""

    def __init__(self):
        super().__init__("Rawlsian (Maximin)")

    def aggregate(self, profile: List[Preference]) -> str:
        alternatives = profile[0].ranking
        min_utilities = {a: float('inf') for a in alternatives}

        for pref in profile:
            for alt in alternatives:
                min_utilities[alt] = min(min_utilities[alt], pref.utility(alt))

        return max(min_utilities.items(), key=lambda x: x[1])[0]

    def social_ranking(self, profile: List[Preference]) -> List[str]:
        alternatives = profile[0].ranking
        min_utilities = {a: float('inf') for a in alternatives}

        for pref in profile:
            for alt in alternatives:
                min_utilities[alt] = min(min_utilities[alt], pref.utility(alt))

        return sorted(alternatives, key=lambda a: -min_utilities[a])


def demo_aggregation_rules():
    """演示各种聚合规则"""

    print("=" * 70)
    print("社会偏好聚合规则演示")
    print("=" * 70)

    # 示例偏好剖面
    profile = [
        Preference(["A", "B", "C"]),
        Preference(["A", "B", "C"]),
        Preference(["B", "C", "A"]),
        Preference(["C", "A", "B"]),
        Preference(["C", "B", "A"]),
    ]

    print("\n偏好剖面:")
    for i, pref in enumerate(profile):
        print(f"  {i+1}: {' > '.join(pref.ranking)}")

    # 各种规则
    rules = [
        PluralityRule(),
        BordaRule(),
        CondorcetRule(),
        InstantRunoffRule(),
    ]

    print("\n各规则结果:")
    for rule in rules:
        winner = rule.aggregate(profile)
        ranking = rule.social_ranking(profile)
        print(f"\n  {rule.name}:")
        print(f"    获胜者: {winner}")
        print(f"    排序: {' > '.join(ranking)}")

    # Condorcet 赢家检查
    condorcet = CondorcetRule()
    cw, exists = condorcet.find_condorcet_winner(profile)
    print(f"\n  Condorcet 赢家存在? {'是' if exists else '否'} ({cw})")

    # 基数效用示例
    print("\n" + "=" * 70)
    print("基数效用聚合")
    print("=" * 70)

    # 带基数效用的偏好
    cardinal_profile = [
        Preference(["A", "B", "C"], {"A": 10, "B": 5, "C": 0}),
        Preference(["A", "B", "C"], {"A": 8, "B": 6, "C": 2}),
        Preference(["B", "C", "A"], {"A": 2, "B": 10, "C": 5}),
    ]

    print("\n基数效用:")
    for i, pref in enumerate(cardinal_profile):
        utils = ", ".join([f"{a}:{pref.utilities[a]}" for a in pref.ranking])
        print(f"  {i+1}: {utils}")

    utilitarian = UtilitarianRule()
    rawlsian = RawlsianRule()

    print(f"\n  功利主义 (总效用最大): {utilitarian.aggregate(cardinal_profile)}")
    print(f"  罗尔斯主义 (最大最小): {rawlsian.aggregate(cardinal_profile)}")


if __name__ == "__main__":
    demo_aggregation_rules()
```
