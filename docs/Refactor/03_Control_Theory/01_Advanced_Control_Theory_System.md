## 1. 引言：现代控制理论的发展

### 1.1 控制理论的历史演进

控制理论从19世纪的经典控制理论发展到现代的高级控制理论，经历了深刻的变革。从PID控制到现代控制理论，再到智能控制理论，控制理论已经成为现代工程和科学的重要基础。

### 1.2 控制理论的重要性

控制理论为系统设计、自动化和工程应用提供了理论基础，是现代工程科学不可或缺的核心理论。

## 2. 控制系统基础

### 2.1 控制系统的基本定义

**定义 2.1.1** (控制系统) 控制系统是一个四元组 $\mathcal{S} = (X, U, f, g)$，其中：

- $X \subseteq \mathbb{R}^n$ 是状态空间
- $U \subseteq \mathbb{R}^m$ 是控制输入空间
- $f: X \times U \rightarrow X$ 是状态转移函数
- $g: X \rightarrow Y$ 是输出函数

**定义 2.1.2** (连续时间系统) 连续时间系统的状态方程：

$$\dot{x}(t) = f(x(t), u(t))$$
$$y(t) = g(x(t))$$

**定义 2.1.3** (离散时间系统) 离散时间系统的状态方程：

$$x(k+1) = f(x(k), u(k))$$
$$y(k) = g(x(k))$$

### 2.2 系统性质

**定义 2.2.1** (可控性) 系统在状态 $x$ 可控，如果存在控制序列将 $x$ 转移到任意目标状态。

**定义 2.2.2** (可观性) 系统可观，如果从输出可以唯一确定初始状态。

**定义 2.2.3** (稳定性) 系统在平衡点 $x_e$ 稳定，如果对于任意 $\epsilon > 0$，存在 $\delta > 0$ 使得 $\|x(0) - x_e\| < \delta$  implies $\|x(t) - x_e\| < \epsilon$ for all $t \geq 0$。

**定理 2.2.1** (李雅普诺夫稳定性) 如果存在正定函数 $V(x)$ 使得 $\dot{V}(x) \leq 0$，则系统稳定。

**证明** 通过李雅普诺夫直接方法：

1. $V(x) > 0$ for $x \neq x_e$
2. $\dot{V}(x) \leq 0$ 保证 $V(x(t))$ 不增
3. 因此 $\|x(t)\|$ 有界，系统稳定

## 3. 线性控制系统

### 3.1 线性系统模型

**定义 3.1.1** (线性系统) 线性系统具有以下形式：

$$\dot{x}(t) = Ax(t) + Bu(t)$$
$$y(t) = Cx(t) + Du(t)$$

其中 $A \in \mathbb{R}^{n \times n}$, $B \in \mathbb{R}^{n \times m}$, $C \in \mathbb{R}^{p \times n}$, $D \in \mathbb{R}^{p \times m}$。

**定义 3.1.2** (传递函数) 线性系统的传递函数：

$$G(s) = C(sI - A)^{-1}B + D$$

**定理 3.1.1** (线性系统可控性) 线性系统可控当且仅当可控性矩阵 $[B \ AB \ A^2B \ \cdots \ A^{n-1}B]$ 满秩。

**证明** 通过凯莱-哈密顿定理和可控性定义。

### 3.2 线性状态反馈

**定义 3.2.1** (状态反馈) 状态反馈控制律：

$$u(t) = -Kx(t)$$

其中 $K \in \mathbb{R}^{m \times n}$ 是反馈增益矩阵。

**定义 3.2.2** (闭环系统) 闭环系统：

$$\dot{x}(t) = (A - BK)x(t)$$

**定理 3.2.1** (极点配置) 如果系统可控，则可以通过状态反馈任意配置闭环极点。

**证明** 通过可控性矩阵和极点配置算法。

### 3.3 线性二次型调节器

**定义 3.3.1** (LQR问题) 线性二次型调节器问题：

$$\min_{u} \int_0^{\infty} [x^T(t)Qx(t) + u^T(t)Ru(t)]dt$$

其中 $Q \geq 0$, $R > 0$。

**定理 3.3.1** (LQR解) LQR问题的最优控制律：

$$u^*(t) = -R^{-1}B^TPx(t)$$

其中 $P$ 是代数黎卡提方程的解：

$$A^TP + PA - PBR^{-1}B^TP + Q = 0$$

**证明** 通过变分法和最优性条件。

## 4. 非线性控制系统

### 4.1 非线性系统模型

**定义 4.1.1** (非线性系统) 非线性系统：

$$\dot{x}(t) = f(x(t), u(t))$$
$$y(t) = g(x(t))$$

其中 $f$ 和 $g$ 是非线性函数。

**定义 4.1.2** (局部线性化) 在平衡点 $(x_e, u_e)$ 的局部线性化：

$$\delta \dot{x} = A\delta x + B\delta u$$
$$\delta y = C\delta x$$

其中 $A = \frac{\partial f}{\partial x}|_{(x_e, u_e)}$, $B = \frac{\partial f}{\partial u}|_{(x_e, u_e)}$。

### 4.2 非线性稳定性

**定义 4.2.1** (李雅普诺夫函数) 李雅普诺夫函数 $V(x)$ 满足：

1. $V(0) = 0$
2. $V(x) > 0$ for $x \neq 0$
3. $\dot{V}(x) \leq 0$ (稳定) 或 $\dot{V}(x) < 0$ (渐近稳定)

**定理 4.2.1** (李雅普诺夫稳定性定理) 如果存在李雅普诺夫函数，则系统稳定。

**定理 4.2.2** (李雅普诺夫渐近稳定性定理) 如果存在严格李雅普诺夫函数，则系统渐近稳定。

### 4.3 反馈线性化

**定义 4.3.1** (反馈线性化) 通过状态反馈和坐标变换将非线性系统转换为线性系统。

**定义 4.3.2** (相对度) 输出 $y$ 相对于输入 $u$ 的相对度是使得 $\frac{\partial y^{(r)}}{\partial u} \neq 0$ 的最小整数 $r$。

**定理 4.3.1** (反馈线性化条件) 如果相对度 $r = n$，则系统可以通过反馈线性化。

**证明** 通过李导数计算和坐标变换。

## 5. 最优控制理论

### 5.1 变分法

**定义 5.1.1** (变分问题) 变分问题：

$$\min_{u} J = \int_{t_0}^{t_f} L(x(t), u(t), t)dt$$

受约束 $\dot{x}(t) = f(x(t), u(t), t)$。

**定理 5.1.1** (欧拉-拉格朗日方程) 最优解满足：

$$\frac{d}{dt}\frac{\partial L}{\partial \dot{x}} - \frac{\partial L}{\partial x} = 0$$

**证明** 通过变分法和积分分部。

### 5.2 动态规划

**定义 5.2.1** (值函数) 值函数：

$$V(x, t) = \min_{u} \int_t^{t_f} L(x(\tau), u(\tau), \tau)d\tau$$

**定理 5.2.1** (哈密顿-雅可比-贝尔曼方程) 值函数满足：

$$-\frac{\partial V}{\partial t} = \min_{u} [L(x, u, t) + \frac{\partial V}{\partial x}f(x, u, t)]$$

**证明** 通过动态规划原理和泰勒展开。

### 5.3 极大值原理

**定义 5.3.1** (哈密顿函数) 哈密顿函数：

$$H(x, u, \lambda, t) = L(x, u, t) + \lambda^T f(x, u, t)$$

**定理 5.3.1** (庞特里亚金极大值原理) 最优控制满足：

1. $\dot{x} = \frac{\partial H}{\partial \lambda}$
2. $\dot{\lambda} = -\frac{\partial H}{\partial x}$
3. $u^* = \arg\min_{u} H(x, u, \lambda, t)$

**证明** 通过变分法和共态变量引入。

## 6. 鲁棒控制理论

### 6.1 不确定性建模

**定义 6.1.1** (不确定性系统) 不确定性系统：

$$\dot{x}(t) = (A + \Delta A)x(t) + (B + \Delta B)u(t)$$

其中 $\Delta A$, $\Delta B$ 是不确定性。

**定义 6.1.2** (结构不确定性) 结构不确定性：

$$\Delta A = \sum_{i=1}^k \delta_i A_i$$

其中 $\delta_i$ 是未知参数，$A_i$ 是已知矩阵。

### 6.2 H∞控制

**定义 6.2.1** (H∞范数) 传递函数 $G(s)$ 的H∞范数：

$$\|G\|_{\infty} = \sup_{\omega} \sigma_{\max}(G(j\omega))$$

**定义 6.2.2** (H∞控制问题) H∞控制问题：

$$\min_{K} \|T_{zw}\|_{\infty}$$

其中 $T_{zw}$ 是从干扰 $w$ 到性能输出 $z$ 的闭环传递函数。

**定理 6.2.1** (H∞控制解) H∞控制问题可以通过求解两个代数黎卡提方程求解。

### 6.3 μ综合

**定义 6.3.1** (μ分析) μ分析用于分析结构不确定性的鲁棒性。

**定义 6.3.2** (μ综合) μ综合设计控制器使得 $\mu_{\Delta}(M) < 1$。

**定理 6.3.1** (μ综合算法) μ综合可以通过D-K迭代算法求解。

## 7. 自适应控制理论

### 7.1 参数自适应控制

**定义 7.1.1** (参数自适应系统) 参数自适应系统：

$$\dot{x}(t) = A(\theta)x(t) + B(\theta)u(t)$$

其中 $\theta$ 是未知参数。

**定义 7.1.2** (参数估计) 参数估计器：

$$\dot{\hat{\theta}}(t) = \Gamma \phi(t)e(t)$$

其中 $\Gamma > 0$ 是学习率，$\phi(t)$ 是回归向量，$e(t)$ 是预测误差。

**定理 7.1.1** (参数收敛性) 如果回归向量持续激励，则参数估计收敛到真值。

### 7.2 模型参考自适应控制

**定义 7.2.1** (参考模型) 参考模型：

$$\dot{x}_m(t) = A_m x_m(t) + B_m r(t)$$

**定义 7.2.2** (跟踪误差) 跟踪误差：

$$e(t) = x(t) - x_m(t)$$

**定理 7.2.1** (MRAC稳定性) 如果参考模型稳定且参数自适应律设计正确，则跟踪误差收敛到零。

## 8. 时态逻辑控制

### 8.1 线性时态逻辑

**定义 8.1.1** (LTL公式) 线性时态逻辑公式：

$$\phi ::= p \mid \neg \phi \mid \phi \land \psi \mid \phi \lor \psi \mid \phi \rightarrow \psi \mid \bigcirc \phi \mid \phi \mathcal{U} \psi \mid \Box \phi \mid \Diamond \phi$$

**定义 8.1.2** (LTL语义) LTL在轨迹 $\pi$ 上的语义：

- $\pi \models p$ 当且仅当 $p \in \pi(0)$
- $\pi \models \bigcirc \phi$ 当且仅当 $\pi^1 \models \phi$
- $\pi \models \phi \mathcal{U} \psi$ 当且仅当存在 $i \geq 0$ 使得 $\pi^i \models \psi$ 且对于所有 $0 \leq j < i$，$\pi^j \models \phi$

### 8.2 控制综合

**定义 8.2.1** (控制综合问题) 给定系统模型和LTL规范 $\phi$，设计控制器使得闭环系统满足 $\phi$。

**定理 8.2.1** (控制综合算法) 控制综合可以通过自动机理论和博弈论方法求解。

**证明** 通过将LTL规范转换为自动机，然后求解可达性游戏。

## 9. 分布式控制

### 9.1 多智能体系统

**定义 9.1.1** (多智能体系统) 多智能体系统：

$$\dot{x}_i(t) = f_i(x_i(t), u_i(t), x_{-i}(t))$$

其中 $x_{-i}$ 表示除智能体 $i$ 外的其他智能体状态。

**定义 9.1.2** (一致性) 系统达到一致性如果：

$$\lim_{t \rightarrow \infty} \|x_i(t) - x_j(t)\| = 0$$

对所有 $i, j$ 成立。

**定理 9.1.1** (一致性条件) 如果通信图连通且控制律设计正确，则系统达到一致性。

### 9.2 分布式优化

**定义 9.2.1** (分布式优化问题) 分布式优化问题：

$$\min_{x} \sum_{i=1}^n f_i(x)$$

受约束 $x \in \mathcal{X}$。

**定义 9.2.2** (分布式梯度下降) 分布式梯度下降：

$$x_i(k+1) = \sum_{j \in \mathcal{N}_i} w_{ij} x_j(k) - \alpha \nabla f_i(x_i(k))$$

**定理 9.2.1** (收敛性) 如果步长选择合适且通信图连通，则分布式梯度下降收敛到最优解。

## 10. 智能控制

### 10.1 模糊控制

**定义 10.1.1** (模糊集合) 模糊集合 $A$ 的隶属函数 $\mu_A: X \rightarrow [0,1]$。

**定义 10.1.2** (模糊规则) 模糊规则：

$$\text{IF } x_1 \text{ is } A_1 \text{ AND } x_2 \text{ is } A_2 \text{ THEN } u \text{ is } B$$

**定理 10.1.1** (模糊控制稳定性) 如果模糊控制器设计正确，则闭环系统稳定。

### 10.2 神经网络控制

**定义 10.2.1** (神经网络) 神经网络：

$$y = \sigma(W_2 \sigma(W_1 x + b_1) + b_2)$$

其中 $\sigma$ 是激活函数。

**定义 10.2.2** (神经网络控制) 神经网络控制器：

$$u(t) = \text{NN}(x(t), \theta)$$

其中 $\theta$ 是网络参数。

**定理 10.2.1** (万能逼近定理) 神经网络可以逼近任意连续函数。

## 11. 控制理论的应用

### 11.1 机器人控制

**应用 11.1.1** (机器人动力学) 机器人动力学：

$$M(q)\ddot{q} + C(q, \dot{q})\dot{q} + G(q) = \tau$$

**应用 11.1.2** (计算力矩控制) 计算力矩控制：

$$\tau = M(q)(\ddot{q}_d + K_v \dot{e} + K_p e) + C(q, \dot{q})\dot{q} + G(q)$$

### 11.2 飞行器控制

**应用 11.2.1** (飞行器动力学) 飞行器动力学：

$$\dot{v} = \frac{1}{m}(F - mg)$$
$$\dot{\omega} = J^{-1}(M - \omega \times J\omega)$$

**应用 11.2.2** (自动驾驶仪) 自动驾驶仪使用PID控制和现代控制方法。

### 11.3 过程控制

**应用 11.3.1** (化工过程) 化工过程控制使用模型预测控制和自适应控制。

**应用 11.3.2** (电力系统) 电力系统控制使用鲁棒控制和分布式控制。
