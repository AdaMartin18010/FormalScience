## 1. 概述

经典控制理论是形式科学理论体系的重要组成部分，为动态系统的分析和控制提供数学基础。本部分涵盖线性系统理论、稳定性分析、控制器设计以及经典控制方法的形式化理论。

### 1.1 理论基础地位

经典控制理论在形式科学理论体系中的核心地位：

- **系统分析**: 提供动态系统分析的理论基础
- **控制器设计**: 为控制器设计提供系统化方法
- **稳定性保证**: 确保控制系统的稳定性
- **性能优化**: 提供系统性能优化的理论工具

### 1.2 理论体系结构

```
经典控制理论
├── 线性系统理论 (Linear System Theory)
├── 稳定性分析 (Stability Analysis)
├── 控制器设计 (Controller Design)
├── 性能分析 (Performance Analysis)
└── 鲁棒性分析 (Robustness Analysis)
```

## 2. 理论基础

### 2.1 动态系统基础

**定义 2.1.1** (动态系统) 动态系统是一个五元组 Σ = (X, U, Y, f, h)，其中：

- X ⊆ ℝⁿ 是状态空间
- U ⊆ ℝᵐ 是输入空间
- Y ⊆ ℝᵖ 是输出空间
- f : X × U → X 是状态转移函数
- h : X → Y 是输出函数

**定义 2.1.2** (连续时间系统) 连续时间系统的状态方程：
```
ẋ(t) = f(x(t), u(t))
y(t) = h(x(t))
```

**定义 2.1.3** (离散时间系统) 离散时间系统的状态方程：
```
x(k+1) = f(x(k), u(k))
y(k) = h(x(k))
```

### 2.2 线性系统理论

**定义 2.2.1** (线性系统) 系统 Σ 是线性的，如果满足叠加原理：
```
f(αx₁ + βx₂, αu₁ + βu₂) = αf(x₁, u₁) + βf(x₂, u₂)
h(αx₁ + βx₂) = αh(x₁) + βh(x₂)
```

**定义 2.2.2** (线性时不变系统) 线性时不变系统的状态空间表示：
```
ẋ(t) = Ax(t) + Bu(t)
y(t) = Cx(t) + Du(t)
```

其中 A ∈ ℝⁿˣⁿ, B ∈ ℝⁿˣᵐ, C ∈ ℝᵖˣⁿ, D ∈ ℝᵖˣᵐ。

**定理 2.2.1** (线性系统解) 线性时不变系统的解为：
```
x(t) = eᴬᵗx(0) + ∫₀ᵗ eᴬ⁽ᵗ⁻ᵗᵃᵘ⁾Bu(τ)dτ
```

**证明** 通过状态方程的积分：

1. **齐次方程**: ẋ = Ax 的解为 x(t) = eᴬᵗx(0)
2. **非齐次方程**: 通过变分常数法求解
3. **完整解**: 利用卷积积分得到完整解

## 3. 核心概念

### 3.1 传递函数

**定义 3.1.1** (传递函数) 线性时不变系统的传递函数：
```
G(s) = C(sI - A)⁻¹B + D
```

**定义 3.1.2** (极点) 传递函数的极点是 det(sI - A) = 0 的根。

**定义 3.1.3** (零点) 传递函数的零点是分子多项式的根。

**定理 3.1.1** (传递函数性质) 传递函数 G(s) 是有理函数，且：
1. 极点数等于系统阶数
2. 零点数不超过极点数
3. 相对阶数 = 极点数 - 零点数

### 3.2 频率响应

**定义 3.2.1** (频率响应) 频率响应是传递函数在虚轴上的值：
```
G(jω) = C(jωI - A)⁻¹B + D
```

**定义 3.2.2** (幅频特性) 幅频特性 |G(jω)| 表示系统对不同频率的放大倍数。

**定义 3.2.3** (相频特性) 相频特性 ∠G(jω) 表示系统对不同频率的相位延迟。

**定理 3.2.1** (频率响应性质) 频率响应具有以下性质：
1. 共轭对称性：G(-jω) = G*(jω)
2. 实系数性：对于实系数系统，频率响应是实系数的
3. 连续性：频率响应是频率的连续函数

### 3.3 根轨迹

**定义 3.3.1** (根轨迹) 根轨迹是系统极点随增益变化的轨迹。

**定义 3.3.2** (根轨迹方程) 根轨迹满足方程：
```
1 + KG(s) = 0
```

**定理 3.3.1** (根轨迹性质) 根轨迹具有以下性质：
1. 对称性：关于实轴对称
2. 分支数：等于系统阶数
3. 起点：在开环极点
4. 终点：在开环零点或无穷远

## 4. 重要定理

### 4.1 稳定性定理

**定理 4.1.1** (劳斯-赫尔维茨判据) 多项式 p(s) = aₙsⁿ + aₙ₋₁sⁿ⁻¹ + ... + a₀ 的所有根都有负实部，当且仅当劳斯阵列的第一列元素都为正。

**证明** 通过劳斯阵列的构造：

1. 构造劳斯阵列
2. 证明第一列符号变化次数等于右半平面根数
3. 所有根在左半平面等价于无符号变化

**算法 4.1.1** (劳斯判据算法)：

```haskell
routhCriterion :: [Double] -> Bool
routhCriterion coeffs = 
  let routhArray = buildRouthArray coeffs
      firstColumn = map head routhArray
  in all (> 0) firstColumn
```

**定理 4.1.2** (奈奎斯特判据) 闭环系统稳定的充要条件是奈奎斯特图绕 (-1, 0) 点的圈数等于开环系统右半平面极点数。

**证明** 通过幅角原理：

1. 幅角原理：函数在围线内的零点数减去极点数等于幅角变化
2. 奈奎斯特围线：包含整个右半平面的围线
3. 稳定性条件：闭环系统在右半平面无极点

### 4.2 可控性定理

**定义 4.2.1** (可控性) 系统在时间 T 内可控，如果对于任意初始状态 x₀ 和目标状态 xf，存在输入 u(t) 使得 x(T) = xf。

**定义 4.2.2** (可控性矩阵) 线性系统的可控性矩阵：
```
C = [B  AB  A²B  ...  Aⁿ⁻¹B]
```

**定理 4.2.1** (可控性判据) 线性系统完全可控当且仅当可控性矩阵 C 满秩。

**证明** 通过凯莱-哈密顿定理：

1. 可控性矩阵的列空间包含可达状态空间
2. 满秩确保可达整个状态空间
3. 凯莱-哈密顿定理限制矩阵幂的线性相关性

### 4.3 可观性定理

**定义 4.3.1** (可观性) 系统可观，如果任意初始状态 x₀ 都可以通过输出 y(t) 唯一确定。

**定义 4.3.2** (可观性矩阵) 线性系统的可观性矩阵：
```
O = [C; CA; CA²; ...; CAⁿ⁻¹]
```

**定理 4.3.1** (可观性判据) 线性系统完全可观当且仅当可观性矩阵 O 满秩。

**证明** 通过输出方程：

1. 可观性矩阵的行空间包含可观测状态空间
2. 满秩确保状态唯一确定
3. 输出序列包含足够信息重构状态

## 5. 稳定性分析

### 5.1 李雅普诺夫稳定性

**定义 5.1.1** (平衡点) 状态 xe ∈ X 是平衡点，如果 f(xe, 0) = 0。

**定义 5.1.2** (李雅普诺夫稳定性) 平衡点 xe 是李雅普诺夫稳定的，如果对于任意 ε > 0，存在 δ > 0 使得：
```
||x(0) - xe|| < δ ⇒ ||x(t) - xe|| < ε for all t ≥ 0
```

**定义 5.1.3** (渐近稳定性) 平衡点 xe 是渐近稳定的，如果它是李雅普诺夫稳定的且：
```
lim_{t→∞} x(t) = xe
```

**定理 5.1.1** (李雅普诺夫直接法) 如果存在连续可微函数 V : X → ℝ 满足：

1. V(xe) = 0
2. V(x) > 0 对于 x ≠ xe
3. V̇(x) ≤ 0 对于 x ≠ xe

则平衡点 xe 是李雅普诺夫稳定的。

**证明** 通过李雅普诺夫函数的单调性：

1. V(x) 在平衡点附近有下界
2. V̇(x) ≤ 0 确保 V(x) 不增加
3. 因此状态轨迹保持在平衡点附近

### 5.2 线性系统稳定性

**定理 5.2.1** (线性系统稳定性) 线性系统 ẋ = Ax 的零平衡点是渐近稳定的当且仅当 A 的所有特征值都有负实部。

**证明** 通过特征值分解：

1. A 的特征值决定系统动态
2. 负实部特征值对应衰减模态
3. 正实部特征值对应增长模态

**算法 5.2.1** (特征值稳定性检查)：

```haskell
isStable :: Matrix -> Bool
isStable a = 
  let eigenvalues = eigenValues a
      realParts = map realPart eigenvalues
  in all (< 0) realParts
```

### 5.3 频域稳定性

**定义 5.3.1** (增益裕度) 增益裕度是系统保持稳定性的最大增益倍数。

**定义 5.3.2** (相位裕度) 相位裕度是系统保持稳定性的最大相位延迟。

**定理 5.3.1** (频域稳定性) 系统稳定的充要条件是：
1. 增益裕度 > 1
2. 相位裕度 > 0

## 6. 控制器设计

### 6.1 PID控制器

**定义 6.1.1** (PID控制器) PID控制器的传递函数：
```
Gc(s) = Kp + Ki/s + Kds
```

其中：
- Kp 是比例增益
- Ki 是积分增益
- Kd 是微分增益

**定理 6.1.1** (PID控制器性质) PID控制器具有以下性质：
1. 比例项：提供基本控制作用
2. 积分项：消除稳态误差
3. 微分项：改善动态响应

**算法 6.1.1** (Ziegler-Nichols调参)：

```haskell
zieglerNichols :: System -> (Double, Double, Double)
zieglerNichols system = 
  let (ku, tu) = findUltimatePoint system
      kp = 0.6 * ku
      ki = 1.2 * ku / tu
      kd = 0.075 * ku * tu
  in (kp, ki, kd)
```

### 6.2 状态反馈

**定义 6.2.1** (状态反馈) 状态反馈控制律：
```
u(t) = -Kx(t) + r(t)
```

其中 K ∈ ℝᵐˣⁿ 是反馈增益矩阵，r(t) 是参考输入。

**定理 6.2.1** (极点配置) 如果系统 (A, B) 可控，则可以通过状态反馈任意配置闭环极点。

**证明** 通过可控性标准形：

1. 可控系统可以变换为标准形
2. 标准形下极点配置直接可得
3. 变换回原坐标系得到反馈增益

**算法 6.2.1** (极点配置算法)：

```haskell
polePlacement :: Matrix -> Matrix -> [Complex Double] -> Matrix
polePlacement a b desiredPoles = 
  let controllableForm = toControllableForm a b
      kStandard = placePoles controllableForm desiredPoles
      transformation = getTransformation a b
  in kStandard * transformation
```

### 6.3 观测器设计

**定义 6.3.1** (全维观测器) 全维观测器：
```
ẋ̂(t) = Ax̂(t) + Bu(t) + L(y(t) - Cx̂(t))
```

其中 L ∈ ℝⁿˣᵖ 是观测器增益矩阵。

**定理 6.3.1** (观测器极点配置) 如果系统 (A, C) 可观，则可以通过选择 L 任意配置观测器极点。

**证明** 通过可观性对偶性：

1. (A, C) 可观等价于 (Aᵀ, Cᵀ) 可控
2. 观测器极点配置等价于对偶系统的状态反馈极点配置
3. 通过极点配置算法得到观测器增益

## 8. 批判分析

### 8.1 理论局限性

#### 8.1.1 线性假设

**批判 8.1.1** (线性假设) 经典控制理论基于线性系统假设，无法处理强非线性系统。

**分析** 线性假设的限制：
- 无法处理饱和、死区等非线性特性
- 无法处理时变系统
- 无法处理多模态系统
- 实际系统往往具有非线性特性

#### 8.1.2 单输入单输出

**批判 8.1.2** (SISO限制) 经典控制理论主要针对单输入单输出系统。

**分析** SISO限制的问题：
- 无法处理多变量系统
- 无法处理耦合效应
- 无法处理分布式系统
- 现代系统往往具有多变量特性

### 8.2 设计挑战

#### 8.2.1 参数整定

**批判 8.2.1** (参数整定) 控制器参数整定需要经验和试错。

**分析** 参数整定的挑战：
- 缺乏系统化的整定方法
- 参数之间相互影响
- 性能指标难以量化
- 鲁棒性要求难以满足

#### 8.2.2 模型不确定性

**批判 8.2.2** (模型不确定性) 实际系统模型往往存在不确定性。

**分析** 模型不确定性的影响：
- 参数变化影响控制性能
- 未建模动态导致不稳定
- 外部扰动影响控制效果
- 传感器噪声影响测量精度

### 8.3 哲学问题

#### 8.3.1 因果性

**问题 8.3.1** (因果性) 控制理论中的因果关系是什么？

**分析** 因果性的哲学问题：
- **决定论**: 系统行为是否完全由输入决定？
- **自由意志**: 控制器是否具有自主性？
- **目的论**: 控制目标是否具有目的性？

#### 8.3.2 最优性

**问题 8.3.2** (最优性) 什么是最优控制？

**分析** 最优性的哲学问题：
- **目标函数**: 如何定义最优性？
- **约束条件**: 如何处理约束？
- **多目标**: 如何处理多个目标？
