## 15.1.1 一般均衡理论

### 概述

一般均衡理论（General Equilibrium Theory）是数理经济学的核心理论框架，由Léon Walras奠基，Kenneth Arrow和Gérard Debreu在1954年通过不动点定理严格证明了均衡的存在性。
该理论描述市场经济中所有市场同时达到均衡的状态。

**参考文献**: Arrow & Debreu (1954), "Existence of an Equilibrium for a Competitive Economy"

---

## 15.1.1.1 Arrow-Debreu模型

### 模型设定

**定义 15.1.1** (Arrow-Debreu经济)

一个Arrow-Debreu经济 $\mathcal{E}$ 定义为五元组：

$$\mathcal{E} = (\mathcal{I}, \mathcal{J}, (X_i, \succsim_i, e_i)_{i \in \mathcal{I}}, (Y_j)_{j \in \mathcal{J}}, (\theta_{ij}))$$

其中：

- $\mathcal{I} = \{1, 2, \ldots, I\}$：消费者集合
- $\mathcal{J} = \{1, 2, \ldots, J\}$：生产者集合
- $X_i \subseteq \mathbb{R}^L_+$：消费者 $i$ 的消费集（$L$ 种商品）
- $\succsim_i \subseteq X_i \times X_i$：消费者 $i$ 的偏好关系
- $e_i \in X_i$：消费者 $i$ 的初始禀赋
- $Y_j \subseteq \mathbb{R}^L$：生产者 $j$ 的生产集
- $\theta_{ij} \in [0,1]$：消费者 $i$ 拥有企业 $j$ 的利润份额，满足 $\sum_{i \in \mathcal{I}} \theta_{ij} = 1$

---

### 消费者行为

**定义 15.1.2** (效用最大化问题)

给定价格向量 $p \in \mathbb{R}^L_+$，消费者 $i$ 求解：

$$\max_{x_i \in X_i} u_i(x_i) \quad \text{s.t.} \quad p \cdot x_i \leq p \cdot e_i + \sum_{j \in \mathcal{J}} \theta_{ij} \pi_j(p)$$

其中 $\pi_j(p) = \max_{y_j \in Y_j} p \cdot y_j$ 为企业 $j$ 的利润函数。

**定义 15.1.3** (马歇尔需求函数)

消费者 $i$ 的马歇尔需求对应为：

$$x_i^*(p) = \{x_i \in X_i : x_i \succsim_i x_i' \text{ for all } x_i' \text{ s.t. } p \cdot x_i' \leq w_i(p)\}$$

其中 $w_i(p) = p \cdot e_i + \sum_{j} \theta_{ij} \pi_j(p)$ 为财富函数。

---

### 生产者行为

**定义 15.1.4** (利润最大化)

生产者 $j$ 求解：

$$\pi_j(p) = \max_{y_j \in Y_j} p \cdot y_j$$

最优生产计划为 $y_j^*(p) \in \arg\max_{y_j \in Y_j} p \cdot y_j$

**公理 15.1.1** (生产集性质)

标准假设下，生产集 $Y_j$ 满足：

1. **闭性**: $Y_j$ 是闭集
2. **凸性**: $Y_j$ 是凸集
3. **自由处置**: 若 $y \in Y_j$ 且 $y' \leq y$，则 $y' \in Y_j$
4. **不可逆性**: $Y_j \cap (-Y_j) \subseteq \{0\}$
5. **无免费午餐**: $Y_j \cap \mathbb{R}^L_+ = \{0\}$

---

## 15.1.1.2 竞争均衡

### 均衡定义

**定义 15.1.5** (Arrow-Debreu竞争均衡)

配置 $(x^*, y^*, p^*)$ 称为竞争均衡，若满足：

1. **消费者最优**: 对所有 $i \in \mathcal{I}$
   $$x_i^* \in x_i^*(p^*)$$

2. **生产者最优**: 对所有 $j \in \mathcal{J}$
   $$y_j^* \in y_j^*(p^*)$$

3. **市场出清**:
   $$\sum_{i \in \mathcal{I}} x_i^* = \sum_{i \in \mathcal{I}} e_i + \sum_{j \in \mathcal{J}} y_j^*$$

---

## 15.1.1.3 均衡存在性定理

### 核心定理

**定理 15.1.1** (Arrow-Debreu存在性定理, 1954)

在满足以下条件下，竞争均衡存在：

**假设条件**:

- **(H1)** 消费集: $X_i = \mathbb{R}^L_+$，凸、闭、下有界
- **(H2)** 偏好: $\succsim_i$ 完备、传递、连续、凸、强单调
- **(H3)** 初始禀赋: $e_i \in \text{int}(X_i)$（内部点）
- **(H4)** 生产集: $Y_j$ 闭、凸、包含原点、满足自由处置

**证明**:

_第一步：构造超额需求函数_

定义超额需求函数：

$$z(p) = \sum_{i \in \mathcal{I}} x_i^*(p) - \sum_{i \in \mathcal{I}} e_i - \sum_{j \in \mathcal{J}} y_j^*(p)$$

_第二步：验证Walras法则_

由预算约束加总：

$$\sum_i p \cdot x_i^*(p) = \sum_i p \cdot e_i + \sum_j p \cdot y_j^*(p)$$

因此 $p \cdot z(p) = 0$（Walras法则）

_第三步：应用Kakutani不动点定理_

定义价格单纯形 $\Delta^{L-1} = \{p \in \mathbb{R}^L_+ : \sum_l p_l = 1\}$

构造对应：

$$\Phi(p) = \arg\max_{q \in \Delta^{L-1}} q \cdot z(p)$$

由Kakutani不动点定理，存在 $p^*$ 使得 $p^* \in \Phi(p^*)$

_第四步：证明 $z(p^*) \leq 0$_

假设存在 $l$ 使得 $z_l(p^*) > 0$，则 $\Phi(p^*)$ 会赋予该商品最大权重，导致矛盾（违反Walras法则）。因此 $z(p^*) \leq 0$。

由自由处置假设，可以实现精确均衡。 $\square$

---

## 15.1.1.4 均衡的福利性质

### 福利经济学基本定理

**定理 15.1.2** (第一福利定理)

若偏好局部非餍足，则竞争均衡配置是帕累托最优的。

**证明**:

设 $(x^*, y^*, p^*)$ 为竞争均衡。假设存在可行配置 $(x, y)$ 帕累托改进 $x^*$。

对改进的消费者 $i$：$x_i \succ_i x_i^* \Rightarrow p^* \cdot x_i > p^* \cdot x_i^*$（否则违反效用最大化）

对无差异的消费者：$p^* \cdot x_i \geq p^* \cdot x_i^*$

加总得：

$$\sum_i p^* \cdot x_i > \sum_i p^* \cdot x_i^* = \sum_i p^* \cdot e_i + \sum_j p^* \cdot y_j^*$$

但由可行性：$\sum_i x_i = \sum_i e_i + \sum_j y_j$

由利润最大化：$p^* \cdot y_j \leq p^* \cdot y_j^*$

矛盾。 $\square$

---

**定理 15.1.3** (第二福利定理)

若偏好凸、连续、强单调，则任何帕累托最优配置可通过适当的初始禀赋再分配实现为竞争均衡。

## 15.1.1.5 计算均衡

### 算法实现

```python
"""
Arrow-Debreu一般均衡计算
基于Scarf算法计算近似均衡
"""

import numpy as np
from scipy.optimize import minimize, linprog
from typing import List, Callable, Tuple
import matplotlib.pyplot as plt

class ArrowDebreuEconomy:
    """
    Arrow-Debreu经济模型

    参数:
        n_consumers: 消费者数量
        n_goods: 商品种类数
        n_firms: 企业数量
    """

    def __init__(self, n_consumers: int, n_goods: int, n_firms: int = 0):
        self.I = n_consumers
        self.L = n_goods
        self.J = n_firms

        # 效用函数参数 (Cobb-Douglas)
        self.alpha = np.random.dirichlet(np.ones(n_goods), n_consumers)

        # 初始禀赋
        self.endowments = np.random.uniform(1, 10, (n_consumers, n_goods))

        # 生产技术 (Leontief)
        if n_firms > 0:
            self.tech_matrix = np.random.uniform(0.5, 2.0, (n_firms, n_goods))
            self.theta = np.random.dirichlet(np.ones(n_consumers), n_firms).T

    def utility(self, i: int, x: np.ndarray) -> float:
        """Cobb-Douglas效用函数"""
        return np.prod(x ** self.alpha[i])

    def marshallian_demand(self, i: int, p: np.ndarray, wealth: float) -> np.ndarray:
        """计算马歇尔需求 (Cobb-Douglas解析解)"""
        return (self.alpha[i] * wealth) / (p + 1e-10)

    def excess_demand(self, p: np.ndarray) -> np.ndarray:
        """计算超额需求函数 z(p)"""
        aggregate_demand = np.zeros(self.L)
        aggregate_supply = np.sum(self.endowments, axis=0)

        for i in range(self.I):
            # 计算财富
            wealth = np.dot(p, self.endowments[i])
            if self.J > 0:
                for j in range(self.J):
                    # 简化：假设线性生产
                    profit = max(0, np.dot(p, np.ones(self.L) - self.tech_matrix[j]))
                    wealth += self.theta[i, j] * profit

            demand = self.marshallian_demand(i, p, wealth)
            aggregate_demand += demand

        return aggregate_demand - aggregate_supply

    def compute_equilibrium(self, max_iter: int = 1000, tol: float = 1e-6) -> np.ndarray:
        """
        使用Tatonnement过程计算均衡价格

        算法: Walrasian Tatonnement
        dp/dt = z(p) (超额需求推动价格调整)
        """
        # 初始价格 (标准化)
        p = np.ones(self.L) / self.L

        for t in range(max_iter):
            z = self.excess_demand(p)

            # 价格调整 (Tatonnement)
            p_new = p + 0.1 * z
            p_new = np.maximum(p_new, 0.01)  # 保证正价格
            p_new = p_new / np.sum(p_new)    # 标准化

            if np.linalg.norm(p_new - p) < tol:
                print(f"均衡收敛于迭代 {t}")
                return p_new

            p = p_new

        print("达到最大迭代次数")
        return p

    def verify_equilibrium(self, p: np.ndarray) -> dict:
        """验证均衡条件"""
        z = self.excess_demand(p)

        # Walras法则验证
        walras_law = np.dot(p, z)

        return {
            'excess_demand': z,
            'walras_law': walras_law,
            'market_clearing': np.allclose(z, 0, atol=1e-4),
            'prices': p
        }

# ==================== 演示 ====================
if __name__ == "__main__":
    # 创建2人2商品经济
    print("=" * 60)
    print("Arrow-Debreu一般均衡计算示例")
    print("=" * 60)

    economy = ArrowDebreuEconomy(n_consumers=2, n_goods=2, n_firms=0)

    print("\n初始禀赋:")
    print(economy.endowments)
    print("\n效用函数参数 (Cobb-Douglas):")
    print(economy.alpha)

    # 计算均衡
    p_eq = economy.compute_equilibrium()

    print(f"\n均衡价格: {p_eq}")

    # 验证均衡
    verification = economy.verify_equilibrium(p_eq)
    print(f"\n超额需求: {verification['excess_demand']}")
    print(f"Walras法则 (p·z): {verification['walras_law']:.10f}")
    print(f"市场出清: {verification['market_clearing']}")

    # 绘制Edgeworth盒
    fig, axes = plt.subplots(1, 2, figsize=(14, 6))

    # 图1: 超额需求收敛
    prices_path = []
    p = np.ones(2) / 2
    for _ in range(50):
        prices_path.append(p.copy())
        z = economy.excess_demand(p)
        p = p + 0.1 * z
        p = np.maximum(p, 0.01)
        p = p / np.sum(p)
    prices_path = np.array(prices_path)

    axes[0].plot(prices_path[:, 0], prices_path[:, 1], 'b-o', markersize=4)
    axes[0].plot(p_eq[0], p_eq[1], 'r*', markersize=15, label='Equilibrium')
    axes[0].set_xlabel('Price of Good 1')
    axes[0].set_ylabel('Price of Good 2')
    axes[0].set_title('Price Convergence (Tatonnement)')
    axes[0].legend()
    axes[0].grid(True)

    # 图2: Edgeworth盒 (契约曲线)
    total = np.sum(economy.endowments, axis=0)
    n_grid = 50
    x1_grid = np.linspace(0.01, total[0] - 0.01, n_grid)
    x2_grid = np.linspace(0.01, total[1] - 0.01, n_grid)

    U1 = np.array([[economy.utility(0, np.array([x1, x2]))
                    for x2 in x2_grid] for x1 in x1_grid])
    U2 = np.array([[economy.utility(1, total - np.array([x1, x2]))
                    for x2 in x2_grid] for x1 in x1_grid])

    axes[1].contour(x1_grid, x2_grid, U1.T, levels=10, colors='blue', alpha=0.5)
    axes[1].contour(x1_grid, x2_grid, U2.T, levels=10, colors='red', alpha=0.5)
    axes[1].scatter([economy.endowments[0, 0]], [economy.endowments[0, 1]],
                    c='green', s=100, marker='s', label='Endowment')

    # 计算均衡配置
    wealth = np.dot(p_eq, economy.endowments[0])
    x_eq = economy.marshallian_demand(0, p_eq, wealth)
    axes[1].scatter([x_eq[0]], [x_eq[1]], c='gold', s=150, marker='*', label='Equilibrium')

    axes[1].set_xlabel('Good 1 (Consumer 1)')
    axes[1].set_ylabel('Good 2 (Consumer 1)')
    axes[1].set_title('Edgeworth Box')
    axes[1].legend()
    axes[1].grid(True)

    plt.tight_layout()
    plt.savefig('general_equilibrium.png', dpi=150, bbox_inches='tight')
    plt.show()
    print("\n图形已保存至 general_equilibrium.png")
```

## 15.1.1.6 均衡的唯一性与稳定性

### 唯一性条件

**定理 15.1.4** (Gross替代性蕴含唯一性)

若对所有 $l, k$ 满足Gross替代性：

$$\frac{\partial z_l(p)}{\partial p_k} > 0 \quad (l \neq k)$$

则均衡价格在比例意义上唯一。

### 稳定性分析

**定义 15.1.6** (Tatonnement动态)

价格调整过程：

$$\frac{dp_l}{dt} = \lambda_l z_l(p(t)), \quad \lambda_l > 0$$

**定理 15.1.5** (全局稳定性)

若超额需求函数满足Gross替代性且Walras法则成立，则Tatonnement过程全局收敛到均衡。

## 15.1.1.7 扩展：不确定性下的均衡

### Arrow证券

**定义 15.1.7** (Arrow-Debreu证券)

状态 $s \in \mathcal{S} = \{1, \ldots, S\}$ 下的Arrow证券在状态 $s$ 支付1单位计价物，其他状态支付0。

**定理 15.1.6** (状态价格)

存在状态价格 $\phi_s > 0$ 使得任何或有索取权 $c = (c_s)_{s \in \mathcal{S}}$ 的价格为：

$$q(c) = \sum_{s \in \mathcal{S}} \phi_s c_s$$
