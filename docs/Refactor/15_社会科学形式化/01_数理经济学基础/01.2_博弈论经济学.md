## 15.1.2 博弈论经济学

### 概述

博弈论经济学将策略互动形式化，研究理性个体在相互依赖决策环境中的行为。
从von Neumann & Morgenstern的奠基性工作到Nash均衡概念，博弈论已成为现代经济分析的核心理论工具。

**参考文献**: von Neumann & Morgenstern (1944), Nash (1950, 1951), Harsanyi (1967)

---

## 15.1.2.1 策略形式博弈

### 基本定义

**定义 15.1.8** (正规形式博弈)

正规形式（策略形式）博弈定义为三元组：

$$\Gamma = (\mathcal{N}, (S_i)_{i \in \mathcal{N}}, (u_i)_{i \in \mathcal{N}})$$

其中：

- $\mathcal{N} = \{1, 2, \ldots, n\}$：参与人集合
- $S_i$：参与人 $i$ 的纯策略集
- $u_i: S \to \mathbb{R}$：参与人 $i$ 的效用函数，$S = \prod_{i \in \mathcal{N}} S_i$

---

### 混合策略

**定义 15.1.9** (混合策略)

参与人 $i$ 的混合策略是 $S_i$ 上的概率分布：

$$\Delta(S_i) = \{\sigma_i : S_i \to [0,1] \mid \sum_{s_i \in S_i} \sigma_i(s_i) = 1\}$$

期望效用函数：

$$U_i(\sigma) = \sum_{s \in S} \left(\prod_{j \in \mathcal{N}} \sigma_j(s_j)\right) u_i(s)$$

---

## 15.1.2.2 Nash均衡

### 均衡概念

**定义 15.1.10** (Nash均衡, Nash 1950)

策略组合 $\sigma^* = (\sigma_1^*, \ldots, \sigma_n^*)$ 是Nash均衡，若对所有 $i \in \mathcal{N}$：

$$U_i(\sigma_i^*, \sigma_{-i}^*) \geq U_i(\sigma_i, \sigma_{-i}^*), \quad \forall \sigma_i \in \Delta(S_i)$$

或等价地，对所有 $s_i \in \text{supp}(\sigma_i^*)$：

$$U_i(s_i, \sigma_{-i}^*) = \max_{s_i' \in S_i} U_i(s_i', \sigma_{-i}^*)$$

---

### 存在性定理

**定理 15.1.7** (Nash存在性定理, 1950)

任何有限博弈（$\mathcal{N}$ 有限，$S_i$ 有限）至少存在一个Nash均衡（可能是混合策略）。

**证明**:

_第一步：构造最佳响应对应_

$$BR_i(\sigma_{-i}) = \arg\max_{\sigma_i \in \Delta(S_i)} U_i(\sigma_i, \sigma_{-i})$$

全局最佳响应：$BR(\sigma) = \prod_{i} BR_i(\sigma_{-i})$

_第二步：应用Kakutani不动点定理_

需验证：

1. $\Delta = \prod_i \Delta(S_i)$ 是非空紧凸集
2. $BR(\sigma)$ 非空（Weierstrass定理）
3. $BR(\sigma)$ 凸值（$U_i$ 关于 $\sigma_i$ 线性）
4. $BR$ 上半连续（Berge极大值定理）

由Kakutani定理，存在 $\sigma^* \in BR(\sigma^*)$，即Nash均衡。 $\square$

## 15.1.2.3 市场博弈

### Bertrand价格竞争

**定义 15.1.11** (Bertrand博弈)

两个企业生产同质产品，边际成本 $c_1, c_2$，同时选择价格 $p_1, p_2$：

需求函数：

$$D_i(p_i, p_j) = \begin{cases} D(p_i) & \text{if } p_i < p_j \\ \frac{1}{2}D(p_i) & \text{if } p_i = p_j \\ 0 & \text{if } p_i > p_j \end{cases}$$

**定理 15.1.8** (Bertrand悖论)

若 $c_1 = c_2 = c$，则唯一Nash均衡为 $p_1^* = p_2^* = c$（边际成本定价）。

**证明**:

1. $p_1 = p_2 = c$ 是均衡：任何企业提价无需求，降价亏损
2. 唯一性：若 $p_1 > p_2 > c$，企业2有利可图偏离到 $p_2 - \epsilon$
   若 $p_1 = p_2 > c$，任何企业有利可图偏离到 $p - \epsilon$ $\square$

---

### Cournot数量竞争

**定义 15.1.12** (Cournot博弈)

$n$ 个企业选择产量 $q_i$，市场价格 $P(Q)$，$Q = \sum_i q_i$

利润函数：$\pi_i(q_i, q_{-i}) = q_i P(q_i + \sum_{j \neq i} q_j) - c_i(q_i)$

**定理 15.1.9** (Cournot均衡)

若 $P(\cdot)$ 递减凹函数，$c_i(\cdot)$ 凸，则存在唯一Cournot-Nash均衡。

**一阶条件**:

$$P(Q) + q_i P'(Q) - c_i'(q_i) = 0$$

---

## 15.1.2.4 动态博弈

### 扩展形式

**定义 15.1.13** (扩展形式博弈)

扩展形式博弈为七元组：

$$\mathcal{G} = (\mathcal{N}, \mathcal{T}, \iota, \mathcal{H}, A, \rho, (u_i))$$

- $\mathcal{T}$：博弈树（有限或无限）
- $\iota: \mathcal{X} \to \mathcal{N}$：参与人函数，$\mathcal{X}$ 为决策节点集
- $\mathcal{H}$：信息划分
- $A(x)$：节点 $x$ 的行动集
- $\rho$：自然（机会）的概率分布

---

### 子博弈精炼均衡

**定义 15.1.14** (子博弈)

子博弈由单节点信息集开始，包含该节点所有后续节点，且保持原博弈信息结构。

**定义 15.1.15** (子博弈精炼Nash均衡, SPE)

策略组合是SPE，若它在每个子博弈上都构成Nash均衡。

**定理 15.1.10** (有限博弈SPE存在性)

每个有限完美信息博弈存在唯一的子博弈精炼均衡（通过逆向归纳）。

---

### 重复博弈

**定义 15.1.16** (重复博弈)

阶段博弈 $G$ 重复 $T$ 次（有限或无限），折现因子 $\delta \in (0,1)$

总支付：$U_i = \sum_{t=1}^{T} \delta^{t-1} u_i(a^t)$

**定理 15.1.11** (民间定理, Folk Theorem)

若参与人足够耐心（$\delta$ 接近1），则任何可行的、个体理性的支付向量都可作为子博弈精炼均衡结果实现。

---

## 15.1.2.5 不完全信息博弈

### Harsanyi转换

**定义 15.1.17** (贝叶斯博弈)

贝叶斯博弈定义为：

$$\mathcal{B} = (\mathcal{N}, (\Theta_i), (S_i), (u_i), \phi)$$

- $\Theta_i$：参与人 $i$ 的类型空间
- $\phi \in \Delta(\Theta)$：共同先验分布
- $u_i: S \times \Theta \to \mathbb{R}$：类型依存的支付函数

**定义 15.1.18** (贝叶斯Nash均衡)

策略组合 $(\sigma_i^*: \Theta_i \to \Delta(S_i))$ 是BNE，若：

$$\mathbb{E}_{\theta_{-i}}[u_i(\sigma_i^*(\theta_i), \sigma_{-i}^*(\theta_{-i}), \theta) | \theta_i] \geq \mathbb{E}_{\theta_{-i}}[u_i(s_i, \sigma_{-i}^*(\theta_{-i}), \theta) | \theta_i]$$

对所有 $i, \theta_i, s_i$ 成立。

---

### 拍卖理论

**定义 15.1.19** (第一价格密封拍卖)

$n$ 个竞标者，私人价值 $\theta_i \sim F$ i.i.d.

均衡报价策略（对称情形）：

$$b(\theta) = \theta - \frac{\int_{\underline{\theta}}^{\theta} F^{n-1}(x)dx}{F^{n-1}(\theta)}$$

**定理 15.1.12** (收益等价定理)

在满足一定条件下，标准拍卖形式（第一价格、第二价格、荷兰式、英式）产生相同的期望收益。

"""

class NormalFormGame:
    """
    正规形式博弈

    参数:
        payoff_matrices: 支付矩阵列表 [(n_actions, n_actions, ...)]
    """

    def __init__(self, payoff_matrices: List[np.ndarray]):
        self.n_players = len(payoff_matrices)
        self.payoffs = payoff_matrices
        self.n_actions = payoff_matrices[0].shape

    def expected_utility(self, i: int, strategies: List[np.ndarray]) -> float:
        """计算参与人i的期望效用"""
        result = self.payoffs[i].copy()
        for j, sigma in enumerate(strategies):
            if j != i:
                # 在其他参与人维度上求期望
                axes = tuple(k for k in range(self.n_players) if k != i and k != j)
                if len(axes) > 0:
                    result = np.tensordot(result, sigma, axes=([j], [0]))
                else:
                    result = result @ sigma
        return float(np.dot(result.flatten(), strategies[i]))

    def best_response(self, i: int, opponent_strategies: List[np.ndarray]) -> np.ndarray:
        """计算最佳响应"""
        # 计算对每个纯策略的期望效用
        n = self.n_actions[i]
        utilities = np.zeros(n)

        for a in range(n):
            strategies = opponent_strategies.copy()
            pure = np.zeros(n)
            pure[a] = 1.0
            strategies[i] = pure
            utilities[a] = self.expected_utility(i, strategies)

        # 找到最大值的所有行动
        max_util = np.max(utilities)
        br = (utilities == max_util).astype(float)
        return br / np.sum(br)  # 均匀混合

    def find_nash_equilibrium(self, method='support_enumeration') -> List[np.ndarray]:
        """
        寻找Nash均衡

        方法:
            - 'support_enumeration': 支持枚举（小博弈）
            - 'fictitious_play': 虚拟博弈（迭代）
        """
        if method == 'fictitious_play':
            return self._fictitious_play()
        else:
            return self._support_enumeration()

    def _fictitious_play(self, max_iter=10000, tol=1e-6) -> List[np.ndarray]:
        """虚拟博弈算法"""
        # 初始化
        beliefs = [np.ones(n) / n for n in self.n_actions]

        for t in range(max_iter):
            # 计算最佳响应
            new_strategies = []
            for i in range(self.n_players):
                br = self.best_response(i, beliefs)
                new_strategies.append(br)

            # 更新信念（经验频率）
            for i in range(self.n_players):
                beliefs[i] = (t * beliefs[i] + new_strategies[i]) / (t + 1)

            # 检查收敛
            if all(np.linalg.norm(new_strategies[i] - beliefs[i]) < tol
                   for i in range(self.n_players)):
                print(f"虚拟博弈收敛于迭代 {t}")
                return beliefs

        return beliefs

    def _support_enumeration(self) -> List[np.ndarray]:
        """支持枚举（2人博弈）"""
        if self.n_players != 2:
            return self._fictitious_play()

        A, B = self.payoffs[0], self.payoffs[1]
        m, n = A.shape

        # 遍历所有可能的支持集
        for supp_size_1 in range(1, m + 1):
            for supp_size_2 in range(1, n + 1):
                for supp1 in self._subsets(range(m), supp_size_1):
                    for supp2 in self._subsets(range(n), supp_size_2):
                        eq = self._solve_for_support(supp1, supp2, A, B)
                        if eq is not None:
                            return eq

        return [np.ones(m) / m, np.ones(n) / n]  # 返回均匀混合

    def _subsets(self, s, k):
        """生成k元子集"""
        from itertools import combinations
        return list(combinations(s, k))

class CournotGame:
    """
    Cournot数量竞争模型

    线性需求: P(Q) = a - bQ
    二次成本: C_i(q_i) = c_i q_i + d_i q_i^2
    """

    def __init__(self, n_firms: int, a: float = 100, b: float = 1,
                 costs: List[Tuple[float, float]] = None):
        self.n = n_firms
        self.a = a  # 需求截距
        self.b = b  # 需求斜率

        if costs is None:
            costs = [(10 + i * 5, 0.1) for i in range(n_firms)]
        self.costs = costs  # (c_i, d_i) 列表

    def price(self, Q: float) -> float:
        """逆需求函数"""
        return max(0, self.a - self.b * Q)

    def profit(self, i: int, q_i: float, q_others: List[float]) -> float:
        """企业i的利润"""
        Q = q_i + sum(q_others)
        c, d = self.costs[i]
        revenue = q_i * self.price(Q)
        cost = c * q_i + d * q_i**2
        return revenue - cost

    def best_response(self, i: int, q_others: List[float]) -> float:
        """企业i的最佳响应函数"""
        Q_others = sum(q_others)
        c, d = self.costs[i]

        # 一阶条件: a - 2bq - bQ_others - c - 2dq = 0
        # q = (a - bQ_others - c) / (2b + 2d)
        q = (self.a - self.b * Q_others - c) / (2 * self.b + 2 * d)
        return max(0, q)

    def find_equilibrium(self, max_iter=1000, tol=1e-6) -> np.ndarray:
        """通过迭代寻找Cournot-Nash均衡"""
        q = np.ones(self.n) * self.a / (2 * self.b * self.n)

        for t in range(max_iter):
            q_new = np.zeros(self.n)
            for i in range(self.n):
                q_others = [q[j] for j in range(self.n) if j != i]
                q_new[i] = self.best_response(i, q_others)

            if np.linalg.norm(q_new - q) < tol:
                print(f"Cournot均衡收敛于迭代 {t}")
                return q_new

            q = 0.5 * q + 0.5 * q_new  # 阻尼迭代

        return q

    def compute_welfare(self, q: np.ndarray) -> Dict:
        """计算社会福利"""
        Q = np.sum(q)
        P = self.price(Q)

        # 消费者剩余
        CS = 0.5 * (self.a - P) * Q

        # 生产者剩余（利润）
        PS = sum(self.profit(i, q[i], [q[j] for j in range(self.n) if j != i])
                 for i in range(self.n))

        # 无谓损失 (与完全竞争比较)
        # 完全竞争: P = MC_min
        mc_min = min(c for c, d in self.costs)
        Q_competitive = (self.a - mc_min) / self.b
        Q_monopoly = (self.a - mc_min) / (2 * self.b)

        return {
            'price': P,
            'total_quantity': Q,
            'consumer_surplus': CS,
            'producer_surplus': PS,
            'total_surplus': CS + PS,
            'quantities': q
        }

    ax2.set_xlabel('$q_1$ (企业1产量)')
    ax2.set_ylabel('$q_2$ (企业2产量)')
    ax2.set_title('Cournot双寡头: 最佳响应函数')
    ax2.legend()
    ax2.grid(True)
    ax2.set_xlim(0, 40)
    ax2.set_ylim(0, 40)

    # 图3: 虚拟博弈收敛
    ax3 = axes[1, 0]
    beliefs = [np.ones(2) / 2 for _ in range(2)]
    history = [beliefs.copy()]

    for t in range(200):
        new_strat = [game.best_response(i, beliefs) for i in range(2)]
        for i in range(2):
            beliefs[i] = (t * beliefs[i] + new_strat[i]) / (t + 1)
        history.append([b.copy() for b in beliefs])

    history = np.array(history)
    ax3.plot(history[:, 0, 0], 'b-', label='参与人1: 合作概率', linewidth=2)
    ax3.plot(history[:, 1, 0], 'r-', label='参与人2: 合作概率', linewidth=2)
    ax3.axhline(y=0, color='gray', linestyle='--', alpha=0.5)
    ax3.set_xlabel('迭代')
    ax3.set_ylabel('策略概率')
    ax3.set_title('虚拟博弈收敛 (囚徒困境)')
    ax3.legend()
    ax3.grid(True)
    ax3.set_ylim(-0.1, 1.1)

    # 图4: 不同企业数量的Cournot均衡
    ax4 = axes[1, 1]
    n_firms_list = range(1, 11)
    prices = []
    quantities = []

    for n in n_firms_list:
        cgame = CournotGame(n_firms=n, a=100, b=1)
        q = cgame.find_equilibrium()
        w = cgame.compute_welfare(q)
        prices.append(w['price'])
        quantities.append(w['total_quantity'])

    ax4_twin = ax4.twinx()
    line1 = ax4.plot(n_firms_list, prices, 'b-o', label='价格', linewidth=2)
    line2 = ax4_twin.plot(n_firms_list, quantities, 'r-s', label='总产量', linewidth=2)

    # 添加完全竞争参考线
    ax4.axhline(y=10, color='green', linestyle='--', alpha=0.7, label='完全竞争价格')
    ax4_twin.axhline(y=90, color='orange', linestyle='--', alpha=0.7, label='完全竞争产量')

    ax4.set_xlabel('企业数量')
    ax4.set_ylabel('价格', color='b')
    ax4_twin.set_ylabel('总产量', color='r')
    ax4.set_title('Cournot市场: 竞争程度的影响')
    ax4.grid(True)

    lines = line1 + line2
    labels = [l.get_label() for l in lines]
    ax4.legend(lines, labels, loc='center right')

    plt.tight_layout()
    plt.savefig('game_theory_economics.png', dpi=150, bbox_inches='tight')
    plt.show()
    print("\n图形已保存至 game_theory_economics.png")

## 15.1.2.7 机制设计

### 显示原理

**定理 15.1.13** (显示原理)

任何机制实现的均衡结果，均可通过一个直接显示机制实现，其中真实报告是均衡策略。

### Vickrey-Clarke-Groves机制

**定义 15.1.20** (VCG机制)

社会选择函数 $f(\theta) \in \arg\max_x \sum_i v_i(x, \theta_i)$

参与人 $i$ 的支付：$t_i(\theta) = h_i(\theta_{-i}) - \sum_{j \neq i} v_j(f(\theta), \theta_j)$

**定理 15.1.14** (VCG机制激励相容性)

在拟线性偏好下，VCG机制满足占优策略激励相容（真实报告是最优策略）。
