# 15.4 形式人类学

## 15.4.2 文化模式

### 概述

文化模式的数学模型研究文化元素的分布、传播和结构化过程。从Boas学派的文化区概念到当代的文化进化理论，形式人类学试图用数学语言描述文化多样性、文化变迁和文化认知的基础结构。

**参考文献**: Kroeber (1939), D'Andrade (1995), Strauss & Quinn (1997)

---

## 15.4.2.1 文化空间模型

### 文化特征空间

**定义 15.4.12** (文化空间)

文化空间 $\mathcal{C}$ 是特征空间的乘积：

$$\mathcal{C} = \mathcal{C}_1 \times \mathcal{C}_2 \times \cdots \times \mathcal{C}_F$$

其中每个 $\mathcal{C}_f$ 是特征 $f$ 的可能取值集合。

**定义 15.4.13** (文化距离)

两个文化配置 $\mathbf{c}^i$ 和 $\mathbf{c}^j$ 的距离：

$$d_{ij} = \frac{1}{F} \sum_{f=1}^F \mathbb{1}[c_f^i \neq c_f^j]$$

或欧氏距离（连续特征）：

$$d_{ij} = \sqrt{\sum_{f=1}^F (c_f^i - c_f^j)^2}$$

---

### 文化区域

**定义 15.4.14** (文化区域)

文化区域 $R$ 是文化空间中的连通子集：

$$\forall x, y \in R, \exists \text{ 路径 } \gamma: [0,1] \to R, \gamma(0) = x, \gamma(1) = y$$

**边界**: $\partial R = \{x : d(x, R) = d(x, R^c)\}$

---

## 15.4.2.2 文化分布

### 文化频率

**定义 15.4.15** (文化特征频率)

特征 $f$ 取值 $v$ 的频率：

$$p_f(v) = \frac{1}{N} \sum_{i=1}^N \mathbb{1}[c_f^i = v]$$

**定义 15.4.16** (文化多样性)

Simpson多样性指数：

$$D = 1 - \sum_{f} \sum_{v} p_f(v)^2$$

或Shannon熵：

$$H = -\sum_f \sum_v p_f(v) \log p_f(v)$$

---

### 文化中心与边缘

**定义 15.4.17** (文化中心)

文化中心是高频特征组合：

$$\mathbf{c}^* = \arg\max_{\mathbf{c}} \prod_f p_f(c_f)$$

**定义 15.4.18** (文化边缘)

边缘文化具有低频率或独特特征组合：

$$\text{边缘度}(i) = \min_j d_{ij}$$

---

## 15.4.2.3 文化动力学

### 文化传播

**模型 15.4.1** ( unbiased transmission)

$$p_f^{t+1}(v) = \frac{1}{N} \sum_i P(c_f^i(t+1) = v | \text{示范者})$$

**模型 15.4.2** (有偏传播)

$$P(i \text{ 采纳 } v) \propto p_f(v)^\alpha \cdot u(v)^\beta$$

其中 $u(v)$ 为特征效用，$\alpha$ 为从众强度，$\beta$ 为效用敏感性。

---

### 文化创新

**定义 15.4.19** (创新率)

$$\lambda = \frac{\text{新特征产生数}}{\text{时间} \times \text{人口}}$$

**定理 15.4.3** (创新-传播平衡)

稳态文化多样性：

$$D^* = \frac{\lambda}{\lambda + \mu}$$

其中 $\mu$ 为特征丢失率。

---

## 15.4.2.4 认知人类学

### 文化图式

**定义 15.4.20** (文化图式)

图式 $S$ 是知识结构：

$$S = (S_0, S_1, \ldots, S_k)$$

- $S_0$: 槽位 (slots)
- $S_1$: 默认值
- $S_2$: 约束条件

**示例** (餐馆图式):

| 槽位 | 默认值 | 约束 |
|------|--------|------|
| 角色 | 顾客 | 服务员、顾客 |
| 活动 | 点餐 | 看菜单→点餐→进食→付款 |
| 地点 | 餐桌 | 厨房(禁入) |

---

### 分布式认知

**定义 15.4.21** (共识分析)

文化共识模型：

$$P(X_{ij} = 1) = \pi_j^{D_i} (1 - e_i)$$

- $X_{ij}$: 个体 $i$ 对项目 $j$ 的回答
- $\pi_j$: 文化正确答案
- $D_i$: 个体 $i$ 的文化能力
- $e_i$: 个体 $i$ 的误差率

---

## 15.4.2.5 计算实现

### 算法代码

```python
"""
文化模式计算分析
文化空间、文化分布、文化动力学的数值实现
"""

import numpy as np
from typing import List, Tuple, Dict, Set
from collections import Counter, defaultdict
import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from sklearn.cluster import KMeans
from scipy.spatial.distance import pdist, squareform
import networkx as nx

class CulturalSpace:
    """
    文化空间模型

    多维度文化特征的数学表示
    """

    def __init__(self, n_populations: int, n_features: int,
                 n_traits_per_feature: List[int] = None):
        """
        参数:
            n_populations: 人口/群体数量
            n_features: 文化特征维度数
            n_traits_per_feature: 每个特征的取值数量，默认随机
        """
        self.N = n_populations
        self.F = n_features

        if n_traits_per_feature is None:
            self.q = np.random.randint(3, 10, n_features)
        else:
            self.q = np.array(n_traits_per_feature)

        # 初始化文化配置
        self.culture = np.zeros((n_populations, n_features), dtype=int)
        for f in range(n_features):
            self.culture[:, f] = np.random.randint(0, self.q[f], n_populations)

        # 地理位置 (用于模拟空间传播)
        self.locations = np.random.rand(n_populations, 2)

    def cultural_distance(self, i: int, j: int) -> float:
        """
        计算文化距离

        使用汉明距离归一化
        """
        diff = np.sum(self.culture[i] != self.culture[j])
        return diff / self.F

    def distance_matrix(self) -> np.ndarray:
        """计算所有群体对的文化距离矩阵"""
        D = np.zeros((self.N, self.N))
        for i in range(self.N):
            for j in range(i + 1, self.N):
                D[i, j] = D[j, i] = self.cultural_distance(i, j)
        return D

    def feature_frequencies(self, feature: int) -> Dict[int, float]:
        """计算特定特征的频率分布"""
        counts = Counter(self.culture[:, feature])
        return {trait: count / self.N for trait, count in counts.items()}

    def cultural_diversity(self, method: str = 'simpson') -> float:
        """
        计算文化多样性

        method: 'simpson' 或 'shannon'
        """
        total_diversity = 0

        for f in range(self.F):
            freqs = list(self.feature_frequencies(f).values())

            if method == 'simpson':
                # Simpson指数: 1 - Σp²
                diversity = 1 - sum(p ** 2 for p in freqs)
            elif method == 'shannon':
                # Shannon熵: -Σp·log(p)
                import math
                diversity = -sum(p * math.log(p) if p > 0 else 0 for p in freqs)
            else:
                diversity = len(freqs) / self.q[f]  # 归一化特征数

            total_diversity += diversity

        return total_diversity / self.F

    def cultural_centers(self, n_centers: int = 3) -> Tuple[np.ndarray, np.ndarray]:
        """
        识别文化中心（使用K-means）

        返回: (中心位置, 群体标签)
        """
        kmeans = KMeans(n_clusters=n_centers, random_state=42, n_init=10)
        labels = kmeans.fit_predict(self.culture)

        return kmeans.cluster_centers_, labels

    def detect_boundaries(self, threshold: float = 0.5) -> List[Tuple[int, int]]:
        """
        检测文化边界

        相邻群体（地理上）且文化距离>阈值的边
        """
        boundaries = []

        # 构建Delaunay三角网或简单近邻
        for i in range(self.N):
            for j in range(i + 1, self.N):
                # 地理距离
                geo_dist = np.linalg.norm(self.locations[i] - self.locations[j])

                # 如果地理相邻且文化距离大，则为边界
                if geo_dist < 0.3 and self.cultural_distance(i, j) > threshold:
                    boundaries.append((i, j))

        return boundaries

    def simulate_diffusion(self, n_steps: int = 100,
                          interaction_prob: float = 0.1) -> List[float]:
        """
        模拟文化传播

        基于地理邻近的同质性传播
        """
        diversity_history = [self.cultural_diversity()]

        for _ in range(n_steps):
            # 随机选择群体对
            i, j = np.random.choice(self.N, 2, replace=False)

            # 地理距离影响互动概率
            geo_dist = np.linalg.norm(self.locations[i] - self.locations[j])
            prob = interaction_prob * np.exp(-geo_dist)

            if np.random.rand() < prob:
                # 文化传播: j影响i
                # 以文化相似度为概率采纳特征
                sim = 1 - self.cultural_distance(i, j)

                for f in range(self.F):
                    if np.random.rand() < sim:
                        self.culture[i, f] = self.culture[j, f]

            if _ % 10 == 0:
                diversity_history.append(self.cultural_diversity())

        return diversity_history

    def visualize(self, ax=None, show_centers: bool = True):
        """可视化文化空间"""
        if ax is None:
            fig, ax = plt.subplots(figsize=(10, 8))

        # 使用PCA降维到2D
        if self.F > 2:
            pca = PCA(n_components=2)
            coords = pca.fit_transform(self.culture)
        else:
            coords = self.culture[:, :2]

        # 聚类着色
        _, labels = self.cultural_centers(n_centers=4)
        scatter = ax.scatter(coords[:, 0], coords[:, 1],
                           c=labels, cmap='viridis', s=100, alpha=0.6)

        # 标记文化中心
        if show_centers:
            centers, _ = self.cultural_centers(n_centers=4)
            if self.F > 2:
                centers_2d = pca.transform(centers)
            else:
                centers_2d = centers[:, :2]
            ax.scatter(centers_2d[:, 0], centers_2d[:, 1],
                      c='red', s=300, marker='*', edgecolors='black',
                      label='Cultural Centers', zorder=5)

        ax.set_xlabel('Dimension 1')
        ax.set_ylabel('Dimension 2')
        ax.set_title('Cultural Space (PCA projection)')
        ax.legend()

        return ax


class CulturalPattern:
    """
    文化模式分析

    模式识别、分布分析
    """

    def __init__(self, culture_data: np.ndarray):
        """
        参数:
            culture_data: 文化配置矩阵 (n_populations × n_features)
        """
        self.data = culture_data
        self.N, self.F = culture_data.shape

    def motif_detection(self, motif_size: int = 3) -> Dict[Tuple, int]:
        """
        检测频繁出现的文化模式（motif）

        返回: 模式→频率 的字典
        """
        motif_counts = defaultdict(int)

        # 简化：统计特征组合
        for i in range(self.N):
            # 将特征组合转为元组
            pattern = tuple(self.data[i])
            motif_counts[pattern] += 1

        return dict(motif_counts)

    def core_periphery_structure(self) -> Tuple[List[int], List[int]]:
        """
        识别核心-边缘结构

        返回: (核心群体, 边缘群体)
        """
        # 计算每个群体的"中心性"
        distances = self.distance_matrix()
        centrality = 1 / (np.mean(distances, axis=1) + 1e-10)

        # 阈值分割
        threshold = np.median(centrality)
        core = [i for i in range(self.N) if centrality[i] >= threshold]
        periphery = [i for i in range(self.N) if centrality[i] < threshold]

        return core, periphery

    def distance_matrix(self) -> np.ndarray:
        """计算距离矩阵"""
        return squareform(pdist(self.data, metric='hamming'))

    def cultural_tree(self) -> nx.Graph:
        """
        构建文化演化树（Neighbor Joining简化版）
        """
        D = self.distance_matrix()

        # 使用层次聚类构建树
        from scipy.cluster.hierarchy import linkage, to_tree
        Z = linkage(squareform(D), method='average')

        # 转换为NetworkX图
        G = nx.Graph()

        # 简化的树结构
        for i in range(self.N):
            G.add_node(f'pop_{i}', type='leaf')

        return G


class CulturalEvolutionSimulator:
    """
    文化演化模拟器

    创新、选择、漂变的综合模型
    """

    def __init__(self, n_features: int = 10, n_traits: int = 5):
        self.F = n_features
        self.q = n_traits
        self.population = None

    def initialize(self, n_individuals: int = 100):
        """初始化随机种群"""
        self.population = np.random.randint(0, self.q, (n_individuals, self.F))
        self.N = n_individuals

    def innovation(self, rate: float = 0.01):
        """创新: 随机改变特征"""
        for i in range(self.N):
            for f in range(self.F):
                if np.random.rand() < rate:
                    self.population[i, f] = np.random.randint(0, self.q)

    def biased_transmission(self, bias_strength: float = 2.0):
        """有偏传播: 依从多数"""
        new_pop = self.population.copy()

        for i in range(self.N):
            # 随机选择示范者
            demonstrators = np.random.choice(self.N, size=3, replace=False)

            for f in range(self.F):
                # 统计示范者特征频率
                traits = self.population[demonstrators, f]
                freqs = Counter(traits)

                # 有偏采纳
                max_trait = max(freqs.items(), key=lambda x: x[1])[0]
                prob = (freqs[max_trait] / len(traits)) ** bias_strength

                if np.random.rand() < prob:
                    new_pop[i, f] = max_trait

        self.population = new_pop

    def conformist_transmission(self, conformity: float = 0.8):
        """从众传播"""
        new_pop = self.population.copy()

        for i in range(self.N):
            # 观察样本
            sample = self.population[np.random.choice(self.N, size=5, replace=False)]

            for f in range(self.F):
                # 多数特征
                majority = Counter(sample[:, f]).most_common(1)[0][0]

                if np.random.rand() < conformity:
                    new_pop[i, f] = majority

        self.population = new_pop

    def run(self, n_generations: int = 100,
           innovation_rate: float = 0.01) -> Dict:
        """
        运行模拟

        返回: 统计历史
        """
        history = {
            'diversity': [],
            'unique_cultures': [],
            'generation': []
        }

        for gen in range(n_generations):
            # 演化步骤
            self.innovation(innovation_rate)
            self.conformist_transmission()

            # 记录统计
            if gen % 10 == 0:
                unique = len(set(tuple(row) for row in self.population))
                diversity = len(unique) / self.N if self.N > 0 else 0

                history['unique_cultures'].append(unique)
                history['diversity'].append(diversity)
                history['generation'].append(gen)

        return history
```

## 15.4.2.6 文化进化理论

### 双继承理论

**模型 15.4.3** (基因-文化共演化)

基因频率 $g$ 和文化频率 $c$ 共同演化：

$$\Delta g = g(1-g) \cdot \frac{\partial w}{\partial g}$$
$$\Delta c = c(1-c) \cdot \frac{\partial w}{\partial c} + \text{传播偏差}$$

其中 $w$ 为适应度。
