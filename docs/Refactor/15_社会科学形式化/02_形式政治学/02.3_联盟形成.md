## 15.2.3 联盟形成

### 概述

联盟形成理论研究政治行为者如何通过合作博弈形成稳定的政府或联盟。
该理论结合了合作博弈论的核心概念（如核、稳定集）与政治学中的具体应用（如议会联盟、政府形成）。

**参考文献**: von Neumann & Morgenstern (1944), Riker (1962), Axelrod (1970), Laver & Schofield (1990)

---

## 15.2.3.1 合作博弈基础

### 特征函数博弈

**定义 15.2.18** (Transferable Utility博弈, TU-Game)

TU博弈定义为 $(N, v)$：

- $N = \{1, 2, \ldots, n\}$：参与人集合
- $v: 2^N \to \mathbb{R}$：特征函数，$v(S)$ 为联盟 $S$ 能保证获得的最大总收益

满足：

- **规范性**: $v(\emptyset) = 0$
- **超可加性**: $S \cap T = \emptyset \Rightarrow v(S \cup T) \geq v(S) + v(T)$

**定义 15.2.19** (分配/Imputation)

分配 $x = (x_1, \ldots, x_n)$ 满足：

- **有效性**: $\sum_{i \in N} x_i = v(N)$
- **个体理性**: $x_i \geq v(\{i\})$ 对所有 $i$

---

### 核心

**定义 15.2.20** (核心, Core)

核心是所有不被任何联盟阻止的分配：

$$C(v) = \{x \in \mathbb{R}^n : \sum_{i \in N} x_i = v(N) \text{ 且 } \sum_{i \in S} x_i \geq v(S) \text{ 对所有 } S \subseteq N\}$$

**解释**: 核心中的分配使得没有任何子联盟能通过偏离获得更高收益。

**定理 15.2.10** (核心存在性)

凸博弈的核心非空：

**凸博弈**: $v(S) + v(T) \leq v(S \cup T) + v(S \cap T)$ 对所有 $S, T$

---

## 15.2.3.2 Shapley值

### 定义

**定义 15.2.21** (Shapley值, 1953)

$$\phi_i(v) = \sum_{S \subseteq N \setminus \{i\}} \frac{|S|!(n-|S|-1)!}{n!} [v(S \cup \{i\}) - v(S)]$$

**解释**: 参与人 $i$ 的边际贡献在所有可能加入顺序下的平均值。

---

### 公理化

**定理 15.2.11** (Shapley公理)

Shapley值是唯一满足以下公理的解概念：

1. **效率性**: $\sum_i \phi_i(v) = v(N)$
2. **对称性**: 若 $v(S \cup \{i\}) = v(S \cup \{j\})$ 对所有 $S$，则 $\phi_i = \phi_j$
3. **哑元性**: 若 $v(S \cup \{i\}) = v(S)$ 对所有 $S$，则 $\phi_i = 0$
4. **可加性**: $\phi(v + w) = \phi(v) + \phi(w)$

---

## 15.2.3.3 政府形成模型

### Riker最小获胜联盟理论

**假设 15.2.1** (Riker, 1962)

政治行为者寻求最大化他们在政府中的权力份额，同时最小化联盟规模。

**定义 15.2.22** (最小获胜联盟)

联盟 $S$ 是最小获胜的，若：

- $v(S) = 1$（获胜）
- $v(S \setminus \{i\}) = 0$ 对所有 $i \in S$（移除任何成员即失败）

**定理 15.2.12** (Riker规模原则)

在零和竞争下，理性行为者将形成最小获胜联盟。

---

### 政策空间模型

**定义 15.2.23** (议会博弈)

- $N$：政党集合
- $w_i$：政党 $i$ 的席位数
- $x_i \in \mathbb{R}^d$：政党 $i$ 的理想政策点
- 配额 $q > \frac{1}{2}\sum_i w_i$

**定义 15.2.24** (封闭规则博弈)

政府政策为执政联盟成员理想点的加权平均：

$$g(S) = \frac{\sum_{i \in S} w_i x_i}{\sum_{i \in S} w_i}$$

政党 $i$ 的效用：$u_i(S) = -||g(S) - x_i||^2$

---

### 最小连接获胜联盟 (MCW)

**定义 15.2.25** (Axelrod, 1970)

最小连接获胜联盟是最小的政策上相连的获胜联盟。

**假设**: 政党按单一政策维度（如左右光谱）排列，联盟在政策空间上必须是连续的。

**示例**:

政党位置: [极左] L -- C -- R [极右]

若 $w_L = 40, w_C = 20, w_R = 40$，配额 $q = 51$

- $\{L, C\}$: 获胜且最小
- $\{C, R\}$: 获胜且最小
- $\{L, R\}$: 获胜但不连接（跳过中间）

---

## 15.2.3.4 联盟稳定性

### 核仁

**定义 15.2.26** (超额向量)

对分配 $x$，联盟 $S$ 的超额为：

$$e(S, x) = v(S) - \sum_{i \in S} x_i$$

超额表示联盟 $S$ 对分配 $x$ 的不满程度。

**定义 15.2.27** (核仁, Nucleolus)

核仁是使最大超额最小化的分配，然后次大超额最小化，依此类推。

形式化：将超额向量按降序排列，核仁是字典序最小的分配。

---

### 稳定集

**定义 15.2.28** (von Neumann-Morgenstern稳定集)

集合 $V$ 是稳定集，若：

1. **内部稳定性**: $x, y \in V \Rightarrow$ $x$ 不支配 $y$
2. **外部稳定性**: $x \notin V \Rightarrow \exists y \in V$ 使得 $y$ 支配 $x$

**定义 15.2.29** (支配)

分配 $y$ 支配 $x$（通过联盟 $S$），若：

- $\sum_{i \in S} y_i \leq v(S)$
- $y_i > x_i$ 对所有 $i \in S$

---

## 15.2.3.5 计算模型

### 算法实现

```python
"""
联盟形成模型计算
合作博弈解概念、政府形成模拟
"""

import numpy as np
from itertools import combinations, permutations
from typing import List, Tuple, Dict, Set, Callable
import matplotlib.pyplot as plt
from scipy.optimize import linprog
from collections import defaultdict

class CooperativeGame:
    """
    合作博弈 (TU-Game)

    (N, v) 其中 v: 2^N → R
    """

    def __init__(self, n: int, characteristic: Callable = None):
        """
        参数:
            n: 参与人数量
            characteristic: 特征函数 v(S)，若None则使用简单多数
        """
        self.n = n
        self.N = set(range(n))

        # 预计算所有联盟的值
        self.coalition_values = {}
        self.all_coalitions = []

        for r in range(n + 1):
            for coalition in combinations(range(n), r):
                S = frozenset(coalition)
                self.all_coalitions.append(S)

                if characteristic is not None:
                    self.coalition_values[S] = characteristic(S)
                else:
                    # 默认：简单投票博弈
                    self.coalition_values[S] = 1.0 if len(S) > n / 2 else 0.0

        self.v = self.coalition_values

    def get_value(self, S: Set[int]) -> float:
        """获取联盟S的特征函数值"""
        return self.v.get(frozenset(S), 0)

    def is_superadditive(self) -> bool:
        """检查超可加性"""
        for S in self.all_coalitions:
            for T in self.all_coalitions:
                if S.isdisjoint(T):
                    lhs = self.v[S] + self.v[T]
                    rhs = self.v[S | T]
                    if lhs > rhs + 1e-10:
                        return False
        return True

    def is_convex(self) -> bool:
        """检查凸性"""
        for S in self.all_coalitions:
            for T in self.all_coalitions:
                lhs = self.v[S] + self.v[T]
                rhs = self.v[S | T] + self.v[S & T]
                if lhs > rhs + 1e-10:
                    return False
        return True

    def core(self) -> List[np.ndarray]:
        """
        计算核心 (使用线性规划)

        寻找满足:
        - Σx_i = v(N)
        - Σ_{i∈S} x_i ≥ v(S) 对所有S
        - x_i ≥ v({i})
        """
        # 构建约束
        n_constraints = len(self.all_coalitions) + self.n

        # 等式约束: sum(x) = v(N)
        A_eq = np.ones((1, self.n))
        b_eq = np.array([self.v[self.N]])

        # 不等式约束: sum_{i∈S} x_i >= v(S)
        A_ub = []
        b_ub = []

        for S in self.all_coalitions:
            if S != self.N and len(S) > 0:
                constraint = np.zeros(self.n)
                for i in S:
                    constraint[i] = -1  # 注意: linprog 是 <= 约束
                A_ub.append(constraint)
                b_ub.append(-self.v[S])

        # 个体理性: x_i >= v({i})
        for i in range(self.n):
            constraint = np.zeros(self.n)
            constraint[i] = -1
            A_ub.append(constraint)
            b_ub.append(-self.v[frozenset({i})])

        A_ub = np.array(A_ub)
        b_ub = np.array(b_ub)

        # 目标函数: 最小化0 (可行性问题)
        c = np.zeros(self.n)

        try:
            result = linprog(c, A_ub=A_ub, b_ub=b_ub, A_eq=A_eq, b_eq=b_eq,
                           bounds=(None, None), method='highs')

            if result.success:
                return [result.x]
            else:
                return []  # 核心为空
        except:
            return []

    def shapley_value(self) -> Dict[int, float]:
        """
        计算Shapley值

        φ_i = Σ |S|!(n-|S|-1)!/n! * [v(S∪{i}) - v(S)]
        """
        phi = np.zeros(self.n)

        for i in range(self.n):
            for S in self.all_coalitions:
                if i not in S:
                    S_with_i = S | {i}
                    marginal = self.v[S_with_i] - self.v[S]

                    s = len(S)
                    weight = np.math.factorial(s) * np.math.factorial(self.n - s - 1)
                    weight /= np.math.factorial(self.n)

                    phi[i] += weight * marginal

        return {i: phi[i] for i in range(self.n)}

    def nucleolus(self, max_iter: int = 100) -> np.ndarray:
        """
        计算核仁 (迭代算法)

        最小化最大超额，然后次大，依此类推
        """
        # 简化为计算近似核仁
        # 第一步：找到核心元素（如果存在）
        core_elements = self.core()

        if len(core_elements) > 0:
            return core_elements[0]

        # 否则，计算最小化最大超额的分配
        # 使用线性规划迭代
        epsilon = 0

        for _ in range(max_iter):
            # 构建LP: 最小化epsilon，满足所有超额 <= epsilon
            c = np.zeros(self.n + 1)
            c[-1] = 1  # epsilon

            A_eq = np.zeros((1, self.n + 1))
            A_eq[0, :self.n] = 1
            b_eq = np.array([self.v[self.N]])

            A_ub = []
            b_ub = []

            for S in self.all_coalitions:
                if S != self.N and len(S) > 0:
                    constraint = np.zeros(self.n + 1)
                    for i in S:
                        constraint[i] = -1
                    constraint[-1] = 1
                    A_ub.append(constraint)
                    b_ub.append(-self.v[S])

            result = linprog(c, A_ub=np.array(A_ub), b_ub=np.array(b_ub),
                           A_eq=A_eq, b_eq=b_eq, bounds=[(None, None)]*self.n + [(0, None)])

            if result.success:
                return result.x[:-1]

        return np.ones(self.n) * self.v[self.N] / self.n  # 均等分配作为备选


class GovernmentFormation:
    """
    政府形成模型

    基于Laver-Schofield政策空间方法
    """

    def __init__(self, seats: List[int], positions: List[float],
                 quota: float = None, party_names: List[str] = None):
        """
        参数:
            seats: 各政党席位数
            positions: 各政党政策位置（一维左右光谱）
            quota: 获胜所需席位，默认多数
            party_names: 政党名称
        """
        self.seats = np.array(seats)
        self.positions = np.array(positions)
        self.n_parties = len(seats)
        self.total_seats = np.sum(seats)

        if quota is None:
            self.quota = self.total_seats / 2 + 0.1
        else:
            self.quota = quota

        if party_names is None:
            self.names = [f"P{i+1}" for i in range(self.n_parties)]
        else:
            self.names = party_names

        # 按位置排序
        sorted_idx = np.argsort(self.positions)
        self.sorted_idx = sorted_idx
        self.seats_sorted = self.seats[sorted_idx]
        self.positions_sorted = self.positions[sorted_idx]
        self.names_sorted = [self.names[i] for i in sorted_idx]

    def government_policy(self, coalition: Set[int]) -> float:
        """计算执政联盟的政策位置（加权平均）"""
        if len(coalition) == 0:
            return 0

        total_seats = sum(self.seats[i] for i in coalition)
        weighted_pos = sum(self.seats[i] * self.positions[i] for i in coalition)

        return weighted_pos / total_seats

    def is_winning(self, coalition: Set[int]) -> bool:
        """检查是否为获胜联盟"""
        return sum(self.seats[i] for i in coalition) >= self.quota

    def is_minimal_winning(self, coalition: Set[int]) -> bool:
        """检查是否为最小获胜联盟"""
        if not self.is_winning(coalition):
            return False

        for i in coalition:
            if self.is_winning(coalition - {i}):
                return False

        return True

    def is_connected(self, coalition: Set[int]) -> bool:
        """检查联盟在政策空间上是否相连"""
        if len(coalition) <= 1:
            return True

        # 检查排序后是否连续
        sorted_coalition = sorted([list(self.sorted_idx).index(i) for i in coalition])
        return sorted_coalition == list(range(sorted_coalition[0], sorted_coalition[-1] + 1))

    def minimal_connected_winning(self) -> List[Set[int]]:
        """寻找所有最小连接获胜联盟 (MCW)"""
        mcw = []

        for r in range(1, self.n_parties + 1):
            for coalition in combinations(range(self.n_parties), r):
                S = set(coalition)
                if self.is_minimal_winning(S) and self.is_connected(S):
                    mcw.append(S)

        return mcw

    def party_utility(self, party: int, coalition: Set[int]) -> float:
        """计算政党对执政联盟的效用"""
        if party not in coalition:
            # 在野党效用：与执政党政策距离
            gov_pos = self.government_policy(coalition)
            return -abs(self.positions[party] - gov_pos)

        # 执政党效用：席位份额 - 政策偏离成本
        gov_pos = self.government_policy(coalition)
        seat_share = self.seats[party] / sum(self.seats[i] for i in coalition)
        policy_cost = abs(self.positions[party] - gov_pos)

        return seat_share - 0.5 * policy_cost

    def simulate_formation(self, protocol: str = 'minimal') -> Dict:
        """
        模拟政府形成过程

        参数:
            protocol: 'minimal' (最小获胜) 或 'mcw' (最小连接获胜)
        """
        if protocol == 'mcw':
            possible = self.minimal_connected_winning()
        else:
            possible = [set(c) for r in range(1, self.n_parties + 1)
                       for c in combinations(range(self.n_parties), r)
                       if self.is_minimal_winning(set(c))]

        if len(possible) == 0:
            return {'status': 'no_stable_government'}

        # 计算各政党对可能联盟的偏好
        preferences = defaultdict(list)

        for party in range(self.n_parties):
            coalition_utils = []
            for coalition in possible:
                util = self.party_utility(party, coalition)
                coalition_utils.append((coalition, util))

            # 按效用排序
            coalition_utils.sort(key=lambda x: -x[1])
            preferences[party] = coalition_utils

        # 寻找稳定联盟（不被任何子联盟阻止）
        stable = []
        for coalition in possible:
            is_stable = True
            for party in coalition:
                # 检查是否有更好的选择
                for alt_coalition, util in preferences[party]:
                    if alt_coalition != coalition and util > self.party_utility(party, coalition):
                        # 检查alt_coalition是否也包含能改进的其他成员
                        can_deviate = True
                        for other in alt_coalition:
                            if self.party_utility(other, alt_coalition) <= self.party_utility(other, coalition):
                                if other != party:
                                    can_deviate = False
                                    break
                        if can_deviate:
                            is_stable = False
                            break

            if is_stable:
                stable.append(coalition)

        return {
            'possible_governments': possible,
            'stable_governments': stable,
            'party_preferences': dict(preferences)
        }

        ax1.barh(0, seat, left=cumulative, color=color, alpha=0.7, edgecolor='black')
        ax1.text(cumulative + seat/2, 0, f'{name}\n{seat}',
                ha='center', va='center', fontsize=10, fontweight='bold')
        cumulative += seat

    ax1.axvline(x=quota, color='green', linestyle='--', linewidth=2, label='Quota')
    ax1.set_xlim(0, 100)
    ax1.set_ylim(-0.5, 0.5)
    ax1.set_xlabel('席位')
    ax1.set_title('议会构成与获胜配额')
    ax1.legend()
    ax1.set_yticks([])
    ax1.grid(True, alpha=0.3, axis='x')

    # 图2: 政党位置与可能联盟
    ax2 = axes[0, 1]

    for i, (pos, seat, name, color) in enumerate(zip(positions, seats, names, colors)):
        ax2.scatter([pos], [seat], s=seat*20, c=color, alpha=0.6, edgecolors='black')
        ax2.annotate(name, (pos, seat), textcoords="offset points",
                    xytext=(0, 10), ha='center', fontsize=10)

    # 绘制MCW
    for coalition in mcw[:3]:
        coalition_positions = [positions[i] for i in coalition]
        coalition_seats = [seats[i] for i in coalition]
        ax2.plot(coalition_positions, coalition_seats, 'k--', alpha=0.3)

    ax2.set_xlabel('政策位置 (左→右)')
    ax2.set_ylabel('席位数')
    ax2.set_title('政策空间与最小连接获胜联盟')
    ax2.grid(True, alpha=0.3)

    # 图3: Shapley值分布
    ax3 = axes[1, 0]

    # 创建加权投票博弈计算Shapley值
    vote_game = CooperativeGame(5, lambda S: 1.0 if sum(seats[i] for i in S) >= quota else 0)
    shapley_parliament = vote_game.shapley_value()

    shapley_values = [shapley_parliament[i] for i in range(5)]
    weight_shares = [s/sum(seats) for s in seats]

    x = np.arange(5)
    width = 0.35

    bars1 = ax3.bar(x - width/2, weight_shares, width, label='席位份额', alpha=0.8, color='lightblue')
    bars2 = ax3.bar(x + width/2, shapley_values, width, label='Shapley权力', alpha=0.8, color='orange')

    ax3.set_ylabel('比例')
    ax3.set_title('席位份额 vs Shapley权力指数')
    ax3.set_xticks(x)
    ax3.set_xticklabels(names, rotation=45)
    ax3.legend()
    ax3.grid(True, alpha=0.3, axis='y')

    # 图4: 核心存在性与博弈凸性
    ax4 = axes[1, 1]

    # 生成随机博弈并测试核心
    np.random.seed(42)
    n_tests = 50
    convex_count = 0
    nonempty_core_count = 0

    for n_players in range(3, 8):
        convex_n = 0
        core_n = 0

        for _ in range(n_tests):
            # 随机特征函数
            v_random = {}
            for r in range(n_players + 1):
                for coalition in combinations(range(n_players), r):
                    v_random[frozenset(coalition)] = np.random.uniform(0, 10)

            # 确保超可加性
            for i in range(n_players):
                v_random[frozenset({i})] = 0

            game_test = CooperativeGame(n_players, lambda S: v_random.get(S, 0))

            if game_test.is_convex():
                convex_n += 1

            if len(game_test.core()) > 0:
                core_n += 1

        ax4.scatter([n_players], [convex_n/n_tests], c='red', s=100, marker='o')
        ax4.scatter([n_players], [core_n/n_tests], c='blue', s=100, marker='s')

    ax4.plot([], [], 'ro', label='凸博弈比例')
    ax4.plot([], [], 'bs', label='非空核心比例')
    ax4.set_xlabel('参与人数量')
    ax4.set_ylabel('比例')
    ax4.set_title('博弈性质随规模变化 (随机博弈)')
    ax4.legend()
    ax4.grid(True)
    ax4.set_ylim(0, 1)

    plt.tight_layout()
    plt.savefig('coalition_formation.png', dpi=150, bbox_inches='tight')
    plt.show()
    print("\n图形已保存至 coalition_formation.png")
```

## 15.2.3.6 动态联盟形成

### 讨价还价模型

**定义 15.2.30** (Rubinstein轮流讨价还价)

两个参与人分配蛋糕，轮流提出分配方案，折现因子 $\delta$

**定理 15.2.13** (唯一子博弈精炼均衡)

均衡分配：

$$x_1^* = \frac{1 - \delta_2}{1 - \delta_1 \delta_2}, \quad x_2^* = \frac{\delta_2(1 - \delta_1)}{1 - \delta_1 \delta_2}$$
