# 范畴论速查卡

> 单页速查 | 基本定义 | 常见范畴 | 函子 | 自然变换

---

## 基本定义

### 范畴 (Category)

| 组成 | 定义 | 记号 |
|------|------|------|
| **对象类** | 对象的集合 | Ob(C) 或 C_0 |
| **态射集** | 对象间的箭头 | Hom_C(A, B) 或 C(A,B) |
| **复合** | 态射组合 | ∘ : C(B,C) × C(A,B) → C(A,C) |
| **恒等** | 每个对象的恒等态射 | id_A : A → A |

### 范畴公理

| 公理 | 陈述 | 公式 |
|------|------|------|
| **结合律** | 复合可结合 | (h ∘ g) ∘ f = h ∘ (g ∘ f) |
| **单位律** | 恒等为左右单位元 | f ∘ id_A = f = id_B ∘ f |

### 特殊态射

| 类型 | 定义 | 直觉 |
|------|------|------|
| **单态射** (Mono) | f ∘ g_1 = f ∘ g_2 ⇒ g_1 = g_2 | 左可消去，单射类比 |
| **满态射** (Epi) | g_1 ∘ f = g_2 ∘ f ⇒ g_1 = g_2 | 右可消去，满射类比 |
| **同构** (Iso) | ∃g: f ∘ g = id, g ∘ f = id | 可逆 |
| **自态射** (Endo) | f : A → A | 从对象到自身 |
| **自同构** (Auto) | 可逆的自态射 | Aut(A) 构成群 |

### 特殊对象

| 对象 | 泛性质 | 定义 |
|------|--------|------|
| **始对象** 0 | ∀A, ∃! 0 → A | 有唯一的出态射 |
| **终对象** 1 | ∀A, ∃! A → 1 | 有唯一的入态射 |
| **零对象** | 既是始又是终 | 0 = 1 |

| 范畴 | 构造方式 |
|------|----------|

## 泛构造

### 积与余积

| 构造 | 泛性质 | 说明 | 例子(Set) |
|------|--------|------|-----------|
| **积** A × B | 投影 π_1, π_2，泛性质 | 对象P到A,B的投影 | 笛卡尔积 |
| **余积** A + B | 嵌入 i_1, i_2，泛性质 | A,B到对象P的嵌入 | 不交并 |
| **等化子** | f ∘ e = g ∘ e，泛 | 最大的使f=g相等的子对象 | {x | f(x) = g(x)} |
| **余等化子** | c ∘ f = c ∘ g，泛 | 最小的使f=g相等的商 | 商集 |
| **拉回** | 泛方块 | 纤维积 | A ×_C B |
| **推出** | 泛余方块 |  | 不交并的商 |

### 极限与余极限

|------|------|--------|

| **余极限** colim F | 图表 J → C | 余锥的始对象 |
| **终对象** | 空图表 | 特殊极限 |
| **始对象** | 空图表 | 特殊余极限 |

## 函子 (Functor)

### 定义

| 组成 | 映射 |
|------|------|
| **对象映射** | F : Ob(C) → Ob(D) |
| **态射映射** | F : C(A,B) → D(FA, FB) |
| **保复合** | F(g ∘ f) = Fg ∘ Ff |
| **保恒等** | F(id_A) = id_{FA} |

### 函子类型

| 类型 | 条件 | 例子 |
|------|------|------|
| **协变函子** | 标准定义 | 大多数函子 |
| **反变函子** | C^op → D | 反变Hom函子 |
| **忘却函子** | 遗忘结构 | U : Grp → Set |
| **自由函子** | 自由构造 | F : Set → Grp (自由群) |
| **忠实函子** | F 在态射上单射 | |
| **全函子** | F 在态射上满射 | |
| **完全忠实** | 双射 | 等价嵌入 |
| **本质满** | 每个D对象同构于某FA | |
| **范畴等价** | 完全忠实+本质满 | C ≅ D |

### 重要函子

| 函子 | 定义 | 范畴 |
|------|------|------|
| **恒等函子** Id_C | Id(A) = A, Id(f) = f | C → C |
| **Hom函子** Hom(A, -) | 协变 | C → Set |
| **Hom函子** Hom(-, B) | 反变 | C^op → Set |
| **积函子** × | (A,B) ↦ A × B | C × C → C |
| **对角函子** Δ | A ↦ (A,A) | C → C × C |

## 自然变换 (Natural Transformation)

### 定义

| 要素 | 说明 |
|------|------|
| **成分** | α_A : FA → GA，对每个 A ∈ C |
| **自然性条件** | α_B ∘ Ff = Gf ∘ α_A |

### 自然性方块

```
     Ff
FA ------> FB
 |          |
 |α_A      | α_B
 v          v
GA ------> GB
     Gf
```

### 自然变换操作

| 操作 | 记号 | 结果 |
|------|------|------|
| **垂直复合** | β ∘ α | (F ⇒ G) × (G ⇒ H) → (F ⇒ H) |
| **水平复合** | β * α | 函子复合的自然变换 |
| **恒等变换** | id_F | F ⇒ F |
| **自然同构** | α : F ≅ G | 每个α_A是同构 |

### 重要定理

| 定理 | 陈述 |
|------|------|
| **Yoneda引理** | Nat(Hom(A, -), F) ≅ FA |
| **Yoneda嵌入** | y : C → [C^op, Set] 完全忠实 |
| **伴随函子** | D(FA, B) ≅ C(A, GB) 自然同构 |

## 伴随

| 概念 | 定义 |
|------|------|
| **伴随** F ⊣ G | 函子 F : C → D, G : D → C |
| **单位** η : Id_C ⇒ GF |  η_A : A → GFA |
| **余单位** ε : FG ⇒ Id_D | ε_B : FGB → B |
| **三角恒等式** | ε_F ∘ Fη = id_F, Gε ∘ η_G = id_G |

| 左伴随 F | 右伴随 G | 范畴 |
|-----------|-----------|------|
| 自由群构造 | 忘却函子 | Set ⊣ Grp |
| 离散拓扑 | 底集 | Set ⊣ Top |
| 积 | 指数 | C 是笛卡尔闭 |
