## 类型规则速查

### 基础类型构造

| 构造 | 直觉 | 类型规则 | 范畴论 |
|------|------|----------|--------|
| **积类型** $A × B$ | 同时有A和B | $(a, b) : A × B$ | 积 (Product) |
| **和类型** $A + B$ | 或有A或有B | `inl a` / `inr b` : $A + B$ | 余积 (Coproduct) |
| **函数类型** $A → B$ | A到B的映射 | $λx.e : A → B$ | 指数对象 |
| **单位类型** $1$ / $()$ | 单元素 | $() : 1$ | 终对象 |
| **空类型** $0$ / $⊥$ | 无元素 | 无构造子 | 始对象 |

### 依赖类型

| 构造 | 符号 | 说明 | 例子 |
|------|------|------|------|
| **依赖积** | $Π(x:A).B(x)$ | 依赖函数 | `(x:A) → B x` |
| **依赖和** | $Σ(x:A).B(x)$ | 依赖对 | `(x:A) × B x` |
| **恒等类型** | $Id_A(a, b)$ / $a =_A b$ | 相等证明 | `refl a : a = a` |

### 类型层级

| 层级 | 名称 | 说明 |
|------|------|------|
| $Type_0$ / $Set$ | 小类型 | 普通数据类型 |
| $Type_1$ | 大类型 | $Type_0$ 的类型 |
| $Type_{n+1}$ | 超类型 | $Type_n$ 的类型 |
| **Prop** / $Type_{-1}$ | 证明无关 | 命题即类型 |
| **Universe** | 宇宙层次 | 避免悖论 |

## 推导规则（自然演绎风格）

### 结构规则

```
Γ ⊢ A : Type    Γ, x:A ⊢ B : Type
---------------------------------  (Π形成)
      Γ ⊢ Π(x:A).B : Type

Γ, x:A ⊢ b : B
---------------------  (Π引入/λ抽象)
Γ ⊢ λx.b : Π(x:A).B

Γ ⊢ f : Π(x:A).B    Γ ⊢ a : A
------------------------------  (Π消除/应用)
        Γ ⊢ f a : B[a/x]
```

### 积类型规则

| 规则 | 形式 |
|------|------|
| **形成** | $Γ ⊢ A : Type \quad Γ ⊢ B : Type \\ \hline Γ ⊢ A × B : Type$ |
| **引入** | $Γ ⊢ a : A \quad Γ ⊢ b : B \\ \hline Γ ⊢ (a, b) : A × B$ |
| **消除(左)** | $Γ ⊢ p : A × B \\ \hline Γ ⊢ π_1 p : A$ |
| **消除(右)** | $Γ ⊢ p : A × B \\ \hline Γ ⊢ π_2 p : B$ |
| **计算** | $π_1(a, b) ≡ a \quad π_2(a, b) ≡ b$ |

### 和类型规则

| 规则 | 形式 |
|------|------|
| **形成** | $Γ ⊢ A : Type \quad Γ ⊢ B : Type \\ \hline Γ ⊢ A + B : Type$ |
| **引入(左)** | $Γ ⊢ a : A \\ \hline Γ ⊢ inl\ a : A + B$ |
| **引入(右)** | $Γ ⊢ b : B \\ \hline Γ ⊢ inr\ b : A + B$ |
| **消除** | $Γ ⊢ s : A+B \quad Γ,x:A⊢c:C \quad Γ,y:B⊢d:C \\ \hline Γ ⊢ case\ s\ of\ inl\ x → c \| inr\ y → d : C$ |

## Curry-Howard 对应

| 逻辑 | 类型论 | 程序/证明 |
|------|--------|-----------|
| **命题** $P$ | **类型** $P$ | 证明即程序 |
| **证明** $p : P$ | **项** $p : P$ | 程序即证明 |
| **真** $⊤$ | **单位类型** $1$ | `()` |
| **假** $⊥$ | **空类型** $0$ | 无构造子 |
| **合取** $P ∧ Q$ | **积类型** $P × Q$ | 对 `(p, q)` |
| **析取** $P ∨ Q$ | **和类型** $P + Q$ | `Left p` / `Right q` |
| **蕴含** $P → Q$ | **函数类型** $P → Q$ | 函数 `λp.q` |
| **否定** $¬P$ | $P → ⊥$ | 到空的函数 |
| **全称** $∀x.P(x)$ | **依赖积** $Π(x:A).P(x)$ | 依赖函数 |
| **存在** $∃x.P(x)$ | **依赖和** $Σ(x:A).P(x)$ | 依赖对 |
| **等式** $a = b$ | **恒等类型** $Id(a, b)$ | `refl` |

### 证明策略 ↔ 程序构造

| 证明策略 | 程序构造 |
|----------|----------|
| 假设引入 | λ抽象 |
| 蕴含消除 (modus ponens) | 函数应用 |
| 合取引入 | 构造对 |
| 合取消除 | 投影 |
| 析取引入 | 构造变体 |
| 析取消除 | case分析 |
| 反证法 | 到空的函数 |
| 存在引入 | 构造依赖对 |
| 存在消除 | 依赖对的投影 |

## 类型系统对比

| 系统 | 特点 | 可表达性 | 一致性 |
|------|------|----------|--------|
| **简单类型λ演算** | 无依赖 | 弱 | 一致 |

## 常见公理/原理

| 名称 | 形式 | 对应类型 |
|------|------|----------|
| **排中律** | $P ∨ ¬P$ | 无对应（经典逻辑） |
| **双重否定消除** | $¬¬P → P$ | 连续性公理 |
| **选择公理** | $(∀x.∃y.R(x,y)) → (∃f.∀x.R(x,fx))$ | 依赖类型可表达 |
| **外延性** | $(∀x. f x = g x) → f = g$ | 函数外延 |
| **唯一性原理** | $K : ∀(A:Type)(x y:A)(p q:x=y). p = q$ | 集合级别成立 |
