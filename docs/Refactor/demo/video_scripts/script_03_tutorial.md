## 第3章: 基础概念 (10:00-18:00)

### 3.1 命题逻辑基础 (10:00-12:00)

**画面**: 概念图解 + 代码演示

**旁白**:
> "在深入学习之前，让我们了解一些基础概念。首先是命题逻辑。命题是一个可以判断真假的陈述句。"

**概念讲解**:

**什么是命题？**
```
命题示例:
✓ "今天是星期一" (可判断真假)
✓ "2 + 2 = 4" (真)
✗ "请关门" (祈使句，不是命题)
```

**代码演示**:
```python
from formal_science import Proposition

# 创建命题
p = Proposition("今天是晴天")
q = Proposition("温度高于30度")

# 设置真值
p.set_value(True)
q.set_value(False)

# 复合命题
conjunction = p.and_(q)  # p ∧ q
disjunction = p.or_(q)   # p ∨ q
implication = p.implies(q)  # p → q

print(f"p 且 q: {conjunction.evaluate()}")
print(f"p 或 q: {disjunction.evaluate()}")
print(f"如果 p 则 q: {implication.evaluate()}")
```

**字幕**: 命题逻辑基础

### 3.3 逻辑等价 (14:00-16:00)

**画面**: 等价公式展示

**旁白**:
> "两个逻辑表达式如果对所有输入都有相同的输出，就称它们是逻辑等价的。这在简化表达式时非常有用。"

**常见等价式**:
```
德摩根律:
  ¬(p ∧ q) ≡ ¬p ∨ ¬q
  ¬(p ∨ q) ≡ ¬p ∧ ¬q

蕴含等价:
  p → q ≡ ¬p ∨ q

双重否定:
  ¬¬p ≡ p
```

**验证代码**:
```python
from formal_science import verify_equivalence

# 验证德摩根律
result = verify_equivalence(
    "not (p and q)",
    "(not p) or (not q)"
)
print(f"德摩根律验证: {'✓ 成立' if result else '✗ 不成立'}")

# 验证蕴含等价
result = verify_equivalence(
    "p implies q",
    "(not p) or q"
)
print(f"蕴含等价验证: {'✓ 成立' if result else '✗ 不成立'}")
```

**字幕**: 逻辑等价验证

### 3.4 量词介绍 (16:00-18:00)

**画面**: 量词概念图解

**旁白**:
> "除了命题逻辑，我们还有谓词逻辑，引入了量词的概念。全称量词 ∀ 表示'对于所有'，存在量词 ∃ 表示'存在'。"

**概念示例**:
```
全称量词 ∀:
  ∀x (Human(x) → Mortal(x))
  所有人都是会死的

存在量词 ∃:
  ∃x (Prime(x) ∧ Even(x))
  存在既是质数又是偶数的数
```

**代码演示**:
```python
from formal_science import Quantifier, Predicate

# 定义谓词
Human = Predicate("Human")
Mortal = Predicate("Mortal")

# 全称命题
universal = Quantifier.forall("x", Human("x").implies(Mortal("x")))
print(f"全称命题: {universal}")

# 存在命题
Even = Predicate("Even")
Prime = Predicate("Prime")
existent = Quantifier.exists("x", Even("x").and_(Prime("x")))
print(f"存在命题: {existent}")
```

**字幕**: 量词基础

```python
# puzzle_solver.py
from formal_science import LogicSolver, Proposition

def solve_puzzle():
    """解决逻辑谜题"""
    
    # 定义命题
    # A_t: 甲是说真话的人
    # B_t: 乙是说真话的人  
    # C_t: 丙是说真话的人
    
    solver = LogicSolver()
    
    # 添加约束：只有一人说真话，一人说谎，一人随机
    # 甲说"乙是说谎者"
    # 如果甲说真话，则乙是说谎者
    # 如果甲说谎，则乙不是说谎者（可能说真话或随机）
    
    # 乙说"丙是说谎者"
    # 类似分析...
    
    # 丙说"甲和乙都是说谎者"
    # 如果丙说真话，则甲和乙都是说谎者
    # 但这与"只有一人说真话"矛盾，所以丙不是说真话的人
    
    # 编码约束...
    solver.add_constraint("...")
    
    # 求解
    solution = solver.solve()
    return solution

if __name__ == "__main__":
    result = solve_puzzle()
    print("谜题解答:")
    print(result)
```
