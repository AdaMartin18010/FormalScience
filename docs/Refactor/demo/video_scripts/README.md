## 视频风格指南

### 通用规范

- **分辨率**: 1920x1080 (1080p)
- **帧率**: 30fps
- **音频**: 清晰人声，背景音乐音量 < -20dB
- **字幕**: 中文/英文双语字幕

### 视觉风格

- **配色**: 使用项目品牌色
- **字体**: 标题 48pt，正文 24pt
- **转场**: 简洁淡入淡出，时长 0.3s
- **标注**: 黄色高亮 + 红色箭头

### 音频规范

- **语速**: 每分钟 200-250 字
- **停顿**: 关键操作后 2-3 秒停顿
- **音效**: 操作成功/失败提示音
