# 项目演示文档

本目录包含 FormalScience 项目的演示文档，帮助用户快速了解如何使用本项目。

## 目录结构

```
demo/
├── README.md              # 本文件 - 演示目录说明
├── usage_examples.md      # 使用示例 - 如何阅读文档、运行代码、使用工具
├── workflow_demo.md       # 工作流演示 - 完整的学习路径和示例
├── screenshots/           # 截图目录 - 界面截图和说明
└── video_scripts/         # 视频脚本目录 - 演示视频大纲
```

## 快速开始

如果你是新用户，建议按以下顺序阅读：

1. **[使用示例](usage_examples.md)** - 了解基本用法
2. **[工作流演示](workflow_demo.md)** - 掌握完整学习路径
3. **查看截图** - 直观了解界面
4. **观看视频** - 跟随视频学习
