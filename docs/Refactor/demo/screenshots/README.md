## 截图规范

### 技术要求

- **格式**: PNG（推荐）或 JPG
- **分辨率**: 1920x1080 或更高
- **文件大小**: 单张不超过 2MB
- **命名**: 小写，下划线连接，英文描述

### 内容要求

1. **清晰可辨** - 文字和界面元素清晰可见
2. **重点突出** - 使用红色方框或箭头标注关键区域
3. **上下文完整** - 包含足够的界面上下文
4. **无敏感信息** - 不包含密码、密钥等敏感信息

### 标注示例

```
┌─────────────────────────────────────────┐
│  [截图内容]                              │
│                                         │
│    ┌─────────────┐                      │
│    │  重点标注区  │ ← 红色箭头标注       │
│    │  [方框]     │                      │
│    └─────────────┘                      │
│                                         │
└─────────────────────────────────────────┘
```

## 使用方式

在文档中引用截图：

```markdown
![描述文字](screenshots/filename.png)
*图 X: 图片描述*
```

示例：
```markdown
![主界面截图](screenshots/main_interface.png)
*图 1: FormalScience 主界面，显示导航菜单和快捷入口*
```
