## 主题说明

本模块从程序语言理论（PLT）的视角探讨类型系统。类型系统是一套规则，它为程序中的每个部分（如变量、表达式）分配一个称为"类型"的属性。语言的类型系统旨在通过在编译或运行时检查类型的一致性，来防止和减少特定类型的程序错误（类型错误）。

本模块是更广泛的 [类型理论](../../04_Type_Theory/README.md) 在程序语言领域的具体应用。
