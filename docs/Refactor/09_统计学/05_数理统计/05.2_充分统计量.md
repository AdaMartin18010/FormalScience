## 9.5.2.1 引言

**充分统计量**（Sufficient Statistic）是统计推断中数据压缩的核心概念。一个充分统计量包含了样本中关于参数的全部信息，使得在已知统计量的条件下，样本不再提供关于参数的额外信息。

## 9.5.2.2 充分统计量的定义

### 9.5.2.2.1 形式化定义

**定义 9.5.2.1**（充分统计量，Sufficient Statistic）

统计量 $T(X)$ 对于参数 $\theta$ 是**充分的**，如果给定 $T(X)$ 时 $X$ 的条件分布不依赖于 $\theta$：

$$f_{X|T}(x | t, \theta) = f_{X|T}(x | t), \quad \forall \theta \in \Theta$$

等价表述：在给定 $T(X)$ 后，$X$ 不包含关于 $\theta$ 的额外信息。

### 9.5.2.2.2 直观解释

充分统计量实现了**数据压缩**而不损失信息。若 $T$ 充分，则基于 $T$ 的推断与基于全部数据 $X$ 的推断等价。

---

## 9.5.2.3 Fisher-Neyman因子分解定理

**定理 9.5.2.2**（因子分解定理，Factorization Theorem）

统计量 $T(X)$ 对于 $\theta$ 充分当且仅当存在函数 $g$ 和 $h$ 使得：

$$f(x | \theta) = g(T(x), \theta) \cdot h(x)$$

即联合密度可分解为仅通过 $T(x)$ 依赖于 $\theta$ 的因子与独立于 $\theta$ 的因子的乘积。

**证明：**

_必要性_：设 $T$ 充分。由条件密度定义：
$$f(x | \theta) = f_{X|T}(x | T(x)) \cdot f_T(T(x) | \theta)$$

取 $h(x) = f_{X|T}(x | T(x))$（由充分性，不依赖 $\theta$），$g(t, \theta) = f_T(t | \theta)$。

_充分性_：设 $f(x | \theta) = g(T(x), \theta)h(x)$。由 $T$ 的边缘密度：
$$f_T(t | \theta) = \int_{T(x)=t} f(x | \theta) dx = g(t, \theta) \int_{T(x)=t} h(x) dx$$

条件密度：
$$f_{X|T}(x | t) = \frac{f(x | \theta)}{f_T(t | \theta)} = \frac{g(t, \theta)h(x)}{g(t, \theta)\int_{T(y)=t}h(y)dy} = \frac{h(x)}{\int_{T(y)=t}h(y)dy}$$

不依赖于 $\theta$，故 $T$ 充分。

**证毕。**

## 9.5.2.4 极小充分统计量

### 9.5.2.4.1 定义

**定义 9.5.2.3**（极小充分统计量）

充分统计量 $T$ 称为**极小的**，如果对任意其他充分统计量 $S$，存在函数 $g$ 使得 $T = g(S)$。

即 $T$ 实现了最大程度的压缩。

### 9.5.2.4.2 Lehmann-Scheffé定理

**定理 9.5.2.4**（Lehmann-Scheffé）

设 $T$ 为完备充分统计量，$\hat{\theta}$ 为基于 $T$ 的无偏估计。则 $\hat{\theta}$ 是UMVUE。

**定义 9.5.2.5**（完备性，Completeness）

统计量 $T$ 是**完备的**，如果对任意函数 $g$：
$$E_\theta[g(T)] = 0, \forall \theta \Rightarrow g(T) = 0 \text{ a.s.}$$

## 9.5.2.5 指数族与充分性

### 9.5.2.5.1 指数族的充分统计量

**定理 9.5.2.6**（指数族的充分统计量）

对于指数族分布：
$$f(x | \theta) = h(x) \exp\left(\sum_{j=1}^{k} \eta_j(\theta) T_j(x) - A(\theta)\right)$$

统计量 $(T_1(X), \ldots, T_k(X))$ 是充分的。

**证明：**

直接应用因子分解定理，$g(t, \theta) = \exp(\sum \eta_j(\theta)t_j - A(\theta))$，$h(x)$ 为原式中的 $h(x)$。

**证毕。**

### 9.5.2.5.2 常见分布的充分统计量

| 分布 | 充分统计量 |
|-----|-----------|
| 正态($\mu$, $\sigma^2$) | $(\sum X_i, \sum X_i^2)$ 或 $(\bar{X}, S^2)$ |
| 泊松($\lambda$) | $\sum X_i$ |
| 二项($n$, $p$) | $\sum X_i$ |
| 指数($\lambda$) | $\sum X_i$ |
| 均匀($0$, $\theta$) | $\max(X_i)$ |

## 9.5.2.6 代码实现

```python
import numpy as np
from scipy import stats
from scipy.special import comb
from typing import Callable, Tuple, List

class Sufficiency:
    """充分统计量相关计算"""

    @staticmethod
    def normal_sufficient_statistics(data: np.ndarray) -> Tuple[float, float]:
        """
        正态分布的充分统计量

        Returns:
            (sum_x, sum_x2) = (ΣXᵢ, ΣXᵢ²)
        """
        return np.sum(data), np.sum(data**2)

    @staticmethod
    def normal_sufficient_to_params(sum_x: float, sum_x2: float, n: int) -> Tuple[float, float]:
        """
        从充分统计量恢复参数估计

        Returns:
            (mu_mle, sigma2_mle)
        """
        mu_mle = sum_x / n
        sigma2_mle = (sum_x2 - sum_x**2 / n) / n
        return mu_mle, sigma2_mle

    @staticmethod
    def poisson_sufficient(data: np.ndarray) -> float:
        """泊松分布的充分统计量"""
        return np.sum(data)

    @staticmethod
    def exponential_sufficient(data: np.ndarray) -> float:
        """指数分布的充分统计量"""
        return np.sum(data)

    @staticmethod
    def uniform_sufficient(data: np.ndarray) -> float:
        """均匀分布U[0,θ]的充分统计量"""
        return np.max(data)


class FactorizationTheorem:
    """因子分解定理验证"""

    @staticmethod
    def verify_factorization_normal(data: np.ndarray) -> bool:
        """
        验证正态分布的因子分解

        f(x|μ,σ²) = (2πσ²)^(-n/2) exp(-1/(2σ²)Σ(xᵢ-μ)²)
                  = g(T(x), θ) · h(x)

        其中 T(x) = (Σxᵢ, Σxᵢ²)
        """
        n = len(data)
        sum_x = np.sum(data)
        sum_x2 = np.sum(data**2)

        # 检查联合密度是否可分解
        # 实际验证通过代数运算
        # Σ(xᵢ - μ)² = Σxᵢ² - 2μΣxᵢ + nμ²

        # 验证代数恒等式
        mu = sum_x / n
        lhs = np.sum((data - mu)**2)
        rhs = sum_x2 - 2*mu*sum_x + n*mu**2

        return np.isclose(lhs, rhs)

    @staticmethod
    def factorization_components_poisson(data: np.ndarray) -> dict:
        """
        泊松分布的因子分解分量

        f(x|λ) = ∏(e^(-λ) λ^(xᵢ) / xᵢ!)
               = e^(-nλ) λ^(Σxᵢ) · ∏(1/xᵢ!)

        g(T, λ) = e^(-nλ) λ^(Σxᵢ)
        h(x) = ∏(1/xᵢ!)
        """
        n = len(data)
        sum_x = np.sum(data)

        # h(x) 独立于λ
        h_x = 1 / np.prod([np.math.factorial(int(xi)) for xi in data])

        # g(T, λ) 需要通过λ参数化
        def g(lambda_param):
            return np.exp(-n * lambda_param) * (lambda_param ** sum_x)

        return {
            'sufficient_statistic': sum_x,
            'h_x': h_x,
            'g_function': g,
            'dimension_reduction': f'{n} → 1'
        }


class RaoBlackwell:
    """Rao-Blackwell定理应用"""

    @staticmethod
    def improve_estimator(data: np.ndarray,
                         unbiased_estimator: Callable,
                         sufficient_stat: Callable) -> float:
        """
        Rao-Blackwell改进估计量

        通过条件期望将任意无偏估计量改进为基于充分统计量的估计量
        """
        # 这是一个概念性实现
        # 实际应用中需要计算条件期望

        t_val = sufficient_stat(data)
        original_est = unbiased_estimator(data)

        # 在实际应用中，这里应该是 E[original_est | T = t_val]
        # 这需要知道条件分布

        return t_val  # 简化返回


# 使用示例
if __name__ == "__main__":
    print("=" * 60)
    print("充分统计量示例")
    print("=" * 60)

    np.random.seed(42)

    # 1. 正态分布的充分统计量
    print("\n1. 正态分布 N(μ=100, σ²=225)")
    print("-" * 40)

    normal_data = np.random.normal(100, 15, 50)
    sum_x, sum_x2 = Sufficiency.normal_sufficient_statistics(normal_data)

    print(f"   样本量: {len(normal_data)}")
    print(f"   充分统计量: T₁ = ΣXᵢ = {sum_x:.2f}")
    print(f"              T₂ = ΣXᵢ² = {sum_x2:.2f}")

    # 验证因子分解
    verified = FactorizationTheorem.verify_factorization_normal(normal_data)
    print(f"   因子分解验证: {'通过' if verified else '失败'}")

    # 从充分统计量恢复MLE
    mu_mle, sigma2_mle = Sufficiency.normal_sufficient_to_params(sum_x, sum_x2, len(normal_data))
    print(f"   从充分统计量恢复: μ̂ = {mu_mle:.2f}, σ̂² = {sigma2_mle:.2f}")
    print(f"   直接计算: μ̂ = {np.mean(normal_data):.2f}, σ̂² = {np.var(normal_data):.2f}")

    # 2. 泊松分布的充分统计量
    print("\n2. 泊松分布 Poisson(λ=4)")
    print("-" * 40)

    poisson_data = np.random.poisson(4, 100)
    sum_x_pois = Sufficiency.poisson_sufficient(poisson_data)

    print(f"   样本量: {len(poisson_data)}")
    print(f"   充分统计量: T = ΣXᵢ = {sum_x_pois}")
    print(f"   降维: 100维 → 1维")
    print(f"   λ的MLE: λ̂ = {sum_x_pois / len(poisson_data):.2f}")

    # 因子分解分量
    factor_comp = FactorizationTheorem.factorization_components_poisson(poisson_data)
    print(f"   h(x) = {factor_comp['h_x']:.2e}")
    print(f"   g(T, λ=4) = {factor_comp['g_function'](4):.2e}")

    # 3. 均匀分布的充分统计量
    print("\n3. 均匀分布 U[0, θ=10]")
    print("-" * 40)

    uniform_data = np.random.uniform(0, 10, 50)
    max_x = Sufficiency.uniform_sufficient(uniform_data)

    print(f"   样本量: {len(uniform_data)}")
    print(f"   充分统计量: T = max(Xᵢ) = {max_x:.2f}")
    print(f"   θ的MLE: θ̂ = {max_x:.2f}")
    print(f"   无偏估计: θ̃ = {(len(uniform_data)+1)/len(uniform_data) * max_x:.2f}")
