# 9.4.4 马尔可夫链蒙特卡洛

## 9.4.4.1 引言

**马尔可夫链蒙特卡洛**（Markov Chain Monte Carlo, MCMC）是一类用于从复杂概率分布采样的算法。
在贝叶斯统计中，MCMC使得后验推断在解析不可解的情况下成为可能。

## 9.4.4.2 马尔可夫链基础

### 9.4.4.2.1 马尔可夫链定义

**定义 9.4.4.1**（马尔可夫链）

随机过程 $\{X_t\}_{t=0}^{\infty}$ 称为**马尔可夫链**，如果满足**马尔可夫性**：

$$P(X_{t+1} = x | X_t, X_{t-1}, \ldots, X_0) = P(X_{t+1} = x | X_t)$$

**转移核**：$P(x, dy) = P(X_{t+1} \in dy | X_t = x)$

### 9.4.4.2.2 遍历性与平稳分布

**定义 9.4.4.2**（平稳分布）

分布 $\pi$ 称为转移核 $P$ 的**平稳分布**，如果：
$$\pi P = \pi$$

即：$\int \pi(x) P(x, dy) dx = \pi(y)$

**定义 9.4.4.3**（遍历性）

马尔可夫链是**遍历的**，如果：

1. **不可约**：从任意状态可到达任意其他状态
2. **非周期**：返回时间的最大公约数为1
3. **正常返**：期望返回时间有限

**定理 9.4.4.1**（遍历定理）

若马尔可夫链遍历且平稳分布为 $\pi$，则：

1. $\lim_{t \to \infty} P(X_t = x) = \pi(x)$（收敛到平稳分布）
2. $\frac{1}{T}\sum_{t=1}^{T} f(X_t) \to \int f(x) \pi(x) dx$（样本平均收敛到期望）

## 9.4.4.3 Metropolis-Hastings算法

### 9.4.4.3.1 算法描述

**算法 9.4.4.2**（Metropolis-Hastings）

目标：从目标分布 $\pi(x)$ 采样

1. 初始化 $X_0$
2. 对于 $t = 0, 1, 2, \ldots$：
   a. 从提议分布 $q(\cdot | X_t)$ 生成候选 $Y$
   b. 计算接受概率：
      $$\alpha(X_t, Y) = \min\left\{1, \frac{\pi(Y)q(X_t | Y)}{\pi(X_t)q(Y | X_t)}\right\}$$
   c. 以概率 $\alpha$ 接受 $X_{t+1} = Y$，否则 $X_{t+1} = X_t$

### 9.4.4.3.2 算法正确性

**定理 9.4.4.3**（细致平衡条件）

若转移核 $P$ 满足**细致平衡**（Detailed Balance）条件：
$$\pi(x) P(x, y) = \pi(y) P(y, x), \quad \forall x, y$$

则 $\pi$ 是平稳分布。

**证明：**

对任意 $y$：
$$\int \pi(x) P(x, y) dx = \int \pi(y) P(y, x) dx = \pi(y) \int P(y, x) dx = \pi(y)$$

**证毕。**

**定理 9.4.4.4**（Metropolis-Hastings的正确性）

MH算法构造的链满足细致平衡条件，因此 $\pi$ 是其平稳分布。

**证明：**

MH转移概率：
$$P(x, y) = q(y | x) \alpha(x, y) + \delta_x(y) r(x)$$

其中 $r(x) = \int q(z | x)(1 - \alpha(x, z)) dz$ 为拒绝概率。

对于 $x \neq y$：
$$\pi(x) P(x, y) = \pi(x) q(y | x) \min\left\{1, \frac{\pi(y)q(x | y)}{\pi(x)q(y | x)}\right\}$$

分两种情况：

- 若 $\pi(y)q(x | y) \leq \pi(x)q(y | x)$：左边 $= \pi(x)q(y|x) \cdot \frac{\pi(y)q(x|y)}{\pi(x)q(y|x)} = \pi(y)q(x|y)$，右边 $= \pi(y)q(x|y) \cdot 1$
- 若 $\pi(y)q(x | y) > \pi(x)q(y | x)$：左边 $= \pi(x)q(y|x)$，右边 $= \pi(y)q(x|y) \cdot \frac{\pi(x)q(y|x)}{\pi(y)q(x|y)} = \pi(x)q(y|x)$

两种情况均满足细致平衡。

**证毕。**

## 9.4.4.4 Gibbs采样

### 9.4.4.4.1 算法描述

**算法 9.4.4.5**（Gibbs采样）

目标：从 $\pi(x_1, \ldots, x_d)$ 采样

1. 初始化 $(X_1^{(0)}, \ldots, X_d^{(0)})$
2. 对于 $t = 0, 1, 2, \ldots$：
   - 从 $\pi(X_1 | X_2^{(t)}, \ldots, X_d^{(t)})$ 采样 $X_1^{(t+1)}$
   - 从 $\pi(X_2 | X_1^{(t+1)}, X_3^{(t)}, \ldots, X_d^{(t)})$ 采样 $X_2^{(t+1)}$
   - $\vdots$
   - 从 $\pi(X_d | X_1^{(t+1)}, \ldots, X_{d-1}^{(t+1)})$ 采样 $X_d^{(t+1)}$

### 9.4.4.4.2 Gibbs是MH的特例

**定理 9.4.4.6**（Gibbs作为MH）

Gibbs采样是Metropolis-Hastings算法的特例，其中提议分布为满条件分布，接受率恒为1。

---

## 9.4.4.5 MCMC诊断

### 9.4.4.5.1 收敛诊断

**定义 9.4.4.7**（收敛）

链在迭代 $T$ 后**收敛**，如果 $X_T$ 的分布足够接近平稳分布 $\pi$。

**常用诊断方法**：

1. **可视化**：迹线图（Trace Plot）检查平稳性
2. **Gelman-Rubin统计量** $\hat{R}$：多链比较，$\hat{R} < 1.1$ 视为收敛
3. **Geweke检验**：比较链的前后部分均值

### 9.4.4.5.2 有效样本量

**定义 9.4.4.8**（有效样本量，ESS）

$$\text{ESS} = \frac{n}{1 + 2\sum_{k=1}^{\infty} \rho(k)}$$

其中 $\rho(k)$ 为滞后 $k$ 的自相关系数。

## 9.4.4.6 代码实现

```python
import numpy as np
from scipy import stats
from scipy.stats import norm, uniform
from typing import Callable, Tuple, List, Optional
import matplotlib.pyplot as plt

class MetropolisHastings:
    """Metropolis-Hastings采样器"""

    def __init__(self, target_logpdf: Callable, proposal_dist: Optional[Callable] = None):
        """
        Args:
            target_logpdf: 目标分布的对数密度函数
            proposal_dist: 提议分布，形式为 (proposal_sample, proposal_logpdf)
                          如果为None，使用随机游走提议
        """
        self.target_logpdf = target_logpdf
        self.proposal = proposal_dist

    def random_walk_proposal(self, current: float, scale: float = 1.0) -> Tuple[float, float, float]:
        """
        对称随机游走提议 N(current, scale²)

        Returns:
            (提议值, 正向对数密度, 反向对数密度)
        """
        proposal = norm.rvs(loc=current, scale=scale)
        log_q_forward = norm.logpdf(proposal, loc=current, scale=scale)
        log_q_backward = norm.logpdf(current, loc=proposal, scale=scale)
        return proposal, log_q_forward, log_q_backward

    def sample(self, x0: float, n_samples: int,
              burn_in: int = 1000, thin: int = 1,
              rw_scale: float = 1.0) -> Tuple[np.ndarray, dict]:
        """
        MCMC采样

        Returns:
            (样本数组, 诊断信息字典)
        """
        samples = []
        x_current = x0
        n_accepted = 0

        for i in range(burn_in + n_samples * thin):
            # 生成提议
            if self.proposal is None:
                x_proposal, log_q_forward, log_q_backward = self.random_walk_proposal(x_current, rw_scale)
            else:
                x_proposal = self.proposal(x_current)
                log_q_forward = 0  # 对称提议
                log_q_backward = 0

            # 计算接受概率
            log_target_current = self.target_logpdf(x_current)
            log_target_proposal = self.target_logpdf(x_proposal)

            log_alpha = min(0, log_target_proposal + log_q_backward - log_target_current - log_q_forward)

            # 接受/拒绝
            if np.log(uniform.rvs()) < log_alpha:
                x_current = x_proposal
                n_accepted += 1

            # 收集样本（Burn-in后，按thin间隔）
            if i >= burn_in and (i - burn_in) % thin == 0:
                samples.append(x_current)

        samples = np.array(samples)

        diagnostics = {
            'acceptance_rate': n_accepted / (burn_in + n_samples * thin),
            'mean': np.mean(samples),
            'std': np.std(samples),
            'effective_sample_size': self._compute_ess(samples)
        }

        return samples, diagnostics

    def _compute_ess(self, samples: np.ndarray, max_lag: int = 100) -> float:
        """计算有效样本量（简化版）"""
        n = len(samples)
        if n < 2:
            return n

        # 计算自相关系数
        x = samples - np.mean(samples)
        autocorr = np.correlate(x, x, mode='full')[n-1:] / np.dot(x, x)
        autocorr = autocorr[:min(max_lag, n//2)]

        # 找到第一个负的自相关系数
        negative_idx = np.where(autocorr < 0)[0]
        if len(negative_idx) > 0:
            autocorr = autocorr[:negative_idx[0]]

        ess = n / (1 + 2 * np.sum(autocorr[1:]))
        return max(1, ess)


class GibbsSampler:
    """Gibbs采样器 - 双变量正态示例"""

    def __init__(self, mu1: float, mu2: float, sigma1: float, sigma2: float, rho: float):
        """
        双变量正态分布 N(μ, Σ)

        其中 Σ = [[σ₁², ρσ₁σ₂], [ρσ₁σ₂, σ₂²]]
        """
        self.mu1 = mu1
        self.mu2 = mu2
        self.sigma1 = sigma1
        self.sigma2 = sigma2
        self.rho = rho

    def sample_conditional_1(self, x2: float) -> float:
        """
        X₁ | X₂ = x₂ 的条件分布

        N(μ₁ + ρ(σ₁/σ₂)(x₂ - μ₂), σ₁²(1 - ρ²))
        """
        mean = self.mu1 + self.rho * (self.sigma1 / self.sigma2) * (x2 - self.mu2)
        var = self.sigma1**2 * (1 - self.rho**2)
        return norm.rvs(loc=mean, scale=np.sqrt(var))

    def sample_conditional_2(self, x1: float) -> float:
        """X₂ | X₁ = x₁ 的条件分布"""
        mean = self.mu2 + self.rho * (self.sigma2 / self.sigma1) * (x1 - self.mu1)
        var = self.sigma2**2 * (1 - self.rho**2)
        return norm.rvs(loc=mean, scale=np.sqrt(var))

    def sample(self, x0: Tuple[float, float], n_samples: int,
              burn_in: int = 1000) -> Tuple[np.ndarray, np.ndarray]:
        """
        Gibbs采样

        Returns:
            (X₁样本, X₂样本)
        """
        x1, x2 = x0
        samples1, samples2 = [], []

        for i in range(burn_in + n_samples):
            # 采样 X₁ | X₂
            x1 = self.sample_conditional_1(x2)
            # 采样 X₂ | X₁
            x2 = self.sample_conditional_2(x1)

            if i >= burn_in:
                samples1.append(x1)
                samples2.append(x2)

        return np.array(samples1), np.array(samples2)


# 使用示例
if __name__ == "__main__":
    print("=" * 60)
    print("MCMC采样示例")
    print("=" * 60)

    np.random.seed(42)

    # 1. Metropolis-Hastings采样标准正态
    print("\n1. Metropolis-Hastings采样标准正态")
    print("-" * 40)

    def standard_normal_logpdf(x):
        return -0.5 * x**2 - 0.5 * np.log(2 * np.pi)

    mh = MetropolisHastings(standard_normal_logpdf)
    samples, diag = mh.sample(x0=0, n_samples=5000, burn_in=1000, rw_scale=2.0)

    print(f"   接受率: {diag['acceptance_rate']:.3f}")
    print(f"   样本均值: {diag['mean']:.4f} (理论: 0)")
    print(f"   样本标准差: {diag['std']:.4f} (理论: 1)")
    print(f"   有效样本量: {diag['effective_sample_size']:.1f}")

    # 2. MH采样双模态分布
    print("\n2. MH采样双模态正态混合")
    print("-" * 40)

    def bimodal_logpdf(x):
        """0.5*N(-3, 1) + 0.5*N(3, 1)"""
        return np.log(0.5 * norm.pdf(x, -3, 1) + 0.5 * norm.pdf(x, 3, 1))

    mh_bimodal = MetropolisHastings(bimodal_logpdf)
    samples_bimodal, diag_bimodal = mh_bimodal.sample(
        x0=0, n_samples=10000, burn_in=2000, rw_scale=1.0
    )

    print(f"   接受率: {diag_bimodal['acceptance_rate']:.3f}")
    print(f"   样本均值: {diag_bimodal['mean']:.4f} (理论: 0)")
    print(f"   有效样本量: {diag_bimodal['effective_sample_size']:.1f}")

    # 检查是否捕获了两个模态
    prop_left = np.mean(samples_bimodal < 0)
    print(f"   左模态比例: {prop_left:.3f} (理论: 0.5)")

    # 3. Gibbs采样双变量正态
    print("\n3. Gibbs采样双变量正态")
    print("-" * 40)

    gibbs = GibbsSampler(mu1=0, mu2=0, sigma1=1, sigma2=1, rho=0.8)
    samples_x1, samples_x2 = gibbs.sample(x0=(0, 0), n_samples=5000, burn_in=1000)

    print(f"   X₁均值: {np.mean(samples_x1):.4f} (理论: 0)")
    print(f"   X₂均值: {np.mean(samples_x2):.4f} (理论: 0)")
    print(f"   样本相关系数: {np.corrcoef(samples_x1, samples_x2)[0,1]:.4f} (理论: 0.8)")
