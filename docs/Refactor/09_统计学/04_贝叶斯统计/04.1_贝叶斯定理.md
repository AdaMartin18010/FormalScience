# 9.4.1 贝叶斯定理

## 9.4.1.1 引言

贝叶斯统计提供了一个与频率学派不同的推断框架。
在贝叶斯框架中，参数被视为随机变量，具有概率分布。
贝叶斯定理是这一方法论的数学基础，它描述了如何根据观测数据更新对参数的信念。

## 9.4.1.2 贝叶斯定理的形式化

### 9.4.1.2.1 离散情形

**定理 9.4.1.1**（贝叶斯定理 - 离散情形）

设 $\{H_1, H_2, \ldots, H_n\}$ 为互斥且穷尽的假设（即样本空间的一个划分），$P(H_i) > 0$。对于观测事件 $D$ 且 $P(D) > 0$：

$$P(H_i | D) = \frac{P(D | H_i) P(H_i)}{P(D)} = \frac{P(D | H_i) P(H_i)}{\sum_{j=1}^{n} P(D | H_j) P(H_j)}$$

其中：

- $P(H_i)$：**先验概率**（Prior）
- $P(D | H_i)$：**似然**（Likelihood）
- $P(H_i | D)$：**后验概率**（Posterior）
- $P(D)$：**证据**（Evidence）或**边际似然**

### 9.4.1.2.2 连续情形

**定理 9.4.1.2**（贝叶斯定理 - 连续参数）

设 $\theta$ 为连续参数，具有先验密度 $\pi(\theta)$，给定 $\theta$ 时数据 $x$ 的似然为 $f(x | \theta)$。则 $\theta$ 的**后验密度**为：

$$\pi(\theta | x) = \frac{f(x | \theta) \pi(\theta)}{m(x)} = \frac{f(x | \theta) \pi(\theta)}{\int f(x | \theta) \pi(\theta) d\theta}$$

其中 $m(x) = \int f(x | \theta) \pi(\theta) d\theta$ 为**边际似然**或**证据**。

**比例形式**（更常用）：

$$\pi(\theta | x) \propto f(x | \theta) \pi(\theta)$$

即：**后验 $\propto$ 似然 $\times$ 先验**

## 9.4.1.3 贝叶斯推断的基本要素

### 9.4.1.3.1 先验分布

**定义 9.4.1.1**（先验分布，Prior Distribution）

**先验分布** $\pi(\theta)$ 表示在观测数据之前对参数 $\theta$ 的信念或知识。

**先验类型**：

- **信息先验**（Informative Prior）：基于先前研究或专家知识
- **无信息先验**（Non-informative Prior）：尽可能少地影响后验
- **共轭先验**（Conjugate Prior）：与似然结合后得到相同族的后验

### 9.4.1.3.2 似然函数

**定义 9.4.1.2**（似然函数，Likelihood）

给定参数 $\theta$ 时观测数据 $x$ 的**似然函数** $f(x | \theta)$，视为 $\theta$ 的函数。

**注**：在贝叶斯框架中，似然与频率学派定义相同，但解释不同。

### 9.4.1.3.3 后验分布

**定义 9.4.1.3**（后验分布，Posterior Distribution）

**后验分布** $\pi(\theta | x)$ 综合了先验信息和数据证据，表示在观测数据后对参数的更新信念。

**定理 9.4.1.3**（后验分布的性质）

若先验 $\pi(\theta)$ 是合法概率密度，似然 $f(x | \theta)$ 满足正则条件，则后验 $\pi(\theta | x)$ 也是合法概率密度。

**证明：**

非负性显然。验证归一化：

$$\int \pi(\theta | x) d\theta = \int \frac{f(x | \theta) \pi(\theta)}{m(x)} d\theta = \frac{m(x)}{m(x)} = 1$$

**证毕。**

### 9.4.1.3.4 边际似然

**定义 9.4.1.4**（边际似然，Marginal Likelihood）

$$m(x) = \int_{\Theta} f(x | \theta) \pi(\theta) d\theta$$

边际似然也称为**证据**，在模型比较中起关键作用（见第 9.4.4 章）。

## 9.4.1.4 条件概率的形式化

### 9.4.1.4.1 条件概率的定义回顾

**定义 9.4.1.5**（条件概率）

设 $(\Omega, \mathcal{F}, P)$ 为概率空间，$A, B \in \mathcal{F}$，$P(B) > 0$：

$$P(A | B) = \frac{P(A \cap B)}{P(B)}$$

### 9.4.1.4.2 乘法公式

**定理 9.4.1.4**（乘法公式）

$$P(A \cap B) = P(A | B) P(B) = P(B | A) P(A)$$

对于多个事件：
$$P(A_1 \cap A_2 \cap \cdots \cap A_n) = P(A_1) P(A_2 | A_1) P(A_3 | A_1 \cap A_2) \cdots P(A_n | A_1 \cap \cdots \cap A_{n-1})$$

---

## 9.4.1.5 贝叶斯定理的推导

**证明 9.4.1.5**（贝叶斯定理的完整推导）

由条件概率定义：
$$P(H_i | D) = \frac{P(D \cap H_i)}{P(D)}$$

又 $P(D \cap H_i) = P(D | H_i) P(H_i)$，代入得：
$$P(H_i | D) = \frac{P(D | H_i) P(H_i)}{P(D)}$$

由全概率公式：
$$P(D) = \sum_{j=1}^{n} P(D | H_j) P(H_j)$$

故：
$$P(H_i | D) = \frac{P(D | H_i) P(H_i)}{\sum_{j=1}^{n} P(D | H_j) P(H_j)}$$

**证毕。**

## 9.4.1.6 示例：硬币偏差推断

**示例 9.4.1.1**（伯努利试验的贝叶斯推断）

设投掷硬币 $n$ 次，观测到 $k$ 次正面。推断正面概率 $\theta$。

**模型**：

- 似然：$f(k | \theta) = \binom{n}{k} \theta^k (1-\theta)^{n-k}$
- 先验：$\pi(\theta) = 1$（Uniform(0,1)，无信息）

**后验**：
$$\pi(\theta | k) \propto \theta^k (1-\theta)^{n-k} \cdot 1 = \theta^k (1-\theta)^{n-k}$$

归一化常数：
$$m(k) = \int_0^1 \theta^k (1-\theta)^{n-k} d\theta = B(k+1, n-k+1) = \frac{k!(n-k)!}{(n+1)!}$$

因此：
$$\pi(\theta | k) = \frac{(n+1)!}{k!(n-k)!} \theta^k (1-\theta)^{n-k} = \text{Beta}(k+1, n-k+1)$$

**后验均值**：$E[\theta | k] = \frac{k+1}{n+2}$

## 9.4.1.7 代码实现

```python
import numpy as np
from scipy import stats
from scipy.integrate import quad
from scipy.special import beta as beta_func
import matplotlib.pyplot as plt
from typing import Callable, Tuple, Dict

class BayesianInference:
    """贝叶斯推断基础"""

    def __init__(self, likelihood_func: Callable, prior_func: Callable):
        """
        Args:
            likelihood_func: 似然函数 f(x|theta)
            prior_func: 先验函数 pi(theta)
        """
        self.likelihood = likelihood_func
        self.prior = prior_func

    def compute_posterior(self, data: np.ndarray, theta_range: Tuple[float, float],
                         n_points: int = 1000) -> Tuple[np.ndarray, np.ndarray, float]:
        """
        计算后验分布（数值方法）

        Returns:
            theta_grid, posterior_values, marginal_likelihood
        """
        theta_grid = np.linspace(theta_range[0], theta_range[1], n_points)

        # 计算未归一化后验
        unnormalized_posterior = []
        for theta in theta_grid:
            likelihood_val = self.likelihood(data, theta)
            prior_val = self.prior(theta)
            unnormalized_posterior.append(likelihood_val * prior_val)

        unnormalized_posterior = np.array(unnormalized_posterior)

        # 数值积分计算边际似然
        marginal_likelihood = np.trapz(unnormalized_posterior, theta_grid)

        # 归一化
        posterior = unnormalized_posterior / marginal_likelihood

        return theta_grid, posterior, marginal_likelihood

    def posterior_mean_numerical(self, data: np.ndarray, theta_range: Tuple[float, float]) -> float:
        """后验均值（数值积分）"""
        theta_grid, posterior, _ = self.compute_posterior(data, theta_range)
        return np.trapz(theta_grid * posterior, theta_grid)

    def posterior_ci(self, data: np.ndarray, theta_range: Tuple[float, float],
                    alpha: float = 0.05) -> Tuple[float, float]:
        """后验可信区间（数值方法，等尾区间）"""
        theta_grid, posterior, _ = self.compute_posterior(data, theta_range)

        # 累积分布
        cdf = np.cumsum(posterior) / np.sum(posterior)

        # 找到分位数
        lower_idx = np.searchsorted(cdf, alpha/2)
        upper_idx = np.searchsorted(cdf, 1 - alpha/2)

        return theta_grid[lower_idx], theta_grid[upper_idx]


class BetaBernoulliConjugate:
    """
    Beta-Bernoulli共轭模型
    先验: Beta(α, β)
    似然: Bernoulli(θ)
    后验: Beta(α + k, β + n - k)
    """

    def __init__(self, alpha: float = 1, beta: float = 1):
        """
        Args:
            alpha, beta: Beta先验参数
        """
        self.alpha_prior = alpha
        self.beta_prior = beta

    def update(self, data: np.ndarray) -> Tuple[float, float]:
        """
        根据数据更新后验参数

        Returns:
            (alpha_posterior, beta_posterior)
        """
        n = len(data)
        k = np.sum(data)  # 成功次数

        alpha_post = self.alpha_prior + k
        beta_post = self.beta_prior + n - k

        return alpha_post, beta_post

    def posterior_stats(self, data: np.ndarray) -> Dict:
        """后验分布统计量"""
        alpha_post, beta_post = self.update(data)

        # Beta分布统计量
        mean = alpha_post / (alpha_post + beta_post)
        var = (alpha_post * beta_post) / ((alpha_post + beta_post)**2 * (alpha_post + beta_post + 1))
        mode = (alpha_post - 1) / (alpha_post + beta_post - 2) if alpha_post > 1 and beta_post > 1 else None

        # 95%等尾可信区间
        ci_lower = stats.beta.ppf(0.025, alpha_post, beta_post)
        ci_upper = stats.beta.ppf(0.975, alpha_post, beta_post)

        return {
            'alpha_posterior': alpha_post,
            'beta_posterior': beta_post,
            'posterior_mean': mean,
            'posterior_variance': var,
            'posterior_mode': mode,
            'posterior_median': stats.beta.median(alpha_post, beta_post),
            'ci_95': (ci_lower, ci_upper)
        }


class NormalNormalConjugate:
    """
    正态-正态共轭模型（已知方差）
    先验: N(μ₀, τ²)
    似然: N(θ, σ²)  # σ²已知
    后验: N(μₙ, τₙ²)
    """

    def __init__(self, mu0: float, tau_sq: float, sigma_sq: float):
        """
        Args:
            mu0: 先验均值
            tau_sq: 先验方差
            sigma_sq: 已知似然方差
        """
        self.mu0 = mu0
        self.tau_sq = tau_sq
        self.sigma_sq = sigma_sq

    def update(self, data: np.ndarray) -> Dict:
        """
        更新后验参数

        Returns:
            后验参数字典
        """
        n = len(data)
        x_bar = np.mean(data)

        # 后验精度 = 先验精度 + 数据精度
        precision_post = 1/self.tau_sq + n/self.sigma_sq
        tau_n_sq = 1 / precision_post

        # 后验均值（精度加权平均）
        mu_n = tau_n_sq * (self.mu0/self.tau_sq + n*x_bar/self.sigma_sq)

        return {
            'mu_posterior': mu_n,
            'tau_sq_posterior': tau_n_sq,
            'sigma_posterior': np.sqrt(tau_n_sq)
        }

    def posterior_predictive(self, x_new: float, data: np.ndarray) -> float:
        """
        后验预测分布密度（单点）

        后验预测: N(μₙ, σ² + τₙ²)
        """
        post_params = self.update(data)
        pred_mean = post_params['mu_posterior']
        pred_var = self.sigma_sq + post_params['tau_sq_posterior']

        return stats.norm.pdf(x_new, pred_mean, np.sqrt(pred_var))
