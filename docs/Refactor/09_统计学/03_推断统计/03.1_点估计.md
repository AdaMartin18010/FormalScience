## 9.3.1.1 引言

**点估计**（Point Estimation）是统计推断的核心问题之一：基于样本数据估计未知总体参数。
本章形式化定义点估计量，介绍极大似然估计（MLE）和矩估计（MOM）两大经典方法，并讨论估计量的优良性质。

## 9.3.1.2 基本概念

### 9.3.1.2.1 参数空间与统计量

**定义 9.3.1.1**（参数族，Parametric Family）

设 $\mathcal{P} = \{P_\theta : \theta \in \Theta\}$ 为概率测度族，$\Theta \subseteq \mathbb{R}^k$ 称为**参数空间**。

**定义 9.3.1.2**（点估计量，Point Estimator）

设 $\mathbf{X} = (X_1, \ldots, X_n)$ 为来自 $P_\theta$ 的样本，**点估计量**是统计量 $T: \mathcal{X}^n \to \Theta$ 的像，记作 $\hat{\theta} = T(\mathbf{X})$。

**注**：估计量是随机变量，估计值是观测数据后的具体数值。

### 9.3.1.2.2 估计量的偏差与方差

**定义 9.3.1.3**（偏差，Bias）

估计量 $\hat{\theta}$ 的**偏差**：

$$\text{Bias}(\hat{\theta}) = E_\theta[\hat{\theta}] - \theta$$

- **无偏估计**：$\text{Bias}(\hat{\theta}) = 0$，即 $E_\theta[\hat{\theta}] = \theta$，$\forall \theta \in \Theta$
- **渐近无偏**：$\lim_{n \to \infty} E_\theta[\hat{\theta}] = \theta$

**定义 9.3.1.4**（标准误，Standard Error）

估计量 $\hat{\theta}$ 的**标准误**：

$$\text{SE}(\hat{\theta}) = \sqrt{\text{Var}(\hat{\theta})}$$

**定义 9.3.1.5**（均方误差，Mean Squared Error）

$$\text{MSE}(\hat{\theta}) = E_\theta[(\hat{\theta} - \theta)^2] = \text{Var}(\hat{\theta}) + \text{Bias}(\hat{\theta})^2$$

## 9.3.1.3 极大似然估计

### 9.3.1.3.1 似然函数

**定义 9.3.1.6**（似然函数，Likelihood Function）

设 $\mathbf{x} = (x_1, \ldots, x_n)$ 为来自 $P_\theta$ 的观测样本，**似然函数**为：

$$L(\theta | \mathbf{x}) = p(\mathbf{x} | \theta) = \prod_{i=1}^{n} p(x_i | \theta)$$

其中 $p(\cdot | \theta)$ 为概率质量/密度函数。

**对数似然函数**：

$$\ell(\theta | \mathbf{x}) = \ln L(\theta | \mathbf{x}) = \sum_{i=1}^{n} \ln p(x_i | \theta)$$

**定义 9.3.1.7**（极大似然估计，MLE）

**极大似然估计量**：

$$\hat{\theta}_{MLE} = \arg\max_{\theta \in \Theta} L(\theta | \mathbf{X}) = \arg\max_{\theta \in \Theta} \ell(\theta | \mathbf{X})$$

### 9.3.1.3.2 MLE的求解

**定理 9.3.1.1**（MLE的求解条件）

若 $\ell(\theta)$ 可微且 $\Theta$ 开，则MLE满足**似然方程**：

$$\frac{\partial \ell(\theta)}{\partial \theta_j} = 0, \quad j = 1, \ldots, k$$

需验证二阶条件确保是最大值。

**定义 9.3.1.8**（Fisher信息矩阵）

$$I(\theta) = -E_\theta\left[\frac{\partial^2 \ell(\theta)}{\partial \theta \partial \theta^T}\right] = E_\theta\left[\frac{\partial \ell(\theta)}{\partial \theta} \frac{\partial \ell(\theta)}{\partial \theta^T}\right]$$

**定理 9.3.1.2**（MLE的渐近正态性）

在正则条件下，MLE是：

1. **相合的**：$\hat{\theta}_{MLE} \stackrel{P}{\to} \theta_0$
2. **渐近正态的**：$\sqrt{n}(\hat{\theta}_{MLE} - \theta_0) \stackrel{d}{\to} N(0, I(\theta_0)^{-1})$
3. **渐近有效的**：达到Cramér-Rao下界

### 9.3.1.3.3 常见分布的MLE

**示例 9.3.1.1**（正态分布参数的MLE）

设 $X_1, \ldots, X_n \stackrel{iid}{\sim} N(\mu, \sigma^2)$。

对数似然：
$$\ell(\mu, \sigma^2) = -\frac{n}{2}\ln(2\pi) - \frac{n}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i - \mu)^2$$

求偏导并令为零：

$$\frac{\partial \ell}{\partial \mu} = \frac{1}{\sigma^2}\sum_{i=1}^{n}(x_i - \mu) = 0 \Rightarrow \hat{\mu} = \bar{X}$$

$$\frac{\partial \ell}{\partial \sigma^2} = -\frac{n}{2\sigma^2} + \frac{1}{2\sigma^4}\sum_{i=1}^{n}(x_i - \mu)^2 = 0 \Rightarrow \hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^{n}(X_i - \bar{X})^2$$

**注**：$\hat{\sigma}^2$ 是有偏的，$E[\hat{\sigma}^2] = \frac{n-1}{n}\sigma^2$。

**示例 9.3.1.2**（指数分布参数的MLE）

设 $X_1, \ldots, X_n \stackrel{iid}{\sim} \text{Exp}(\lambda)$。

$$\ell(\lambda) = n\ln\lambda - \lambda\sum_{i=1}^{n}X_i$$

$$\frac{d\ell}{d\lambda} = \frac{n}{\lambda} - \sum_{i=1}^{n}X_i = 0 \Rightarrow \hat{\lambda} = \frac{n}{\sum_{i=1}^{n}X_i} = \frac{1}{\bar{X}}$$

## 9.3.1.4 矩估计

### 9.3.1.4.1 矩方法

**定义 9.3.1.9**（理论矩与样本矩）

第 $k$ **理论矩**：$\mu_k(\theta) = E_\theta[X^k]$

第 $k$ **样本矩**：$M_k = \frac{1}{n}\sum_{i=1}^{n}X_i^k$

**定义 9.3.1.10**（矩估计，Method of Moments）

设参数 $\theta \in \mathbb{R}^k$，矩估计量 $\hat{\theta}_{MOM}$ 通过解方程组获得：

$$\mu_j(\theta) = M_j, \quad j = 1, \ldots, k$$

### 9.3.1.4.2 矩估计的性质

**定理 9.3.1.3**（矩估计的相合性）

若 $g$ 连续，$M_j \stackrel{P}{\to} \mu_j$，则 $\hat{\theta}_{MOM} \stackrel{P}{\to} \theta$。

**证明：** 由大数定律和连续映射定理。

**示例 9.3.1.3**（伽马分布的矩估计）

设 $X \sim \text{Gamma}(\alpha, \beta)$，$E[X] = \alpha/\beta$，$\text{Var}(X) = \alpha/\beta^2$。

方程组：
$$\bar{X} = \frac{\alpha}{\beta}, \quad \frac{1}{n}\sum_{i=1}^{n}X_i^2 = \frac{\alpha}{\beta^2} + \left(\frac{\alpha}{\beta}\right)^2$$

解得：
$$\hat{\beta} = \frac{\bar{X}}{\frac{1}{n}\sum X_i^2 - \bar{X}^2}, \quad \hat{\alpha} = \bar{X}\hat{\beta}$$

## 9.3.1.5 估计量的优良性质

### 9.3.1.5.1 无偏性与UMVUE

**定义 9.3.1.11**（一致最小方差无偏估计，UMVUE）

估计量 $\hat{\theta}^*$ 称为**UMVUE**，如果：

1. $E_\theta[\hat{\theta}^*] = \theta$（无偏）
2. 对任意其他无偏估计量 $\tilde{\theta}$，$\text{Var}(\hat{\theta}^*) \leq \text{Var}(\tilde{\theta})$，$\forall \theta \in \Theta$

**定理 9.3.1.4**（Cramér-Rao下界）

在正则条件下，任何无偏估计量 $\hat{\theta}$ 满足：

$$\text{Var}(\hat{\theta}) \geq \frac{1}{nI(\theta)}$$

其中 $I(\theta)$ 是Fisher信息量。

### 9.3.1.5.2 相合性

**定义 9.3.1.12**（弱相合与强相合）

- **弱相合**：$\hat{\theta}_n \stackrel{P}{\to} \theta$
- **强相合**：$\hat{\theta}_n \stackrel{a.s.}{\to} \theta$

**定理 9.3.1.5**（MSE相合性）

若 $\lim_{n \to \infty} \text{MSE}(\hat{\theta}_n) = 0$，则 $\hat{\theta}_n$ 弱相合。

## 9.3.1.6 代码实现

```python
import numpy as np
from scipy import stats
from scipy.optimize import minimize
from scipy.special import digamma, polygamma
from typing import Callable, Tuple, Dict, Optional
import warnings

class PointEstimation:
    """点估计：MLE和矩估计"""

    def __init__(self, data: np.ndarray):
        self.data = np.asarray(data)
        self.n = len(data)

    # ========== 极大似然估计 ==========

    def normal_mle(self) -> Dict:
        """正态分布参数的MLE"""
        mu_mle = np.mean(self.data)
        sigma2_mle = np.mean((self.data - mu_mle) ** 2)  # 有偏估计
        sigma2_unbiased = np.var(self.data, ddof=1)  # 无偏估计

        # 对数似然值
        log_likelihood = -0.5 * self.n * np.log(2 * np.pi * sigma2_mle) - self.n / 2

        return {
            'mu_mle': mu_mle,
            'sigma2_mle': sigma2_mle,
            'sigma2_unbiased': sigma2_unbiased,
            'log_likelihood': log_likelihood,
            'bias_sigma2': -sigma2_mle / self.n  # 偏差 = -σ²/n
        }

    def exponential_mle(self) -> Dict:
        """指数分布参数的MLE"""
        lambda_mle = 1 / np.mean(self.data)
        log_likelihood = self.n * np.log(lambda_mle) - lambda_mle * np.sum(self.data)

        return {
            'lambda_mle': lambda_mle,
            'log_likelihood': log_likelihood
        }

    def bernoulli_mle(self) -> Dict:
        """伯努利分布参数的MLE"""
        p_mle = np.mean(self.data)
        log_likelihood = np.sum(self.data) * np.log(p_mle) + \
                        (self.n - np.sum(self.data)) * np.log(1 - p_mle)

        return {
            'p_mle': p_mle,
            'log_likelihood': log_likelihood
        }

    def poisson_mle(self) -> Dict:
        """泊松分布参数的MLE"""
        lambda_mle = np.mean(self.data)
        log_likelihood = -self.n * lambda_mle + np.sum(self.data) * np.log(lambda_mle) - \
                        np.sum([np.log(np.math.factorial(int(x))) for x in self.data])

        return {
            'lambda_mle': lambda_mle,
            'log_likelihood': log_likelihood
        }

    def gamma_mle(self, method: str = 'numerical') -> Dict:
        """伽马分布参数的MLE"""
        x_bar = np.mean(self.data)
        log_x_bar = np.mean(np.log(self.data))

        if method == 'numerical':
            # 数值优化
            def neg_log_likelihood(params):
                alpha, beta = params
                if alpha <= 0 or beta <= 0:
                    return np.inf
                return -(self.n * alpha * np.log(beta) - self.n * np.log(stats.gamma(alpha).pdf(1)) +
                        (alpha - 1) * np.sum(np.log(self.data)) - beta * np.sum(self.data))

            # 矩估计作为初值
            alpha_mom = x_bar ** 2 / np.var(self.data)
            beta_mom = x_bar / np.var(self.data)

            result = minimize(neg_log_likelihood, [alpha_mom, beta_mom],
                            method='L-BFGS-B', bounds=[(0.01, None), (0.01, None)])

            alpha_mle, beta_mle = result.x

        else:
            # 使用Digamma函数的近似解
            # 解方程: ln(α) - digamma(α) = ln(x̄) - ln(x)̄
            s = np.log(x_bar) - log_x_bar

            # 近似公式
            alpha_mle = (3 - s + np.sqrt((s - 3)**2 + 24 * s)) / (12 * s)
            beta_mle = alpha_mle / x_bar

        return {
            'alpha_mle': alpha_mle,
            'beta_mle': beta_mle,
            'mean_estimated': alpha_mle / beta_mle
        }

    # ========== 矩估计 ==========

    def method_of_moments(self, k: int = 2) -> Dict:
        """
        通用矩估计

        Args:
            k: 矩的阶数
        """
        moments = {}
        for i in range(1, k + 1):
            moments[f'moment_{i}'] = np.mean(self.data ** i)

        # 方差（使用二阶中心矩）
        moments['variance'] = moments['moment_2'] - moments['moment_1'] ** 2

        return moments

    def gamma_mom(self) -> Dict:
        """伽马分布的矩估计"""
        x_bar = np.mean(self.data)
        s2 = np.var(self.data, ddof=0)  # 使用n而非n-1

        beta_mom = x_bar / s2
        alpha_mom = x_bar * beta_mom

        return {
            'alpha_mom': alpha_mom,
            'beta_mom': beta_mom,
            'mean': x_bar,
            'variance': s2
        }

    def beta_mom(self) -> Dict:
        """贝塔分布的矩估计"""
        x_bar = np.mean(self.data)
        s2 = np.var(self.data, ddof=0)

        # 样本均值和方差估计参数
        if s2 >= x_bar * (1 - x_bar):
            warnings.warn("样本方差过大，边界情况")
            s2 = x_bar * (1 - x_bar) * 0.99

        common_factor = x_bar * (1 - x_bar) / s2 - 1
        alpha_mom = x_bar * common_factor
        beta_mom = (1 - x_bar) * common_factor

        return {
            'alpha_mom': alpha_mom,
            'beta_mom': beta_mom
        }

    # ========== 估计量评价 ==========

    def evaluate_estimator(self, true_value: float,
                          estimator_func: Callable,
                          n_simulations: int = 1000,
                          sample_size: Optional[int] = None) -> Dict:
        """
        通过模拟评价估计量
        """
        if sample_size is None:
            sample_size = self.n

        estimates = []
        for _ in range(n_simulations):
            # 假设数据生成机制已知（从原始数据bootstrap）
            bootstrap_sample = np.random.choice(self.data, size=sample_size, replace=True)
            est = estimator_func(bootstrap_sample)
            estimates.append(est)

        estimates = np.array(estimates)

        return {
            'bias': np.mean(estimates) - true_value,
            'variance': np.var(estimates, ddof=1),
            'mse': np.mean((estimates - true_value) ** 2),
            'std_error': np.std(estimates, ddof=1),
            'coverage_95': np.mean((estimates >= true_value - 1.96 * np.std(estimates)) &
                                   (estimates <= true_value + 1.96 * np.std(estimates)))
        }


# 使用示例
if __name__ == "__main__":
    print("=" * 60)
    print("点估计示例")
    print("=" * 60)

    np.random.seed(42)

    # 1. 正态分布
    print("\n1. 正态分布 N(μ=10, σ²=25)")
    print("-" * 40)
    normal_data = np.random.normal(10, 5, 100)
    pe_normal = PointEstimation(normal_data)
    result = pe_normal.normal_mle()
    print(f"   μ_MLE = {result['mu_mle']:.4f}")
    print(f"   σ²_MLE = {result['sigma2_mle']:.4f} (有偏)")
    print(f"   σ²_无偏 = {result['sigma2_unbiased']:.4f}")
    print(f"   σ²偏差 = {result['bias_sigma2']:.4f}")

    # 2. 泊松分布
    print("\n2. 泊松分布 Poisson(λ=4)")
    print("-" * 40)
    poisson_data = np.random.poisson(4, 200)
    pe_pois = PointEstimation(poisson_data)
    result = pe_pois.poisson_mle()
    print(f"   λ_MLE = {result['lambda_mle']:.4f}")
    print(f"   理论值 = 4.0")

    # 3. 指数分布
    print("\n3. 指数分布 Exp(λ=2)")
    print("-" * 40)
    exp_data = np.random.exponential(1/2, 100)
    pe_exp = PointEstimation(exp_data)
    result = pe_exp.exponential_mle()
    print(f"   λ_MLE = {result['lambda_mle']:.4f}")
    print(f"   理论值 = 2.0")

    # 4. 伽马分布 - MLE vs MOM
    print("\n4. 伽马分布 Gamma(α=2, β=3) - MLE vs MOM 比较")
    print("-" * 40)
    gamma_data = np.random.gamma(2, 1/3, 500)
    pe_gamma = PointEstimation(gamma_data)

    mle_result = pe_gamma.gamma_mle()
    mom_result = pe_gamma.gamma_mom()

    print(f"   MLE: α̂ = {mle_result['alpha_mle']:.4f}, β̂ = {mle_result['beta_mle']:.4f}")
    print(f"   MOM: α̃ = {mom_result['alpha_mom']:.4f}, β̃ = {mom_result['beta_mom']:.4f}")
    print(f"   理论值: α = 2.0, β = 3.0")
    print(f"   MLE估计均值: {mle_result['mean_estimated']:.4f} (理论: 0.667)")
```
