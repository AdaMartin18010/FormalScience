## 9.2.1.2 样本空间

**定义 9.2.1.1**（样本空间，Sample Space）

一个**随机试验**（random experiment）的**样本空间** $\Omega$ 是所有可能结果的集合。样本空间的元素 $\omega \in \Omega$ 称为**样本点**或**基本事件**。

**示例 9.2.1.1**（样本空间示例）

| 试验 | 样本空间 $\Omega$ | 说明 |
|-----|------------------|------|
| 抛硬币 | $\{H, T\}$ | $H$=正面，$T$=反面 |
| 掷骰子 | $\{1, 2, 3, 4, 5, 6\}$ | 点数 |
| 等待时间 | $[0, \infty)$ | 连续时间 |
| 股票收益 | $\mathbb{R}$ | 实数值 |
| 序列试验 | $\Omega^{\mathbb{N}}$ | 无限序列 |

## 9.2.1.3 事件与σ-代数

### 9.2.1.3.1 事件的基本概念

**定义 9.2.1.2**（事件，Event）

**事件**是样本空间 $\Omega$ 的子集，表示某些可能结果的集合。若试验结果 $\omega \in A$，则称事件 $A$ **发生**。

**事件运算**：

- **并** $A \cup B$：$A$ 或 $B$ 发生
- **交** $A \cap B$：$A$ 和 $B$ 都发生
- **补** $A^c = \Omega \setminus A$：$A$ 不发生
- **差** $A \setminus B = A \cap B^c$：$A$ 发生但 $B$ 不发生

### 9.2.1.3.2 σ-代数的形式化定义

**定义 9.2.1.3**（$\sigma$-代数，Sigma-Algebra）

设 $\Omega$ 为样本空间，$\mathcal{F} \subseteq 2^{\Omega}$ 为 $\Omega$ 的子集族。$\mathcal{F}$ 称为**$\sigma$-代数**（或 $\sigma$-域），如果满足：

1. **非空**：$\Omega \in \mathcal{F}$
2. **对补封闭**：若 $A \in \mathcal{F}$，则 $A^c \in \mathcal{F}$
3. **对可数并封闭**：若 $\{A_i\}_{i=1}^{\infty} \subseteq \mathcal{F}$，则 $\bigcup_{i=1}^{\infty} A_i \in \mathcal{F}$

**定义 9.2.1.4**（可测空间，Measurable Space）

二元组 $(\Omega, \mathcal{F})$ 称为**可测空间**，$\mathcal{F}$ 中的元素称为**可测集**或**事件**。

**定理 9.2.1.1**（$\sigma$-代数的基本性质）

设 $\mathcal{F}$ 为 $\Omega$ 上的 $\sigma$-代数，则：

1. $\emptyset \in \mathcal{F}$
2. 若 $\{A_i\}_{i=1}^{\infty} \subseteq \mathcal{F}$，则 $\bigcap_{i=1}^{\infty} A_i \in \mathcal{F}$
3. 若 $A, B \in \mathcal{F}$，则 $A \setminus B \in \mathcal{F}$
4. 若 $\{A_i\}_{i=1}^{n} \subseteq \mathcal{F}$（有限），则 $\bigcup_{i=1}^{n} A_i \in \mathcal{F}$，$\bigcap_{i=1}^{n} A_i \in \mathcal{F}$

**证明：**

(1) 由 $\Omega \in \mathcal{F}$ 和对补封闭，$\emptyset = \Omega^c \in \mathcal{F}$。

(2) 由德摩根律：$\bigcap_{i=1}^{\infty} A_i = \left(\bigcup_{i=1}^{\infty} A_i^c\right)^c$。每个 $A_i^c \in \mathcal{F}$，可数并 $\in \mathcal{F}$，再取补 $\in \mathcal{F}$。

(3) $A \setminus B = A \cap B^c$，由 $B^c \in \mathcal{F}$ 和有限交封闭（由(2)）得证。

(4) 对有限集，可补充 $\emptyset$ 使之成为可数集，应用可数并/交封闭性。

**证毕。**

### 9.2.1.3.3 生成σ-代数与Borel σ-代数

**定义 9.2.1.5**（生成σ-代数，Generated σ-Algebra）

设 $\mathcal{C} \subseteq 2^{\Omega}$，包含 $\mathcal{C}$ 的最小 $\sigma$-代数称为由 $\mathcal{C}$ **生成**的 $\sigma$-代数，记作 $\sigma(\mathcal{C})$。

**定理 9.2.1.2**（生成σ-代数的存在唯一性）

对于任意 $\mathcal{C} \subseteq 2^{\Omega}$，存在唯一的包含 $\mathcal{C}$ 的最小 $\sigma$-代数。

**证明：**

令 $\mathcal{G} = \{\mathcal{F} : \mathcal{F} \text{ 是 } \sigma\text{-代数且 } \mathcal{C} \subseteq \mathcal{F}\}$。

- $\mathcal{G} \neq \emptyset$，因为 $2^{\Omega} \in \mathcal{G}$。
- 令 $\sigma(\mathcal{C}) = \bigcap_{\mathcal{F} \in \mathcal{G}} \mathcal{F}$。

验证 $\sigma(\mathcal{C})$ 是 $\sigma$-代数：

1. $\Omega \in$ 每个 $\mathcal{F} \in \mathcal{G}$，故 $\Omega \in \sigma(\mathcal{C})$。
2. 若 $A \in \sigma(\mathcal{C})$，则 $A \in$ 每个 $\mathcal{F}$，故 $A^c \in$ 每个 $\mathcal{F}$，$A^c \in \sigma(\mathcal{C})$。
3. 可数并同理。

最小性：任何包含 $\mathcal{C}$ 的 $\sigma$-代数都在 $\mathcal{G}$ 中，故包含 $\sigma(\mathcal{C})$。

**证毕。**

**定义 9.2.1.6**（Borel σ-代数，Borel σ-Algebra）

在 $\mathbb{R}$ 上，由所有开区间生成的 $\sigma$-代数称为**Borel $\sigma$-代数**，记作 $\mathcal{B}(\mathbb{R})$。

$\mathcal{B}(\mathbb{R})$ 也可由以下任一类集合生成：

- 闭区间 $[a, b]$
- 半开区间 $(a, b]$ 或 $[a, b)$
- 开射线 $(-\infty, a)$ 或 $(a, \infty)$

## 9.2.1.4 Kolmogorov公理

### 9.2.1.4.1 概率测度的定义

**定义 9.2.1.7**（概率测度，Probability Measure）

设 $(\Omega, \mathcal{F})$ 为可测空间，函数 $P: \mathcal{F} \to [0, 1]$ 称为**概率测度**，如果满足**Kolmogorov公理**：

**公理1**（非负性）：$\forall A \in \mathcal{F}, \quad P(A) \geq 0$

**公理2**（规范性）：$P(\Omega) = 1$

**公理3**（可列可加性）：对于两两不交的事件序列 $\{A_i\}_{i=1}^{\infty} \subseteq \mathcal{F}$（即 $i \neq j \Rightarrow A_i \cap A_j = \emptyset$）：

$$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$$

**定义 9.2.1.8**（概率空间，Probability Space）

三元组 $(\Omega, \mathcal{F}, P)$ 称为**概率空间**。

### 9.2.1.4.2 概率测度的基本性质

**定理 9.2.1.3**（概率的基本性质）

设 $(\Omega, \mathcal{F}, P)$ 为概率空间，$A, B \in \mathcal{F}$：

1. $P(\emptyset) = 0$
2. **有限可加性**：若 $A_1, \ldots, A_n$ 两两不交，则 $P(\bigcup_{i=1}^{n} A_i) = \sum_{i=1}^{n} P(A_i)$
3. **补事件**：$P(A^c) = 1 - P(A)$
4. **单调性**：若 $A \subseteq B$，则 $P(A) \leq P(B)$
5. **有界性**：$0 \leq P(A) \leq 1$
6. **加法公式**：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$

**证明：**

(1) 令 $A_1 = \Omega, A_2 = A_3 = \cdots = \emptyset$，由可列可加性：
$$P(\Omega) = P(\Omega) + \sum_{i=2}^{\infty} P(\emptyset)$$
故 $\sum_{i=2}^{\infty} P(\emptyset) = 0$，由非负性 $P(\emptyset) = 0$。

(2) 补充 $A_{n+1} = A_{n+2} = \cdots = \emptyset$，应用可列可加性。

(3) $A \cup A^c = \Omega$ 不交，故 $P(A) + P(A^c) = 1$。

(4) $B = A \cup (B \setminus A)$ 不交，$P(B) = P(A) + P(B \setminus A) \geq P(A)$。

(5) 由单调性和 $P(\emptyset) = 0 \leq P(A) \leq P(\Omega) = 1$。

(6) $A \cup B = A \cup (B \setminus A)$，$B = (B \setminus A) \cup (A \cap B)$ 均不交。
$$P(A \cup B) = P(A) + P(B \setminus A) = P(A) + P(B) - P(A \cap B)$$

**证毕。**

**定理 9.2.1.4**（连续性定理）

设 $\{A_n\}_{n=1}^{\infty} \subseteq \mathcal{F}$：

1. **下连续性**：若 $A_n \uparrow A$（$A_1 \subseteq A_2 \subseteq \cdots$，$A = \bigcup_n A_n$），则 $P(A_n) \uparrow P(A)$
2. **上连续性**：若 $A_n \downarrow A$（$A_1 \supseteq A_2 \supseteq \cdots$，$A = \bigcap_n A_n$），则 $P(A_n) \downarrow P(A)$

**证明：**

(1) 令 $B_1 = A_1, B_n = A_n \setminus A_{n-1}$（$n \geq 2$），则 $\{B_n\}$ 不交且 $\bigcup_{n=1}^{\infty} B_n = A$。

$$P(A) = \sum_{n=1}^{\infty} P(B_n) = \lim_{N \to \infty} \sum_{n=1}^{N} P(B_n) = \lim_{N \to \infty} P(A_N)$$

(2) $A_n \downarrow A$ 则 $A_n^c \uparrow A^c$，由(1)：
$$P(A_n) = 1 - P(A_n^c) \to 1 - P(A^c) = P(A)$$

**证毕。**

**定理 9.2.1.5**（Boole不等式 / Union Bound）

对于任意事件序列 $\{A_i\}_{i=1}^{\infty}$：

$$P\left(\bigcup_{i=1}^{\infty} A_i\right) \leq \sum_{i=1}^{\infty} P(A_i)$$

**证明：**

构造不交序列 $B_1 = A_1, B_n = A_n \setminus \bigcup_{i=1}^{n-1} A_i$。则 $B_n \subseteq A_n$，故 $P(B_n) \leq P(A_n)$。

且 $\bigcup_{i=1}^{\infty} A_i = \bigcup_{i=1}^{\infty} B_i$（不交）。

$$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(B_i) \leq \sum_{i=1}^{\infty} P(A_i)$$

**证毕。**

## 9.2.1.5 条件概率与独立性

### 9.2.1.5.1 条件概率

**定义 9.2.1.9**（条件概率，Conditional Probability）

设 $P(B) > 0$，在事件 $B$ 发生的条件下事件 $A$ 发生的**条件概率**为：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

**定理 9.2.1.6**（条件概率是概率测度）

固定 $B$ 且 $P(B) > 0$，定义 $P_B(A) = P(A|B)$，则 $(\Omega, \mathcal{F}, P_B)$ 是概率空间。

**证明：**
验证Kolmogorov公理：

1. $P_B(A) = \frac{P(A \cap B)}{P(B)} \geq 0$
2. $P_B(\Omega) = \frac{P(\Omega \cap B)}{P(B)} = \frac{P(B)}{P(B)} = 1$
3. 对于不交序列 $\{A_i\}$，$\{A_i \cap B\}$ 也不交：
$$P_B\left(\bigcup_i A_i\right) = \frac{P(\bigcup_i (A_i \cap B))}{P(B)} = \sum_i \frac{P(A_i \cap B)}{P(B)} = \sum_i P_B(A_i)$$

**证毕。**

### 9.2.1.5.2 独立性

**定义 9.2.1.10**（事件独立性，Independence）

两个事件 $A, B$ **独立**，如果：

$$P(A \cap B) = P(A)P(B)$$

等价地，若 $P(B) > 0$，则 $P(A|B) = P(A)$。

**定义 9.2.1.11**（事件类的独立性）

事件类 $\mathcal{A}_1, \ldots, \mathcal{A}_n$ **独立**，如果对任意 $A_i \in \mathcal{A}_i$：

$$P\left(\bigcap_{i=1}^{n} A_i\right) = \prod_{i=1}^{n} P(A_i)$$

**定义 9.2.1.12**（σ-代数的独立性）

$\sigma$-代数 $\mathcal{F}_1, \ldots, \mathcal{F}_n \subseteq \mathcal{F}$ **独立**，如果对所有 $A_i \in \mathcal{F}_i$ 满足上述乘积公式。

## 9.2.1.6 代码实现

### 9.2.1.6.1 Python实现

```python
from typing import Set, FrozenSet, Callable, Dict, List, Tuple
from functools import reduce
from itertools import combinations
import numpy as np

class FiniteProbabilitySpace:
    """有限概率空间的实现"""

    def __init__(self, omega: Set, probabilities: Dict[FrozenSet, float]):
        """
        初始化有限概率空间

        Args:
            omega: 样本空间（有限集合）
            probabilities: 基本事件的概率字典 {frozenset({ω}): P({ω})}
        """
        self.omega = frozenset(omega)
        self.probabilities = probabilities

        # 验证概率测度
        self._validate()

    def _validate(self):
        """验证Kolmogorov公理"""
        # 非负性
        for event, prob in self.probabilities.items():
            if prob < 0 or prob > 1:
                raise ValueError(f"概率必须在[0,1]之间: {prob}")

        # 基本事件概率和为1
        total = sum(self.probabilities.values())
        if not np.isclose(total, 1.0):
            raise ValueError(f"概率之和必须等于1: {total}")

    def probability(self, event: Set) -> float:
        """计算事件的概率"""
        event_frozen = frozenset(event)

        # 对于有限空间，假设基本事件独立
        if event_frozen in self.probabilities:
            return self.probabilities[event_frozen]

        # 计算复合事件概率
        prob = 0.0
        for basic_event, basic_prob in self.probabilities.items():
            if basic_event.issubset(event_frozen):
                prob += basic_prob
        return prob

    def conditional_probability(self, event_a: Set, event_b: Set) -> float:
        """
        计算条件概率 P(A|B)

        Raises:
            ValueError: 如果 P(B) = 0
        """
        prob_b = self.probability(event_b)
        if prob_b == 0:
            raise ValueError("P(B) 不能为零")

        prob_intersection = self.probability(set(event_a) & set(event_b))
        return prob_intersection / prob_b

    def are_independent(self, event_a: Set, event_b: Set) -> bool:
        """检查两个事件是否独立"""
        prob_a = self.probability(event_a)
        prob_b = self.probability(event_b)
        prob_intersection = self.probability(set(event_a) & set(event_b))

        return np.isclose(prob_intersection, prob_a * prob_b)

    def union_bound(self, events: List[Set]) -> Tuple[float, float]:
        """
        计算并集概率和Boole上界

        Returns:
            (实际概率, Boole上界)
        """
        # 实际概率（容斥原理，对于小集合）
        actual_prob = 0.0
        n = len(events)

        for k in range(1, n + 1):
            for combo in combinations(events, k):
                intersection = reduce(lambda x, y: x & y, (set(e) for e in combo))
                prob_intersection = self.probability(intersection)
                actual_prob += ((-1) ** (k + 1)) * prob_intersection

        # Boole上界
        boole_bound = sum(self.probability(e) for e in events)

        return actual_prob, min(boole_bound, 1.0)  # 概率不超过1


def create_coin_toss_space():
    """创建抛硬币概率空间"""
    omega = {'H', 'T'}
    probabilities = {
        frozenset({'H'}): 0.5,
        frozenset({'T'}): 0.5
    }
    return FiniteProbabilitySpace(omega, probabilities)


def create_die_space():
    """创建掷骰子概率空间"""
    omega = {1, 2, 3, 4, 5, 6}
    probabilities = {frozenset({i}): 1/6 for i in omega}
    return FiniteProbabilitySpace(omega, probabilities)


# 使用示例
if __name__ == "__main__":
    # 掷骰子示例
    print("=" * 50)
    print("掷骰子概率空间")
    print("=" * 50)

    die = create_die_space()

    # 基本事件概率
    print(f"P({{1}}) = {die.probability({1})}")
    print(f"P(偶数) = {die.probability({2, 4, 6})}")
    print(f"P(大于4) = {die.probability({5, 6})}")

    # 条件概率
    print(f"\nP(偶数 | 大于3) = {die.conditional_probability({2, 4, 6}, {4, 5, 6}):.4f}")

    # 独立性检验
    even = {2, 4, 6}
    greater_than_3 = {4, 5, 6}
    print(f"\n'偶数'与'大于3'独立? {die.are_independent(even, greater_than_3)}")

    # 独立事件示例
    # 创建两个独立骰子的联合空间（简化）
    print("\n" + "=" * 50)
    print("双骰子概率空间")
    print("=" * 50)

    # 简化为：事件A=第一个骰子为6，事件B=第二个骰子为6
    # 假设独立性，P(A∩B) = P(A)P(B) = 1/36
    prob_a = 1/6
    prob_b = 1/6
    prob_intersection = prob_a * prob_b
    print(f"P(双6) = {prob_intersection:.4f} = 1/36")
```

### 9.2.1.6.2 Lean 4形式化

```lean4
import Mathlib

open Set MeasureTheory

namespace ProbabilityTheory

/-- Kolmogorov公理：概率测度 -/
structure ProbabilitySpace (Ω : Type*) where
  /-- σ-代数 -/
  F : MeasurableSpace Ω
  /-- 概率测度 -/
  P : @Measure Ω F.toMeasurableSpace
  /-- 规范性：P(Ω) = 1 -/
  isProbability : P .univ = 1

variable {Ω : Type*} [MeasurableSpace Ω]

/-- 概率测度的基本性质 -/
section ProbabilityProperties

variable (P : Measure Ω) [IsProbabilityMeasure P]

/-- P(∅) = 0 -/
theorem prob_empty : P ∅ = 0 := by
  exact measure_empty

/-- 补事件概率 -/
theorem prob_compl {A : Set Ω} (hA : MeasurableSet A) :
    P Aᶜ = 1 - P A := by
  rw [← @isProbabilityMeasure_iff Ω _ P]
  rw [measure_compl hA]
  rw [isProbabilityMeasure_iff.1 (by infer_instance)]
  all_goals try tauto

/-- 单调性 -/
theorem prob_mono {A B : Set Ω} (hA : MeasurableSet A) (hB : MeasurableSet B)
    (hAB : A ⊆ B) : P A ≤ P B := by
  exact measure_mono hAB

/-- Boole不等式 / Union Bound -/
theorem boole_inequality {ι : Type*} [Countable ι] (A : ι → Set Ω)
    (hA : ∀ i, MeasurableSet (A i)) :
    P (⋃ i, A i) ≤ ∑' i, P (A i) := by
  apply measure_iUnion_le

/-- 有限可加性 -/
theorem prob_finite_add {n : ℕ} (A : Fin n → Set Ω)
    (hA : ∀ i, MeasurableSet (A i))
    (hDisjoint : Pairwise (Disjoint on A)) :
    P (⋃ i, A i) = ∑ i, P (A i) := by
  rw [measure_iUnion]
  · rfl
  · exact hDisjoint
  · exact hA

end ProbabilityProperties

/-- 条件概率 -/
noncomputable def conditionalProbability (P : Measure Ω) [IsProbabilityMeasure P]
    (B : Set Ω) (hB : MeasurableSet B) (hPos : P B > 0) (A : Set Ω) : ℝ :=
  P (A ∩ B) / P B

/-- 事件独立性 -/
def IndependentEvents (P : Measure Ω) [IsProbabilityMeasure P]
    (A B : Set Ω) : Prop :=
  P (A ∩ B) = P A * P B

/-- 独立性对称 -/
theorem independent_symm {P : Measure Ω} [IsProbabilityMeasure P] {A B : Set Ω} :
    IndependentEvents P A B ↔ IndependentEvents P B A := by
  rw [IndependentEvents, IndependentEvents]
  rw [Set.inter_comm]

end ProbabilityTheory
```

## 9.2.1.9 练习

**练习 9.2.1.1** 证明若 $\mathcal{F}$ 是 $\sigma$-代数，则对任意 $A, B \in \mathcal{F}$，有 $A \Delta B = (A \setminus B) \cup (B \setminus A) \in \mathcal{F}$。

**练习 9.2.1.2** 设 $P$ 是概率测度，证明对于递增序列 $A_n \uparrow A$，有 $\lim_{n \to \infty} P(A_n) = P(A)$。

**练习 9.2.1.3** 证明Bonferroni不等式：$P(A \cap B) \geq P(A) + P(B) - 1$。

**练习 9.2.1.4** 设 $A, B, C$ 独立，证明 $A \cup B$ 与 $C$ 独立。
