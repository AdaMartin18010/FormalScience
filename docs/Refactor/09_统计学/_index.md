# 09 统计学模块索引

> **模块**: 统计学
> **描述**: 从数据到推断的数学科学，涵盖描述统计、概率基础、推断方法与统计计算
> **文件数**: 22
> **状态**: 📝 内容填充中

---

## 📋 内容概述

本模块系统介绍统计学的理论与方法，从基础的数据描述与可视化出发，深入概率论基础、统计推断理论，以及现代贝叶斯统计和统计计算方法。强调统计思维的形式化表达与计算实现。

### 核心主题

- **描述统计**: 数据集中趋势与离散度、可视化方法、探索性数据分析
- **概率论基础**: 概率空间、随机变量、概率分布、极限定理
- **推断统计**: 点估计、区间估计、假设检验、方差分析
- **贝叶斯统计**: 贝叶斯定理、先验与后验分布、MCMC方法
- **数理统计**: 统计决策理论、充分统计量、渐近理论
- **统计计算**: 随机数生成、重采样方法、计算贝叶斯方法
