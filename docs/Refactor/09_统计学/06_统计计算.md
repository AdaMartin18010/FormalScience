## 1. 随机数生成

### 1.1 伪随机数

#### 1.1.1 线性同余生成器 (LCG)

**算法**:

$$
X_{n+1} = (a X_n + c) \mod m
$$

其中：

- $m > 0$: 模数
- $0 < a < m$: 乘数
- $0 \leq c < m$: 增量
- $0 \leq X_0 < m$: 种子

**归一化**: $U_n = X_n / m \in [0, 1)$

**常用参数** (RANDU问题后改进):

- Numerical Recipes: $m = 2^{32}$, $a = 1664525$, $c = 1013904223$
- glibc: $m = 2^{31}$, $a = 1103515245$, $c = 12345$

**周期**: 最大周期为 $m$，当且仅当：

1. $c$ 与 $m$ 互质
2. $a-1$ 被 $m$ 的所有素因子整除
3. 若 $m$ 被4整除，则 $a-1$ 也被4整除

#### 1.1.2 梅森旋转算法 (MT19937)

**特点**:

- 周期极长: $2^{19937} - 1$ (梅森素数)
- 623维均匀分布
- 速度快，现代标准

**算法概要**:

1. 初始化624个状态字
2. 扭曲操作混合位
3. tempering步骤进一步打乱

**Python/NumPy默认随机数生成器**

#### 1.1.3 随机数检验

| 检验类型 | 检验内容 | 典型测试 |
|----------|----------|----------|
| **频率检验** | 0/1出现频率 | Monobit test |
| **游程检验** | 连续序列长度 | Runs test |
| **序列检验** | 相邻数关系 | Serial test |
| **间隔检验** | 特定模式间隔 | Gap test |
| **扑克检验** | 模式分布 | Poker test |
| **Diehard** | 综合检验 | Marsaglia's suite |
| **TestU01** | 现代综合 | L'Ecuyer's suite |

### 1.2 非均匀随机变量生成

#### 1.2.1 逆变换法

**定理 1.1** (逆变换法)

设 $U \sim \text{Uniform}(0,1)$，$F$ 是CDF，则：

$$
X = F^{-1}(U)
$$

服从分布 $F$。

**证明**: $P(X \leq x) = P(F^{-1}(U) \leq x) = P(U \leq F(x)) = F(x)$ ∎

**应用示例**

| 分布 | CDF | 逆变换 |
|------|-----|--------|
| **Exponential(λ)** | $1 - e^{-\lambda x}$ | $-\frac{1}{\lambda}\log(1-U)$ |
| **Cauchy** | $\frac{1}{2} + \frac{\arctan x}{\pi}$ | $\tan(\pi(U - 1/2))$ |
| **Logistic** | $\frac{1}{1+e^{-x}}$ | $\log\frac{U}{1-U}$ |
| **Pareto** | $1 - (\frac{x_m}{x})^\alpha$ | $\frac{x_m}{(1-U)^{1/\alpha}}$ |

#### 1.2.2 接受-拒绝法

**算法**:

1. 选择提议分布 $g(x)$ 和常数 $M$ 使得 $f(x) \leq Mg(x)$ 对所有 $x$
2. 从 $g$ 生成 $Y$
3. 从 $\text{Uniform}(0,1)$ 生成 $U$
4. 若 $U \leq \frac{f(Y)}{Mg(Y)}$，返回 $X = Y$；否则重复

**效率**: 接受概率 = $1/M$，期望迭代次数 = $M$

**定理 1.2**: 接受-拒绝法生成的 $X$ 服从目标分布 $f$。

#### 1.2.3 复合方法

**混合分布**: 若 $f(x) = \sum_{i=1}^{k} p_i f_i(x)$，其中 $\sum p_i = 1$：

1. 生成 $I \sim \text{Categorical}(p_1, \ldots, p_k)$
2. 从 $f_I$ 生成 $X$

### 1.3 特殊分布采样

#### 1.3.1 正态分布 (Box-Muller)

**定理 1.3** (Box-Muller变换)

设 $U_1, U_2 \overset{iid}{\sim} \text{Uniform}(0,1)$，则：

$$
Z_1 = \sqrt{-2\ln U_1} \cos(2\pi U_2)
$$

$$
Z_2 = \sqrt{-2\ln U_1} \sin(2\pi U_2)
$$

满足 $Z_1, Z_2 \overset{iid}{\sim} \mathcal{N}(0, 1)$。

#### 1.3.2 多元正态

**算法**:

1. 对 $\boldsymbol{\Sigma}$ 进行Cholesky分解: $\boldsymbol{\Sigma} = \mathbf{L}\mathbf{L}^T$
2. 生成 $\mathbf{Z} = (Z_1, \ldots, Z_p)^T$，$Z_i \overset{iid}{\sim} \mathcal{N}(0,1)$
3. 返回 $\mathbf{X} = \boldsymbol{\mu} + \mathbf{L}\mathbf{Z}$

#### 1.3.3 Gamma与Beta

**Gamma采样**:

- $\alpha < 1$: 使用接受-拒绝
- $\alpha > 1$: Marsaglia-Tsang方法
- 整数 $\alpha$: $k$ 个指数分布之和

**Beta采样**: 若 $X \sim \text{Gamma}(\alpha, 1)$，$Y \sim \text{Gamma}(\beta, 1)$，则：

$$
\frac{X}{X+Y} \sim \text{Beta}(\alpha, \beta)
$$

## 2. 重采样方法

### 2.1 Bootstrap方法

#### 2.1.1 非参数Bootstrap

**算法**:

1. 从原始数据 $\mathbf{x} = (x_1, \ldots, x_n)$ 中有放回地抽取 $n$ 个样本
2. 计算统计量 $\hat{\theta}^* = T(\mathbf{x}^*)$
3. 重复 $B$ 次，得到 $\hat{\theta}^*_1, \ldots, \hat{\theta}^*_B$
4. 用 $\{\hat{\theta}^*_i\}$ 估计 $\hat{\theta}$ 的抽样分布

**用途**:

- 标准误估计: $\widehat{SE} = \sqrt{\frac{1}{B-1}\sum(\hat{\theta}^*_i - \bar{\theta}^*)^2}$
- 置信区间构造
- 偏差校正

#### 2.1.2 参数Bootstrap

**适用**: 数据来自已知分布族 $f(x;\theta)$

**算法**:

1. 估计参数 $\hat{\theta}$ (如MLE)
2. 从 $f(x;\hat{\theta})$ 生成样本 $\mathbf{x}^*$
3. 计算 $\hat{\theta}^*$
4. 重复 $B$ 次

#### 2.1.3 Bootstrap置信区间

| 方法 | 公式 | 特点 |
|------|------|------|
| **正态近似** | $\hat{\theta} \pm z_{\alpha/2} \cdot \widehat{SE}$ | 简单，假设对称 |
| **百分位法** | $(\hat{\theta}^*_{(\alpha/2)}, \hat{\theta}^*_{(1-\alpha/2)})$ | 简单，可能有偏差 |
| **BCa** | 偏差校正加速 | 最准确，计算复杂 |
| **t-Bootstrap** | Bootstrap-t统计量 | 二阶精确 |

#### 2.1.4 Jackknife方法

**算法** (Leave-one-out):

$$
\hat{\theta}_{(i)} = T(x_1, \ldots, x_{i-1}, x_{i+1}, \ldots, x_n)
$$

**用途**:

- 偏差估计: $\widehat{\text{Bias}} = (n-1)(\bar{\theta}_{(·)} - \hat{\theta})$
- 方差估计: $\widehat{\text{Var}} = \frac{n-1}{n}\sum(\hat{\theta}_{(i)} - \bar{\theta}_{(·)})^2$

### 2.2 置换检验

#### 2.2.1 置换检验原理

**思想**: 在 $H_0$ 下，数据标签可交换

**算法**:

1. 计算观测统计量 $T_{obs}$
2. 置换标签 $M$ 次，计算 $T_1, \ldots, T_M$
3. p值 = $\frac{1}{M}\sum_{i=1}^{M}\mathbb{I}(T_i \geq T_{obs})$

**精确性**: 若枚举所有置换，得到**精确检验**。

### 2.3 交叉验证

#### 2.3.1 K折交叉验证

**算法**:

1. 将数据分为 $K$ 个子集
2. 对于 $k = 1, \ldots, K$:
   - 用第 $k$ 折作为验证集
   - 其余作为训练集训练模型
   - 记录验证误差
3. 返回平均验证误差

#### 2.3.2 留一法 (LOOCV)

K折的特例，$K = n$。

**线性模型下的快捷计算**:

$$
\text{LOOCV} = \frac{1}{n}\sum_{i=1}^{n}\left(\frac{y_i - \hat{y}_i}{1 - h_{ii}}\right)^2
$$

其中 $h_{ii}$ 是帽子矩阵对角元。

## 3. 优化方法

### 3.1 经典优化

#### 3.1.1 Newton-Raphson

**算法**:

$$
\theta^{(t+1)} = \theta^{(t)} - [\nabla^2 \ell(\theta^{(t)})]^{-1} \nabla \ell(\theta^{(t)})
$$

**收敛**: 二次收敛 (接近最优解时)

#### 3.1.2 Fisher得分法

用Fisher信息矩阵代替Hessian：

$$
\theta^{(t+1)} = \theta^{(t)} + [I(\theta^{(t)})]^{-1} \nabla \ell(\theta^{(t)})
$$

#### 3.1.3 EM算法

**适用**: 缺失数据或隐变量模型

**算法**:

**E步**: 计算Q函数

$$
Q(\theta | \theta^{(t)}) = E_{Z|\mathbf{X},\theta^{(t)}}[\log L(\theta; \mathbf{X}, Z)]
$$

**M步**: 最大化Q函数

$$
\theta^{(t+1)} = \arg\max_\theta Q(\theta | \theta^{(t)})
**

**收敛性**: 保证似然不减小，收敛到局部极大。

### 3.2 随机优化

#### 3.2.1 随机梯度下降 (SGD)

**算法**:

$$
\theta^{(t+1)} = \theta^{(t)} - \eta_t \nabla_\theta L(\theta^{(t)}; x_i)
$$

其中 $x_i$ 是随机选取的样本。

**学习率调度**:
- 固定: $\eta_t = \eta_0$
- 衰减: $\eta_t = \eta_0 / (1 + \lambda t)$
- AdaGrad, Adam等自适应方法

#### 3.2.2 模拟退火

**思想**: 以一定概率接受劣解，概率随"温度"降低

**接受准则**:

$$
P(\text{接受新解}) = \begin{cases}
1 & \text{if } \Delta E < 0 \\
e^{-\Delta E/T} & \text{if } \Delta E \geq 0
\end{cases}
$$

#### 3.2.3 遗传算法

**算子**:
1. **选择**: 适应度高的个体更可能被选
2. **交叉**: 交换染色体片段
3. **变异**: 随机改变基因

## 4. 数值线性代数

### 4.1 矩阵分解

#### 4.1.1 LU分解

**分解**: $\mathbf{A} = \mathbf{L}\mathbf{U}$

- $\mathbf{L}$: 下三角，对角线为1
- $\mathbf{U}$: 上三角

**应用**: 解线性方程组

#### 4.1.2 QR分解

**分解**: $\mathbf{A} = \mathbf{Q}\mathbf{R}$

- $\mathbf{Q}$: 正交矩阵
- $\mathbf{R}$: 上三角矩阵

**应用**: 最小二乘、特征值计算

#### 4.1.3 Cholesky分解

**分解**: 对于正定矩阵 $\mathbf{A}$：

$$
\mathbf{A} = \mathbf{L}\mathbf{L}^T
$$

其中 $\mathbf{L}$ 是下三角。

**应用**: 多元正态采样、线性系统求解

#### 4.1.4 SVD

**分解**: $\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^T$

**应用**:
- 伪逆计算
- 主成分分析
- 矩阵低秩逼近

### 4.2 线性回归计算

#### 4.2.1 正规方程

$$
\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}
$$

**计算复杂度**: $O(p^2n + p^3)$

#### 4.2.2 QR方法

$$
\mathbf{X} = \mathbf{Q}\mathbf{R} \Rightarrow \hat{\boldsymbol{\beta}} = \mathbf{R}^{-1}\mathbf{Q}^T\mathbf{y}
$$

**优势**: 数值稳定性更好

#### 4.2.3 岭回归计算

$$
\hat{\boldsymbol{\beta}}_{ridge} = (\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}^T\mathbf{y}
$$

**SVD方法**:

$$
\hat{\boldsymbol{\beta}}_{ridge} = \mathbf{V}\text{diag}\left(\frac{\sigma_i}{\sigma_i^2 + \lambda}\right)\mathbf{U}^T\mathbf{y}
$$

## 5. 计算复杂性

### 5.1 算法复杂性

#### 5.1.1 时间复杂性

| 操作 | 复杂度 | 说明 |
|------|--------|------|
| 矩阵乘法 | $O(n^3)$ (标准) | $O(n^{2.37})$ (Strassen) |
| 矩阵求逆 | $O(n^3)$ | 通过LU分解 |
| SVD | $O(mn^2)$ | $m \geq n$ |
| FFT | $O(n\log n)$ | 快速傅里叶变换 |
| 排序 | $O(n\log n)$ | 比较排序下界 |

#### 5.1.2 空间复杂性

| 算法 | 空间需求 |
|------|----------|
| 原位排序 | $O(1)$ 额外空间 |
| 矩阵分解 | $O(n^2)$ |
| Bootstrap | $O(Bn)$ |
| MCMC | $O(kn)$ (k为链数) |

### 5.2 统计问题的复杂性

#### 5.2.1 NP难统计问题

| 问题 | 复杂性 | 说明 |
|------|--------|------|
| 最优子集选择 | NP难 | 穷举 $2^p$ 个子集 |
| 混合模型MLE | NP难 | 多峰似然 |
| 贝叶斯网络结构学习 | NP难 | 评分搜索 |

#### 5.2.2 近似算法

**松弛方法**:
- LASSO近似最优子集
- 变分推断近似后验
- 贪心算法

### 5.3 并行计算

#### 5.3.1 并行随机数

**问题**: 多流需要不相关

**解决方案**:
- **Leapfrog**: 每个流取第 $k$ 个值
- **序列分割**: 不同子序列
- **参数化生成器**: 不同参数
- **跳跃 ahead**: 直接跳到未来状态

#### 5.3.2 GPU加速

**适用场景**:
- 大规模矩阵运算
- 并行MCMC链
- 深度学习训练
- Bootstrap重采样

#### 5.3.3 分布式统计计算

**框架**:
- **MapReduce**: 大数据处理
- **Spark**: 迭代算法
- **Dask**: Python生态

**统计应用**:
- 分布式Bootstrap
- 分块GLM拟合
- 并行贝叶斯推断
