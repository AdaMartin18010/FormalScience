## 13.2.3.2 贝叶斯脑假说

### 13.2.3.2.1 核心假设

**假设 13.2.3.1** (贝叶斯脑假说)

神经系统实现（近似）贝叶斯推断，通过编码和操纵概率分布来表征不确定性。

**形式化**: 脑计算后验分布：

$$
P(s \mid o) = \frac{P(o \mid s) P(s)}{P(o)}
$$

其中 $s$ 为隐藏状态（原因），$o$ 为观测。

### 13.2.3.2.2 概率编码

**定义 13.2.3.1** (群体编码)

神经元群体 $\mathbf{r} = (r_1, \ldots, r_n)$ 编码概率分布的参数：

$$
P(s \mid \mathbf{r}) = \mathcal{N}(s; \mu(\mathbf{r}), \sigma^2(\mathbf{r}))
$$

**定理 13.2.3.1** (贝叶斯群体编码)

若各神经元的调谐曲线为 $f_i(s)$，则最大似然估计：

$$
\hat{s} = \arg\max_s \sum_i \log P(r_i \mid s)
$$

## 13.2.3.3 感知作为推断

### 13.2.3.3.1 视觉感知

**定义 13.2.3.2** (视觉感知模型)

**Helmholtz机器**将视觉视为反问题求解：

$$
P(\text{场景} \mid \text{图像}) \propto P(\text{图像} \mid \text{场景}) \cdot P(\text{场景})

$$

**定理 13.2.3.2** (多稳态感知的贝叶斯解释)

双稳态感知（如Necker立方体）源于后验分布的多峰性：

$$
P(s \mid o) = \alpha P_1(s) + (1-\alpha) P_2(s)
$$

### 13.2.3.3.2 因果推断

**定义 13.2.3.3** (因果推断模型)

多感官整合中的因果推断：

$$
P(C=1 \mid x_v, x_a) = \frac{P(x_v, x_a \mid C=1)P(C=1)}{\sum_C P(x_v, x_a \mid C)P(C)}
$$

其中 $C=1$ 表示视听刺激来自同一来源。

## 13.2.3.4 预测编码

### 13.2.3.4.1 自由能原理

**定义 13.2.3.4** (变分自由能)

$$
\mathcal{F}(q, o) = \mathbb{E}_q[\ln q(s) - \ln P(o, s)] = D_{KL}(q(s) \| P(s \mid o)) - \ln P(o)
$$

**定理 13.2.3.3** (自由能最小化)

最小化自由能等价于：

1. 使变分分布 $q(s)$ 接近后验 $P(s \mid o)$
2. 最大化证据（模型证据）

**证明**:

$$
\begin{aligned}
\mathcal{F}(q, o) &= \mathbb{E}_q[\ln q(s) - \ln P(s \mid o) - \ln P(o)] \\
&= D_{KL}(q(s) \| P(s \mid o)) - \ln P(o)
\end{aligned}
$$

由于 $\ln P(o)$ 与 $q$ 无关，最小化 $\mathcal{F}$ 使 $q \rightarrow P(s \mid o)$。

### 13.2.3.4.2 预测编码网络

**定义 13.2.3.5** (预测编码层级)

层级预测编码模型（Rao & Ballard, 1999）：

$$
\mathbf{r}^{(l)} = f(\mathbf{e}^{(l-1)})
$$

$$
\mathbf{e}^{(l)} = \mathbf{r}^{(l)} - g(\mathbf{r}^{(l+1)})
$$

其中 $\mathbf{r}$ 为表征，$\mathbf{e}$ 为预测误差。

## 13.2.3.5 主动推断

### 13.2.3.5.1 感知-行动循环

**定义 13.2.3.6** (主动推断)

智能体通过行动最小化期望自由能：

$$
\mathcal{G}(\pi) = \mathbb{E}_{q(o, s \mid \pi)}[\ln q(s \mid \pi) - \ln P(o, s)]
$$

### 13.2.3.5.2 目标导向行为

**定理 13.2.3.4** (目标作为先验)

目标导向行为通过设置先验 $P(s_{target})$ 实现：

$$
\pi^* = \arg\min_\pi \mathcal{G}(\pi) = \arg\max_\pi \mathbb{E}[\ln P(s_{target} \mid s_\pi)]
$$

## 13.2.3.6 贝叶斯决策

### 13.2.3.6.1 概率推理

**定义 13.2.3.7** (贝叶斯决策理论)

最优决策最大化期望效用：

$$
a^* = \arg\max_a \mathbb{E}_{P(s \mid o)}[U(a, s)]
$$

### 13.2.3.6.2 概率心理测量学

**定理 13.2.3.5** (不确定性下的反应时)

反应时与后验熵相关：

$$
RT \propto H(P(s \mid o)) = -\sum_s P(s \mid o) \ln P(s \mid o)
$$

"""
贝叶斯认知模型实现
基于贝叶斯脑假说与预测编码
"""

import numpy as np
from scipy import stats
from scipy.special import logsumexp
from typing import Tuple, List, Callable, Optional, Dict
from dataclasses import dataclass
import matplotlib.pyplot as plt


@dataclass
class BayesianPerception:
    """
    贝叶斯感知模型
    实现感知作为概率推断
    """

    def __init__(self, prior_mean: float = 0.0, prior_std: float = 1.0):
        self.prior_mean = prior_mean
        self.prior_std = prior_std

    def posterior(self, observation: float,
                  likelihood_std: float) -> Tuple[float, float]:
        """
        计算高斯先验-似然的后验

        先验: P(s) = N(μ_0, σ_0²)
        似然: P(o|s) = N(s, σ²)
        后验: P(s|o) = N(μ_n, σ_n²)

        Returns:
            (后验均值, 后验标准差)
        """
        # 精度（逆方差）
        prior_precision = 1.0 / (self.prior_std ** 2)
        likelihood_precision = 1.0 / (likelihood_std ** 2)

        # 后验精度 = 先验精度 + 似然精度
        posterior_precision = prior_precision + likelihood_precision
        posterior_var = 1.0 / posterior_precision

        # 后验均值 (加权平均)
        posterior_mean = posterior_var * (
            prior_precision * self.prior_mean +
            likelihood_precision * observation
        )

        return posterior_mean, np.sqrt(posterior_var)

    def multisensory_integration(self,
                                 visual_obs: float, visual_std: float,
                                 auditory_obs: float, auditory_std: float,
                                 causal_prior: float = 0.5) -> Dict:
        """
        多感官整合的因果推断模型

        实现Kording et al. (2007)的因果推断模型
        """
        # 场景1: 共同来源 (C=1)
        # 强制融合
        combined_precision = 1.0/visual_std**2 + 1.0/auditory_std**2
        combined_mean = (visual_obs/visual_std**2 + auditory_obs/auditory_std**2) / combined_precision
        combined_std = 1.0 / np.sqrt(combined_precision)

        # 场景1的似然
        likelihood_common = stats.norm.pdf(visual_obs, combined_mean,
                                           np.sqrt(visual_std**2 + combined_std**2)) * \
                           stats.norm.pdf(auditory_obs, combined_mean,
                                         np.sqrt(auditory_std**2 + combined_std**2))

        # 场景2: 分离来源 (C=2)
        likelihood_separate = stats.norm.pdf(visual_obs, self.prior_mean,
                                             np.sqrt(self.prior_std**2 + visual_std**2)) * \
                             stats.norm.pdf(auditory_obs, self.prior_mean,
                                           np.sqrt(self.prior_std**2 + auditory_std**2))

        # 后验因果概率
        p_common = causal_prior * likelihood_common
        p_separate = (1 - causal_prior) * likelihood_separate
        p_c = p_common / (p_common + p_separate)

        # 模型平均估计
        final_estimate = p_c * combined_mean + (1 - p_c) * visual_obs  # 以视觉为主导

        return {
            'causal_probability': p_c,
            'combined_estimate': combined_mean,
            'visual_estimate': visual_obs,
            'auditory_estimate': auditory_obs,
            'final_estimate': final_estimate,
            'strategy': 'fusion' if p_c > 0.5 else 'segregation'
        }


class PredictiveCoding:
    """
    预测编码模型
    基于Rao & Ballard (1999)和Friston (2005)
    """

    def __init__(self, n_levels: int, n_units_per_level: List[int],
                 learning_rate: float = 0.01):
        """
        初始化层级预测编码网络

        Args:
            n_levels: 层级数
            n_units_per_level: 每级单元数
            learning_rate: 学习率
        """
        self.n_levels = n_levels
        self.n_units = n_units_per_level
        self.lr = learning_rate

        # 初始化表征和误差单元
        self.r = [np.zeros(n) for n in n_units_per_level]
        self.e = [np.zeros(n) for n in n_units_per_level]

        # 生成模型参数
        self.W = []  # 前向权重
        self.V = []  # 后向权重
        for i in range(n_levels - 1):
            self.W.append(np.random.randn(n_units_per_level[i+1],
                                         n_units_per_level[i]) * 0.01)
            self.V.append(np.random.randn(n_units_per_level[i],
                                         n_units_per_level[i+1]) * 0.01)

    def predict(self, level: int) -> np.ndarray:
        """
        从高层预测低层
        prediction = g(r_{l+1})
        """
        if level >= len(self.V):
            return np.zeros(self.n_units[level])
        return np.tanh(np.dot(self.V[level], self.r[level + 1]))

    def encode(self, level: int) -> np.ndarray:
        """
        从低层编码高层
        """
        if level == 0:
            return self.r[level]
        return np.tanh(np.dot(self.W[level-1], self.r[level-1]))

    def update_errors(self, observation: np.ndarray):
        """
        更新预测误差
        e_l = r_l - predict(l)
        """
        self.r[0] = observation.copy()

        for l in range(self.n_levels):
            prediction = self.predict(l)
            self.e[l] = self.r[l] - prediction

    def update_representations(self, n_iterations: int = 10):
        """
        使用梯度下降更新表征以最小化预测误差

        能量函数: F = Σ_l e_l²
        """
        for _ in range(n_iterations):
            for l in range(1, self.n_levels):
                # 梯度: dF/dr_l
                grad = self.e[l].copy()

                # 来自低层的误差
                if l < self.n_levels - 1:
                    back_pred = np.tanh(np.dot(self.V[l], self.r[l+1]))
                    grad -= (self.r[l] - back_pred)

                # 更新表征
                self.r[l] += self.lr * grad

                # 重新计算误差
                self.update_errors(self.r[0])

    def learn_weights(self):
        """
        Hebbian-like学习更新权重
        """
        for l in range(len(self.W)):
            # 更新前向权重
            self.W[l] += self.lr * np.outer(self.r[l+1], self.r[l])
            # 更新后向权重
            self.V[l] += self.lr * np.outer(self.r[l], self.r[l+1])

    def free_energy(self) -> float:
        """
        计算变分自由能 (总预测误差)
        """
        return sum(np.sum(e ** 2) for e in self.e)


class ActiveInference:
    """
    主动推断代理
    实现Friston (2010)的自由能原理
    """

    def __init__(self, n_states: int, n_observations: int, n_actions: int,
                 prior_preferences: Optional[np.ndarray] = None):
        """
        初始化主动推断代理

        Args:
            n_states: 状态数
            n_observations: 观测数
            n_actions: 动作数
            prior_preferences: 先验偏好 (目标分布)
        """
        self.n_states = n_states
        self.n_obs = n_observations
        self.n_actions = n_actions

        # 生成模型
        self.A = np.random.dirichlet(np.ones(n_observations),
                                     n_states).T  # 似然 P(o|s)
        self.B = [np.random.dirichlet(np.ones(n_states), n_states).T
                  for _ in range(n_actions)]  # 转移 P(s'|s,a)

        # 先验偏好
        if prior_preferences is None:
            self.C = np.ones(n_observations) / n_observations
        else:
            self.C = prior_preferences

        # 先验信念
        self.D = np.ones(n_states) / n_states

        # 当前状态信念
        self.q_s = self.D.copy()

    def infer_states(self, observation: int):
        """
        状态推断 (感知)
        更新 q(s) ∝ P(o|s) P(s)
        """
        likelihood = self.A[observation, :]
        self.q_s = likelihood * self.q_s
        self.q_s /= np.sum(self.q_s)  # 归一化

    def expected_free_energy(self, action: int) -> float:
        """
        计算动作的期望自由能

        G(π) = Σ_s q(s|π) [ln q(s|π) - ln P(s)]
             + Σ_s q(s|π) Σ_o q(o|s) ln q(o|s) - ln C(o)]
        """
        # 预期状态
        expected_s = np.dot(self.B[action], self.q_s)

        # 第一项: 与先验的KL散度
        kl_prior = np.sum(expected_s * (np.log(expected_s + 1e-10) -
                                       np.log(self.D + 1e-10)))

        # 第二项: 预期观测的负信息增益 + 与偏好的差异
        exp_obs = np.dot(self.A, expected_s)  # P(o|π)
        ambiguity = np.sum(expected_s * np.sum(self.A * np.log(self.A + 1e-10),
                                               axis=0))
        risk = np.sum(exp_obs * (np.log(exp_obs + 1e-10) - np.log(self.C + 1e-10)))

        return kl_prior + ambiguity + risk

    def select_action(self) -> int:
        """
        选择最小化期望自由能的动作
        """
        efe_values = [self.expected_free_energy(a) for a in range(self.n_actions)]

        # Softmax动作选择
        probs = np.exp(-np.array(efe_values))
        probs /= np.sum(probs)

        return np.random.choice(self.n_actions, p=probs)

    def step(self, observation: int) -> int:
        """
        完整的主动推断步骤

        Returns:
            选择的动作
        """
        # 感知: 更新状态信念
        self.infer_states(observation)

        # 行动: 选择最优动作
        action = self.select_action()

        # 更新状态信念 (预测)
        self.q_s = np.dot(self.B[action], self.q_s)

        return action


class BayesianDecision:
    """
    贝叶斯决策模型
    实现不确定性下的最优决策
    """

    def __init__(self,
                 utility_function: Callable[[int, int], float],
                 prior: Optional[np.ndarray] = None):
        """
        初始化贝叶斯决策者

        Args:
            utility_function: U(action, state) -> utility
            prior: 先验信念 P(state)
        """
        self.U = utility_function
        self.prior = prior

    def expected_utility(self, action: int,
                        posterior: np.ndarray) -> float:
        """
        计算动作的期望效用
        EU(a) = Σ_s U(a,s) P(s)
        """
        eu = 0.0
        for state, prob in enumerate(posterior):
            eu += prob * self.U(action, state)
        return eu

    def optimal_action(self, posterior: np.ndarray) -> Tuple[int, float]:
        """
        选择最优动作

        Returns:
            (最优动作, 期望效用)
        """
        n_actions = 2  # 假设二选一
        eus = [self.expected_utility(a, posterior) for a in range(n_actions)]

        best_action = np.argmax(eus)
        return best_action, eus[best_action]

    def choice_probability(self, posterior: np.ndarray,
                          temperature: float = 1.0) -> np.ndarray:
        """
        Softmax选择概率 (Luce选择公理)

        P(a) ∝ exp(EU(a) / τ)
        """
        n_actions = 2
        eus = np.array([self.expected_utility(a, posterior)
                       for a in range(n_actions)])

        exp_eus = np.exp(eus / temperature)
        return exp_eus / np.sum(exp_eus)

class PsychophysicsSimulator:
    """
    心理物理学实验模拟器
    """

    @staticmethod
    def simulate_psychometric_curve(stimulus_levels: np.ndarray,
                                   threshold: float = 0.0,
                                   slope: float = 1.0) -> np.ndarray:
        """
        模拟心理测量曲线 (logistic)

        P(detect) = 1 / (1 + exp(-slope * (s - threshold)))
        """
        return 1.0 / (1.0 + np.exp(-slope * (stimulus_levels - threshold)))

    @staticmethod
    def signal_detection_theory(d_prime: float,
                                 criterion: float = 0.0,
                                 n_trials: int = 1000) -> Dict:
        """
        信号检测理论模拟

        Returns:
            命中率和虚报率
        """
        # 信号呈现试验
        signal_trials = np.random.randn(n_trials) + d_prime
        # 噪音试验
        noise_trials = np.random.randn(n_trials)

        hits = np.sum(signal_trials > criterion) / n_trials
        fa = np.sum(noise_trials > criterion) / n_trials

        return {
            'hit_rate': hits,
            'fa_rate': fa,
            'd_prime': d_prime,
            'criterion': criterion,
            'sensitivity': hits - fa
        }
