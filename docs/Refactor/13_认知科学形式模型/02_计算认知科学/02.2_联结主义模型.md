## 13.2.2.1 引言

联结主义（Connectionism）将认知视为神经网络中并行分布式处理的结果。
本节基于McClelland、Rumelhart等人的Parallel Distributed Processing（PDP）研究计划，阐述神经网络作为认知计算的形式模型。

> **参考**: Rumelhart, D. E., McClelland, J. L., & PDP Research Group. (1986). _Parallel Distributed Processing: Explorations in the Microstructure of Cognition_. MIT Press.

## 13.2.2.2 联结主义基础

### 13.2.2.2.1 并行分布式处理

**定义 13.2.2.1** (PDP系统)

一个PDP系统是一个七元组：

$$
\mathcal{P} = (U, A, W, D, P, F, L)
$$

其中：

- $U$：单元集合
- $A \subseteq U$：输入单元
- $W \subseteq U$：输出单元
- $D: U \times U \rightarrow \mathbb{R}$：连接权重函数
- $P: U \rightarrow \mathcal{P}(U)$：投射函数
- $F$: 激活函数
- $L$: 学习规则

### 13.2.2.2.2 分布式表征

**定义 13.2.2.2** (微特征表征)

概念 $C$ 的表征是单元激活模式 $\mathbf{a}_C \in [0,1]^n$：

$$
\mathbf{a}_C = (a_1, a_2, \ldots, a_n), \quad a_i \in [0,1]
$$

**定理 13.2.2.1** (分布式表征的连续性)

相似概念的表征在激活空间中接近：

$$
\text{Sim}(C_1, C_2) \approx \|\mathbf{a}_{C_1} - \mathbf{a}_{C_2}\|^{-1}
$$

## 13.2.2.3 前馈网络

### 13.2.2.3.1 感知机与多层网络

**定义 13.2.2.3** (感知机)

感知机计算：

$$
y = \sigma\left(\sum_{i=1}^{n} w_i x_i + b\right) = \sigma(\mathbf{w} \cdot \mathbf{x} + b)
$$

其中 $\sigma$ 为激活函数（如sigmoid、ReLU）。

**定理 13.2.2.2** (感知机收敛定理)

若训练数据线性可分，则感知机学习算法在有限步内收敛。

### 13.2.2.3.2 反向传播算法

**定义 13.2.2.4** (多层前馈网络)

网络计算：

$$
\mathbf{a}^{(l)} = \sigma(\mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)})
$$

**定理 13.2.2.3** (反向传播)

损失函数 $\mathcal{L}$ 对权重 $w_{ij}^{(l)}$ 的梯度：

$$
\frac{\partial \mathcal{L}}{\partial w_{ij}^{(l)}} = \delta_i^{(l)} \cdot a_j^{(l-1)}
$$

其中误差信号 $\delta_i^{(l)}$ 递归计算：

$$
\delta_i^{(L)} = (t_i - a_i^{(L)}) \cdot \sigma'(z_i^{(L)})
$$

$$
\delta_i^{(l)} = \left(\sum_k w_{ki}^{(l+1)} \delta_k^{(l+1)}\right) \cdot \sigma'(z_i^{(l)})
$$

## 13.2.2.4 递归网络

### 13.2.2.4.1 联想记忆

**定义 13.2.2.5** (Hopfield网络)

Hopfield网络是能量基模型，状态 $\mathbf{s} \in \{-1, +1\}^n$：

$$
E(\mathbf{s}) = -\frac{1}{2}\sum_{i,j} w_{ij} s_i s_j - \sum_i b_i s_i
$$

**定理 13.2.2.4** (Hopfield存储容量)

具有 $n$ 个单元的Hopfield网络可稳定存储约 $0.14n$ 个随机模式。

**证明概要**:

考虑存储模式 $\{\mathbf{p}^\mu\}_{\mu=1}^{M}$。Hebb学习规则：

$$
w_{ij} = \frac{1}{n}\sum_{\mu=1}^{M} p_i^\mu p_j^\mu \quad (i \neq j)
$$

稳定性分析涉及信号项与噪声项的平衡。

### 13.2.2.4.2 内容寻址记忆

**定理 13.2.2.5** (内容寻址性)

给定部分输入 $\tilde{\mathbf{p}}$，网络通过能量最小化收敛至最相似存储模式：

$$
\mathbf{s}(t+1) = \text{sgn}(\mathbf{W}\mathbf{s}(t))
$$

## 13.2.2.5 认知建模应用

### 13.2.2.5.1 联结主义与符号系统的争论

**Smolensky的和谐性**:

联结主义提供**亚符号**（subsymbolic）层面的解释，位于符号认知与神经实现之间。

| 层面 | 描述 | 实例 |
|------|------|------|
| 符号层面 | 显式规则 | "if A then B" |
| 亚符号层面 | 激活模式 | 分布式向量表征 |
| 神经层面 | 神经元活动 | 脉冲发放 |

### 13.2.2.5.2 语言处理模型

**定义 13.2.2.6** (简单循环网络SRN)

SRN在隐藏层保留上下文信息：

$$
\mathbf{h}_t = \sigma(\mathbf{W}_{xh}\mathbf{x}_t + \mathbf{W}_{hh}\mathbf{h}_{t-1} + \mathbf{b}_h)
$$

**定理 13.2.2.6** (序列学习)

SRN可通过梯度下降学习有限状态语法（正则语言）。

## 13.2.2.6 形式化认知理论

### 13.2.2.6.1 知识表征

**定义 13.2.2.7** (默示知识)

网络权重编码的**默示知识**（tacit knowledge）：

$$
\mathcal{K}_{\text{implicit}} = \{(\mathbf{x}, \mathbf{y}) \mid \mathbf{y} = f_{\mathbf{W}}(\mathbf{x})\}
$$

### 13.2.2.6.2 概念学习

**定义 13.2.2.8** (原型表征)

概念的原型为训练样本的隐式平均：

$$
\mathbf{p}_{\text{prototype}} \approx \frac{1}{N}\sum_{i=1}^{N} \mathbf{a}(\mathbf{x}_i)
$$

"""
联结主义认知模型实现
基于Rumelhart & McClelland (1986) PDP框架
"""

import numpy as np
from typing import List, Tuple, Callable, Optional
from dataclasses import dataclass
import matplotlib.pyplot as plt
from abc import ABC, abstractmethod


class ActivationFunction:
    """激活函数及其导数"""

    @staticmethod
    def sigmoid(x: np.ndarray) -> np.ndarray:
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))

    @staticmethod
    def sigmoid_derivative(x: np.ndarray) -> np.ndarray:
        s = ActivationFunction.sigmoid(x)
        return s * (1 - s)

    @staticmethod
    def tanh(x: np.ndarray) -> np.ndarray:
        return np.tanh(x)

    @staticmethod
    def tanh_derivative(x: np.ndarray) -> np.ndarray:
        return 1 - np.tanh(x) ** 2

    @staticmethod
    def relu(x: np.ndarray) -> np.ndarray:
        return np.maximum(0, x)

    @staticmethod
    def relu_derivative(x: np.ndarray) -> np.ndarray:
        return (x > 0).astype(float)

    @staticmethod
    def linear(x: np.ndarray) -> np.ndarray:
        return x

    @staticmethod
    def linear_derivative(x: np.ndarray) -> np.ndarray:
        return np.ones_like(x)


class NeuralNetwork:
    """
    多层前馈神经网络
    实现反向传播学习
    """

    def __init__(self,
                 layer_sizes: List[int],
                 activation: str = 'sigmoid',
                 learning_rate: float = 0.1,
                 momentum: float = 0.9):
        """
        初始化神经网络

        Args:
            layer_sizes: 各层单元数 [输入, 隐藏..., 输出]
            activation: 激活函数类型
            learning_rate: 学习率 η
            momentum: 动量系数 α
        """
        self.layer_sizes = layer_sizes
        self.n_layers = len(layer_sizes)
        self.lr = learning_rate
        self.momentum = momentum

        # 选择激活函数
        self.activation = getattr(ActivationFunction, activation)
        self.activation_deriv = getattr(ActivationFunction,
                                       f"{activation}_derivative")

        # 初始化权重 (Xavier初始化)
        self.weights = []
        self.biases = []
        for i in range(self.n_layers - 1):
            w = np.random.randn(layer_sizes[i], layer_sizes[i+1]) * \
                np.sqrt(2.0 / layer_sizes[i])
            b = np.zeros((1, layer_sizes[i+1]))
            self.weights.append(w)
            self.biases.append(b)

        # 动量项
        self.weight_velocities = [np.zeros_like(w) for w in self.weights]
        self.bias_velocities = [np.zeros_like(b) for b in self.biases]

        # 存储前向传播结果
        self.activations = []
        self.z_values = []

    def forward(self, X: np.ndarray) -> np.ndarray:
        """
        前向传播

        Args:
            X: 输入矩阵 (n_samples, n_features)

        Returns:
            输出激活 (n_samples, n_outputs)
        """
        self.activations = [X]
        self.z_values = []

        current = X
        for w, b in zip(self.weights, self.biases):
            z = np.dot(current, w) + b
            self.z_values.append(z)
            current = self.activation(z)
            self.activations.append(current)

        return current

    def backward(self, X: np.ndarray, y: np.ndarray) -> float:
        """
        反向传播

        Returns:
            损失值
        """
        n_samples = X.shape[0]

        # 前向传播
        output = self.forward(X)

        # 计算损失 (MSE)
        loss = np.mean((output - y) ** 2)

        # 反向传播误差
        deltas = []

        # 输出层误差
        delta = (output - y) * self.activation_deriv(self.z_values[-1])
        deltas.append(delta)

        # 隐藏层误差 (反向传播)
        for i in range(self.n_layers - 3, -1, -1):
            delta = np.dot(deltas[-1], self.weights[i+1].T) * \
                   self.activation_deriv(self.z_values[i])
            deltas.append(delta)

        deltas.reverse()

        # 更新权重 (带动量)
        for i in range(len(self.weights)):
            grad_w = np.dot(self.activations[i].T, deltas[i]) / n_samples
            grad_b = np.mean(deltas[i], axis=0, keepdims=True)

            # 动量更新
            self.weight_velocities[i] = \
                self.momentum * self.weight_velocities[i] - self.lr * grad_w
            self.bias_velocities[i] = \
                self.momentum * self.bias_velocities[i] - self.lr * grad_b

            self.weights[i] += self.weight_velocities[i]
            self.biases[i] += self.bias_velocities[i]

        return loss

    def train(self, X: np.ndarray, y: np.ndarray,
              epochs: int = 1000, verbose: int = 100) -> List[float]:
        """训练网络"""
        losses = []

        for epoch in range(epochs):
            loss = self.backward(X, y)
            losses.append(loss)

            if verbose > 0 and epoch % verbose == 0:
                print(f"Epoch {epoch}: Loss = {loss:.6f}")

        return losses

    def predict(self, X: np.ndarray) -> np.ndarray:
        """预测输出"""
        return self.forward(X)


class HopfieldNetwork:
    """
    Hopfield联想记忆网络
    内容寻址记忆的形式化实现
    """

    def __init__(self, n_units: int):
        """
        初始化Hopfield网络

        Args:
            n_units: 单元数 (神经元数量)
        """
        self.n = n_units
        self.weights = np.zeros((n_units, n_units))
        self.patterns: List[np.ndarray] = []

    def train_hebbian(self, patterns: List[np.ndarray]):
        """
        Hebbian学习规则
        w_ij = (1/n) Σ_μ p_i^μ p_j^μ
        """
        self.patterns = patterns
        self.weights = np.zeros((self.n, self.n))

        for p in patterns:
            p = p.reshape(-1, 1)
            self.weights += np.dot(p, p.T)

        self.weights /= self.n
        np.fill_diagonal(self.weights, 0)  # 无自连接

    def update_sync(self, state: np.ndarray,
                    max_iter: int = 100) -> Tuple[np.ndarray, int]:
        """
        同步更新
        s(t+1) = sign(W · s(t))
        """
        current = state.copy()

        for i in range(max_iter):
            new_state = np.sign(np.dot(self.weights, current))
            new_state[new_state == 0] = 1  # 零值处理

            if np.array_equal(new_state, current):
                return current, i + 1

            current = new_state

        return current, max_iter

    def update_async(self, state: np.ndarray,
                     max_iter: int = 1000) -> Tuple[np.ndarray, int]:
        """
        异步更新 (随机顺序)
        """
        current = state.copy()
        n_iter = 0

        for _ in range(max_iter):
            # 随机选择更新单元
            for i in np.random.permutation(self.n):
                activation = np.dot(self.weights[i], current)
                current[i] = 1 if activation >= 0 else -1
                n_iter += 1

                # 检查是否收敛到存储模式
                for p in self.patterns:
                    if np.array_equal(current, p):
                        return current, n_iter

        return current, n_iter

    def energy(self, state: np.ndarray) -> float:
        """
        计算网络能量
        E = -0.5 Σ_ij w_ij s_i s_j
        """
        return -0.5 * np.dot(state, np.dot(self.weights, state))

    def capacity_analysis(self, n_patterns: int,
                         n_trials: int = 100) -> float:
        """
        分析存储容量
        返回成功回忆的比例
        """
        success = 0

        for _ in range(n_trials):
            # 生成随机模式
            patterns = [np.random.choice([-1, 1], self.n)
                       for _ in range(n_patterns)]

            self.train_hebbian(patterns)

            # 测试回忆
            test_pattern = patterns[0].copy()
            # 添加噪声
            noise_indices = np.random.choice(self.n, self.n // 10, replace=False)
            test_pattern[noise_indices] *= -1

            recalled, _ = self.update_sync(test_pattern)

            if np.array_equal(recalled, patterns[0]):
                success += 1

        return success / n_trials


class SimpleRecurrentNetwork:
    """
    简单循环网络 (SRN / Elman网络)
    用于序列学习
    """

    def __init__(self,
                 input_size: int,
                 hidden_size: int,
                 output_size: int,
                 learning_rate: float = 0.01):
        self.input_size = input_size
        self.hidden_size = hidden_size
        self.output_size = output_size
        self.lr = learning_rate

        # 权重初始化
        self.W_xh = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hh = np.random.randn(hidden_size, hidden_size) * 0.01
        self.W_hy = np.random.randn(hidden_size, output_size) * 0.01

        self.b_h = np.zeros(hidden_size)
        self.b_y = np.zeros(output_size)

        # 状态
        self.h_prev = np.zeros(hidden_size)

    def forward_step(self, x: np.ndarray,
                     h_prev: Optional[np.ndarray] = None) -> Tuple[np.ndarray, np.ndarray]:
        """
        单步前向传播

        h_t = tanh(W_xh · x_t + W_hh · h_{t-1} + b_h)
        y_t = W_hy · h_t + b_y
        """
        if h_prev is None:
            h_prev = self.h_prev

        h = np.tanh(np.dot(x, self.W_xh) +
                   np.dot(h_prev, self.W_hh) + self.b_h)
        y = np.dot(h, self.W_hy) + self.b_y

        return y, h

    def forward_sequence(self, X: List[np.ndarray]) -> List[np.ndarray]:
        """序列前向传播"""
        outputs = []
        h = np.zeros(self.hidden_size)

        for x in X:
            y, h = self.forward_step(x, h)
            outputs.append(y)

        return outputs

    def train_sequence(self, X: List[np.ndarray],
                      targets: List[np.ndarray],
                      epochs: int = 100):
        """BPTT训练"""
        for epoch in range(epochs):
            total_loss = 0

            # 前向传播并存储中间结果
            h_states = [np.zeros(self.hidden_size)]
            outputs = []

            for x in X:
                h = np.tanh(np.dot(x, self.W_xh) +
                           np.dot(h_states[-1], self.W_hh) + self.b_h)
                y = np.dot(h, self.W_hy) + self.b_y
                h_states.append(h)
                outputs.append(y)

            # BPTT反向传播
            dh_next = np.zeros(self.hidden_size)

            for t in reversed(range(len(X))):
                # 输出误差
                dy = outputs[t] - targets[t]
                total_loss += np.sum(dy ** 2)

                # 梯度
                dW_hy = np.outer(h_states[t+1], dy)
                db_y = dy

                dh = np.dot(dy, self.W_hy.T) + dh_next
                dh_raw = dh * (1 - h_states[t+1] ** 2)  # tanh导数

                dW_xh = np.outer(X[t], dh_raw)
                dW_hh = np.outer(h_states[t], dh_raw)
                db_h = dh_raw

                dh_next = np.dot(dh_raw, self.W_hh.T)

                # 更新
                self.W_hy -= self.lr * dW_hy
                self.b_y -= self.lr * db_y
                self.W_xh -= self.lr * dW_xh
                self.W_hh -= self.lr * dW_hh
                self.b_h -= self.lr * db_h

            if epoch % 10 == 0:
                print(f"Epoch {epoch}: Loss = {total_loss:.4f}")

class PastTenseLearner:
    """
    过去式学习模型
    基于Rumelhart & McClelland (1986)
    """

    def __init__(self, phoneme_size: int = 30):
        self.net = NeuralNetwork(
            layer_sizes=[phoneme_size * 3, 50, phoneme_size * 3],
            activation='sigmoid',
            learning_rate=0.01
        )

    def encode_word(self, word: str, phonemes: List[str]) -> np.ndarray:
        """Wickelfeature编码（简化版）"""
        # 简化的音素编码
        encoding = np.zeros(len(phonemes) * 3)
        for i, p in enumerate(word):
            if p in phonemes:
                idx = phonemes.index(p)
                encoding[idx] = 1
        return encoding

    def learn(self, pairs: List[Tuple[str, str]], phonemes: List[str]]):
        """学习动词过去式映射"""
        X = np.array([self.encode_word(w, phonemes) for w, _ in pairs])
        y = np.array([self.encode_word(p, phonemes) for _, p in pairs])

        return self.net.train(X, y, epochs=500, verbose=50)

    print(f"  带噪声模式收敛于 {steps} 步")
    print(f"  正确回忆: {np.array_equal(recalled, patterns[0])}")

    # 容量分析
    print("\n  容量分析:")
    for n_p in [1, 2, 3, 5, 8]:
        accuracy = hopfield.capacity_analysis(n_p, n_trials=50)
        print(f"    {n_p} 模式: {accuracy*100:.1f}% 成功率")
