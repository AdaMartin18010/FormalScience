# 13.4.3 类比推理

## 13.4.3.1 引言

类比推理是人类认知的核心能力，通过发现不同领域间的结构相似性来迁移知识和解决问题。
本节基于Dedre Gentner的结构映射理论（Structure-Mapping Theory），形式化类比推理的结构、过程与约束。

> **参考**: Gentner, D. (1983). "Structure-mapping: A theoretical framework for analogy". _Cognitive Science_.
>
> Gentner, D., & Markman, A. B. (1997). "Structure mapping in analogy and similarity". _American Psychologist_.

## 13.4.3.2 结构映射理论

### 13.4.3.2.1 知识表征

**定义 13.4.3.1** (谓词演算表征)

知识表示为一阶谓词演算：

| 谓词类型 | 语法 | 示例 |
|----------|------|------|
| **实体** | 对象常元 | $a, b, c$ |
| **属性** | $P(x)$ | $RED(x), LARGE(y)$ |
| **关系** | $R(x, y)$ | $CAUSE(x, y), GREATER(x, y)$ |
| **高阶关系** | $R(R_1, R_2)$ | $CAUSE(COLLIDE(x,y), BREAK(y))$ |

### 13.4.3.2.2 结构映射

**定义 13.4.3.2** (类比映射)

从**基**（base）到**目标**（target）的映射 $M$：

$$
M: \mathcal{B} \rightarrow \mathcal{T}
$$

满足：

1. **一一对应**: $M$ 是单射
2. **平行连通性**: 若 $R(x, y) \in \mathcal{B}$，则 $R(M(x), M(y)) \in \mathcal{T}$
3. **系统性**: 优先映射关系连接的关系

## 13.4.3.3 类比约束

### 13.4.3.3.1 结构一致性

**定义 13.4.3.3** (结构一致性)

映射 $M$ 是结构一致的当且仅当：

$$
\forall R, x, y: R(x, y) \in \mathcal{B} \Rightarrow R(M(x), M(y)) \in \mathcal{T}
$$

### 13.4.3.3.2 系统性原则

**原则 13.4.3.1** (系统性原则)

好的类比应映射**相互连接的关系系统**而非孤立特性。

**形式化**: 设 $S$ 为关系系统（关系的连通子图），映射质量：

$$
\text{Quality}(M) = \sum_{S \subseteq \mathcal{B}} w(S) \cdot |S| \cdot \mathbb{1}_{M(S) \subseteq \mathcal{T}}
$$

其中 $w(S)$ 随关系阶数递增。

### 13.4.3.3.3 一一映射约束

**约束 13.4.3.1** (唯一性)

$$
\forall x, y \in \text{dom}(M): x \neq y \Rightarrow M(x) \neq M(y)
$$

## 13.4.3.4 SME算法

### 13.4.3.4.1 算法步骤

**定义 13.4.3.4** (SME - Structure Mapping Engine)

Falkenhainer、Forbus和Gentner的SME算法：

**步骤 1**: **局部匹配**

找出基和目标间的候选对应：

$$
\text{Candidates} = \{(b, t) \mid \text{Match}(b, t) > \theta\}
$$

**步骤 2**: **结构合并**

构建连接一致的对应（gmap）：

$$
\text{GM} = \{M \subseteq \text{Candidates} \mid \text{StructurallyConsistent}(M)\}
$$

**步骤 3**: **评估与选择**

$$
M^* = \arg\max_{M \in \text{GM}} \text{Evaluate}(M)
$$

### 13.4.3.4.2 评估函数

**定义 13.4.3.5** (评估函数)

$$
\text{Evaluate}(M) = \sum_{r \in \text{Relations}(M)} \text{Importance}(r) + \alpha \cdot \text{Systematicity}(M)
$$

## 13.4.3.5 类比类型

### 13.4.3.5.1 字面相似

**定义 13.4.3.6** (字面相似)

基和目标共享对象属性和关系：

$$
\text{LiteralSimilarity} = \text{Attributes}(\mathcal{B} \cap \mathcal{T}) + \text{Relations}(\mathcal{B} \cap \mathcal{T})
$$

### 13.4.3.5.2 真类比

**定义 13.4.3.7** (真类比)

共享关系结构但对象不同：

$$
\text{TrueAnalogy}: \text{Relations}(\mathcal{B} \cap \mathcal{T}) \gg \text{Attributes}(\mathcal{B} \cap \mathcal{T})
$$

### 13.4.3.5.3 语用类比

**定义 13.4.3.8** (语用类比)

基于特定目的的映射：

$$
M_{pragmatic} = \arg\max_M \text{Relevance}(M, \text{Goal})
$$

## 13.4.3.6 隐喻与概念变化

### 13.4.3.6.1 隐喻的理解

**定义 13.4.3.9** (概念隐喻)

Lakoff & Johnson的概念隐喻：

$$
\text{Source Domain} \xrightarrow{\text{MAPPING}} \text{Target Domain}
$$

**示例**: "时间是金钱"隐喻

$$
\begin{aligned}
&\text{MONEY} \rightarrow \text{TIME} \\
&\text{SPEND}(\text{money}) \rightarrow \text{SPEND}(\text{time}) \\
&\text{WASTE}(\text{money}) \rightarrow \text{WASTE}(\text{time})
\end{aligned}
$$

### 13.4.3.6.2 通过类比学习

**定理 13.4.3.1** (类比学习)

通过类比获得的新知识：

$$
\mathcal{T}_{new} = \mathcal{T}_{old} \cup \{M(R) \mid R \in \mathcal{B}_{candidate}, \text{Consistent}(M(R), \mathcal{T})\}
$$

"""
类比推理的形式化实现
基于Gentner的结构映射理论 (Structure-Mapping Theory)
"""

from typing import List, Dict, Set, Tuple, Optional, Any
from dataclasses import dataclass, field
from collections import defaultdict
import itertools
from copy import deepcopy


@dataclass
class Entity:
    """领域中的实体/对象"""
    name: str
    attributes: Dict[str, Any] = field(default_factory=dict)

    def __hash__(self):
        return hash(self.name)

    def __eq__(self, other):
        return isinstance(other, Entity) and self.name == other.name


@dataclass
class Relation:
    """
    关系谓词

    可以是：
    - 一阶关系: P(entity)
    - 二阶关系: R(entity1, entity2)
    - 高阶关系: R(Relation1, Relation2)
    """
    name: str
    arguments: Tuple[Any, ...]
    order: int = 1  # 关系阶数

    def __hash__(self):
        return hash((self.name, self.arguments))

    def __eq__(self, other):
        return (isinstance(other, Relation) and
                self.name == other.name and
                self.arguments == other.arguments)

    def __str__(self):
        args_str = ', '.join(str(a) for a in self.arguments)
        return f"{self.name}({args_str})"


class Domain:
    """
    知识领域
    包含实体和关系
    """

    def __init__(self, name: str):
        self.name = name
        self.entities: Set[Entity] = set()
        self.relations: Set[Relation] = set()
        self.attributes: Dict[Entity, Dict[str, Any]] = {}

    def add_entity(self, entity: Entity):
        """添加实体"""
        self.entities.add(entity)

    def add_relation(self, relation: Relation):
        """添加关系"""
        self.relations.add(relation)

        # 提取实体参数
        for arg in relation.arguments:
            if isinstance(arg, Entity):
                self.entities.add(arg)

    def get_relations_involving(self, entity: Entity) -> Set[Relation]:
        """获取涉及特定实体的所有关系"""
        result = set()
        for rel in self.relations:
            if entity in rel.arguments:
                result.add(rel)
        return result

    def get_higher_order_relations(self) -> Set[Relation]:
        """获取高阶关系（关系之间的关系）"""
        return {r for r in self.relations if r.order > 1}

    def to_dict(self) -> Dict:
        """转换为字典表示"""
        return {
            'name': self.name,
            'entities': [e.name for e in self.entities],
            'relations': [str(r) for r in self.relations]
        }


class Mapping:
    """
    类比映射
    从基到目标的对应
    """

    def __init__(self, base: Domain, target: Domain):
        self.base = base
        self.target = target
        self.entity_map: Dict[Entity, Entity] = {}
        self.relation_map: Dict[Relation, Relation] = {}
        self.score: float = 0.0

    def add_entity_correspondence(self,
                                   base_entity: Entity,
                                   target_entity: Entity):
        """添加实体对应"""
        self.entity_map[base_entity] = target_entity

    def add_relation_correspondence(self,
                                     base_relation: Relation,
                                     target_relation: Relation):
        """添加关系对应"""
        self.relation_map[base_relation] = target_relation

    def is_one_to_one(self) -> bool:
        """检查是否满足一一对应约束"""
        target_values = set(self.entity_map.values())
        return len(target_values) == len(self.entity_map)

    def check_parallel_connectivity(self) -> bool:
        """
        检查平行连通性约束
        如果R(a,b)在基中，则应该有R(M(a), M(b))在目标中
        """
        for base_rel, target_rel in self.relation_map.items():
            # 检查参数是否一致映射
            for i, base_arg in enumerate(base_rel.arguments):
                if isinstance(base_arg, Entity):
                    if base_arg in self.entity_map:
                        expected_target = self.entity_map[base_arg]
                        actual_target = target_rel.arguments[i]
                        if expected_target != actual_target:
                            return False
        return True

    def get_systematicity_score(self) -> float:
        """
        计算系统性得分
        奖励相互连接的关系系统
        """
        score = 0.0

        # 统计高阶关系
        for base_rel, target_rel in self.relation_map.items():
            # 基础分数
            score += 1.0

            # 高阶关系奖励
            if base_rel.order > 1:
                score += base_rel.order * 2.0

            # 连通性奖励
            # 检查此关系是否与其他映射关系相连
            connected = False
            for other_base_rel in self.relation_map.keys():
                if other_base_rel != base_rel:
                    # 共享参数?
                    if set(base_rel.arguments) & set(other_base_rel.arguments):
                        connected = True
                        score += 0.5

            if connected:
                score += 1.0

        return score

    def evaluate(self) -> float:
        """评估映射质量"""
        if not self.is_one_to_one():
            return 0.0

        if not self.check_parallel_connectivity():
            return 0.0

        # 系统性得分
        systematicity = self.get_systematicity_score()

        # 结构一致性奖励
        consistency_bonus = 2.0 if self.check_parallel_connectivity() else 0.0

        self.score = systematicity + consistency_bonus
        return self.score

    def project_inference(self) -> Set[Relation]:
        """
        从基向目标投影推断
        返回可以添加到目标的新关系
        """
        inferences = set()

        # 找到基中有但未映射到目标的关系
        mapped_base_relations = set(self.relation_map.keys())
        unmapped_relations = self.base.relations - mapped_base_relations

        for base_rel in unmapped_relations:
            # 检查是否所有参数都可以映射
            can_map = True
            new_args = []

            for arg in base_rel.arguments:
                if isinstance(arg, Entity):
                    if arg in self.entity_map:
                        new_args.append(self.entity_map[arg])
                    else:
                        can_map = False
                        break
                elif isinstance(arg, Relation):
                    if arg in self.relation_map:
                        new_args.append(self.relation_map[arg])
                    else:
                        can_map = False
                        break

            if can_map:
                # 创建新关系投影到目标
                new_relation = Relation(base_rel.name, tuple(new_args), base_rel.order)

                # 检查是否已存在于目标中
                if new_relation not in self.target.relations:
                    inferences.add(new_relation)

        return inferences


class StructureMappingEngine:
    """
    结构映射引擎 (SME)
    实现Gentner的SME算法
    """

    def __init__(self, base: Domain, target: Domain):
        self.base = base
        self.target = target
        self.candidates: List[Tuple[Entity, Entity]] = []
        self.gmaps: List[Mapping] = []

    def local_match(self, similarity_threshold: float = 0.0) -> List[Tuple[Entity, Entity]]:
        """
        步骤1: 局部匹配
        找出可能的实体对应
        """
        candidates = []

        for base_entity in self.base.entities:
            for target_entity in self.target.entities:
                # 基于属性和关系的相似性
                similarity = self._compute_similarity(base_entity, target_entity)

                if similarity >= similarity_threshold:
                    candidates.append((base_entity, target_entity))

        self.candidates = candidates
        return candidates

    def _compute_similarity(self, e1: Entity, e2: Entity) -> float:
        """计算两个实体的相似性"""
        # 基于属性的相似性
        attr_sim = 0.0

        common_attrs = set(e1.attributes.keys()) & set(e2.attributes.keys())
        if common_attrs:
            matches = sum(1 for a in common_attrs
                        if e1.attributes[a] == e2.attributes[a])
            attr_sim = matches / len(common_attrs)

        # 基于关系的相似性
        rel1 = self.base.get_relations_involving(e1)
        rel2 = self.target.get_relations_involving(e2)

        rel_sim = 0.0
        if rel1 and rel2:
            # 关系名称匹配
            names1 = {r.name for r in rel1}
            names2 = {r.name for r in rel2}
            rel_sim = len(names1 & names2) / max(len(names1), len(names2))

        return 0.3 * attr_sim + 0.7 * rel_sim

    def structurally_consistent(self, mapping: Mapping) -> bool:
        """检查映射的结构一致性"""
        return mapping.is_one_to_one() and mapping.check_parallel_connectivity()

    def build_gmaps(self, max_mappings: int = 10) -> List[Mapping]:
        """
        步骤2: 构建全局映射 (gmaps)

        使用启发式搜索找到结构一致的映射
        """
        gmaps = []

        # 按实体相似性排序候选
        sorted_candidates = sorted(
            self.candidates,
            key=lambda x: self._compute_similarity(x[0], x[1]),
            reverse=True
        )

        # 贪心构建映射
        for seed in sorted_candidates[:5]:  # 从多个种子开始
            mapping = Mapping(self.base, self.target)
            mapping.add_entity_correspondence(seed[0], seed[1])

            # 扩展映射
            self._expand_mapping(mapping, sorted_candidates)

            # 添加关系对应
            self._match_relations(mapping)

            if self.structurally_consistent(mapping):
                mapping.evaluate()
                gmaps.append(mapping)

        # 按得分排序
        gmaps.sort(key=lambda m: m.score, reverse=True)

        self.gmaps = gmaps[:max_mappings]
        return self.gmaps

    def _expand_mapping(self, mapping: Mapping,
                       candidates: List[Tuple[Entity, Entity]]):
        """扩展实体映射"""
        added = True
        while added:
            added = False
            for base_e, target_e in candidates:
                if base_e in mapping.entity_map:
                    continue
                if target_e in mapping.entity_map.values():
                    continue

                # 检查是否有助于关系匹配
                for base_rel in self.base.relations:
                    if base_e in base_rel.arguments:
                        # 检查是否有对应关系
                        for target_rel in self.target.relations:
                            if (target_rel.name == base_rel.name and
                                target_e in target_rel.arguments):
                                mapping.add_entity_correspondence(base_e, target_e)
                                added = True
                                break
                    if added:
                        break

    def _match_relations(self, mapping: Mapping):
        """匹配关系"""
        for base_rel in self.base.relations:
            # 寻找可以对应的目标关系
            for target_rel in self.target.relations:
                if base_rel.name != target_rel.name:
                    continue
                if len(base_rel.arguments) != len(target_rel.arguments):
                    continue

                # 检查参数对应
                matches = True
                for i, base_arg in enumerate(base_rel.arguments):
                    target_arg = target_rel.arguments[i]

                    if isinstance(base_arg, Entity):
                        if base_arg in mapping.entity_map:
                            if mapping.entity_map[base_arg] != target_arg:
                                matches = False
                                break
                        else:
                            matches = False
                            break

                if matches:
                    mapping.add_relation_correspondence(base_rel, target_rel)

    def select_best_mapping(self) -> Optional[Mapping]:
        """
        步骤3: 选择最佳映射
        """
        if not self.gmaps:
            return None
        return self.gmaps[0]

    def run(self) -> Optional[Mapping]:
        """运行完整的SME算法"""
        self.local_match()
        self.build_gmaps()
        return self.select_best_mapping()


class AnalogyReasoning:
    """
    类比推理系统
    包含经典的类比问题示例
    """

    @staticmethod
    def create_solar_system_analogy() -> Tuple[Domain, Domain]:
        """
        创建太阳系-原子类比
        Rutherford的经典类比
        """
        # 基：太阳系
        solar = Domain("Solar System")

        sun = Entity("Sun", {"massive": True, "central": True})
        planet = Entity("Planet", {"orbits": True})
        gravity = Entity("Gravity")

        solar.add_entity(sun)
        solar.add_entity(planet)
        solar.add_entity(gravity)

        solar.add_relation(Relation("ATTRACTS", (sun, planet)))
        solar.add_relation(Relation("REVOLVES_AROUND", (planet, sun)))
        solar.add_relation(Relation("CAUSE", (Relation("ATTRACTS", (sun, planet)),
                                             Relation("REVOLVES_AROUND", (planet, sun))) ,
                                   order=2))

        # 目标：原子
        atom = Domain("Atom")

        nucleus = Entity("Nucleus", {"massive": True, "central": True, "positive": True})
        electron = Entity("Electron", {"orbits": True, "negative": True})

        atom.add_entity(nucleus)
        atom.add_entity(electron)

        atom.add_relation(Relation("ATTRACTS", (nucleus, electron)))
        atom.add_relation(Relation("REVOLVES_AROUND", (electron, nucleus)))

        return solar, atom

    @staticmethod
    def create_water_flow_electricity() -> Tuple[Domain, Domain]:
        """
        水流-电流类比
        Gentner的经典示例
        """
        # 基：水流
        water = Domain("Water Flow")

        pipe = Entity("Pipe")
        water_substance = Entity("Water")
        pump = Entity("Pump")
        valve = Entity("Valve")

        water.add_entity(pipe)
        water.add_entity(water_substance)
        water.add_entity(pump)
        water.add_entity(valve)

        water.add_relation(Relation("CONTAINS", (pipe, water_substance)))
        water.add_relation(Relation("CAUSE_FLOW", (pump, water_substance)))
        water.add_relation(Relation("IMPEDES", (valve, water_substance)))
        water.add_relation(Relation("FLOW_THROUGH", (water_substance, pipe)))

        # 目标：电流
        electric = Domain("Electricity")

        wire = Entity("Wire")
        current = Entity("Current")
        battery = Entity("Battery")
        switch = Entity("Switch")

        electric.add_entity(wire)
        electric.add_entity(current)
        electric.add_entity(battery)
        electric.add_entity(switch)

        electric.add_relation(Relation("CONTAINS", (wire, current)))
        electric.add_relation(Relation("CAUSE_FLOW", (battery, current)))

        return water, electric
