## 13.4.1.1 引言

演绎推理是从一般到特殊的推理形式。本节基于心智逻辑（Mental Logic）和心智模型（Mental Models）理论，阐述人类演绎推理的心理现实性形式化。

> **参考**: Rips, L. J. (1994). _The Psychology of Proof: Deductive Reasoning in Human Thinking_. MIT Press.
>
> Johnson-Laird, P. N. (2006). _How We Reason_. Oxford University Press.

## 13.4.1.2 心智逻辑理论

### 13.4.1.2.1 自然推理系统

**定义 13.4.1.1** (自然推理规则)

Braine和O'Brien的自然推理形式：

| 推理规则 | 形式 | 心理难度 |
|----------|------|----------|
| MP (肯定前件) | $p \rightarrow q, p \vdash q$ | 基础 |
| MT (否定后件) | $p \rightarrow q, \neg q \vdash \neg p$ | 中等 |
| AC (肯定后件) | $p \rightarrow q, q \nvdash p$ | 易错 |
| DA (否定前件) | $p \rightarrow q, \neg p \nvdash \neg q$ | 易错 |

### 13.4.1.2.2 推理程序

**定义 13.4.1.2** (推理链)

从前提集 $\Gamma$ 到结论 $\varphi$ 的推理链：

$$
\Gamma = \Gamma_0 \vdash \Gamma_1 \vdash \cdots \vdash \Gamma_n = \Gamma \cup \{\varphi\}
$$

**定理 13.4.1.1** (推理难度)

推理难度与推理链长度正相关：

$$
\text{Difficulty}(\Gamma \vdash \varphi) \propto \min\{|\text{chain}|: \Gamma \vdash_{\text{chain}} \varphi\}
$$

## 13.4.1.3 心智模型理论

### 13.4.1.3.1 模型构建

**定义 13.4.1.3** (心智模型)

命题 $\varphi$ 的心智模型 $\mathcal{M}(\varphi)$ 是满足 $\varphi$ 的可能状态集合：

$$
\mathcal{M}(\varphi) = \{s \in \mathcal{S} \mid s \models \varphi\}
$$

### 13.4.1.3.2 真值表与模型

**定义 13.4.1.4** (合取的模型)

$$
\mathcal{M}(A \land B) = \mathcal{M}(A) \cap \mathcal{M}(B)
$$

**定义 13.4.1.5** (条件句的模型)

Johnson-Laird的三模型条件：

$$
\mathcal{M}(A \rightarrow B) = \{[A, B], [\neg A, B], [\neg A, \neg B]\}
$$

**初始表征**（简化模型）：

$$
\mathcal{M}_0(A \rightarrow B) = \{[A, B]\}
$$

### 13.4.1.3.3 推论过程

**定义 13.4.1.6** (模型融合)

$$
\mathcal{M}(\Gamma) = \bigcap_{\gamma \in \Gamma} \mathcal{M}(\gamma)
$$

**定理 13.4.1.2** (有效性检验)

结论 $C$ 从前提 $\Gamma$ 有效推出当且仅当：

$$
\mathcal{M}(\Gamma) \subseteq \mathcal{M}(C)
$$

**定义 13.4.1.7** (反例搜索)

若 $\mathcal{M}(\Gamma) \not\subseteq \mathcal{M}(C)$，则寻找反例：

$$
\text{Counterexample} = \mathcal{M}(\Gamma) \setminus \mathcal{M}(C)
$$

## 13.4.1.4 三段论推理

### 13.4.1.4.1 三段论结构

**定义 13.4.1.8** (直言命题)

| 类型 | 形式 | 模型数 |
|------|------|--------|
| A (全称肯定) | 所有 $M$ 是 $P$ | 2 |
| E (全称否定) | 没有 $M$ 是 $P$ | 1 |
| I (特称肯定) | 有些 $M$ 是 $P$ | 1 |
| O (特称否定) | 有些 $M$ 不是 $P$ | 2 |

### 13.4.1.4.2 图示方法

**定义 13.4.1.9** (Euler圈)

Euler图表示集合关系：

```
M ⊂ P:    M
         /  \
        ◯    ◯
              P

M ∩ P = ∅:  M     P
           ◯     ◯
```

**定理 13.4.1.3** (三段论有效性)

有效三段论满足：

1. 恰好三个项（大项、小项、中项）
2. 中项周延至少一次
3. 结论中周延的项在前提中周延

## 13.4.1.5 条件推理

### 13.4.1.5.1 条件句语义

**定义 13.4.1.10** (心理模型扩展)

充分条件句的完整模型集：

$$
\begin{aligned}
A \rightarrow B: \quad &[A] \quad [B] \\
                 &[\neg A] \quad [B] \\
                 &[\neg A] \quad [\neg B]
\end{aligned}
$$

### 13.4.1.5.2 推理偏差的形式解释

**定理 13.4.1.4** (肯定后件谬误)

$AC$ 谬误源于使用初始模型而非完整模型：

$$
\mathcal{M}_0(A \rightarrow B) = \{[A, B]\}, \mathcal{M}(B) = \{[A, B], [\neg A, B]\}
$$

由于 $\mathcal{M}_0(A \rightarrow B) \cap \mathcal{M}(B) = \{[A, B]\}$，被误认为支持 $A$。

## 13.4.1.6 双过程理论

### 13.4.1.6.1 系统1与系统2

**定义 13.4.1.11** (双过程模型)

| 特征 | 系统1 (直觉) | 系统2 (分析) |
|------|-------------|--------------|
| 速度 | 快 | 慢 |
| 意识 | 无意识 | 有意识 |
| 努力 | 低认知负荷 | 高认知负荷 |

### 13.4.1.6.2 认知反射

**定义 13.4.1.12** (认知反射测试)

Frederick的CRT：

> 球拍和球共11元，球拍比球贵10元，球多少钱？

直觉答案：10元（错误）
分析答案：0.5元（正确）

CRT分数与演绎推理准确性正相关。

"""
演绎推理形式模型
基于心智逻辑 (Braine, Rips) 和心智模型 (Johnson-Laird)
"""

from typing import Set, List, Dict, Tuple, Optional, FrozenSet
from dataclasses import dataclass, field
from enum import Enum, auto
from itertools import combinations
import re


class PropositionType(Enum):
    """命题类型"""
    ATOMIC = auto()
    NEGATION = auto()
    CONJUNCTION = auto()
    DISJUNCTION = auto()
    CONDITIONAL = auto()
    UNIVERSAL = auto()  # 全称
    EXISTENTIAL = auto()  # 存在


@dataclass(frozen=True)
class Proposition:
    """
    命题的递归定义
    """
    ptype: PropositionType
    content: str
    components: Tuple['Proposition', ...] = field(default_factory=tuple)

    def __hash__(self):
        return hash((self.ptype, self.content, self.components))

    def __str__(self):
        if self.ptype == PropositionType.ATOMIC:
            return self.content
        elif self.ptype == PropositionType.NEGATION:
            return f"¬{self.components[0]}"
        elif self.ptype == PropositionType.CONJUNCTION:
            return f"({self.components[0]} ∧ {self.components[1]})"
        elif self.ptype == PropositionType.DISJUNCTION:
            return f"({self.components[0]} ∨ {self.components[1]})"
        elif self.ptype == PropositionType.CONDITIONAL:
            return f"({self.components[0]} → {self.components[1]})"
        return self.content


class MentalModel:
    """
    心智模型理论实现
    Johnson-Laird的心智模型
    """

    def __init__(self, propositions: List[Proposition]):
        self.propositions = propositions
        self.models: List[Dict[str, Optional[bool]]] = []
        self.mental_models: List[Dict[str, str]] = []

    def build_models(self, exhaustive: bool = False):
        """
        构建心智模型

        Args:
            exhaustive: 是否构建完整模型（系统2）或初始模型（系统1）
        """
        self.models = []
        self.mental_models = []

        for prop in self.propositions:
            models = self._model_of_proposition(prop, exhaustive)
            self.mental_models.extend(models)

    def _model_of_proposition(self, prop: Proposition,
                             exhaustive: bool) -> List[Dict[str, str]]:
        """
        为单个命题构建模型
        """
        models = []

        if prop.ptype == PropositionType.ATOMIC:
            # 原子命题：构建单模型
            models.append({prop.content: True})

        elif prop.ptype == PropositionType.NEGATION:
            # 否定：标记为假
            inner = prop.components[0]
            if inner.ptype == PropositionType.ATOMIC:
                models.append({inner.content: False})

        elif prop.ptype == PropositionType.CONJUNCTION:
            # 合取：两个都为真
            left, right = prop.components
            if (left.ptype == PropositionType.ATOMIC and
                right.ptype == PropositionType.ATOMIC):
                models.append({
                    left.content: True,
                    right.content: True
                })

        elif prop.ptype == PropositionType.CONDITIONAL:
            # 条件句：关键的心理模型
            antecedent = prop.components[0]
            consequent = prop.components[1]

            if (antecedent.ptype == PropositionType.ATOMIC and
                consequent.ptype == PropositionType.ATOMIC):

                # 初始模型（系统1）：只有前件+后件
                models.append({
                    antecedent.content: True,
                    consequent.content: True
                })

                if exhaustive:
                    # 完整模型（系统2）：添加前件假的情况
                    models.append({
                        antecedent.content: False,
                        consequent.content: True
                    })
                    models.append({
                        antecedent.content: False,
                        consequent.content: False
                    })

        return models

    def combine_models(self, other: 'MentalModel') -> 'MentalModel':
        """
        模型融合（对应前提组合）
        """
        combined = MentalModel(self.propositions + other.propositions)

        # 找到兼容的模型对
        new_models = []
        for m1 in self.mental_models:
            for m2 in other.mental_models:
                merged = self._merge_compatible(m1, m2)
                if merged is not None:
                    new_models.append(merged)

        combined.mental_models = new_models
        return combined

    def _merge_compatible(self, m1: Dict, m2: Dict) -> Optional[Dict]:
        """合并两个兼容的模型"""
        merged = dict(m1)
        for key, value in m2.items():
            if key in merged:
                if merged[key] != value:
                    return None  # 不兼容
            else:
                merged[key] = value
        return merged

    def entails(self, conclusion: Proposition) -> Tuple[bool, Optional[Dict]]:
        """
        检验结论是否被蕴涵
        返回 (是否有效, 反例)
        """
        # 构建结论的否定模型
        neg_conclusion = self._negate(conclusion)

        # 检查是否存在相容的反例
        for model in self.mental_models:
            if self._satisfies(model, neg_conclusion):
                return False, model  # 找到反例

        return True, None

    def _negate(self, prop: Proposition) -> Proposition:
        """构建命题的否定"""
        return Proposition(
            PropositionType.NEGATION,
            f"not_{prop.content}",
            (prop,)
        )

    def _satisfies(self, model: Dict, prop: Proposition) -> bool:
        """检查模型是否满足命题"""
        if prop.ptype == PropositionType.ATOMIC:
            return model.get(prop.content, False)
        elif prop.ptype == PropositionType.NEGATION:
            return not self._satisfies(model, prop.components[0])
        return False

    def fleshed_out(self) -> 'MentalModel':
        """
        模型充实（System 2过程）
        从初始模型到完整模型
        """
        new_model = MentalModel(self.propositions)
        new_model.build_models(exhaustive=True)
        return new_model


class SyllogisticReasoning:
    """
    三段论推理
    实现Aristotle的直言三段论
    """

    # 四种直言命题类型
    A_TYPE = "A"  # 所有 M 是 P
    E_TYPE = "E"  # 没有 M 是 P
    I_TYPE = "I"  # 有些 M 是 P
    O_TYPE = "O"  # 有些 M 不是 P

    def __init__(self):
        self.valid_forms = self._init_valid_forms()

    def _init_valid_forms(self) -> Set[str]:
        """初始化有效三段论形式"""
        # 传统有效式 (Barbara, Celarent, Darii, Ferio等)
        return {
            "AAA-1", "EAE-1", "AII-1", "EIO-1",  # 第一格
            "EAE-2", "AEE-2", "EIO-2", "AOO-2",  # 第二格
            "AAI-3", "IAI-3", "AII-3", "EAO-3", "OAO-3", "EIO-3",  # 第三格
            "AAI-4", "AEE-4", "IAI-4", "EAO-4", "EIO-4",  # 第四格
        }

    def parse_proposition(self, text: str) -> Tuple[str, str, str]:
        """
        解析直言命题
        返回 (类型, 主项, 谓项)
        """
        text = text.lower().strip()

        if text.startswith("all") or text.startswith("所有"):
            # A型：所有 S 是 P
            match = re.match(r"all\s+(\w+)\s+(?:is|are)\s+(\w+)", text)
            if match:
                return (self.A_TYPE, match.group(1), match.group(2))

        elif text.startswith("no") or text.startswith("没有"):
            # E型：没有 S 是 P
            match = re.match(r"no\s+(\w+)\s+(?:is|are)\s+(\w+)", text)
            if match:
                return (self.E_TYPE, match.group(1), match.group(2))

        elif text.startswith("some") or text.startswith("有些"):
            if "not" in text or "不是" in text:
                # O型：有些 S 不是 P
                match = re.match(r"some\s+(\w+)\s+(?:is|are)\s+not\s+(\w+)", text)
                if match:
                    return (self.O_TYPE, match.group(1), match.group(2))
            else:
                # I型：有些 S 是 P
                match = re.match(r"some\s+(\w+)\s+(?:is|are)\s+(\w+)", text)
                if match:
                    return (self.I_TYPE, match.group(1), match.group(2))

        return None

    def figure(self, major: Tuple, minor: Tuple) -> int:
        """
        确定三段论格

        第一格: M-P, S-M -> S-P
        第二格: P-M, S-M -> S-P
        第三格: M-P, M-S -> S-P
        第四格: P-M, M-S -> S-P
        """
        _, m1, p = major  # 大前提
        _, s, m2 = minor  # 小前提

        if m1 == m2:  # 中项相同
            if major[1] == m1 and minor[2] == m1:
                return 1
            elif major[2] == m1 and minor[2] == m1:
                return 2
            elif major[1] == m1 and minor[1] == m1:
                return 3
            elif major[2] == m1 and minor[1] == m1:
                return 4
        return 0

    def validate(self, premise1: str, premise2: str,
                conclusion: str) -> bool:
        """
        验证三段论是否有效
        """
        p1 = self.parse_proposition(premise1)
        p2 = self.parse_proposition(premise2)
        conc = self.parse_proposition(conclusion)

        if not all([p1, p2, conc]):
            return False

        # 确定大小前提
        if p1[2] == p2[1] or p1[2] == p2[2]:
            major, minor = p1, p2
        else:
            major, minor = p2, p1

        # 确定格
        fig = self.figure(major, minor)

        # 构建形式
        form = f"{major[0]}{minor[0]}{conc[0]}-{fig}"

        return form in self.valid_forms


class MentalLogic:
    """
    心智逻辑理论 (Braine, O'Brien, Rips)
    """

    def __init__(self):
        # 自然推理规则
        self.rules = {
            'MP': self._modus_ponens,
            'MT': self._modus_tollens,
            'HS': self._hypothetical_syllogism,
            'DS': self._disjunctive_syllogism,
        }

    def _modus_ponens(self, premises: List[Proposition]) -> List[Proposition]:
        """肯定前件式"""
        conclusions = []
        conditionals = [p for p in premises
                       if p.ptype == PropositionType.CONDITIONAL]

        for cond in conditionals:
            antecedent = cond.components[0]
            consequent = cond.components[1]

            # 检查是否有肯定前件
            for p in premises:
                if p == antecedent:
                    conclusions.append(consequent)

        return conclusions

    def _modus_tollens(self, premises: List[Proposition]) -> List[Proposition]:
        """否定后件式"""
        conclusions = []
        conditionals = [p for p in premises
                       if p.ptype == PropositionType.CONDITIONAL]

        for cond in conditionals:
            antecedent = cond.components[0]
            consequent = cond.components[1]

            # 检查是否有否定后件
            for p in premises:
                if (p.ptype == PropositionType.NEGATION and
                    p.components[0] == consequent):
                    # 否定前件
                    neg_ant = Proposition(
                        PropositionType.NEGATION,
                        f"not_{antecedent.content}",
                        (antecedent,)
                    )
                    conclusions.append(neg_ant)

        return conclusions

    def _disjunctive_syllogism(self, premises: List[Proposition]) -> List[Proposition]:
        """选言三段论"""
        conclusions = []
        disjunctions = [p for p in premises
                       if p.ptype == PropositionType.DISJUNCTION]

        for disj in disjunctions:
            left, right = disj.components

            # 检查是否有否定一边
            for p in premises:
                if p.ptype == PropositionType.NEGATION:
                    if p.components[0] == left:
                        conclusions.append(right)
                    elif p.components[0] == right:
                        conclusions.append(left)

        return conclusions

    def reason(self, premises: List[Proposition],
              max_depth: int = 3) -> List[Proposition]:
        """
        执行推理链

        Args:
            premises: 初始前提
            max_depth: 最大推理深度

        Returns:
            所有可推出的结论
        """
        all_conclusions = set(premises)
        current = set(premises)

        for depth in range(max_depth):
            new_conclusions = set()

            for rule_name, rule_fn in self.rules.items():
                conclusions = rule_fn(list(current))
                for c in conclusions:
                    if c not in all_conclusions:
                        new_conclusions.add(c)

            if not new_conclusions:
                break

            all_conclusions.update(new_conclusions)
            current = new_conclusions

        return list(all_conclusions)

class WasonSelectionTask:
    """
    Wason选择任务
    测试条件推理
    """

    def __init__(self, rule: str = "If P then Q"):
        self.rule = rule
        self.cards = ['P', 'not P', 'Q', 'not Q']
        self.correct_selection = {'P', 'not Q'}  # MP + MT

    def analyze_response(self, selected: Set[str]) -> Dict:
        """
        分析被试的选择
        """
        # 匹配偏差分析
        matching_bias = 'Q' in selected and 'not Q' not in selected

        # 逻辑正确性
        logically_correct = selected == self.correct_selection

        # 常见错误模式
        if selected == {'P', 'Q'}:
            error_type = "Verification bias (确认偏误)"
        elif selected == {'P'}:
            error_type = "Only MP"
        elif 'not P' in selected:
            error_type = "DA fallacy (否定前件谬误)"
        elif 'Q' in selected and 'not Q' not in selected:
            error_type = "AC fallacy (肯定后件谬误)"
        else:
            error_type = "Other"

        return {
            'logically_correct': logically_correct,
            'matching_bias': matching_bias,
            'error_type': error_type,
            'selected': selected,
            'correct': self.correct_selection
        }
