## 13.4.2.1 引言

归纳推理是从特殊到一般的推理形式，是科学发现与日常学习的认知基础。本节基于贝叶斯归纳理论和统计学习理论，形式化归纳推理的本质、证据评估与假设选择。

> **参考**: Tenenbaum, J. B., Kemp, C., Griffiths, T. L., & Goodman, N. D. (2011). "How to Grow a Mind: Statistics, Structure, and Abstraction". _Science_.

## 13.4.2.2 归纳问题

### 13.4.2.2.1 Hume的归纳问题

**问题 13.4.2.1** (Hume问题)

从有限观察推导一般规律的合理性能否得到证明？

**形式化表述**:

给定观察 $D = \{x_1, \ldots, x_n\}$，推断一般规律 $H$：

$$
P(H \mid D) = \frac{P(D \mid H) P(H)}{P(D)}
$$

**Hume的挑战**: 先验 $P(H)$ 的来源问题。

### 13.4.2.2.2 Goodman的新归纳之谜

**定义 13.4.2.1** (Grue谓词)

$$
\text{Grue}(x) \iff \begin{cases}
\text{Green}(x) & \text{if observed before } t \\
\text{Blue}(x) & \text{otherwise}
\end{cases}
$$

**问题 13.4.2.2** (投影问题)

为何"所有翡翠是绿色"比"所有翡翠是grue"更可投射？

**贝叶斯解答**: 简单性先验 $P(H_{green}) > P(H_{grue})$

## 13.4.2.3 贝叶斯归纳

### 13.4.2.3.1 层级贝叶斯模型

**定义 13.4.2.2** (层级贝叶斯归纳)

$$
\begin{aligned}
P(H \mid D) &= \int P(H \mid D, \theta) P(\theta \mid D) d\theta \\
P(\theta \mid D) &= \frac{P(D \mid \theta) P(\theta)}{P(D)}
\end{aligned}
$$

### 13.4.2.3.2 大小原理

**定理 13.4.2.1** (大小原理 - Tenenbaum & Griffiths)

假设空间 $\mathcal{H}$ 中的假设越**小**（限制越多），获得一致性证据后的后验概率越高：

$$
P(H_i \mid D) \propto P(D \mid H_i) P(H_i) \propto \frac{1}{|H_i|^{n}} P(H_i)
$$

其中 $|H_i|$ 为假设 $H_i$ 的扩展大小。

### 13.4.2.3.3 收敛定理

**定理 13.4.2.2** (贝叶斯一致性)

在温和条件下，贝叶斯后验随样本量增加几乎必然收敛于真实假设：

$$
P(H^* \mid D_n) \xrightarrow{a.s.} 1 \quad \text{当 } n \rightarrow \infty
$$

## 13.4.2.4 归纳推理的类型

### 13.4.2.4.1 属性归纳

**定义 13.4.2.3** (属性归纳)

从已知对象的属性推断新对象：

$$
P(\text{Robins have property } X \mid \text{Sparrows have } X) > P(X \mid \emptyset)
$$

**相似性效应**:

$$
P(\text{Property projects to } B \mid A \text{ has it}) \propto \text{Sim}(A, B)
$$

### 13.4.2.4.2 关系归纳

**定义 13.4.2.4** (类比归纳)

基于结构相似性的归纳：

$$
P(\phi(B) \mid \phi(A), \text{Sim}_{struct}(A, B)) > P(\phi(B))
$$

### 13.4.2.4.3 因果归纳

**定义 13.4.2.5** (因果归纳)

从共变推断因果：

$$
P(C \rightarrow E \mid \{x_i, y_i\}_{i=1}^n)
$$

## 13.4.2.5 假设生成与选择

### 13.4.2.5.1 假设空间

**定义 13.4.2.6** (良定义假设空间)

$$
\mathcal{H} = \{H_1, H_2, \ldots, H_n\}
$$

**示例**: 数字游戏假设空间

| 假设 | 描述 | 扩展 |
|------|------|------|
| $H_{even}$ | 偶数 | $\{2, 4, 6, 8, \ldots\}$ |
| $H_{odd}$ | 奇数 | $\{1, 3, 5, 7, \ldots\}$ |
| $H_{powers2}$ | 2的幂 | $\{2, 4, 8, 16, \ldots\}$ |
| $H_{mult3}$ | 3的倍数 | $\{3, 6, 9, 12, \ldots\}$ |

### 13.4.2.5.2 奥卡姆剃刀

**定理 13.4.2.3** (奥卡姆剃刀的贝叶斯形式)

简单假设的先验概率更高：

$$
P(H) \propto 2^{-K(H)}
$$

其中 $K(H)$ 为假设的Kolmogorov复杂度。

## 13.4.2.6 Python实现

```python
"""
归纳推理的形式化实现
基于贝叶斯归纳学习和统计学习理论
"""

import numpy as np
from typing import List, Set, Tuple, Dict, Callable, Optional
from dataclasses import dataclass
from collections import defaultdict
from scipy.special import comb, gammaln
from scipy.stats import beta as beta_dist
import matplotlib.pyplot as plt


@dataclass
class Hypothesis:
    """
    归纳假设

    属性:
        name: 假设名称
        extension: 假设的扩展（满足假设的实例集合）
        prior: 先验概率
        complexity: 复杂度度量
    """
    name: str
    extension: Set[int]
    prior: float
    complexity: float

    def likelihood(self, data: Set[int]) -> float:
        """
        计算数据在假设下的似然

        假设数据从扩展中均匀采样
        P(D|H) = 1/|H|^n if D ⊆ H else 0
        """
        if not data.issubset(self.extension):
            return 0.0

        n = len(data)
        return (1.0 / len(self.extension)) ** n if len(self.extension) > 0 else 0.0

    def posterior_predictive(self, x: int,
                            observed: Set[int]) -> float:
        """
        后验预测：P(x ∈ H | D)
        """
        if x in self.extension and observed.issubset(self.extension):
            return 1.0
        elif x not in self.extension and observed.issubset(self.extension):
            return 0.0
        return 0.0


class BayesianInduction:
    """
    贝叶斯归纳推理
    实现Tenenbaum & Griffiths的层级贝叶斯框架
    """

    def __init__(self, hypotheses: List[Hypothesis],
                 domain: Set[int]):
        """
        初始化贝叶斯归纳器

        Args:
            hypotheses: 假设空间
            domain: 全域
        """
        self.hypotheses = hypotheses
        self.domain = domain

        # 归一化先验
        total_prior = sum(h.prior for h in hypotheses)
        for h in hypotheses:
            h.prior /= total_prior

    def compute_posteriors(self, data: Set[int]) -> Dict[str, float]:
        """
        计算给定数据的假设后验

        P(H|D) ∝ P(D|H) * P(H)
        """
        posteriors = {}

        for h in self.hypotheses:
            likelihood = h.likelihood(data)
            posteriors[h.name] = likelihood * h.prior

        # 归一化
        total = sum(posteriors.values())
        if total > 0:
            for name in posteriors:
                posteriors[name] /= total

        return posteriors

    def generalization_probability(self,
                                   query: int,
                                   data: Set[int]) -> float:
        """
        泛化概率：P(query ∈ C | D)

        通过对假设求和
        """
        posteriors = self.compute_posteriors(data)

        prob = 0.0
        for h in self.hypotheses:
            if query in h.extension:
                prob += posteriors[h.name]

        return prob

    def predict_next(self, data: Set[int]) -> Dict[int, float]:
        """
        预测下一个样本

        Returns:
            每个可能值的预测概率
        """
        posteriors = self.compute_posteriors(data)

        predictions = defaultdict(float)

        for h in self.hypotheses:
            if posteriors[h.name] > 0:
                # 在假设扩展中均匀分布
                for x in h.extension:
                    if x not in data:  # 排除已观察的
                        predictions[x] += posteriors[h.name] / len(h.extension)

        # 归一化
        total = sum(predictions.values())
        if total > 0:
            for x in predictions:
                predictions[x] /= total

        return dict(predictions)

    def model_selection(self, data: Set[int],
                       criterion: str = 'map') -> Hypothesis:
        """
        模型选择

        Args:
            criterion: 'map' (最大后验), 'ml' (最大似然), 'bayes' (贝叶斯平均)
        """
        posteriors = self.compute_posteriors(data)

        if criterion == 'map':
            best_name = max(posteriors, key=posteriors.get)
            return next(h for h in self.hypotheses if h.name == best_name)

        elif criterion == 'ml':
            best = max(self.hypotheses, key=lambda h: h.likelihood(data))
            return best

        elif criterion == 'bayes':
            # 返回加权平均（简化：返回MAP）
            return self.model_selection(data, 'map')


class NumberGame:
    """
    数字游戏 (Tenenbaum, 2000)
    归纳推理的经典示例
    """

    def __init__(self, max_number: int = 100):
        self.max_number = max_number
        self.domain = set(range(1, max_number + 1))
        self.hypotheses = self._create_hypotheses()

    def _create_hypotheses(self) -> List[Hypothesis]:
        """创建数字游戏的假设空间"""
        hypotheses = []

        # 1. 数学概念
        math_concepts = [
            ("even", lambda x: x % 2 == 0),
            ("odd", lambda x: x % 2 == 1),
            ("powers_of_2", lambda x: x > 0 and (x & (x-1)) == 0),
            ("powers_of_3", lambda x: x > 0 and 3**int(np.log(x)/np.log(3)) == x),
            ("squares", lambda x: int(np.sqrt(x))**2 == x),
            ("primes", self._is_prime),
        ]

        for name, predicate in math_concepts:
            extension = {x for x in self.domain if predicate(x)}
            complexity = len(name) * 0.1  # 简化的复杂度
            prior = np.exp(-complexity)  # 简单性先验
            hypotheses.append(Hypothesis(name, extension, prior, complexity))

        # 2. 区间假设
        for i in range(1, self.max_number + 1):
            for j in range(i, self.max_number + 1):
                extension = set(range(i, j + 1))
                complexity = 2  # 区间需要两个端点
                prior = 0.001 / self.max_number  # 较低的先验
                hypotheses.append(
                    Hypothesis(f"[{i},{j}]", extension, prior, complexity)
                )

        # 3. 倍数假设
        for k in range(2, 20):
            extension = {x for x in self.domain if x % k == 0}
            complexity = 1.0
            prior = 0.01 / k  # 较大的倍数先验较低
            hypotheses.append(
                Hypothesis(f"multiples_of_{k}", extension, prior, complexity)
            )

        return hypotheses

    def _is_prime(self, n: int) -> bool:
        """素数检验"""
        if n < 2:
            return False
        for i in range(2, int(np.sqrt(n)) + 1):
            if n % i == 0:
                return False
        return True

    def induction_engine(self) -> BayesianInduction:
        """创建归纳引擎"""
        return BayesianInduction(self.hypotheses, self.domain)


class PropertyInduction:
    """
    属性归纳 (Osherson et al., 1990)
    基于相似性的属性投影
    """

    def __init__(self, similarity_matrix: np.ndarray,
                 category_names: List[str]):
        """
        初始化属性归纳模型

        Args:
            similarity_matrix: n×n相似性矩阵
            category_names: 类别名称列表
        """
        self.sim = similarity_matrix
        self.names = category_names
        self.n = len(category_names)

    def premise_conclusion_similarity(self,
                                      premise_idx: int,
                                      conclusion_idx: int) -> float:
        """
        计算前提-结论相似性
        """
        return self.sim[premise_idx, conclusion_idx]

    def induction_strength(self,
                          premises: List[int],
                          conclusion: int,
                          diversity: bool = True) -> float:
        """
        计算归纳强度

        考虑多样性和覆盖性
        """
        if not diversity:
            # 简单平均相似性
            return np.mean([self.sim[p, conclusion] for p in premises])

        # 多样性增强：不同前提的相似性取最大
        max_sims = []
        for p in premises:
            max_sims.append(self.sim[p, conclusion])

        # 多样性：低相似性前提提供更多证据
        diversity_bonus = 1.0
        if len(premises) > 1:
            # 前提间的平均相似性越低，多样性越高
            premise_sim = np.mean([
                self.sim[p1, p2]
                for i, p1 in enumerate(premises)
                for p2 in premises[i+1:]
            ])
            diversity_bonus = 1 - premise_sim

        return np.mean(max_sims) * (1 + 0.5 * diversity_bonus)


class CausalInduction:
    """
    因果归纳
    从共变推断因果
    """

    def __init__(self, alpha: float = 1.0, beta: float = 1.0):
        """
        Beta先验参数
        P(cause strength) ~ Beta(α, β)
        """
        self.alpha = alpha
        self.beta = beta

    def infer_causal_strength(self,
                              n_cause_effect: int,   # a: 原因和结果都出现
                              n_cause_no_effect: int,  # b: 原因出现但结果不出现
                              n_no_cause_effect: int,  # c: 原因不出现但结果出现
                              n_no_cause_no_effect: int) -> Tuple[float, float]:
        """
        使用Beta分布推断因果强度

        Returns:
            (后验均值, 后验方差)
        """
        # 后验参数
        alpha_post = self.alpha + n_cause_effect
        beta_post = self.beta + n_cause_no_effect

        # 后验均值 (因果强度估计)
        mean = alpha_post / (alpha_post + beta_post)

        # 后验方差
        variance = (alpha_post * beta_post) / \
                  ((alpha_post + beta_post)**2 * (alpha_post + beta_post + 1))

        return mean, variance

    def causal_support(self,
                      n_cause_effect: int,
                      n_cause_no_effect: int,
                      n_no_cause_effect: int,
                      n_no_cause_no_effect: int) -> float:
        """
        计算因果支持度 (Griffiths & Tenenbaum, 2005)

        因果 vs 非因果模型的贝叶斯因子
        """
        a, b, c, d = (n_cause_effect, n_cause_no_effect,
                     n_no_cause_effect, n_no_cause_no_effect)
        n = a + b + c + d

        # 因果模型 (两个独立Beta)
        log_p_data_given_causal = \
            gammaln(a + b + 1) + gammaln(c + d + 1) - gammaln(n + 2) + \
            gammaln(a + 1) + gammaln(b + 1) - gammaln(a + b + 2) + \
            gammaln(c + 1) + gammaln(d + 1) - gammaln(c + d + 2)

        # 非因果模型 (一个Beta)
        log_p_data_given_noncausal = \
            gammaln(a + c + 1) + gammaln(b + d + 1) - gammaln(n + 2) + \
            gammaln(a + c + 1) + gammaln(b + d + 1) - gammaln(n + 2)

        # 贝叶斯因子
        return np.exp(log_p_data_given_causal - log_p_data_given_noncausal)

if __name__ == "__main__":
    print("=" * 60)
    print("归纳推理形式化示例")
    print("=" * 60)

    # 1. 数字游戏
    print("\n1. 数字游戏 (Number Game):")
    game = NumberGame(max_number=50)
    induction = game.induction_engine()

    # 观察 {8, 16, 32}
    data = {8, 16, 32}
    posteriors = induction.compute_posteriors(data)

    print(f"   观察数据: {data}")
    print("   顶级假设后验:")
    for name, prob in sorted(posteriors.items(),
                            key=lambda x: -x[1])[:5]:
        print(f"      {name}: {prob:.4f}")

    # 泛化预测
    print("\n   泛化预测:")
    for x in [4, 64, 12]:
        prob = induction.generalization_probability(x, data)
        print(f"      P({x} ∈ concept | data) = {prob:.4f}")

    # 2. 属性归纳
    print("\n2. 属性归纳:")
    # 动物相似性矩阵（简化）
    animals = ['robin', 'sparrow', 'hawk', 'penguin', 'chicken']
    sim_matrix = np.array([
        [1.0, 0.9, 0.7, 0.3, 0.4],
        [0.9, 1.0, 0.6, 0.3, 0.4],
        [0.7, 0.6, 1.0, 0.2, 0.3],
        [0.3, 0.3, 0.2, 1.0, 0.5],
        [0.4, 0.4, 0.3, 0.5, 1.0],
    ])

    prop_ind = PropertyInduction(sim_matrix, animals)

    # Robin有X属性，Hawk有X属性的概率
    robin_idx = 0
    hawk_idx = 2
    strength = prop_induction.induction_strength([robin_idx], hawk_idx)
    print(f"   P(Hawk has X | Robin has X) 强度: {strength:.3f}")

    # 多样性效应
    diverse_premises = [0, 3]  # Robin和Penguin
    similar_premises = [0, 1]  # Robin和Sparrow

    strength_diverse = prop_ind.induction_strength(diverse_premises, hawk_idx)
    strength_similar = prop_ind.induction_strength(similar_premises, hawk_idx)

    print(f"   多样性前提归纳强度: {strength_diverse:.3f}")
    print(f"   相似前提归纳强度: {strength_similar:.3f}")

    # 3. 因果归纳
    print("\n3. 因果归纳:")
    causal = CausalInduction()

    # 经典示例: 药物试验
    # a=药物有效, b=药物无效, c=安慰剂有效, d=安慰剂无效
    a, b, c, d = 18, 12, 6, 24

    strength, var = causal.infer_causal_strength(a, b, c, d)
    support = causal.causal_support(a, b, c, d)

    print(f"   数据: 药物有效={a}, 药物无效={b}, 安慰剂有效={c}, 安慰剂无效={d}")
    print(f"   因果强度估计: {strength:.3f} ± {np.sqrt(var):.3f}")
    print(f"   因果支持度: {support:.3f}")
