## 13.3.3.1 引言

范畴化（Categorization）是人类认知的核心能力，涉及将对象归入概念类别的过程。
本节形式化范畴学习的计算模型，包括基于规则、基于相似性和基于理论的范畴化。

> **参考**: Nosofsky, R. M. (1986). "Attention, similarity, and the identification-categorization relationship". _Journal of Experimental Psychology_.
>
> Ashby, F. G., & Maddox, W. T. (2005). "Human category learning". _Annual Review of Psychology_.

## 13.3.3.2 范畴学习的形式模型

### 13.3.3.2.1 范畴学习任务

**定义 13.3.3.1** (范畴学习任务)

给定：

- 刺激空间 $\mathcal{X} \subseteq \mathbb{R}^n$
- 范畴标签集 $\mathcal{C} = \{C_1, \ldots, C_k\}$
- 训练样本 $\{(\mathbf{x}_i, c_i)\}_{i=1}^N$

目标：学习分类器 $f: \mathcal{X} \rightarrow \mathcal{C}$

### 13.3.3.2.2 范畴化系统类型

| 系统类型 | 神经基础 | 特征 |
|----------|----------|------|
| **规则型** | 前额叶皮层 | 显性规则，可言语报告 |
| **信息整合** | 纹状体 | 程序性学习，隐式 |
| **程序性** | 尾状核 | 习惯形成，奖励驱动 |

## 13.3.3.3 基于规则的模型

### 13.3.3.3.1 单维度规则

**定义 13.3.3.2** (单维度分类器)

基于维度 $j$ 和阈值 $\theta$ 的分类器：

$$
f(\mathbf{x}) = \begin{cases}
C_1 & \text{if } x_j \leq \theta \\
C_2 & \text{if } x_j > \theta
\end{cases}
$$

### 13.3.3.3.2 合取规则

**定义 13.3.3.3** (合取分类器)

$$
f(\mathbf{x}) = C_1 \iff \bigwedge_{j \in J} (x_j \in I_j)
$$

其中 $I_j$ 为维度 $j$ 上的区间。

## 13.3.3.4 基于相似性的模型

### 13.3.3.4.1 广义语境模型(GCM)

**定义 13.3.3.4** (GCM)

Nosofsky的广义语境模型：

$$
P(C_j \mid \mathbf{x}) = \frac{\left(\sum_{\mathbf{x}_i \in C_j} \eta_i \cdot s(\mathbf{x}, \mathbf{x}_i)\right)^\gamma}{\sum_k \left(\sum_{\mathbf{x}_i \in C_k} \eta_i \cdot s(\mathbf{x}, \mathbf{x}_i)\right)^\gamma}
$$

其中：

- $\eta_i$：范例 $i$ 的熟悉度
- $s(\mathbf{x}, \mathbf{x}_i)$：相似性函数
- $\gamma$：确定性参数

### 13.3.3.4.2 ALCOVE模型

**定义 13.3.3.5** (ALCOVE)

Kruschke的ALCOVE连接主义模型：

**相似性计算**:

$$
a_j^{hid} = \exp\left(-c \cdot \sum_i w_i^{att} \cdot |x_i - h_{ji}|\right)
$$

**类别激活**:

$$
a_k^{out} = \sum_j w_{kj} \cdot a_j^{hid}
$$

**定理 13.3.3.1** (ALCOVE学习动态)

权重更新遵循Delta规则：

$$
\Delta w_{kj} = \lambda_w \cdot (t_k - a_k^{out}) \cdot a_j^{hid}
$$

$$
\Delta w_i^{att} = -\lambda_a \cdot \frac{\partial E}{\partial w_i^{att}}
$$

## 13.3.3.5 原型与范例模型

### 13.3.3.5.1 原型模型

**定义 13.3.3.6** (原型分类器)

基于类别原型的分类：

$$
P(C_j \mid \mathbf{x}) = \frac{\exp\left(-d(\mathbf{x}, \mathbf{p}_j)\right)}{\sum_k \exp\left(-d(\mathbf{x}, \mathbf{p}_k)\right)}
$$

其中 $\mathbf{p}_j = \frac{1}{|C_j|}\sum_{\mathbf{x} \in C_j} \mathbf{x}$

### 13.3.3.5.2 范例模型

**定义 13.3.3.7** (范例存储)

所有训练样本作为范例存储：

$$
\mathcal{E} = \{(\mathbf{x}_i, c_i, \eta_i)\}_{i=1}^N
$$

分类基于与所有范例的相似性加权。

## 13.3.3.6 基于理论的范畴化

### 13.3.3.6.1 理论驱动的分类

**定义 13.3.3.8** (理论基础的先验)

先验分布受领域理论约束：

$$
P(C \mid \mathbf{x}, \mathcal{T}) = \frac{P(\mathbf{x} \mid C, \mathcal{T}) P(C \mid \mathcal{T})}{P(\mathbf{x} \mid \mathcal{T})}
$$

### 13.3.3.6.2 因果模型理论

**定义 13.3.3.9** (因果分类)

Rehder的因果模型：

$$
P(F_i = 1 \mid C) = \sum_{\text{causes } j} w_{ji} \cdot P(F_j = 1 \mid C)
$$

特征依赖由因果网络结构决定。

"""
范畴化过程的形式化实现
基于Nosofsky, Kruschke, 和Ashby的理论
"""

import numpy as np
from typing import List, Tuple, Dict, Callable, Optional
from dataclasses import dataclass
from collections import defaultdict
from scipy.spatial.distance import cdist, euclidean
from scipy.optimize import minimize
from scipy.special import softmax
import matplotlib.pyplot as plt


@dataclass
class Stimulus:
    """
    刺激表征
    """
    features: np.ndarray
    category: int
    label: Optional[str] = None

    def __array__(self):
        return self.features


class SimilarityFunction:
    """
    相似性函数
    实现各种相似性度量
    """

    @staticmethod
    def exponential(distance: float, c: float = 1.0) -> float:
        """
        指数相似性: s = exp(-c * d)
        Shepard定律
        """
        return np.exp(-c * distance)

    @staticmethod
    def gaussian(distance: float, sigma: float = 1.0) -> float:
        """
        高斯相似性: s = exp(-d² / (2σ²))
        """
        return np.exp(-distance ** 2 / (2 * sigma ** 2))

    @staticmethod
    def minkowski(x: np.ndarray, y: np.ndarray,
                  p: float = 2.0,
                  weights: Optional[np.ndarray] = None) -> float:
        """
        加权Minkowski距离
        """
        diff = np.abs(x - y)
        if weights is not None:
            diff = diff * weights
        return np.sum(diff ** p) ** (1 / p)


class PrototypeModel:
    """
    原型范畴化模型
    基于类别原型进行分类
    """

    def __init__(self, n_categories: int, similarity_fn: str = 'gaussian'):
        self.n_categories = n_categories
        self.prototypes: Dict[int, np.ndarray] = {}
        self.prototype_counts: Dict[int, int] = defaultdict(int)

        if similarity_fn == 'gaussian':
            self.similarity = SimilarityFunction.gaussian
        elif similarity_fn == 'exponential':
            self.similarity = SimilarityFunction.exponential
        else:
            raise ValueError(f"未知的相似性函数: {similarity_fn}")

    def fit(self, stimuli: List[Stimulus]):
        """
        学习类别原型
        原型 = 类别内样本的均值
        """
        category_sums = defaultdict(lambda: np.zeros(len(stimuli[0].features)))

        for s in stimuli:
            category_sums[s.category] += s.features
            self.prototype_counts[s.category] += 1

        for cat in category_sums:
            self.prototypes[cat] = (category_sums[cat] /
                                   self.prototype_counts[cat])

    def predict_proba(self, x: np.ndarray,
                     temperature: float = 1.0) -> np.ndarray:
        """
        预测类别概率

        P(C|x) ∝ exp(-d(x,p_C) / T)
        """
        distances = []
        for cat in range(self.n_categories):
            if cat in self.prototypes:
                d = euclidean(x, self.prototypes[cat])
                distances.append(-d / temperature)
            else:
                distances.append(-np.inf)

        return softmax(distances)

    def predict(self, x: np.ndarray) -> int:
        """预测类别标签"""
        probs = self.predict_proba(x)
        return np.argmax(probs)

    def similarity_to_prototype(self, x: np.ndarray,
                                category: int) -> float:
        """计算与特定原型的相似性"""
        if category not in self.prototypes:
            return 0.0
        dist = euclidean(x, self.prototypes[category])
        return self.similarity(dist)


class ExemplarModel:
    """
    范例模型 (GCM - Generalized Context Model)
    Nosofsky (1986)
    """

    def __init__(self,
                 similarity_fn: str = 'gaussian',
                 gamma: float = 2.0):
        """
        初始化范例模型

        Args:
            similarity_fn: 相似性函数类型
            gamma: 确定性参数
        """
        self.exemplars: List[Stimulus] = []
        self.gamma = gamma
        self.attention_weights: Optional[np.ndarray] = None

        if similarity_fn == 'gaussian':
            self.sim_fn = SimilarityFunction.gaussian
        else:
            self.sim_fn = SimilarityFunction.exponential

        # 范例熟悉度
        self.familiarity: Dict[int, float] = defaultdict(lambda: 1.0)

    def fit(self, stimuli: List[Stimulus]):
        """
        存储所有范例
        """
        self.exemplars = stimuli
        n_features = len(stimuli[0].features)
        self.attention_weights = np.ones(n_features) / n_features

    def weighted_distance(self, x: np.ndarray, y: np.ndarray) -> float:
        """
        加权距离
        d²(x,y) = Σ w_i (x_i - y_i)²
        """
        diff = x - y
        weighted_sq = self.attention_weights * (diff ** 2)
        return np.sqrt(np.sum(weighted_sq))

    def predict_proba(self, x: np.ndarray) -> np.ndarray:
        """
        GCM概率预测

        P(C|x) = [Σ_{x_i ∈ C} η_i * s(x, x_i)]^γ /
                 Σ_k [Σ_{x_i ∈ C_k} η_i * s(x, x_i)]^γ
        """
        category_similarities = defaultdict(float)

        for exemplar in self.exemplars:
            dist = self.weighted_distance(x, exemplar.features)
            sim = self.sim_fn(dist)
            eta = self.familiarity[id(exemplar)]
            category_similarities[exemplar.category] += eta * sim

        # 应用gamma并归一化
        scores = np.zeros(max(self.n_categories(), 1))
        for cat, sim_sum in category_similarities.items():
            scores[cat] = sim_sum ** self.gamma

        total = np.sum(scores)
        return scores / total if total > 0 else scores

    def predict(self, x: np.ndarray) -> int:
        """预测类别"""
        probs = self.predict_proba(x)
        return np.argmax(probs)

    def n_categories(self) -> int:
        """类别数"""
        return len(set(e.category for e in self.exemplars))

    def learn_attention(self, stimuli: List[Stimulus],
                       learning_rate: float = 0.01,
                       n_epochs: int = 100):
        """
        学习最优注意权重
        通过最小化分类误差
        """
        n_features = len(stimuli[0].features)

        def loss(weights):
            self.attention_weights = softmax(weights)
            error = 0
            for s in stimuli:
                probs = self.predict_proba(s.features)
                # 交叉熵损失
                error -= np.log(probs[s.category] + 1e-10)
            return error

        # 优化注意权重
        init_weights = np.zeros(n_features)
        result = minimize(loss, init_weights, method='L-BFGS-B')
        self.attention_weights = softmax(result.x)


class ALCOVE:
    """
    ALCOVE模型 (Kruschke, 1992)
    连接主义范畴学习
    """

    def __init__(self,
                 n_features: int,
                 n_categories: int,
                 hidden_units: List[np.ndarray],
                 c: float = 6.25):
        """
        初始化ALCOVE

        Args:
            n_features: 特征维度
            n_categories: 类别数
            hidden_units: 隐藏单元位置（通常=范例位置）
            c: 特异性参数
        """
        self.n_features = n_features
        self.n_categories = n_categories
        self.c = c

        # 隐藏单元（范例）
        self.hidden_locs = np.array(hidden_units)
        self.n_hidden = len(hidden_units)

        # 注意权重
        self.attention = np.ones(n_features) / n_features

        # 输出权重 (类别 × 隐藏单元)
        self.weights = np.zeros((n_categories, self.n_hidden))

        # 类别代表 (one-hot)
        self.category_reps = np.eye(n_categories)

    def hidden_activation(self, x: np.ndarray) -> np.ndarray:
        """
        计算隐藏层激活
        a_j^hid = exp(-c * Σ w_i^att |x_i - h_ji|)
        """
        # 加权绝对距离
        diff = np.abs(x - self.hidden_locs)
        weighted_diff = diff * self.attention
        distances = np.sum(weighted_diff, axis=1)

        return np.exp(-self.c * distances)

    def output_activation(self, hidden_act: np.ndarray) -> np.ndarray:
        """
        计算输出层激活
        a_k^out = Σ_j w_kj * a_j^hid
        """
        return np.dot(self.weights, hidden_act)

    def predict_proba(self, x: np.ndarray) -> np.ndarray:
        """预测类别概率"""
        hidden = self.hidden_activation(x)
        output = self.output_activation(hidden)
        return softmax(output)

    def train(self, stimuli: List[Stimulus],
             learning_rate_w: float = 0.1,
             learning_rate_a: float = 0.1,
             n_epochs: int = 50):
        """
        训练ALCOVE

        使用Delta规则更新权重
        """
        for epoch in range(n_epochs):
            total_error = 0

            for s in stimuli:
                # 前向传播
                hidden = self.hidden_activation(s.features)
                output = self.output_activation(hidden)

                # 目标
                target = self.category_reps[s.category]

                # 误差
                error = target - output
                total_error += np.sum(error ** 2)

                # 更新输出权重 (Delta规则)
                delta_w = learning_rate_w * np.outer(error, hidden)
                self.weights += delta_w

                # 更新注意权重
                # 计算梯度
                for i in range(self.n_features):
                    # 注意权重对输出的影响
                    grad = 0
                    for k in range(self.n_categories):
                        for j in range(self.n_hidden):
                            diff = abs(s.features[i] - self.hidden_locs[j, i])
                            grad -= error[k] * self.weights[k, j] * \
                                   hidden[j] * self.c * diff

                    self.attention[i] += learning_rate_a * grad

                # 归一化注意权重
                self.attention = np.maximum(self.attention, 0.01)
                self.attention /= np.sum(self.attention)

            if epoch % 10 == 0:
                print(f"Epoch {epoch}: Error = {total_error:.4f}")


class RuleBasedModel:
    """
    基于规则的范畴化模型
    实现单维度和合取规则
    """

    def __init__(self, rule_type: str = 'unidimensional'):
        """
        Args:
            rule_type: 'unidimensional', 'conjunctive', 'disjunctive'
        """
        self.rule_type = rule_type
        self.rule: Optional[Dict] = None

    def fit(self, stimuli: List[Stimulus]):
        """
        学习最优规则
        通过搜索最佳分割
        """
        if self.rule_type == 'unidimensional':
            self.rule = self._fit_unidimensional(stimuli)
        elif self.rule_type == 'conjunctive':
            self.rule = self._fit_conjunctive(stimuli)

    def _fit_unidimensional(self, stimuli: List[Stimulus]) -> Dict:
        """
        学习单维度规则
        x_j ≤ θ -> C_0, x_j > θ -> C_1
        """
        n_features = len(stimuli[0].features)
        best_acc = 0
        best_rule = None

        for dim in range(n_features):
            # 按维度排序
            values = [(s.features[dim], s.category) for s in stimuli]
            values.sort()

            # 尝试不同阈值
            for i in range(len(values) - 1):
                threshold = (values[i][0] + values[i+1][0]) / 2

                # 测试两个方向
                for flip in [False, True]:
                    correct = 0
                    for s in stimuli:
                        pred = 0 if s.features[dim] <= threshold else 1
                        if flip:
                            pred = 1 - pred
                        if pred == s.category:
                            correct += 1

                    acc = correct / len(stimuli)
                    if acc > best_acc:
                        best_acc = acc
                        best_rule = {
                            'dimension': dim,
                            'threshold': threshold,
                            'flip': flip,
                            'accuracy': acc
                        }

        return best_rule

    def _fit_conjunctive(self, stimuli: List[Stimulus]) -> Dict:
        """
        学习合取规则
        ∧_j (x_j ∈ [low_j, high_j]) -> C
        """
        # 简化的合取规则学习
        categories = set(s.category for s in stimuli)
        best_rule = None
        best_acc = 0

        for cat in categories:
            cat_stimuli = [s for s in stimuli if s.category == cat]
            other_stimuli = [s for s in stimuli if s.category != cat]

            # 找包含边界
            if len(cat_stimuli) > 0:
                bounds = []
                for dim in range(len(cat_stimuli[0].features)):
                    values = [s.features[dim] for s in cat_stimuli]
                    bounds.append((min(values), max(values)))

                # 测试规则
                correct = 0
                for s in stimuli:
                    in_bounds = all(
                        bounds[d][0] <= s.features[d] <= bounds[d][1]
                        for d in range(len(bounds))
                    )
                    pred = cat if in_bounds else (1 - cat)
                    if pred == s.category:
                        correct += 1

                acc = correct / len(stimuli)
                if acc > best_acc:
                    best_acc = acc
                    best_rule = {
                        'category': cat,
                        'bounds': bounds,
                        'accuracy': acc
                    }

        return best_rule

    def predict(self, x: np.ndarray) -> int:
        """使用规则预测"""
        if self.rule is None:
            raise ValueError("模型尚未训练")

        if self.rule_type == 'unidimensional':
            dim = self.rule['dimension']
            threshold = self.rule['threshold']
            pred = 0 if x[dim] <= threshold else 1
            if self.rule['flip']:
                pred = 1 - pred
            return pred

        elif self.rule_type == 'conjunctive':
            cat = self.rule['category']
            bounds = self.rule['bounds']
            in_bounds = all(
                bounds[d][0] <= x[d] <= bounds[d][1]
                for d in range(len(bounds))
            )
            return cat if in_bounds else (1 - cat)


# ===== 示例：Shepard分类任务 =====
class ShepardTasks:
    """
    Shepard, Hovland & Jenkins (1961) 六类分类任务
    """

    TASKS = {
        'Type I': lambda f: f[0],      # 单维度
        'Type II': lambda f: f[0] ^ f[1],  # 异或
        'Type III': lambda f: f[0] & (f[1] | f[2]),
        'Type IV': lambda f: (f[0] == f[1]) & (f[1] == f[2]),
        'Type V': lambda f: f[0] | f[1],
        'Type VI': lambda f: f[0] ^ f[1] ^ f[2],  # 奇偶校验
    }

    @staticmethod
    def generate_task(task_type: str, n_dims: int = 3) -> Tuple[List[Stimulus], Callable]:
        """
        生成Shepard任务数据

        Returns:
            (刺激列表, 目标函数)
        """
        stimuli = []
        rule_fn = ShepardTasks.TASKS[task_type]

        # 生成所有可能的刺激 (2^n_dims)
        for i in range(2 ** n_dims):
            features = np.array([(i >> j) & 1 for j in range(n_dims)])
            category = int(rule_fn(features))
            stimuli.append(Stimulus(features, category))

        return stimuli, rule_fn

    @staticmethod
    def evaluate_models(task_type: str):
        """
        评估不同模型在Shepard任务上的表现
        """
        stimuli, rule = ShepardTasks.generate_task(task_type)

        results = {}

        # 原型模型
        proto = PrototypeModel(n_categories=2)
        proto.fit(stimuli)
        proto_acc = sum(
            proto.predict(s.features) == s.category
            for s in stimuli
        ) / len(stimuli)
        results['Prototype'] = proto_acc

        # 范例模型
        exemplar = ExemplarModel()
        exemplar.fit(stimuli)
        exemp_acc = sum(
            exemplar.predict(s.features) == s.category
            for s in stimuli
        ) / len(stimuli)
        results['Exemplar (GCM)'] = exemp_acc

        # 规则模型
        rule_model = RuleBasedModel('unidimensional')
        rule_model.fit(stimuli)
        rule_acc = sum(
            rule_model.predict(s.features) == s.category
            for s in stimuli
        ) / len(stimuli)
        results['Rule-based'] = rule_acc

        return results
