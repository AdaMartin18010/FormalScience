## 13.3.2.1 引言

原型理论由Eleanor Rosch提出，挑战了经典范畴理论的充要条件观点。
本节将原型理论形式化为模糊集合论框架，建立隶属度函数、典型性度量与范畴边界的严格数学模型。

> **参考**: Rosch, E. (1978). "Principles of Categorization". In E. Rosch & B. B. Lloyd (Eds.), _Cognition and Categorization_.
>
> Zadeh, L. A. (1965). "Fuzzy Sets". _Information and Control_.

## 13.3.2.2 模糊集合基础

### 13.3.2.2.1 模糊集合定义

**定义 13.3.2.1** (模糊集合)

论域 $X$ 上的**模糊集合** $A$ 由隶属函数表征：

$$
\mu_A: X \rightarrow [0,1]
$$

其中 $\mu_A(x)$ 表示 $x$ 属于 $A$ 的程度。

### 13.3.2.2.2 模糊运算

**定义 13.3.2.2** (模糊集合运算)

| 运算 | 公式 | 名称 |
|------|------|------|
| 并 | $\mu_{A \cup B}(x) = \max(\mu_A(x), \mu_B(x))$ | 标准并 |
| 交 | $\mu_{A \cap B}(x) = \min(\mu_A(x), \mu_B(x))$ | 标准交 |
| 补 | $\mu_{\bar{A}}(x) = 1 - \mu_A(x)$ | 标准补 |
| 包含 | $A \subseteq B \iff \forall x: \mu_A(x) \leq \mu_B(x)$ | 模糊包含 |

**定理 13.3.2.1** (模糊集合代数性质)

标准模糊运算满足：

- 幂等律: $A \cup A = A$, $A \cap A = A$
- 交换律: $A \cup B = B \cup A$
- 结合律: $(A \cup B) \cup C = A \cup (B \cup C)$
- **不满足**: 排中律 ($A \cup \bar{A} \neq X$)、矛盾律 ($A \cap \bar{A} \neq \emptyset$)

## 13.3.2.3 原型模型

### 13.3.2.3.1 原型作为隶属度生成器

**定义 13.3.2.3** (原型函数)

设原型 $p \in X$，以 $p$ 为中心的概念隶属函数：

$$
\mu_p(x) = f(d(x, p))
$$

其中 $d$ 为距离度量，$f$ 为单调递减函数。

**常用形式**:

- 高斯型: $\mu(x) = \exp\left(-\frac{d(x,p)^2}{2\sigma^2}\right)$
- 三角型: $\mu(x) = \max(0, 1 - \frac{d(x,p)}{r})$
- Sigmoid型: $\mu(x) = \frac{1}{1 + e^{-\alpha(r - d(x,p))}}$

### 13.3.2.3.2 多原型系统

**定义 13.3.2.4** (多原型概念)

概念 $C$ 有原型集合 $P = \{p_1, \ldots, p_n\}$：

$$
\mu_C(x) = \max_{p \in P} \mu_p(x) \quad \text{(析取原型)}
$$

或

$$
\mu_C(x) = \frac{1}{n}\sum_{p \in P} \mu_p(x) \quad \text{(平均原型)}
$$

## 13.3.2.4 典型性理论

### 13.3.2.4.1 典型性度量

**定义 13.3.2.5** (典型性)

元素 $x$ 对概念 $C$ 的**典型性**：

$$
\text{Typ}(x, C) = \mu_C(x) \cdot \text{Sal}(x)
$$

其中 $\text{Sal}(x)$ 为 $x$ 的显著性（与上下文相关）。

### 13.3.2.4.2 反应时与典型性

**定理 13.3.2.2** (典型性效应)

范畴判断反应时与典型性负相关：

$$
RT(x \in^? C) = a - b \cdot \text{Typ}(x, C) + \epsilon
$$

实验验证（Rosch, 1973）:

| 鸟类 | 典型性评分 | 平均RT(ms) |
|------|-----------|-----------|
| 知更鸟 | 1.00 | 558 |
| 鹰 | 0.83 | 570 |
| 鸡 | 0.52 | 593 |
| 企鹅 | 0.21 | 620 |
| 蝙蝠 | 0.05 | 642 |

## 13.3.2.5 范畴边界

### 13.3.2.5.1 α-截集

**定义 13.3.2.6** (α-截集)

模糊集合 $A$ 的 $\alpha$-截集为经典集合：

$$
A_\alpha = \{x \in X \mid \mu_A(x) \geq \alpha\}
$$

**定理 13.3.2.3** (分解定理)

任何模糊集合可表示为：

$$
\mu_A(x) = \sup_{\alpha \in [0,1]} \min(\alpha, \mu_{A_\alpha}(x))
$$

### 13.3.2.5.2 边界模糊性

**定义 13.3.2.7** (边界宽度)

概念 $C$ 的边界区域：

$$
\text{Boundary}(C) = \{x \mid 0 < \mu_C(x) < 1\}
$$

**边界宽度度量**：

$$
\text{BW}(C) = \int_{\partial C} \left|\nabla \mu_C(x)\right|^{-1} dx
$$

## 13.3.2.6 概念层次结构

### 13.3.2.6.1 基本层次

**定义 13.3.2.8** (基本层次)

基本层次概念 $C_{basic}$ 满足：

1. **知觉整体性**: 高内部相似性
2. **动作程序**: 共同的行为互动
3. **信息丰富性**: 高预测力

**形式化**: 最大化组内相似性与组间差异性的层次：

$$
C^* = \arg\max_C \left(\frac{\text{Sim}_{\text{within}}(C)}{\text{Sim}_{\text{between}}(C, C_{\text{others}})}\right)
$$

### 13.3.2.6.2 上位与下位概念

**定理 13.3.2.4** (包含关系)

若 $C_{sub}$ 是 $C_{super}$ 的下位概念，则：

$$
\mu_{C_{super}}(x) \geq \mu_{C_{sub}}(x) \quad \forall x \in X
$$

"""
原型理论形式化实现
基于Rosch (1978) 和 Zadeh (1965)
"""

import numpy as np
from typing import List, Tuple, Callable, Dict, Optional, Set
from dataclasses import dataclass
from collections import defaultdict
import matplotlib.pyplot as plt
from scipy.spatial.distance import euclidean, cosine
from scipy.optimize import minimize


class MembershipFunction:
    """
    隶属函数生成器
    """

    @staticmethod
    def gaussian(prototype: np.ndarray, sigma: float) -> Callable:
        """
        高斯型隶属函数
        μ(x) = exp(-||x - p||² / (2σ²))
        """
        def mu(x: np.ndarray) -> float:
            dist = euclidean(x, prototype)
            return np.exp(-dist ** 2 / (2 * sigma ** 2))
        return mu

    @staticmethod
    def triangular(prototype: np.ndarray, radius: float) -> Callable:
        """
        三角型隶属函数
        μ(x) = max(0, 1 - d(x,p)/r)
        """
        def mu(x: np.ndarray) -> float:
            dist = euclidean(x, prototype)
            return max(0, 1 - dist / radius)
        return mu

    @staticmethod
    def trapezoidal(prototype: np.ndarray,
                   inner_radius: float,
                   outer_radius: float) -> Callable:
        """
        梯形隶属函数
        """
        def mu(x: np.ndarray) -> float:
            dist = euclidean(x, prototype)
            if dist <= inner_radius:
                return 1.0
            elif dist <= outer_radius:
                return (outer_radius - dist) / (outer_radius - inner_radius)
            else:
                return 0.0
        return mu

    @staticmethod
    def sigmoid(prototype: np.ndarray,
               steepness: float,
               crossover: float) -> Callable:
        """
        Sigmoid型隶属函数
        μ(x) = 1 / (1 + exp(-α(r - d)))
        """
        def mu(x: np.ndarray) -> float:
            dist = euclidean(x, prototype)
            return 1.0 / (1.0 + np.exp(-steepness * (crossover - dist)))
        return mu


@dataclass
class FuzzySet:
    """
    模糊集合
    """
    name: str
    membership_fn: Callable[[np.ndarray], float]
    domain: Optional[Tuple[float, float]] = None

    def __call__(self, x: np.ndarray) -> float:
        """计算隶属度"""
        return np.clip(self.membership_fn(x), 0, 1)

    def alpha_cut(self, alpha: float,
                  samples: np.ndarray) -> Set[int]:
        """
        α-截集
        返回隶属度≥α的样本索引
        """
        return {i for i, x in enumerate(samples)
               if self(x) >= alpha}

    def support(self, samples: np.ndarray) -> Set[int]:
        """
        支撑集 (α=0)
        """
        return {i for i, x in enumerate(samples) if self(x) > 0}

    def core(self, samples: np.ndarray,
            tolerance: float = 1e-6) -> Set[int]:
        """
        核心 (α=1)
        """
        return {i for i, x in enumerate(samples)
               if abs(self(x) - 1.0) < tolerance}

    def height(self, samples: np.ndarray) -> float:
        """
        集合高度
        """
        return max(self(x) for x in samples)

    def cardinality(self, samples: np.ndarray) -> float:
        """
        模糊基数 (sigma-count)
        |A| = Σ μ_A(x)
        """
        return sum(self(x) for x in samples)


class FuzzyOperations:
    """
    模糊集合运算
    """

    @staticmethod
    def union(a: FuzzySet, b: FuzzySet,
             t_conorm: str = 'max') -> FuzzySet:
        """
        模糊并集

        t-conorms:
        - max: 标准并
        - algebraic_sum: a + b - ab
        - bounded_sum: min(1, a + b)
        """
        def mu(x):
            ma, mb = a(x), b(x)
            if t_conorm == 'max':
                return max(ma, mb)
            elif t_conorm == 'algebraic_sum':
                return ma + mb - ma * mb
            elif t_conorm == 'bounded_sum':
                return min(1.0, ma + mb)
            else:
                raise ValueError(f"未知的t-conorm: {t_conorm}")

        return FuzzySet(f"{a.name}_union_{b.name}", mu)

    @staticmethod
    def intersection(a: FuzzySet, b: FuzzySet,
                    t_norm: str = 'min') -> FuzzySet:
        """
        模糊交集

        t-norms:
        - min: 标准交
        - algebraic_product: ab
        - bounded_diff: max(0, a + b - 1)
        """
        def mu(x):
            ma, mb = a(x), b(x)
            if t_norm == 'min':
                return min(ma, mb)
            elif t_norm == 'algebraic_product':
                return ma * mb
            elif t_norm == 'bounded_diff':
                return max(0.0, ma + mb - 1)
            else:
                raise ValueError(f"未知的t-norm: {t_norm}")

        return FuzzySet(f"{a.name}_inter_{b.name}", mu)

    @staticmethod
    def complement(a: FuzzySet,
                  negation: str = 'standard') -> FuzzySet:
        """
        模糊补集
        """
        def mu(x):
            ma = a(x)
            if negation == 'standard':
                return 1.0 - ma
            elif negation == 'sugeno':
                # Sugeno补
                lambda_param = 1.0
                return (1 - ma) / (1 + lambda_param * ma)
            else:
                raise ValueError(f"未知的否定: {negation}")

        return FuzzySet(f"not_{a.name}", mu)

    @staticmethod
    def implication(a: FuzzySet, b: FuzzySet,
                   implication_type: str = 'lukasiewicz') -> FuzzySet:
        """
        模糊蕴含
        """
        def mu(x):
            ma, mb = a(x), b(x)
            if implication_type == 'lukasiewicz':
                return min(1.0, 1 - ma + mb)
            elif implication_type == 'goedel':
                return 1.0 if ma <= mb else mb
            elif implication_type == 'goguen':
                return 1.0 if ma == 0 else min(1.0, mb / ma)
            else:
                raise ValueError(f"未知的蕴含: {implication_type}")

        return FuzzySet(f"{a.name}_implies_{b.name}", mu)


class PrototypeConcept:
    """
    基于原型的概念
    实现Rosch原型理论
    """

    def __init__(self,
                 name: str,
                 prototypes: List[np.ndarray],
                 sigma: float = 1.0,
                 weights: Optional[List[float]] = None):
        """
        初始化原型概念

        Args:
            name: 概念名称
            prototypes: 原型位置列表
            sigma: 隶属函数的宽度参数
            weights: 各原型的权重（用于多原型）
        """
        self.name = name
        self.prototypes = prototypes
        self.sigma = sigma
        self.weights = weights or [1.0 / len(prototypes)] * len(prototypes)

        # 生成隶属函数
        self.membership_fns = [
            MembershipFunction.gaussian(p, sigma)
            for p in prototypes
        ]

    def membership(self, x: np.ndarray,
                   combination: str = 'max') -> float:
        """
        计算隶属度

        Args:
            combination: 'max' (析取), 'average' (平均), 'weighted' (加权)
        """
        mus = [fn(x) for fn in self.membership_fns]

        if combination == 'max':
            return max(mus)
        elif combination == 'average':
            return sum(mus) / len(mus)
        elif combination == 'weighted':
            return sum(w * m for w, m in zip(self.weights, mus))
        else:
            raise ValueError(f"未知的组合方式: {combination}")

    def typicality(self, x: np.ndarray,
                   salience_fn: Optional[Callable] = None) -> float:
        """
        计算典型性

        Typ(x,C) = μ_C(x) * Sal(x)
        """
        membership = self.membership(x)
        salience = salience_fn(x) if salience_fn else 1.0
        return membership * salience

    def reaction_time(self, x: np.ndarray,
                     a: float = 700, b: float = 200) -> float:
        """
        预测范畴判断反应时
        RT = a - b * Typ(x,C)
        """
        typ = self.typicality(x)
        return a - b * typ

    def to_fuzzy_set(self) -> FuzzySet:
        """转换为FuzzySet对象"""
        return FuzzySet(self.name, lambda x: self.membership(x))

    def similarity_to(self, other: 'PrototypeConcept',
                     samples: np.ndarray) -> float:
        """
        计算与另一个概念的相似性
        基于原型间的平均距离
        """
        similarities = []
        for p1 in self.prototypes:
            for p2 in other.prototypes:
                # 使用隶属度作为相似性
                sim = np.exp(-euclidean(p1, p2) / (self.sigma + other.sigma))
                similarities.append(sim)
        return sum(similarities) / len(similarities)


class CategoryHierarchy:
    """
    概念层次结构
    实现基本层次理论
    """

    def __init__(self):
        self.concepts: Dict[str, PrototypeConcept] = {}
        self.parent_of: Dict[str, str] = {}
        self.children_of: Dict[str, List[str]] = defaultdict(list)

    def add_concept(self, concept: PrototypeConcept,
                   parent: Optional[str] = None):
        """添加概念到层次结构"""
        self.concepts[concept.name] = concept
        if parent:
            self.parent_of[concept.name] = parent
            self.children_of[parent].append(concept.name)

    def basic_level(self, samples: np.ndarray) -> str:
        """
        识别基本层次概念

        基本层次 = 最大化(内部相似性/外部相似性)
        """
        best_concept = None
        best_score = -float('inf')

        for name, concept in self.concepts.items():
            # 计算内部相似性（样本间的平均相似性）
            within_sims = []
            for x in samples:
                if concept.membership(x) > 0.5:
                    for y in samples:
                        if concept.membership(y) > 0.5:
                            within_sims.append(
                                np.exp(-euclidean(x, y))
                            )

            within_sim = sum(within_sims) / len(within_sims) if within_sims else 0

            # 计算外部相似性
            between_sims = []
            for other_name, other_concept in self.concepts.items():
                if other_name != name:
                    sim = concept.similarity_to(other_concept, samples)
                    between_sims.append(sim)

            between_sim = sum(between_sims) / len(between_sims) if between_sims else 1

            # 基本层次分数
            score = within_sim / (between_sim + 1e-10)

            if score > best_score:
                best_score = score
                best_concept = name

        return best_concept

    def verify_inclusion(self, sub_name: str,
                        super_name: str,
                        samples: np.ndarray) -> bool:
        """
        验证包含关系
        检查 sub ⊆ super
        """
        sub = self.concepts[sub_name]
        super_c = self.concepts[super_name]

        # 检查所有样本
        for x in samples:
            if sub.membership(x) > super_c.membership(x) + 1e-6:
                return False
        return True

class BirdCategorySystem:
    """
    鸟类范畴系统 (Rosch, 1975)
    """

    def __init__(self):
        # 使用2D特征空间：大小、飞行能力
        self.features = ['size', 'flight_ability']

        # 定义原型
        birds = {
            'robin': np.array([0.3, 0.9]),       # 知更鸟：小，会飞
            'sparrow': np.array([0.25, 0.85]),   # 麻雀：小，会飞
            'eagle': np.array([0.8, 0.95]),      # 鹰：大，会飞
            'chicken': np.array([0.5, 0.1]),     # 鸡：中等，不会飞
            'penguin': np.array([0.6, 0.0]),     # 企鹅：中等，不会飞
            'ostrich': np.array([1.0, 0.0]),     # 鸵鸟：大，不会飞
        }

        # 创建鸟类概念
        self.bird_concept = PrototypeConcept(
            name='bird',
            prototypes=list(birds.values()),
            sigma=0.3
        )

        self.individual_birds = {
            name: PrototypeConcept(name, [features], sigma=0.15)
            for name, features in birds.items()
        }

        # 典型性评分（基于Rosch实验）
        self.typicality_ratings = {
            'robin': 1.00,
            'eagle': 0.83,
            'chicken': 0.52,
            'penguin': 0.21,
        }

    def typicality_study(self) -> Dict[str, float]:
        """模拟典型性实验"""
        results = {}
        for name, concept in self.individual_birds.items():
            # 计算相对于鸟类原型的典型性
            typ = self.bird_concept.typicality(concept.prototypes[0])
            results[name] = typ
        return results

    def categorize(self, features: np.ndarray) -> str:
        """将特征向量归类"""
        best_match = None
        best_score = -1

        for name, concept in self.individual_birds.items():
            score = concept.membership(features)
            if score > best_score:
                best_score = score
                best_match = name

        # 检查是否为鸟
        bird_score = self.bird_concept.membership(features)

        if bird_score > 0.3:
            return f"{best_match} (bird: {bird_score:.2f})"
        else:
            return "non-bird"

    print("\n4. α-截集分析:")
    # 生成样本
    samples = np.random.randn(100, 2) * 20 + np.array([170, 70])
    alphas = [0.2, 0.5, 0.8]
    for alpha in alphas:
        cut = set_a.alpha_cut(alpha, samples)
        print(f"   α={alpha}: {len(cut)}个样本属于'tall'")
