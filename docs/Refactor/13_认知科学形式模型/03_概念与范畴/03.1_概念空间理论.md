## 13.3.1.1 引言

概念空间理论（Conceptual Spaces Theory）由Peter Gärdenfors提出，作为连接符号认知模型与联结主义模型的中间层面。
本节系统阐述概念的几何表征与认知语义学。

> **参考**: Gärdenfors, P. (2000). _Conceptual Spaces: The Geometry of Thought_. MIT Press.
>
> Gärdenfors, P. (2014). _The Geometry of Meaning: Semantics Based on Conceptual Spaces_. MIT Press.

## 13.3.1.2 概念空间基础

### 13.3.1.2.1 域结构

**定义 13.3.1.1** (域)

**域** $\mathcal{D}$ 是具有特定认知意义的品质维度集合：

$$
\mathcal{D} = (D_1, D_2, \ldots, D_n)
$$

**定义 13.3.1.2** (概念空间)

**概念空间** $\mathcal{C}$ 是域的集合：

$$
\mathcal{C} = \{\mathcal{D}_1, \mathcal{D}_2, \ldots, \mathcal{D}_m\}
$$

### 13.3.1.2.2 品质维度

**定义 13.3.1.3** (品质维度类型)

| 类型 | 描述 | 实例 |
|------|------|------|
| **区间尺度** | 有序，有相等单位 | 温度、长度 |
| **比率尺度** | 有绝对零点 | 重量、持续时间 |
| **循环尺度** | 首尾相连 | 色相、时间周期 |
| **等级尺度** | 仅有序 | 偏好排序 |
| **名义尺度** | 仅分类 | 性别、国籍 |

## 13.3.1.3 几何表征

### 13.3.1.3.1 距离与相似性

**定义 13.3.1.4** (概念距离)

点 $\mathbf{x}, \mathbf{y} \in \mathcal{D}$ 之间的**曼哈顿距离**：

$$
d_c(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^{n} c_i |x_i - y_i|
$$

其中 $c_i$ 为维度权重。

**定义 13.3.1.5** (相似性度量)

基于Shepard的泛化理论：

$$
\text{Sim}(\mathbf{x}, \mathbf{y}) = e^{-\alpha \cdot d_c(\mathbf{x}, \mathbf{y})}
$$

### 13.3.1.3.2 凸性假设

**假设 13.3.1.1** (概念凸性假设)

自然概念对应于概念空间中的**凸区域**：

$$
\forall \mathbf{x}, \mathbf{y} \in C, \forall \lambda \in [0,1]: \lambda\mathbf{x} + (1-\lambda)\mathbf{y} \in C
$$

**定理 13.3.1.1** (凸性与插值)

若概念是凸的，则任何介于两个概念实例之间的点也属于该概念。

## 13.3.1.4 概念结构

### 13.3.1.4.1 原型与范畴

**定义 13.3.1.6** (原型)

概念 $C$ 的原型 $\mathbf{p}_C$ 是最大化范畴内相似性的点：

$$
\mathbf{p}_C = \arg\max_{\mathbf{x}} \sum_{\mathbf{y} \in C} \text{Sim}(\mathbf{x}, \mathbf{y})
$$

### 13.3.1.4.2 Voronoi图划分

**定义 13.3.1.7** (Voronoi单元)

给定原型集合 $\{\mathbf{p}_1, \ldots, \mathbf{p}_n\}$，Voronoi单元为：

$$
V_i = \{\mathbf{x} \in \mathcal{D} \mid d(\mathbf{x}, \mathbf{p}_i) \leq d(\mathbf{x}, \mathbf{p}_j), \forall j \neq i\}
$$

**定理 13.3.1.2** (Voronoi凸性)

在欧氏距离下，Voronoi单元是凸集。

## 13.3.1.5 颜色概念空间

### 13.3.1.5.1 颜色固体

**定义 13.3.1.8** (颜色空间)

颜色空间 $\mathcal{CS}$ 的三维结构：

$$
\mathcal{CS} = \text{Hue} \times \text{Saturation} \times \text{Value/Brightness}
$$

### 13.3.1.5.2 焦点色

**定义 13.3.1.9** (Berlin & Kay焦点色)

跨语言普遍的颜色原型：

| 顺序 | 基本颜色词 | 原型色相(度) |
|------|-----------|-------------|
| 1 | 白/黑 | - (明度) |
| 2 | 红 | 0° |
| 3 | 绿或黄 | 120°或60° |
| 4 | 蓝 | 240° |
| 5 | 棕 | 30° |

## 13.3.1.6 语义组合

### 13.3.1.6.1 概念组合

**定义 13.3.1.10** (合取概念)

两个概念的**交**（若有）：

$$
C_1 \land C_2 = C_1 \cap C_2 \quad \text{(若非空且凸)}
$$

### 13.3.1.6.2 隐喻映射

**定义 13.3.1.11** (域映射)

隐喻 $M: \mathcal{D}_S \rightarrow \mathcal{D}_T$ 保持几何结构：

$$
d_T(M(\mathbf{x}), M(\mathbf{y})) \approx d_S(\mathbf{x}, \mathbf{y})
$$

## 13.3.1.7 Python实现

```python
"""
概念空间理论的形式化实现
基于Gärdenfors (2000, 2014)
"""

import numpy as np
from typing import List, Tuple, Dict, Optional, Callable
from dataclasses import dataclass
from scipy.spatial import Voronoi, voronoi_plot_2d
from scipy.spatial.distance import cdist
import matplotlib.pyplot as plt
from matplotlib.patches import Polygon
from matplotlib.collections import PatchCollection


@dataclass
class QualityDimension:
    """
    品质维度

    属性:
        name: 维度名称
        scale_type: 尺度类型 (interval, ratio, circular, ordinal, nominal)
        domain: 值域 (min, max)
        weight: 在距离计算中的权重
    """
    name: str
    scale_type: str
    domain: Tuple[float, float]
    weight: float = 1.0

    def normalize(self, value: float) -> float:
        """归一化到[0,1]"""
        min_val, max_val = self.domain
        return (value - min_val) / (max_val - min_val)

    def distance(self, x: float, y: float) -> float:
        """计算维度上的距离"""
        if self.scale_type == 'circular':
            # 循环距离
            period = self.domain[1] - self.domain[0]
            diff = abs(x - y)
            return min(diff, period - diff) / period
        elif self.scale_type == 'nominal':
            # 名义尺度：相等或不等于
            return 0.0 if x == y else 1.0
        else:
            # 区间/比率/等级尺度
            return abs(self.normalize(x) - self.normalize(y))


class ConceptualDomain:
    """
    概念域：一组品质维度的笛卡尔积
    """

    def __init__(self, name: str, dimensions: List[QualityDimension]):
        self.name = name
        self.dimensions = dimensions
        self.n_dims = len(dimensions)

    def distance(self, x: np.ndarray, y: np.ndarray,
                 metric: str = 'manhattan') -> float:
        """
        计算域中的距离

        Args:
            x, y: 点坐标
            metric: 'manhattan', 'euclidean', 'weighted_euclidean'
        """
        if metric == 'manhattan':
            return sum(d.weight * d.distance(x[i], y[i])
                      for i, d in enumerate(self.dimensions))
        elif metric == 'euclidean':
            return np.sqrt(sum(d.weight * (d.distance(x[i], y[i]) ** 2)
                              for i, d in enumerate(self.dimensions)))
        elif metric == 'weighted_euclidean':
            weighted_diff = np.array([
                np.sqrt(d.weight) * d.distance(x[i], y[i])
                for i, d in enumerate(self.dimensions)
            ])
            return np.linalg.norm(weighted_diff)
        else:
            raise ValueError(f"未知度量: {metric}")

    def similarity(self, x: np.ndarray, y: np.ndarray,
                   alpha: float = 5.0) -> float:
        """
        Shepard相似性度量
        Sim(x,y) = exp(-α * d(x,y))
        """
        dist = self.distance(x, y)
        return np.exp(-alpha * dist)

    def interpolate(self, x: np.ndarray, y: np.ndarray,
                   t: float) -> np.ndarray:
        """
        凸组合插值

        对每个维度分别插值
        """
        result = np.zeros(self.n_dims)
        for i, dim in enumerate(self.dimensions):
            if dim.scale_type == 'nominal':
                # 名义尺度：选择较接近的
                result[i] = x[i] if t < 0.5 else y[i]
            elif dim.scale_type == 'circular':
                # 循环插值
                period = dim.domain[1] - dim.domain[0]
                diff = y[i] - x[i]
                if abs(diff) > period / 2:
                    diff = -np.sign(diff) * (period - abs(diff))
                result[i] = x[i] + t * diff
                # 归一化
                result[i] = (result[i] - dim.domain[0]) % period + dim.domain[0]
            else:
                # 线性插值
                x_norm = dim.normalize(x[i])
                y_norm = dim.normalize(y[i])
                result[i] = dim.domain[0] + (t * y_norm + (1-t) * x_norm) * \
                           (dim.domain[1] - dim.domain[0])
        return result

    def is_convex(self, points: List[np.ndarray],
                  test_point: np.ndarray) -> bool:
        """
        检查点集是否构成凸区域
        测试点是否可以表示为凸组合
        """
        # 简化：检查测试点是否在所有点对的插值路径上
        for p1 in points:
            for p2 in points:
                for t in np.linspace(0, 1, 10):
                    interp = self.interpolate(p1, p2, t)
                    if self.distance(interp, test_point) < 0.01:
                        return True
        return False


class Concept:
    """
    概念：概念空间中的凸区域
    """

    def __init__(self,
                 name: str,
                 domain: ConceptualDomain,
                 prototype: np.ndarray,
                 region: Optional[List[np.ndarray]] = None,
                 salience: float = 1.0):
        """
        初始化概念

        Args:
            name: 概念名称
            domain: 所属域
            prototype: 原型位置
            region: 区域边界点（可选）
            salience: 显著性权重
        """
        self.name = name
        self.domain = domain
        self.prototype = prototype
        self.region = region or [prototype]
        self.salience = salience
        self._cache = {}

    def membership(self, point: np.ndarray,
                   metric: str = 'euclidean') -> float:
        """
        计算点在概念中的隶属度
        基于与原型距离的高斯函数
        """
        dist = self.domain.distance(point, self.prototype, metric)
        # 高斯隶属函数
        return np.exp(-dist ** 2 / (2 * self.salience ** 2))

    def similarity_to(self, other: 'Concept') -> float:
        """
        计算两个概念间的相似性
        基于原型距离
        """
        dist = self.domain.distance(self.prototype, other.prototype)
        return np.exp(-dist)

    def intersect(self, other: 'Concept') -> Optional['Concept']:
        """
        概念合取（交集）
        如果区域相交则返回新概念
        """
        # 检查原型间距离
        dist = self.domain.distance(self.prototype, other.prototype)

        # 简化的相交判断
        if dist < (self.salience + other.salience):
            # 创建新的交集概念
            new_prototype = (self.prototype + other.prototype) / 2
            return Concept(
                name=f"{self.name}_{other.name}",
                domain=self.domain,
                prototype=new_prototype,
                salience=min(self.salience, other.salience)
            )
        return None


class ConceptualSpace:
    """
    概念空间：多个域的集合
    """

    def __init__(self):
        self.domains: Dict[str, ConceptualDomain] = {}
        self.concepts: Dict[str, Concept] = {}

    def add_domain(self, domain: ConceptualDomain):
        """添加域"""
        self.domains[domain.name] = domain

    def add_concept(self, concept: Concept):
        """添加概念"""
        self.concepts[concept.name] = concept

    def voronoi_partition(self, domain_name: str) -> Dict[str, List[np.ndarray]]:
        """
        计算Voronoi划分
        将域划分为基于原型的区域
        """
        domain = self.domains[domain_name]

        # 收集该域中的所有概念原型
        prototypes = []
        concept_names = []

        for name, concept in self.concepts.items():
            if concept.domain == domain:
                prototypes.append(concept.prototype)
                concept_names.append(name)

        if len(prototypes) < 2:
            return {name: [concept.prototype]
                   for name, concept in self.concepts.items()
                   if concept.domain == domain}

        prototypes = np.array(prototypes)

        # 计算Voronoi图（限于2D）
        if domain.n_dims == 2:
            vor = Voronoi(prototypes)

            regions = {}
            for i, name in enumerate(concept_names):
                # 找到对应区域
                region_idx = vor.point_region[i]
                vertices = vor.regions[region_idx]
                if -1 not in vertices:
                    regions[name] = [vor.vertices[v] for v in vertices]

            return regions
        else:
            # 高维：基于最近原型的划分
            regions = {name: [] for name in concept_names}
            # 这里可以添加采样点进行划分
            return regions

    def metaphor_map(self,
                     source_domain: str,
                     target_domain: str,
                     mapping: Dict[int, int]) -> Callable:
        """
        创建域之间的隐喻映射

        Args:
            mapping: 维度映射 {source_dim: target_dim}

        Returns:
            映射函数
        """
        source = self.domains[source_domain]
        target = self.domains[target_domain]

        def map_point(x: np.ndarray) -> np.ndarray:
            y = np.zeros(target.n_dims)
            for s_dim, t_dim in mapping.items():
                y[t_dim] = x[s_dim]
            return y

        return map_point


# ===== 颜色空间示例 =====
class ColorSpace:
    """
    颜色概念空间的实现
    基于HSV模型和Berlin & Kay的基本颜色词研究
    """

    def __init__(self):
        # 定义颜色域
        self.hue = QualityDimension('hue', 'circular', (0, 360), weight=1.0)
        self.saturation = QualityDimension('saturation', 'ratio', (0, 1), weight=0.5)
        self.value = QualityDimension('value', 'ratio', (0, 1), weight=0.5)

        self.domain = ConceptualDomain('color',
                                       [self.hue, self.saturation, self.value])
        self.space = ConceptualSpace()
        self.space.add_domain(self.domain)

        self._setup_focal_colors()

    def _setup_focal_colors(self):
        """设置Berlin & Kay的焦点色原型"""
        focal_colors = {
            'red': (0, 0.9, 0.9),
            'yellow': (60, 0.9, 0.9),
            'green': (120, 0.9, 0.9),
            'blue': (240, 0.9, 0.9),
            'purple': (300, 0.9, 0.8),
            'orange': (30, 0.9, 0.9),
            'brown': (30, 0.6, 0.4),
            'pink': (350, 0.4, 0.9),
            'white': (0, 0, 1.0),
            'black': (0, 0, 0),
            'grey': (0, 0, 0.5)
        }

        for name, (h, s, v) in focal_colors.items():
            concept = Concept(
                name=name,
                domain=self.domain,
                prototype=np.array([h, s, v]),
                salience=0.3
            )
            self.space.add_concept(concept)

    def name_color(self, hsv: Tuple[float, float, float]) -> str:
        """
        根据HSV值命名颜色
        找到最相似的概念
        """
        point = np.array(hsv)

        best_match = None
        best_sim = -1

        for name, concept in self.space.concepts.items():
            sim = concept.membership(point)
            if sim > best_sim:
                best_sim = sim
                best_match = name

        return best_match if best_sim > 0.5 else "unknown"

    def color_distance(self, c1: Tuple, c2: Tuple) -> float:
        """计算两种颜色的感知距离"""
        return self.domain.distance(np.array(c1), np.array(c2))

if __name__ == "__main__":
    print("=" * 60)
    print("概念空间理论示例")
    print("=" * 60)

    # 1. 颜色空间
    print("\n1. 颜色概念空间:")
    cs = ColorSpace()

    test_colors = [

        print(f"   HSV{hsv} -> {name}")

    # 2. Voronoi划分

    regions = cs.space.voronoi_partition('color')

    # 3. 概念相似性
    print("\n3. 概念间相似性:")
    red = cs.space.concepts['red']
    orange = cs.space.concepts['orange']
    blue = cs.space.concepts['blue']

    print(f"   red ~ orange: {red.similarity_to(orange):.3f}")
    print(f"   red ~ blue: {red.similarity_to(blue):.3f}")

    # 4. 概念插值（渐变）
    print("\n4. 颜色渐变 (red -> blue):")
    red_point = red.prototype
    blue_point = blue.prototype

    for t in [0, 0.25, 0.5, 0.75, 1.0]:
        interp = cs.domain.interpolate(red_point, blue_point, t)
        name = cs.name_color(tuple(interp))
        print(f"   t={t:.2f}: HSV({interp[0]:.0f}, {interp[1]:.2f}, {interp[2]:.2f}) -> {name}")
