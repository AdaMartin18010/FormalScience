## 13.5.3.1 引言

语言习得是认知科学的核心问题，涉及人类如何从零开始掌握复杂的语言系统。本节基于Chomsky的普遍语法理论、Gold的形式可学性结果，以及统计学习理论，形式化第一语言习得与语言学习机制。

> **参考**: Chomsky, N. (1986). _Knowledge of Language: Its Nature, Origin, and Use_. Praeger.
>
> Gold, E. M. (1967). "Language identification in the limit". _Information and Control_.

## 13.5.3.2 形式语言理论

### 13.5.3.2.1 Chomsky层级

**定义 13.5.3.1** (形式文法层级)

| 层级 | 文法 | 自动机 | 表达能力 |
|------|------|--------|----------|
| Type-3 | 正则文法 | 有限自动机 | $a^*$ |
| Type-2 | 上下文无关 | 下推自动机 | $a^n b^n$ |
| Type-1 | 上下文有关 | 线性有界 | $a^n b^n c^n$ |
| Type-0 | 无限制 | 图灵机 | 递归可枚举 |

### 13.5.3.2.2 自然语言的位置

**Chomsky观点**: 自然语言不是正则的（中心嵌入结构）。

**例**: 中心嵌入

```
The rat [the cat [the dog chased] killed] ate the malt.
```

**定理 13.5.3.1** (自然语言的上下文无关性争议)

Shieber (1985) 证明瑞士德语跨序列依赖不是上下文无关的：

$$
\{w w \mid w \in \{a,b\}^*\} \text{ 不是CFL}
$$

## 13.5.3.3 Gold定理

### 13.5.3.3.1 文本呈现

**定义 13.5.3.2** (正例文本)

语言 $L$ 的**文本**是 $L$ 中句子的枚举：

$$
t = (w_1, w_2, w_3, \ldots), \quad \forall i: w_i \in L
$$

### 13.5.3.3.2 极限识别

**定义 13.5.3.3** (极限识别)

算法 $A$ 在极限中识别语言 $L$ 如果：

$$
\exists N: \forall n \geq N: A(t_n) = G_L
$$

其中 $G_L$ 是生成 $L$ 的文法，$t_n$ 是前 $n$ 个例子。

### 13.5.3.3.3 Gold定理

**定理 13.5.3.2** (Gold, 1967)

**正例学习的不可能性**:

任何包含所有有限语言以及至少一个无限语言的**超有限类**都不是可学习的。

**推论**: 仅基于正例的语法归纳是不可能的。

**证明概要**:

设语言类 $\mathcal{L}$ 包含 $\{w\}$ 对所有 $w \in \Sigma^*$，以及 $\Sigma^*$。

给定文本 $t$，对于任何前缀，$\Sigma^*$ 始终是可能的假设，但无法确定何时停止。

## 13.5.3.4 普遍语法

### 13.5.3.4.1 原则与参数

**定义 13.5.3.4** (原则与参数框架)

普遍语法 (UG) 由**固定原则**和**待设参数**组成：

$$
\text{UG} = \text{Principles} \times \text{Parameters}
$$

**参数示例**:

| 参数 | 选项 | 语言 |
|------|------|------|
| 中心语方向 | 头初/头末 | 英语/日语 |
| 空主语 | +/- | 意大利语/英语 |
| 话题突出 | +/- | 汉语/英语 |

### 13.5.3.4.2 刺激贫乏论

**论点 13.5.3.1** (刺激贫乏)

儿童习得语言的输入不足以仅从数据诱导语法：

$$
\text{Input} \ll \text{Grammar Complexity}
$$

**证据**:

- 儿童没有系统的负例反馈
- 输入包含错误和不完整句子
- 习得速度（约3年掌握核心语法）

## 13.5.3.5 统计学习理论

### 13.5.3.5.1 过渡概率学习

**定义 13.5.3.5** (统计分割)

从连续语流中提取词汇单元：

$$
P(w_i \mid w_{i-1}) = \frac{C(w_{i-1}, w_i)}{C(w_{i-1})}
$$

低过渡概率指示词边界。

### 13.5.3.5.2 分布学习

**定义 13.5.3.6** (句法范畴的分布线索)

词 $w$ 的句法范畴由其上下文分布决定：

$$
\vec{v}_w = (P(\text{_ the } \mid w), P(\text{_ is } \mid w), P(\text{will _ } \mid w), \ldots)
$$

**聚类**: $\text{Category}(w) = \arg\min_k \|\vec{v}_w - \vec{c}_k\|$

## 13.5.3.6 语义引导

### 13.5.3.6.1 符号接地问题

**问题 13.5.3.1** (符号接地)

语言符号如何获得意义？

$$
\text{Symbol} \xrightarrow{\text{?}} \text{Meaning} \xrightarrow{\text{?}} \text{World}
$$

### 13.5.3.6.2 跨情境学习

**定义 13.5.3.7** (跨情境统计)

Yu & Smith (2008) 的跨情境词学习：

$$
P(\text{word} = \text{object}) \propto \sum_{\text{contexts}} P(\text{word} \mid \text{context}) \cdot P(\text{object} \mid \text{context})
$$

"""
语言习得的形式化实现
基于Chomsky的普遍语法、Gold的可学性理论和统计学习
"""

import numpy as np
from typing import List, Set, Dict, Tuple, Optional, Callable
from dataclasses import dataclass
from collections import defaultdict, Counter
import random
import re
from functools import lru_cache


class FormalGrammar:
    """
    形式文法
    实现Chomsky层级中的文法类型
    """

    def __init__(self, grammar_type: str = "regular"):
        """
        Args:
            grammar_type: 'regular', 'context_free', 'context_sensitive'
        """
        self.grammar_type = grammar_type
        self.rules: Dict[str, List[str]] = defaultdict(list)
        self.start_symbol = "S"
        self.terminals: Set[str] = set()
        self.non_terminals: Set[str] = set()

    def add_rule(self, lhs: str, rhs: str):
        """添加产生式规则"""
        self.rules[lhs].append(rhs)
        self.non_terminals.add(lhs)

        # 提取终结符和非终结符
        for symbol in rhs.split():
            if symbol.isupper():
                self.non_terminals.add(symbol)
            else:
                self.terminals.add(symbol)

    def generate(self, max_depth: int = 10) -> str:
        """
        从文法生成句子
        """
        def expand(symbol: str, depth: int) -> str:
            if depth > max_depth:
                return ""

            if symbol not in self.rules:
                return symbol

            # 随机选择产生式
            production = random.choice(self.rules[symbol])

            result = []
            for sym in production.split():
                result.append(expand(sym, depth + 1))

            return " ".join(result)

        return expand(self.start_symbol, 0)

    def recognize(self, sentence: str) -> bool:
        """
        检查句子是否属于语言
        (简化的CYK算法用于CFL)
        """
        # 简化的成员测试
        tokens = sentence.split()

        # 使用动态规划
        n = len(tokens)
        if n == 0:
            return False

        # 初始化解析表
        table = [[set() for _ in range(n)] for _ in range(n)]

        # 填充对角线（长度为1的子串）
        for i, token in enumerate(tokens):
            for lhs, rhs_list in self.rules.items():
                for rhs in rhs_list:
                    if rhs == token:
                        table[i][i].add(lhs)

        # 填充其余部分
        for length in range(2, n + 1):
            for i in range(n - length + 1):
                j = i + length - 1
                for k in range(i, j):
                    for lhs, rhs_list in self.rules.items():
                        for rhs in rhs_list:
                            parts = rhs.split()
                            if len(parts) == 2:
                                B, C = parts
                                if B in table[i][k] and C in table[k + 1][j]:
                                    table[i][j].add(lhs)

        return self.start_symbol in table[0][n - 1]


class GoldLanguageLearner:
    """
    Gold语言学习算法
    极限识别框架
    """

    def __init__(self, hypothesis_space: List[FormalGrammar]):
        """
        Args:
            hypothesis_space: 候选文法列表
        """
        self.hypothesis_space = hypothesis_space
        self.consistent_grammars: List[FormalGrammar] = list(hypothesis_space)
        self.current_hypothesis: Optional[FormalGrammar] = None
        self.examples_seen: List[str] = []
        self.converged = False

    def present_example(self, sentence: str, is_positive: bool = True):
        """
        呈现学习样例

        Args:
            sentence: 句子
            is_positive: 是否为正例
        """
        self.examples_seen.append(sentence)

        # 过滤不一致的假设
        new_consistent = []
        for grammar in self.consistent_grammars:
            recognized = grammar.recognize(sentence)

            if is_positive and recognized:
                new_consistent.append(grammar)
            elif not is_positive and not recognized:
                new_consistent.append(grammar)

        self.consistent_grammars = new_consistent

        # 更新当前假设（选择最简单的）
        if self.consistent_grammars:
            self.current_hypothesis = min(
                self.consistent_grammars,
                key=lambda g: len(g.rules)
            )

        # 检查是否收敛
        if len(self.consistent_grammars) == 1:
            self.converged = True

    def learn_from_text(self, text: List[str], max_examples: int = 1000):
        """
        从文本学习（仅正例）

        演示Gold定理：可能无法收敛
        """
        for i, sentence in enumerate(text[:max_examples]):
            self.present_example(sentence, is_positive=True)

            if self.converged:
                print(f"  收敛于第{i+1}个例子")
                return True

        print(f"  未收敛，剩余{len(self.consistent_grammars)}个一致假设")
        return False


class UniversalGrammar:
    """
    普遍语法原则与参数模型
    Chomsky的原则与参数框架简化实现
    """

    def __init__(self):
        # 固定原则（所有语言共有）
        self.principles = {
            'structure_dependency': True,  # 结构依赖
            'recursion': True,              # 递归性
            'discrete_infinity': True,      # 离散无限性
        }

        # 待设参数
        self.parameters = {
            'head_direction': None,     # 'initial' or 'final'
            'null_subject': None,       # True or False
            'topic_prominent': None,    # True or False
            'wh_movement': None,        # True or False
        }

        # 参数空间大小
        self.parameter_space_size = 2 ** len(self.parameters)

    def set_parameter(self, param: str, value):
        """设置参数值"""
        if param in self.parameters:
            self.parameters[param] = value

    def generate_language(self) -> FormalGrammar:
        """
        根据当前参数设置生成特定语法
        """
        grammar = FormalGrammar("context_free")

        # 根据head_direction生成不同的VP规则
        if self.parameters['head_direction'] == 'initial':
            # 英语型: VP -> V NP
            grammar.add_rule("VP", "V NP")
        else:
            # 日语型: VP -> NP V
            grammar.add_rule("VP", "NP V")

        # 添加其他规则
        grammar.add_rule("S", "NP VP")
        grammar.add_rule("NP", "Det N")
        grammar.add_rule("NP", "N")

        # 词汇
        grammar.add_rule("Det", "the")
        grammar.add_rule("Det", "a")
        grammar.add_rule("N", "cat")
        grammar.add_rule("N", "dog")
        grammar.add_rule("V", "chases")
        grammar.add_rule("V", "sees")

        return grammar

    def parameter_setting_process(self,
                                   input_sentences: List[str],
                                   n_iterations: int = 100):
        """
        参数设置过程
        模拟儿童根据输入触发参数设置
        """
        print("\n参数设置过程:")

        for sentence in input_sentences[:n_iterations]:
            # 分析句子特征来触发参数
            tokens = sentence.split()

            # 检测head direction
            if "chases" in tokens or "sees" in tokens:
                verb_idx = tokens.index("chases") if "chases" in tokens else tokens.index("sees")

                if verb_idx < len(tokens) - 1 and tokens[verb_idx + 1] in ["the", "a", "cat", "dog"]:
                    # V在NP前
                    if self.parameters['head_direction'] is None:
                        self.parameters['head_direction'] = 'initial'
                        print(f"  设置 head_direction = initial (基于: {sentence})")
                elif verb_idx > 0:
                    # V在NP后
                    if self.parameters['head_direction'] is None:
                        self.parameters['head_direction'] = 'final'
                        print(f"  设置 head_direction = final (基于: {sentence})")

        print(f"\n最终参数设置: {self.parameters}")


class StatisticalLanguageLearner:
    """
    统计语言学习
    基于分布学习和过渡概率
    """

    def __init__(self):
        self.word_counts: Counter = Counter()
        self.bigram_counts: Counter = Counter()
        self.word_contexts: Dict[str, Dict[str, int]] = defaultdict(lambda: defaultdict(int))
        self.vocabulary: Set[str] = set()

        # 学习到的词类
        self.word_classes: Dict[str, int] = {}
        self.n_classes = 0

    def learn_from_corpus(self, corpus: List[str]):
        """
        从语料库学习统计规律
        """
        for sentence in corpus:
            tokens = sentence.lower().split()

            # 统计词频
            self.word_counts.update(tokens)
            self.vocabulary.update(tokens)

            # 统计bigram
            for i in range(len(tokens) - 1):
                self.bigram_counts[(tokens[i], tokens[i + 1])] += 1

                # 记录上下文
                self.word_contexts[tokens[i + 1]]['prev_' + tokens[i]] += 1
                self.word_contexts[tokens[i]]['next_' + tokens[i + 1]] += 1

    def segment_words(self, continuous_input: str) -> List[str]:
        """
        词分割
        基于过渡概率
        """
        # 简化的分割：基于字符bigram统计
        chars = list(continuous_input)
        boundaries = [0]  # 词边界位置

        for i in range(1, len(chars) - 1):
            # 计算过渡概率
            bigram_count = self.bigram_counts.get((chars[i], chars[i + 1]), 0)
            char_count = self.word_counts.get(chars[i], 0)

            if char_count > 0:
                trans_prob = bigram_count / char_count
                # 低过渡概率 -> 词边界
                if trans_prob < 0.1:
                    boundaries.append(i + 1)

        boundaries.append(len(chars))

        # 分割
        words = []
        for i in range(len(boundaries) - 1):
            word = ''.join(chars[boundaries[i]:boundaries[i + 1]])
            words.append(word)

        return words

    def induce_syntactic_categories(self, n_categories: int = 3) -> Dict[str, int]:
        """
        归纳句法范畴
        基于分布相似性
        """
        # 为每个词构建上下文向量
        all_contexts = set()
        for contexts in self.word_contexts.values():
            all_contexts.update(contexts.keys())

        all_contexts = list(all_contexts)

        # 构建词-上下文矩阵
        word_vectors = {}
        for word in self.vocabulary:
            vector = np.zeros(len(all_contexts))
            for i, context in enumerate(all_contexts):
                vector[i] = self.word_contexts[word].get(context, 0)

            # 归一化
            norm = np.linalg.norm(vector)
            if norm > 0:
                vector = vector / norm
            word_vectors[word] = vector

        # K-means聚类
        from sklearn.cluster import KMeans

        words = list(word_vectors.keys())
        vectors = np.array([word_vectors[w] for w in words])

        kmeans = KMeans(n_clusters=n_categories, random_state=42)
        labels = kmeans.fit_predict(vectors)

        self.word_classes = {words[i]: labels[i] for i in range(len(words))}
        self.n_classes = n_categories

        return self.word_classes

    def predict_next_word(self, context: str) -> Dict[str, float]:
        """
        预测下一个词
        基于bigram概率
        """
        context = context.lower().split()[-1] if context else ""

        predictions = {}
        for (w1, w2), count in self.bigram_counts.items():
            if w1 == context:
                prob = count / self.word_counts[w1]
                predictions[w2] = prob

        # 归一化
        total = sum(predictions.values())
        if total > 0:
            predictions = {k: v / total for k, v in predictions.items()}

        return predictions


class CrossSituationalLearner:
    """
    跨情境词学习
    Yu & Smith (2008)
    """

    def __init__(self):
        self.word_object_associations: Dict[str, Dict[str, float]] = defaultdict(lambda: defaultdict(float))
        self.word_counts: Counter = Counter()
        self.object_counts: Counter = Counter()

    def observe_scene(self, words: List[str], objects: List[str]):
        """
        观察一个情境（词-场景配对）
        """
        # 更新计数
        self.word_counts.update(words)
        self.object_counts.update(objects)

        # 建立共现关联
        for word in words:
            for obj in objects:
                self.word_object_associations[word][obj] += 1

    def learn_mappings(self, threshold: float = 0.5) -> Dict[str, str]:
        """
        学习词-物体映射

        Returns:
            word -> object 映射
        """
        mappings = {}

        for word, object_counts in self.word_object_associations.items():
            if self.word_counts[word] > 0:
                # 计算条件概率 P(object|word)
                probs = {
                    obj: count / self.word_counts[word]
                    for obj, count in object_counts.items()
                }

                # 选择最可能的映射
                best_obj = max(probs, key=probs.get)
                if probs[best_obj] >= threshold:
                    mappings[word] = best_obj

        return mappings

    print("\n5. Gold可学性 (简化的有限语言类):")
    # 创建有限语言假设空间
    g1 = FormalGrammar("regular")
    g1.add_rule("S", "a")
    g1.add_rule("S", "aa")

    g2 = FormalGrammar("regular")
    g2.add_rule("S", "b")
    g2.add_rule("S", "bb")

    g3 = FormalGrammar("regular")  # 无限语言
    g3.add_rule("S", "a")
    g3.add_rule("S", "a S")

    gold_learner = GoldLanguageLearner([g1, g2, g3])

    # 从g1生成正例文本
    text = [g1.generate() for _ in range(20)]
    print(f"   训练文本 (来自g1): {text[:5]}")
    gold_learner.learn_from_text(text)
