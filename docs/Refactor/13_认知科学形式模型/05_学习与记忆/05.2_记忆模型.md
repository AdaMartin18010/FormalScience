## 13.5.2.1 引言

记忆是认知系统的信息存储与提取功能，涵盖编码、存储和检索三个核心过程。本节基于Atkinson-Shiffrin的多存储模型和Baddeley的工作记忆模型，形式化记忆的层级结构、容量限制与遗忘机制。

> **参考**: Baddeley, A. D. (2012). "Working memory: Theories, models, and controversies". _Annual Review of Psychology_.
>
> Cowan, N. (2001). "The magical number 4 in short-term memory". _Behavioral and Brain Sciences_.

## 13.5.2.2 记忆的多存储模型

### 13.5.2.2.1 记忆系统层级

**定义 13.5.2.1** (多存储模型)

Atkinson-Shiffrin的三存储模型：

```
┌─────────────────────────────────────────────────────────────┐
│                    感觉登记 (Sensory Register)               │
│  视觉 ( iconic ): ~250-500ms, 大容量                          │
│  听觉 ( echoic ): ~2-4s, 大容量                               │
└─────────────────────────────────────────────────────────────┘
                            ↓ 注意
┌─────────────────────────────────────────────────────────────┐
│                    短时记忆 (Short-Term Memory)              │
│  容量: 7±2 组块 (Miller), 实际约4个 (Cowan)                   │
│  持续时间: ~15-30s (无复述)                                   │
└─────────────────────────────────────────────────────────────┘
                            ↓ 复述/编码
┌─────────────────────────────────────────────────────────────┐
│                    长时记忆 (Long-Term Memory)               │
│  容量: 几乎无限                                              │
│  持续时间: 分钟到终身                                         │
└─────────────────────────────────────────────────────────────┘
```

### 13.5.2.2.2 容量限制

**定理 13.5.2.1** (Miller定律)

短时记忆的容量约为 $7 \pm 2$ 组块：

$$
C_{STM} = 7 \pm 2 \text{ chunks}
$$

**定理 13.5.2.2** (Cowan的聚焦容量)

注意聚焦的容量约为4个对象：

$$
C_{focus} \approx 4 \text{ items}
$$

## 13.5.2.3 工作记忆

### 13.5.2.3.1 多成分模型

**定义 13.5.2.2** (Baddeley工作记忆模型)

$$
\text{WM} = (\text{CE}, \text{PL}, \text{VSSP}, \text{EP})
$$

| 成分 | 全称 | 功能 |
|------|------|------|
| **CE** | Central Executive | 注意控制，协调子系统 |
| **PL** | Phonological Loop | 语音短时存储 |
| **VSSP** | Visuospatial Sketchpad | 视觉空间短时存储 |
| **EP** | Episodic Buffer | 多模态整合，连接LTM |

### 13.5.2.3.2 语音环路

**定义 13.5.2.3** (语音存储)

$$
\text{PL} = (\text{PS}, \text{ART})
$$

- **PS** (Phonological Store): 语音存储，约2秒衰减
- **ART** (Articulatory Rehearsal): 发音复述过程

**定理 13.5.2.3** (词长效应)

短时记忆广度与发音时长负相关：

$$
\text{Span} \propto \frac{1}{\text{Articulation Time}}
$$

### 13.5.2.3.3 中央执行系统

**定义 13.5.2.4** (注意控制)

中央执行的功能：

$$
\text{CE}(t) = f(\text{Attention}(t), \text{Inhibition}(t), \text{Switching}(t))
$$

## 13.5.2.4 长时记忆

### 13.5.2.4.1 陈述性记忆

**定义 13.5.2.5** (情景与语义)

| 类型 | 内容 | 神经基础 |
|------|------|----------|
| **情景记忆** | 个人经历，时空标记 | 海马体，前额叶 |
| **语义记忆** | 一般知识，事实 | 颞叶皮层 |

**定义 13.5.2.6** (编码特异性)

$$
\text{Retrieval} \propto \text{Match}(\text{Encoding Context}, \text{Retrieval Context})
$$

### 13.5.2.4.2 记忆痕迹巩固

**定理 13.5.2.4** (双迹理论)

$$
M(t) = M_{hippocampal}(t) + M_{cortical}(t)
$$

- 早期：海马体痕迹主导
- 晚期：皮层痕迹独立

### 13.5.2.4.3 遗忘曲线

**定义 13.5.2.7** (Ebbinghaus遗忘曲线)

$$
R(t) = R_0 \cdot e^{-\lambda t / \log(t+1)^\beta}
$$

或简化形式：

$$
R(t) = a \cdot (\log t + b)^{-c}
$$

## 13.5.2.5 记忆的形式模型

### 13.5.2.5.1 搜索模型

**定义 13.5.2.8** (SAM模型)

检索作为联想搜索：

$$
P(\text{Recall } I \mid C) = \frac{S(I, C)}{\sum_J S(J, C)}
$$

其中 $S(I, C)$ 为项 $I$ 与线索 $C$ 的关联强度。

### 13.5.2.5.2 痕迹强度

**定义 13.5.2.9** (痕迹强度与存储强度)

Bjork的区分：

$$
\text{Retrievability} = f(\text{Trace Strength}, \text{Storage Strength})
$$

- **痕迹强度**: 决定提取概率，随提取成功增加
- **存储强度**: 决定长期保留，随练习增加

### 13.5.2.5.3 分布式表征

**定义 13.5.2.10** (联结主义记忆)

记忆作为权重模式：

$$
\mathbf{w}^* = \arg\min_{\mathbf{w}} \sum_i (\mathbf{y}_i - f(\mathbf{w}, \mathbf{x}_i))^2
$$

**定理 13.5.2.5** (Hopfield存储容量)

$$
C_{max} \approx 0.14 \cdot N
$$

其中 $N$ 为单元数。

"""
记忆模型的形式化实现
基于多存储模型、工作记忆和联结主义框架
"""

import numpy as np
from typing import List, Dict, Tuple, Optional, Any, Set
from dataclasses import dataclass, field
from collections import deque, defaultdict
from datetime import datetime, timedelta
import time
import random


@dataclass
class MemoryItem:
    """
    记忆项目
    """
    content: Any
    timestamp: float
    context: Dict[str, Any] = field(default_factory=dict)
    rehearsal_count: int = 0
    emotional_tag: float = 0.0  # -1到1

    def age(self, current_time: float) -> float:
        """记忆的年龄"""
        return current_time - self.timestamp

    def strength(self, current_time: float,
                decay_rate: float = 0.1) -> float:
        """
        计算当前痕迹强度
        基于时间衰减和复述
        """
        age = self.age(current_time)
        # Ebbinghaus型遗忘
        time_decay = np.exp(-decay_rate * age / (np.log(age + 2)))
        # 复述增强
        rehearsal_boost = 1 + 0.5 * self.rehearsal_count
        # 情绪增强
        emotion_boost = 1 + abs(self.emotional_tag)

        return time_decay * rehearsal_boost * emotion_boost


class SensoryRegister:
    """
    感觉登记
    大容量、极短持续时间
    """

    def __init__(self, modality: str, duration_ms: float):
        self.modality = modality  # 'visual', 'auditory'
        self.duration = duration_ms / 1000.0  # 转换为秒
        self.buffer: List[MemoryItem] = []
        self.last_update = time.time()

    def present(self, stimulus: Any, context: Dict = None):
        """呈现刺激"""
        item = MemoryItem(
            content=stimulus,
            timestamp=time.time(),
            context=context or {}
        )
        self.buffer.append(item)
        self._decay()

    def _decay(self):
        """衰减过期的感觉痕迹"""
        current_time = time.time()
        self.buffer = [
            item for item in self.buffer
            if current_time - item.timestamp < self.duration
        ]

    def attend(self, selector: Callable = None) -> Optional[MemoryItem]:
        """
        注意转移，将项目转移到短时记忆

        Args:
            selector: 选择函数，None则随机选择
        """
        self._decay()
        if not self.buffer:
            return None

        if selector:
            item = selector(self.buffer)
        else:
            item = random.choice(self.buffer)

        # 从感觉登记移除
        self.buffer.remove(item)
        return item

    def get_iconic_memory(self) -> List[Any]:
        """获取视觉感觉登记内容"""
        self._decay()
        return [item.content for item in self.buffer]


class ShortTermMemory:
    """
    短时记忆
    有限容量，快速衰减
    """

    def __init__(self, capacity: int = 7, decay_time: float = 30.0):
        """
        Args:
            capacity: 容量 (默认7±2)
            decay_time: 衰减时间（秒）
        """
        self.capacity = capacity
        self.decay_time = decay_time
        self.items: deque = deque(maxlen=capacity)
        self.rehearsal_buffer: Set[int] = set()  # 正在复述的项目索引

    def add(self, item: MemoryItem) -> bool:
        """
        添加项目到STM

        Returns:
            是否成功添加（如果已满，可能需要替换）
        """
        self._decay()

        if len(self.items) >= self.capacity:
            # 移除最旧的项目（除非正在复述）
            removed = False
            for old_item in list(self.items):
                if id(old_item) not in self.rehearsal_buffer:
                    self.items.remove(old_item)
                    removed = True
                    break

            if not removed:
                return False  # 无法添加，所有项目都在复述

        self.items.append(item)
        return True

    def _decay(self):
        """移除过期的项目"""
        current_time = time.time()
        self.items = deque(
            [item for item in self.items
             if current_time - item.timestamp < self.decay_time],
            maxlen=self.capacity
        )

    def rehearse(self, item_id: int):
        """
        复述项目
        重置时间戳并增加复述计数
        """
        for item in self.items:
            if id(item) == item_id:
                item.rehearsal_count += 1
                item.timestamp = time.time()  # 刷新
                self.rehearsal_buffer.add(item_id)
                break

    def stop_rehearsal(self, item_id: int):
        """停止复述"""
        self.rehearsal_buffer.discard(item_id)

    def get_contents(self) -> List[Any]:
        """获取当前内容"""
        self._decay()
        return [item.content for item in self.items]

    def is_full(self) -> bool:
        """检查是否已满"""
        return len(self.items) >= self.capacity


class WorkingMemory:
    """
    工作记忆模型
    Baddeley多成分模型
    """

    def __init__(self,
                 phonological_capacity: int = 7,
                 visuospatial_capacity: int = 4,
                 central_capacity: int = 4):
        self.phonological_loop = ShortTermMemory(phonological_capacity)
        self.visuospatial_sketchpad = ShortTermMemory(visuospatial_capacity)
        self.episodic_buffer: deque = deque(maxlen=central_capacity)

        # 中央执行系统状态
        self.attention_focus: Optional[Any] = None
        self.inhibition_active: Set[Any] = set()

    def phonological_store(self, verbal_item: Any):
        """存储到语音环路"""
        item = MemoryItem(
            content=verbal_item,
            timestamp=time.time()
        )
        self.phonological_loop.add(item)

    def visuospatial_store(self, visual_item: Any):
        """存储到视觉空间画板"""
        item = MemoryItem(
            content=visual_item,
            timestamp=time.time()
        )
        self.visuospatial_sketchpad.add(item)

    def central_executive(self, operation: str, *args):
        """
        中央执行系统操作

        Args:
            operation: 'focus', 'switch', 'inhibit', 'integrate'
        """
        if operation == 'focus':
            self.attention_focus = args[0]
        elif operation == 'switch':
            self.attention_focus = args[0]
            self.inhibition_active.clear()
        elif operation == 'inhibit':
            self.inhibition_active.add(args[0])
        elif operation == 'integrate':
            # 整合多模态信息到情景缓冲器
            integrated = {
                'phonological': self.phonological_loop.get_contents(),
                'visuospatial': self.visuospatial_sketchpad.get_contents(),
                'focus': self.attention_focus
            }
            self.episodic_buffer.append(integrated)

    def word_length_effect(self, words: List[str]) -> int:
        """
        演示词长效应
        长单词的记忆广度小于短单词
        """
        # 简单模拟：假设发音时间 ∝ 音节数
        articulation_times = [len(word) * 0.3 for word in words]

        # 在2秒循环内可容纳的单词数
        loop_duration = 2.0
        capacity = 0
        total_time = 0

        for art_time in articulation_times:
            if total_time + art_time <= loop_duration:
                total_time += art_time
                capacity += 1
            else:
                break

        return capacity


class LongTermMemory:
    """
    长时记忆
    大容量，持久存储
    """

    def __init__(self,
                 consolidation_rate: float = 0.1,
                 retrieval_noise: float = 0.1):
        self.storage: Dict[str, MemoryItem] = {}
        self.consolidation_rate = consolidation_rate
        self.retrieval_noise = retrieval_noise

        # 语义网络
        self.semantic_links: Dict[str, Set[str]] = defaultdict(set)

    def encode(self, item: MemoryItem,
               rehearsal_count: int = 0) -> str:
        """
        编码项目到LTM

        Returns:
            存储键
        """
        # 增强存储强度
        item.rehearsal_count += rehearsal_count

        key = f"item_{id(item)}_{time.time()}"
        self.storage[key] = item

        return key

    def consolidate(self, stm_item: MemoryItem,
                   context: Dict) -> str:
        """
        从STM巩固到LTM
        模拟睡眠/时间的记忆巩固
        """
        # 添加上下文信息
        stm_item.context.update(context)

        # 存储强度增长
        storage_strength = 1 + self.consolidation_rate * stm_item.rehearsal_count

        key = self.encode(stm_item)

        # 建立语义链接
        for related_key, related_item in self.storage.items():
            if related_key != key:
                similarity = self._compute_similarity(stm_item, related_item)
                if similarity > 0.7:
                    self.semantic_links[key].add(related_key)

        return key

    def _compute_similarity(self, item1: MemoryItem,
                           item2: MemoryItem) -> float:
        """计算两个项目的语义相似性"""
        # 简化的相似性计算
        if type(item1.content) != type(item2.content):
            return 0.0

        # 基于内容的相似性
        if isinstance(item1.content, str):
            words1 = set(item1.content.lower().split())
            words2 = set(item2.content.lower().split())
            intersection = len(words1 & words2)
            union = len(words1 | words2)
            return intersection / union if union > 0 else 0.0

        return 0.0

    def retrieve(self, cue: Any,
                context: Dict = None,
                method: str = 'cue') -> List[Tuple[str, float]]:
        """
        检索项目

        Args:
            cue: 检索线索
            context: 上下文匹配
            method: 'cue', 'associative', 'free'

        Returns:
            (项目键, 激活水平) 列表
        """
        results = []
        current_time = time.time()

        for key, item in self.storage.items():
            # 计算痕迹强度
            trace_strength = item.strength(current_time)

            # 上下文匹配
            context_match = 1.0
            if context:
                matches = sum(1 for k, v in context.items()
                            if item.context.get(k) == v)
                context_match = matches / len(context) if context else 1.0

            # 线索相似性
            cue_similarity = self._cue_match(cue, item)

            # 总激活水平
            activation = trace_strength * context_match * cue_similarity

            # 添加噪声
            activation += random.gauss(0, self.retrieval_noise)

            if activation > 0.1:  # 阈值
                results.append((key, activation))

        # 按激活排序
        results.sort(key=lambda x: x[1], reverse=True)
        return results

    def _cue_match(self, cue: Any, item: MemoryItem) -> float:
        """计算线索匹配度"""
        if cue == item.content:
            return 1.0
        if isinstance(cue, str) and isinstance(item.content, str):
            return 1.0 if cue.lower() in item.content.lower() else 0.0
        return 0.0

    def recall_probability(self, key: str,
                          time_since_encoding: float) -> float:
        """
        计算给定时间的回忆概率
        基于痕迹强度和遗忘曲线
        """
        if key not in self.storage:
            return 0.0

        item = self.storage[key]
        current_time = item.timestamp + time_since_encoding
        strength = item.strength(current_time)

        # sigmoid转换到概率
        return 1 / (1 + np.exp(-5 * (strength - 0.5)))


class MemorySystem:
    """
    完整记忆系统
    整合感觉登记、STM、WM和LTM
    """

    def __init__(self):
        self.visual_register = SensoryRegister('visual', 250)
        self.auditory_register = SensoryRegister('auditory', 2000)
        self.short_term_memory = ShortTermMemory(capacity=7)
        self.working_memory = WorkingMemory()
        self.long_term_memory = LongTermMemory()

    def process_stimulus(self, stimulus: Any,
                        modality: str = 'visual',
                        attend: bool = True):
        """
        处理刺激通过记忆系统
        """
        # 1. 感觉登记
        if modality == 'visual':
            self.visual_register.present(stimulus)
            if attend:
                item = self.visual_register.attend()
        else:
            self.auditory_register.present(stimulus)
            if attend:
                item = self.auditory_register.attend()

        if attend and 'item' in dir() and item:
            # 2. 进入短时记忆
            success = self.short_term_memory.add(item)

            if success:
                # 3. 工作记忆处理
                if modality == 'visual':
                    self.working_memory.visuospatial_store(item.content)
                else:
                    self.working_memory.phonological_store(item.content)

            return item

        return None

    rehearse_item = None
    def rehearse(self, item_content: Any):
        """复述项目"""
        for item in self.short_term_memory.items:
            if item.content == item_content:
                self.short_term_memory.rehearse(id(item))
                item.rehearsal_count += 1

                # 巩固到LTM的机会
                if item.rehearsal_count >= 3:
                    self.long_term_memory.consolidate(
                        item,
                        context={'source': 'rehearsal'}
                    )
                break

    def recall(self, cue: Any) -> List[Any]:
        """从LTM回忆"""
        results = self.long_term_memory.retrieve(cue)
        return [self.long_term_memory.storage[key].content
                for key, _ in results[:5]]

    keys = []
    for item in items:
        key = ltm.encode(item, rehearsal_count=item.rehearsal_count)
        keys.append(key)
        print(f"   编码: {item.content} (复述{item.rehearsal_count}次)")

    # 模拟不同时间的回忆概率
    print("\n   不同时间的回忆概率:")
    for delay in [1, 60, 600, 3600]:  # 1秒, 1分, 10分, 1小时
        print(f"   t={delay}s后:")
        for i, key in enumerate(keys):
            prob = ltm.recall_probability(key, delay)
            content = items[i].content
            print(f"      {content}: P(recall)={prob:.3f}")

    # 4. 完整记忆系统演示
    print("\n4. 完整记忆系统:")
    memory = MemorySystem()

    # 处理一系列刺激
    stimuli = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J']
    for s in stimuli:
        memory.process_stimulus(s, attend=True)

    print(f"   处理{len(stimuli)}个刺激后STM: {memory.short_term_memory.get_contents()}")

    # 复述
    memory.rehearse('H')
    memory.rehearse('I')
    memory.rehearse('J')

    print(f"   复述后STM: {memory.short_term_memory.get_contents()}")
