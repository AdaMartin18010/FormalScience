## 13.5.1.1 引言

学习是认知系统的核心功能，涉及知识获取、技能掌握和行为适应。本节基于Rescorla-Wagner模型、贝叶斯学习理论和强化学习框架，形式化经典条件作用、工具性条件作用和统计学习的数学原理。

> **参考**: Rescorla, R. A., & Wagner, A. R. (1972). "A theory of Pavlovian conditioning: Variations in the effectiveness of reinforcement and nonreinforcement". In _Classical Conditioning II_.
>
> Sutton, R. S., & Barto, A. G. (2018). _Reinforcement Learning: An Introduction_. MIT Press.

## 13.5.1.2 Rescorla-Wagner模型

### 13.5.1.2.1 基本方程

**定义 13.5.1.1** (R-W学习规则)

联想强度 $V$ 的更新：

$$
\Delta V_{CS} = \alpha \cdot \beta \cdot (\lambda - V_{total})
$$

其中：

- $\alpha$: CS显著性（可学习性）
- $\beta$: US强度/学习率参数
- $\lambda$: US支持的最大条件作用
- $V_{total} = \sum_i V_{CS_i}$: 所有CS的联想强度之和

### 13.5.1.2.2 阻塞效应

**定理 13.5.1.1** (阻塞)

先验条件化的CS会阻塞对新CS的学习：

若 $V_A \approx \lambda$，则 $\Delta V_B \approx 0$ 在 $A+B \rightarrow US$ 训练中。

**证明**:

$$
V_{total} = V_A + V_B \approx \lambda + V_B \Rightarrow \lambda - V_{total} \approx -V_B
$$

当 $V_B$ 很小时，$\Delta V_B \approx 0$。

### 13.5.1.2.3 预测误差

**定义 13.5.1.2** (预测误差)

$$\delta = \lambda - V_{total}$$

**解释**: 强化学习中的TD误差前身。

## 13.5.1.3 贝叶斯学习

### 13.5.1.3.1 参数学习

**定义 13.5.1.3** (贝叶斯参数更新)

$$
P(\theta \mid D) = \frac{P(D \mid \theta) P(\theta)}{\int P(D \mid \theta') P(\theta') d\theta'}
$$

### 13.5.1.3.2 共轭先验

**定理 13.5.1.2** (Beta-Binomial共轭)

若 $\theta \sim \text{Beta}(\alpha, \beta)$，观测 $k$ 次成功于 $n$ 次试验：

$$
\theta \mid D \sim \text{Beta}(\alpha + k, \beta + n - k)
$$

**后验均值**: $\mathbb{E}[\theta \mid D] = \frac{\alpha + k}{\alpha + \beta + n}$

### 13.5.1.3.3 累积预测

**定义 13.5.1.4** (预测分布)

$$
P(x_{n+1} \mid D) = \int P(x_{n+1} \mid \theta) P(\theta \mid D) d\theta
$$

## 13.5.1.4 强化学习

### 13.5.1.4.1 马尔可夫决策过程

**定义 13.5.1.5** (MDP)

马尔可夫决策过程五元组 $(S, A, T, R, \gamma)$：

| 符号 | 含义 | 描述 |
|------|------|------|
| $S$ | 状态空间 | 环境状态的集合 |
| $A$ | 动作空间 | 可用动作的集合 |
| $T(s' \mid s, a)$ | 转移概率 | 状态转移函数 |
| $R(s, a)$ | 奖励函数 | 即时奖励 |
| $\gamma$ | 折扣因子 | 未来奖励的折现率 |

### 13.5.1.4.2 价值函数

**定义 13.5.1.6** (状态价值函数)

$$
V^\pi(s) = \mathbb{E}\left[\sum_{t=0}^{\infty} \gamma^t R(s_t, a_t) \mid s_0 = s, \pi\right]
$$

**定义 13.5.1.7** (动作价值函数)

$$
Q^\pi(s, a) = R(s, a) + \gamma \sum_{s'} T(s' \mid s, a) V^\pi(s')
$$

### 13.5.1.4.3 Bellman方程

**定理 13.5.1.3** (最优Bellman方程)

$$
V^*(s) = \max_a \left[ R(s, a) + \gamma \sum_{s'} T(s' \mid s, a) V^*(s') \right]
$$

$$
Q^*(s, a) = R(s, a) + \gamma \sum_{s'} T(s' \mid s, a) \max_{a'} Q^*(s', a')
$$

## 13.5.1.5 时序差分学习

### 13.5.1.5.1 TD学习

**定义 13.5.1.8** (TD学习规则)

$$
V(s_t) \leftarrow V(s_t) + \alpha [R_{t+1} + \gamma V(s_{t+1}) - V(s_t)]
$$

**TD误差**: $\delta_t = R_{t+1} + \gamma V(s_{t+1}) - V(s_t)$

### 13.5.1.5.2 Q-Learning

**定义 13.5.1.9** (Q-Learning更新)

$$
Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha [R_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t)]
$$

**定理 13.5.1.4** (Q-Learning收敛)

在适当条件下（所有状态-动作对被无限访问，学习率递减），Q-Learning收敛于 $Q^*$。

## 13.5.1.6 统计学习

### 13.5.1.6.1 过渡概率学习

**定义 13.5.1.10** (隐马尔可夫模型)

序列学习的隐马尔可夫模型：

$$
P(s_{1:T}) = P(s_1) \prod_{t=2}^{T} P(s_t \mid s_{t-1})
$$

### 13.5.1.6.2 神经预测编码

**定义 13.5.1.11** (预测误差最小化)

学习通过最小化预测误差实现：

$$
\mathcal{E} = \sum_t \|o_t - \hat{o}_t\|^2
$$

## 13.5.1.7 Python实现

```python
"""
学习理论形式化实现
基于Rescorla-Wagner、贝叶斯学习和强化学习
"""

import numpy as np
from typing import List, Tuple, Dict, Callable, Optional
from dataclasses import dataclass
from collections import defaultdict
import random


class RescorlaWagner:
    """
    Rescorla-Wagner模型
    经典条件作用的形式化模型
    """

    def __init__(self,
                 n_cues: int,
                 alpha: float = 0.1,  # 学习率
                 beta: float = 1.0,   # US强度
                 lambda_max: float = 1.0):  # 渐近水平
        self.n_cues = n_cues
        self.alpha = alpha
        self.beta = beta
        self.lambda_max = lambda_max

        # 联想强度
        self.V = np.zeros(n_cues)

        # 历史记录
        self.history: List[Dict] = []

    def trial(self,
              cues: np.ndarray,  # 二值向量，表示哪些CS呈现
              us_present: bool) -> float:
        """
        单次试验

        Args:
            cues: CS呈现向量 (0/1)
            us_present: US是否呈现

        Returns:
            预测误差
        """
        # 总联想强度
        V_total = np.sum(self.V * cues)

        # 渐近水平
        lambda_val = self.lambda_max if us_present else 0.0

        # 预测误差
        delta = lambda_val - V_total

        # 更新联想强度
        for i in range(self.n_cues):
            if cues[i] > 0:
                self.V[i] += self.alpha * self.beta * delta

        # 记录
        self.history.append({
            'cues': cues.copy(),
            'us': us_present,
            'V_total': V_total,
            'delta': delta,
            'V': self.V.copy()
        })

        return delta

    def train(self, trials: List[Tuple[np.ndarray, bool]]):
        """训练序列"""
        for cues, us in trials:
            self.trial(cues, us)

    def blocking_demonstration(self):
        """
        演示阻塞效应
        阶段1: A -> US (直到V_A ≈ λ)
        阶段2: AB -> US (V_B保持接近0)
        """
        print("\n=== 阻塞效应演示 ===")

        # 阶段1: A单独训练
        print("阶段1: A -> US")
        for i in range(20):
            cues = np.array([1, 0])  # 只有A
            delta = self.trial(cues, us_present=True)
            if i < 5 or i % 5 == 0:
                print(f"  Trial {i+1}: V_A={self.V[0]:.3f}, V_B={self.V[1]:.3f}, δ={delta:.3f}")

        # 阶段2: AB复合训练
        print("\n阶段2: AB -> US")
        for i in range(20):
            cues = np.array([1, 1])  # A和B
            delta = self.trial(cues, us_present=True)
            if i < 5 or i % 5 == 0:
                print(f"  Trial {i+1}: V_A={self.V[0]:.3f}, V_B={self.V[1]:.3f}, δ={delta:.3f}")

        print(f"\n最终结果: V_A={self.V[0]:.3f}, V_B={self.V[1]:.3f}")
        print("B的联想强度被阻塞！")

    def overshadowing_demonstration(self):
        """
        演示遮蔽效应
        AB同时训练，较强的CS获得更多联想
        """
        print("\n=== 遮蔽效应演示 ===")

        # 重置
        self.V = np.zeros(2)
        self.alpha = 0.1

        # A和B同时训练，但A有更高的显著性
        print("AB同时训练（A显著性=0.1, B显著性=0.05）")

        for i in range(30):
            cues = np.array([1, 1])

            # 临时调整学习率以模拟不同显著性
            orig_alpha = self.alpha

            # A的学习率
            self.alpha = 0.1
            V_A_total = self.V[0] * cues[0]
            delta_A = self.lambda_max - (V_A_total + self.V[1])
            self.V[0] += self.alpha * self.beta * delta_A

            # B的学习率
            self.alpha = 0.05
            V_B_total = self.V[1] * cues[1]
            delta_B = self.lambda_max - (self.V[0] + V_B_total)
            self.V[1] += self.alpha * self.beta * delta_B

            self.alpha = orig_alpha

            if i < 5 or i % 5 == 0:
                print(f"  Trial {i+1}: V_A={self.V[0]:.3f}, V_B={self.V[1]:.3f}")

        print(f"\n最终结果: V_A={self.V[0]:.3f}, V_B={self.V[1]:.3f}")
        print("显著性更高的A获得更多联想强度")


class BayesianLearner:
    """
    贝叶斯学习模型
    Beta-Binomial共轭先验
    """

    def __init__(self, alpha: float = 1.0, beta: float = 1.0):
        """
        Beta(α, β)先验
        """
        self.alpha_prior = alpha
        self.beta_prior = beta
        self.alpha_post = alpha
        self.beta_post = beta

        self.n_trials = 0
        self.n_successes = 0

    def observe(self, success: bool):
        """
        观察结果

        Args:
            success: 是否成功/正面结果
        """
        self.n_trials += 1
        if success:
            self.n_successes += 1
            self.alpha_post += 1
        else:
            self.beta_post += 1

    def posterior_mean(self) -> float:
        """后验均值"""
        return self.alpha_post / (self.alpha_post + self.beta_post)

    def posterior_mode(self) -> float:
        """后验众数 (MAP)"""
        if self.alpha_post > 1 and self.beta_post > 1:
            return (self.alpha_post - 1) / (self.alpha_post + self.beta_post - 2)
        return self.posterior_mean()

    def posterior_variance(self) -> float:
        """后验方差"""
        a, b = self.alpha_post, self.beta_post
        return (a * b) / ((a + b)**2 * (a + b + 1))

    def credible_interval(self, level: float = 0.95) -> Tuple[float, float]:
        """
        计算可信区间
        使用Beta分布的分位数
        """
        from scipy.stats import beta
        lower = (1 - level) / 2
        upper = 1 - lower
        return (beta.ppf(lower, self.alpha_post, self.beta_post),
                beta.ppf(upper, self.alpha_post, self.beta_post))

    def predict(self) -> float:
        """
        预测下一个结果的概率
        (后验预测分布的期望)
        """
        return self.posterior_mean()

    def surprise(self, success: bool) -> float:
        """
        计算观察的惊讶度
        -log P(结果|先前信念)
        """
        pred = self.predict()
        if success:
            return -np.log(pred + 1e-10)
        else:
            return -np.log(1 - pred + 1e-10)


class QLearning:
    """
    Q-Learning算法
    离策略时序差分控制
    """

    def __init__(self,
                 n_states: int,
                 n_actions: int,
                 alpha: float = 0.1,   # 学习率
                 gamma: float = 0.9,   # 折扣因子
                 epsilon: float = 0.1): # ε-贪心探索
        self.n_states = n_states
        self.n_actions = n_actions
        self.alpha = alpha
        self.gamma = gamma
        self.epsilon = epsilon

        # Q表
        self.Q = np.zeros((n_states, n_actions))

        # 历史
        self.episode_rewards: List[float] = []

    def select_action(self, state: int, explore: bool = True) -> int:
        """
        ε-贪心动作选择
        """
        if explore and random.random() < self.epsilon:
            return random.randint(0, self.n_actions - 1)
        else:
            return int(np.argmax(self.Q[state]))

    def update(self,
               state: int,
               action: int,
               reward: float,
               next_state: int,
               done: bool):
        """
        Q-Learning更新
        """
        # 当前Q值
        current_q = self.Q[state, action]

        # 目标Q值
        if done:
            target = reward
        else:
            target = reward + self.gamma * np.max(self.Q[next_state])

        # TD误差
        td_error = target - current_q

        # 更新
        self.Q[state, action] += self.alpha * td_error

        return td_error

    def train_episode(self,
                     env: Callable[[int, int], Tuple[int, float, bool]],
                     start_state: int,
                     max_steps: int = 100) -> float:
        """
        训练一个回合

        Args:
            env: 环境函数 (state, action) -> (next_state, reward, done)
            start_state: 初始状态

        Returns:
            回合总奖励
        """
        state = start_state
        total_reward = 0.0

        for step in range(max_steps):
            action = self.select_action(state)
            next_state, reward, done = env(state, action)

            self.update(state, action, reward, next_state, done)

            total_reward += reward
            state = next_state

            if done:
                break

        self.episode_rewards.append(total_reward)
        return total_reward


class StatisticalLearning:
    """
    统计学习模型
    过渡概率和序列学习
    """

    def __init__(self, n_elements: int):
        self.n_elements = n_elements

        # 转移计数
        self.transition_counts = np.zeros((n_elements, n_elements))
        self.element_counts = np.zeros(n_elements)

        # 转移概率 (学习到的)
        self.P = np.ones((n_elements, n_elements)) / n_elements

    def observe_sequence(self, sequence: List[int]):
        """
        观察序列并更新转移概率
        """
        for i in range(len(sequence) - 1):
            from_elem = sequence[i]
            to_elem = sequence[i + 1]

            self.transition_counts[from_elem, to_elem] += 1
            self.element_counts[from_elem] += 1

        # 更新概率估计 (加平滑)
        for i in range(self.n_elements):
            if self.element_counts[i] > 0:
                self.P[i] = (self.transition_counts[i] + 1) / \
                           (self.element_counts[i] + self.n_elements)

    def predict_next(self, current: int) -> np.ndarray:
        """
        预测下一个元素的概率分布
        """
        return self.P[current]

    def sequence_surprise(self, sequence: List[int]) -> float:
        """
        计算序列的惊讶度 (负对数似然)
        """
        surprise = 0.0
        for i in range(len(sequence) - 1):
            prob = self.P[sequence[i], sequence[i + 1]]
            surprise -= np.log(prob + 1e-10)
        return surprise

    def detect_boundary(self, sequence: List[int],
                       position: int,
                       window: int = 3) -> float:
        """
        检测序列边界
        边界处的过渡概率通常较低

        Returns:
            边界概率 (0-1)
        """
        if position <= 0 or position >= len(sequence):
            return 0.0

        # 边界前后的过渡概率差异
        before_prob = self.P[sequence[position-1], sequence[position]]

        # 如果该位置是边界，下一个过渡应该是新单元的开始
        # 我们检查该元素作为序列起始的概率
        start_prob = self.element_counts[sequence[position]] / \
                    np.sum(self.element_counts)

        # 边界指标：低过渡概率 + 高起始概率
        boundary_score = (1 - before_prob) * start_prob

        return boundary_score

    print("=" * 60)
    print("学习理论形式化示例")

    # 2. 贝叶斯学习
    print("\n\n2. 贝叶斯学习 (Beta-Binomial):")

    # 模拟抛硬币
    print("模拟抛硬币（真实p=0.7）")
    true_prob = 0.7

        result = random.random() < true_prob
        bayes.observe(result)

        if i in [0, 4, 9, 19]:
            mean = bayes.posterior_mean()
            ci = bayes.credible_interval()
            print(f"  观测{i+1}次后: 估计p={mean:.3f}, "
                  f"95% CI=[{ci[0]:.3f}, {ci[1]:.3f}]")

    # 3. Q-Learning
    print("\n3. Q-Learning (简单网格世界):")

    # 简单的3状态链
    # 0 -> 1 -> 2 (goal)
    def simple_env(state: int, action: int) -> Tuple[int, float, bool]:
        # 动作0: 向左, 动作1: 向右
        if action == 0 and state > 0:
            next_state = state - 1
        elif action == 1 and state < 2:
            next_state = state + 1
        else:
            next_state = state

        reward = 1.0 if next_state == 2 else 0.0
        done = (next_state == 2)

        return next_state, reward, done

    ql = QLearning(n_states=3, n_actions=2, alpha=0.2, gamma=0.9, epsilon=0.2)

    for episode in range(50):
        reward = ql.train_episode(simple_env, start_state=0)
        if episode % 10 == 0:
            print(f"  Episode {episode}: Q-table = {ql.Q}")

    print(f"\n  学习到的Q值:")
    print(f"  State 0: Q(left)={ql.Q[0,0]:.3f}, Q(right)={ql.Q[0,1]:.3f}")
    print(f"  State 1: Q(left)={ql.Q[1,0]:.3f}, Q(right)={ql.Q[1,1]:.3f}")

    # 4. 统计学习
    print("\n4. 统计学习 (序列过渡概率):")
    sl = StatisticalLearning(n_elements=3)

    # 训练序列: 0->1->2重复
    training = [0, 1, 2, 0, 1, 2, 0, 1, 2, 0, 1, 2]
    sl.observe_sequence(training)

    print("学习到的转移概率矩阵:")
    print(sl.P)

    # 测试序列惊讶度
    test_familiar = [0, 1, 2, 0, 1]  # 熟悉的模式
    test_unusual = [0, 2, 1, 0, 2]   # 不熟悉的模式

    print(f"\n  熟悉序列惊讶度: {sl.sequence_surprise(test_familiar):.3f}")
