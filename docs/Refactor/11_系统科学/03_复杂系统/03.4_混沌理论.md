## 11.12.1 混沌的定义

### 11.12.1.1 Devaney混沌

**定义 11.12.1**（Devaney混沌）：映射 $f: X \to X$ 是混沌的，若：

1. **初值敏感依赖**：$\exists \delta > 0$，对任意 $x$ 和邻域 $N$，$\exists y \in N$ 和 $n \geq 0$ 使得 $|f^n(x) - f^n(y)| > \delta$

2. **拓扑传递性**：对任意开集 $U, V \subset X$，$\exists n > 0$ 使 $f^n(U) \cap V \neq \emptyset$

3. **周期点稠密**：周期点在 $X$ 中稠密

**定理 11.12.1**（Banks et al., 1992）：周期点稠密 + 拓扑传递性 $\Rightarrow$ 初值敏感依赖

### 11.12.1.2 Lyapunov指数

**定义 11.12.2**（Lyapunov指数）：相邻轨迹的指数分离率：

$$\lambda = \lim_{t \to \infty} \lim_{\|\delta x_0\| \to 0} \frac{1}{t} \ln \frac{\|\delta x(t)\|}{\|\delta x_0\|}$$

**定义 11.12.3**（Lyapunov谱）：多维系统的Lyapunov指数集合 $\{\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_n\}$。

**定理 11.12.2**（混沌判据）：若最大Lyapunov指数 $\lambda_1 > 0$，则系统混沌。

---

## 11.12.2 经典混沌系统

### 11.12.2.1 Logistic映射

**定义 11.12.4**（Logistic映射）：

$$x_{n+1} = r x_n (1 - x_n), \quad x \in [0, 1]$$

**定理 11.12.3**（倍周期分岔）：当 $r$ 增加时，系统经历一系列倍周期分岔：

$$r_1 = 3, \quad r_2 = 1 + \sqrt{6} \approx 3.449, \quad \ldots$$

**定理 11.12.4**（Feigenbaum常数）：分岔间距比收敛于：

$$\delta = \lim_{n \to \infty} \frac{r_n - r_{n-1}}{r_{n+1} - r_n} \approx 4.6692$$

### 11.12.2.2 Lorenz系统

**定义 11.12.5**（Lorenz方程）：

$$\begin{cases}
\dot{x} = \sigma(y - x) \\
\dot{y} = x(\rho - z) - y \\
\dot{z} = xy - \beta z
\end{cases}$$

标准参数：$\sigma = 10$，$\beta = 8/3$，$\rho = 28$

**定义 11.12.6**（奇怪吸引子）：Lorenz吸引子具有：
- 分形结构
- 对初值敏感
- 非周期轨道

---

## 11.12.3 混沌的度量

### 11.12.3.1 Kolmogorov-Sinai熵

**定义 11.12.7**（KS熵）：

$$h_{KS} = \sum_{\lambda_i > 0} \lambda_i$$

即所有正Lyapunov指数之和。

### 11.12.3.2 分形维度

**定义 11.12.8**（Lyapunov维度）：

$$D_L = k + \frac{\sum_{i=1}^k \lambda_i}{|\lambda_{k+1}|}$$

其中 $k$ 满足 $\sum_{i=1}^k \lambda_i \geq 0$ 但 $\sum_{i=1}^{k+1} \lambda_i < 0$。

```python
"""
复杂系统：混沌理论
Lyapunov指数、分岔图和奇怪吸引子
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint
from scipy.optimize import fsolve


class LogisticMap:
    """Logistic映射"""

    def __init__(self, r: float, x0: float = 0.5):
        self.r = r
        self.x = x0
        self.history = [x0]

    def iterate(self, n: int):
        """迭代n次"""
        for _ in range(n):
            self.x = self.r * self.x * (1 - self.x)
            self.history.append(self.x)
        return np.array(self.history)

    def lyapunov_exponent(self, n_transient: int = 1000, n_compute: int = 10000) -> float:
        """
        计算Lyapunov指数

        λ = lim (1/n) Σ ln|f'(x_i)|
        """
        # 热化
        x = 0.5
        for _ in range(n_transient):
            x = self.r * x * (1 - x)

        # 计算
        lyap_sum = 0
        for _ in range(n_compute):
            x = self.r * x * (1 - x)
            lyap_sum += np.log(abs(self.r * (1 - 2 * x)))

        return lyap_sum / n_compute


class LorenzSystem:
    """Lorenz系统"""

    def __init__(self, sigma: float = 10.0, rho: float = 28.0, beta: float = 8/3):
        self.sigma = sigma
        self.rho = rho
        self.beta = beta

    def dynamics(self, state, t):
        """Lorenz动力学"""
        x, y, z = state
        dx = self.sigma * (y - x)
        dy = x * (self.rho - z) - y
        dz = x * y - self.beta * z
        return [dx, dy, dz]

    def jacobian(self, state):
        """雅可比矩阵"""
        x, y, z = state
        return np.array([
            [-self.sigma, self.sigma, 0],
            [self.rho - z, -1, -x],
            [y, x, -self.beta]
        ])

    def simulate(self, x0: np.ndarray, t_span: tuple = (0, 50), n_points: int = 5000):
        """仿真"""
        t = np.linspace(t_span[0], t_span[1], n_points)
        trajectory = odeint(self.dynamics, x0, t)
        return t, trajectory

    def lyapunov_exponents(self, x0: np.ndarray, dt: float = 0.01, n_steps: int = 10000):
        """
        计算Lyapunov指数谱（使用QR分解法）
        """
        n = len(x0)
        Q = np.eye(n)
        lyap_sum = np.zeros(n)

        x = x0.copy()

        for i in range(n_steps):
            # 演化状态
            t_temp = [0, dt]
            sol = odeint(self.dynamics, x, t_temp)
            x = sol[-1]

            # 演化切向量
            J = self.jacobian(x)
            A = np.eye(n) + dt * J
            Q = A @ Q

            # QR分解
            Q, R = np.linalg.qr(Q)

            # 累加对角线元素的对数
            lyap_sum += np.log(np.abs(np.diag(R)))

        return lyap_sum / (n_steps * dt)


def bifurcation_diagram(r_range: np.ndarray, n_transient: int = 1000,
                        n_plot: int = 100) -> tuple:
    """
    生成分岔图

    Returns:
        (r值数组, x值数组)
    """
    r_values = []
    x_values = []

    for r in r_range:
        log_map = LogisticMap(r)
        # 热化
        x = 0.5
        for _ in range(n_transient):
            x = r * x * (1 - x)

        # 收集点
        for _ in range(n_plot):
            x = r * x * (1 - x)
            r_values.append(r)
            x_values.append(x)

    return np.array(r_values), np.array(x_values)


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 3, figsize=(15, 10))

    # 1. Logistic映射迭代
    ax1 = axes[0, 0]
    log_map = LogisticMap(r=3.8)
    trajectory = log_map.iterate(100)
    ax1.plot(trajectory, 'b-', linewidth=1)
    ax1.set_xlabel('n')
    ax1.set_ylabel('x_n')
    ax1.set_title('Logistic Map (r=3.8)')
    ax1.grid(True, alpha=0.3)

    # 2. 分岔图
    ax2 = axes[0, 1]
    r_range = np.linspace(2.5, 4.0, 2000)
    r_vals, x_vals = bifurcation_diagram(r_range)
    ax2.scatter(r_vals, x_vals, s=0.1, c='blue', alpha=0.5)
    ax2.set_xlabel('r')
    ax2.set_ylabel('x')
    ax2.set_title('Bifurcation Diagram')
    ax2.grid(True, alpha=0.3)

    # 3. Lyapunov指数随r变化
    ax3 = axes[0, 2]
    r_lyap = np.linspace(2.5, 4.0, 500)
    lyap_exponents = [LogisticMap(r).lyapunov_exponent(n_transient=500, n_compute=2000)
                      for r in r_lyap]
    ax3.plot(r_lyap, lyap_exponents, 'b-', linewidth=1)
    ax3.axhline(y=0, color='r', linestyle='--', alpha=0.5)
    ax3.set_xlabel('r')
    ax3.set_ylabel('λ')
    ax3.set_title('Lyapunov Exponent')
    ax3.grid(True, alpha=0.3)

    # 4. Lorenz吸引子 3D
    ax4 = axes[1, 0]
    lorenz = LorenzSystem()
    t, traj = lorenz.simulate([1.0, 1.0, 1.0], t_span=(0, 50))
    ax4.plot(traj[:, 0], traj[:, 2], 'b-', linewidth=0.5, alpha=0.8)
    ax4.set_xlabel('x')
    ax4.set_ylabel('z')
    ax4.set_title('Lorenz Attractor (x-z)')
    ax4.grid(True, alpha=0.3)

    # 5. 初值敏感依赖
    ax5 = axes[1, 1]
    t1, traj1 = lorenz.simulate([1.0, 1.0, 1.0], t_span=(0, 30))
    t2, traj2 = lorenz.simulate([1.0001, 1.0, 1.0], t_span=(0, 30))

    distance = np.linalg.norm(traj1 - traj2, axis=1)
    ax5.semilogy(t1, distance, 'b-', linewidth=2)
    ax5.set_xlabel('t')
    ax5.set_ylabel('||δx||')
    ax5.set_title('Sensitive Dependence (δx₀=10⁻⁴)')
    ax5.grid(True, alpha=0.3)

    # 6. 相空间重构（延迟嵌入）
    ax6 = axes[1, 2]
    # 使用Logistic映射的延迟嵌入
    log_map2 = LogisticMap(r=4.0)
    traj_1d = log_map2.iterate(1000)
    # 延迟嵌入
    delay = 1
    x_embed = traj_1d[:-delay]
    y_embed = traj_1d[delay:]
    ax6.scatter(x_embed, y_embed, s=1, c='blue', alpha=0.5)
    ax6.set_xlabel('x_n')
    ax6.set_ylabel('x_{n+1}')
    ax6.set_title('Delay Embedding (Logistic r=4)')
    ax6.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('chaos_theory.png', dpi=150)
    plt.show()

    # 计算Lorenz系统的Lyapunov指数
    print("Lorenz System Lyapunov Exponents:")
    lyaps = lorenz.lyapunov_exponents([1.0, 1.0, 1.0], n_steps=5000)
    for i, lam in enumerate(lyaps):
        print(f"  λ_{i+1} = {lam:.4f}")
    print(f"  KS Entropy ≈ {sum(l for l in lyaps if l > 0):.4f}")

    print("\nChaos theory visualization saved!")
