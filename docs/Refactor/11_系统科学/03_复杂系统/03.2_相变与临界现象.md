## 11.10.1 相变的分类

### 11.10.1.1 Ehrenfest分类

**定义 11.10.1**（相变阶数）：根据自由能的导数分类：

- **一级相变**：自由能的一阶导数（熵、体积）不连续
- **二级相变**：自由能的一阶导数连续，二阶导数（热容、压缩率）不连续或发散

**定义 11.10.2**（序参量）：描述有序程度的量 $\eta$，在无序相为0，有序相非零。

### 11.10.1.2 临界现象的特征

**定义 11.10.3**（临界点）：相变曲线的终点，此处相变成为连续。

**定义 11.10.4**（临界指数）：在临界点附近，物理量随约化温度 $t = (T - T_c)/T_c$ 的变化：

| 物理量 | 行为 | 指数 |
|--------|------|------|
| 序参量 | $\eta \sim |t|^\beta$ | $\beta$ |
| 热容 | $C \sim |t|^{-\alpha}$ | $\alpha$ |
| 磁化率 | $\chi \sim |t|^{-\gamma}$ | $\gamma$ |
| 相关长度 | $\xi \sim |t|^{-\nu}$ | $\nu$ |
| 相关函数 | $G(r) \sim r^{-(d-2+\eta)}$ | $\eta$ |

## 11.10.2 Ising模型

### 11.10.2.1 模型定义

**定义 11.10.5**（Ising哈密顿量）：

$$H = -J \sum_{\langle i,j \rangle} s_i s_j - h \sum_i s_i$$

其中：

- $s_i \in \{-1, +1\}$：自旋变量
- $J$：交换相互作用（$J > 0$ 铁磁，$J < 0$ 反铁磁）
- $h$：外场
- $\langle i,j \rangle$：最近邻对

### 11.10.2.2 平均场近似

**定义 11.10.6**（平均场哈密顿量）：

$$H_{MF} = -\sum_i s_i (zJm + h)$$

其中 $z$ 为配位数，$m = \langle s_i \rangle$ 为平均磁化强度。

**定理 11.10.1**（平均场方程）：自洽方程：

$$m = \tanh\left(\frac{zJm + h}{k_B T}\right)$$

**证明**：

在平均场近似下，每个自旋感受到有效场 $h_{eff} = zJm + h$。配分函数：

$$Z = \sum_{\{s_i\}} e^{-\beta H_{MF}} = [2\cosh(\beta h_{eff})]^N$$

磁化强度：

$$m = \frac{1}{N} \sum_i \langle s_i \rangle = \tanh(\beta h_{eff}) = \tanh\left(\frac{zJm + h}{k_B T}\right) \quad \square$$

**定理 11.10.2**（临界温度）：零场临界温度：

$$T_c = \frac{zJ}{k_B}$$

**证明**：

在 $h = 0$ 附近展开自洽方程：

$$m = \tanh\left(\frac{T_c}{T}m\right) \approx \frac{T_c}{T}m - \frac{1}{3}\left(\frac{T_c}{T}m\right)^3$$

非零解存在的条件是 $T_c/T > 1$，即 $T < T_c$。$\square$

## 11.10.3 自组织临界性

### 11.10.3.1 Bak-Tang-Wiesenfeld模型

**定义 11.10.7**（沙堆模型）：元胞上的沙粒数 $z_i$，当 $z_i \geq z_c$（临界斜率）时发生崩塌：

$$z_i \to z_i - 4, \quad z_{nn} \to z_{nn} + 1$$

**定义 11.10.8**（幂律分布）：崩塌大小 $s$ 的概率分布：

$$P(s) \sim s^{-\tau}$$

其中 $\tau \approx 1.1$（二维沙堆模型）。

### 11.10.3.2 临界性的特征

**定义 11.10.9**（1/f噪声）：功率谱密度：

$$S(f) \sim f^{-\alpha}, \quad \alpha \approx 1$$

**定义 11.10.10**（标度不变性）：系统在不同尺度上表现出相似的行为。

---

## 11.10.4 重整化群理论

### 11.10.4.1 重整化变换

**定义 11.10.11**（重整化群变换）：

$$R_b: \{K\} \to \{K'\}$$

其中 $b$ 为长度标度因子，$\{K\}$ 为耦合常数集合。

**定义 11.10.12**（固定点）：$R_b(K^*) = K^*$

### 11.10.4.2 标度律

**定理 11.10.3**（标度律）：临界指数满足标度关系：

$$\alpha + 2\beta + \gamma = 2$$
$$\gamma = \nu(2 - \eta)$$
$$\alpha = 2 - d\nu$$

**定义 11.10.13**（普适类）：具有相同临界指数和维数的系统属于同一普适类。

## 11.10.5 Python实现：相变与临界现象

```python
"""
复杂系统：相变与临界现象
Ising模型和自组织临界性仿真
"""

import numpy as np
from typing import Tuple, List, Callable
from dataclasses import dataclass
import matplotlib.pyplot as plt
from scipy.ndimage import convolve
from collections import Counter


class IsingModel:
    """
    二维Ising模型
    """

    def __init__(self, size: int, temperature: float, J: float = 1.0):
        """
        初始化Ising模型

        Args:
            size: 晶格大小
            temperature: 温度（以k_B为单位）
            J: 交换耦合常数
        """
        self.size = size
        self.T = temperature
        self.J = J
        self.beta = 1.0 / temperature

        # 随机初始构型
        self.spins = np.random.choice([-1, 1], size=(size, size))

        # 邻居卷积核
        self.kernel = np.array([[0, 1, 0],
                               [1, 0, 1],
                               [0, 1, 0]])

    def energy(self) -> float:
        """计算总能量"""
        # 邻居相互作用
        neighbors = convolve(self.spins, self.kernel, mode='wrap')
        interaction = -self.J * np.sum(self.spins * neighbors) / 2  # 每对只算一次
        return interaction

    def magnetization(self) -> float:
        """计算磁化强度"""
        return np.mean(self.spins)

    def magnetization_abs(self) -> float:
        """计算磁化强度绝对值"""
        return abs(np.mean(self.spins))

    def susceptibility(self) -> float:
        """计算磁化率"""
        m = self.magnetization()
        m_squared = np.mean(self.spins**2)
        return self.beta * (m_squared - m**2) * self.size**2

    def specific_heat(self) -> float:
        """计算比热"""
        E = self.energy()
        E_squared = self.energy_squared()
        return self.beta**2 * (E_squared - E**2) / self.size**2

    def energy_squared(self) -> float:
        """计算能量平方的期望值（简化）"""
        return self.energy()**2  # 单次测量近似

    def monte_carlo_step(self, n_sweeps: int = 1):
        """
        Metropolis Monte Carlo更新

        Args:
            n_sweeps: 扫过整个晶格的次数
        """
        for _ in range(n_sweeps):
            for _ in range(self.size**2):
                # 随机选择一个格点
                i, j = np.random.randint(0, self.size, 2)

                # 计算能量变化
                neighbor_sum = (
                    self.spins[(i+1) % self.size, j] +
                    self.spins[(i-1) % self.size, j] +
                    self.spins[i, (j+1) % self.size] +
                    self.spins[i, (j-1) % self.size]
                )

                delta_E = 2 * self.J * self.spins[i, j] * neighbor_sum

                # Metropolis准则
                if delta_E < 0 or np.random.random() < np.exp(-self.beta * delta_E):
                    self.spins[i, j] *= -1

    def simulate(self, n_steps: int, n_thermalization: int = 1000):
        """
        运行模拟并收集统计

        Returns:
            磁化强度、能量历史
        """
        # 热化
        self.monte_carlo_step(n_thermalization)

        # 收集数据
        mag_history = []
        energy_history = []

        for _ in range(n_steps):
            self.monte_carlo_step(10)  # 每隔10步采样
            mag_history.append(self.magnetization_abs())
            energy_history.append(self.energy())

        return np.array(mag_history), np.array(energy_history)

    def correlation_function(self, max_distance: int = 10) -> np.ndarray:
        """计算自旋-自旋关联函数"""
        correlations = []

        center = self.size // 2
        s0 = self.spins[center, center]

        for r in range(1, max_distance + 1):
            # 四个方向的平均
            corr = 0
            if center + r < self.size:
                corr += s0 * self.spins[center + r, center]
            if center - r >= 0:
                corr += s0 * self.spins[center - r, center]
            if center + r < self.size:
                corr += s0 * self.spins[center, center + r]
            if center - r >= 0:
                corr += s0 * self.spins[center, center - r]

            correlations.append(corr / 4)

        return np.array(correlations)


class SandpileModel:
    """
    Bak-Tang-Wiesenfeld沙堆模型
    """

    def __init__(self, size: int, critical_slope: int = 4):
        """
        初始化沙堆模型

        Args:
            size: 晶格大小
            critical_slope: 临界斜率
        """
        self.size = size
        self.zc = critical_slope
        self.heights = np.zeros((size, size), dtype=int)
        self.avalanche_sizes = []
        self.avalanche_durations = []

    def add_grain(self) -> Tuple[int, int]:
        """
        添加一粒沙子

        Returns:
            (崩塌大小, 崩塌持续时间)
        """
        # 在中心添加
        center = self.size // 2
        self.heights[center, center] += 1

        # 处理崩塌
        size, duration = self._topple()

        if size > 0:
            self.avalanche_sizes.append(size)
            self.avalanche_durations.append(duration)

        return size, duration

    def _topple(self) -> Tuple[int, int]:
        """处理崩塌，返回崩塌大小和持续时间"""
        size = 0
        duration = 0

        while True:
            # 找到所有超过临界斜率的格点
            unstable = self.heights >= self.zc

            if not np.any(unstable):
                break

            duration += 1

            # 处理崩塌
            for i in range(self.size):
                for j in range(self.size):
                    if self.heights[i, j] >= self.zc:
                        self.heights[i, j] -= self.zc
                        size += 1

                        # 向邻居传递
                        if i + 1 < self.size:
                            self.heights[i + 1, j] += 1
                        if i - 1 >= 0:
                            self.heights[i - 1, j] += 1
                        if j + 1 < self.size:
                            self.heights[i, j + 1] += 1
                        if j - 1 >= 0:
                            self.heights[i, j - 1] += 1

        return size, duration

    def run(self, n_grains: int):
        """运行n次添加沙粒"""
        for _ in range(n_grains):
            self.add_grain()

    def get_power_law_exponent(self) -> float:
        """估计幂律指数"""
        if len(self.avalanche_sizes) < 100:
            return 0

        # 使用最大似然估计
        sizes = np.array(self.avalanche_sizes)
        sizes = sizes[sizes > 0]

        if len(sizes) == 0:
            return 0

        # 简化估计
        log_sizes = np.log(sizes)
        n = len(log_sizes)

        # 幂律指数估计: P(s) ~ s^(-τ)
        # 对于累积分布: τ ≈ 1 + n / Σln(s_i/s_min)
        s_min = np.min(sizes)
        tau = 1 + n / np.sum(np.log(sizes / s_min))

        return tau


def simulate_phase_transition():
    """
    Ising模型相变仿真
    """
    print("=" * 60)
    print("Ising Model Phase Transition")
    print("=" * 60)

    size = 20
    temperatures = np.linspace(1.5, 4.0, 20)

    magnetizations = []
    susceptibilities = []
    energies = []

    for T in temperatures:
        print(f"  Simulating T = {T:.2f}...", end='\r')

        ising = IsingModel(size, T)
        mag_hist, energy_hist = ising.simulate(n_steps=100, n_thermalization=500)

        magnetizations.append(np.mean(mag_hist))
        energies.append(np.mean(energy_hist))

        # 磁化率估计
        susc = np.var(mag_hist) * size**2 / T
        susceptibilities.append(susc)

    print("\nSimulation completed!")

    # 理论临界温度 (二维Ising模型)
    Tc_theory = 2 / np.log(1 + np.sqrt(2))
    print(f"\nTheoretical Tc = {Tc_theory:.4f}")

    # 估计临界温度
    max_susc_idx = np.argmax(susceptibilities)
    Tc_estimated = temperatures[max_susc_idx]
    print(f"Estimated Tc (from susceptibility peak) = {Tc_estimated:.4f}")

    return temperatures, magnetizations, susceptibilities, energies, Tc_theory


def simulate_self_organized_criticality():
    """
    自组织临界性仿真
    """
    print("\n" + "=" * 60)
    print("Self-Organized Criticality (Sandpile Model)")
    print("=" * 60)

    size = 50
    n_grains = 50000

    sandpile = SandpileModel(size)

    print(f"\nRunning sandpile model...")
    print(f"  Grid size: {size}x{size}")
    print(f"  Grains to add: {n_grains}")

    sandpile.run(n_grains)

    print(f"  Total avalanches: {len(sandpile.avalanche_sizes)}")
    print(f"  Max avalanche size: {max(sandpile.avalanche_sizes) if sandpile.avalanche_sizes else 0}")

    # 估计幂律指数
    tau = sandpile.get_power_law_exponent()
    print(f"  Estimated power-law exponent τ ≈ {tau:.3f}")
    print(f"  (Theoretical 2D value: τ ≈ 1.1)")

    return sandpile


def visualize_phase_transition():
    """可视化相变"""
    fig = plt.figure(figsize=(16, 10))

    # 1. Ising模型相变
    ax1 = plt.subplot(2, 3, 1)
    temps, mags, suscs, energies, Tc = simulate_phase_transition()

    ax1.plot(temps, mags, 'b-o', linewidth=2, markersize=6)
    ax1.axvline(Tc, color='r', linestyle='--', label=f'Tc = {Tc:.3f}')
    ax1.set_xlabel('Temperature T')
    ax1.set_ylabel('Magnetization |m|')
    ax1.set_title('Ising Model: Magnetization vs Temperature')
    ax1.legend()
    ax1.grid(True, alpha=0.3)

    # 2. 磁化率峰值
    ax2 = plt.subplot(2, 3, 2)
    ax2.plot(temps, suscs, 'r-s', linewidth=2, markersize=6)
    ax2.axvline(Tc, color='k', linestyle='--', label=f'Tc = {Tc:.3f}')
    ax2.set_xlabel('Temperature T')
    ax2.set_ylabel('Susceptibility χ')
    ax2.set_title('Ising Model: Susceptibility')
    ax2.legend()
    ax2.grid(True, alpha=0.3)

    # 3. 不同温度下的构型
    ax3 = plt.subplot(2, 3, 3)
    ising_low = IsingModel(30, 1.5)
    ising_low.monte_carlo_step(1000)
    ax3.imshow(ising_low.spins, cmap='RdBu_r', interpolation='nearest')
    ax3.set_title(f'Low T={1.5}: Ordered Phase')
    ax3.axis('off')

    # 4. 沙堆模型构型
    ax4 = plt.subplot(2, 3, 4)
    sandpile = simulate_self_organized_criticality()
    ax4.imshow(sandpile.heights, cmap='YlOrRd', interpolation='nearest')
    ax4.set_title('Sandpile: Critical State')
    ax4.axis('off')

    # 5. 崩塌大小分布
    ax5 = plt.subplot(2, 3, 5)

    if len(sandpile.avalanche_sizes) > 100:
        sizes = np.array(sandpile.avalanche_sizes)
        sizes = sizes[sizes > 0]

        # 对数分箱
        bins = np.logspace(np.log10(min(sizes)), np.log10(max(sizes)), 30)
        counts, edges = np.histogram(sizes, bins=bins)

        # 概率密度
        bin_centers = (edges[:-1] * edges[1:])**0.5
        probabilities = counts / len(sizes) / (edges[1:] - edges[:-1])

        ax5.loglog(bin_centers, probabilities, 'b.', markersize=10)

        # 拟合幂律
        mask = (bin_centers > 10) & (probabilities > 0)
        if np.sum(mask) > 5:
            log_x = np.log(bin_centers[mask])
            log_y = np.log(probabilities[mask])
            coeffs = np.polyfit(log_x, log_y, 1)
            ax5.loglog(bin_centers, np.exp(coeffs[1]) * bin_centers**coeffs[0],
                      'r--', linewidth=2, label=f'τ ≈ {-coeffs[0]:.2f}')

        ax5.set_xlabel('Avalanche Size s')
        ax5.set_ylabel('P(s)')
        ax5.set_title('Power-Law Distribution')
        ax5.legend()
        ax5.grid(True, alpha=0.3, which='both')

    # 6. 临界指数随维度变化（示意）
    ax6 = plt.subplot(2, 3, 6)

    dimensions = [1, 2, 3, 4, '≥4']
    beta_values = [0.125, 0.125, 0.326, 0.365, 0.5]  # 磁性指数
    nu_values = [1, 1, 0.63, 0.54, 0.5]  # 相关长度指数

    x_pos = np.arange(len(dimensions))
    width = 0.35

    ax6.bar(x_pos - width/2, beta_values, width, label='β', color='#3498db')
    ax6.bar(x_pos + width/2, nu_values, width, label='ν', color='#e74c3c')

    ax6.set_xlabel('Dimension')
    ax6.set_ylabel('Critical Exponent')
    ax6.set_title('Critical Exponents vs Dimension')
    ax6.set_xticks(x_pos)
    ax6.set_xticklabels(dimensions)
    ax6.legend()
    ax6.grid(True, alpha=0.3, axis='y')

    plt.tight_layout()
    plt.savefig('phase_transitions.png', dpi=150, bbox_inches='tight')
    plt.show()


if __name__ == "__main__":
    visualize_phase_transition()
    print("\nVisualization saved to 'phase_transitions.png'")
