## 11.11.1 分形维度

### 11.11.1.1 Hausdorff维度

**定义 11.11.1**（Hausdorff测度）：对于集合 $S \subset \mathbb{R}^n$，$d$-维Hausdorff测度：

$$H^d(S) = \lim_{\delta \to 0} \inf \left\{\sum_i (\text{diam } U_i)^d : S \subset \bigcup_i U_i, \text{diam } U_i < \delta\right\}$$

**定义 11.11.2**（Hausdorff维度）：

$$\dim_H(S) = \inf\{d : H^d(S) = 0\} = \sup\{d : H^d(S) = \infty\}$$

### 11.11.1.2 盒计数维度

**定义 11.11.3**（盒计数维度）：用边长为 $\epsilon$ 的盒子覆盖集合所需的最小盒子数 $N(\epsilon)$：

$$\dim_B(S) = \lim_{\epsilon \to 0} \frac{\log N(\epsilon)}{\log(1/\epsilon)}$$

**定理 11.11.1**：对于紧集，$\dim_H(S) \leq \dim_B(S)$。

---

## 11.11.2 经典分形

### 11.11.2.1 Cantor集

**定义 11.11.4**（Cantor集）：迭代删除中间三分之一的开区间：

$$C = \bigcap_{n=0}^{\infty} C_n$$

其中 $C_0 = [0, 1]$，$C_{n+1} = \frac{1}{3}C_n \cup \left(\frac{2}{3} + \frac{1}{3}C_n\right)$。

**定理 11.11.2**（Cantor集的维度）：

$$\dim_H(C) = \dim_B(C) = \frac{\log 2}{\log 3} \approx 0.6309$$

**证明**：在第 $n$ 步，有 $2^n$ 个区间，每个长度为 $(1/3)^n$。

$$N(\epsilon) = 2^n \text{ 当 } \epsilon = (1/3)^n$$

$$\dim_B = \lim_{n \to \infty} \frac{\log 2^n}{\log 3^n} = \frac{\log 2}{\log 3} \quad \square$$

### 11.11.2.2 Koch雪花

**定义 11.11.5**（Koch曲线）：将线段分为三等分，中间用等边三角形替代。

**定理 11.11.3**（Koch曲线维度）：

$$\dim_H(K) = \frac{\log 4}{\log 3} \approx 1.2619$$

---

## 11.11.3 幂律分布

### 11.11.3.1 幂律的形式

**定义 11.11.6**（幂律分布）：概率密度函数：

$$p(x) = C x^{-\alpha}, \quad x \geq x_{min}$$

其中 $\alpha > 1$ 为幂指数，$C = (\alpha - 1)x_{min}^{\alpha - 1}$ 为归一化常数。

**定义 11.11.7**（累积分布）：

$$P(X > x) = \left(\frac{x}{x_{min}}\right)^{-\alpha + 1}$$

### 11.11.3.2 产生幂律的机制

**定理 11.11.4**（偏好依附）：Yule过程产生幂律：

$$p(k) \sim k^{-(1 + \frac{1}{\beta})}$$

**定理 11.11.5**（自组织临界性）：沙堆模型产生幂律分布的崩塌大小。

"""
复杂系统：分形与幂律
分形维度计算和幂律分布分析
"""

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.patches import Rectangle
from scipy.optimize import curve_fit
from scipy import stats


class FractalDimension:
    """分形维度计算"""

    @staticmethod
    def box_counting(points: np.ndarray, epsilon_range: np.ndarray) -> float:
        """
        盒计数法计算分形维度

        Args:
            points: N x d 的点集
            epsilon_range: 盒子大小范围
        """
        counts = []

        for eps in epsilon_range:
            # 计算需要的盒子数
            min_coords = np.min(points, axis=0)
            max_coords = np.max(points, axis=0)

            # 离散化到网格
            grid_coords = np.floor((points - min_coords) / eps).astype(int)

            # 统计非空盒子
            unique_boxes = len(np.unique(grid_coords, axis=0))
            counts.append(unique_boxes)

        # 线性拟合 log(N) vs log(1/ε)
        log_counts = np.log(counts)
        log_inv_eps = np.log(1 / epsilon_range)

        coeffs = np.polyfit(log_inv_eps, log_counts, 1)
        return coeffs[0]  # 斜率即维度


class CantorSet:
    """Cantor集"""

    def __init__(self, n_iterations: int = 6):
        self.n = n_iterations
        self.points = self._generate()

    def _generate(self) -> np.ndarray:
        """生成Cantor集"""
        intervals = [(0.0, 1.0)]

        for _ in range(self.n):
            new_intervals = []
            for a, b in intervals:
                length = (b - a) / 3
                new_intervals.append((a, a + length))
                new_intervals.append((a + 2*length, b))
            intervals = new_intervals

        # 返回区间端点
        points = []
        for a, b in intervals:
            points.extend([a, b])
        return np.array(sorted(set(points)))

    def theoretical_dimension(self) -> float:
        """理论维度 log(2)/log(3)"""
        return np.log(2) / np.log(3)


class KochCurve:
    """Koch曲线"""

    def __init__(self, n_iterations: int = 4):
        self.n = n_iterations
        self.points = self._generate()

    def _generate(self) -> np.ndarray:
        """生成Koch曲线"""
        # 初始线段
        points = np.array([[0, 0], [1, 0]])

        for _ in range(self.n):
            new_points = [points[0]]

            for i in range(len(points) - 1):
                p1, p2 = points[i], points[i + 1]

                # 分割点
                v = p2 - p1
                a = p1 + v / 3
                b = p1 + 2 * v / 3

                # 顶点（旋转60度）
                # 旋转矩阵
                angle = np.pi / 3
                R = np.array([[np.cos(angle), -np.sin(angle)],
                             [np.sin(angle), np.cos(angle)]])

                c = a + (R @ (v / 3).T).T

                new_points.extend([a, c, b, p2])

            points = np.array(new_points)

        return points


class PowerLawAnalyzer:
    """幂律分布分析"""

    @staticmethod
    def estimate_exponent(data: np.ndarray, x_min: float = None) -> tuple:
        """
        估计幂律指数

        使用最大似然估计
        """
        if x_min is None:
            x_min = np.min(data)

        # 筛选大于x_min的数据
        filtered = data[data >= x_min]
        n = len(filtered)

        if n == 0:
            return 0, 0

        # 最大似然估计
        # α = 1 + n / Σ ln(x_i / x_min)
        alpha = 1 + n / np.sum(np.log(filtered / x_min))

        # 标准误差
        sigma = (alpha - 1) / np.sqrt(n)

        return alpha, sigma

    @staticmethod
    def log_likelihood_ratio(data: np.ndarray, x_min: float) -> float:
        """
        比较幂律和指数分布的对数似然比
        正值表示幂律更好
        """
        filtered = data[data >= x_min]
        n = len(filtered)

        # 幂律参数
        alpha, _ = PowerLawAnalyzer.estimate_exponent(data, x_min)

        # 幂律对数似然
        L_pl = n * np.log((alpha - 1) / x_min) - alpha * np.sum(np.log(filtered / x_min))

        # 指数分布参数
        lambda_exp = 1 / np.mean(filtered - x_min)

        # 指数对数似然
        L_exp = n * np.log(lambda_exp) - lambda_exp * np.sum(filtered - x_min)

        return L_pl - L_exp


def generate_power_law(n: int, alpha: float, x_min: float = 1.0) -> np.ndarray:
    """
    生成幂律分布数据

    使用逆变换采样
    """
    u = np.random.uniform(0, 1, n)
    x = x_min * (1 - u) ** (-1 / (alpha - 1))
    return x


# 可视化
if __name__ == "__main__":
    fig, axes = plt.subplots(2, 3, figsize=(15, 10))

    # 1. Cantor集
    ax1 = axes[0, 0]
    cantor = CantorSet(n_iterations=6)
    for i, x in enumerate(cantor.points):
        ax1.plot([x, x], [0, 1], 'b-', linewidth=2)
    ax1.set_xlim(-0.1, 1.1)
    ax1.set_ylim(-0.1, 1.1)
    ax1.set_title(f"Cantor Set (D = {cantor.theoretical_dimension():.4f})")
    ax1.set_aspect('equal')
    ax1.axis('off')

    # 2. Koch曲线
    ax2 = axes[0, 1]
    koch = KochCurve(n_iterations=4)
    ax2.plot(koch.points[:, 0], koch.points[:, 1], 'b-', linewidth=1)
    ax2.set_title(f"Koch Curve (D = {np.log(4)/np.log(3):.4f})")
    ax2.set_aspect('equal')
    ax2.axis('off')

    # 3. 幂律分布
    ax3 = axes[0, 2]
    data = generate_power_law(10000, alpha=2.5, x_min=1.0)
    bins = np.logspace(np.log10(1), np.log10(np.max(data)), 50)
    ax3.hist(data, bins=bins, density=True, alpha=0.7, color='blue')
    ax3.set_xscale('log')
    ax3.set_yscale('log')
    ax3.set_xlabel('x')
    ax3.set_ylabel('P(x)')
    ax3.set_title('Power Law Distribution')
    ax3.grid(True, alpha=0.3)

    # 4. 盒计数维度估计
    ax4 = axes[1, 0]
    # 用Koch曲线点估计维度
    eps_range = np.logspace(-4, -1, 20)
    dim_estimator = FractalDimension()
    dim_est = dim_estimator.box_counting(koch.points, eps_range)

    counts = []
    for eps in eps_range:
        min_coords = np.min(koch.points, axis=0)
        grid_coords = np.floor((koch.points - min_coords) / eps).astype(int)
        unique_boxes = len(np.unique(grid_coords, axis=0))
        counts.append(unique_boxes)

    ax4.loglog(1/eps_range, counts, 'bo', label=f'Estimated D ≈ {dim_est:.3f}')
    ax4.loglog(1/eps_range, (1/eps_range)**(np.log(4)/np.log(3)), 'r--',
               label=f'Theoretical D = {np.log(4)/np.log(3):.3f}')
    ax4.set_xlabel('1/ε')
    ax4.set_ylabel('N(ε)')
    ax4.set_title('Box Counting Dimension')
    ax4.legend()
    ax4.grid(True, alpha=0.3)

    # 5. 幂律指数估计
    ax5 = axes[1, 1]
    alpha_est, sigma = PowerLawAnalyzer.estimate_exponent(data, x_min=1.0)

    # 累积分布
    sorted_data = np.sort(data)
    cdf = 1 - np.arange(1, len(sorted_data) + 1) / len(sorted_data)

    ax5.loglog(sorted_data, cdf, 'b.', alpha=0.5, label='Data')
    x_fit = np.logspace(0, np.log10(np.max(data)), 100)
    y_fit = (x_fit / 1.0) ** (-alpha_est + 1)
    ax5.loglog(x_fit, y_fit, 'r-', linewidth=2,
               label=f'Fit: α = {alpha_est:.2f} ± {sigma:.2f}')
    ax5.set_xlabel('x')
    ax5.set_ylabel('P(X > x)')
    ax5.set_title('Power Law Fitting')
    ax5.legend()
    ax5.grid(True, alpha=0.3)

    # 6. 不同α的幂律比较
    ax6 = axes[1, 2]
    alphas = [2.0, 2.5, 3.0, 4.0]
    colors = ['red', 'blue', 'green', 'orange']

    for alpha, color in zip(alphas, colors):
        data_alpha = generate_power_law(5000, alpha, x_min=1.0)
        sorted_d = np.sort(data_alpha)
        cdf_d = 1 - np.arange(1, len(sorted_d) + 1) / len(sorted_d)
        ax6.loglog(sorted_d, cdf_d, '.', alpha=0.3, color=color, label=f'α={alpha}')

    ax6.set_xlabel('x')
    ax6.set_ylabel('P(X > x)')
    ax6.set_title('Power Laws with Different α')
    ax6.legend()
    ax6.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('fractals_powerlaw.png', dpi=150)
    plt.show()

    print("Fractals and Power Law visualization saved!")
    print(f"Cantor set dimension: {cantor.theoretical_dimension():.4f}")
    print(f"Koch curve estimated dimension: {dim_est:.4f}")
    print(f"Power law estimated exponent: {alpha_est:.4f} ± {sigma:.4f}")
