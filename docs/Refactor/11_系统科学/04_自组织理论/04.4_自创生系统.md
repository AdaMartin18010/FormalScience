## 11.16.1 自创生的概念

### 11.16.1.1 定义

**定义 11.16.1**（自创生）：一个系统如果其组成元素的产生过程（1）通过相互作用和转化，（2）在空间中构成有边界的网络，（3）产生它们自己，（4）且这个网络构成系统作为一个统一体的边界，则称该系统为自创生系统。

**定义 11.16.2**（操作封闭性）：系统内部的因果关系循环，不依赖外部指令。

**定义 11.16.3**（结构确定性）：系统的状态变化由其自身结构决定，而非由环境"指令"。

### 11.16.1.2 自创生 vs 异创生

| 特征 | 异创生 (Allopoiesis) | 自创生 (Autopoiesis) |
|------|---------------------|---------------------|
| 产物 | 生产不同于自身的东西 | 生产自身 |
| 例子 | 工厂、机器 | 细胞、生命 |
| 边界 | 由外部定义 | 由系统自身产生 |
| 组织 | 可替换 | 必须维持 |

## 11.16.2 形式化描述

### 11.16.2.1 自创生网络

**定义 11.16.4**（自创生网络）：过程网络 $\mathcal{N} = (\mathcal{P}, \mathcal{R})$，其中：

- $\mathcal{P} = \{p_1, \ldots, p_n\}$：过程集合
- $\mathcal{R} \subseteq \mathcal{P} \times \mathcal{P}$：关系（过程 $p_i$ 产生执行 $p_j$ 所需的组件）

**定义 11.16.5**（自创生条件）：网络 $\mathcal{N}$ 是自创生的，若：

$$\forall p_i \in \mathcal{P}: \exists \{p_j\} \subseteq \mathcal{P}: p_j \to p_i \land p_i \to \{p_k\}$$

即每个过程都被系统内的其他过程产生，并产生其他过程所需的组件。

### 11.16.2.2 认知作为行动

**定义 11.16.6**（认知域）：系统能够维持其自创生的可能状态空间。

**定理 11.16.1**（结构耦合）：自创生系统与环境之间通过反复互动达到协调的关系称为结构耦合。

"""
自组织理论：自创生系统
Autopoiesis模型和结构耦合
"""

import numpy as np
import matplotlib.pyplot as plt
import networkx as nx
from matplotlib.patches import Circle


class AutopoieticNetwork:
    """
    自创生网络模型

    简化的细胞代谢网络
    """

    def __init__(self, n_processes: int = 6, connectivity: float = 0.4):
        """
        初始化自创生网络

        Args:
            n_processes: 过程/反应数量
            connectivity: 连接密度
        """
        self.n = n_processes
        self.graph = nx.DiGraph()

        # 添加过程节点
        for i in range(n_processes):
            self.graph.add_node(i, type='process', activity=1.0)

        # 添加组件节点（代谢物）
        for i in range(n_processes, 2 * n_processes):
            self.graph.add_node(i, type='component', concentration=1.0)

        # 建立自创生连接
        self._build_autopoietic_structure()

    def _build_autopoietic_structure(self):
        """构建自创生结构：每个过程产生下一个过程所需的组件"""
        for i in range(self.n):
            # 过程i消耗组件i，产生组件(i+1)%n
            process_node = i
            substrate = self.n + i
            product = self.n + (i + 1) % self.n

            # 底物 -> 过程
            self.graph.add_edge(substrate, process_node, weight=1.0)
            # 过程 -> 产物
            self.graph.add_edge(process_node, product, weight=1.0)

            # 产物激活下一个过程（自创生循环）
            next_process = (i + 1) % self.n
            self.graph.add_edge(product, next_process, weight=0.5)

    def is_autopoietic(self) -> bool:
        """
        检查网络是否满足自创生条件

        每个过程必须：
        1. 有输入（被产生）
        2. 有输出（产生其他）
        """
        for i in range(self.n):
            in_degree = self.graph.in_degree(i)
            out_degree = self.graph.out_degree(i)

            if in_degree == 0 or out_degree == 0:
                return False

        return True

    def visualize(self, ax=None):
        """可视化自创生网络"""
        if ax is None:
            fig, ax = plt.subplots(figsize=(10, 8))

        pos = nx.spring_layout(self.graph, k=2, iterations=50)

        # 分开绘制不同类型的节点
        process_nodes = [n for n, attr in self.graph.nodes(data=True) if attr.get('type') == 'process']
        component_nodes = [n for n, attr in self.graph.nodes(data=True) if attr.get('type') == 'component']

        nx.draw_networkx_nodes(self.graph, pos, nodelist=process_nodes,
                              node_color='lightblue', node_size=2000, ax=ax, label='Processes')
        nx.draw_networkx_nodes(self.graph, pos, nodelist=component_nodes,
                              node_color='lightgreen', node_size=1500, ax=ax, label='Components')

        nx.draw_networkx_edges(self.graph, pos, arrows=True,
                              arrowsize=20, width=2, ax=ax,
                              connectionstyle='arc3,rad=0.1')

        # 标签
        labels = {i: f'P{i}' for i in range(self.n)}
        labels.update({self.n + i: f'C{i}' for i in range(self.n)})
        nx.draw_networkx_labels(self.graph, pos, labels, ax=ax)

        ax.set_title('Autopoietic Network Structure')
        ax.axis('off')

        return ax


class StructuralCoupling:
    """
    结构耦合模型

    系统与环境之间的协调演化
    """

    def __init__(self, system_state: np.ndarray, environment_state: np.ndarray):
        self.system = system_state.copy()
        self.environment = environment_state.copy()

        # 历史记录
        self.system_history = [self.system.copy()]
        self.env_history = [self.environment.copy()]

    def interaction(self, coupling_strength: float = 0.1):
        """
        系统与环境之间的结构耦合

        相互影响但不决定对方的结构变化
        """
        # 系统扰动环境
        env_perturbation = coupling_strength * self.system

        # 环境扰动系统
        system_perturbation = coupling_strength * self.environment

        # 内部动态（操作封闭）
        self.system += system_perturbation + 0.05 * self._internal_dynamics()
        self.environment += env_perturbation + 0.05 * self._env_dynamics()

        # 记录
        self.system_history.append(self.system.copy())
        self.env_history.append(self.environment.copy())

    def _internal_dynamics(self) -> np.ndarray:
        """系统内部动态"""
        # 简化的自创生维持
        return -0.1 * self.system + 0.1 * np.sin(self.system)

    def _env_dynamics(self) -> np.ndarray:
        """环境动态"""
        return -0.05 * self.environment + 0.1 * np.random.randn(len(self.environment))

    def run(self, n_steps: int = 1000):
        """运行耦合仿真"""
        for _ in range(n_steps):
            self.interaction()

    def coordination_measure(self) -> float:
        """测量系统与环境的协调性"""
        sys_hist = np.array(self.system_history)
        env_hist = np.array(self.env_history)

        # 计算相关性
        correlations = []
        for i in range(len(self.system)):
            corr = np.corrcoef(sys_hist[:, i], env_hist[:, i])[0, 1]
            if not np.isnan(corr):
                correlations.append(abs(corr))

        return np.mean(correlations) if correlations else 0
