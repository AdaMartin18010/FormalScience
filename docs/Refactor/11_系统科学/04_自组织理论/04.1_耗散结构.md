## 11.13.1 远离平衡态的热力学

### 11.13.1.1 熵平衡方程

**定义 11.13.1**（熵平衡）：开放系统的熵变化：

$$dS = d_e S + d_i S$$

其中：

- $d_e S$：熵流（与环境交换）
- $d_i S$：熵产生（内部不可逆过程）

**定理 11.13.1**（熵产生非负）：

$$d_i S \geq 0$$

**定义 11.13.2**（定态）：系统性质不随时间变化的状态，但内部存在持续的流。

### 11.13.1.2 最小熵产生原理

**定理 11.13.2**（Prigogine最小熵产生原理）：在线性非平衡区，定态使熵产生取极小值。

**证明**：设熵产生：

$$P = \int \sigma dV = \int \sum_i J_i X_i dV$$

在定态，流 $J_i$ 稳定，力 $X_i$ 达到使 $\delta P = 0$ 且 $\delta^2 P > 0$。$\square$

## 11.13.2 Brusselator模型

### 11.13.2.1 三分子模型

**定义 11.13.3**（Brusselator反应）：

$$\begin{cases}
A \xrightarrow{k_1} X \\
B + X \xrightarrow{k_2} Y + D \\
2X + Y \xrightarrow{k_3} 3X \\
X \xrightarrow{k_4} E
\end{cases}$$

**定义 11.13.4**（动力学方程）：

$$\begin{cases}
\frac{dX}{dt} = A - (B + 1)X + X^2 Y \\
\frac{dY}{dt} = BX - X^2 Y
\end{cases}$$

### 11.13.2.2 稳定性分析

**定理 11.13.3**（Brusselator分岔）：当 $B > B_c = 1 + A^2$ 时，定态失稳，出现极限环。

**证明**：在定态 $(X_0, Y_0) = (A, B/A)$ 线性化，Jacobian：

$$J = \begin{pmatrix} B - 1 & A^2 \\ -B & -A^2 \end{pmatrix}$$

特征值实部为零的条件给出分岔点 $B_c = 1 + A^2$。$\square$

## 11.13.3 Python实现

```python
"""
自组织理论：耗散结构
Brusselator模型和熵产生分析
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint


class Brusselator:
    """Brusselator模型"""

    def __init__(self, A: float = 1.0, B: float = 2.5):
        self.A = A
        self.B = B
        self.Bc = 1 + A**2  # 临界值

    def dynamics(self, state, t):
        """动力学方程"""
        X, Y = state
        dX = self.A - (self.B + 1) * X + X**2 * Y
        dY = self.B * X - X**2 * Y
        return [dX, dY]

    def steady_state(self):
        """定态"""
        return np.array([self.A, self.B / self.A])

    def is_stable(self):
        """检查定态稳定性"""
        return self.B < self.Bc

    def simulate(self, X0: np.ndarray, t_span: tuple = (0, 50), n_points: int = 5000):
        """仿真"""
        t = np.linspace(t_span[0], t_span[1], n_points)
        trajectory = odeint(self.dynamics, X0, t)
        return t, trajectory

    def entropy_production(self, X: float, Y: float) -> float:
        """计算局域熵产生（简化）"""
        # 熵产生率 ~ 反应流 × 亲和力
        # 简化计算
        sigma = (self.A * np.log(self.A / X) +
                self.B * X / Y * np.log(self.B * X / Y))
        return max(0, sigma)


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))

    # 1. 稳定情况
    ax1 = axes[0, 0]
    bruss_stable = Brusselator(A=1.0, B=1.5)  # B < Bc=2
    t, traj = bruss_stable.simulate([0.5, 0.5])
    ax1.plot(t, traj[:, 0], 'b-', label='X')
    ax1.plot(t, traj[:, 1], 'r-', label='Y')
    ax1.axhline(bruss_stable.steady_state()[0], color='b', linestyle='--', alpha=0.5)
    ax1.axhline(bruss_stable.steady_state()[1], color='r', linestyle='--', alpha=0.5)
    ax1.set_xlabel('t')
    ax1.set_ylabel('Concentration')
    ax1.set_title(f'Stable (B={bruss_stable.B} < Bc={bruss_stable.Bc:.2f})')
    ax1.legend()
    ax1.grid(True, alpha=0.3)

    # 2. 不稳定情况 - 极限环
    ax2 = axes[0, 1]
    bruss_unstable = Brusselator(A=1.0, B=3.0)  # B > Bc=2
    t, traj = bruss_unstable.simulate([0.5, 0.5])
    ax2.plot(t, traj[:, 0], 'b-', label='X')
    ax2.plot(t, traj[:, 1], 'r-', label='Y')
    ax2.set_xlabel('t')
    ax2.set_ylabel('Concentration')
    ax2.set_title(f'Limit Cycle (B={bruss_unstable.B} > Bc={bruss_unstable.Bc:.2f})')
    ax2.legend()
    ax2.grid(True, alpha=0.3)

    # 3. 相平面图
    ax3 = axes[1, 0]
    # 不同初始条件
    for x0 in np.linspace(0.1, 2.0, 5):
        for y0 in np.linspace(0.1, 3.0, 5):
            t, traj = bruss_unstable.simulate([x0, y0], t_span=(0, 30))
            ax3.plot(traj[:, 0], traj[:, 1], 'b-', alpha=0.3, linewidth=0.5)

    # 标记定态
    ss = bruss_unstable.steady_state()
    ax3.scatter([ss[0]], [ss[1]], c='r', s=100, zorder=5)
    ax3.set_xlabel('X')
    ax3.set_ylabel('Y')
    ax3.set_title('Phase Portrait (Unstable)')
    ax3.grid(True, alpha=0.3)

    # 4. 分岔图示意
    ax4 = axes[1, 1]
    A = 1.0
    B_range = np.linspace(0.5, 4.0, 100)
    Bc = 1 + A**2

    # 定态值
    X_steady = np.ones_like(B_range) * A
    ax4.plot(B_range, X_steady, 'b-', linewidth=2, label='Steady state X')
    ax4.axvline(Bc, color='r', linestyle='--', label=f'Critical Bc={Bc:.2f}')

    # 极限环幅度（示意）
    B_unstable = B_range[B_range > Bc]
    amplitude = np.sqrt(B_unstable - Bc)  # 简化的幅度增长
    ax4.plot(B_unstable, A + amplitude, 'r--', linewidth=2, label='Limit cycle max')
    ax4.plot(B_unstable, A - amplitude, 'r--', linewidth=2, label='Limit cycle min')

    ax4.set_xlabel('B')
    ax4.set_ylabel('X')
    ax4.set_title('Bifurcation Diagram')
    ax4.legend()
    ax4.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('dissipative_structures.png', dpi=150)
    plt.show()

    print("Dissipative structures visualization saved!")
    print(f"Critical value Bc = 1 + A² = {bruss_stable.Bc}")
