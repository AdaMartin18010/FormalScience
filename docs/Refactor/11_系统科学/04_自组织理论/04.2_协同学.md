## 11.14.1 协同学基本原理

### 11.14.1.1 序参量概念

**定义 11.14.1**（序参量）：描述系统宏观有序程度的变量 $\xi$，它支配子系统的行为。

**定理 11.14.1**（绝热消去原理）：快弛豫变量追随慢变量（序参量）：

$$\dot{q}_s = -\gamma_s q_s + f_s(q_f, q_s), \quad \gamma_s \ll \gamma_f$$

定态时：$q_f = q_f(q_s)$（快变量表示为序参量的函数）

### 11.14.1.2 役使原理

**定义 11.14.2**（役使原理）：序参量支配子系统，子系统通过反馈影响序参量。

**定理 11.14.2**（役使方程）：消去快变量后，序参量方程：

$$\dot{\xi} = N(\xi) + F(t)$$

其中 $N$ 为非线性函数，$F$ 为涨落力。

---

## 11.14.2 典型模型

### 11.14.2.1 激光模型

**定义 11.14.3**（激光序参量方程）：光场振幅 $E$ 的方程：

$$\dot{E} = (G - \kappa)E - \beta |E|^2 E$$

其中 $G$ 为增益，$\kappa$ 为损耗，$\beta$ 为饱和系数。

**定理 11.14.3**（激光阈值）：当 $G > \kappa$ 时，零场失稳，产生相干光。

### 11.14.2.2 Bénard对流

**定义 11.14.4**（Rayleigh数）：

$$Ra = \frac{g \alpha \Delta T d^3}{\nu \kappa}$$

**定理 11.14.4**（对流不稳定性）：当 $Ra > Ra_c \approx 1708$ 时，静止状态失稳，形成对流卷。

## 11.14.3 Python实现

```python
"""
自组织理论：协同学
序参量方程和役使原理
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint


class OrderParameter:
    """序参量动力学"""

    def __init__(self, a: float = -1.0, b: float = 1.0):
        """
        标准序参量方程: dξ/dt = aξ - bξ³

        a: 控制参数
        b: 非线性系数
        """
        self.a = a
        self.b = b

    def dynamics(self, xi, t):
        """序参量方程"""
        return self.a * xi - self.b * xi**3

    def steady_states(self):
        """定态解"""
        if self.a < 0:
            return [0]
        else:
            return [0, np.sqrt(self.a/self.b), -np.sqrt(self.a/self.b)]

    def potential(self, xi):
        """势函数 V(ξ) = -∫(dξ/dt)dξ"""
        return -0.5 * self.a * xi**2 + 0.25 * self.b * xi**4


class SlavedSystem:
    """
    被役使的子系统模型

    快变量 q 追随慢变量 ξ:
    dq/dt = -γq + cξ + ξ² (γ >> 1)
    """

    def __init__(self, gamma: float = 10.0, c: float = 1.0):
        self.gamma = gamma
        self.c = c

    def adiabatic_solution(self, xi):
        """绝热近似解: q ≈ (cξ + ξ²)/γ"""
        return (self.c * xi + xi**2) / self.gamma


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))

    # 1. 超临界分岔的势函数
    ax1 = axes[0, 0]
    xi_range = np.linspace(-2, 2, 200)

    for a in [-1, -0.5, 0, 0.5, 1]:
        op = OrderParameter(a=a, b=1)
        V = [op.potential(xi) for xi in xi_range]
        ax1.plot(xi_range, V, label=f'a={a}')

    ax1.set_xlabel('ξ')
    ax1.set_ylabel('V(ξ)')
    ax1.set_title('Potential Function (Supercritical Bifurcation)')
    ax1.legend()
    ax1.grid(True, alpha=0.3)

    # 2. 分岔图
    ax2 = axes[0, 1]
    a_range = np.linspace(-2, 2, 100)
    stable_branches = []
    unstable_branches = []

    for a in a_range:
        op = OrderParameter(a=a, b=1)
        states = op.steady_states()
        if a < 0:
            stable_branches.append((a, 0))
        else:
            stable_branches.extend([(a, states[1]), (a, states[2])])
            unstable_branches.append((a, 0))

    if stable_branches:
        a_stable, xi_stable = zip(*stable_branches)
        ax2.plot(a_stable, xi_stable, 'b-', linewidth=2, label='Stable')

    if unstable_branches:
        a_unstable, xi_unstable = zip(*unstable_branches)
        ax2.plot(a_unstable, xi_unstable, 'r--', linewidth=2, label='Unstable')

    ax2.axvline(0, color='k', linestyle=':', alpha=0.5)
    ax2.set_xlabel('a (control parameter)')
    ax2.set_ylabel('ξ (order parameter)')
    ax2.set_title('Bifurcation Diagram')
    ax2.legend()
    ax2.grid(True, alpha=0.3)

    # 3. 序参量演化
    ax3 = axes[1, 0]
    t = np.linspace(0, 10, 1000)

    for xi0 in [-1.5, -0.5, 0.5, 1.5]:
        op = OrderParameter(a=1, b=1)  # a > 0: 双稳态
        sol = odeint(op.dynamics, xi0, t)
        ax3.plot(t, sol, label=f'ξ(0)={xi0}')

    ax3.axhline(np.sqrt(1), color='k', linestyle='--', alpha=0.3)
    ax3.axhline(-np.sqrt(1), color='k', linestyle='--', alpha=0.3)
    ax3.set_xlabel('t')
    ax3.set_ylabel('ξ')
    ax3.set_title('Order Parameter Evolution (a=1)')
    ax3.legend()
    ax3.grid(True, alpha=0.3)

    # 4. 役使原理示意图
    ax4 = axes[1, 1]

    # 慢变量（序参量）
    t_slow = np.linspace(0, 10, 100)
    xi_slow = np.sin(0.5 * t_slow)  # 慢振荡

    # 快变量（被役使）
    slave = SlavedSystem(gamma=10)
    q_fast = [slave.adiabatic_solution(xi) for xi in xi_slow]

    ax4.plot(t_slow, xi_slow, 'b-', linewidth=2, label='Order parameter ξ(t)')
    ax4.plot(t_slow, q_fast, 'r--', linewidth=2, label='Slaved variable q(t)')
    ax4.set_xlabel('t')
    ax4.set_ylabel('Amplitude')
    ax4.set_title('Slaving Principle')
    ax4.legend()
    ax4.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('synergetics.png', dpi=150)
    plt.show()

    print("Synergetics visualization saved!")
