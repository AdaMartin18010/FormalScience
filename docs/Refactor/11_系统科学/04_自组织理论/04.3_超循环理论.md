## 11.15.1 超循环概念

### 11.15.1.1 分子自组织

**定义 11.15.1**（准种）：一组密切相关的RNA序列，通过突变相互连接。

**定义 11.15.2**（误差阈值）：可维持遗传信息的最大突变率：

$$q_{min} = \sigma_m^{-1}$$

其中 $\sigma_m$ 为最大复制率，$q = 1 - \epsilon$ 为复制保真度。

**定理 11.15.1**（误差灾变）：当突变率超过误差阈值时，遗传信息丢失。

### 11.15.1.2 超循环结构

**定义 11.15.3**（n-成员超循环）：

$$I_1 \to I_2 \to \cdots \to I_n \to I_1$$

其中 $I_i$ 为自复制单元（如RNA分子）。

**定义 11.15.4**（超循环方程）：

$$\dot{x}_i = x_i (A_i Q_i x_i + \sum_{j \neq i} k_{ij} x_j - \Phi)$$

其中 $\Phi = \sum_j A_j x_j$ 为通量归一化因子。

---

## 11.15.2 进化动力学

### 11.15.2.1 选择方程

**定义 11.15.5**（复制-突变方程）：

$$\dot{x}_i = \sum_j W_{ij} x_j - \bar{f} x_i$$

其中 $W_{ij} = A_j Q_{ij}$ 为复制-突变矩阵，$\bar{f} = \sum_i f_i x_i$ 为平均适应度。

**定理 11.15.2**（选择定理）：复制率最高的准种将主导系统。

### 11.15.2.2 协同进化

**定义 11.15.6**（协同进化）：多个物种（或分子）相互适应的演化过程。

"""
自组织理论：超循环理论
分子演化和超循环动力学
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint


class Quasispecies:
    """
    准种模型

    dx_i/dt = Σ_j W_ij x_j - f̄ x_i
    """

    def __init__(self, n_species: int, mutation_rate: float = 0.01):
        self.n = n_species
        self.mu = mutation_rate

        # 复制率（不同物种有不同的适应度）
        self.replication_rates = 1 + np.random.exponential(0.5, n_species)
        self.replication_rates[0] = 2.0  # 主序列有最高适应度

        # 构建W矩阵（复制+突变）
        self.W = self._build_W_matrix()

    def _build_W_matrix(self) -> np.ndarray:
        """构建复制-突变矩阵"""
        W = np.zeros((self.n, self.n))

        for j in range(self.n):
            for i in range(self.n):
                if i == j:
                    # 忠实复制
                    W[i, j] = self.replication_rates[j] * (1 - self.mu)
                else:
                    # 突变（简化模型：等概率突变到任何其他序列）
                    W[i, j] = self.replication_rates[j] * self.mu / (self.n - 1)

        return W

    def dynamics(self, x, t):
        """准种动力学"""
        # 平均适应度
        f_mean = np.sum(self.replication_rates * x)

        # 动力学方程
        dx = self.W @ x - f_mean * x
        return dx

    def simulate(self, x0: np.ndarray, t_span: tuple = (0, 50), n_points: int = 1000):
        """仿真"""
        t = np.linspace(t_span[0], t_span[1], n_points)
        trajectory = odeint(self.dynamics, x0, t)
        return t, trajectory


class Hypercycle:
    """
    超循环模型

    dx_i/dt = k_i x_i x_{i-1} - x_i Φ
    其中 Φ = Σ_j k_j x_j x_{j-1}
    """

    def __init__(self, n_members: int, k: float = 1.0):
        """
        Args:
            n_members: 超循环成员数
            k: 催化速率常数
        """
        self.n = n_members
        self.k = k

    def dynamics(self, x, t):
        """超循环动力学"""
        dx = np.zeros(self.n)

        # 计算通量
        flux = np.sum([self.k * x[i] * x[(i-1) % self.n] for i in range(self.n)])

        for i in range(self.n):
            # 催化增长 + 通量归一化
            growth = self.k * x[i] * x[(i-1) % self.n]
            dx[i] = x[i] * (growth - flux)

        return dx

    def simulate(self, x0: np.ndarray, t_span: tuple = (0, 100), n_points: int = 2000):
        """仿真"""
        t = np.linspace(t_span[0], t_span[1], n_points)
        trajectory = odeint(self.dynamics, x0, t)
        return t, trajectory


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))

    # 1. 准种演化
    ax1 = axes[0, 0]
    n_species = 5
    qs = Quasispecies(n_species, mutation_rate=0.05)

    # 初始：均匀分布
    x0 = np.ones(n_species) / n_species
    t, traj = qs.simulate(x0, t_span=(0, 30))

    for i in range(n_species):
        ax1.plot(t, traj[:, i], label=f'Species {i} (A={qs.replication_rates[i]:.2f})')

    ax1.set_xlabel('Time')
    ax1.set_ylabel('Concentration')
    ax1.set_title('Quasispecies Evolution')
    ax1.legend(fontsize=8)
    ax1.grid(True, alpha=0.3)

    # 2. 准种最终分布 vs 复制率
    ax2 = axes[0, 1]
    final_dist = traj[-1]
    ax2.scatter(qs.replication_rates, final_dist, s=100)

    for i, (A, x) in enumerate(zip(qs.replication_rates, final_dist)):
        ax2.annotate(f'{i}', (A, x), fontsize=10, ha='center')

    ax2.set_xlabel('Replication Rate A')
    ax2.set_ylabel('Final Concentration')
    ax2.set_title('Selection vs Fitness')
    ax2.grid(True, alpha=0.3)

    # 3. 超循环动力学
    ax3 = axes[1, 0]
    n_members = 4
    hyper = Hypercycle(n_members)

    # 初始条件：稍有差异
    x0_hyper = np.array([0.3, 0.25, 0.25, 0.2])
    t_hyper, traj_hyper = hyper.simulate(x0_hyper, t_span=(0, 50))

    for i in range(n_members):
        ax3.plot(t_hyper, traj_hyper[:, i], label=f'I_{i}')

    ax3.set_xlabel('Time')
    ax3.set_ylabel('Concentration')
    ax3.set_title(f'Hypercycle ({n_members} members)')
    ax3.legend()
    ax3.grid(True, alpha=0.3)

    # 4. 超循环相空间
    ax4 = axes[1, 1]
    # 简化：只画前3个成员的三维投影
    ax4.plot(traj_hyper[:, 0], traj_hyper[:, 1], 'b-', alpha=0.7, linewidth=1)
    ax4.scatter([traj_hyper[0, 0]], [traj_hyper[0, 1]], c='g', s=100, label='Start')
    ax4.scatter([traj_hyper[-1, 0]], [traj_hyper[-1, 1]], c='r', s=100, label='End')
    ax4.set_xlabel('I_0')
    ax4.set_ylabel('I_1')
    ax4.set_title('Hypercycle Phase Space')
    ax4.legend()
    ax4.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('hypercycle.png', dpi=150)
    plt.show()

    print("Hypercycle theory visualization saved!")
