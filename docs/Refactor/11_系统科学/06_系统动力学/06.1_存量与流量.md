## 11.21.1 Stock & Flow的形式化

### 11.21.1.1 存量（Stock）

**定义 11.21.1**（存量）：累积量，系统的状态变量：

$$S(t) = S(t_0) + \int_{t_0}^{t} [I(s) - O(s)] ds$$

其中 $I$ 为流入量，$O$ 为流出量。

**定义 11.21.2**（离散形式）：

$$S_{t+\Delta t} = S_t + \Delta t \cdot (Inflow_t - Outflow_t)$$

### 11.21.1.2 流量（Flow）

**定义 11.21.3**（流量）：单位时间内存量的变化率，不累积。

**定义 11.21.4**（净流量）：

$$Net\ Flow = Inflow - Outflow = \frac{dS}{dt}$$

---

## 11.21.2 延迟结构

### 11.21.2.1 物质延迟

**定义 11.21.5**（一阶物质延迟）：

$$\frac{dS}{dt} = \frac{Inflow - S/D}{1} = \frac{Inflow}{D} - \frac{S}{D}$$

其中 $D$ 为延迟时间。

**定理 11.21.1**（延迟输出）：输出等于存量除以延迟时间：

$$Outflow = \frac{S}{D}$$

### 11.21.2.2 信息延迟

**定义 11.21.6**（平滑函数）：

$$\frac{dS}{dt} = \frac{Input - S}{D}$$

即指数平滑：$S$ 是对 $Input$ 的平滑估计。

## 11.21.3 Python实现

```python
"""
系统动力学：存量与流量
Stock & Flow建模基础
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint


class Stock:
    """存量模型"""

    def __init__(self, initial_value: float, inflow_func=None, outflow_func=None):
        self.value = initial_value
        self.history = [initial_value]
        self.inflow_func = inflow_func
        self.outflow_func = outflow_func

    def update(self, dt: float, t: float, **kwargs):
        """更新存量"""
        inflow = self.inflow_func(t, **kwargs) if self.inflow_func else 0
        outflow = self.outflow_func(t, **kwargs) if self.outflow_func else 0

        self.value += dt * (inflow - outflow)
        self.history.append(self.value)


class MaterialDelay:
    """物质延迟"""

    def __init__(self, delay_time: float, initial_input: float = 0):
        self.D = delay_time
        self.stock = initial_input * delay_time
        self.history = [self.stock]

    def output(self) -> float:
        """延迟输出"""
        return self.stock / self.D

    def update(self, inflow: float, dt: float):
        """更新延迟存量"""
        outflow = self.output()
        self.stock += dt * (inflow - outflow)
        self.history.append(self.stock)
        return outflow


class Smooth:
    """信息延迟（平滑）"""

    def __init__(self, delay_time: float, initial_value: float = 0):
        self.D = delay_time
        self.smoothed = initial_value
        self.history = [initial_value]

    def update(self, input_val: float, dt: float):
        """指数平滑"""
        self.smoothed += dt * (input_val - self.smoothed) / self.D
        self.history.append(self.smoothed)
        return self.smoothed


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))

    # 1. 简单存量流量
    ax1 = axes[0, 0]
    dt = 0.1
    t_max = 50
    t = np.arange(0, t_max, dt)

    # 人口模型
    population = [100]
    birth_rate = 0.05
    death_rate = 0.02

    for i in range(1, len(t)):
        pop = population[-1]
        births = birth_rate * pop
        deaths = death_rate * pop
        population.append(pop + dt * (births - deaths))

    ax1.plot(t, population, 'b-', linewidth=2)
    ax1.set_xlabel('Time')
    ax1.set_ylabel('Population')
    ax1.set_title('Exponential Growth (Stock & Flow)')
    ax1.grid(True, alpha=0.3)

    # 2. 物质延迟
    ax2 = axes[0, 1]
    delay = MaterialDelay(delay_time=5, initial_input=10)
    input_signal = [10 if i < 100 else 20 for i in range(len(t))]
    output_signal = [delay.output()]

    for i in range(1, len(t)):
        out = delay.update(input_signal[i], dt)
        output_signal.append(out)

    ax2.plot(t, input_signal, 'r--', label='Input', linewidth=2)
    ax2.plot(t, output_signal, 'b-', label='Delayed Output', linewidth=2)
    ax2.set_xlabel('Time')
    ax2.set_ylabel('Flow')
    ax2.set_title(f'Material Delay (D={delay.D})')
    ax2.legend()
    ax2.grid(True, alpha=0.3)

    # 3. 平滑（信息延迟）
    ax3 = axes[1, 0]
    smooth = Smooth(delay_time=10, initial_value=0)

    # 阶跃输入
    step_input = [0 if i < 200 else 100 for i in range(len(t))]
    smoothed = [smooth.smoothed]

    for i in range(1, len(t)):
        val = smooth.update(step_input[i], dt)
        smoothed.append(val)

    ax3.plot(t, step_input, 'r--', label='Input', linewidth=2)
    ax3.plot(t, smoothed, 'b-', label='Smoothed', linewidth=2)
    ax3.axhline(y=63.2, color='g', linestyle=':', alpha=0.7, label='63.2% (1 time constant)')
    ax3.set_xlabel('Time')
    ax3.set_ylabel('Value')
    ax3.set_title(f'Information Delay/Smooth (D={smooth.D})')
    ax3.legend()
    ax3.grid(True, alpha=0.3)

    # 4. 多阶延迟比较
    ax4 = axes[1, 1]

    delays = [1, 3, 5]
    colors = ['blue', 'green', 'red']

    for D, color in zip(delays, colors):
        delay = MaterialDelay(delay_time=D, initial_input=0)

        # 脉冲输入
        pulse_input = [10 if 20 <= i < 50 else 0 for i in range(len(t))]
        output = [delay.output()]

        for i in range(1, len(t)):
            out = delay.update(pulse_input[i], dt)
            output.append(out)

        ax4.plot(t, output, color=color, label=f'D={D}', linewidth=2)

    ax4.set_xlabel('Time')
    ax4.set_ylabel('Output')
    ax4.set_title('Multi-Order Delays')
    ax4.legend()
    ax4.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('stock_flow.png', dpi=150)
    plt.show()

    print("Stock & Flow visualization saved!")
