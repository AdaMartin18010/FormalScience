## 11.22.1 增强回路

### 11.22.1.1 正反馈

**定义 11.22.1**（增强回路）：变化导致同方向更多变化的回路。

**数学形式**：

$$\frac{dX}{dt} = rX \Rightarrow X(t) = X_0 e^{rt}$$

**例子**：

- 复利增长
- 病毒传播
- 军备竞赛

### 11.22.1.2 超指数增长

**定义 11.22.2**（有限增长）：考虑承载能力的增长：

$$\frac{dX}{dt} = rX\left(1 - \frac{X}{K}\right)$$

---

## 11.22.2 调节回路

### 11.22.2.1 负反馈

**定义 11.22.3**（调节回路）：变化导致反方向响应的回路，追求平衡或目标。

**数学形式**：

$$\frac{dX}{dt} = -k(X - X^*)$$

解为指数趋近目标：$X(t) = X^* + (X_0 - X^*)e^{-kt}$

**例子**：

- 恒温器控制
- 库存管理
- 血糖调节

### 11.22.2.2 延迟与振荡

**定义 11.22.4**（振荡产生）：调节回路中的延迟导致超调和振荡。

## 11.22.3 Python实现

```python
"""
系统动力学：反馈回路
增强回路和调节回路
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint


def reinforcing_loop(initial: float, rate: float, t: np.ndarray) -> np.ndarray:
    """增强回路: dx/dt = r*x"""
    return initial * np.exp(rate * t)


def balancing_loop(initial: float, target: float, rate: float, t: np.ndarray) -> np.ndarray:
    """调节回路: dx/dt = -k*(x - target)"""
    return target + (initial - target) * np.exp(-rate * t)


def limited_growth(initial: float, rate: float, capacity: float, t: np.ndarray) -> np.ndarray:
    """有限增长: dx/dt = r*x*(1 - x/K)"""
    def model(x, t):
        return rate * x * (1 - x / capacity)

    return odeint(model, initial, t).flatten()


def oscillation_loop(initial: float, target: float, rate: float, delay: float, t: np.ndarray) -> np.ndarray:
    """带延迟的调节回路导致振荡（简化）"""
    # 使用延迟微分方程的近似
    result = [initial]
    dt = t[1] - t[0]

    for i in range(1, len(t)):
        # 当前误差
        if i < int(delay / dt):
            delayed_val = initial
        else:
            delayed_val = result[i - int(delay / dt)]

        error = target - delayed_val
        dx = rate * error * dt
        result.append(result[-1] + dx)

    return np.array(result)


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))
    t = np.linspace(0, 10, 1000)

    # 1. 增强回路（指数增长）
    ax1 = axes[0, 0]
    rates = [0.1, 0.3, 0.5]
    colors = ['blue', 'green', 'red']

    for rate, color in zip(rates, colors):
        growth = reinforcing_loop(1.0, rate, t)
        ax1.semilogy(t, growth, color=color, label=f'r={rate}', linewidth=2)

    ax1.set_xlabel('Time')
    ax1.set_ylabel('Stock (log scale)')
    ax1.set_title('Reinforcing Loop: Exponential Growth')
    ax1.legend()
    ax1.grid(True, alpha=0.3)

    # 2. 调节回路（指数趋近）
    ax2 = axes[0, 1]
    target = 100
    rates_bal = [0.5, 1.0, 2.0]

    for rate, color in zip(rates_bal, colors):
        balance = balancing_loop(0, target, rate, t)
        ax2.plot(t, balance, color=color, label=f'k={rate}', linewidth=2)

    ax2.axhline(target, color='k', linestyle='--', alpha=0.5, label='Target')
    ax2.set_xlabel('Time')
    ax2.set_ylabel('Stock')
    ax2.set_title('Balancing Loop: Exponential Approach')
    ax2.legend()
    ax2.grid(True, alpha=0.3)

    # 3. 有限增长（S曲线）
    ax3 = axes[1, 0]
    capacities = [50, 100, 200]

    for cap, color in zip(capacities, colors):
        s_curve = limited_growth(1.0, 0.5, cap, t)
        ax3.plot(t, s_curve, color=color, label=f'K={cap}', linewidth=2)

    ax3.set_xlabel('Time')
    ax3.set_ylabel('Stock')
    ax3.set_title('Limited Growth (S-Curve)')
    ax3.legend()
    ax3.grid(True, alpha=0.3)

    # 4. 延迟导致的振荡
    ax4 = axes[1, 1]
    delays = [0.5, 1.0, 2.0]

    for delay, color in zip(delays, colors):
        osc = oscillation_loop(0, 100, 1.0, delay, t)
        ax4.plot(t, osc, color=color, label=f'Delay={delay}', linewidth=2)

    ax4.axhline(100, color='k', linestyle='--', alpha=0.5, label='Target')
    ax4.set_xlabel('Time')
    ax4.set_ylabel('Stock')
    ax4.set_title('Oscillation from Delay in Balancing Loop')
    ax4.legend()
    ax4.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('feedback_loops.png', dpi=150)
    plt.show()

    print("Feedback loops visualization saved!")
