## 11.19.1 网络上的传播

### 11.19.1.1 SI模型

**定义 11.19.1**（SI模型）：个体处于易感(S)或感染(I)状态：

$$\frac{d\rho_k}{dt} = \lambda k (1 - \rho_k) \Theta(\lambda)$$

其中 $\rho_k$ 为度$k$节点的感染概率，$\Theta$ 为随机指向的边连接到感染节点的概率。

### 11.19.1.2 SIR和SIS模型

**定义 11.19.2**（SIR模型）：加入恢复(R)状态：

$$\begin{cases}
\frac{dS}{dt} = -\lambda SI \\
\frac{dI}{dt} = \lambda SI - \mu I \\
\frac{dR}{dt} = \mu I
\end{cases}$$

**定理 11.19.1**（无标度网络的零阈值）：对于度分布 $P(k) \sim k^{-\gamma}$ 且 $\gamma \leq 3$ 的网络，流行病阈值 $\lambda_c = 0$。

## 11.19.2 网络同步

### 11.19.2.1 Kuramoto模型

**定义 11.19.3**（网络Kuramoto模型）：

$$\frac{d\theta_i}{dt} = \omega_i + \lambda \sum_j A_{ij} \sin(\theta_j - \theta_i)$$

**定义 11.19.4**（序参量）：

$$r e^{i\psi} = \frac{1}{N} \sum_j e^{i\theta_j}$$

$r = 1$ 表示完全同步。

### 11.19.2.2 主稳定函数方法

**定理 11.19.2**（同步稳定性）：线性化后，同步流形的稳定性由Laplacian特征值决定。

"""
网络科学：网络动力学
传播模型和同步
"""

import numpy as np
import matplotlib.pyplot as plt
import networkx as nx
from scipy.integrate import odeint


class SIRModel:
    """SIR流行病模型"""

    def __init__(self, G: nx.Graph, beta: float = 0.3, gamma: float = 0.1):
        """
        Args:
            G: 接触网络
            beta: 感染率
            gamma: 恢复率
        """
        self.G = G
        self.beta = beta
        self.gamma = gamma
        self.n = G.number_of_nodes()

    def simulate(self, initial_infected: list, t_max: int = 100) -> dict:
        """仿真SIR动力学"""
        S = set(self.G.nodes()) - set(initial_infected)
        I = set(initial_infected)
        R = set()

        history = {'S': [len(S)], 'I': [len(I)], 'R': [len(R)]}

        for t in range(t_max):
            new_I = set()
            new_R = set()

            # 感染过程
            for infected in I:
                for neighbor in self.G.neighbors(infected):
                    if neighbor in S and np.random.random() < self.beta:
                        new_I.add(neighbor)

            # 恢复过程
            for infected in I:
                if np.random.random() < self.gamma:
                    new_R.add(infected)

            # 更新状态
            S -= new_I
            I = (I - new_R) | new_I
            R |= new_R

            history['S'].append(len(S))
            history['I'].append(len(I))
            history['R'].append(len(R))

            if len(I) == 0:
                break

        return history


class KuramotoNetwork:
    """网络上的Kuramoto模型"""

    def __init__(self, G: nx.Graph, omega_mean: float = 0, omega_std: float = 1.0):
        self.G = G
        self.n = G.number_of_nodes()
        self.omega = np.random.normal(omega_mean, omega_std, self.n)
        self.A = nx.to_numpy_array(G)

    def dynamics(self, theta, t, lambda_coupling):
        """Kuramoto动力学"""
        dtheta = self.omega.copy()

        for i in range(self.n):
            for j in range(self.n):
                if self.A[i, j] > 0:
                    dtheta[i] += lambda_coupling * np.sin(theta[j] - theta[i])

        return dtheta

    def order_parameter(self, theta):
        """计算序参量 r"""
        return abs(np.mean(np.exp(1j * theta)))

    def simulate(self, theta0: np.ndarray, lambda_coupling: float,
                t_span: tuple = (0, 20), n_points: int = 200):
        """仿真"""
        t = np.linspace(t_span[0], t_span[1], n_points)

        def dyn(theta, t):
            return self.dynamics(theta, t, lambda_coupling)

        trajectory = odeint(dyn, theta0, t)

        r_values = [self.order_parameter(theta) for theta in trajectory]

        return t, trajectory, r_values


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))

    # 1. SIR在不同网络上的传播
    ax1 = axes[0, 0]

    networks = {
        'Erdős-Rényi': nx.erdos_renyi_graph(1000, 0.01),
        'Barabási-Albert': nx.barabasi_albert_graph(1000, 3)
    }

    for name, G in networks.items():
        sir = SIRModel(G, beta=0.3, gamma=0.1)
        initial = list(np.random.choice(G.nodes(), 10, replace=False))
        history = sir.simulate(initial, t_max=100)

        ax1.plot(history['I'], label=f'{name} (I)', linewidth=2)

    ax1.set_xlabel('Time')
    ax1.set_ylabel('Infected')
    ax1.set_title('SIR Spread on Different Networks')
    ax1.legend()
    ax1.grid(True, alpha=0.3)

    # 2. 最终感染比例 vs 感染率
    ax2 = axes[0, 1]
    G = nx.barabasi_albert_graph(500, 3)
    beta_range = np.linspace(0.05, 0.5, 15)
    final_infected = []

    for beta in beta_range:
        sir = SIRModel(G, beta=beta, gamma=0.1)
        initial = list(np.random.choice(G.nodes(), 5, replace=False))
        history = sir.simulate(initial, t_max=150)
        final_infected.append(history['R'][-1] / G.number_of_nodes())

    ax2.plot(beta_range, final_infected, 'b-o', linewidth=2)
    ax2.set_xlabel('Infection rate β')
    ax2.set_ylabel('Final recovered fraction')
    ax2.set_title('Epidemic Threshold')
    ax2.grid(True, alpha=0.3)

    # 3. Kuramoto同步
    ax3 = axes[1, 0]
    G_kuramoto = nx.watts_strogatz_graph(50, 6, 0.3)
    kuramoto = KuramotoNetwork(G_kuramoto, omega_std=1.0)

    lambda_values = np.linspace(0, 2, 20)
    r_final = []

    for lam in lambda_values:
        theta0 = np.random.uniform(0, 2*np.pi, kuramoto.n)
        t, traj, r_vals = kuramoto.simulate(theta0, lam, t_span=(0, 50))
        r_final.append(np.mean(r_vals[-50:]))  # 取后期平均值

    ax3.plot(lambda_values, r_final, 'r-s', linewidth=2)
    ax3.set_xlabel('Coupling strength λ')
    ax3.set_ylabel('Order parameter r')
    ax3.set_title('Kuramoto Synchronization')
    ax3.grid(True, alpha=0.3)

    # 4. 相位演化
    ax4 = axes[1, 1]
    theta0 = np.random.uniform(0, 2*np.pi, kuramoto.n)
    t, traj, r_vals = kuramoto.simulate(theta0, lambda_coupling=1.5, t_span=(0, 30))

    for i in range(0, kuramoto.n, 5):  # 只画部分轨迹
        ax4.plot(t, np.sin(traj[:, i]), alpha=0.5, linewidth=1)

    ax4.set_xlabel('Time')
    ax4.set_ylabel('sin(θ)')
    ax4.set_title(f'Phase Evolution (λ=1.5, r_final={r_final[-1]:.3f})')
    ax4.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('network_dynamics.png', dpi=150)
    plt.show()

    print("Network dynamics visualization saved!")
