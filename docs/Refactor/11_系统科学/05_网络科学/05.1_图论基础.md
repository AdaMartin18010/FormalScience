## 11.17.1 图的形式化定义

### 11.17.1.1 基本定义

**定义 11.17.1**（图）：图 $G$ 是有序对 $G = (V, E)$，其中：

- $V$：顶点（节点）集合
- $E \subseteq V \times V$：边集合

**定义 11.17.2**（邻接矩阵）：对于 $n$ 个顶点的图：

$$A_{ij} = \begin{cases} 1 & \text{if } (i, j) \in E \\ 0 & \text{otherwise} \end{cases}$$

**定义 11.17.3**（度）：顶点 $i$ 的度 $k_i$：

$$k_i = \sum_j A_{ij}$$

**定理 11.17.1**（握手引理）：

$$\sum_i k_i = 2|E|$$

### 11.17.1.2 特殊图类

**定义 11.17.4**（完全图 $K_n$）：每对顶点都相连的图。

**定义 11.17.5**（二分图）：顶点可分为两组，组内无边。

**定义 11.17.6**（树）：连通无环图，$|E| = |V| - 1$。

## 11.17.2 网络度量

### 11.17.2.1 中心性度量

**定义 11.17.7**（度中心性）：

$$C_D(i) = \frac{k_i}{n-1}$$

**定义 11.17.8**（介数中心性）：

$$C_B(i) = \sum_{s \neq i \neq t} \frac{\sigma_{st}(i)}{\sigma_{st}}$$

其中 $\sigma_{st}$ 为 $s$ 到 $t$ 的最短路径数，$\sigma_{st}(i)$ 为经过 $i$ 的路径数。

**定义 11.17.9**（接近中心性）：

$$C_C(i) = \frac{n-1}{\sum_{j \neq i} d_{ij}}$$

**定义 11.17.10**（特征向量中心性）：

$$C_E(i) = \frac{1}{\lambda} \sum_j A_{ij} C_E(j)$$

即邻接矩阵主特征向量的分量。

### 11.17.2.2 全局度量

**定义 11.17.11**（平均路径长度）：

$$L = \frac{1}{n(n-1)} \sum_{i \neq j} d_{ij}$$

**定义 11.17.12**（聚类系数）：

$$C = \frac{1}{n} \sum_i C_i = \frac{1}{n} \sum_i \frac{2e_i}{k_i(k_i-1)}$$

其中 $e_i$ 为邻居之间的边数。

## 11.17.3 Python实现

```python
"""
网络科学：图论基础
网络度量计算
"""

import numpy as np
import matplotlib.pyplot as plt
import networkx as nx


class NetworkMetrics:
    """网络度量计算"""

    def __init__(self, adjacency_matrix: np.ndarray):
        self.A = adjacency_matrix
        self.n = adjacency_matrix.shape[0]
        self.G = nx.from_numpy_array(adjacency_matrix)

    def degree_centrality(self) -> dict:
        """度中心性"""
        return nx.degree_centrality(self.G)

    def betweenness_centrality(self) -> dict:
        """介数中心性"""
        return nx.betweenness_centrality(self.G)

    def closeness_centrality(self) -> dict:
        """接近中心性"""
        return nx.closeness_centrality(self.G)

    def eigenvector_centrality(self) -> dict:
        """特征向量中心性"""
        try:
            return nx.eigenvector_centrality(self.G, max_iter=1000)
        except:
            return {i: 0 for i in range(self.n)}

    def clustering_coefficient(self) -> float:
        """全局聚类系数"""
        return nx.average_clustering(self.G)

    def average_path_length(self) -> float:
        """平均路径长度"""
        try:
            return nx.average_shortest_path_length(self.G)
        except:
            return float('inf')


def create_special_graphs():
    """创建特殊图"""
    graphs = {
        'Complete': nx.complete_graph(8),
        'Star': nx.star_graph(7),
        'Cycle': nx.cycle_graph(8),
        'Wheel': nx.wheel_graph(8),
        'Random': nx.erdos_renyi_graph(20, 0.15),
        'Scale-Free': nx.barabasi_albert_graph(20, 2)
    }
    return graphs


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 3, figsize=(15, 10))

    graphs = create_special_graphs()

    for idx, (name, G) in enumerate(graphs.items()):
        ax = axes[idx // 3, idx % 3]

        # 布局
        if name == 'Star':
            pos = nx.spring_layout(G, k=2)
        elif name in ['Cycle', 'Wheel']:
            pos = nx.circular_layout(G)
        else:
            pos = nx.spring_layout(G, k=1, iterations=50)

        # 绘制
        nx.draw(G, pos, ax=ax, node_color='lightblue',
               node_size=300, with_labels=True, font_size=8)
        ax.set_title(f'{name} Graph\n(n={G.number_of_nodes()}, m={G.number_of_edges()})')

    plt.tight_layout()
    plt.savefig('graph_basics.png', dpi=150)
    plt.show()

    # 计算并显示度量
    print("Network Metrics Comparison:")
    print("-" * 60)
    print(f"{'Graph':<15} {'Avg Degree':<12} {'Clustering':<12} {'Avg Path':<12}")
    print("-" * 60)

    for name, G in graphs.items():
        metrics = NetworkMetrics(nx.to_numpy_array(G))
        avg_deg = 2 * G.number_of_edges() / G.number_of_nodes()
        clustering = metrics.clustering_coefficient()
        avg_path = metrics.average_path_length()

        print(f"{name:<15} {avg_deg:<12.3f} {clustering:<12.3f} {avg_path:<12.3f}")

    print("\nGraph theory visualization saved!")
