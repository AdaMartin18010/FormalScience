## 11.18.1 随机图模型

### 11.18.1.1 Erdős-Rényi模型

**定义 11.18.1**（$G(n, p)$模型）：$n$个顶点，每对顶点以概率$p$独立连接。

**定理 11.18.1**（度分布）：$G(n, p)$的度分布为二项分布：

$$P(k) = \binom{n-1}{k} p^k (1-p)^{n-1-k} \approx \frac{\lambda^k e^{-\lambda}}{k!}$$

其中 $\lambda = (n-1)p$。

**定理 11.18.2**（相变）：当 $p = \frac{\ln n}{n}$ 时，图以高概率连通。

**定理 11.18.3**（巨分量相变）：当 $p = \frac{1}{n}$ 时，出现巨分量。

---

## 11.18.2 小世界网络

### 11.18.2.1 Watts-Strogatz模型

**定义 11.18.2**（WS模型）：

1. 从规则格点开始（每个节点连接$k$个最近邻）
2. 以概率$p$重连每条边

**定理 11.18.4**（小世界特性）：对于小的$p$：

- 平均路径长度 $L \sim \ln n$（小世界）
- 聚类系数 $C$ 保持较高

### 11.18.2.2 小世界特性度量

**定义 11.18.3**（小世界系数）：

$$\sigma = \frac{C/C_{rand}}{L/L_{rand}}$$

若 $\sigma > 1$，则为小世界网络。

---

## 11.18.3 无标度网络

### 11.18.3.1 Barabási-Albert模型

**定义 11.18.4**（BA模型）：

1. **增长**：网络从$m_0$个节点开始，每次添加一个带$m$条边的新节点
2. **偏好依附**：新节点连接到节点$i$的概率：

$$P_i = \frac{k_i}{\sum_j k_j}$$

**定理 11.18.5**（度分布）：BA网络的度分布服从幂律：

$$P(k) \sim k^{-\gamma}, \quad \gamma = 3$$

### 11.18.3.2 度相关性

**定义 11.18.5**（同配系数）：

$$r = \frac{\sum_{ij} (A_{ij} - k_i k_j / 2m) k_i k_j}{\sum_{ij} (k_i \delta_{ij} - k_i k_j / 2m) k_i k_j}$$

- $r > 0$：同配（高度节点连接高度节点）
- $r < 0$：异配

## 11.18.4 Python实现

```python
"""
网络科学：复杂网络模型
随机图、小世界和无标度网络
"""

import numpy as np
import matplotlib.pyplot as plt
import networkx as nx
from scipy.stats import powerlaw


def analyze_degree_distribution(G, ax=None, label=''):
    """分析度分布"""
    degrees = [d for n, d in G.degree()]

    if ax is None:
        fig, ax = plt.subplots()

    # 累积分布
    unique_degrees = sorted(set(degrees))
    cumprob = [np.mean(np.array(degrees) >= d) for d in unique_degrees]

    ax.loglog(unique_degrees, cumprob, 'o', label=label, alpha=0.7)
    ax.set_xlabel('Degree k')
    ax.set_ylabel('P(K ≥ k)')
    ax.legend()
    ax.grid(True, alpha=0.3)

    return degrees


if __name__ == "__main__":
    fig, axes = plt.subplots(2, 3, figsize=(15, 10))
    n = 500

    # 1. Erdős-Rényi随机图
    ax1 = axes[0, 0]
    p = 0.02
    G_er = nx.erdos_renyi_graph(n, p)
    pos = nx.spring_layout(G_er, k=0.5, iterations=50)
    nx.draw(G_er, pos, ax=ax1, node_size=10, node_color='blue', alpha=0.6)
    ax1.set_title(f'Erdős-Rényi (n={n}, p={p})')

    # 2. Watts-Strogatz小世界
    ax2 = axes[0, 1]
    k, p_ws = 6, 0.3
    G_ws = nx.watts_strogatz_graph(n, k, p_ws)
    pos = nx.circular_layout(G_ws)
    nx.draw(G_ws, pos, ax=ax2, node_size=10, node_color='green', alpha=0.6)
    ax2.set_title(f'Watts-Strogatz (n={n}, k={k}, p={p_ws})')

    # 3. Barabási-Albert无标度
    ax3 = axes[0, 2]
    m = 3
    G_ba = nx.barabasi_albert_graph(n, m)
    pos = nx.spring_layout(G_ba, k=0.5, iterations=50)
    nx.draw(G_ba, pos, ax=ax3, node_size=10, node_color='red', alpha=0.6)
    ax3.set_title(f'Barabási-Albert (n={n}, m={m})')

    # 4. 度分布比较
    ax4 = axes[1, 0]
    analyze_degree_distribution(G_er, ax4, 'Erdős-Rényi')
    analyze_degree_distribution(G_ws, ax4, 'Watts-Strogatz')
    analyze_degree_distribution(G_ba, ax4, 'Barabási-Albert')
    ax4.set_title('Degree Distributions')

    # 5. 网络特性比较
    ax5 = axes[1, 1]
    networks = {
        'Erdős-Rényi': G_er,
        'Watts-Strogatz': G_ws,
        'Barabási-Albert': G_ba
    }

    metrics = ['Avg Degree', 'Clustering', 'Path Length', 'Max Degree']
    data = {name: [] for name in networks}

    for name, G in networks.items():
        avg_deg = 2 * G.number_of_edges() / G.number_of_nodes()
        clustering = nx.average_clustering(G)
        try:
            path_len = nx.average_shortest_path_length(G)
        except:
            path_len = 0
        max_deg = max(dict(G.degree()).values())

        data[name] = [avg_deg, clustering, path_len, max_deg]

    x = np.arange(len(metrics))
    width = 0.25

    for i, (name, values) in enumerate(data.items()):
        # 归一化用于显示
        ax5.bar(x + i*width, values, width, label=name)

    ax5.set_xticks(x + width)
    ax5.set_xticklabels(metrics, rotation=15)
    ax5.legend()
    ax5.set_title('Network Properties Comparison')
    ax5.grid(True, alpha=0.3, axis='y')

    # 6. 重连概率对小世界特性的影响
    ax6 = axes[1, 2]
    p_values = np.logspace(-4, 0, 20)
    L_values = []
    C_values = []

    for p in p_values:
        G = nx.watts_strogatz_graph(200, 6, p)
        try:
            L_values.append(nx.average_shortest_path_length(G))
        except:
            L_values.append(0)
        C_values.append(nx.average_clustering(G))

    # 归一化
    L_norm = np.array(L_values) / L_values[0]
    C_norm = np.array(C_values) / C_values[0]

    ax6.semilogx(p_values, L_norm, 'b-o', label='L(p)/L(0)')
    ax6.semilogx(p_values, C_norm, 'r-s', label='C(p)/C(0)')
    ax6.axvline(x=0.01, color='k', linestyle='--', alpha=0.5, label='Small-world regime')
    ax6.set_xlabel('Rewiring probability p')
    ax6.set_ylabel('Normalized value')
    ax6.set_title('Small-World Transition')
    ax6.legend()
    ax6.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('complex_networks.png', dpi=150)
    plt.show()

    # 打印详细统计
    print("Complex Network Models Analysis:")
    print("-" * 70)
    print(f"{'Network':<20} {'<k>':<10} {'C':<10} {'L':<10} {'r':<10}")
    print("-" * 70)

    for name, G in networks.items():
        avg_deg = 2 * G.number_of_edges() / G.number_of_nodes()
        clustering = nx.average_clustering(G)
        try:
            path_len = nx.average_shortest_path_length(G)
        except:
            path_len = 0
        try:
            assortativity = nx.degree_assortativity_coefficient(G)
        except:
            assortativity = 0

        print(f"{name:<20} {avg_deg:<10.2f} {clustering:<10.3f} {path_len:<10.2f} {assortativity:<10.3f}")

    print("\nComplex networks visualization saved!")
