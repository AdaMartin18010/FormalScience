## 11.7.1 Pontryagin极大值原理

### 11.7.1.1 最优控制问题表述

**定义 11.7.1**（最优控制问题）：寻找控制 $u(t) \in \mathcal{U}$ 使性能指标最小：

$$J = \Phi(x(t_f), t_f) + \int_{t_0}^{t_f} L(x(t), u(t), t) dt$$

约束条件：
$$\dot{x} = f(x, u, t), \quad x(t_0) = x_0$$

其中：

- $x \in \mathbb{R}^n$：状态向量
- $u \in \mathbb{R}^m$：控制向量
- $\mathcal{U}$：容许控制集合
- $\Phi$：终端代价
- $L$：运行代价

### 11.7.1.2 Hamiltonian与协态方程

**定义 11.7.2**（Hamiltonian函数）：

$$H(x, u, \lambda, t) = L(x, u, t) + \lambda^T f(x, u, t)$$

其中 $\lambda \in \mathbb{R}^n$ 为协态（costate）向量。

**定理 11.7.1**（Pontryagin极大值原理）：最优控制 $u^*(t)$ 满足：

1. **状态方程**：
   $$\dot{x}^* = \frac{\partial H}{\partial \lambda} = f(x^*, u^*, t)$$

2. **协态方程**：
   $$\dot{\lambda} = -\frac{\partial H}{\partial x}$$

3. **极小值条件**：
   $$H(x^*, u^*, \lambda^*, t) = \min_{u \in \mathcal{U}} H(x^*, u, \lambda^*, t)$$

4. **横截条件**：
   $$\lambda(t_f) = \frac{\partial \Phi}{\partial x}|_{t_f}$$

**证明概要**：

通过变分法，考虑状态和控制的变化 $\delta x$ 和 $\delta u$。引入拉格朗日乘子 $\lambda$ 处理动力学约束，构造增广泛函：

$$\bar{J} = \Phi + \int_{t_0}^{t_f} [L + \lambda^T(f - \dot{x})] dt$$

分部积分并令变分为零，导出必要条件。$\square$

### 11.7.1.3 Bang-Bang控制

**定义 11.7.3**（Bang-Bang控制）：控制仅在容许集的边界取值：

$$u^*(t) \in \{u_{min}, u_{max}\}$$

**定理 11.7.2**（线性系统的Bang-Bang原理）：对于系统 $\dot{x} = Ax + Bu$，若控制约束为 $|u_i| \leq 1$，则最优控制为Bang-Bang型。

## 11.7.2 动态规划

### 11.7.2.1 Bellman最优性原理

**定理 11.7.3**（Bellman最优性原理）：最优策略具有如下性质：无论初始状态和初始决策如何，剩余的决策对于由初始决策产生的状态构成最优策略。

**定义 11.7.4**（值函数）：从状态 $x$ 时刻 $t$ 开始的最优代价：

$$V(x, t) = \min_{u(\cdot)} \left[ \Phi(x(t_f), t_f) + \int_{t}^{t_f} L(x(\tau), u(\tau), \tau) d\tau \right]$$

### 11.7.2.2 Hamilton-Jacobi-Bellman方程

**定理 11.7.4**（HJB方程）：值函数满足：

$$-\frac{\partial V}{\partial t} = \min_{u} \left[ L(x, u, t) + \frac{\partial V}{\partial x} f(x, u, t) \right]$$

边界条件：$V(x, t_f) = \Phi(x, t_f)$

**证明**：

对小区间 $[t, t + \Delta t]$ 应用最优性原理：

$$V(x, t) = \min_{u} \left[ L(x, u, t)\Delta t + V(x + f(x,u,t)\Delta t, t + \Delta t) \right]$$

Taylor展开并令 $\Delta t \to 0$ 即得。$\square$

**定理 11.7.5**（验证定理）：若 $V(x, t)$ 满足HJB方程且对应控制 $u^*(x, t)$ 可容许，则 $u^*$ 最优。

---

## 11.7.3 线性二次调节器（LQR）

### 11.7.3.1 LQR问题表述

**定义 11.7.5**（LQR问题）：对于线性系统：

$$\dot{x} = Ax + Bu$$

最小化二次性能指标：

$$J = \int_{0}^{\infty} (x^T Q x + u^T R u) dt$$

其中 $Q \succeq 0$，$R \succ 0$。

### 11.7.3.2 Riccati方程

**定理 11.7.6**（LQR最优解）：最优控制为状态反馈：

$$u^*(t) = -Kx(t) = -R^{-1}B^T P x(t)$$

其中 $P$ 满足代数Riccati方程：

$$A^T P + PA - PBR^{-1}B^T P + Q = 0$$

**证明**：

设值函数 $V(x) = x^T P x$，则：

$$\frac{\partial V}{\partial x} = 2x^T P, \quad \frac{\partial V}{\partial t} = 0$$

代入HJB方程：

$$0 = \min_{u} [x^T Q x + u^T R u + 2x^T P(Ax + Bu)]$$

对 $u$ 求导得最优控制：

$$2R u + 2B^T P x = 0 \Rightarrow u = -R^{-1}B^T P x$$

代回得Riccati方程。$\square$

**定理 11.7.7**（Riccati方程解的存在性）：若 $(A, B)$ 能控，$(A, Q^{1/2})$ 能观，则存在唯一正定解 $P$。

---

## 11.7.4 最优控制的数值方法

### 11.7.4.1 直接法

**定义 11.7.6**（直接配点法）：将连续问题离散化：

$$\min_{u_0, \ldots, u_N} J_N = \Phi(x_N) + \sum_{k=0}^{N-1} L(x_k, u_k) \Delta t$$

约束：$x_{k+1} = x_k + f(x_k, u_k) \Delta t$

### 11.7.4.2 间接法

**定义 11.7.7**（打靶法）：通过迭代求解两点边值问题：

$$\dot{x} = f(x, u^*(x, \lambda), t)$$
$$\dot{\lambda} = -\frac{\partial H}{\partial x}$$

边界条件：$x(0) = x_0$，$\lambda(t_f) = \frac{\partial \Phi}{\partial x}|_{t_f}$

"""

基于Pontryagin极大值原理和动态规划的实现
"""

from typing import Callable, Tuple, Optional, List
from dataclasses import dataclass
import matplotlib.pyplot as plt
from scipy.integrate import odeint, solve_bvp
from scipy.optimize import minimize
from scipy.linalg import solve_continuous_are

class OptimalControlSolver:
    """
    最优控制问题求解器
    """

    def __init__(self, dynamics: Callable, running_cost: Callable,
                 terminal_cost: Optional[Callable] = None):
        """
        初始化

        Args:
            dynamics: f(x, u, t) 系统动力学
            running_cost: L(x, u, t) 运行代价
            terminal_cost: Φ(x, t) 终端代价 (可选)
        """
        self.f = dynamics
        self.L = running_cost
        self.Phi = terminal_cost if terminal_cost else lambda x, t: 0

    def hamiltonian(self, x: np.ndarray, u: np.ndarray,
                   lambda_: np.ndarray, t: float) -> float:
        """计算Hamiltonian"""
        return self.L(x, u, t) + lambda_.T @ self.f(x, u, t)

    def solve_indirect(self, x0: np.ndarray, tf: float,
                      u_guess: Callable = None,
                      n_grid: int = 100) -> Tuple[np.ndarray, np.ndarray, np.ndarray]:
        """
        使用间接法（打靶法）求解

        Returns:
            (时间数组, 状态轨迹, 控制轨迹)
        """
        n_states = len(x0)
        t = np.linspace(0, tf, n_grid)

        # 初始化控制和协态猜测
        if u_guess is None:
            u_guess = lambda t: np.zeros(1)

        # 简化的梯度下降求解（实际应用需要更复杂的方法）
        u_traj = np.array([u_guess(ti) for ti in t])
        x_traj = self._forward_integrate(x0, u_traj, t)

        return t, x_traj, u_traj

    def _forward_integrate(self, x0: np.ndarray, u_traj: np.ndarray,
                          t: np.ndarray) -> np.ndarray:
        """前向积分"""
        x_traj = [x0]
        x = x0.copy()

        for i in range(len(t) - 1):
            dt = t[i+1] - t[i]
            u = u_traj[i]
            x = x + dt * self.f(x, u, t[i])
            x_traj.append(x.copy())

        return np.array(x_traj)


class LQRSolver:
    """
    线性二次调节器 (LQR) 求解器
    """

    def __init__(self, A: np.ndarray, B: np.ndarray,
                 Q: np.ndarray, R: np.ndarray):
        """
        初始化LQR问题

        Args:
            A: 系统矩阵
            B: 输入矩阵
            Q: 状态权重矩阵
            R: 控制权重矩阵
        """
        self.A = A
        self.B = B
        self.Q = Q
        self.R = R
        self.n = A.shape[0]
        self.m = B.shape[1]

        # 求解Riccati方程
        self.P = self._solve_riccati()

        # 计算最优反馈增益
        self.K = self._compute_gain()

    def _solve_riccati(self) -> np.ndarray:
        """求解代数Riccati方程"""
        try:
            P = solve_continuous_are(self.A, self.B, self.Q, self.R)
            return P
        except:
            # 使用迭代方法
            P = np.eye(self.n)
            for _ in range(1000):
                P_new = self.Q + self.A.T @ P + P @ self.A - \
                       P @ self.B @ np.linalg.inv(self.R) @ self.B.T @ P
                if np.linalg.norm(P_new - P) < 1e-6:
                    break
                P = P_new
            return P

    def _compute_gain(self) -> np.ndarray:
        """计算最优反馈增益 K = R^(-1) B^T P"""
        return np.linalg.inv(self.R) @ self.B.T @ self.P

    def optimal_control(self, x: np.ndarray) -> np.ndarray:
        """计算最优控制 u = -Kx"""
        return -self.K @ x

    def closed_loop_system(self) -> np.ndarray:
        """返回闭环系统矩阵 A_cl = A - BK"""
        return self.A - self.B @ self.K

    def simulate(self, x0: np.ndarray, t_span: Tuple[float, float] = (0, 10),
                n_points: int = 1000) -> Tuple[np.ndarray, np.ndarray, np.ndarray]:
        """
        仿真闭环系统

        Returns:
            (时间数组, 状态轨迹, 控制轨迹)
        """
        t = np.linspace(t_span[0], t_span[1], n_points)

        def closed_loop_dynamics(x, t):
            u = self.optimal_control(x)
            return self.A @ x + self.B @ u

        x_traj = odeint(closed_loop_dynamics, x0, t)
        u_traj = np.array([self.optimal_control(x) for x in x_traj])

        return t, x_traj, u_traj

    def cost_to_go(self, x: np.ndarray) -> float:
        """计算从状态x出发的最优代价 x^T P x"""
        return x.T @ self.P @ x


class DynamicProgramming:
    """
    动态规划求解器
    """

    def __init__(self, state_bounds: Tuple[np.ndarray, np.ndarray],
                 control_bounds: Tuple[np.ndarray, np.ndarray],
                 n_state_grid: int = 50,
                 n_control_grid: int = 20):
        """
        初始化

        Args:
            state_bounds: (x_min, x_max) 状态空间边界
            control_bounds: (u_min, u_max) 控制空间边界
        """
        self.x_min, self.x_max = state_bounds
        self.u_min, self.u_max = control_bounds
        self.n_x = n_state_grid
        self.n_u = n_control_grid

        # 创建网格
        self.state_grid = [np.linspace(self.x_min[i], self.x_max[i], self.n_x)
                          for i in range(len(self.x_min))]
        self.control_grid = [np.linspace(self.u_min[i], self.u_max[i], self.n_u)
                            for i in range(len(self.u_min))]

    def solve(self, dynamics: Callable, cost: Callable,
             terminal_cost: Callable, dt: float, n_steps: int) -> np.ndarray:
        """
        使用值迭代求解

        Returns:
            值函数数组
        """
        # 简化实现：1D情况
        n_states = len(self.state_grid[0])
        V = np.zeros(n_states)

        # 反向递推
        for k in range(n_steps - 1, -1, -1):
            V_new = np.zeros(n_states)

            for i, x in enumerate(self.state_grid[0]):
                # 在每个状态找到最优控制
                min_cost = float('inf')

                for u in self.control_grid[0]:
                    # 计算下一状态
                    x_next = x + dt * dynamics(x, u)

                    # 插值得到V(x_next)
                    if self.x_min[0] <= x_next <= self.x_max[0]:
                        v_next = np.interp(x_next, self.state_grid[0], V)
                    else:
                        v_next = terminal_cost(x_next)

                    # 总代价
                    total_cost = cost(x, u) * dt + v_next

                    if total_cost < min_cost:
                        min_cost = total_cost

                V_new[i] = min_cost

            V = V_new

        return V


def lqr_double_integrator():
    """
    双积分器LQR控制示例
    """
    print("=" * 60)
    print("LQR: Double Integrator Control")
    print("=" * 60)

    # 双积分器: dx1/dt = x2, dx2/dt = u
    A = np.array([[0, 1], [0, 0]])
    B = np.array([[0], [1]])

    # 权重矩阵
    Q = np.array([[1, 0], [0, 0.1]])  # 强调位置
    R = np.array([[0.1]])  # 控制代价

    print(f"\nSystem: Double integrator")
    print(f"A = \n{A}")
    print(f"B = \n{B}")
    print(f"Q = \n{Q}")
    print(f"R = \n{R}")

    # 求解LQR
    lqr = LQRSolver(A, B, Q, R)

    print(f"\nSolution:")
    print(f"P = \n{lqr.P}")
    print(f"K = {lqr.K.flatten()}")

    # 验证闭环稳定性
    A_cl = lqr.closed_loop_system()
    eigenvalues = np.linalg.eigvals(A_cl)
    print(f"\nClosed-loop eigenvalues: {eigenvalues}")
    print(f"System is {'stable' if np.all(np.real(eigenvalues) < 0) else 'unstable'}")

    # 仿真
    x0 = np.array([1.0, 0.0])  # 初始位置偏移

    print(f"  Final position: {x_traj[-1, 0]:.4f}")
    print(f"  Final velocity: {x_traj[-1, 1]:.4f}")
    print(f"  Max control: {np.max(np.abs(u_traj)):.4f}")

    return lqr, t, x_traj, u_traj


def minimum_time_control():
    """
    最短时间控制示例（Bang-Bang控制）
    """
    print("\n" + "=" * 60)
    print("Minimum Time Control (Bang-Bang)")
    print("=" * 60)

    # 双积分器最短时间控制
    # 切换曲线: x1 = -0.5 * x2 * |x2| (对于 |u| <= 1)

    def switching_curve(x2):
        """切换曲线"""
        return -0.5 * x2 * np.abs(x2)

    def optimal_control_bangbang(x1, x2):
        """Bang-Bang最优控制律"""
        s = x1 - switching_curve(x2)
        if s > 0:
            return -1
        elif s < 0:
            return 1
        else:
            return 0

    # 仿真
    def dynamics(state, t):
        x1, x2 = state
        u = optimal_control_bangbang(x1, x2)
        return [x2, u]

    x0 = [2.0, 0.0]  # 从位置2，速度0开始
    t = np.linspace(0, 5, 1000)
    traj = odeint(dynamics, x0, t)

    print(f"\nBang-Bang control from x0 = {x0}:")
    print(f"  Final position: {traj[-1, 0]:.4f}")
    print(f"  Final velocity: {traj[-1, 1]:.4f}")

    # 估计到达时间
    settled = np.where(np.abs(traj[:, 0]) < 0.01)[0]
    if len(settled) > 0:
        t_settle = t[settled[0]]
        print(f"  Settling time: {t_settle:.4f}")

    return t, traj, switching_curve


def trajectory_optimization():
    """
    轨迹优化示例
    """
    print("\n" + "=" * 60)
    print("Trajectory Optimization")
    print("=" * 60)

    # 简单二维轨迹优化
    # 最小化 ∫(u₁² + u₂²)dt，约束: d²x/dt² = u

    # 使用直接配点法（简化）
    N = 50  # 离散点数
    dt = 0.1

    # 初始和终端条件
    x0 = np.array([0, 0])
    xf = np.array([1, 1])

    # 优化变量: [x0, y0, x1, y1, ..., u0_x, u0_y, ...]
    def objective(vars):
        """目标函数: 控制能量"""
        u_start = 2 * N
        u = vars[u_start:].reshape(-1, 2)
        return np.sum(u**2) * dt

    def constraint_dynamics(vars):
        """动力学约束"""
        pos = vars[:2*N].reshape(-1, 2)
        u = vars[2*N:].reshape(-1, 2)

        constraints = []
        for i in range(N-1):
            # x_{i+1} = x_i + dt * v_i + 0.5 * dt^2 * u_i
            # 简化: 只考虑速度
            vel = (pos[i+1] - pos[i]) / dt
            constraints.extend(u[i] - (pos[i+1] - 2*pos[i] + pos[max(0, i-1)])/dt**2)

        return np.array(constraints)

    print(f"\nTrajectory from {x0} to {xf}")
    print(f"  Discretization points: {N}")
    print(f"  Time step: {dt}")
    print(f"  Total time: {N * dt}")

    # 简化的直线轨迹
    t = np.linspace(0, N*dt, N)
    x_traj = np.linspace(x0[0], xf[0], N)
    y_traj = np.linspace(x0[1], xf[1], N)

    # 计算所需控制
    ux = np.zeros(N)
    uy = np.zeros(N)
    for i in range(1, N-1):
        ux[i] = (x_traj[i+1] - 2*x_traj[i] + x_traj[i-1]) / dt**2
        uy[i] = (y_traj[i+1] - 2*y_traj[i] + y_traj[i-1]) / dt**2

    cost = np.sum(ux**2 + uy**2) * dt
    print(f"  Control cost (straight line): {cost:.4f}")

    return t, x_traj, y_traj, ux, uy

    ax4.plot(x1_switch, x2_range, 'r--', linewidth=2, label='Switching curve')

    ax4.scatter([traj_bb[0, 0]], [traj_bb[0, 1]], c='g', s=100, zorder=5)
    ax4.scatter([traj_bb[-1, 0]], [traj_bb[-1, 1]], c='r', s=100, zorder=5)
    ax4.set_xlabel('Position')
    ax4.set_ylabel('Velocity')
    ax4.set_title('Minimum Time Control (Bang-Bang)')
    ax4.legend()
    ax4.grid(True, alpha=0.3)
    ax4.axis('equal')

    # 5. 轨迹优化
    ax5 = plt.subplot(2, 3, 5)
    t_traj, x_traj_opt, y_traj_opt, ux, uy = trajectory_optimization()
    ax5.plot(x_traj_opt, y_traj_opt, 'b-', linewidth=2, marker='o', markersize=3)
    ax5.scatter([x_traj_opt[0]], [y_traj_opt[0]], c='g', s=100, label='Start')
    ax5.scatter([x_traj_opt[-1]], [y_traj_opt[-1]], c='r', s=100, label='Goal')
    ax5.set_xlabel('x')
    ax5.set_ylabel('y')
    ax5.set_title('Optimal Trajectory')
    ax5.legend()
    ax5.grid(True, alpha=0.3)
    ax5.axis('equal')

    # 6. 值函数可视化（简化的LQR值函数）
    ax6 = plt.subplot(2, 3, 6)

    x1_range = np.linspace(-2, 2, 50)
    x2_range = np.linspace(-2, 2, 50)
    X1, X2 = np.meshgrid(x1_range, x2_range)

    V = np.zeros_like(X1)
    for i in range(len(x1_range)):
        for j in range(len(x2_range)):
            x = np.array([X1[i,j], X2[i,j]])
            V[i,j] = lqr.cost_to_go(x)

    contour = ax6.contour(X1, X2, V, levels=15, cmap='viridis')
    plt.colorbar(contour, ax=ax6)
    ax6.set_xlabel('x₁')
    ax6.set_ylabel('x₂')
    ax6.set_title('LQR Value Function V(x)')
    ax6.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('optimal_control.png', dpi=150, bbox_inches='tight')
    plt.show()
