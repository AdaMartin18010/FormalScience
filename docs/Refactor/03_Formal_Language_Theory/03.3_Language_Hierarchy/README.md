# 03.3 语言层次 (Language Hierarchy)

## 📚 概述

语言层次理论研究形式语言的分类和层次结构，以乔姆斯基谱系为核心，建立了语言、文法和自动机之间的对应关系。本目录包含语言层次理论的详细内容，包括各类语言的定义、性质、关系和应用。
