# 并发示例

本目录是 Rust 并发编程示例的预留位置，隶属于 `docs/Refactor/examples/rust` 形式化系统示例库。

## 目录目的

- 展示 Rust 中的线程安全原语（Mutex、RwLock、Arc）
- 演示消息传递通道（mpsc、crossbeam-channel）
- 提供原子操作与无锁数据结构示例
- 对应父目录中的 [`concurrent_patterns.rs`](../concurrent_patterns.rs) 并发模式理论

## 当前状态

本目录目前为占位状态，示例代码将逐步补充。如需立即学习并发内容，请参考父目录的 `concurrent_patterns.rs`。
