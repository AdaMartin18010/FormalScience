# 隐私政策 (Privacy Policy)

## 概述

FormalScience（"我们"、"我们的"或"本项目"）致力于保护您的隐私。本隐私政策说明了我们如何收集、使用、存储和保护您的个人信息。

**生效日期**: 2026年4月11日

**最后更新**: 2026年4月11日

## 数据收集

### 我们收集的信息类型

#### 1. 自动收集的信息

当您使用我们的服务时，我们可能自动收集以下信息：

| 数据类型 | 描述 | 收集目的 |
|----------|------|----------|
| 日志数据 | IP地址、浏览器类型、访问时间 | 服务优化、安全监控 |
| 使用数据 | 功能使用、点击流、性能指标 | 产品改进、用户体验优化 |
| 设备信息 | 操作系统、设备类型、分辨率 | 兼容性优化、故障排除 |
| Cookie数据 | 会话标识、偏好设置 | 功能实现、个性化体验 |

#### 2. 您提供的信息

| 数据类型 | 描述 | 提供方式 |
|----------|------|----------|
| 账户信息 | 用户名、邮箱地址 | 注册账户 |
| 个人资料 | 头像、个人简介 | 自愿填写 |
| 通讯内容 | 邮件、Issue、评论 | 社区参与 |
| 贡献数据 | 代码提交、Pull Request | 开源贡献 |

#### 3. 第三方提供的信息

- GitHub API 提供的公开资料信息
- Gravatar 等头像服务
- 其他公开的开源活动数据

### 我们不收集的信息

我们明确**不收集**以下敏感信息：

- ❌ 密码或密钥（除非您主动分享，但强烈建议不要这样做）
- ❌ 财务信息
- ❌ 政府颁发的身份证件号码
- ❌ 精确的地理位置数据
- ❌ 生物识别数据
- ❌ 种族、民族、宗教等敏感分类数据

## 数据保护

### 安全措施

我们实施以下技术和组织措施保护您的数据：

#### 1. 技术措施

```
┌─────────────────────────────────────────────────────────┐
│                    数据保护架构                          │
├─────────────────────────────────────────────────────────┤
│  传输层: TLS 1.3 加密                                   │
```

- 定期安全培训
- 访问权限最小化原则
- 员工保密协议
- 定期安全审计
- 事件响应计划

#### 3. 数据处理协议

与第三方服务提供商签订数据处理协议 (DPA)，确保：

- 仅按我们的指示处理数据
- 实施适当的安全措施
- 及时报告数据泄露
- 合同终止后删除或返回数据

## 用户权利

根据适用的数据保护法律，您享有以下权利：

### 1. 访问权

您有权获取我们持有的关于您的个人数据的副本。

**如何行使**:

- 发送邮件至 privacy@formalscience.org
- 在 GitHub 上提交隐私请求 Issue

**响应时间**: 30 天内

### 2. 更正权

您有权要求更正不准确的个人数据。

**如何行使**:

- 通过账户设置更新信息
- 联系 privacy@formalscience.org

### 3. 删除权（被遗忘权）

在特定情况下，您有权要求删除您的个人数据。

**例外情况**:

- 法律要求保留
- 行使言论自由
- 法律索赔的建立、行使或辩护

### 4. 限制处理权

您有权要求限制对您的数据的处理。

### 5. 数据可携带权

您有权以结构化、常用和机器可读的格式获取您的数据。

### 6. 反对权

您有权反对某些类型的数据处理，特别是基于合法利益的处理。

### 7. 撤回同意权

如果处理基于您的同意，您可以随时撤回同意。

### 权利行使流程

```
提交请求 → 身份验证 → 处理请求 → 响应 → 申诉（如需要）
```

1. **提交请求**: 通过指定渠道提交
2. **身份验证**: 验证您的身份
3. **处理请求**: 评估请求并采取行动
4. **响应**: 在法定时间内响应
5. **申诉**: 如不满意可向监管机构申诉

## Cookie 和跟踪技术

### 我们使用的 Cookie

| 类型 | 目的 | 示例 |
|------|------|------|
| 必要 | 基本功能 | 会话 Cookie、CSRF 令牌 |
