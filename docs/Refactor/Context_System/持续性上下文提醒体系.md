## 中断恢复指令

如果系统中断，请按以下步骤恢复：

1. **检查当前状态**: 查看本文件中的状态标记
2. **加载上下文**: 读取已保存的分析结果
3. **继续任务**: 从标记的断点继续处理
4. **更新状态**: 完成后更新状态标记

## 紧急处理策略

### 网络中断

- 保存当前状态到本地
- 记录处理位置
- 等待网络恢复后继续

### 处理超时

- 分批处理大型文件
- 优先处理核心内容
- 保持关键上下文

### 内容冲突

- 记录冲突点
- 分析冲突原因
- 制定统一解决方案
