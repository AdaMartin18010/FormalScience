# 行为准则 (Code of Conduct)

## 我们的承诺

作为 FormalScience 项目的成员、贡献者和领导者，我们承诺为每个人营造一个开放、友好、包容和无骚扰的参与环境，无论其年龄、体型、可见或不可见的残疾、种族、性特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

我们承诺以有助于建立开放、友好、多元、包容和健康社区的方式行事和互动。

### 我们的价值观

- **尊重**: 尊重每一位社区成员的观点和经验
- **包容**: 欢迎来自不同背景的新成员
- **协作**: 通过建设性合作解决问题
- **透明**: 在决策和沟通中保持开放
- **卓越**: 追求技术和社区建设的最高标准

## 行为标准

### 积极行为的示例

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心
- 帮助新成员融入社区
- 分享知识和经验
- 认可他人的贡献

### 不可接受的行为

以下行为被视为不可接受：

#### 骚扰行为

- 使用性化的语言或图像
- 不受欢迎的性注意或挑逗
- 恶意攻击、侮辱/贬损评论
- 人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如物理或电子邮件地址）
- 其他在专业环境中可能被合理认为不适当的行为

#### 歧视行为

- 基于个人特征的歧视
- 种族主义、性别歧视、仇外言论
- 对残疾人士的不当行为
- 基于宗教或信仰的排斥

#### 破坏性行为

- 故意传播错误信息
- 恶意破坏项目资源
- 滥用报告机制
- 骚扰项目维护者
- 故意制造冲突或分裂

#### 技术不当行为

- 故意引入恶意代码
- 滥用项目基础设施
- 违反安全政策
- 未经授权访问系统

## 执行责任

### 社区领导者职责

社区领导者有责任：

1. **澄清标准**: 明确并公布可接受行为的标准
2. **公正执行**: 公正、一致地执行行为准则
3. **及时响应**: 对违规行为报告做出及时响应
4. **保护举报者**: 保护举报违规行为的个人免受报复
5. **持续改进**: 定期审查和更新行为准则

### 执行流程

#### 报告违规

如果您观察到或经历了不可接受的行为，请通过以下方式报告：

| 方式 | 联系方式 | 适用情况 |
|------|----------|----------|
| 邮箱 | conduct@formalscience.org | 一般违规报告 |
| GitHub | 私信项目维护者 | 涉及 GitHub 平台的问题 |
| 匿名表单 | [匿名报告](https://formalscience.org/report) | 需要匿名的情况 |

#### 报告内容

请尽量提供以下信息：

- 事件发生的时间和地点
- 涉及的人员（如知道）
- 违规行为的描述
- 任何相关证据（截图、链接等）
- 其他证人（如有）

#### 处理流程

```
收到报告 → 初步评估 → 调查 → 决定 → 执行 → 跟进
```

1. **初步评估** (24小时内)
   - 确认收到报告
   - 评估紧急程度
   - 确定是否需要立即采取行动

2. **调查** (3-7天)
   - 收集证据
   - 与相关方沟通
   - 咨询其他维护者（如需要）

3. **决定**
   - 基于证据做出决定
   - 确定适当的处理措施

4. **执行**
   - 实施处理措施
   - 通知相关方

5. **跟进**
   - 监控后续情况
   - 记录案例（匿名化）
   - 评估政策改进需求

### 处理措施

根据违规的严重程度和频率，社区领导者可能采取以下措施：

#### 轻度违规

- **口头警告**: 私下提醒注意行为准则
- **书面警告**: 正式的书面警告
- **临时限制**: 暂时限制某些权限（如 7 天）

#### 中度违规

- **长期限制**: 延长权限限制（如 30 天）
- **要求道歉**: 要求向受影响方公开道歉
- **强制培训**: 要求完成相关培训

#### 严重违规

- **永久封禁**: 永久禁止参与项目
- **公开声明**: 发布关于违规行为的公开声明
- **法律行动**: 在必要时采取法律行动

### 申诉流程

如果您对处理决定有异议，可以通过以下方式申诉：

1. **提交申诉**: 发送邮件至 conduct-appeal@formalscience.org
2. **说明理由**: 详细说明申诉理由
3. **审核委员会**: 由未参与原始决定的维护者组成委员会审核
4. **最终决定**: 委员会的决定为最终决定

## 适用范围

本行为准则适用于所有 FormalScience 项目空间，包括：

### 在线空间

- GitHub 仓库和 Issues
- 讨论区和论坛
- 官方社交媒体账号
- 项目网站和文档
- 在线会议和网络研讨会
- 官方聊天频道（Discord、Slack 等）
- 邮件列表

### 线下活动

- 项目组织的会议和活动
- 项目成员的演讲和展示
- 社区聚会和研讨会
- 黑客马拉松和编程活动
- 任何代表项目的情况

### 个人空间

本行为准则也适用于：

- 在公共场合代表项目或社区时
- 使用项目官方电子邮件地址时
- 通过官方社交媒体账号发布时
- 作为指定代表参加线上或线下活动时
