## 3.3.1 引言

高阶归纳类型(Higher Inductive Types, HITs)是HoTT的核心创新，允许在定义类型时不仅指定**点构造子**（元素），还指定**路径构造子**（相等关系）。这使得我们可以在类型论中直接表达拓扑空间。

**传统归纳类型 vs HITs**：

| 特征 | 归纳类型 | HITs |
|------|---------|------|
| 点构造子 | ✓ | ✓ |
| 路径构造子 | ✗ | ✓ |
| 高阶路径构造子 | ✗ | ✓（可选）|

> **引用**: 恒等类型见 [03.2_恒等类型.md](./03.2_恒等类型.md)，HoTT基础见 [03.1_同伦基础.md](./03.1_同伦基础.md)。

## 3.3.2 HIT的定义

### 3.3.2.1 构造子

**定义 3.3.1 (HIT)** 高阶归纳类型由以下构造子定义：

1. **0-构造子**：引入该类型的元素（点）
2. **1-构造子**：引入该类型的路径（相等证明）
3. **n-构造子**：引入n阶路径（路径之间的相等）

**一般形式**：

```
inductive T where
  | pt₁ : T              -- 点构造子
  | pt₂ : A → T          -- 带参数的点构造子
  | path₁ : pt₁ = pt₂    -- 路径构造子
  | path₂ : (x : A) → f x = g x  -- 依赖路径构造子
  | ...
```

### 3.3.2.2 归纳原理

**定义 3.3.2 (HIT的归纳原理)** 要定义函数 $f : T \rightarrow P$，需要指定：

- 每个点构造子在 $P$ 中的像
- 每个路径构造子在 $P$ 中映射为适当的路径
- 每个高阶路径构造子映射为适当的高阶路径

## 3.3.3 圆

### 3.3.3.1 圆的定义

**定义 3.3.3 (圆 $S^1$)**

$$
S^1 := \begin{cases}
\text{base} : S^1 \\
\text{loop} : \text{base} =_{S^1} \text{base}
\end{cases}
$$

```mermaid
graph LR
    subgraph S¹圆
    B["base"]
    end

    B -.->|"loop"| B
```

**几何解释**：

- $\text{base}$：圆上的一个基点
- $\text{loop}$：从基点出发绕圆一周的环路

### 3.3.3.2 圆的归纳

**定理 3.3.1 (圆的归纳原理)** 给定：

- $P : S^1 \rightarrow \mathcal{U}$
- $b : P(\text{base})$
- $\ell : \text{transport}^{P}(\text{loop}, b) = b$

存在 $f : \Pi (x:S^1). P(x)$ 使得：

- $f(\text{base}) \equiv b$
- $\text{apd}_f(\text{loop}) = \ell$

**定理 3.3.2 ($S^1$的基本群)**

$$\Omega(S^1, \text{base}) \simeq \mathbb{Z}$$

即圆的环路空间等价于整数群（由环绕数分类）。

## 3.3.4 区间

**定义 3.3.4 (区间 $I$)**

$$
I := \begin{cases}
0_I : I \\
1_I : I \\
\text{seg} : 0_I =_I 1_I
\end{cases}
$$

**定理 3.3.3 (区间的可缩性)** 区间 $I$ 是可缩类型（收缩到任一点）。

**函数外延性**：

使用区间可以证明函数外延性：

$$\text{funext} : (\Pi (x:A). f(x) = g(x)) \rightarrow (f = g)$$

---

## 3.3.5 球面

**定义 3.3.5 (2维球面 $S^2$)**

$$
S^2 := \begin{cases}
\text{base} : S^2 \\
\text{surf} : \text{refl}_{\text{base}} =_{\text{base} = \text{base}} \text{refl}_{\text{base}}
\end{cases}
$$

**几何解释**：

- $\text{base}$：球面上的基点
- $\text{surf}$：将自反路径"填充"成二维面的2-路径

**一般球面 $S^n$**：

- $S^0 = \mathbf{2}$（两个点）
- $S^{n+1}$ 的构造包含一个(n+1)-路径

## 3.3.6 推出与怀特和

**定义 3.3.6 (推出, Pushout)** 给定 $f : C \rightarrow A$ 和 $g : C \rightarrow B$：

$$
A \sqcup_C B := \begin{cases}
\text{inl} : A \rightarrow A \sqcup_C B \\
\text{inr} : B \rightarrow A \sqcup_C B \\
\text{glue} : \Pi (c:C). \text{inl}(f(c)) = \text{inr}(g(c))
\end{cases}
$$

**怀特和 (Wedge Sum)**：

$$A \vee B := A \sqcup_{\mathbf{1}} B$$

**smash积**：

$$A \wedge B := (A \times B) / (A \vee B)$$

---

## 3.3.7 截断

**定义 3.3.7 (n-截断, $\|A\|_n$)** 将类型 $A$ "截断"为n-截断类型：

$$
\|A\|_{-1} := \begin{cases}
|a| : A \rightarrow \|A\|_{-1} \\
\text{squash} : \Pi (x,y:\|A\|_{-1}). x = y
\end{cases}
$$

**命题截断**（-1-截断）：

- 将类型变为命题（最多一个元素）
- $|a|$：将元素嵌入截断类型
- $\text{squash}$：证明所有元素相等

**集合截断**（0-截断）：

$$
\|A\|_0 := \begin{cases}
|a| : A \rightarrow \|A\|_0 \\
\text{trunc} : \Pi (x,y:\|A\|_0). \Pi (p,q:x=y). p = q
\end{cases}
$$

## 3.3.8 形式化证明

### Lean 4：HIT的形式化

```lean4
-- 圆 (使用Quotient模拟HIT)
inductive S1Pre where
  | base

def S1Loop : S1Pre → S1Pre → Prop
  | .base, .base => True
  | _, _ => False

def S1 := Quotient (⟨S1Loop, sorry, sorry, sorry⟩ : Setoid S1Pre)

def S1.base : S1 := Quotient.mk _ S1Pre.base

-- loop路径（需要使用商类型的性质）
def S1.loop : Eq S1.base S1.base :=
  -- 在Lean中需要额外构造
  sorry

-- 圆的归纳原理
def S1.rec {P : S1 → Type}
  (b : P S1.base)
  (ℓ : HEq (cast (by rw [S1.loop]) b) b)
  (x : S1) : P x :=
  Quotient.rec (fun x => match x with | .base => b) (fun _ _ _ => sorry) x

-- 区间
def Interval := Unit

-- 命题截断 (使用内置)
inductive Trunc (A : Type) : Type where
  | mk : A → Trunc A
  | eq : ∀ (x y : Trunc A), x = y

-- 集合截断
def SetTrunc (A : Type) : Type :=
  Quotient (⟨fun x y => Nonempty (x = y),
             sorry, sorry, sorry⟩ : Setoid A)

-- 悬垂 (Suspension)
inductive Susp (A : Type) where
  | north
  | south
  | merid : A → Eq north south

-- S^n via iterated suspension
def Sphere : Nat → Type
  | 0 => Bool
  | n + 1 => Susp (Sphere n)

-- S^1 is equivalent to Sphere 1
def S1_equiv_Sphere1 : S1 ≃ Susp Bool :=
  sorry
```

### HoTT风格的形式化

```lean4
-- 使用高阶归纳类型的定义风格
namespace HIT

-- 圆的完整定义
codef S¹ : Type where
  | base
  | loop : base = base

codef S¹.rec (P : S¹ → Type)
  (b : P base)
  (ℓ : PathOver P loop b b)
  : (x : S¹) → P x
  | base => b
  | loop => ℓ

-- 区间
codef Interval : Type where
  | zero
  | one
  | seg : zero = one

-- 推出
codef Pushout {A B C : Type} (f : C → A) (g : C → B) : Type where
  | inl (a : A)
  | inr (b : B)
  | glue (c : C) : inl (f c) = inr (g c)

-- 命题截断
codef ∥_∥₋₁ (A : Type) : Type where
  | ∣_∣ (a : A)
  | trunc (x y : ∥_∥₋₁ A) : x = y

end HIT
```

## 3.3.9 总结

**常见HITs**：

| HIT | 构造子 | 用途 |
|-----|--------|------|
| **$S^1$** | base, loop | 基本群、环路空间 |
| **$I$** | $0_I, 1_I$, seg | 函数外延性 |
| **$\|A\|_{-1}$** | $\|-\|$, squash | 命题逻辑 |
| **$\|A\|_0$** | $\|-\|$, trunc | 集合论 |
| **推出** | inl, inr, glue | 同伦推出 |

**HIT的核心思想**：

- 类型可以有非平凡的相等结构
- 相等本身可以作为构造对象
- 几何直观指导类型定义
