# 02 形式语言模块索引

> **模块**: 形式语言
> **描述**: 研究形式化语言的理论基础，从自动机到类型论的完整谱系
> **文件数**: 32
> **状态**: 📝 内容填充中

---

## 📋 内容概述

本模块涵盖形式化语言的完整理论谱系，从经典的自动机理论和形式语法，到现代的类型论、同伦类型论和范畴论。这些理论为编程语言设计、形式验证和数学基础提供了严格的数学框架。

### 核心主题

- **形式语言基础**: Chomsky层次、自动机、λ演算
- **类型论**: 简单类型、多态类型、依赖类型
- **同伦类型论**: 同伦基础、恒等类型、高阶归纳类型
- **范畴论**: 范畴基础、极限、伴随、在PL中的应用
