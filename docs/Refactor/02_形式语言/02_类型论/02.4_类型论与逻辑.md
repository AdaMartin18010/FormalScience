## 1. Curry-Howard 同构

### 1.1 历史背景

**Curry-Howard 同构**（也称为 **Curry-Howard 对应**、**命题即类型**）是由 Haskell Curry (1934)、William Alvin Howard (1969) 等人逐步发现的一个深刻对应关系。它揭示了**逻辑证明**与**程序计算**在结构上的同一性，为现代依赖类型证明助手（如 Lean、Coq、Agda）奠定了理论基础。

> **核心洞见**: 证明一个命题 $P$ 等价于构造一个类型 $P$ 的程序（项）。

### 1.2 形式化对应

**定义 1.2.1** (Curry-Howard 对应). 直觉主义命题逻辑的自然演绎系统 $  ext{NJ}$ 与简单类型λ演算 $\lambda^\to$ 之间存在结构保持的一一对应：

| 逻辑 ($\text{NJ}$) | 类型论 ($\lambda^\to$) | 说明 |
|:---:|:---|:---|
| **命题** $P$ | **类型** $P$ | 命题的可证性 ↔ 类型的可居住性 |
| **证明** $p$ | **项/程序** $t : P$ | 证明 $P$ ↔ 构造类型 $P$ 的项 |
| **蕴涵** $P \to Q$ | **函数类型** $P \to Q$ | 从 $P$ 的证明得到 $Q$ 的证明 |
| **合取** $P \land Q$ | **积类型** $P \times Q$ | 配对构造子 |
| **析取** $P \lor Q$ | **和类型** $P + Q$ | 左/右注入 |
| **真** $\top$ | **单位类型** $\mathbf{1}$ | 单元素类型 |
| **假** $\bot$ | **空类型** $\mathbf{0}$ | 无构造子的类型 |
| **否定** $\neg P$ | **函数类型** $P \to \bot$ | 从 $P$ 导出矛盾 |

**定理 1.2.2** (同构的完备性).
$$\Gamma \vdash_{\text{NJ}} P \quad \Longleftrightarrow \quad \exists t. \Gamma \vdash_{\lambda^\to} t : P$$

即：在自然演绎中，假设集 $\Gamma$ 能证明命题 $P$ 当且仅当在简单类型λ演算中存在项 $t$ 使得在上下文 $\Gamma$ 下 $t$ 具有类型 $P$。

### 1.3 归约与证明规范化

**定理 1.3.1** (证明规范化 ↔ β归约). Curry-Howard 对应下：

| 证明论 | 计算论 |
|:---|:---|
| 消去后紧跟引入的**截断** (cut) | β-可约子项 (β-redex) |
| 证明规范化 (cut-elimination) | β归约 $t \to_\beta t'$ |
| 规范证明 (normal proof) | β范式 (normal form) |
| 规范证明的唯一性 | Church-Rosser 定理 |

**推论 1.3.2**. 类型检查对应证明验证；程序求值对应证明简化。

## 2. 自然演绎与λ项的精确对应

### 2.1 蕴涵规则

**引入规则 ($\to I$)**:

$$
\frac{\Gamma, x : P \vdash t : Q}{\Gamma \vdash \lambda x : P.\, t : P \to Q}
$$

**消去规则 ($\to E$)**:

$$
\frac{\Gamma \vdash f : P \to Q \quad \Gamma \vdash a : P}{\Gamma \vdash f\, a : Q}
$$

**对应**: 蕴涵引入 = λ抽象；蕴涵消去 = 函数应用。

### 2.2 合取规则

**引入规则 ($\land I$)**:

$$
\frac{\Gamma \vdash p : P \quad \Gamma \vdash q : Q}{\Gamma \vdash \langle p, q \rangle : P \times Q}
$$

**消去规则 ($\land E_1$, $\land E_2$)**:

$$
\frac{\Gamma \vdash r : P \times Q}{\Gamma \vdash \pi_1(r) : P}
\quad
\frac{\Gamma \vdash r : P \times Q}{\Gamma \vdash \pi_2(r) : Q}
$$

### 2.3 析取规则

**引入规则 ($\lor I_1$, $\lor I_2$)**:

$$
\frac{\Gamma \vdash p : P}{\Gamma \vdash \text{inl}(p) : P + Q}
\quad
\frac{\Gamma \vdash q : Q}{\Gamma \vdash \text{inr}(q) : P + Q}
$$

**消去规则 ($\lor E$)**:

$$
\frac{\Gamma \vdash s : P + Q \quad \Gamma, x : P \vdash t : R \quad \Gamma, y : Q \vdash u : R}{\Gamma \vdash \text{case}(s, \lambda x.t, \lambda y.u) : R}
$$

### 2.4 常见命题的证明项构造

| 命题 | 证明项 (λ表达式) | 类型 |
|:---|:---|:---|
| $P \to P$ | $\lambda x : P.\, x$ | 恒等函数 |
| $P \to Q \to P$ | $\lambda x : P.\, \lambda y : Q.\, x$ | 投影/常函数 $K$ |
| $(P \to Q \to R) \to (P \to Q) \to P \to R$ | $\lambda f.\, \lambda g.\, \lambda x.\, f\, x\, (g\, x)$ | $S$组合子 |
| $P \land Q \to Q \land P$ | $\lambda r.\, \langle \pi_2(r), \pi_1(r) \rangle$ | 交换配对 |
| $P \lor Q \to Q \lor P$ | $\lambda s.\, \text{case}(s, \text{inr}, \text{inl})$ | 交换和 |

## 3. 命题即类型：Lean 4 形式化

```lean4
-- ========================================
-- Curry-Howard 对应的形式化演示 (Lean 4)
-- ========================================

-- 单位类型 True 对应逻辑真 ⊤
inductive MyTrue : Prop where
  | tt : MyTrue

-- 空类型 False 对应逻辑假 ⊥
-- 无构造子，因此无法构造该类型的值
inductive MyFalse : Prop

-- 否定定义为蕴涵到假
def MyNot (P : Prop) : Prop := P → MyFalse

-- ----------------------------------------
-- 1. 恒等证明：P → P
-- 对应恒等函数 λx.x
-- ----------------------------------------
theorem id_proof (P : Prop) : P → P :=
  λ (p : P) => p

-- ----------------------------------------
-- 2. K 组合子：P → Q → P
-- 对应 λx.λy.x
-- ----------------------------------------
theorem K_proof (P Q : Prop) : P → Q → P :=
  λ (p : P) => λ (_ : Q) => p

-- ----------------------------------------
-- 3. 合取交换律：P ∧ Q → Q ∧ P
-- 对应交换序对
-- ----------------------------------------
theorem and_comm {P Q : Prop} : P ∧ Q → Q ∧ P :=
  λ h => ⟨h.2, h.1⟩

-- ----------------------------------------
-- 4. 析取交换律：P ∨ Q → Q ∨ P
-- 对应 case 分析
-- ----------------------------------------
theorem or_comm {P Q : Prop} : P ∨ Q → Q ∨ P :=
  λ h => match h with
    | .inl p => .inr p
    | .inr q => .inl q

-- ----------------------------------------
-- 5. 传递性：(P → Q) → (Q → R) → (P → R)
-- 对应函数复合
-- ----------------------------------------
theorem compose {P Q R : Prop} : (P → Q) → (Q → R) → (P → R) :=
  λ f => λ g => λ p => g (f p)

-- ----------------------------------------
-- 6. 否定与蕴涵的关系：¬P ↔ (P → False)
-- ----------------------------------------
theorem not_iff_implies_false {P : Prop} : ¬P ↔ (P → False) := by
  -- 在 Lean 中，¬P 就是 P → False 的语法糖
  -- 因此这是自反的
  exact Iff.rfl
```

## 4. 量词的对应

### 4.1 全称量词与 Π 类型

**定理 4.1.1** (全称量词 = Π 类型). 在依赖类型论中：

$$\forall x : A.\, P(x) \quad \longleftrightarrow \quad \Pi(x : A).\, P(x)$$

- **引入规则** ($\forall I$): 假设 $x : A$，构造 $p : P(x)$，则得到 $\lambda x : A.\, p : \forall x : A.\, P(x)$
- **消去规则** ($\forall E$): 给定 $p : \forall x : A.\, P(x)$ 和 $a : A$，得到 $p\, a : P(a)$

**对应**: 全称量词的证明是一个**依赖函数**，它接受任意 $x : A$ 并返回 $P(x)$ 的证明。

```lean4
-- ∀x:A, P(x) 的证明是一个函数
theorem forall_is_function {A : Type} {P : A → Prop}
    (h : ∀ x : A, P x) (a : A) : P a :=
  h a  -- 函数应用 = 全称消去
```

### 4.2 存在量词与 Σ 类型

**定理 4.2.1** (存在量词 = Σ 类型). 在依赖类型论中：

$$\exists x : A.\, P(x) \quad \longleftrightarrow \quad \Sigma(x : A).\, P(x)$$

- **引入规则** ($\exists I$): 给出证据 $a : A$ 和证明 $p : P(a)$，构造 $\langle a, p \rangle : \exists x : A.\, P(x)$
- **消去规则** ($\exists E$): 从 $\Sigma(x : A).\, P(x)$ 中提取第一投影（证据）和第二投影（证明）

```lean4
-- ∃引入：构造依赖序对
theorem exists_intro {A : Type} {P : A → Prop}
    (a : A) (h : P a) : ∃ x, P x :=
  ⟨a, h⟩

-- ∃消去：模式匹配提取证据和证明
theorem exists_elim {A : Type} {P : A → Prop} {Q : Prop}
    (h : ∃ x, P x) (f : ∀ x, P x → Q) : Q :=
  match h with
  | ⟨a, ha⟩ => f a ha
```

## 5. 构造主义逻辑与经典逻辑

### 5.1 直觉主义逻辑的基本原则

**定义 5.1.1** (直觉主义逻辑, Brouwer-Heyting). 直觉主义逻辑拒绝以下经典逻辑原则：

1. **排中律** (Law of Excluded Middle): $P \lor \neg P$
2. **双重否定消去** (Double Negation Elimination): $\neg\neg P \to P$
3. **反证法** (Proof by Contradiction): $(\neg P \to \bot) \to P$

**哲学基础**: 要证明 $P \lor Q$，必须**构造性地**给出 $P$ 的证明或 $Q$ 的证明。要证明 $\exists x. P(x)$，必须**显式构造**满足条件的 $x$。

### 5.2 直觉主义中可证的命题

**定理 5.2.1**. 以下命题在直觉主义逻辑中可证：

- $P \to \neg\neg P$
- $\neg P \leftrightarrow (P \to \bot)$
- $(P \to Q) \to (\neg Q \to \neg P)$ （逆否命题）
- $\neg(P \land \neg P)$ （矛盾律）

```lean4
-- P → ¬¬P
theorem dn_intro (P : Prop) : P → ¬¬P :=
  λ p np => np p

-- (P → Q) → (¬Q → ¬P)
theorem contrapositive {P Q : Prop} : (P → Q) → (¬Q → ¬P) :=
  λ h nq p => nq (h p)
```

### 5.3 经典逻辑的嵌入：双重否定翻译

**定理 5.3.1** (Gödel-Gentzen 双重否定翻译). 任何经典逻辑命题 $P$ 可翻译为直觉主义逻辑命题 $P^*$，使得：

$$P \text{ 在经典逻辑中可证} \quad \Longleftrightarrow \quad P^* \text{ 在直觉主义逻辑中可证}$$

**翻译规则**:

| 原始命题 | 翻译 $P^*$ |
|:---|:---|
| 原子命题 $A$ | $\neg\neg A$ |
| $\top$ | $\top$ |
| $\bot$ | $\bot$ |
| $A \land B$ | $A^* \land B^*$ |
| $A \lor B$ | $\neg(\neg A^* \land \neg B^*)$ |
| $A \to B$ | $A^* \to B^*$ |
| $\forall x. A$ | $\forall x. A^*$ |
| $\exists x. A$ | $\neg\forall x. \neg A^*$ |

**推论 5.3.2**. 经典一阶逻辑在直觉主义一阶逻辑中是**一致的相对化**。

```lean4
-- 排中律的双重否定翻译在直觉主义中可证
theorem dn_em (P : Prop) : ¬¬(P ∨ ¬P) :=
  λ h => h (.inr λ p => h (.inl p))

-- 双重否定消去的翻译：¬¬(¬¬P → P)
-- 注意：这不是 ¬¬P → P 本身
theorem dn_dne_translation (P : Prop) : ¬¬(¬¬P → P) := by
  intro h
  apply h
  intro hnnp
  -- 在直觉主义中，¬¬P → P 本身不可证
  -- 但 ¬¬(¬¬P → P) 可证：从矛盾可导出任意命题（ex falso）
  have hnp : ¬P := λ p => h (λ _ => p)
  exact False.elim (hnnp hnp)
```

### 5.4 构造性与计算性

在 Curry-Howard 框架下，**构造性证明**对应**可计算程序**。这意味着：

- 一个 $\exists x. P(x)$ 的证明项可以被**执行**以提取出具体的见证 $x$
- 一个 $P \lor Q$ 的证明项包含信息告诉我们哪一边是成立的
- 经典逻辑中的不可计算原则（如选择公理、排中律）对应**不可计算的项**

```lean4
-- 从存在证明中提取见证（需要构造性证明）
def extractWitness {A : Type} {P : A → Prop}
    (h : ∃ x, P x) : A :=
  Classical.choose h  -- 非构造性提取（使用选择公理）

-- 注意：在直觉主义类型论中，这样的非构造性提取被禁止或需要显式标记
```

## 6. 线性逻辑与资源敏感类型

### 6.1 线性逻辑的基本 connectives

**定义 6.1.1** (Girard, 1987). 线性逻辑区分**可复用资源**与**一次性资源**：

| 线性逻辑 | 直觉 | 类型对应 |
|:---|:---|:---|
| $A \otimes B$ | 同时拥有 $A$ 和 $B$ | 积类型（线性） |
| $A \multimap B$ | 消耗 $A$ 一次得到 $B$ | 线性函数类型 |
| $A \& B$ | 选择拥有 $A$ 或 $B$ | 加法积（&-类型） |
| $A \oplus B$ | 要么是 $A$ 要么是 $B$ | 和类型 |
| $!A$ | $A$ 可任意复制/丢弃 | 指数模态（comonad） |

**定理 6.1.2**. 经典蕴涵 $A \to B$ 可在线性逻辑中编码为：
$$A \to B \;\equiv\; !A \multimap B$$

即：经典逻辑中的假设可任意使用，等价于线性逻辑中经过 `!` 包装后的资源。

### 6.2 资源敏感的类型系统

现代编程语言中的**所有权类型**（如 Rust 的借用检查器）和**线性类型**（如 Haskell 的 `LinearTypes` 扩展）均可视为线性逻辑在工程中的实现：

```rust
// Rust: 线性/所有权类型的体现
fn consume(v: Vec<i32>) -> usize {
    v.len()  // v 被移动(消耗)，不能再次使用
}

fn borrow(v: &Vec<i32>) -> usize {
    v.len()  // v 被线性借用，所有权不变
}
```
