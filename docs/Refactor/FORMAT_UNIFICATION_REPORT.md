## 📝 格式统一标准

### 1. 标题格式

- 一级标题 `#` - 文档标题
- 二级标题 `##` - 主要章节
- 三级标题 `###` - 子章节
- 四级标题 `####` - 小节

### 2. 数学公式格式

- 行内公式：`$...$`
- 块级公式：`$$...$$` 或 `\[...\]`
- 定理环境统一使用：

```markdown
**定理 X.X** (定理名称)
> 定理内容

*证明*：
证明步骤... ∎
```

### 3. 代码块格式

- 使用 ```language 标记
- 主要语言：lean4, rust, python, haskell, go, java, cpp, c

### 4. 表格格式

- 使用标准Markdown表格
- 对齐方式：表头居中对齐，内容左对齐

### 5. 列表格式

- 有序列表：1. 2. 3.
- 无序列表：- 或 *

### 6. 引用和链接

- 内部链接：`文本`
- 外部链接：`文本`
- 引用标记：`[^1]`

### 7. 特殊标记

- 完成状态：✅ 🔄 ⏸️
- 优先级：🔴 🟠 🟡 🟢
- 注意：⚠️ 💡
