## 主题分类与说明

计算机架构理论是研究计算机系统的概念结构和操作行为的学科。它关注的是程序员所能看到的系统属性，即**指令集架构 (Instruction Set Architecture, ISA)**，以及如何通过**微架构 (Microarchitecture)** 的设计来高效地实现ISA，同时管理好**内存**和**并行计算**等关键资源。
