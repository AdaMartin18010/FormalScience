## 10.5.1.1 引言

**量子信息论**将经典信息论的概念推广到量子系统。
量子熵（Quantum Entropy）是这一推广的核心，由冯·诺依曼（John von Neumann）于1927年在统计力学背景下首次提出。
**von Neumann熵**是量子态不确定性的基本度量，在量子通信、量子计算和量子统计力学中具有核心地位。

## 10.5.1.2 量子态与密度算子

### 密度算子

**定义 10.5.1.1（密度算子）**：

量子系统的**密度算子**（或密度矩阵）$\rho$ 是满足以下条件的算子：

1. **厄米性**：$\rho = \rho^\dagger$
2. **半正定性**：$\rho \geq 0$
3. **迹为1**：$\text{Tr}(\rho) = 1$

**纯态**：若 $\rho = |\psi\rangle\langle\psi|$，称为纯态。

**混合态**：若 $\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|$，$p_i > 0$，$\sum p_i = 1$，称为混合态。

## 10.5.1.3 von Neumann熵

### 定义 10.5.1.2（von Neumann熵）

密度算子 $\rho$ 的**von Neumann熵**定义为：

$$S(\rho) = -\text{Tr}(\rho \log \rho)$$

**谱分解形式**：

若 $\rho = \sum_i \lambda_i |i\rangle\langle i|$，其中 $\lambda_i$ 是特征值，则：
$$S(\rho) = -\sum_i \lambda_i \log \lambda_i = H(\{\lambda_i\})$$

即von Neumann熵等于密度算子特征值的香农熵。

### 约定

- $0 \log 0 = 0$
- 对数的底为2时，单位为**量子比特**（qubits）
- 对数的底为 $e$ 时，单位为**量子奈特**（nats）

## 10.5.1.4 量子熵的性质

### 定理 10.5.1.1（基本性质）

1. **非负性**：$S(\rho) \geq 0$，等号当且仅当 $\rho$ 是纯态
2. **有界性**：$S(\rho) \leq \log d$，等号当且仅当 $\rho = I/d$（最大混合态）
3. **酉不变性**：$S(U\rho U^\dagger) = S(\rho)$，对任意酉算子 $U$
4. **凹性**：$S(\sum_i p_i \rho_i) \geq \sum_i p_i S(\rho_i)$

### 定理 10.5.1.2（熵的次可加性）

对于复合系统 $AB$：
$$S(\rho_{AB}) \leq S(\rho_A) + S(\rho_B)$$

等号成立当且仅当 $\rho_{AB} = \rho_A \otimes \rho_B$（乘积态）。

### 定理 10.5.1.3（Araki-Lieb三角不等式）

$$|S(\rho_A) - S(\rho_B)| \leq S(\rho_{AB})$$

## 10.5.1.5 量子相对熵

### 定义 10.5.1.3（量子相对熵）

两个密度算子 $\rho$ 和 $\sigma$ 之间的**量子相对熵**定义为：

$$D(\rho \| \sigma) = \text{Tr}(\rho \log \rho) - \text{Tr}(\rho \log \sigma)$$

约定：若 $\text{supp}(\rho) \not\subseteq \text{supp}(\sigma)$，则 $D(\rho \| \sigma) = +\infty$。

### 定理 10.5.1.4（Klein不等式）

$$D(\rho \| \sigma) \geq 0$$

等号成立当且仅当 $\rho = \sigma$。

### 定理 10.5.1.5（单調性）

对于任意完全正定保迹映射（量子信道）$\mathcal{N}$：
$$D(\mathcal{N}(\rho) \| \mathcal{N}(\sigma)) \leq D(\rho \| \sigma)$$

## 10.5.1.6 量子条件熵与互信息

### 量子条件熵

**定义 10.5.1.4**：
$$S(A|B) = S(\rho_{AB}) - S(\rho_B)$$

**注意**：量子条件熵**可以为负**！这是量子信息论与经典信息论的重要区别。

### 量子互信息

**定义 10.5.1.5**：
$$I(A:B) = S(\rho_A) + S(\rho_B) - S(\rho_{AB})$$

**性质**：$I(A:B) \geq 0$（由次可加性保证）

### 纠缠与条件熵

**定理 10.5.1.6**：若 $\rho_{AB}$ 是可分态（非纠缠），则 $S(A|B) \geq 0$。

**推论**：$S(A|B) < 0$ 蕴含纠缠存在。

## 10.5.1.7 代码实现

### Python 实现

```python
import numpy as np
from typing import Tuple, List
import scipy.linalg as la

def von_neumann_entropy(rho: np.ndarray, base: float = 2) -> float:
    """
    计算von Neumann熵 S(ρ) = -Tr(ρ log ρ)

    Args:
        rho: 密度矩阵
        base: 对数底（默认2）

    Returns:
        von Neumann熵
    """
    # 计算特征值
    eigenvalues = la.eigvalsh(rho)

    # 计算熵（忽略零特征值）
    entropy = 0.0
    for lam in eigenvalues:
        if lam > 1e-12:  # 避免log(0)
            entropy -= lam * np.log(lam) / np.log(base)

    return entropy

def quantum_relative_entropy(rho: np.ndarray, sigma: np.ndarray,
                             base: float = 2) -> float:
    """
    计算量子相对熵 D(ρ||σ) = Tr(ρ log ρ) - Tr(ρ log σ)
    """
    # 检查支撑集
    rho_eig = la.eigh(rho)
    sigma_eig = la.eigh(sigma)

    # 如果ρ的支撑集不在σ的支撑集内，返回无穷
    rho_support = set(i for i, val in enumerate(rho_eig[0]) if val > 1e-12)
    sigma_support = set(i for i, val in enumerate(sigma_eig[0]) if val > 1e-12)

    if not rho_support.issubset(sigma_support):
        return float('inf')

    # 计算 D(ρ||σ)
    log_rho = la.logm(rho) / np.log(base)
    log_sigma = la.logm(sigma) / np.log(base)

    D = np.trace(rho @ (log_rho - log_sigma))
    return np.real(D)

def partial_trace(rho: np.ndarray, dim_A: int, dim_B: int,
                  trace_system: str = 'B') -> np.ndarray:
    """
    计算偏迹

    Args:
        rho: 复合系统的密度矩阵 (dim_A * dim_B) × (dim_A * dim_B)
        dim_A: 子系统A的维度
        dim_B: 子系统B的维度
        trace_system: 'B' 或 'A'，要迹掉的子系统

    Returns:
        约化密度矩阵
    """
    # 重塑为张量形式
    rho_tensor = rho.reshape(dim_A, dim_B, dim_A, dim_B)

    if trace_system == 'B':
        # 对B求迹：ρ_A = Tr_B(ρ_AB)
        reduced = np.trace(rho_tensor, axis1=1, axis2=3)
    else:
        # 对A求迹：ρ_B = Tr_A(ρ_AB)
        reduced = np.trace(rho_tensor, axis1=0, axis2=2)

    return reduced

def quantum_conditional_entropy(rho_AB: np.ndarray, dim_A: int,
                                 dim_B: int) -> float:
    """
    计算量子条件熵 S(A|B) = S(ρ_AB) - S(ρ_B)
    """
    S_AB = von_neumann_entropy(rho_AB)
    rho_B = partial_trace(rho_AB, dim_A, dim_B, trace_system='A')
    S_B = von_neumann_entropy(rho_B)
    return S_AB - S_B

def quantum_mutual_information(rho_AB: np.ndarray, dim_A: int,
                                dim_B: int) -> float:
    """
    计算量子互信息 I(A:B) = S(ρ_A) + S(ρ_B) - S(ρ_AB)
    """
    rho_A = partial_trace(rho_AB, dim_A, dim_B, trace_system='B')
    rho_B = partial_trace(rho_AB, dim_A, dim_B, trace_system='A')

    S_A = von_neumann_entropy(rho_A)
    S_B = von_neumann_entropy(rho_B)
    S_AB = von_neumann_entropy(rho_AB)

    return S_A + S_B - S_AB

# 创建常用的量子态
def pure_state(state_vector: np.ndarray) -> np.ndarray:
    """从态矢创建纯态密度矩阵"""
    return np.outer(state_vector, state_vector.conj())

def bell_state(state_type: str = 'phi_plus') -> np.ndarray:
    """
    创建贝尔态
    |Φ+⟩ = (|00⟩ + |11⟩)/√2
    |Φ-⟩ = (|00⟩ - |11⟩)/√2
    |Ψ+⟩ = (|01⟩ + |10⟩)/√2
    |Ψ-⟩ = (|01⟩ - |10⟩)/√2
    """
    if state_type == 'phi_plus':
        psi = np.array([1, 0, 0, 1]) / np.sqrt(2)
    elif state_type == 'phi_minus':
        psi = np.array([1, 0, 0, -1]) / np.sqrt(2)
    elif state_type == 'psi_plus':
        psi = np.array([0, 1, 1, 0]) / np.sqrt(2)
    elif state_type == 'psi_minus':
        psi = np.array([0, 1, -1, 0]) / np.sqrt(2)
    else:
        raise ValueError(f"Unknown Bell state: {state_type}")

    return pure_state(psi)

# 示例测试
print("=== 量子熵示例 ===")

# 例1：纯态的熵
print("\n例1：纯态的von Neumann熵")
psi_0 = np.array([1, 0])
rho_pure = pure_state(psi_0)
S_pure = von_neumann_entropy(rho_pure)
print(f"|0⟩态的熵: {S_pure:.6f} qubits (应为0)")

# 例2：最大混合态
print("\n例2：最大混合态")
I_2 = np.eye(2) / 2
S_max = von_neumann_entropy(I_2)
print(f"I/2的熵: {S_max:.6f} qubits (应为1)")

# 例3：一般混合态
print("\n例3：一般混合态")
p = 0.7
rho_mixed = p * pure_state(np.array([1, 0])) + (1-p) * pure_state(np.array([0, 1]))
S_mixed = von_neumann_entropy(rho_mixed)
print(f"ρ = {p}|0⟩⟨0| + {1-p}|1⟩⟨1|")
print(f"熵: {S_mixed:.6f} qubits")
print(f"经典二元熵 H({p}): {-(p*np.log2(p) + (1-p)*np.log2(1-p)):.6f}")

# 例4：贝尔态的熵
print("\n例4：贝尔态（纠缠态）")
phi_plus = bell_state('phi_plus')
S_bell = von_neumann_entropy(phi_plus)
print(f"|Φ+⟩的熵: {S_bell:.6f} qubits (纯态，应为0)")

# 计算约化密度矩阵
rho_A = partial_trace(phi_plus, 2, 2, trace_system='B')
S_A = von_neumann_entropy(rho_A)
print(f"约化密度矩阵ρ_A的熵: {S_A:.6f} qubits")
print(f"这是最大混合态: {np.allclose(rho_A, np.eye(2)/2)}")

# 例5：量子条件熵（可以为负！）
print("\n例5：量子条件熵（纠缠态可以为负）")
S_cond = quantum_conditional_entropy(phi_plus, 2, 2)
print(f"S(A|B) for |Φ+⟩: {S_cond:.6f} qubits")
print("负的条件熵是量子纠缠的标志！")

# 例6：量子互信息
print("\n例6：量子互信息")
I_AB = quantum_mutual_information(phi_plus, 2, 2)
print(f"I(A:B) for |Φ+⟩: {I_AB:.6f} qubits")

# 例7：可分态
print("\n例7：可分态（乘积态）")
rho_product = np.kron(pure_state(np.array([1, 0])), I_2)
I_product = quantum_mutual_information(rho_product, 2, 2)
print(f"可分态的互信息: {I_product:.6f} qubits (应为0)")

# 例8：量子相对熵
print("\n例8：量子相对熵")
rho_test = p * pure_state(np.array([1, 0])) + (1-p) * pure_state(np.array([0, 1]))
sigma_test = np.eye(2) / 2
D_rel = quantum_relative_entropy(rho_test, sigma_test)
print(f"D(ρ||I/2): {D_rel:.6f}")
print(f"相对熵非负: {D_rel >= -1e-10}")

# 例9：多个贝尔态
print("\n" + "="*50)
print("\n不同贝尔态的比较:")
for state_name in ['phi_plus', 'phi_minus', 'psi_plus', 'psi_minus']:
    bell = bell_state(state_name)
    S_total = von_neumann_entropy(bell)
    S_A = von_neumann_entropy(partial_trace(bell, 2, 2, 'B'))
    S_cond = quantum_conditional_entropy(bell, 2, 2)
    I_AB = quantum_mutual_information(bell, 2, 2)

    print(f"\n|{state_name.replace('_', '')}⟩:")
    print(f"  S(ρ) = {S_total:.4f}")
    print(f"  S(ρ_A) = {S_A:.4f}")
    print(f"  S(A|B) = {S_cond:.4f}")
    print(f"  I(A:B) = {I_AB:.4f}")
```

## 10.5.1.8 总结

```mermaid
flowchart TB
    A[密度算子 ρ] --> B[von Neumann熵]
    B --> C[S(ρ) = -Tr(ρ log ρ)]

    C --> D[性质]
    D --> E[S(ρ) ≥ 0]
    D --> F[S(ρ) ≤ log d]
    D --> G[凹性]

    H[量子相对熵] --> I[D(ρ||σ) ≥ 0]

    J[条件熵] --> K[S(A|B) = S(ρ_AB) - S(ρ_B)]
    K --> L[可以为负！]

    M[互信息] --> N[I(A:B) ≥ 0]
```

**核心结论**：

1. **von Neumann熵**：$S(\rho) = -\text{Tr}(\rho \log \rho)$，是量子态的不确定度量
2. **性质**：非负、有界、酉不变、凹
3. **相对熵**：$D(\rho \| \sigma) \geq 0$（Klein不等式）
4. **条件熵**：可以为负，是量子纠缠的标志
5. **互信息**：非负，度量量子关联
