# 10.1.4 互信息与相对熵

> 基于 Shannon (1948) 和 Cover & Thomas (2006)

## 10.1.4.1 引言

**互信息**（Mutual Information）和**相对熵**（Relative Entropy，又称Kullback-Leibler散度）是信息论中两个核心概念。
互信息度量两个随机变量之间的统计依赖性，而相对熵度量两个概率分布之间的差异。
这两个概念在机器学习、统计推断、信号处理等领域有广泛应用。

## 10.1.4.2 互信息的定义

### 定义 10.1.4.1（互信息）

随机变量 $X$ 和 $Y$ 之间的**互信息**定义为：

$$I(X; Y) = \sum_{x \in \mathcal{X}} \sum_{y \in \mathcal{Y}} p(x, y) \log \frac{p(x, y)}{p(x)p(y)}$$

等价形式：
$$I(X; Y) = \mathbb{E}_{p(x,y)}\left[\log \frac{p(X, Y)}{p(X)p(Y)}\right]$$

### 互信息的解释

互信息度量：

1. **信息增益**：知道 $Y$ 后，关于 $X$ 的不确定性的减少量
2. **依赖性**：$X$ 和 $Y$ 之间统计依赖性的强度
3. **KL散度**：联合分布与独立分布之间的差异

## 10.1.4.3 互信息与其他熵量的关系

### 定理 10.1.4.1（互信息的熵表示）

$$I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$

**证明**：

$$
\begin{aligned}
I(X; Y) &= \sum_{x,y} p(x,y) \log \frac{p(x,y)}{p(x)p(y)} \\
&= \sum_{x,y} p(x,y) \log \frac{p(x|y)}{p(x)} \\
&= \sum_{x,y} p(x,y) \log p(x|y) - \sum_{x,y} p(x,y) \log p(x) \\
&= -H(X|Y) + H(X) \\
&= H(X) - H(X|Y)
\end{aligned}
$$

### 定理 10.1.4.2（互信息的对称性）

$$I(X; Y) = I(Y; X)$$

### 定理 10.1.4.3（联合熵分解）

$$I(X; Y) = H(X) + H(Y) - H(X, Y)$$

## 10.1.4.4 互信息的性质

### 定理 10.1.4.4（非负性）

$$I(X; Y) \geq 0$$

等号成立当且仅当 $X$ 和 $Y$ 独立。

**证明**：由Jensen不等式和 $\log x$ 的凹性：

$$\begin{aligned}
-I(X; Y) &= \sum_{x,y} p(x,y) \log \frac{p(x)p(y)}{p(x,y)} \\
&\leq \log \sum_{x,y} p(x,y) \frac{p(x)p(y)}{p(x,y)} \\
&= \log \sum_{x,y} p(x)p(y) \\
&= \log 1 = 0
\end{aligned}$$

因此 $I(X; Y) \geq 0$。

### 定理 10.1.4.5（有界性）

$$I(X; Y) \leq \min\{H(X), H(Y)\}$$

### 定理 10.1.4.6（凸性/凹性）

- 固定 $p(x|y)$ 时，$I(X;Y)$ 是关于 $p(x)$ 的**凹函数**
- 固定 $p(x)$ 时，$I(X;Y)$ 是关于 $p(y|x)$ 的**凸函数**

## 10.1.4.5 Kullback-Leibler散度

### 定义 10.1.4.2（KL散度）

概率分布 $p$ 相对于 $q$ 的**相对熵**或**KL散度**定义为：

$$D_{KL}(p \| q) = \sum_{x \in \mathcal{X}} p(x) \log \frac{p(x)}{q(x)}$$

约定：$0 \log(0/q) = 0$，$p \log(p/0) = +\infty$（当 $p > 0$）。

### KL散度的性质

**定理 10.1.4.7**（非负性）：
$$D_{KL}(p \| q) \geq 0$$

等号成立当且仅当 $p = q$（几乎处处）。

**证明**：由Jensen不等式：
$$\begin{aligned}
-D_{KL}(p \| q) &= \sum_x p(x) \log \frac{q(x)}{p(x)} \\
&\leq \log \sum_x p(x) \frac{q(x)}{p(x)} \\
&= \log \sum_x q(x) \\
&= \log 1 = 0
\end{aligned}$$

**注意**：KL散度**不是**真正的距离度量，因为它：
1. 不对称：$D_{KL}(p \| q) \neq D_{KL}(q \| p)$（一般情况）
2. 不满足三角不等式

### KL散度与互信息的关系

$$I(X; Y) = D_{KL}(p(x,y) \| p(x)p(y))$$

这表明互信息是联合分布与独立分布之间的"距离"。

## 10.1.4.6 条件互信息

### 定义 10.1.4.3（条件互信息）

$$I(X; Y|Z) = \sum_z p(z) \sum_{x,y} p(x,y|z) \log \frac{p(x,y|z)}{p(x|z)p(y|z)}$$

或：
$$I(X; Y|Z) = H(X|Z) - H(X|Y, Z)$$

### 链式法则

$$I(X; Y, Z) = I(X; Y) + I(X; Z|Y)$$

## 10.1.4.7 多变量互信息

### 交互信息（Interaction Information）

对于三个变量：
$$I(X; Y; Z) = I(X; Y) - I(X; Y|Z)$$

注意：交互信息可以为负！

## 10.1.4.8 应用：特征选择

互信息在机器学习中常用于特征选择：

```mermaid
flowchart LR
    A[特征X] --> C[互信息 I(X;Y)]
    B[目标Y] --> C
    C --> D{阈值判断}
    D -->|高互信息| E[选择特征]
    D -->|低互信息| F[舍弃特征]
```

选择标准：最大化 $I(X_i; Y)$，即选择与目标变量最相关的特征。

## 10.1.4.9 代码实现

### Python 实现

```python
import math
import numpy as np
from typing import Dict, Tuple, List
from collections import defaultdict

def mutual_information(joint_dist: Dict[Tuple[str, str], float]) -> float:
    """
    计算互信息 I(X;Y)

    Args:
        joint_dist: 联合概率分布

    Returns:
        互信息值（单位：bits）
    """
    # 计算边缘分布
    p_x = defaultdict(float)
    p_y = defaultdict(float)

    for (x, y), p in joint_dist.items():
        p_x[x] += p
        p_y[y] += p

    # 计算互信息
    I = 0.0
    for (x, y), p_xy in joint_dist.items():
        if p_xy > 0:
            p_x_val = p_x[x]
            p_y_val = p_y[y]
            if p_x_val > 0 and p_y_val > 0:
                I += p_xy * math.log2(p_xy / (p_x_val * p_y_val))

    return I

def kl_divergence(p: Dict[str, float], q: Dict[str, float]) -> float:
    """
    计算KL散度 D_KL(P || Q)

    Args:
        p: 真实分布
        q: 近似分布

    Returns:
        KL散度值
    """
    dkl = 0.0
    for x, p_x in p.items():
        if p_x > 0:
            q_x = q.get(x, 0)
            if q_x == 0:
                return float('inf')  # 未定义
            dkl += p_x * math.log2(p_x / q_x)
    return dkl

def entropy_from_dist(dist: Dict[str, float]) -> float:
    """计算熵"""
    H = 0.0
    for p in dist.values():
        if p > 0:
            H -= p * math.log2(p)
    return H

def conditional_entropy_from_joint(joint_dist: Dict[Tuple[str, str], float],
                                    condition_axis: int = 1) -> float:
    """从联合分布计算条件熵"""
    # condition_axis=1: H(X|Y), =0: H(Y|X)
    p_cond = defaultdict(float)
    H_cond = 0.0

    for (x, y), p_xy in joint_dist.items():
        key = y if condition_axis == 1 else x
        p_cond[key] += p_xy

    for (x, y), p_xy in joint_dist.items():
        if p_xy > 0:
            cond_key = y if condition_axis == 1 else x
            p_given = p_xy / p_cond[cond_key]
            H_cond -= p_xy * math.log2(p_given)

    return H_cond

def joint_entropy(joint_dist: Dict[Tuple[str, str], float]) -> float:
    """计算联合熵"""
    H = 0.0
    for p in joint_dist.values():
        if p > 0:
            H -= p * math.log2(p)
    return H

# 示例计算
print("=== 互信息与KL散度示例 ===")

# 例1：天气与活动（相关）
weather_activity = {
    ('晴天', '进行'): 0.30,
    ('晴天', '取消'): 0.05,
    ('阴天', '进行'): 0.15,
    ('阴天', '取消'): 0.05,
    ('雨天', '进行'): 0.05,
    ('雨天', '取消'): 0.40
}

print("\n例1：天气与户外活动")
I1 = mutual_information(weather_activity)
print(f"互信息 I(X;Y) = {I1:.4f} bits")

# 验证：I(X;Y) = H(X) - H(X|Y)
from collections import defaultdict
p_x = defaultdict(float)
for (x, y), p in weather_activity.items():
    p_x[x] += p
H_X = entropy_from_dist(dict(p_x))
H_X_given_Y = conditional_entropy_from_joint(weather_activity, 1)
print(f"验证: H(X) - H(X|Y) = {H_X:.4f} - {H_X_given_Y:.4f} = {H_X - H_X_given_Y:.4f}")

# 例2：独立随机变量
independent_joint = {
    ('A', '1'): 0.25,
    ('A', '2'): 0.25,
    ('B', '1'): 0.25,
    ('B', '2'): 0.25
}
print("\n例2：独立随机变量")
I2 = mutual_information(independent_joint)
print(f"互信息 I(X;Y) = {I2:.6f} bits (应为0)")

# 例3：完全相关（函数关系）
functional_joint = {
    ('A', '1'): 0.5,
    ('A', '2'): 0.0,
    ('B', '1'): 0.0,
    ('B', '2'): 0.5
}
print("\n例3：完全相关（Y = f(X)）")
I3 = mutual_information(functional_joint)
print(f"互信息 I(X;Y) = {I3:.4f} bits")
H_X_func = entropy_from_dist({'A': 0.5, 'B': 0.5})
print(f"H(X) = {H_X_func:.4f} bits")
print(f"说明：互信息等于H(X)，因为Y完全由X决定")

# KL散度示例
print("\n=== KL散度示例 ===")
p_true = {'A': 0.5, 'B': 0.3, 'C': 0.2}
q_approx1 = {'A': 0.5, 'B': 0.3, 'C': 0.2}  # 相同
q_approx2 = {'A': 0.4, 'B': 0.4, 'C': 0.2}  # 不同

dkl1 = kl_divergence(p_true, q_approx1)
dkl2 = kl_divergence(p_true, q_approx2)

print(f"D_KL(P||P) = {dkl1:.6f} (应为0)")
print(f"D_KL(P||Q) = {dkl2:.4f} bits")

# 验证非对称性
dkl_reverse = kl_divergence(q_approx2, p_true)
print(f"D_KL(Q||P) = {dkl_reverse:.4f} bits")
print(f"非对称性: D_KL(P||Q) ≠ D_KL(Q||P)")

# 高斯分布的互信息示例
print("\n=== 连续变量互信息（高斯分布） ===")
def gaussian_mutual_info(rho: float) -> float:
    """
    对于相关系数为rho的二元高斯分布，
    互信息 I(X;Y) = -0.5 * log2(1 - rho^2)
    """
    if abs(rho) >= 1:
        return float('inf')
    return -0.5 * math.log2(1 - rho**2)

for rho in [0, 0.3, 0.5, 0.7, 0.9, 0.99]:
    mi = gaussian_mutual_info(rho)
    print(f"ρ = {rho:.2f}: I(X;Y) = {mi:.4f} bits")

# 特征选择应用示例
print("\n=== 特征选择应用 ===")
features = {
    '特征1': {('高', '患病'): 0.4, ('高', '健康'): 0.1,
              ('低', '患病'): 0.1, ('低', '健康'): 0.4},
    '特征2': {('正', '患病'): 0.25, ('正', '健康'): 0.25,
              ('负', '患病'): 0.25, ('负', '健康'): 0.25}
}

for fname, joint in features.items():
    mi = mutual_information(joint)
    print(f"{fname}: I(特征;疾病) = {mi:.4f} bits")
print("结论：特征1与疾病更相关")
```

**核心要点**：

| 概念 | 公式 | 解释 |
|------|------|------|
| 互信息 | $I(X;Y) = \sum p(x,y)\log\frac{p(x,y)}{p(x)p(y)}$ | 变量间信息共享量 |
| KL散度 | $D_{KL}(p\|q) = \sum p(x)\log\frac{p(x)}{q(x)}$ | 分布间"距离" |
| 关系 | $I(X;Y) = D_{KL}(p(x,y)\|p(x)p(y))$ | 互信息是特殊KL散度 |

**应用**：
- 特征选择：最大化 $I(X_i; Y)$
- 变分推断：最小化 $D_{KL}(q\|p)$
- 信息瓶颈：优化 $I(X; T) - \beta I(T; Y)$
