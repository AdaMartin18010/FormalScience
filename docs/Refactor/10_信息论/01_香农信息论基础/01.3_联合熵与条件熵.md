# 10.1.3 联合熵与条件熵

> 基于 Shannon (1948) 和 Cover & Thomas (2006)

## 10.1.3.1 引言

在信息论中，我们常常需要处理多个随机变量之间的关系。
**联合熵**（Joint Entropy）和**条件熵**（Conditional Entropy）是描述多变量系统不确定性的重要工具。
这些概念为定义互信息和推导信息论基本不等式奠定了基础。

## 10.1.3.2 联合熵

### 定义 10.1.3.1（联合熵）

设离散随机变量对 $(X, Y)$ 的联合概率质量函数为 $p(x, y)$，则**联合熵**定义为：

$$H(X, Y) = -\sum_{x \in \mathcal{X}} \sum_{y \in \mathcal{Y}} p(x, y) \log p(x, y)$$

等价地：
$$H(X, Y) = \mathbb{E}\left[\log \frac{1}{p(X, Y)}\right]$$

### 联合熵的性质

**定理 10.1.3.1**（非负性与有界性）：
$$0 \leq H(X, Y) \leq \log(|\mathcal{X}| \cdot |\mathcal{Y}|)$$

**证明**：

- 非负性：由定义直接可得，每项 $-p(x,y)\log p(x,y) \geq 0$
- 上界：由Jensen不等式，当 $(X,Y)$ 均匀分布时取等

**定理 10.1.3.2**（单调性）：
$$H(X, Y) \geq \max\{H(X), H(Y)\}$$

**证明**：由链式法则和条件熵的非负性可得（见下文）。

### 多元联合熵

对于 $n$ 个随机变量 $X_1, X_2, \ldots, X_n$：

$$H(X_1, X_2, \ldots, X_n) = -\sum_{x_1, \ldots, x_n} p(x_1, \ldots, x_n) \log p(x_1, \ldots, x_n)$$

## 10.1.3.3 条件熵

### 定义 10.1.3.2（条件熵）

给定 $Y$ 时 $X$ 的**条件熵**定义为：

$$H(X|Y) = \sum_{y \in \mathcal{Y}} p(y) H(X|Y=y)$$

其中 $H(X|Y=y) = -\sum_{x \in \mathcal{X}} p(x|y) \log p(x|y)$

等价形式：
$$H(X|Y) = -\sum_{x, y} p(x, y) \log p(x|y) = \mathbb{E}\left[\log \frac{1}{p(X|Y)}\right]$$

### 条件熵的直观理解

条件熵 $H(X|Y)$ 表示：在已知 $Y$ 的情况下，$X$ 剩余的**平均不确定性**。

- 若 $X$ 和 $Y$ 独立，则 $H(X|Y) = H(X)$（$Y$ 不提供关于 $X$ 的信息）
- 若 $X = f(Y)$（$X$ 是 $Y$ 的函数），则 $H(X|Y) = 0$（$Y$ 完全确定 $X$）

## 10.1.3.4 熵的链式法则

### 定理 10.1.3.3（链式法则）

$$H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$$

**证明**：

$$\begin{aligned}
H(X, Y) &= -\sum_{x,y} p(x,y) \log p(x,y) \\
&= -\sum_{x,y} p(x,y) \log [p(x) \cdot p(y|x)] \\
&= -\sum_{x,y} p(x,y) [\log p(x) + \log p(y|x)] \\
&= -\sum_{x} p(x) \log p(x) - \sum_{x,y} p(x,y) \log p(y|x) \\
&= H(X) + H(Y|X)
\end{aligned}$$

### 多元链式法则

对于 $n$ 个随机变量：
$$H(X_1, \ldots, X_n) = \sum_{i=1}^n H(X_i | X_1, \ldots, X_{i-1})$$

特别地，当 $X_i$ 相互独立时：
$$H(X_1, \ldots, X_n) = \sum_{i=1}^n H(X_i)$$

## 10.1.3.5 条件熵的性质

### 定理 10.1.3.4（条件熵的非负性）

$$H(X|Y) \geq 0$$

等号成立当且仅当 $X$ 是 $Y$ 的函数（在给定 $Y$ 时 $X$ 确定）。

### 定理 10.1.3.5（条件作用减少熵）

$$H(X|Y) \leq H(X)$$

等号成立当且仅当 $X$ 和 $Y$ 独立。

**证明**：这等价于互信息的非负性（将在下一节证明）。直观上，知道 $Y$ 不会增加对 $X$ 的不确定性。

### 定理 10.1.3.6（条件熵的链式法则）

$$H(X, Y|Z) = H(X|Z) + H(Y|X, Z)$$

### 定理 10.1.3.7（条件熵的界）

$$0 \leq H(X|Y) \leq H(X) \leq H(X, Y) \leq H(X) + H(Y)$$

## 10.1.3.7 条件熵与数据处理

### 数据处理不等式（预览）

若 $X \to Y \to Z$ 形成马尔可夫链，则：
$$H(X|Z) \geq H(X|Y)$$

这表明：经过数据处理，关于 $X$ 的信息不会增加，只会减少或保持不变。

## 10.1.3.8 代码实现

### Python 实现

```python
import math
import numpy as np
from typing import List, Tuple, Dict

def joint_entropy(joint_dist: Dict[Tuple[str, str], float]) -> float:
    """
    计算联合熵 H(X,Y)

    Args:
        joint_dist: 联合概率分布，键为(x, y)元组

    Returns:
        联合熵值
    """
    H = 0.0
    for (x, y), p in joint_dist.items():
        if p > 0:
            H -= p * math.log2(p)
    return H

def marginal_dist(joint_dist: Dict[Tuple[str, str], float], axis: int = 0) -> Dict[str, float]:
    """
    计算边缘分布
    axis=0: 对Y求和，得到X的分布
    axis=1: 对X求和，得到Y的分布
    """
    marginal = {}
    for (x, y), p in joint_dist.items():
        key = x if axis == 0 else y
        marginal[key] = marginal.get(key, 0) + p
    return marginal

def conditional_entropy(joint_dist: Dict[Tuple[str, str], float],
                        condition_axis: int = 1) -> float:
    """
    计算条件熵
    condition_axis=1: H(X|Y)
    condition_axis=0: H(Y|X)
    """
    # 条件变量的边缘分布
    cond_marginal = marginal_dist(joint_dist, axis=1-condition_axis)

    H_cond = 0.0
    for (x, y), p_xy in joint_dist.items():
        if p_xy > 0:
            cond_var = y if condition_axis == 1 else x
            p_cond = cond_marginal[cond_var]
            p_given = p_xy / p_cond
            H_cond -= p_xy * math.log2(p_given)

    return H_cond

def entropy_from_dist(dist: Dict[str, float]) -> float:
    """计算熵"""
    H = 0.0
    for p in dist.values():
        if p > 0:
            H -= p * math.log2(p)
    return H

# 示例：天气与活动
print("=== 联合熵与条件熵示例 ===")
print("例：天气(X)与户外活动(Y)的关系\n")

# 联合分布
weather_activity = {
    ('晴天', '进行'): 0.30,
    ('晴天', '取消'): 0.05,
    ('阴天', '进行'): 0.15,
    ('阴天', '取消'): 0.05,
    ('雨天', '进行'): 0.05,
    ('雨天', '取消'): 0.40
}

# 验证概率和为1
total = sum(weather_activity.values())
print(f"联合概率总和: {total}")

# 计算联合熵
H_XY = joint_entropy(weather_activity)
print(f"\n联合熵 H(X,Y) = {H_XY:.4f} bits")

# 边缘分布
weather_dist = marginal_dist(weather_activity, axis=0)
activity_dist = marginal_dist(weather_activity, axis=1)

print(f"\n天气分布: {weather_dist}")
print(f"活动分布: {activity_dist}")

# 计算各熵
H_X = entropy_from_dist(weather_dist)
H_Y = entropy_from_dist(activity_dist)
H_X_given_Y = conditional_entropy(weather_activity, condition_axis=1)
H_Y_given_X = conditional_entropy(weather_activity, condition_axis=0)

print(f"\n熵 H(X) = {H_X:.4f} bits")
print(f"熵 H(Y) = {H_Y:.4f} bits")
print(f"条件熵 H(X|Y) = {H_X_given_Y:.4f} bits")
print(f"条件熵 H(Y|X) = {H_Y_given_X:.4f} bits")

# 验证链式法则
print(f"\n=== 验证链式法则 ===")
print(f"H(X) + H(Y|X) = {H_X + H_Y_given_X:.4f}")
print(f"H(Y) + H(X|Y) = {H_Y + H_X_given_Y:.4f}")
print(f"H(X,Y) = {H_XY:.4f}")
print(f"链式法则验证: {math.isclose(H_XY, H_X + H_Y_given_X)}")

# 验证条件作用减少熵
print(f"\n=== 验证条件作用减少熵 ===")
print(f"H(X|Y) = {H_X_given_Y:.4f} ≤ H(X) = {H_X:.4f}: {H_X_given_Y <= H_X + 1e-10}")
print(f"H(Y|X) = {H_Y_given_X:.4f} ≤ H(Y) = {H_Y:.4f}: {H_Y_given_X <= H_Y + 1e-10}")

# 互信息（预告下一节）
mutual_info = H_X - H_X_given_Y
print(f"\n互信息 I(X;Y) = H(X) - H(X|Y) = {mutual_info:.4f} bits")

# 独立随机变量的例子
print("\n=== 独立随机变量示例 ===")
independent_joint = {
    ('A', '1'): 0.25,
    ('A', '2'): 0.25,
    ('B', '1'): 0.25,
    ('B', '2'): 0.25
}
H_XY_ind = joint_entropy(independent_joint)
H_X_ind = entropy_from_dist(marginal_dist(independent_joint, 0))
H_Y_ind = entropy_from_dist(marginal_dist(independent_joint, 1))
H_X_given_Y_ind = conditional_entropy(independent_joint, 1)

print(f"独立时: H(X,Y) = {H_XY_ind:.4f}")
print(f"H(X) + H(Y) = {H_X_ind + H_Y_ind:.4f}")
print(f"H(X|Y) = H(X): {math.isclose(H_X_given_Y_ind, H_X_ind)}")
```

## 10.1.3.10 小结

**关键概念**：

| 概念 | 定义 | 解释 |
|------|------|------|
| 联合熵 $H(X,Y)$ | $-\sum p(x,y)\log p(x,y)$ | $(X,Y)$ 的总不确定性 |
| 条件熵 $H(X|Y)$ | $\sum p(y)H(X|Y=y)$ | 已知 $Y$ 后 $X$ 的不确定性 |
| 链式法则 | $H(X,Y) = H(X) + H(Y|X)$ | 熵的可加分解 |

**重要不等式**：
- $H(X|Y) \geq 0$（条件熵非负）
- $H(X|Y) \leq H(X)$（条件作用减少熵）
- $H(X,Y) \leq H(X) + H(Y)$（独立时取等）

**参考**：
- Shannon, C. E. (1948). A mathematical theory of communication.
- Cover, T. M., & Thomas, J. A. (2006). *Elements of information theory*.
