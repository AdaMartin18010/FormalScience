## 如何成为贡献者

我们欢迎各种形式的贡献！请参阅以下指南：

1. **代码贡献** - 阅读 [CONTRIBUTING.md](../CONTRIBUTING.md) 了解代码提交规范
2. **文档贡献** - 帮助改进或翻译文档
3. **问题反馈** - 通过 Issue 系统报告 bug 或建议
4. **代码审查** - 参与 Pull Request 的审查

### 贡献者等级
| 等级 | 要求 | 特权 |
|------|------|------|
| 🥉 贡献者 | 1+ PR 被合并 | 名字列入贡献者名单 |
| 🥈 活跃贡献者 | 5+ PR 被合并 | 参与功能讨论 |
| 🥇 核心贡献者 | 20+ PR 被合并 | 代码审查权限 |
| 💎 维护者 | 长期贡献 | 项目管理权限 |
