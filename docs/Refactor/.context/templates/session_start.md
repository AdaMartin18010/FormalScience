# 会话启动检查清单

> 每次会话开始前执行此检查清单

## 📋 启动步骤

### 1. 读取断点状态

- [ ] 打开 `CHECKPOINT.md`
- [ ] 查看最新断点编号和状态
- [ ] 阅读上下文摘要
- [ ] 确认待继续事项

### 2. 确认当前焦点

- [ ] 打开 `CURRENT_FOCUS.md`
- [ ] 了解当前任务详情
- [ ] 查看任务进度
- [ ] 阅读注意事项

### 3. 检查任务队列

- [ ] 打开 `TASK_QUEUE.md`
- [ ] 查看是否有新的紧急任务
- [ ] 确认当前任务的优先级
- [ ] 了解依赖关系

### 4. 环境检查

- [ ] 确认工作目录正确
- [ ] 检查相关文件是否存在
- [ ] 验证文件权限
- [ ] 确保无未保存的冲突变更
