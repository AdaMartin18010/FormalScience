## 如何支持本项目

如果您希望支持本项目的发展，可以通过以下方式：

1. **贡献代码** - 提交 Pull Request 改进功能
2. **完善文档** - 帮助改进和翻译文档
3. **报告问题** - 提交 Issue 反馈 bug 或建议
4. **推广项目** - 向更多人介绍本项目
5. **成为赞助者** - 查看 [SUPPORTERS.md](./SUPPORTERS.md) 了解详情
