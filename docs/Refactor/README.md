## 🎯 内容规范

### 形式化标准

1. **数学符号**: 使用标准 LaTeX 数学符号
2. **证明结构**: 定理-引理-证明 标准结构
3. **代码示例**: 优先 Lean 4，次选 Haskell/Rust
4. **多表征**: 图表、公式、代码三位一体

### 文档格式

- **标题层级**: 最多 6 层，严格树形编号
- **交叉引用**: 相对路径链接，支持本地跳转
- **元数据**: 每文件包含主题、依赖、状态标签
