## 🎯 项目概述

本项目是对形式科学内容的全面重构与形式化，涵盖数学基础、形式语言、编程范式、软件工程、形式化理论、调度系统，以及新增的统计学、信息论、系统科学、决策与博弈论、认知科学、形式语言学和社会科学形式化等七大领域。

### 核心特色

- 📚 **1,190+ 文件** - 史诗级内容规模 (新增120个文件)

- 🐳 **Docker支持** - 一键部署
- 🔄 **CI/CD集成** - 自动化工作流
- 🌐 **项目网站** - 完整展示
- ⭐ **7个新增模块** - 数据科学、系统科学、认知社会科学

## 📚 新增模块概览

### 09 统计学 📊

从描述统计到统计计算的完整统计理论体系：
- 描述统计：集中趋势、离散度、数据可视化
- 概率论基础：概率空间、随机变量、分布
- 推断统计：估计、假设检验、方差分析
- 贝叶斯统计：先验/后验、MCMC方法
- 数理统计：决策理论、充分统计量、渐近理论
- 统计计算：随机数、Bootstrap、计算贝叶斯

### 10 信息论 📡

从香农理论到量子信息的完整信息理论：
- 香农信息论：熵、互信息、相对熵
- 信源编码：Kraft不等式、Huffman编码、算术编码
- 信道编码：信道容量、纠错码、线性分组码
- 算法信息论：Kolmogorov复杂度、MDL原理
- 量子信息论：量子熵、量子信道、纠缠

### 11 系统科学 🌐

系统论、控制论与复杂性科学的综合：
- 一般系统论：Bertalanffy理论、涌现、系统思维
- 控制论：反馈系统、稳定性、最优控制
- 复杂系统：复杂性度量、相变、混沌
- 自组织理论：耗散结构、协同学、超循环
- 网络科学：复杂网络、网络动力学、鲁棒性
- 系统动力学：存量流量、反馈回路、建模

### 12 决策与博弈论 🎯

理性决策与策略互动的形式化：
- 决策理论基础：偏好、效用、风险
- 博弈论基础：纳什均衡、子博弈精炼、重复博弈
- 机制设计：显示原理、拍卖、投票、匹配
- 社会选择理论：阿罗定理、偏好聚合、公平分配
- 行为博弈论：有限理性、演化博弈

### 13 认知科学形式模型 🧠

认知过程的形式化理论与计算模型：
- 形式认识论：信念逻辑、知识逻辑
- 计算认知科学：ACT-R、联结主义、贝叶斯认知
- 概念与范畴：概念空间、原型理论
- 推理与问题解决：演绎、归纳、类比推理
- 学习与记忆：学习理论、记忆模型、语言习得

### 14 形式语言学 🗣️

自然语言的形式化理论与计算：
- 形式语法：Chomsky层次、转换语法、范畴语法
- 形式语义学：Montague语法、类型逻辑、DRT
- 音系学形式理论：生成音系学、优选论
- 计算语言学：句法分析、语义解析

### 15 社会科学形式化 👥

经济学、政治学等社会科学的形式化：
- 数理经济学：一般均衡、博弈论经济、信息经济
- 形式政治学：投票理论、权力指数、联盟
- 计算社会学：社会网络、社会模拟、文化演化
- 形式人类学：亲属关系、文化模式
