### 5.1 æ–æ³¢é‚£å¥‘æ•°åˆ—

**é—®é¢˜**ï¼šæ±‚è§£æ–æ³¢é‚£å¥‘æ•°åˆ— $F_n = F_{n-1} + F_{n-2}$ï¼Œå…¶ä¸­ $F_0 = 0, F_1 = 1$ã€‚

$$= x + \sum_{n=2}^{\infty} F_{n-1} x^n + \sum_{n=2}^{\infty} F_{n-2} x^n$$

$$= x + x \sum_{n=1}^{\infty} F_n x^n + x^2 \sum_{n=0}^{\infty} F_n x^n$$

$$= x + x F(x) + x^2 F(x)$$

### 5.2 ç»„åˆé—®é¢˜
