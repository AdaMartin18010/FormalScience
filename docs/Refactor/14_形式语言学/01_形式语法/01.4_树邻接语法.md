## 14.1.4.1 引言

树邻接语法（Tree Adjoining Grammar, TAG）由Joshi, Levy & Takahashi（1975）提出，是一种介于上下文无关文法和上下文有关文法之间的 mildly context-sensitive 形式系统。TAG通过**树邻接**（tree adjoining）操作而非字符串重写来生成语言。

> **核心特征**：TAG能够自然描述自然语言中的长距离依存和嵌套结构，同时保持多项式时间可解析性。

## 14.1.4.2 基本定义

### 14.1.4.2.1 基本树与辅助树

> **定义 14.1.4.1**（TAG文法）
> TAG文法是一个五元组 G = (N, Σ, S, I, A)，其中：
>
> - N：非终结符集合
> - Σ：终结符集合
> - S ∈ N：起始符号
> - I：初始树（initial trees）的有限集合
> - A：辅助树（auxiliary trees）的有限集合

> **定义 14.1.4.2**（初始树）
> 初始树（α树）是满足以下条件的树：
>
> - 所有内部节点标记为非终结符
> - 所有叶子节点标记为终结符或非终结符
> - 若叶子标记为非终结符，则标记为**足迹**（foot node）或**空替换节点**（标记为↓）
> - 根节点标记为S或其他非终结符
> - **没有足迹节点**

> **定义 14.1.4.3**（辅助树）
> 辅助树（β树）是满足以下条件的树：
>
> - 根节点和某个叶子节点标记相同的非终结符 X
> - 该叶子节点称为**足迹节点**（foot node），标记为 X*
> - 足迹节点必须位于树的**前沿**（frontier）

```
初始树示例：                   辅助树示例：
      S                              NP
     / | \                          / | \
    NP VP  .                       D  N' NP*
    ↓   |                          |   |   |
       V'                         the N'  who VP
        |                              |      |
       V                              N'      V'
        |                              |       |
      sleeps                          man      V
                                          |
                                        loves
```

```haskell
-- TAG的Haskell实现
data NodeLabel = NonTerm String
               | Term String
               | FootNode String    -- 足迹节点 X*
               | SubstNode String   -- 替换节点 X↓
               deriving (Eq, Show)

data Tree a = Node a [Tree a]
            | Leaf a
            deriving (Show)

type TAGTree = Tree NodeLabel

-- TAG文法
data TAG = TAG {
    tagNonTerms :: [String],
    tagTerms    :: [String],
    tagStart    :: String,
    tagInitial  :: [TAGTree],  -- I: 初始树
    tagAuxiliary :: [TAGTree]   -- A: 辅助树
} deriving (Show)

-- 检查是否为初始树
isInitialTree :: TAGTree -> Bool
isInitialTree tree = case tree of
    Node (NonTerm _) children -> all validChild children
    _ -> False
  where
    validChild (Leaf (Term _)) = True
    validChild (Leaf (SubstNode _)) = True
    validChild (Node (NonTerm _) cs) = all validChild cs
    validChild _ = False

-- 检查是否为辅助树
isAuxiliaryTree :: TAGTree -> Bool
isAuxiliaryTree tree = case tree of
    Node (NonTerm rootLabel) children ->
        any (isFootNode rootLabel) (frontier tree)
    _ -> False
  where
    isFootNode label (Leaf (FootNode f)) = label == f
    isFootNode _ _ = False

    frontier (Leaf x) = [Leaf x]
    frontier (Node _ cs) = concatMap frontier cs
```

### 14.1.4.2.2 邻接操作

> **定义 14.1.4.4**（邻接 Adjoining）
> 设 β 是一个辅助树，其根节点和足迹节点标记为 X，
> τ 是一个内部节点标记为 X 的树。
> **邻接**操作 Adj(τ, β, n) 将 β 插入到 τ 的节点 n 处：
>
> 1. 从 τ 中移除以 n 为根的子树 τ/n（称为**切除**）
> 2. 将 β 插入 n 的位置
> 3. 将 τ/n 连接到 β 的足迹节点处

```
邻接操作示意：

    树 τ                辅助树 β            结果 Adj(τ, β)
      X                    X                    X
     / \                  / \                  /|\
    A   Y      +         B  X*       =        B X' Y
       / \                  |                  |    |
      Z1  Z2                C                  C    Z1
                                              |
                                             Z2
```

### 14.1.4.2.3 替换操作

> **定义 14.1.4.5**（替换 Substitution）
> 设 τ 是一个包含叶子节点 n 标记为 X↓（替换节点）的树，
> γ 是一个初始树（或派生树），其根节点标记为 X。
> **替换**操作将 γ 连接到 n 处，替换 X↓ 标记。

```haskell
-- 邻接操作
data DerivationTree = DT {
    dtTree :: TAGTree,
    dtDerivation :: [DerivationStep]
} deriving (Show)

data DerivationStep = Adjunction TAGTree NodePath
                    | Substitution TAGTree NodePath
                    deriving (Show)

type NodePath = [Int]  -- 从根到节点的路径

-- 在指定路径执行邻接
adjoin :: TAGTree -> TAGTree -> NodePath -> Maybe TAGTree
adjoin target aux path = do
    node <- getNodeAt path target
    rootLabel <- getRootLabel aux
    footLabel <- getFootLabel aux
    guard (nodeLabel node == rootLabel && rootLabel == footLabel)
    let (before, after) = removeSubtree target path
    let auxWithSubtree = attachAtFoot after aux
    return (combine before auxWithSubtree)

-- 获取节点
getNodeAt :: NodePath -> TAGTree -> Maybe TAGTree
getNodeAt [] tree = Just tree
getNodeAt (i:is) (Node _ cs) | i < length cs = getNodeAt is (cs !! i)
getNodeAt _ _ = Nothing

-- 获取根标签
getRootLabel :: TAGTree -> Maybe NodeLabel
getRootLabel (Node label _) = Just label
getRootLabel (Leaf label) = Just label

-- 获取足迹标签
getFootLabel :: TAGTree -> Maybe String
getFootLabel (Node _ children) =
    case filter isFoot (concatMap frontier children) of
        [Leaf (FootNode f)] -> Just f
        _ -> Nothing
  where
    isFoot (Leaf (FootNode _)) = True
    isFoot _ = False
    frontier t@(Leaf _) = [t]
    frontier (Node _ cs) = concatMap frontier cs

-- 辅助函数占位符
removeSubtree :: TAGTree -> NodePath -> (TAGTree, TAGTree)
removeSubtree = undefined

attachAtFoot :: TAGTree -> TAGTree -> TAGTree
attachAtFoot = undefined

combine :: TAGTree -> TAGTree -> TAGTree
combine = undefined

nodeLabel :: TAGTree -> NodeLabel
nodeLabel (Node l _) = l
nodeLabel (Leaf l) = l
```

## 14.1.4.3 派生树与派生

### 14.1.4.3.1 派生树

> **定义 14.1.4.6**（派生树 Derivation Tree）
> 派生树记录了树邻接和替换操作的历史。它是一个有序树，其中：
>
> - 节点标记为 (τ, p)，τ 是基本树，p 是操作位置
> - 子节点对应邻接或替换的树

```
派生树示例：

           (α_s, root)
            /       \
           /         \
      (β_np, NP)  (β_vp, VP)
         |              |
      (α_np, ε)    (α_vp, ε)

表示：
1. 从初始树 α_s 开始
2. 在NP节点邻接辅助树 β_np
3. 在VP节点邻接辅助树 β_vp
4. β_np 的替换节点被 α_np 替换
5. β_vp 的替换节点被 α_vp 替换
```

### 14.1.4.3.2 派生树语言

> **定义 14.1.4.7**（TAG生成的语言）
> TAG文法 G 生成的语言 L(G) 是：
> L(G) = { w ∈ Σ* | 存在初始树 α ∈ I 和派生 D，
> 使得 D(α) 的前缀（frontier）等于 w }

**示例**：生成 "the man that John loves sleeps"

```
初始树 α_s:              辅助树 β_rel:
      S                        NP
     / \                      / | \
    NP  VP                   D  N' NP*
    ↓   |                    |   |   |
       V'                   the N'  that S
        |                        |      |
       V                        N'      NP↓ VP
        |                        |       ↓   |
      sleeps                    N        .   V'
                                 |           |
                                man          V
                                              |
                                            loves

派生过程：
1. α_s 的NP节点邻接 β_rel
2. β_rel 的NP*处连接原NP子树（空，仅替换节点）
3. β_rel 的S↓被 "John loves" 替换
```

```haskell
-- 构建派生树
data DerivationNode = DerivationNode {
    dnTree :: TAGTree,
    dnPosition :: Maybe NodePath,
    dnChildren :: [(Operation, DerivationNode)]
} deriving (Show)

data Operation = AdjOp TAGTree
               | SubstOp TAGTree
               deriving (Show)

-- 派生树到导出树（派生结果）
derivedTree :: DerivationNode -> TAGTree
derivedTree (DerivationNode tree Nothing []) = tree
derivedTree (DerivationNode tree (Just pos) ops) =
    foldl applyOp tree ops
  where
    applyOp t (AdjOp aux, child) =
        case adjoin t aux pos of
            Just t' -> attach child t'
            Nothing -> t
    applyOp t (SubstOp _, child) =
        attach child t

    attach child tree = undefined  -- 在替换节点处连接

-- 获取前沿（终结符串）
frontierString :: TAGTree -> String
frontierString (Leaf (Term t)) = t ++ " "
frontierString (Leaf _) = ""  -- 足迹或替换节点不贡献终结符
frontierString (Node _ cs) = concatMap frontierString cs
```

## 14.1.4.4 形式性质

### 14.1.4.4.1 弱上下文敏感性

**定理 14.1.4.1**（TAG的生成能力）
TAG严格强于CFG但弱于CSG。具体而言：

1. TAG可以生成 { aⁿbⁿcⁿ | n ≥ 0 }
2. TAG不能生成 { ww | w ∈ {a,b}* }
3. 若 L 是TAG语言，则 { ww | w ∈ L } 不一定是TAG语言

**证明**：
对于 { aⁿbⁿcⁿ }：

```
辅助树 β:
      S
     /|\
    a S* c
      |
      b

初始树: a b c
通过n次邻接 β 到 S*，生成 aⁿbⁿcⁿ
```

### 14.1.4.4.2 形式语言包含关系

```
CFG ⊂ TAG ⊂ Indexed Grammar ⊂ CSL
     ↓
   LIG (Linear Indexed Grammar)
     ↓
   TAG 的多项式可解析性
```

### 14.1.4.4.3 可解析性

**定理 14.1.4.2**（TAG的解析复杂度）
TAG可以在 O(n⁶) 时间内解析，其中 n 是输入串长度。

**证明概要**：
使用Vijay-Shanker & Joshi（1985）的算法：

1. 将TAG转换为等价的LIG
2. 使用动态规划算法，状态为 (i, j, k, l, m, n)
3. 表示子串 [i,j] 到 [k,l] 的推导

```haskell
-- TAG解析器（简化版）
type Position = Int
type Item = (TAGTree, Position, Position)  -- 树，开始，结束

data ParseState = ParseState {
    items :: [Item],
    completed :: Map (String, Position, Position) TAGTree
}

-- Earley风格TAG解析
parseTAG :: TAG -> String -> [TAGTree]
parseTAG grammar input =
    let words = words input
        n = length words
        initState = initialize grammar words
        finalState = iterateUntilFixpoint (processState grammar words) initState
    in [tree | (tree, 0, n) <- items finalState]

initialize :: TAG -> [String] -> ParseState
initialize grammar words = ParseState {
    items = [(tree, 0, 0) | tree <- tagInitial grammar],
    completed = Map.empty
}

processState :: TAG -> [String] -> ParseState -> ParseState
processState grammar words state =
    let newByPrediction = predict grammar state
        newByScanning = scan words state
        newByCompletion = complete state
    in state { items = nub (items state ++ newByPrediction ++ newByScanning ++ newByCompletion) }

predict :: TAG -> ParseState -> [Item]
predict grammar state = undefined  -- 预测新初始树或辅助树

scan :: [String] -> ParseState -> [Item]
scan words state = undefined  -- 匹配终结符

complete :: ParseState -> [Item]
complete state = undefined  -- 完成替换/邻接

iterateUntilFixpoint :: Eq a => (a -> a) -> a -> a
iterateUntilFixpoint f x =
    let x' = f x
    in if x == x' then x else iterateUntilFixpoint f x'
```

## 14.1.4.5 词汇化TAG（LTAG）

### 14.1.4.5.1 词汇化原理

> **定义 14.1.4.8**（词汇化TAG）
> 词汇化TAG（Lexicalized TAG, LTAG）是每个基本树都包含至少一个词汇锚点（lexical anchor）的TAG。

**词汇树示例**：

```
loves:                    thinks:
      VP                         VP
     /  \                       /  \
    V    NP                    V    S
    |     ↓                    |     ↓
  loves                       thinks
```

### 14.1.4.5.2 特征统一

LTAG通过特征结构实现更精细的语法控制：

```haskell
-- 特征结构
data FeatureValue = FAtom String
                  | FVar String
                  | FComplex (Map String FeatureValue)
                  deriving (Show)

data FeatureStructure = FS {
    fsType :: String,
    fsAttributes :: Map String FeatureValue
} deriving (Show)

-- 特征统一
unify :: FeatureValue -> FeatureValue -> Maybe FeatureValue
unify (FVar x) y = Just y  -- 变量被实例化
unify x (FVar y) = Just x
unify (FAtom a) (FAtom b) | a == b = Just (FAtom a)
                          | otherwise = Nothing
unify (FComplex m1) (FComplex m2) =
    FComplex <$> sequence (Map.unionWith unifyVal (Just <$> m1) (Just <$> m2))
  where
    unifyVal (Just x) (Just y) = unify x y
    unifyVal x Nothing = x
    unifyVal Nothing y = y
unify _ _ = Nothing

-- 带特征的词汇树
data LexTree = LexTree {
    ltAnchor :: String,
    ltTree :: TAGTree,
    ltTopFeatures :: Map String FeatureStructure,  -- 根节点特征
    ltBotFeatures :: Map String FeatureStructure   -- 替换/足迹节点特征
} deriving (Show)
```

## 14.1.4.6 语义解释

### 14.1.4.6.1 TAG的语义组合

TAG的语义解释可以通过**派生树**进行组合：

```haskell
-- TAG语义组合
data SemExpr = SVar String
             | SLam String SemExpr
             | SApp SemExpr SemExpr
             | SConst String
             deriving (Show)

-- 词汇语义
lexicalSemantics :: String -> TAGTree -> SemExpr
lexicalSemantics "john" _ = SConst "john"
lexicalSemantics "loves" tree =
    SLam "y" (SLam "x" (SApp (SApp (SConst "love") (SVar "x")) (SVar "y")))
lexicalSemantics "sleeps" _ = SLam "x" (SApp (SConst "sleep") (SVar "x"))
lexicalSemantics "every" tree =
    SLam "P" (SLam "Q" (SApp (SApp (SConst "every") (SVar "P")) (SVar "Q")))
lexicalSemantics _ _ = SConst "unknown"

-- 语义组合（按派生树结构）
composeSemantics :: DerivationNode -> SemExpr
composeSemantics (DerivationNode tree _ []) =
    case getAnchor tree of
        Just word -> lexicalSemantics word tree
        Nothing -> SConst "empty"
composeSemantics (DerivationNode tree _ children) =
    let baseSem = case getAnchor tree of
            Just word -> lexicalSemantics word tree
            Nothing -> SConst "empty"
        childSems = map (composeSemantics . snd) children
    in foldl applyArg baseSem childSems
  where
    applyArg f a = SApp f a
    getAnchor (Leaf (Term t)) = Just t
    getAnchor (Node _ cs) = listToMaybe (mapMaybe getAnchor cs)
    getAnchor _ = Nothing
```

## 14.1.4.7 TAG与自然语言

### 14.1.4.7.1 长距离依存

TAG通过**提取树**（extraction tree）处理wh-移位：

```
what-thinks树：
      S
     / \
    NP  VP
    |    |
  what  thinks
         |
         S'
         |
         ...

通过递归邻接，可以生成长距离wh-问句：
"What does John think Mary believes Bill saw?"
```

### 14.1.4.7.2 照应语约束

TAG的局部性限制自然地解释了照应语约束：

```
*Himself_i thinks that John_i is smart

himself所在树的局部域不包含John，
违反了约束原则A的局部性要求
```

## 14.1.4.8 总结

| 特性 | CFG | TAG |
|------|-----|-----|
| 基本操作 | 重写规则 | 树邻接/替换 |
| 生成能力 | 上下文无关 | 轻度上下文敏感 |
| 长距离依存 | 有限处理 | 自然处理 |
| 解析复杂度 | O(n³) | O(n⁶) |
| 结构描述 | 字符串推导 | 树推导 |
