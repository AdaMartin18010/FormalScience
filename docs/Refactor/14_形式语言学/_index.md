# 14 形式语言学模块索引

> **模块**: 形式语言学
> **描述**: 语言的形式理论，从形式语法到语义学，涵盖音系学与计算语言学
> **文件数**: 13
> **状态**: 📝 内容填充中

---

## 📋 内容概述

本模块系统介绍形式语言学的理论与方法，从Chomsky语法层次出发，深入形式语义学的逻辑基础，涵盖音系学形式理论与计算语言学算法。强调语言结构的形式化刻画与计算实现。

### 核心主题

- **形式语法**: Chomsky层次、转换语法、范畴语法、树邻接语法
- **形式语义学**: Montague语法、类型逻辑语法、话语表示理论、博弈论语义学
- **音系学形式理论**: 生成音系学、优选论
- **计算语言学**: 形式语言与自动机、句法分析算法、语义解析
