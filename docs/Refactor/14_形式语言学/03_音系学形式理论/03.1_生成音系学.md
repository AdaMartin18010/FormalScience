# 14.3.1 生成音系学

## 14.3.1.1 引言

生成音系学（Generative Phonology）是Noam Chomsky和Morris Halle在《The Sound Pattern of English》（SPE, 1968）中建立的音系学理论框架。
该理论将音系视为语法的一个组件，通过形式规则将底层音位表示映射到表层语音实现。

> **核心思想**：音系表达分为底层形式（Underlying Representation, UR）和表层形式（Surface Representation, SR），通过有序的音系规则序列连接。

## 14.3.1.2 区别特征系统

### 14.3.1.2.1 Jakobsonian特征

> **定义 14.3.1.1**（区别特征）
> 区别特征是语音的最小对立单位，每个音段可表示为特征值的向量。

**主要特征类别**：

| 特征 | 描述 | 示例 |
|------|------|------|
| [±syllabic] | 音节核 | /i/, /a/ = [+syll]；/s/, /t/ = [-syll] |
| [±consonantal] | 辅音性 | /p/, /t/ = [+cons]；/i/, /u/ = [-cons] |
| [±sonorant] | 响音性 | /m/, /n/, /l/, /r/, /i/, /u/ = [+son] |
| [±voiced] | 浊音性 | /b/, /d/, /g/ = [+voice]；/p/, /t/, /k/ = [-voice] |
| [±nasal] | 鼻音性 | /m/, /n/, /ŋ/ = [+nasal] |
| [±continuant] | 延续性 | /f/, /s/ = [+cont]；/p/, /t/ = [-cont] |
| [±anterior] | 前部性 | /t/, /n/ = [+ant]；/k/, /ŋ/ = [-ant] |
| [±coronal] | 舌冠性 | /t/, /s/ = [+cor]；/p/, /k/ = [-cor] |
| [±high] | 高 | /i/, /u/ = [+high]；/a/, /o/ = [-high] |
| [±low] | 低 | /a/, /æ/ = [+low]；/i/, /u/ = [-low] |
| [±back] | 后 | /u/, /o/ = [+back]；/i/, /e/ = [-back] |
| [±round] | 圆唇 | /u/, /o/ = [+round]；/i/, /a/ = [-round] |
| [±tense] | 紧 | /i/, /u/ = [+tense]；/ɪ/, /ʊ/ = [-tense] |

### 14.3.1.2.2 特征几何

Clements（1985）提出了特征的分层组织——**特征几何**（Feature Geometry）。

> **定义 14.3.1.2**（特征树）
> 特征按层级组织为树结构：
>
> - **根节点**：音段（Segment）
> - **层级节点**：根节点（Root）、喉节点（Laryngeal）、部位节点（Place）
> - **终端节点**：具体特征

```
特征几何结构：

        音段（Root）
           |
    ┌─────┼─────┐
    |     |     |
 喉节点   manner  部位(Place)
    |              |
 [±voice]      ┌───┼───┐
 [±sg]         |   |   |
 [±cg]      舌冠 舌体  唇
              |    |    |
            [±cor] [±high] [±round]
            [±ant] [±low]
                   [±back]

多值特征：
舌冠：[+coronal]
  └─ 舌冠前：[+distributed]
  └─ 舌尖：[-distributed]
  └─ 舌叶：[+distributed]
```

```haskell
-- 特征几何
data FeatureTree = RootNode [FeatureTree]
                 | LaryngealNode FeatureSet
                 | PlaceNode [FeatureTree]
                 | CoronalNode FeatureSet
                 | DorsalNode FeatureSet
                 | LabialNode FeatureSet
                 | MannerNode FeatureSet
                 deriving (Show)

-- 构建特征几何树
buildFeatureTree :: Segment -> FeatureTree
buildFeatureTree seg = RootNode [
    LaryngealNode $ filterFeatures [Voiced] (segFeatures seg),
    MannerNode $ filterFeatures [Syllabic, Consonantal, Sonorant, Continuant] (segFeatures seg),
    buildPlaceNode (segFeatures seg)
]

filterFeatures :: [Feature] -> FeatureSet -> FeatureSet
filterFeatures feats fs = Map.filterWithKey (\k _ -> k `elem` feats) fs

buildPlaceNode :: FeatureSet -> FeatureTree
buildPlaceNode fs = PlaceNode [
    if hasFeature Coronal fs then CoronalNode (filterFeatures [Coronal, Anterior, Distributed] fs) else CoronalNode Map.empty,
    if hasFeature High fs then DorsalNode (filterFeatures [High, Low, Back] fs) else DorsalNode Map.empty,
    if hasFeature Round fs then LabialNode (filterFeatures [Round] fs) else LabialNode Map.empty
]

hasFeature :: Feature -> FeatureSet -> Bool
hasFeature f fs = case Map.lookup f fs of
    Just Plus -> True
    _ -> False

data Distributed = Distributed | NotDistributed deriving (Show)
```

## 14.3.1.3 音系规则

### 14.3.1.3.1 规则形式

> **定义 14.3.1.3**（音系规则）
> SPE规则形式：A → B / C __ D
> 含义：A 在 C 后 D 前变为 B

```
规则形式化：

元音和谐（匈牙利语）：
[+back] → [-back] / [-back] C₀ __
（后元音在非后元音后变为非后元音）

清化（德语）：
[+voiced] → [-voiced] / __ [-voiced]
（浊音在清音前清化）

鼻化（英语）：
V → [+nasal] / __ [+nasal]
（元音在鼻音前鼻化）
```

```haskell
-- 音系规则
data PhonRule = PhonRule {
    ruleName :: String,
    ruleTarget :: FeatureSet,      -- A
    ruleChange :: FeatureSet,      -- B (变化结果)
    ruleLeft :: FeatureSet,        -- C
    ruleRight :: FeatureSet,       -- D
    ruleDirection :: Direction
} deriving (Show)

data Direction = LeftToRight | RightToLeft | Simultaneous deriving (Show)

-- 规则应用
applyRule :: PhonRule -> [Segment] -> [Segment]
applyRule rule segs = map (\(i, s) -> applyAt rule segs i s) (zip [0..] segs)

applyAt :: PhonRule -> [Segment] -> Int -> Segment -> Segment
applyAt rule segs idx seg =
    if matchesTarget (ruleTarget rule) seg &&
       matchesContext rule segs idx
    then Segment (segIPA seg) (Map.union (ruleChange rule) (segFeatures seg))
    else seg

matchesTarget :: FeatureSet -> Segment -> Bool
matchesTarget target seg = all (\(f, v) -> Map.lookup f (segFeatures seg) == Just v) (Map.toList target)

matchesContext :: PhonRule -> [Segment] -> Int -> Bool
matchesContext rule segs idx =
    let leftEnv = take idx segs
        rightEnv = drop (idx + 1) segs
        leftMatch = case Map.toList (ruleLeft rule) of
            [] -> True
            _ -> any (\s -> matchesTarget (ruleLeft rule) s) (take 1 (reverse leftEnv))
        rightMatch = case Map.toList (ruleRight rule) of
            [] -> True
            _ -> any (\s -> matchesTarget (ruleRight rule) s) (take 1 rightEnv)
    in leftMatch && rightMatch

-- 示例规则：元音鼻化
nasalization :: PhonRule
nasalization = PhonRule "Nasalization"
    (Map.fromList [(Syllabic, Plus)])  -- 目标：元音
    (Map.fromList [(Nasal, Plus)])      -- 变化：鼻化
    Map.empty                           -- 左环境：任意
    (Map.fromList [(Nasal, Plus)])      -- 右环境：鼻音
    LeftToRight

-- 示例规则：词尾清化（德语）
finalDevoicing :: PhonRule
finalDevoicing = PhonRule "Final Devoicing"
    (Map.fromList [(Voiced, Plus)])     -- 目标：浊音
    (Map.fromList [(Voiced, Minus)])    -- 变化：清化
    Map.empty                           -- 左环境：任意
    (Map.fromList [(Syllabic, Minus)])  -- 右环境：非音节（词尾或辅音）
    LeftToRight
```

### 14.3.1.3.2 规则排序

> **定义 14.3.1.4**（规则有序性）
> 音系规则按特定顺序应用，顺序影响结果。

**规则应用模式**：

- **线性（Linear）**：规则应用一次，从左到右
- **循环（Cyclic）**：规则在句法域上循环应用
- **全局（Global）**：规则在所有可能位置同时应用

```haskell
-- 有序规则系统
data RuleSystem = RuleSystem {
    rsRules :: [PhonRule],
    rsOrder :: [Int],  -- 规则应用顺序
    rsMode :: ApplicationMode
} deriving (Show)

data ApplicationMode = LinearMode | CyclicMode Int | GlobalMode deriving (Show)

-- 应用规则系统
applyRuleSystem :: RuleSystem -> [Segment] -> [Segment]
applyRuleSystem rs segs = case rsMode rs of
    LinearMode -> foldl (\s i -> applyRule ((rsRules rs) !! i) s) segs (rsOrder rs)
    CyclicMode maxIter -> applyCyclically (rsRules rs) segs maxIter 0
    GlobalMode -> applyGlobally (rsRules rs) segs

applyCyclically :: [PhonRule] -> [Segment] -> Int -> Int -> [Segment]
applyCyclically rules segs maxIter iter
    | iter >= maxIter = segs
    | otherwise =
        let newSegs = foldl (flip applyRule) segs rules
        in if newSegs == segs
           then segs
           else applyCyclically rules newSegs maxIter (iter + 1)

applyGlobally :: [PhonRule] -> [Segment] -> [Segment]
applyGlobally rules segs =
    let changes = concatMap (findChanges segs) rules
    in applyAllChanges segs changes

findChanges :: [Segment] -> PhonRule -> [(Int, Segment)]
findChanges = undefined

applyAllChanges :: [Segment] -> [(Int, Segment)] -> [Segment]
applyAllChanges = undefined
```

## 14.3.1.4 音系表达层级

### 14.3.1.4.1 底层与表层

> **定义 14.3.1.5**（UR与SR）
>
> - **底层表示（UR）**：词汇的抽象音系形式
> - **表层表示（SR）**：经过所有音系规则后的语音实现

```
英语复数示例：

UR: /kæt/ + /z/
     ↓
同化规则: z → s / [-voiced] __
     ↓
SR: [kæts]

UR: /dɔg/ + /z/
     ↓
（无规则应用）
     ↓
SR: [dɔgz]

UR: /bʊʃ/ + /z/
     ↓
插入规则: ∅ → ə / [sibilant] __ z
     ↓
SR: [bʊʃəz]
```

```haskell
-- 形态音系学：词缀与底层形式
data Morpheme = Morpheme {
    morphemeUR :: [Segment],
    morphemeGloss :: String,
    morphemeCategory :: MorphCategory
} deriving (Show)

data MorphCategory = Root | Affix AffixType deriving (Show)
data AffixType = Prefix | Suffix | Infix | Circumfix deriving (Show)

-- 英语复数规则系统
englishPluralSystem :: RuleSystem
englishPluralSystem = RuleSystem {
    rsRules = [pluralAssimilation, pluralInsertion],
    rsOrder = [0, 1],
    rsMode = LinearMode
}

pluralAssimilation :: PhonRule
pluralAssimilation = PhonRule "Plural Assimilation"
    (Map.fromList [(Syllabic, Minus), (Voiced, Plus)])  -- /z/
    (Map.fromList [(Voiced, Minus)])                     -- → /s/
    (Map.fromList [(Voiced, Minus)])                     -- / [-voiced] __
    Map.empty
    LeftToRight

pluralInsertion :: PhonRule
pluralInsertion = PhonRule "Schwa Insertion"
    Map.empty  -- 目标：音渡
    (Map.fromList [(Syllabic, Plus), (High, Minus), (Low, Minus), (Back, Minus)])  -- 插入/ə/
    (Map.fromList [(Strident, Plus)])  -- [sibilant]
    (Map.fromList [(Coronal, Plus)])   -- __ alveolar
    LeftToRight

-- 构建词汇形式
buildWord :: [Morpheme] -> RuleSystem -> [Segment]
buildWord morphs rs =
    let ur = concatMap morphemeUR morphs
    in applyRuleSystem rs ur
```

### 14.3.1.4.2 韵律层级

> **定义 14.3.1.6**（韵律层级）
> 音系单位按层级组织：
> 音段（Segment）< 莫拉（Mora）< 音节（Syllable）< 音步（Foot）< 韵律词（Prosodic Word）< 音系短语（Phonological Phrase）< 语调短语（Intonation Phrase）< 话语（Utterance）

```
韵律层级示例：

      话语
        |
    语调短语
        |
    音系短语
        |
    韵律词 "university"
        |
    ┌───┴───┬───────┐
   音步   音步    音步
  (u-ni) (ver)   (si-ty)
    |      |       |
  音节   音节    音节 音节
  (u)    (ni)   (ver) (si) (ty)
   |      |      |     |    |
莫拉 莫拉 莫拉 莫拉 莫拉 莫拉 莫拉 莫拉
 μ    μ    μ    μ    μ    μ    μ    μ
 |    |    |    |    |    |    |    |
ʌ    n    ɪ    v    ɜ    r    s    ɪ    t    i
```

```haskell
-- 韵律层级
data ProsodicUnit = SegmentUnit Segment
                  | Mora [SegmentUnit]
                  | Syllable { syllableOnset :: [SegmentUnit],
                               syllableNucleus :: [SegmentUnit],
                               syllableCoda :: [SegmentUnit] }
                  | Foot [Syllable] FootType
                  | ProsodicWord [Foot]
                  | PhonologicalPhrase [ProsodicWord]
                  | IntonationPhrase [PhonologicalPhrase]
                  | Utterance [IntonationPhrase]
                  deriving (Show)

data FootType = Trochee | Iamb | Dactyl | Anapest deriving (Show)

-- 音节划分
syllabify :: [Segment] -> [Syllable]
syllabify segs =
    let nuclei = findIndices (isVowel . segFeatures) segs
    in map (\n -> extractSyllable segs n) nuclei

isVowel :: FeatureSet -> Bool
isVowel fs = Map.lookup Syllabic fs == Just Plus &&
             Map.lookup Consonantal fs == Just Minus

extractSyllable :: [Segment] -> Int -> Syllable
extractSyllable segs nucleusIdx =
    let (before, after) = splitAt nucleusIdx segs
        onset = takeWhile (not . isVowel . segFeatures) (reverse before)
        nucleus = takeWhile (isVowel . segFeatures) (drop nucleusIdx segs)
        coda = takeWhile (isConsonant . segFeatures)
                        (drop (nucleusIdx + length nucleus) segs)
    in Syllable (map SegmentUnit (reverse onset))
                (map SegmentUnit nucleus)
                (map SegmentUnit coda)

isConsonant :: FeatureSet -> Bool
isConsonant fs = Map.lookup Consonantal fs == Just Plus

-- 音步构建
buildFeet :: [Syllable] -> [Foot]
buildFeet [] = []
buildFeet (s1:s2:ss) = Foot [s1, s2] Trochee : buildFeet ss
buildFeet [s] = [Foot [s] Trochee]
```

## 14.3.1.5 自主音段音系学

### 14.3.1.5.1 多层级表达

Goldsmith（1976）提出了**自主音段音系学**（Autosegmental Phonology），将不同特征放在不同的音层（tier）上。

> **定义 14.3.1.7**（音层）
> 音系表达包含多个并行的音层：
>
> - **骨架层（Skeleton）**：时序单位（C, V 或 X）
> - **音段层（Segmental）**：音段特征
> - **声调层（Tone）**：声调特征（H, L）
> - 其他层：鼻化、喉化等

```
多层级表达示例（声调）：

声调层：   H     L
          |     |
骨架层：   X  X  X  X
          |  |  |  |
音段层：   m  a  n  a

实现：[mána]（H调在第一个音节）

浮游声调：
声调层：   H

骨架层：   X  X  X
          |  |  |
音段层：   b  a  n

实现：[bán] 或 [bāná]（浮游声调连接）
```

```haskell
-- 自主音段音系学
data Tier a = Tier [a] deriving (Show)

data SkeletonUnit = C | V | X deriving (Show)

data Tone = H | L | M | HL | LH | Rising | Falling deriving (Show)

data AutosegmentalRep = AR {
    arSkeleton :: Tier SkeletonUnit,
    arSegments :: Tier (Maybe Segment),
    arTones :: Tier (Maybe Tone),
    arNasal :: Tier (Maybe FeatureVal),
    arAssociations :: [(Int, Int)]  -- (骨架位置, 特征位置)
} deriving (Show)

-- 关联约定（Association Convention）
associate :: AutosegmentalRep -> AutosegmentalRep
associate ar = ar { arAssociations = newAssocs }
  where
    skelLen = length (let Tier xs = arSkeleton ar in xs)
    toneLen = length (let Tier xs = arTones ar in xs)
    newAssocs = if toneLen <= skelLen
                then [(i, min i (toneLen - 1)) | i <- [0..skelLen-1]]
                else [(min i (skelLen - 1), i) | i <- [0..toneLen-1]]

-- 声调扩展（Spreading）
spreadTone :: AutosegmentalRep -> AutosegmentalRep
spreadTone ar = ar  -- 简化实现

-- 声调层构建
toneAssociation :: [Tone] -> [Segment] -> [(Int, Int)]
toneAssociation tones segs =
    let n = length segs
        m = length tones
    in if m == n
       then zip [0..n-1] [0..m-1]
       else if m < n
            then [(i, min i (m-1)) | i <- [0..n-1]]  -- 右扩展
            else [(min i (n-1), i) | i <- [0..m-1]]  -- 多调连接
```
