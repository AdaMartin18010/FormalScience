## 14.4.3.1 引言

语义解析（Semantic Parsing）是将自然语言映射到形式意义表示的过程。
本章探讨组合语义（Compositional Semantics）和语义合成（Semantic Composition）的计算实现，结合λ演算和类型理论。

## 14.4.3.2 语义表示

### 14.4.3.2.1 一阶逻辑表示

> **定义 14.4.3.1**（FOL语义）
> 自然语言的逻辑形式使用一阶谓词逻辑表示：
>
> - 常量：john, mary, book1
> - 谓词：man(x), love(x,y)
> - 量词：∀x, ∃x

```
语义表示示例：

"John loves Mary"
→ love(john, mary)

"Every man walks"
→ ∀x(man(x) → walk(x))

"A woman reads a book"
→ ∃x(woman(x) ∧ ∃y(book(y) ∧ read(x,y)))

"The king is tall"
→ ∃x(king(x) ∧ ∀y(king(y) → x=y) ∧ tall(x))
```

```haskell
-- 一阶逻辑表示
data FOL = FPred String [Term]        -- 谓词
         | FNeg FOL                  -- 否定
         | FConj FOL FOL             -- 合取
         | FDisj FOL FOL             -- 析取
         | FImp FOL FOL              -- 蕴涵
         | FForall String FOL        -- 全称量词
         | FExists String FOL        -- 存在量词
         | FEquals Term Term         -- 等式
         deriving (Show)

data Term = TVar String             -- 变元
          | TConst String           -- 常量
          | TFunc String [Term]     -- 函数
          deriving (Show)

-- 语义表示生成器
data SemRep = SemRep {
    srFormula :: FOL,
    srType :: SemanticType,
    srFreeVars :: [String]
} deriving (Show)

data SemanticType = EType | TType | Arrow SemanticType SemanticType
                  deriving (Show)

-- 标准表示
johnTerm :: Term
johnTerm = TConst "john"

maryTerm :: Term
maryTerm = TConst "mary"

lovePred :: Term -> Term -> FOL
lovePred x y = FPred "love" [x, y]

manPred :: Term -> FOL
manPred x = FPred "man" [x]

walkPred :: Term -> FOL
walkPred x = FPred "walk" [x]
```

### 14.4.3.2.2 λ表达式表示

```haskell
-- λ演算语义表示
data LambdaTerm = LVar String
                | LLam String LambdaTerm
                | LApp LambdaTerm LambdaTerm
                | LConst String
                deriving (Show)

-- 类型标注的λ项
data TypedLambda = TypedLambda {
    tlTerm :: LambdaTerm,
    tlType :: SemanticType
} deriving (Show)

-- 从λ项到FOL的翻译
translateToFOL :: LambdaTerm -> FOL
translateToFOL (LConst c) = FPred c []
translateToFOL (LApp f x) = applyPred (translateToFOL f) (translateToFOL x)
translateToFOL (LLam x body) = FForall x (translateToFOL body)  -- 简化
translateToFOL _ = FPred "unknown" []

applyPred :: FOL -> FOL -> FOL
applyPred (FPred p args) arg = FPred p (args ++ [termFromFOL arg])
applyPred _ _ = FPred "error" []

termFromFOL :: FOL -> Term
termFromFOL (FPred c []) = TConst c
termFromFOL _ = TConst "complex"
```

## 14.4.3.3 语义合成

### 14.4.3.3.1 函子应用规则

> **定义 14.4.3.2**（语义合成规则）
> 句法组合对应语义函数应用：
>
> - **FA**（前向应用）：(A→B) × A → B
> - **BA**（后向应用）：A × (A→B) → B
> - **FC**（前向组合）：(A→B) × (B→C) → (A→C)
> - **BC**（后向组合）：(B→C) × (A→B) → (A→C)

```haskell
-- 语义合成规则
data CompositionRule = FA | BA | FC | BC | PM | SM deriving (Show)

-- 合成函数
composeSemantics :: CompositionRule -> LambdaTerm -> LambdaTerm -> Maybe LambdaTerm
composeSemantics FA f x =
    case inferType f of
        Arrow a b | inferType x == a -> Just (LApp f x)
        _ -> Nothing
composeSemantics BA x f = composeSemantics FA f x
composeSemantics FC f g =
    case (inferType f, inferType g) of
        (Arrow a b, Arrow b' c) | b == b' ->
            Just (LLam "x" (LApp g (LApp f (LVar "x"))))
        _ -> Nothing
composeSemantics BC g f = composeSemantics FC f g
composeSemantics _ _ _ = Nothing

-- 类型推断
inferType :: LambdaTerm -> SemanticType
inferType (LVar x) = EType  -- 简化
inferType (LConst "john") = EType
inferType (LConst "mary") = EType
inferType (LConst "love") = Arrow EType (Arrow EType TType)
inferType (LConst "man") = Arrow EType TType
inferType (LConst "walk") = Arrow EType TType
inferType (LApp f x) = case inferType f of
    Arrow _ b -> b
    _ -> EType
inferType (LLam x body) = Arrow EType (inferType body)
```

### 14.4.3.3.2 量词提升

```haskell
-- 量词处理
data Quantifier = Every | Some | The | No deriving (Show)

-- 量词语义
quantifierMeaning :: Quantifier -> (LambdaTerm -> LambdaTerm) -> LambdaTerm -> LambdaTerm
quantifierMeaning Every restrict scope =
    let x = "x"
    in LApp (LApp (LConst "forall")
                  (LLam x (LApp (LApp (LConst "implies") (restrict (LVar x)))
                                (scope (LVar x)))))
            (LVar "dummy")

quantifierMeaning Some restrict scope =
    let x = "x"
    in LApp (LApp (LConst "exists")
                  (LLam x (LApp (LApp (LConst "and") (restrict (LVar x)))
                                (scope (LVar x)))))
            (LVar "dummy")

-- 量词提升
type Lift = LambdaTerm -> LambdaTerm

quantifierRaise :: Quantifier -> Lift -> Lift -> LambdaTerm
quantifierRaise q restrict scope = quantifierMeaning q restrict scope

-- 示例: "Every man walks"
everyManWalks :: LambdaTerm
everyManWalks =
    let restrict = \x -> LApp (LConst "man") x
        scope = \x -> LApp (LConst "walk") x
    in quantifierMeaning Every restrict scope
```

## 14.4.3.4 词汇语义

### 14.4.3.4.1 词汇条目

```haskell
-- 词汇条目
data LexicalEntry = LexicalEntry {
    leForm :: String,
    leCategory :: SyntacticCat,
    leSemType :: SemanticType,
    leMeaning :: LambdaTerm
} deriving (Show)

data SyntacticCat = S | NP | VP | V | Det | N | PP | Adj | Adv deriving (Show)

-- 词汇库
type Lexicon = [LexicalEntry]

englishLexicon :: Lexicon
englishLexicon = [
    -- 专名
    LexicalEntry "John" NP EType (LConst "john"),
    LexicalEntry "Mary" NP EType (LConst "mary"),

    -- 普通名词
    LexicalEntry "man" N (Arrow EType TType) (LConst "man"),
    LexicalEntry "woman" N (Arrow EType TType) (LConst "woman"),
    LexicalEntry "book" N (Arrow EType TType) (LConst "book"),

    -- 动词
    LexicalEntry "walks" VP (Arrow EType TType) (LConst "walk"),
    LexicalEntry "loves" V (Arrow EType (Arrow EType TType)) (LConst "love"),
    LexicalEntry "reads" V (Arrow EType (Arrow EType TType)) (LConst "read"),

    -- 限定词
    LexicalEntry "every" Det (Arrow (Arrow EType TType) (Arrow (Arrow EType TType) TType))
        (LConst "every"),
    LexicalEntry "a" Det (Arrow (Arrow EType TType) (Arrow (Arrow EType TType) TType))
        (LConst "some"),
    LexicalEntry "the" Det (Arrow (Arrow EType TType) (Arrow (Arrow EType TType) TType))
        (LConst "the"),

    -- 形容词
    LexicalEntry "tall" Adj (Arrow (Arrow EType TType) (Arrow EType TType))
        (LConst "tall")
]

-- 词汇查找
lookupLex :: String -> Lexicon -> Maybe LexicalEntry
lookupLex form lex = find (\e -> leForm e == form) lex
```

### 14.4.3.4.2 语义组合语法

```haskell
-- 语义语法规则
data SemRule = SemRule {
    srSyntax :: [SyntacticCat],
    srResult :: SyntacticCat,
    srComposition :: [LambdaTerm] -> Maybe LambdaTerm
} deriving (Show)

-- 语义规则库
semRules :: [SemRule]
semRules = [
    -- S -> NP VP
    SemRule [NP, VP] S (\[np, vp] -> composeSemantics BA np vp),

    -- VP -> V NP
    SemRule [V, NP] VP (\[v, np] -> composeSemantics BA np v),

    -- NP -> Det N
    SemRule [Det, N] NP (\[det, n] -> composeSemantics FA det n),

    -- N -> Adj N
    SemRule [Adj, N] N (\[adj, n] -> composeSemantics FA adj n)
]

-- 带语义的句法树
data SemTree = SemTree {
    stCat :: SyntacticCat,
    stForm :: String,
    stSem :: LambdaTerm,
    stChildren :: [SemTree]
} deriving (Show)

-- 语义解释
interpret :: SemTree -> Maybe LambdaTerm
interpret (SemTree cat form sem []) = Just sem
interpret (SemTree cat form sem children) = do
    childSems <- mapM interpret children
    applyRule cat childSems

applyRule :: SyntacticCat -> [LambdaTerm] -> Maybe LambdaTerm
applyRule S [np, vp] = composeSemantics BA np vp
applyRule VP [v, np] = composeSemantics BA np v
applyRule NP [det, n] = composeSemantics FA det n
applyRule _ _ = Nothing
```

### 14.4.3.5.1 基于图表的解析

-- 图表语义解析
data ChartSemItem = ChartSemItem {
    csiCat :: SyntacticCat,
    csiSpan :: (Int, Int),
    csiSem :: LambdaTerm,
    csiTree :: ParseNode
} deriving (Show)

type SemChart = [[ChartSemItem]]

-- Earley风格语义解析
earleySemParse :: Lexicon -> [SemRule] -> [String] -> [LambdaTerm]
earleySemParse lexicon rules words =
    let chart = initializeSemChart lexicon words
        finalChart = processSemChart rules words chart
    in [csiSem item | item <- last finalChart, csiCat item == S]

initializeSemChart :: Lexicon -> [String] -> SemChart
initializeSemChart lexicon words =
    [[ChartSemItem (leCategory entry) (i, i) (leMeaning entry) (PLeaf word)
      | (i, word) <- zip [0..] words,
        Just entry <- [lookupLex word lexicon]]
     | _ <- [0..length words]]

processSemChart :: [SemRule] -> [String] -> SemChart -> SemChart
processSemChart = undefined  -- 类似标准Earley算法，但传递语义

## 14.4.3.6 语义合一

### 14.4.3.6.1 特征结构合一

```haskell
-- 特征结构
data FeatureValue = FAtom String
                  | FVar String
                  | FComplex (Map String FeatureValue)
                  deriving (Show)

data FeatureStructure = FS {
    fsType :: String,
    fsFeatures :: Map String FeatureValue
} deriving (Show)

-- 特征合一
unify :: FeatureValue -> FeatureValue -> Maybe FeatureValue
unify (FVar x) y = Just y  -- 变量实例化
unify x (FVar y) = Just x
unify (FAtom a) (FAtom b) | a == b = Just (FAtom a)
                          | otherwise = Nothing
unify (FComplex m1) (FComplex m2) = do
    unified <- sequence (Map.intersectionWith unify m1 m2)
    return (FComplex (Map.union unified m1 `Map.union` m2))
unify _ _ = Nothing

-- 语义特征合一
unifySem :: LambdaTerm -> LambdaTerm -> Maybe LambdaTerm
unifySem t1 t2 = if alphaEquivalent t1 t2 then Just t1 else Nothing

alphaEquivalent :: LambdaTerm -> LambdaTerm -> Bool
alphaEquivalent (LVar x) (LVar y) = x == y
alphaEquivalent (LConst c1) (LConst c2) = c1 == c2
alphaEquivalent (LApp f1 x1) (LApp f2 x2) =
    alphaEquivalent f1 f2 && alphaEquivalent x1 x2
alphaEquivalent (LLam x1 b1) (LLam x2 b2) =
    alphaEquivalent (rename x1 x2 b1) b2
alphaEquivalent _ _ = False

rename :: String -> String -> LambdaTerm -> LambdaTerm
rename old new (LVar x) | x == old = LVar new
                        | otherwise = LVar x
rename old new (LApp f x) = LApp (rename old new f) (rename old new x)
rename old new (LLam x body) | x == old = LLam x body
                             | otherwise = LLam x (rename old new body)
rename _ _ t = t
```

## 14.4.3.7 实现示例

```haskell
-- 完整示例：解析 "John loves Mary"
exampleParse :: Maybe ParseResult
exampleParse =
    let sentence = ["John", "loves", "Mary"]
        -- 构建句法树
        john = SemTree NP "John" (LConst "john") []
        mary = SemTree NP "Mary" (LConst "mary") []
        loves = SemTree V "loves" (LConst "love") []
        vp = SemTree VP "loves Mary"
             (fromJust $ composeSemantics BA (LConst "mary") (LConst "love"))
             [loves, mary]
        s = SemTree S "John loves Mary"
            (fromJust $ composeSemantics BA (LConst "john")
                       (fromJust $ composeSemantics BA (LConst "mary") (LConst "love")))
            [john, vp]
        sem = stSem s
        fol = translateToFOL sem
    in Just $ ParseResult s sem fol

-- 结果: love(john, mary)
```
