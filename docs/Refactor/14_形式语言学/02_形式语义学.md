## 1. 引言

### 1.1 语义学的形式化转向

形式语义学（Formal Semantics）运用数理逻辑和数学工具精确刻画自然语言的意义。
其核心目标是回答：**"句子在什么条件下为真？"**

**形式语义学的三个支柱**:

```
        ┌─────────────────┐
        │   形式语义学    │
        └────────┬────────┘
                 │
    ┌────────────┼────────────┐
    ▼            ▼            ▼
┌────────┐  ┌──────────┐  ┌──────────┐
│ 逻辑学 │  │  语言学   │  │ 数学/哲学 │
│(真值条件)│  │(句法结构) │  │(模型论语义)│
└────────┘  └──────────┘  └──────────┘
```

### 1.2 组合性原则

**Frege组合性原则** (Frege's Principle of Compositionality):
> 复合表达式的意义是其组成部分的意义及其组合方式的函数。

**形式化表述**:
$$[[α ★ β]] = f([[α]], [[β]])$$

其中 $[[·]]$ 是语义解释函数，$★$ 是句法组合操作，$f$ 是对应的语义组合规则。

### 1.3 语义与句法的关系

| 层面 | 表征 | 核心问题 |
|------|------|---------|
| 句法 | 句法树/派生 | 什么形式是合法的？ |
| 语义 | 逻辑形式 | 合法形式的意义是什么？ |
| 语用 | 语境/使用 | 如何使用意义进行交流？ |

**句法-语义接口**:

```
句法树 (Syntax Tree)
      │
      │  同态映射
      ▼
逻辑形式 (Logical Form)
      │
      │  模型论语义解释
      ▼
真值条件 (Truth Conditions)
```

## 2. 逻辑基础

### 2.1 命题逻辑

**定义 2.1.1** (命题逻辑语法)

- 原子命题: $p, q, r, ...$
- 逻辑连接词: $\neg$ (否定), $\wedge$ (合取), $\vee$ (析取), $\rightarrow$ (蕴含), $\leftrightarrow$ (等价)

**递归定义**:

1. 任何原子命题是合式公式
2. 若 $\phi$ 是合式公式，则 $\neg\phi$ 也是
3. 若 $\phi, \psi$ 是合式公式，则 $(\phi \wedge \psi)$, $(\phi \vee \psi)$, $(\phi \rightarrow \psi)$ 也是

**真值表**:

| $p$ | $q$ | $\neg p$ | $p \wedge q$ | $p \vee q$ | $p \rightarrow q$ |
|:---:|:---:|:---:|:---:|:---:|:---:|
| T | T | F | T | T | T |
| T | F | F | F | T | F |
| F | T | T | F | T | T |
| F | F | T | F | F | T |

### 2.2 一阶谓词逻辑

**定义 2.2.1** (一阶语言)
一阶语言 $\mathcal{L}$ 由以下组成：

- 个体常量: $a, b, c, ...$
- 个体变元: $x, y, z, ...$
- 谓词符号: $P, Q, R, ...$（带元数）
- 函项符号: $f, g, h, ...$（带元数）
- 量词: $\forall$ (全称), $\exists$ (存在)
- 逻辑连接词

**定义 2.2.2** (模型)
模型 $\mathcal{M} = \langle D, I \rangle$，其中：

- $D$：非空个体域
- $I$：解释函数，将符号映射到论域对象

**语义解释**:

- $[[a]]^\mathcal{M} = I(a) \in D$
- $[[P(t_1, ..., t_n)]]^\mathcal{M} = T$ 当且仅当 $\langle [[t_1]]^\mathcal{M}, ..., [[t_n]]^\mathcal{M} \rangle \in I(P)$
- $[[\forall x \phi]]^\mathcal{M} = T$ 当且仅当对所有 $d \in D$，$[[\phi]]^{\mathcal{M}[x/d]} = T$

**示例 2.2.1**:

```
句子: "Every student read a book"

逻辑形式: ∀x[student(x) → ∃y[book(y) ∧ read(x,y)]]

模型 M = ⟨D, I⟩:
  D = {j, m, b1, b2}
  I(student) = {j, m}
  I(book) = {b1, b2}
  I(read) = {⟨j,b1⟩, ⟨j,b2⟩, ⟨m,b1⟩}

真值条件验证:
  对所有x∈{j,m}:
    j: student(j)→∃y[book(y)∧read(j,y)] = T→T = T
    m: student(m)→∃y[book(y)∧read(m,y)] = T→T = T

  整体真值 = T
```

### 2.3 模态逻辑与可能世界语义

**定义 2.3.1** (可能世界模型)
模型 $\mathcal{M} = \langle W, R, V \rangle$，其中：

- $W$：可能世界的非空集合
- $R \subseteq W \times W$：可及关系
- $V: Prop \times W \rightarrow \{T, F\}$：赋值函数

**模态算子语义**:

- $\mathcal{M}, w \Vdash \Box \phi$ 当且仅当对所有 $w' \in W$，若 $wRw'$ 则 $\mathcal{M}, w' \Vdash \phi$
- $\mathcal{M}, w \Vdash \Diamond \phi$ 当且仅当存在 $w' \in W$，$wRw'$ 且 $\mathcal{M}, w' \Vdash \phi$

**示例 2.3.1** (模态歧义):

```
"John must leave"

道义解释 (Deontic): □leave(j)
  □ 表示"根据规则/义务"

认知解释 (Epistemic): □leave(j)
  □ 表示"根据已有知识"

不同真值条件:
  道义: 存在规则要求John离开
  认知: 根据所有可及知识状态，John离开
```

### 2.4 高阶逻辑与类型论

**定义 2.4.1** (简单类型)
类型 $\tau$ 递归定义：

- $e$：个体类型
- $t$：真值类型
- 若 $\sigma, \tau$ 是类型，则 $\langle \sigma, \tau \rangle$ 是函数类型

**语义域**:

- $D_e = D$（个体域）
- $D_t = \{T, F\}$
- $D_{\langle\sigma,\tau\rangle} = D_\tau^{D_\sigma}$（从 $D_\sigma$ 到 $D_\tau$ 的函数集）

**示例 2.4.1** (类型标注):

```
run: ⟨e,t⟩          (从个体到真值)
john: e
run(john): t

love: ⟨e,⟨e,t⟩⟩     (二元关系)
love(mary)(john): t  (John loves Mary)

student: ⟨e,t⟩
every: ⟨⟨e,t⟩,⟨⟨e,t⟩,t⟩⟩  (广义量词类型)
```

## 3. λ演算

### 3.1 λ演算的语法

**定义 3.1.1** (λ项)
λ项递归定义：

1. 变元 $x, y, z, ...$ 是λ项
2. 若 $M, N$ 是λ项，则 $(M N)$（应用）是λ项
3. 若 $M$ 是λ项，$x$ 是变元，则 $(\lambda x.M)$（抽象）是λ项

**约定**:

- 左结合: $M N P = ((M N) P)$
- 体最大化: $\lambda x. M N = \lambda x. (M N)$，但 $(\lambda x. M) N$ 是应用
- 多重抽象: $\lambda xyz.M = \lambda x.\lambda y.\lambda z.M$

### 3.2 α、β、η转换

**定义 3.2.1** (α-转换)
变元重命名：$\lambda x.M \xrightarrow{\alpha} \lambda y.M[y/x]$（$y$ 不在 $M$ 中自由出现）

**定义 3.2.2** (β-归约)
函数应用：$(\lambda x.M) N \xrightarrow{\beta} M[N/x]$

**示例 3.2.1**:

```
(λx.x+1) 5 →β 5+1 = 6
(λx.λy.x+y) 3 →β λy.3+y
(λx.λy.x+y) 3 4 →β (λy.3+y) 4 →β 3+4 = 7
```

**定义 3.2.3** (η-转换)
外延等价：$\lambda x.(M x) \xrightarrow{\eta} M$（$x$ 不在 $M$ 中自由出现）

### 3.3 类型化λ演算

**定义 3.3.1** (简单类型λ演算)
λ项的类型推导规则：

$$\frac{x : \sigma \in \Gamma}{\Gamma \vdash x : \sigma} \text{(Var)}$$

$$\frac{\Gamma, x : \sigma \vdash M : \tau}{\Gamma \vdash \lambda x.M : \sigma \rightarrow \tau} \text{(Abs)}$$

$$\frac{\Gamma \vdash M : \sigma \rightarrow \tau \quad \Gamma \vdash N : \sigma}{\Gamma \vdash M N : \tau} \text{(App)}$$

**Curry-Howard对应**:

```
逻辑                     类型系统
────────────────────────────────────────
命题 φ                   类型 τ
证明 φ→ψ                 函数 τ→σ
证明 φ∧ψ                 对 (τ,σ)
证明 φ∨ψ                 和类型 τ|σ
矛盾 ⊥                   空类型
```

### 3.4 在语义学中的应用

**示例 3.4.1** (形容词修饰):

```
red: λP.λx.red(x) ∧ P(x)  (交集性形容词)
former: λP.λx.former(P)(x)  (非交集性)

red ball: (λP.λx.red(x)∧P(x))(ball)
         →β λx.red(x)∧ball(x)
```

**示例 3.4.2** (关系从句):

```
that runs: λP.λx.P(x) ∧ run(x)
          : ⟨⟨e,t⟩,⟨e,t⟩⟩

student that runs: (λP.λx.P(x)∧run(x))(student)
                  →β λx.student(x)∧run(x)
```

## 4. Montague语法

### 4.1 Montague形式主义

**核心思想**: 自然语言是形式语言的" disguised version "，可直接用形式语义学处理。

**Montague文法结构**:

```
句法规则: Fᵢ(α₁, ..., αₙ) = β
语义解释: [[β]] = Gᵢ([[α₁]], ..., [[αₙ]])
```

**PTQ系统** (Proper Treatment of Quantification):

- 英语片段的形式化
- 直接生成内涵逻辑表达式

### 4.2 PTQ系统详解

**基本范畴**: $t$ (句子), $e$ (实体), $CN$ (普通名词), $T$ (词项)

**句法规则示例**:

```
S1:  F₁(α,δ) = αδ'
    若α∈P(词项), δ∈IV(不及物动词),
    则αδ'∈T(句子)
    语义: α'(δ')

S4:  F₄(α,β) = α such that β'
    若α∈CN, β∈T, 则结果∈CN
    语义: λx[α'(x) ∧ β'(x)]
```

### 4.3 类型论语义

**类型层级**:

```
类型e: 个体 (John, Mary)
类型t: 真值 (真, 假)
类型⟨e,t⟩: 一元谓词 (run, student)
类型⟨e,⟨e,t⟩⟩: 二元关系 (love, see)
类型⟨⟨e,t⟩,t⟩: 广义量词 (every student, a book)
```

**量词提升** (Quantifier Raising):

```
原始结构:
  [S [NP every student] [VP read [NP a book]]]

量化结构 (LF):
  [S [NP a book]₂ [S [NP every student]₁
      [S t₁ read t₂]]]

语义组合 (从内向外的函数应用):
  read': λy.λx.read(x,y)
  t₂: y
  t₁: x
  VP: λx.read(x,y)
  every student': λP.∀x[student(x)→P(x)]
  S': ∀x[student(x)→read(x,y)]
  a book': λQ.∃y[book(y)∧Q(y)]
  最终: ∃y[book(y)∧∀x[student(x)→read(x,y)]]
```

### 4.4 组合规则

**函数应用** (Functional Application):
$$\frac{\alpha : \sigma \rightarrow \tau \quad \beta : \sigma}{\alpha(\beta) : \tau}$$

**非分支节点** (Non-branching Nodes):
$$\frac{\alpha : \sigma}{\alpha : \sigma}$$

**谓词修饰** (Predicate Modification):
$$\frac{\alpha : \langle e,t \rangle \quad \beta : \langle e,t \rangle}{\lambda x.[\alpha(x) \wedge \beta(x)] : \langle e,t \rangle}$$

### 4.5 广义量词理论

**定义 4.5.1** (广义量词)
类型为 $\langle\langle e,t\rangle,t\rangle$ 的表达式，将谓词映射为真值。

**量词作为集合的集合**:

- $[[every]] = \lambda P.\lambda Q.P \subseteq Q$
- $[[some]] = \lambda P.\lambda Q.P \cap Q \neq \emptyset$
- $[[no]] = \lambda P.\lambda Q.P \cap Q = \emptyset$

**单调性**:

- 向上单调: $Q(A)(B) \wedge A \subseteq A' \rightarrow Q(A')(B)$
- 向下单调: $Q(A)(B) \wedge A' \subseteq A \rightarrow Q(A')(B)$

**量词关系的同构**:

```
Every A is B  ⟺  A ⊆ B
Some A is B   ⟺  A ∩ B ≠ ∅
No A is B     ⟺  A ∩ B = ∅
Most A are B  ⟸  |A ∩ B| > |A - B|
```

## 5. 类型逻辑语法

### 5.1 语法-逻辑对应

**Curry-Howard 对应在语言学中的应用**:

```
证明                     派生
─────────────────────────────────
公理                     词汇项
蕴含消除 (→E)            函数应用
蕴含引入 (→I)            类型提升/协调
合取 (∧)                 协调/附加
```

**自然演绎格式**:

```
     [x:A]¹
       ⋮
       M:B
  ─────────── (→I, 1)
  λx.M : A→B

  M : A→B    N : A
  ───────────────── (→E)
        M N : B
```

### 5.2 多模态类型逻辑

**结构规则控制**:

- 结合性: $(A \bullet B) \bullet C \leftrightarrow A \bullet (B \bullet C)$
- 交换性: $A \bullet B \leftrightarrow B \bullet A$
- 弱中心化: $A \bullet B \rightarrow B \bullet A$（受限）

**示例 5.2.1** (非结合Lambek演算):

```
处理无括号结构:

句子结构: (John (sees Mary)) 而非 ((John sees) Mary)

规则限制:
  右应用仅当右侧是完整成分
  A/B  B ⊢ A  仅当 B 是饱和的
```

### 5.3 证明网络

**证明网络 (Proof Nets)** 提供无序列演算的图形化证明表示。

**示例 5.3.1**:

```
"John likes Mary"

证明网络:
  john ──► NP
            │
            ▼
  likes ──► (NP\S)/NP ──► NP\S
                              │
  mary ──► NP ────────────────►
                              │
                              ▼
                              S
```

## 6. 话语表示理论（DRT）

### 6.1 DRS的结构

**定义 6.1.1** (话语表示结构 DRS)
DRS是二元组 $\langle U, Con \rangle$，其中：

- $U$：话语参照标的集合（discourse referents）
- $Con$：条件的集合（conditions）

**DRS框表示法**:

```
┌─────────────┐
│ x y         │  ← 参照标 (U)
│             │
│ John(x)     │  ← 条件 (Con)
│ book(y)     │
│ read(x,y)   │
└─────────────┘
```

### 6.2 构造算法

**DRS构造规则** (归纳定义):

**基本规则**:

```
处理名词短语 "a N":
  引入新参照标 x
  添加条件 N(x)

处理专有名词 "John":
  引入新参照标 x
  添加条件 John(x)
  添加条件 x=j（外部锚定）
```

**合并规则**:

```
处理 "S₁ and S₂":
  K₁ + K₂ = ⟨U₁∪U₂, Con₁∪Con₂⟩

处理条件句 "if S₁ then S₂":
  创建子DRS嵌套
  K_if = ⟨{ }, {K₁ ⇒ K₂}⟩
```

### 6.3 预设处理

**预设触发语** (Presupposition Triggers):

- 定指描述: "the N" → 预设存在唯一满足N的个体
- 事实动词: "regret", "know" → 预设补语为真
- 迭代词: "too", "again" → 预设前设情境存在

**预设投射问题**:

```
句子: "If John has a cat, the cat is black"

分析:
  预设: 存在猫（在条件后件中）
  处理: 条件局部满足

DRS:
  ┌─────────────────────┐
  │ x                   │
  │ cat(x)              │
  │ John-has(x) ───────►│  ┌─────────────┐
  │                     │  │ y           │
  │                     │  │ y=x         │ (等同)
  │                     │  │ black(y)    │
  │                     │  └─────────────┘
  └─────────────────────┘
```

### 6.4 回指解析

**回指规则** (Accessibility):

- 参照标在DRS中可被后续话语访问
- 子DRS中的参照标对母DRS不可访问（反之可）

**示例 6.4.1**:

```
话语: "A man came in. He smiled."

DRS 1 (第一句):
  ┌─────────────┐
  │ x           │
  │ man(x)      │
  │ came-in(x)  │
  └─────────────┘

DRS 2 (整合第二句):
  ┌─────────────┐
  │ x           │
  │ man(x)      │
  │ came-in(x)  │
  │ smiled(x)   │ ← "he" 解析为 x
  └─────────────┘
```

**约束理论 (DRT版本)**:

```
话语: "Every man likes his mother"

┌─────────────────────────────────────┐
│                                     │
│ ┌─────────────┐  ┌─────────────┐   │
│ │ x           │  │ y z         │   │
│ │ man(x)      │⇒ │ mother(z,x) │   │
│ │             │  │ like(y,x,z) │   │
│ └─────────────┘  └─────────────┘   │
│                                     │
└─────────────────────────────────────┘

"his" 绑定于条件前件引入的x
```

### 6.5 动态语义学

**动态意义观**: 句子的意义是输入话语语境与输出话语语境之间的关系。

**形式化**:
$$[[S]] = \{(g, h) : g \text{通过处理} S \text{转换为} h\}$$

**测试 vs 更新**:

- 测试: 仅当条件满足时保留语境
- 更新: 添加新信息到语境

## 7. 博弈论语义学

### 7.1 验证博弈

**核心思想**: 意义即验证博弈中证明者的获胜策略。

**定义 7.1.1** (逻辑博弈)
博弈 $G(\mathcal{M}, \phi, g)$，其中：

- $\mathcal{M}$：模型
- $\phi$：待验证/证伪的公式
- $g$：赋值函数

**博弈规则**:

```
位置 (φ, g):

原子: P(t₁,...,tₙ)
  - 若为真，证明者胜；否则反驳者胜

否定: ¬φ
  - 玩家角色互换，继续 (φ, g)

合取: φ ∧ ψ
  - 反驳者选择左或右子博弈

析取: φ ∨ ψ
  - 证明者选择左或右子博弈

全称: ∀xφ
  - 反驳者选择d∈D，继续 (φ, g[x/d])

存在: ∃xφ
  - 证明者选择d∈D，继续 (φ, g[x/d])
```

### 7.2 博弈规则与策略

**策略定义**:

- 证明者策略: 在每个证明者选择节点指定选择
- 反驳者策略: 在每个反驳者选择节点指定选择

**定理 7.2.1** (博弈语义与经典语义等价)
$\phi$ 在 $\mathcal{M}$ 中为真当且仅当证明者在 $G(\mathcal{M}, \phi)$ 中有获胜策略。

**示例 7.2.1**:

```
验证: ∀x∃y R(x,y)

博弈过程:
  反驳者选 a₁ ∈ D
  证明者选 b₁ ∈ D 使得 R(a₁,b₁)

若对所有 a₁, 证明者都能选到 b₁ 使得 R(a₁,b₁) 成立，
则证明者有获胜策略。
```

### 7.3 交互式语义

**自然语言博弈**: 对话可视为意义协商的博弈。

**示例 7.3.1** (量词博弈):

```
句子: "Every student read some book"

解读1 (∃∀):
  反驳者选学生s
  证明者选书b
  验证: read(s,b)
  （每个学生有不同的书）

解读2 (∀∃):
  证明者选书b
  反驳者选学生s
  验证: read(s,b)
  （所有学生读同一本书）

自然语言通常偏好解读1（量词提升顺序决定）
```

## 10. 应用案例

### 10.1 案例1：问答系统语义解析

**背景**: 将自然语言问题转换为数据库查询。

**示例**:

```
问题: "Which students passed the exam?"

语义解析:
  句法: [Which [NP students]] [VP passed [NP the exam]]]

  λ项: λQ.∃x[student(x) ∧ passed(x, exam) ∧ Q(x)]

  SQL: SELECT student_id FROM students
       WHERE passed = true AND exam_id = 'exam'

组合过程:
  which: λP.λQ.∃x[P(x)∧Q(x)]
  students: student
  passed the exam: λx.passed(x, e)

  组合: λQ.∃x[student(x)∧passed(x,e)∧Q(x)](λy.y)
      → ∃x[student(x)∧passed(x,e)]
```

### 10.2 案例2：机器翻译的语义保持

**背景**: 确保翻译过程中语义等价。

**方法**: 使用语义中间表示（DRS/逻辑形式）。

```
英语: "Every student read some book"
     ├─► DRS: [x][student(x)] ⇒ [y][book(y), read(x,y)]
     │
     ▼
汉语: "每个学生都读了一本书"

结构保持:
  ∀>∃ 辖域关系在DRS中显式编码
  翻译系统保留量化结构
```

### 10.3 案例3：语义角色标注系统

**背景**: 自动识别句子中的语义角色（施事、受事等）。

**形式化**:

```
句子: "John gave Mary a book"

谓词: give
角色:
  AGENT: John (施事)
  RECIPIENT: Mary (接受者)
  THEME: a book (主题)

逻辑形式: ∃x∃y∃z[John(x) ∧ Mary(y) ∧ book(z) ∧
                  give(e) ∧ agent(e,x) ∧ recipient(e,y) ∧ theme(e,z)]
```

## 附录：λ项库

### 常用词汇语义

```
个体: john, mary : e

不及物动词: run, walk : ⟨e,t⟩
  run = λx.run(x)

及物动词: love, see : ⟨e,⟨e,t⟩⟩
  love = λy.λx.love(x,y)

三价动词: give : ⟨e,⟨e,⟨e,t⟩⟩⟩
  give = λz.λy.λx.give(x,y,z)

普通名词: student, book : ⟨e,t⟩

限定词:
  every = λP.λQ.∀x[P(x)→Q(x)] : ⟨⟨e,t⟩,⟨⟨e,t⟩,t⟩⟩
  some  = λP.λQ.∃x[P(x)∧Q(x)] : ⟨⟨e,t⟩,⟨⟨e,t⟩,t⟩⟩
  the   = λP.ιx[P(x)] : ⟨⟨e,t⟩,e⟩

形容词:
  red = λP.λx.red(x)∧P(x) : ⟨⟨e,t⟩,⟨e,t⟩⟩

介词:
  in = λy.λx.in(x,y) : ⟨e,⟨e,t⟩⟩
```
