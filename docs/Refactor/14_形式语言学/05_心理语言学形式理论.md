## 1. 引言

### 1.1 心理语言学的计算转向

心理语言学（Psycholinguistics）研究语言的心理现实性，即语言知识如何在大脑中表征、语言如何被理解和产生。
形式理论为心理语言学提供了可计算、可验证的模型框架。

**核心问题**:

```
1. 语言习得: 儿童如何在有限输入下习得复杂语言系统？
2. 语言理解: 听者如何实时处理连续语音流？
3. 语言产生: 说话者如何将思想转化为语言？
4. 语言表征: 语言知识在大脑中如何存储？
```

### 1.2 形式模型 vs 处理模型

| 维度 | 形式模型 (如GB/MP) | 处理模型 (如ACT-R) |
|------|-------------------|-------------------|
| 目标 | 描述语言能力 | 描述语言使用 |
| 表征 | 抽象结构 | 心理表征 |
| 时间 | 无时态 | 实时动态 |
| 限制 | 理论约束 | 认知资源限制 |
| 可证伪性 | 语法判断 | 反应时间、眼动等 |

### 1.3 语言能力的表征

**三成分模型**:

```
语言能力 (Competence)
    ├── 句法知识 ───► 形式文法理论
    ├── 语义知识 ───► 模型论语义
    └── 词汇知识 ───► 词汇表征理论

语言使用 (Performance)
    ├── 理解机制 ───► 句法分析模型
    ├── 产生机制 ───► 语言产生模型
    └── 学习机制 ───► 习得模型
```

## 2. 语言习得的形式理论

### 2.1 习得逻辑问题（Plato's Problem）

**核心问题**: 儿童如何在有限、有缺陷的输入（刺激贫乏）下习得复杂的语言能力？

**形式化表述**:

```
给定:
  - 输入语料 D (有限、含噪音)
  - 目标语法 G_target (生成无限语言)

寻找:
  - 学习算法 L: D → G

约束:
  - L 在合理时间内收敛到 G_target
  - L 使用人类儿童可及的认知资源
```

**Gold定理** (语言习得理论的经典负结果):
> 在仅基于正例（文本呈现）且要求极限识别精确语法的条件下，任何包含所有有限语言和一个无限语言的语言类都是不可学习的。

**启示**: 语言习得需要额外的归纳偏置（inductive bias）。

### 2.2 原则与参数理论

**核心思想**: 普遍语法（UG）提供原则（普遍规则）和参数（语言间变异点），儿童通过设置参数习得特定语言。

**参数示例**:

| 参数 | 选项 | 语言例子 |
|------|------|---------|
| 空主语参数 | [+pro-drop] | 汉语、意大利语 (允许无主语) |
| | [-pro-drop] | 英语、法语 (需要显性主语) |
| 中心语方向 | 中心语在前 | 英语 (VP: V + NP) |
| | 中心语在后 | 日语 (VP: NP + V) |

**形式化习得模型**:

```
UG = ⟨UniversalPrinciples, Parameters⟩

习得过程:
  G₀ = 初始状态 (所有参数未设定)

  对输入句子 s ∈ D:
    若 s 与当前参数设定一致:
      保持当前设定
    若 s 与某些参数设定冲突:
      调整参数以兼容s

  收敛: 参数设定稳定
```

**触发经验 (Triggering Experience)**:

```
输入: "Giovanni parla" (意大利语)
      (Giovanni speaks)

分析: 无主语但句子完整

触发: 设置 [+pro-drop] 参数
```

### 2.3 最简方案下的习得

**核心变化**: 从原则-参数的"开关"模型转向"特征组装"模型。

**特征组装理论**:

```
词汇项由以下成分组成:
  - 语音形式 (PF可解读特征)
  - 语义形式 (LF可解读特征)
  - 不可解读特征 (uF) —— 需要一致操作删除

习得任务: 学习哪些特征组合是可解读的
```

**形式化**:

```
输入: 可解读特征集合 {iF}
输出: 词汇条目集合 {Lexᵢ}

约束:
  - 每个 Lexᵢ = ⟨PF, LF, {uF}⟩
  - 一致操作: Probe-Goal 关系删除uF
  - 成功推导: 所有uF被删除，得到完整解释
```

### 2.4 统计学习理论

**核心思想**: 儿童利用统计规律（如转换概率）从输入中提取语言结构。

**统计学习机制**:

| 类型 | 机制 | 证据 |
|------|------|------|
| 转换概率 | 音位切分 | Saffran et al. (1996) |
| 条件概率 | 词汇范畴学习 | 高预测词→功能词 |
| 共现模式 | 范畴化 | 分布相似性→同类 |

**形式化模型**:

```
隐马尔可夫模型用于词范畴学习:

状态: 词汇范畴 {N, V, Adj, Det, ...}
观测: 词汇实例

学习: 估计转移概率 P(catᵢ | catᵢ₋₁)
      和发射概率 P(word | cat)
```

**最小描述长度 (MDL) 学习**:

```
目标: 找到文法 G 最小化
      L(G) + L(D|G)

其中:
  L(G) = 编码文法的比特数
  L(D|G) = 用G编码数据的比特数

平衡: 简单文法 vs 数据拟合
```

### 2.5 联结主义模型

**核心思想**: 语言知识以分布式连接权重形式存储，学习是权重调整过程。

**简单递归网络 (SRN)**:

```
架构:
  输入层 ──► 隐藏层 ──► 输出层 (预测下一词)
     ▲          │
     └──────────┘ (上下文层，复制上一时刻隐藏层)

学习规则: 反向传播通过时间 (BPTT)

目标: 最小化预测误差
      E = Σₜ (targetₜ - outputₜ)²
```

**联结主义习得模型的预测**:

- 渐进式习得（非突变）
- 错误模式反映统计规律
- 对频率敏感
- 可解释发育性语言障碍

## 3. 语言处理的形式模型

### 3.1 实时句法分析

**理解任务的时间进程**:

```
0-200ms:  声学/视觉处理，词汇访问
200-400ms: 词汇识别，范畴分配
400-600ms: 句法结构构建
600ms+:   语义整合，语用推理
```

**左角分析器 (Left-Corner Parser)**:

```
结合自顶向下预测与自底向上验证:

规则: S → NP VP

左角分析:
  1. 遇到词 w
  2. 预测 w 可作为哪个非终结符的左角
  3. 若 w ∈ First(NP)，启动 NP 分析
  4. 完成 NP 后，期望 VP 完成 S

心理现实性: 符合人类处理逐步、增量的特点
```

### 3.2 花园路径现象

**定义**: 句法分析器最初采用错误分析路径，需要回溯重分析的现象。

**经典例子**:

```
"The horse raced past the barn fell"

初始分析 (花园路径):
  [The horse] [raced past the barn] (主谓结构)
  预期: 句子结束

遇到 "fell":
  冲突！需要重分析

正确分析:
  [The horse [raced past the barn]] fell
  (减少关系从句结构)
```

**形式化解释**:

```
最小依附原则 (Minimal Attachment):
  使用最少节点的新分析

迟关闭原则 (Late Closure):
  将新词依附到当前处理成分

这两个原则导致花园路径
```

**预测 vs 频率**:

```
模型对比:
   garden-path theory: 基于结构复杂性
   constraint-satisfaction: 基于多种信息源的竞争

眼动证据支持约束满足模型:
  - 频率信息即时影响解析
  - 语义信息早期介入
```

### 3.3 句法处理的竞争模型

**同步分析假设**: 多个分析并行激活，竞争选择。

**联系主义竞争模型**:

```
输入: 词序列 w₁, w₂, ..., wₙ

每个时刻:
  1. 激活与输入兼容的分析假设
  2. 分析间根据支持证据竞争
  3. 强分析抑制弱分析

选择: 最激活的分析
```

**surprisal 理论**:

```
单词 wₜ 的 surprisal:
  S(wₜ) = -log P(wₜ | w₁...wₜ₋₁, context)

处理难度 ∝ surprisal

预测:
  意外词 → 高 surprisal → 慢处理
  预期词 → 低 surprisal → 快处理
```

### 3.4 预测性处理

**预测编码框架**:

```
大脑作为预测机器:
  - 基于上下文生成语言输入的预测
  - 比较预测与实际输入
  - 预测误差驱动更新

层级预测编码:
  低层: 音位/字形预测
  中层: 词汇预测
  高层: 句法/语义预测
```

**N400效应的形式解释**:

```
N400 (ERP成分):
  - 语义整合难度指标
  - 振幅 ∝ 语义 surprisal

形式化:
  N400振幅 ~ -log P(semantic_features | context)

 cloze概率（完形填空概率）高度预测N400
```

## 4. 词汇访问与识别

### 4.1 词汇通达的形式模型

**词汇访问**: 从语音/字形输入到词汇表征激活的过程。

**瀑布模型**:

```
声学输入 ──► 音位识别 ──► 词汇选择 ──► 语义访问
                │
                ▼
           词汇竞争
           (cohort模型)
```

**Cohort模型**:

```
阶段1: 词初始cohort激活
  听到 /kæ/ → 激活 "cat", "can", "captain", ...

阶段2: 词汇竞争
  后续输入 /t/ → "cat" 胜出，抑制竞争者

形式化:
  候选词 w 的激活:
    A(w, t) = f(match(input(t), w), frequency(w), inhibition)
```

**Trace模型** (McClelland & Elman):

```
特征层 ──► 音位层 ──► 词汇层 ──► 语义层
   ▲          │           │
   └──────────┴───────────┘ (反馈连接)

交互激活: 自下而上输入 + 自上而下预测
```

### 4.2 歧义词处理

**歧义词类型**:

| 类型 | 例子 | 处理特点 |
|------|------|---------|
| 语音歧义 | "bank" (河岸/银行) | 语境即时消歧 |
| 词类歧义 | "race" (名/动) | 句法语境消歧 |
| 语义歧义 | "bug" (昆虫/窃听器) | 主题语境消歧 |

**歧义消解模型**:

**穷尽访问模型**:

```
所有意义并行激活，后选择
证据: 主导意义和次要意义都有启动效应
```

**顺序访问模型**:

```
主导意义先激活，需要时访问次要意义
证据: 主导意义反应时优势
```

**重排序模型**:

```
语境提高预期意义的激活
证据: 强语境下无主导意义优势
```

**形式化 (贝叶斯视角)**:

```
P(meaning | word, context) ∝
  P(word | meaning) · P(meaning | context)

选择: argmax_{meaning} P(meaning | word, context)
```

### 4.3 词汇决策模型

**词汇决策任务**: 判断字母串是否为真词。

**扩散模型 (Diffusion Model)**:

```
证据累积过程:
  dX = v·dt + s·dW

其中:
  v: 漂移率 (证据质量)
  s: 扩散系数 (噪声)

决策: 达到上阈值 → "词"
      达到下阈值 → "非词"

预测: RT ~ 证据强度、阈值设置
```

## 5. 句子理解与加工

### 5.1 句法加工的复杂性度量

**依存局域性理论 (DLT)**:

```
加工负荷 ∝ 整合新词所需保持的依存数量

成本函数:
  IntegrationCost(wᵢ) = Σ_{d ∈ D(wᵢ)} distance(d)

其中 D(wᵢ) 是 wᵢ 需要建立的依存关系集合
```

**示例**:

```
"The rat the cat the dog chased killed ate the malt"

高嵌入导致多重依存:
  rat ──► ate
  cat ──► killed
  dog ──► chased

每个词需要保持多个未闭合依存，负荷极高
```

### 5.2 依存距离理论

**依存距离假设**:
> 语言倾向于最小化依存距离（词语与其句法头之间的距离）。

**形式化**:

```
依存距离:
  DD(w) = |position(w) - position(head(w))|

句子总代价:
  DD(S) = Σ_{w ∈ S} DD(w)

跨语言证据:
  - 语言DD均值在2-3之间
  - 长距离依存使用句法标记降低处理负担
```

**依存距离最小化 (DDM)**:

```
语言演变约束:
  语言变化倾向于降低平均依存距离

解释:
  - 词序自由度与形态丰富度的权衡
  - SOV语言有丰富格标记
  - SVO语言依赖语序
```

### 5.3 期望理论

**基于经验的期望**:

```
处理难度 ∝ -log P(structure | experience)

经验来源:
  - 词汇共现频率
  - 结构频率
  - 特定动词的子范畴偏好
```

**形式化**:

```
动词子范畴偏好:
  "put": V NP PP (into the box)
  "give": V NP NP (John a book) 或 V NP PP (to John)
  "sleep": V

偏好违反 → 处理困难
```

### 5.4 基于经验的不确定性 reduction

**不确定性 reduction框架**:

```
处理目标: 最小化句法结构的不确定性

熵减少:
  H(G | w₁...wₜ₋₁) - H(G | w₁...wₜ)

其中 H 是句法结构分布的熵

预测:
  高信息词 (高熵减少) → 强ERP反应 (P600)
```

## 6. 神经计算模型

### 6.1 简单递归网络（SRN）

**架构回顾**:

```
输入层: x(t)  (当前词one-hot编码)
隐藏层: h(t) = tanh(W_{ih}·x(t) + W_{hh}·h(t-1))
输出层: y(t) = softmax(W_{ho}·h(t))

训练目标: 预测下一词
  L = -Σₜ log P(wₜ₊₁ | h(t))
```

**语言习得模拟**:

```
训练语料: 父母话语语料 ( CHILDES )
网络结构: 100-150 隐藏单元

学习结果:
  - 能够预测合理的下一词
  - 隐状态编码句法信息
  - 可模拟儿童错误模式
```

**SRN作为心理模型**:

- 增量处理
- 分布式表征
- 渐进学习
- 频率敏感

### 6.2 长短期记忆网络（LSTM）

**动机**: 解决SRN的长期依赖问题。

**LSTM单元**:

```
遗忘门: fₜ = σ(W_f·[hₜ₋₁, xₜ] + b_f)
输入门: iₜ = σ(W_i·[hₜ₋₁, xₜ] + b_i)
候选:   C̃ₜ = tanh(W_C·[hₜ₋₁, xₜ] + b_C)
细胞状态: Cₜ = fₜ * Cₜ₋₁ + iₜ * C̃ₜ
输出门: oₜ = σ(W_o·[hₜ₋₁, xₜ] + b_o)
隐藏状态: hₜ = oₜ * tanh(Cₜ)
```

**心理语言学应用**:

```
长距离依存建模:
  "The boy who the girl saw ___ smiled"

LSTM能够保持主语信息跨越从句
```

### 6.3 Transformer与注意力机制

**自注意力**:

```
Attention(Q, K, V) = softmax(QK^T / √d_k)V

在语言处理中:
  Q, K, V 都来自同一序列的不同线性变换

效果: 每个位置可"关注"序列中任何位置
```

**心理语言学启示**:

```
与人类的异同:
  同: 并行处理、长距离连接
  异: 人类是增量处理，Transformer是全序列

Transformer-XL / XLNet: 引入循环段，更接近增量处理
```

### 6.4 神经符号整合

**动机**: 结合神经网络的感知能力与符号系统的推理能力。

**神经模块网络 (NMN)**:

```
问题: "How many red cubes are left of the sphere?"

解析:
  find[sphere] →
  filter[red] →
  filter[cube] →
  relate[left] →
  count

每个模块由神经网络实现，组合由符号控制
```

**神经定理证明**:

```
目标: 可微分逻辑推理

方法:
  - 原子: 向量嵌入
  - 规则: 神经网络学习推理步骤
  - 证明: 网络前向传播
```

**心理语言学相关性**:

```
神经符号模型可解释:
  - 规则知识与统计知识的交互
  - 显性推理与隐式学习的分工
  - 语言理解与通用认知的关系
```

## 7. 发展性语言障碍

### 7.1 特定型语言障碍（SLI）

**定义**: 语言发展显著落后于其他认知能力，无明确神经或环境原因。

**形式化模型**:

```
特征缺失假设:
  SLI儿童在特征组装/核查上有困难

延长敏感期假设:
  参数设定窗口延长，习得速率减慢

统计学习缺陷:
  从输入中提取统计规律的能力受损
```

**预测与证据**:

```
形态句法障碍:
  - 时态标记错误率高
  - 一致关系困难

解释: 功能性范畴（T, C）的弱势
```

### 7.2 发展性阅读障碍

**语音缺陷假设**:

```
核心问题: 语音意识缺陷

级联效应:
  语音处理 → 字形-音位对应学习 → 阅读流畅性

形式化:
  语音表征的颗粒度粗化
  /ba/ 和 /pa/ 的区分表征弱
```

**双通路模型**:

```
词汇通路: 整词识别 (语义 → 语音)
亚词汇通路: 形音转换 (字形 → 音位 → 语音)

阅读障碍:
  亚词汇通路受损 → 依赖词汇通路
  结果: 不规则词 > 规则词, 假词困难
```

## 8. 方法对比矩阵

### 8.1 习得理论对比

## 10. 应用案例

### 10.1 案例1：计算模型预测阅读时间

**应用场景**: 网页可读性评估。

**实现**:

```python
def predict_reading_time(sentence, surprisal_model):
    """
    基于surprisal理论预测阅读时间
    """
    tokens = tokenize(sentence)
    total_surprisal = 0

    for i, token in enumerate(tokens):
        context = tokens[:i]
        prob = surprisal_model.next_word_prob(token, context)
        surprisal = -log(prob)
        total_surprisal += surprisal

    # 阅读时间 ~ surprisal + 基线时间
    reading_time = baseline_time + k * total_surprisal
    return reading_time
```

**验证**: 与眼动数据相关性 r > 0.6
