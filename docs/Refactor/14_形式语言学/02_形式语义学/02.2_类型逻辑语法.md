## 14.2.2.1 引言

类型逻辑语法（Type Logical Grammar, TLG），也称为**类型逻辑范畴语法**（Type-Logical Categorial Grammar），是将Lambek演算与类型理论相结合的语法框架。
它由Morrill（1994）、Moortgat（1997）等学者发展，为范畴语法提供了坚实的逻辑基础。

> **核心思想**：句法推导对应于类型逻辑中的证明，语义解释通过Curry-Howard同构直接获得。

## 14.2.2.2 Lambek演算的逻辑基础

### 14.2.2.2.1 逻辑重构

Lambek演算可以看作**非交换线性逻辑**的子系统。

> **定义 14.2.2.1**（Lambek演算 L）
> 相继式形式：
>
> - 结构：Γ ::= A | Γ, Γ（序列组合，非交换）
> - 判断：Γ ⇒ A（从Γ可证A）

**公理与规则**：

```
恒等:           切割:
  ----- (Id)     Γ ⇒ A    Δ, A, Δ' ⇒ B
  A ⇒ A          ------------------------ (Cut)
                 Δ, Γ, Δ' ⇒ B

右蕴涵 /R:      左蕴涵 /L:
  Γ, A ⇒ B       Γ ⇒ A    Δ, B, Δ' ⇒ C
  ---------      ------------------------
  Γ ⇒ B/A        Δ, B/A, Γ, Δ' ⇒ C

右蕴涵 \R:      左蕴涵 \L:
  A, Γ ⇒ B       Γ ⇒ A    Δ, B, Δ' ⇒ C
  ---------      ------------------------
  Γ ⇒ A\B        Δ, Γ, A\B, Δ' ⇒ C
```

```haskell
-- Lambek演算的Haskell实现
data LType = LAtom String
           | LSlash LType LType    -- A/B
           | LBackslash LType LType -- A\B
           | LProd LType LType     -- A•B (积)
           deriving (Eq, Show)

-- 结构
data Structure = StrSingle LType
               | StrComma Structure Structure
               deriving (Show)

-- 相继式
data Sequent = Sequent Structure LType deriving (Show)

-- 证明树
data Proof = Ax LType                           -- 公理
           | Cut Proof Proof LType              -- 切割
           | SlashR Proof LType LType           -- /R
           | SlashL Proof Proof LType LType     -- /L
           | BackslashR Proof LType LType       -- \R
           | BackslashL Proof Proof LType LType -- \L
           | ProdR Proof Proof LType LType      -- •R
           | ProdL Proof LType LType            -- •L
           deriving (Show)

-- 类型检查证明
validProof :: Proof -> Maybe Sequent
validProof (Ax a) = Just (Sequent (StrSingle a) a)
validProof (Cut left right a) = do
    Sequent gamma b <- validProof left
    Sequent (StrComma delta (StrSingle a') delta') c <- validProof right
    guard (a == a')
    return (Sequent (StrComma delta (StrComma gamma delta')) c)
validProof (SlashR proof a b) = do
    Sequent (StrComma gamma (StrSingle a')) c <- validProof proof
    guard (a == a')
    return (Sequent gamma (LSlash b a))
validProof (SlashL left right a b) = do
    Sequent gamma a' <- validProof left
    Sequent (StrComma delta (StrSingle b') delta') c <- validProof right
    guard (a == a' && b == b')
    return (Sequent (StrComma delta (StrComma (StrSingle (LSlash b a)) gamma) delta') c)
validProof _ = Nothing
```

### 14.2.2.2.2 Curry-Howard对应

> **定理 14.2.2.1**（Curry-Howard-Lambek对应）
> Lambek演算的证明与简单类型λ演算中的项同构：
>
> - 命题 ↔ 类型
> - 证明 ↔ λ项
> - 证明归约 ↔ β归约

```
对应关系:

Lambek        λ演算
---------------------
  A/B        B → A
  A\B        B → A
  A•B        B × A (积类型)

  Γ, A ⇒ B   Γ, x:A ⊢ t:B
  ---------  ------------
  Γ ⇒ B/A    Γ ⊢ λx.t : B → A
```

```haskell
-- Curry-Howard翻译
data LambdaTerm = LVar String
                | LLam String LType LambdaTerm
                | LApp LambdaTerm LambdaTerm
                | LPair LambdaTerm LambdaTerm
                | LFst LambdaTerm
                | LSnd LambdaTerm
                deriving (Show)

-- 从证明提取λ项
extractTerm :: Proof -> LambdaTerm
extractTerm (Ax a) = LVar "x"  -- 基础变量
extractTerm (SlashR proof a b) =
    let body = extractTerm proof
    in LLam "x" a body
extractTerm (SlashL left right a b) =
    let f = extractTerm left
        arg = extractTerm right
    in LApp (LVar "op") (LApp f arg)  -- 简化的组合
extractTerm _ = LVar "complex"

-- 语义翻译：将Lambek证明翻译为逻辑形式
toLogicalForm :: LambdaTerm -> ILTerm
toLogicalForm (LVar x) = Var x E
toLogicalForm (LLam x t body) = Lam x (toMType t) (toLogicalForm body)
toLogicalForm (LApp f a) = App (toLogicalForm f) (toLogicalForm a)
toLogicalForm (LPair a b) = undefined  -- 积类型需要额外定义

toMType :: LType -> MType
toMType (LAtom s) = case s of
    "np" -> E
    "s" -> T
    "n" -> Arrow E T
    _ -> E
toMType (LSlash a b) = Arrow (toMType b) (toMType a)
toMType (LBackslash a b) = Arrow (toMType b) (toMType a)
toMType _ = E
```

## 14.2.2.3 模态扩展

### 14.2.2.3.1 结构模态

为了控制资源的使用，TLG引入了模态算子：

> **定义 14.2.2.2**（模态算子）
>
> - ◇A（可能）：A可任意使用
> - □A（必然）：A必须恰好使用一次
> - ↑A, ↓A：域转换算子

```
模态规则:

□L:  Γ, A, Δ ⇒ B          ◇L:  Γ, Δ ⇒ B
    ------------------        ------------------
    Γ, □A, Δ ⇒ B              Γ, ◇A, Δ ⇒ B

□R:  ◇Γ ⇒ A                ◇R:  Γ, A, Δ ⇒ B
    -----------               ------------------
    ◇Γ ⇒ □A                   Γ, ◇A, Δ ⇒ B
```

```haskell
-- 模态Lambek演算
data ModalLType = MLAtom String
                | MLSlash ModalLType ModalLType
                | MLBox ModalLType      -- □A
                | MLDiamond ModalLType  -- ◇A
                | MLUp ModalLType       -- ↑A
                | MLDown ModalLType     -- ↓A
                deriving (Eq, Show)

-- 模态结构
data ModalStructure = MSAtom ModalLType
                    | MSSeq ModalStructure ModalStructure
                    | MSBox ModalStructure   -- □环境
                    | MSDiamond ModalStructure -- ◇环境
                    deriving (Show)

-- 模态转换
diamondIntro :: ModalLType -> ModalLType
diamondIntro = MLDiamond

boxElim :: ModalLType -> Maybe ModalLType
boxElim (MLBox a) = Just a
boxElim _ = Nothing

-- 域转换
up :: ModalLType -> ModalLType
up = MLUp

down :: ModalLType -> Maybe ModalLType
down (MLDown a) = Just a
down _ = Nothing
```

### 14.2.2.3.2 括号模态

括号模态用于限制结合性：

> **定义 14.2.2.3**（括号模态）
>
> - ◇^A：在特定括号域内A可用
> - □^A：跨越括号边界时A必须被使用

```
括号规则:

  Γ[(Δ)] ⇒ A          Γ, Δ, Γ' ⇒ A
  ---------------     -----------------
  Γ[◇^B Δ] ⇒ A        Γ, ◇^B Δ, Γ' ⇒ A

其中 Γ[(Δ)] 表示Δ位于括号上下文中
```

## 14.2.2.4 多模态系统

### 14.2.2.4.1 多种蕴含

Moortgat（1995）提出了**多模态系统**，包含多种蕴含：

> **定义 14.2.2.4**（多模态蕴含）
>
> - A/B：右蕴含（普通）
> - A\B：左蕴含（普通）
> - A⊙B：积
> - A•ᵢB：不同模态的积（i为模态索引）

**结构假设**：

```
A•B : 序列组合（默认）
A⊗B : 无序组合
A⊕B : 交替组合
```

```haskell
-- 多模态类型逻辑
data MMType = MMAtom String
            | MMSlash MMType Int MMType    -- A /ᵢ B
            | MMBackslash MMType Int MMType -- A \ᵢ B
            | MMProd Int MMType MMType      -- A •ᵢ B
            deriving (Eq, Show)

-- 模态索引
data ModalIdx = MPlain    -- 默认
              | MUnordered -- ⊗
              | MAlternating -- ⊕
              | MCommutative -- 交换
              | MAssociative -- 结合
              deriving (Eq, Show, Enum)

intToModal :: Int -> ModalIdx
intToModal 0 = MPlain
intToModal 1 = MUnordered
intToModal 2 = MAlternating
intToModal 3 = MCommutative
intToModal 4 = MAssociative
intToModal _ = MPlain

-- 结构组合
data MMStructure = MMSingle MMType
                 | MMComb ModalIdx MMStructure MMStructure
                 deriving (Show)

-- 结构规则
structuralRule :: ModalIdx -> MMStructure -> Maybe MMStructure
structuralRule MCommutative (MMComb _ a b) = Just (MMComb MCommutative b a)
structuralRule MAssociative (MMComb i (MMComb j a b) c) | i == j =
    Just (MMComb i a (MMComb j b c))
structuralRule MAssociative (MMComb i a (MMComb j b c)) | i == j =
    Just (MMComb i (MMComb j a b) c)
structuralRule _ _ = Nothing
```

## 14.2.2.5 词汇语法接口

### 14.2.2.5.1 词汇条目

data LexEntry = LexEntry {

lovesEntry = LexEntry "loves"

### 14.2.2.5.2 特征统一

```haskell
-- 特征统一
unifyFeatures :: [Feature] -> [Feature] -> Maybe [Feature]
unifyFeatures f1 f2 = sequence $ map (uncurry unifyF) pairs
  where
    pairs = [(lookupFeature t f1, lookupFeature t f2) | t <- featureTypes]
    featureTypes = [Num Sing, Pers Third]  -- 简化

    lookupFeature :: Feature -> [Feature] -> Maybe Feature
    lookupFeature template feats = find (sameType template) feats

    sameType (Num _) (Num _) = True
    sameType (Pers _) (Pers _) = True
    sameType (Case _) (Case _) = True
    sameType _ _ = False

    unifyF :: Maybe Feature -> Maybe Feature -> Maybe Feature
    unifyF Nothing x = x
    unifyF x Nothing = x
    unifyF (Just x) (Just y) = if x == y then Just x else Nothing
```

## 14.2.2.6 证明搜索与解析

### 14.2.2.6.1 定理证明作为解析

在TLG中，句法分析就是定理证明：

> **定义 14.2.2.5**（解析即证明）
> 句子 w₁...wₙ 是合式句子当且仅当存在证明：
> A₁, ..., Aₙ ⇒ S
> 其中 Aᵢ 是 wᵢ 的范畴。

```haskell
-- 证明搜索（解析）
type ParseGoal = (MMStructure, MMType)

prove :: [MMType] -> MMType -> [Proof]
prove gamma goal = prove' (foldr1 (MMComb MPlain) (map MMSingle gamma)) goal []

prove' :: MMStructure -> MMType -> [Proof] -> [Proof]
prove' (MMSingle a) goal proofs
    | a == goal = proofs ++ [Ax (toLType a)]
    | otherwise = []
prove' (MMComb i left right) goal proofs =
    -- 尝试左规则
    let leftResults = proveStruct left goal proofs
        -- 尝试右规则
        rightResults = proveStruct right goal proofs
        -- 尝试切割
        cutResults = [Cut p1 p2 a | a <- subformulas goal,
                                     p1 <- proveStruct left a proofs,
                                     p2 <- proveStruct right goal proofs]
    in leftResults ++ rightResults ++ cutResults

proveStruct :: MMStructure -> MMType -> [Proof] -> [Proof]
proveStruct struct goal proofs = prove' struct goal proofs

toLType :: MMType -> LType
toLType (MMAtom s) = LAtom s
toLType (MMSlash a _ b) = LSlash (toLType a) (toLType b)
toLType (MMBackslash a _ b) = LBackslash (toLType a) (toLType b)
toLType _ = LAtom "complex"

subformulas :: MMType -> [MMType]
subformulas a@(MMAtom _) = [a]
subformulas (MMSlash a _ b) = a : b : subformulas a ++ subformulas b
subformulas (MMBackslash a _ b) = a : b : subformulas a ++ subformulas b
subformulas (MMProd _ a b) = a : b : subformulas a ++ subformulas b
```

### 14.2.2.6.2 证明网络

```prolog
% Prolog: 多模态Lambek解析器

% 证明搜索
prove(Gamma, Goal, Proof) :- prove_struct(Gamma, Goal, [], Proof).

% 基本情形：单类型
prove_struct([A], A, _, ax(A)).

% /R 规则
prove_struct(Gamma, slash(B, A), Hist, slashR(Proof)) :-
    append(Gamma, [A], GammaA),
    prove_struct(GammaA, B, Hist, Proof).

% \R 规则
prove_struct(Gamma, backslash(A, B), Hist, backslashR(Proof)) :-
    append([A], Gamma, AGamma),
    prove_struct(AGamma, B, Hist, Proof).

% /L 规则
prove_struct(Gamma, Goal, Hist, slashL(Proof1, Proof2)) :-
    select(slash(B, A), Gamma, Rest),
    prove_struct([A], _, Hist, Proof1),
    prove_struct([B|Rest], Goal, Hist, Proof2).

% \L 规则
prove_struct(Gamma, Goal, Hist, backslashL(Proof1, Proof2)) :-
    select(backslash(B, A), Gamma, Rest),
    prove_struct([A], _, Hist, Proof1),
    prove_struct([B|Rest], Goal, Hist, Proof2).

% 结构规则：交换
prove_struct(Gamma, Goal, Hist, Proof) :-
    member(commutative, Hist),
    permutation(Gamma, GammaPerm),
    Gamma \= GammaPerm,
    prove_struct(GammaPerm, Goal, Hist, Proof).

% 结构规则：结合
prove_struct([prod(A, B), C|Rest], Goal, Hist, Proof) :-
    member(associative, Hist),
    prove_struct([A, prod(B, C)|Rest], Goal, Hist, Proof).
```

## 14.2.2.7 语义解释

### 14.2.2.7.1 蒙太古语义

TLG通过Curry-Howard对应直接获得蒙太古风格的语义：

```haskell
-- 语义组合
data SemValue = SEntity String
              | SProp (Entity -> Bool)
              | SFun (SemValue -> SemValue)
              | SPred (Entity -> Entity -> Bool)
              deriving (Show)

type Entity = String
type World = Int
type Assignment = Map String Entity

-- 语义求值
evaluate :: LambdaTerm -> Assignment -> World -> SemValue
evaluate (LVar x) assign _ = case Map.lookup x assign of
    Just e -> SEntity e
    Nothing -> SEntity x
evaluate (LLam x t body) assign w =
    SFun (\v -> evaluate body (Map.insert x (extractEntity v) assign) w)
  where
    extractEntity (SEntity e) = e
    extractEntity _ = "unknown"
evaluate (LApp f a) assign w =
    case evaluate f assign w of
        SFun fun -> fun (evaluate a assign w)
        _ -> SEntity "error"
evaluate _ _ _ = SEntity "unknown"

-- 类型到语义域的映射
typeToDomain :: MMType -> [SemValue]
typeToDomain (MMAtom "e") = map SEntity entities
  where entities = ["john", "mary", "book1"]
typeToDomain (MMAtom "t") = [SEntity "true", SEntity "false"]
typeToDomain (MMSlash a _ b) =
    [SFun (\x -> head (typeToDomain a)) | _ <- typeToDomain b]
typeToDomain _ = []
```

### 14.2.2.7.2 动态语义

```haskell
-- 动态语义：上下文更新
type Context = ([Entity], Map String Int)  --  discourse referents + 约束

data DRS = DRS {
    drsRefs :: [String],
    drsConds :: [DRSCond]
} deriving (Show)

data DRSCond = Eq String String
             | Pred String [String]
             | Imp DRS DRS
             | Neg DRS
             deriving (Show)

-- 从TLG证明构建DRS
toDRS :: Proof -> DRS
toDRS proof = case proof of
    Ax (LAtom "np") -> DRS ["x"] [Pred "NP" ["x"]]
    SlashR p _ _ -> toDRS p
    SlashL left right _ _ -> mergeDRS (toDRS left) (toDRS right)
    _ -> DRS [] []

mergeDRS :: DRS -> DRS -> DRS
mergeDRS (DRS refs1 conds1) (DRS refs2 conds2) =
    DRS (refs1 ++ refs2) (conds1 ++ conds2)

-- 动态更新
update :: Context -> DRS -> Maybe Context
update (refs, assign) (DRS newRefs conds) =
    let updatedRefs = refs ++ newRefs
        newAssign = foldr (\r m -> Map.insert r (length refs + 1) m) assign newRefs
    in Just (updatedRefs, newAssign)
```

## 14.2.2.8 总结

| 特征 | Lambek演算 | TLG |
|------|------------|-----|
| 逻辑基础 | 直觉主义 | 线性逻辑 |
| 结构控制 | 无 | 模态算子 |
| 结合性 | 非结合 | 可控 |
| 交换性 | 非交换 | 模态控制 |
| 语义 | Curry-Howard | Montague风格 |
| 复杂度 | 多项式 | 多项式 |
