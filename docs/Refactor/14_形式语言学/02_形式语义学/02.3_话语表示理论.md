## 14.2.3.1 引言

话语表示理论（Discourse Representation Theory, DRT）由Hans Kamp（1981）提出，旨在解决形式语义学中的一些问题，特别是无定名词短语的指称和跨句的照应用法。DRT引入了**话语表示结构**（Discourse Representation Structure, DRS）作为语义表示的中间层。

> **核心动机**：Montague语法将每个句子独立解释为真值条件，但自然语言理解需要考虑话语的上下文和动态更新。

## 14.2.3.2 基本概念

### 14.2.3.2.1 话语表示结构

> **定义 14.2.3.1**（DRS）
> 话语表示结构是一个二元组 ⟨U, Con⟩，其中：
>
> - **U**：话语指称（discourse referents）的集合，如 x, y, z
> - **Con**：条件的集合，每个条件为以下之一：
>   - 原子条件：P(x₁, ..., xₙ)
>   - 等式条件：x = y
>   - 否定条件：¬DRS'
>   - 条件DRS：DRS' → DRS''
>   - 析取条件：DRS' ∨ DRS''

```
DRS示例：

"A man walks. He sleeps."

DRS 1:            DRS 2 (组合后):
┌──────────┐      ┌─────────────┐
│ x        │      │ x           │
│ man(x)   │  +   │ man(x)      │
│ walk(x)  │      │ walk(x)     │
└──────────┘      │ sleep(x)    │
                  └─────────────┘

"x" 是话语指称，对应 "a man" 和 "he"
```

```haskell
-- DRT的Haskell实现

type DiscourseRef = String

data DRS = DRS {
    drsUniverse :: [DiscourseRef],
    drsConditions :: [DRSCondition]
} deriving (Eq, Show)

data DRSCondition = Pred String [DiscourseRef]      -- P(x,y)
                  | Equals DiscourseRef DiscourseRef -- x = y
                  | Neg DRS                           -- ¬DRS
                  | Imp DRS DRS                       -- DRS → DRS
                  | Or DRS DRS                        -- DRS ∨ DRS
                  | Diamond DRS                       -- ◇DRS (可能)
                  | Box DRS                           -- □DRS (必然)
                  deriving (Eq, Show)

-- 空DRS
emptyDRS :: DRS
emptyDRS = DRS [] []

-- 单话语指称DRS
singletonDRS :: DiscourseRef -> DRS
singletonDRS x = DRS [x] []

-- 添加条件
addCondition :: DRS -> DRSCondition -> DRS
drs@(DRS u c) `addCondition` cond = DRS u (c ++ [cond])

-- 添加话语指称
addRef :: DRS -> DiscourseRef -> DRS
drs@(DRS u c) `addRef` x = DRS (u ++ [x]) c
```

### 14.2.3.2.2 可及性

> **定义 14.2.3.2**（可及性）
> 话语指称 x 在DRS D 中是可及的，如果 x ∈ U(D)。
> 对于嵌套DRS，可及性关系递归定义：
>
> - 若 x ∈ U(D)，则 x 在 D 中可及
> - 若 x 在 D 中可及，且 D' 是 D 的直接子DRS，则 x 在 D' 中可及（除非 D' 在否定或条件前件中）

```
可及性结构：

┌──────────────────────────┐
│ x                        │  ← x, y, z 相互可及
│ man(x)                   │
│ ┌──────────────────────┐ │
│ │ y                    │ │  ← y, z 可及x
│ │ woman(y)             │ │
│ │ love(x,y)            │ │
│ │ ┌──────────────────┐ │ │
│ │ │ z                │ │ │  ← z 可及x,y
│ │ │ ¬leave(z)        │ │ │
│ │ └──────────────────┘ │ │  ← z 可及于否定外
│ └──────────────────────┘ │
└──────────────────────────┘

否定和条件前件阻断可及性向外传播
```

```haskell
-- 可及性检查
accessible :: DRS -> DiscourseRef -> Bool
accessible (DRS u _) x = x `elem` u

-- 获取所有可及的话语指称
allAccessible :: DRS -> [DiscourseRef]
allAccessible (DRS u conds) = u ++ concatMap accessibleFromCond conds
  where
    accessibleFromCond (Neg drs) = []  -- 否定阻断
    accessibleFromCond (Imp antec _) = allAccessible antec  -- 前件内可及
    accessibleFromCond (Or drs1 drs2) = allAccessible drs1 ++ allAccessible drs2
    accessibleFromCond _ = []

-- 可及性合并（DRS合并操作）
mergeDRS :: DRS -> DRS -> DRS
mergeDRS (DRS u1 c1) (DRS u2 c2) = DRS (u1 ++ u2) (c1 ++ c2)
```

## 14.2.3.3 DRS构造算法

### 14.2.3.3.1 构造规则

> **定义 14.2.3.3**（DRS构造规则）
> 从句法树到DRS的转换规则：

**名词短语规则**：

```
NP → Det N
如果 Det = "a" 或 "some"：引入新话语指称 x，添加 N(x)
如果 Det = "every"：构建条件DRS ⟨{x}, N(x)⟩ → ⟨∅, VP(x)⟩
如果 Det = "the"：引入 x，添加 N(x) 和唯一性条件
```

**代词规则**：

```
NP → Pro
选择可及的话语指称 x，添加条件 x = y（y为先行语）
```

```haskell
-- DRS构造规则
data SyntacticNode = SN String [SyntacticNode]
                   | Leaf String
                   deriving (Show)

-- 从句法树构造DRS
constructDRS :: SyntacticNode -> DRS -> DRS
constructDRS (SN "S" [np, vp]) drs =
    let (drs1, x) = processNP np drs
        drs2 = processVP vp x drs1
    in mergeDRS drs1 drs2
constructDRS (SN "S" [s1, SN "CONJ" [Leaf "and"], s2]) drs =
    let drs1 = constructDRS s1 drs
        drs2 = constructDRS s2 (mergeDRS drs drs1)
    in mergeDRS drs1 drs2
constructDRS _ drs = drs

-- 处理NP
processNP :: SyntacticNode -> DRS -> (DRS, DiscourseRef)
processNP (SN "NP" [Leaf "a", n]) drs =
    let x = freshRef drs
        drs' = addRef drs x
        drs'' = addNounCondition n x drs'
    in (drs'', x)
processNP (SN "NP" [Leaf "every", n]) drs =
    let x = freshRef drs
        antecedent = addNounCondition n x (singletonDRS x)
    in (DRS [] [Imp antecedent emptyDRS], x)  -- VP将在后续添加
processNP (SN "NP" [Leaf "the", n]) drs =
    let x = freshRef drs
        drs' = addRef drs x
        drs'' = addNounCondition n x drs'
        -- 添加唯一性条件
        drs''' = drs'' `addCondition` Pred "unique" [x]
    in (drs''', x)
processNP (SN "NP" [Leaf pronoun]) drs | isPronoun pronoun =
    -- 代词：选择可及的先行语
    case selectAntecedent pronoun drs of
        Just y -> (drs, y)
        Nothing -> let x = freshRef drs in (addRef drs x, x)
processNP _ drs = (drs, "x")

-- 处理VP
processVP :: SyntacticNode -> DiscourseRef -> DRS -> DRS
processVP (SN "VP" [v, np]) x drs =
    let (drs1, y) = processNP np drs
        vform = getVerbForm v
    in mergeDRS drs1 (DRS [] [Pred vform [x, y]])
processVP (Leaf v) x drs =
    DRS [] [Pred v [x]]
processVP _ x drs = drs

-- 添加名词条件
addNounCondition :: SyntacticNode -> DiscourseRef -> DRS -> DRS
drs `addNounCondition` (Leaf noun) x = drs `addCondition` Pred noun [x]
drs `addNounCondition` _ x = drs

-- 生成新的话语指称
freshRef :: DRS -> DiscourseRef
freshRef (DRS u _) = head [x | n <- [1..], let x = "x" ++ show n, x `notElem` u]

-- 选择先行语
selectAntecedent :: String -> DRS -> Maybe DiscourseRef
selectAntecedent pronoun drs =
    let candidates = allAccessible drs
        -- 简单的性别/数匹配
        matching = filter (matchesPronoun pronoun) candidates
    in listToMaybe matching

matchesPronoun :: String -> DiscourseRef -> Bool
matchesPronoun _ _ = True  -- 简化处理

isPronoun :: String -> Bool
isPronoun w = w `elem` ["he", "she", "it", "they", "him", "her", "them"]

getVerbForm :: SyntacticNode -> String
getVerbForm (Leaf v) = v
getVerbForm _ = "verb"
```

### 14.2.3.3.2 示例推导

```
句子: "A man loves a woman. She smiles."

步骤1: "A man loves a woman"
┌─────────────────────────────┐
│ x, y                        │
│ man(x)                      │
│ woman(y)                    │
│ love(x,y)                   │
└─────────────────────────────┘

步骤2: "She smiles"
- "She" 选择可及的 y（女性）
┌─────────────────────────────┐
│ x, y                        │
│ man(x)                      │
│ woman(y)                    │
│ love(x,y)                   │
│ y = z  (z是"she"的话语指称) │
│ smile(z)  →  smile(y)       │
└─────────────────────────────┘
```

## 14.2.3.4 DRS的真值条件

### 14.2.3.4.1 嵌入函子

> **定义 14.2.3.4**（嵌入函子）
> DRS K = ⟨U, Con⟩ 在模型 M = ⟨D, I⟩ 中为真，当且仅当存在嵌入函子 f: U → D 使得所有条件在 M 中被满足。

**形式定义**：

- f ⊨_M P(x₁, ..., xₙ) 当且仅当 ⟨f(x₁), ..., f(xₙ)⟩ ∈ I(P)
- f ⊨_M x = y 当且仅当 f(x) = f(y)
- f ⊨_M ¬K' 当且仅当不存在 f' 扩展 f 使得 f' ⊨_M K'
- f ⊨_M K₁ → K₂ 当且仅当对所有 f' 扩展 f，若 f' ⊨_M K₁ 则存在 f'' 扩展 f' 使得 f'' ⊨_M K₂

```haskell
-- 模型定义
data Model = Model {
    domain :: [Entity],
    interpretation :: Map String [[Entity]]
} deriving (Show)

type Entity = String
type Embedding = Map DiscourseRef Entity

-- 嵌入函子
data EmbeddingFunc = EmbeddingFunc {
    embMap :: Map DiscourseRef Entity,
    embDomain :: [Entity]
} deriving (Show)

-- 验证DRS
verifyDRS :: Model -> Embedding -> DRS -> Bool
verifyDRS model emb (DRS u conds) =
    all (verifyCondition model emb) conds

verifyCondition :: Model -> Embedding -> DRSCondition -> Bool
verifyCondition model emb (Pred p args) =
    case Map.lookup p (interpretation model) of
        Just tuples ->
            let entities = map (emb Map.!) args
            in entities `elem` tuples
        Nothing -> False
verifyCondition model emb (Equals x y) =
    (emb Map.! x) == (emb Map.! y)
verifyCondition model emb (Neg drs) =
    not (existsEmbedding model emb drs)
verifyCondition model emb (Imp antec consequent) =
    forallEmbeddings model emb antec (\emb' -> existsEmbedding model emb' consequent)
verifyCondition model emb (Or drs1 drs2) =
    existsEmbedding model emb drs1 || existsEmbedding model emb drs2
verifyCondition _ _ _ = False

-- 存在扩展嵌入
existsEmbedding :: Model -> Embedding -> DRS -> Bool
existsEmbedding model emb (DRS u conds) =
    let needed = filter (`Map.notMember` emb) u
        possibleEmbeddings = generateEmbeddings model emb needed
    in any (\e -> all (verifyCondition model e) conds) possibleEmbeddings

generateEmbeddings :: Model -> Embedding -> [DiscourseRef] -> [Embedding]
generateEmbeddings model emb [] = [emb]
generateEmbeddings model emb (r:rs) =
    [Map.insert r e emb' | e <- domain model,
                           emb' <- generateEmbeddings model emb rs]

-- 全称扩展嵌入
forallEmbeddings :: Model -> Embedding -> DRS -> (Embedding -> Bool) -> Bool
forallEmbeddings model emb drs prop =
    all prop (generateEmbeddings model emb (drsUniverse drs))
```

## 14.2.3.5 预设与投射问题

### 14.2.3.5.1 预设的DRS表示

> **定义 14.2.3.5**（预设）
> 预设是由特定触发词（定指描述、事实动词、迭代副词等）引入的背景条件。

```
预设触发示例：

"John stopped smoking"
预设: "John used to smoke"
断言: "John no longer smokes"

DRS表示：
┌───────────────────────────────┐
│ x                             │
│ john(x)                       │
│ ┌───────────────────────────┐ │
│ │ used-to-smoke(x)          │ │  ← 预设条件
│ └───────────────────────────┘ │
│ ¬smoke(x)                     │  ← 断言
└───────────────────────────────┘
```

```haskell
-- 预设的DRS表示
data Presupposition = Presupposition {
    presTrigger :: String,
    presDRS :: DRS,
    presAccommodation :: Bool  -- 是否允许调适
} deriving (Show)

-- 预设投射
projectPresupposition :: Presupposition -> DRS -> DRS
projectPresupposition pres drs =
    -- 尝试在主DRS中满足预设
    if canSatisfy drs (presDRS pres)
    then bindPresupposition pres drs
    else if presAccommodation pres
         then accommodatePresupposition pres drs
         else error "Presupposition failure"

canSatisfy :: DRS -> DRS -> Bool
canSatisfy context pres =
    -- 检查context是否蕴含pres
    not (null [() | _ <- generateEmbeddings undefined Map.empty (drsUniverse pres)]),
    -- 简化处理
    True

bindPresupposition :: Presupposition -> DRS -> DRS
bindPresupposition pres drs =
    -- 将预设变量绑定到可及的话语指称
    mergeDRS drs (presDRS pres)

accommodatePresupposition :: Presupposition -> DRS -> DRS
accommodatePresupposition pres drs =
    -- 将预设条件添加到主DRS
    mergeDRS drs (presDRS pres)
```

## 14.2.3.6 时态与体

### 14.2.3.6.1 时间DRS

```
时态标记：

"John walked" (过去时)
┌─────────────────────────────┐
│ x, t                        │
│ john(x)                     │
│ t < n                       │  ← t在说话时间n之前
│ walk(x,t)                   │
└─────────────────────────────┘

"John will walk" (将来时)
┌─────────────────────────────┐
│ x, t                        │
│ john(x)                     │
│ n < t                       │  ← t在说话时间n之后
│ walk(x,t)                   │
└─────────────────────────────┘
```

```haskell
-- 时间DRS
data TemporalDRS = TemporalDRS {
    tdrsRefs :: [DiscourseRef],
    tdrsTimeRefs :: [DiscourseRef],  -- 时间话语指称
    tdrsConds :: [TemporalCond]
} deriving (Show)

data TemporalCond = TPred String [DiscourseRef] DiscourseRef  -- P(x,t)
                  | Before DiscourseRef DiscourseRef           -- t1 < t2
                  | After DiscourseRef DiscourseRef            -- t1 > t2
                  | During DiscourseRef DiscourseRef           -- t1 ⊆ t2
                  | TEquals DiscourseRef DiscourseRef          -- t1 = t2
                  | Now DiscourseRef                           -- t = n
                  deriving (Show)

-- 时态标记处理
processTense :: String -> DiscourseRef -> TemporalDRS -> TemporalDRS
tprocessTense "past" eventRef tdrs =
    let nowRef = "n"
        pastCond = Before eventRef nowRef
    in tdrs { tdrsConds = tdrsConds tdrs ++ [pastCond] }
processTense "present" eventRef tdrs =
    let nowRef = "n"
        presCond = TEquals eventRef nowRef
    in tdrs { tdrsConds = tdrsConds tdrs ++ [presCond] }
processTense "future" eventRef tdrs =
    let nowRef = "n"
        futCond = After eventRef nowRef
    in tdrs { tdrsConds = tdrsConds tdrs ++ [futCond] }
processTense _ _ tdrs = tdrs
```

## 14.2.3.7 DRT与动态语义学

### 14.2.3.7.1 动态谓词逻辑（DPL）

> **定义 14.2.3.6**（DPL语义）
> DPL将公式解释为上下文更新函数：⟦φ⟧: C → ℘(C)

**DPL与DRT的关系**：

```
DPL公式 ⟷ DRS
∃xφ      ⟷ ⟨{x}, φ⟩
φ ∧ ψ    ⟷ merge(DRS(φ), DRS(ψ))
φ → ψ    ⟷ ¬(φ ∧ ¬ψ)
```

```haskell
-- 动态谓词逻辑
data DPLFormula = DExist String DPLFormula
                | DConj DPLFormula DPLFormula
                | DImp DPLFormula DPLFormula
                | DNeg DPLFormula
                | DAtom String [String]
                | DEquals String String
                deriving (Show)

-- DPL到DRS的转换
dplToDRS :: DPLFormula -> DRS
dplToDRS (DExist x phi) =
    let DRS u c = dplToDRS phi
    in DRS (x:u) c
dplToDRS (DConj phi psi) =
    mergeDRS (dplToDRS phi) (dplToDRS psi)
dplToDRS (DImp phi psi) =
    DRS [] [Imp (dplToDRS phi) (dplToDRS psi)]
dplToDRS (DNeg phi) =
    DRS [] [Neg (dplToDRS phi)]
dplToDRS (DAtom p args) =
    DRS [] [Pred p args]
dplToDRS (DEquals x y) =
    DRS [] [Equals x y]
```

## 14.2.3.8 实现示例

```haskell
-- 完整示例："Every man loves a woman"
example1 :: DRS
example1 =
    let x = "x"
        y = "y"
        -- 全称量词：every man
        antecedent = DRS [x] [Pred "man" [x]]
        -- 存在量词：a woman（在蕴涵后件中）
        consequent = DRS [y] [Pred "woman" [y], Pred "love" [x, y]]
    in DRS [] [Imp antecedent consequent]

-- 示例："If a farmer owns a donkey, he beats it"
donationSentence :: DRS
donationSentence =
    let x = "x"  -- farmer
        y = "y"  -- donkey
        z = "z"  -- he (farmer)
        w = "w"  -- it (donkey)
        -- 前件：a farmer owns a donkey
        antecedent = DRS [x, y] [Pred "farmer" [x], Pred "donkey" [y], Pred "own" [x, y]]
        -- 后件：he beats it
        consequent = DRS [] [Pred "beat" [x, y]]  -- x和y直接绑定
    in DRS [] [Imp antecedent consequent]

-- DRS输出
printDRS :: DRS -> String
printDRS (DRS u conds) =
    "┌" ++ replicate 30 '-' ++ "┐\n" ++
    concatMap (\x -> "│ " ++ x ++ replicate (29 - length x) ' ' ++ "│\n") u ++
    concatMap printCondition conds ++
    "└" ++ replicate 30 '-' ++ "┘"

printCondition :: DRSCondition -> String
printCondition (Pred p args) =
    "│ " ++ p ++ "(" ++ intercalate "," args ++ ")" ++ replicate (30 - length p - length (intercalate "," args) - 3) ' ' ++ "│\n"
printCondition (Equals x y) =
    "│ " ++ x ++ "=" ++ y ++ replicate (30 - length x - length y - 1) ' ' ++ "│\n"
printCondition (Neg drs) =
    "│ NOT:\n" ++ indentDRS (printDRS drs) ++ "\n"
printCondition (Imp d1 d2) =
    "│ IF:\n" ++ indentDRS (printDRS d1) ++ "\n" ++
    "│ THEN:\n" ++ indentDRS (printDRS d2) ++ "\n"
printCondition _ = "│ ... │\n"

indentDRS :: String -> String
indentDRS = unlines . map ("│   " ++) . lines
```

## 14.2.3.9 总结

| 特性 | Montague语义 | DRT |
|------|--------------|-----|
| 表示层 | 直接真值条件 | DRS中间层 |
| 无定名词 | 存在量词 | 话语指称 |
| 照应语 | λ抽象 | 话语指称绑定 |
| 跨句解释 | 句子组合 | DRS合并 |
| 预设 | 复杂处理 | 自然表示 |
| 动态性 | 静态 | 动态更新 |
