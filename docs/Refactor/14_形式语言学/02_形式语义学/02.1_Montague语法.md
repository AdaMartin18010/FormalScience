## 14.2.1.1 引言

Montague语法（Montague Grammar）由Richard Montague在20世纪70年代提出，是形式语义学的奠基性理论。Montague在其经典论文"The Proper Treatment of Quantification in Ordinary English"（PTQ, 1973）中展示了如何将自然语言翻译成内涵类型λ演算（Intensional Type Theory），实现句法与语义的同构映射。

> **Montague假说**：
> "There is in my opinion no important theoretical difference between natural languages and the artificial languages of logicians; indeed, I consider it possible to comprehend the syntax and semantics of both kinds of languages within a single natural and mathematically precise theory."
> — Richard Montague, 1970

## 14.2.1.2 类型理论基础

### 14.2.1.2.1 类型层次

> **定义 14.2.1.1**（Montague类型系统）
> 类型集合 T 递归定义如下：
>
> 1. e ∈ T（实体类型）
> 2. t ∈ T（真值类型）
> 3. 若 a, b ∈ T，则 ⟨a, b⟩ ∈ T（从a到b的函数类型）
> 4. 若 a ∈ T，则 ⟨s, a⟩ ∈ T（从世界/时间到a的内涵类型）

**类型记法**：

- e：实体（entity）
- t：真值（truth value）
- ⟨e, t⟩：性质（从实体到真值的函数，即集合）
- ⟨⟨e, t⟩, t⟩：广义量词（从性质到真值的函数）
- ⟨s, a⟩：a的内涵（s指"世界-时间对"）

```haskell
-- Montague类型的Haskell实现
data MType = E           -- 实体类型
           | T           -- 真值类型
           | Arrow MType MType   -- 函数类型 <a,b>
           | Intension MType     -- 内涵类型 <s,a>
           deriving (Eq, Show)

-- 类型简写
type Property = Arrow E T        -- <e,t>
type GQ = Arrow Property T       -- <<e,t>,t>
type IndividualConcept = Intension E  -- <s,e>

-- 判断是否为内涵类型
isIntensional :: MType -> Bool
isIntensional (Intension _) = True
isIntensional _ = False

-- 类型的外延
extensionOf :: MType -> MType
extensionOf (Intension a) = a
extensionOf a = a

-- 类型的内涵
intensionOf :: MType -> MType
intensionOf a = Intension a
```

### 14.2.1.2.2 内涵类型λ演算

> **定义 14.2.1.2**（IL：内涵逻辑）
> Montague的内涵逻辑是一个类型λ演算，包含：
>
> - 变元：对每个类型 a，可数无穷变元集 Varₐ
> - 项形成规则：
>   1. 变元是项
>   2. 若 α 是类型⟨a,b⟩的项，β是类型a的项，则 α(β) 是类型b的项
>   3. 若 α 是类型b的项，u是类型a的变元，则 λu.α 是类型⟨a,b⟩的项
>   4. 若 α 是类型a的项，则 ^α 是类型⟨s,a⟩的项（内涵化）
>   5. 若 α 是类型⟨s,a⟩的项，则 ˇα 是类型a的项（外延化）

```haskell
-- 内涵逻辑项
data ILTerm = Var String MType
            | App ILTerm ILTerm
            | Lam String MType ILTerm
            | Int ILTerm            -- ^α (内涵化)
            | Ext ILTerm            -- ˇα (外延化)
            | Const String MType
            | Equals ILTerm ILTerm
            deriving (Show)

-- 类型检查
typeOf :: ILTerm -> MType
typeOf (Var _ t) = t
typeOf (Const _ t) = t
typeOf (App f x) = case typeOf f of
    Arrow _ b -> b
    _ -> error "Application to non-function"
typeOf (Lam _ t body) = Arrow t (typeOf body)
typeOf (Int t) = Intension (typeOf t)
typeOf (Ext t) = case typeOf t of
    Intension a -> a
    _ -> error "Extension of non-intensional type"
typeOf (Equals x y) = T

-- β归约
betaReduce :: ILTerm -> ILTerm
betaReduce (App (Lam x _ body) arg) = subst x arg body
betaReduce (App f x) = App (betaReduce f) (betaReduce x)
betaReduce (Lam x t body) = Lam x t (betaReduce body)
betaReduce (Int t) = Int (betaReduce t)
betaReduce (Ext t) = Ext (betaReduce t)
betaReduce (Equals x y) = Equals (betaReduce x) (betaReduce y)
betaReduce t = t

-- 替换操作
subst :: String -> ILTerm -> ILTerm -> ILTerm
subst var val (Var v t) | v == var = val
                        | otherwise = Var v t
subst var val (App f x) = App (subst var val f) (subst var val x)
subst var val (Lam v t body) | v == var = Lam v t body
                             | otherwise = Lam v t (subst var val body)
subst var val (Int t) = Int (subst var val t)
subst var val (Ext t) = Ext (subst var val t)
subst var val (Equals x y) = Equals (subst var val x) (subst var val y)
subst var val t = t
```

## 14.2.1.3 PTQ语法

### 14.2.1.3.1 句法规则

PTQ定义了一组句法规则和对应的翻译规则。

> **定义 14.2.1.3**（PTQ句法范畴）
>
> - **t**：句子
> - **CN**：普通名词（common noun）
> - **T**：术语（term，主语位置）
> - **IV**：不及物动词短语（intransitive verb phrase）
> - **TV**：及物动词
> - **IAV**：IV修饰语

**核心句法规则**：

| 规则 | 句法 | 语义翻译 |
|------|------|----------|
| S1 | T IV → t | IV'(T') |
| S2 | TV T → IV | λx. TV'(x, T') |
| S3 | CN → t | ∃x[CN'(x)] |
| S4 | T CN → t | CN'(T') |
| S5 | every CN → T | λP.∀x[CN'(x) → P(x)] |
| S6 | the CN → T | λP.∃x[∀y[CN'(y) ↔ x=y] ∧ P(x)] |
| S7 | a CN → T | λP.∃x[CN'(x) ∧ P(x)] |

```
语义推导示例：

"Every man walks"

句法结构: [every man] walks
          T          IV

语义翻译:
  [every man]' = λP.∀x[man'(x) → P(x)]
  [walks]' = walk'

组合: (λP.∀x[man'(x) → P(x)])(walk')
   = ∀x[man'(x) → walk'(x)]

最终语义: "所有走路的都是男人"（逻辑等价于"所有人走路"）
```

```haskell
-- PTQ范畴
data PTQCat = T_t | T_cn | T_t' | T_iv | T_tv | T_iav | T_det
            deriving (Eq, Show)

-- PTQ词汇项
data PTQItem = PTQItem {
    ptqForm :: String,
    ptqCat :: PTQCat,
    ptqMeaning :: ILTerm
} deriving (Show)

-- 翻译函数：从PTQ到IL
translateToIL :: PTQCat -> ILTerm -> ILTerm -> ILTerm
translateToIL T_t term iv = App iv term  -- S1: IV(T)
translateToIL T_tv tv term =
    let x = Var "x" E
    in Lam "x" E (App (App tv x) term)  -- S2: λx.TV(x,T)
translateToIL T_cn cn =
    let x = Var "x" E
    in App (Const "exists" (Arrow T T))
           (Lam "x" E (App cn x))  -- S3: ∃x[CN(x)]
translateToIL T_det det cn = App det cn  -- S5-S7: DET(CN)

-- 限定词翻译
everyMeaning :: ILTerm
everyMeaning = Lam "P" (Arrow E T)
    (Lam "Q" (Arrow E T)
        (App (Const "forall" (Arrow (Arrow E T) T))
             (Lam "x" E
                 (App (App (Const "implies" (Arrow T (Arrow T T)))
                           (App (Var "P" (Arrow E T)) (Var "x" E)))
                      (App (Var "Q" (Arrow E T)) (Var "x" E))))))

someMeaning :: ILTerm
someMeaning = Lam "P" (Arrow E T)
    (Lam "Q" (Arrow E T)
        (App (Const "exists" (Arrow (Arrow E T) T))
             (Lam "x" E
                 (App (App (Const "and" (Arrow T (Arrow T T)))
                           (App (Var "P" (Arrow E T)) (Var "x" E)))
                      (App (Var "Q" (Arrow E T)) (Var "x" E))))))

theMeaning :: ILTerm
theMeaning = Lam "P" (Arrow E T)
    (Lam "Q" (Arrow E T)
        (App (Const "exists" (Arrow (Arrow E T) T))
             (Lam "x" E
                 (App (App (Const "and" (Arrow T (Arrow T T)))
                           (App (Const "forall" (Arrow (Arrow E T) T))
                                (Lam "y" E
                                    (App (App (Const "iff" (Arrow T (Arrow T T)))
                                              (App (Var "P" (Arrow E T)) (Var "y" E)))
                                         (Equals (Var "x" E) (Var "y" E))))))
                      (App (Var "Q" (Arrow E T)) (Var "x" E))))))
```

### 14.2.1.3.2 广义量词

> **定义 14.2.1.4**（广义量词）
> 广义量词是类型 ⟨⟨e, t⟩, t⟩ 的函数，将性质映射到真值。
> 对于限定词 D，其语义为：
> D'(P)(Q) 当且仅当集合 {x | P(x) ∧ Q(x)} 满足特定条件

```
量词类型关系：
  every: ⟨⟨e,t⟩, ⟨⟨e,t⟩, t⟩⟩

  every man walks:
    every' = λP.λQ.∀x[P(x) → Q(x)]
    man' = λx.man(x)
    walk' = λx.walk(x)

    every'(man')(walk')
    = (λP.λQ.∀x[P(x) → Q(x)])(man')(walk')
    = ∀x[man(x) → walk(x)]
```

```haskell
-- 广义量词定义
data GeneralizedQuantifier = GQ {
    gqCond :: Set Entity -> Bool,
    gqLeft :: Set Entity,   -- 限制集合（如man）
    gqRight :: Set Entity   -- 核集合（如walk）
} deriving (Show)

type Entity = String
type Set a = [a]

-- 量词条件
everyCond :: Ord a => Set a -> Set a -> Bool
everyCond restrict nuclear = all (`elem` nuclear) restrict

someCond :: Eq a => Set a -> Set a -> Bool
someCond restrict nuclear = any (`elem` nuclear) restrict

noCond :: Eq a => Set a -> Set a -> Bool
noCond restrict nuclear = not (any (`elem` nuclear) restrict)

mostCond :: Eq a => Set a -> Set a -> Bool
mostCond restrict nuclear =
    let intersection = filter (`elem` nuclear) restrict
    in length intersection > length restrict `div` 2

-- 转换为IL项
gqToIL :: String -> ILTerm
gqToIL "every" = everyMeaning
gqToIL "some" = someMeaning
gqToIL "the" = theMeaning
gqToIL "no" = Lam "P" (Arrow E T)
    (Lam "Q" (Arrow E T)
        (App (Const "not" (Arrow T T))
             (App (App someMeaning (Var "P" (Arrow E T)))
                  (Var "Q" (Arrow E T)))))
gqToIL _ = Const "unknown" (Arrow (Arrow E T) (Arrow (Arrow E T) T))
```

## 14.2.1.4 类型提升与函数组合

### 14.2.1.4.1 类型提升

> **定义 14.2.1.5**（类型提升）
> 类型提升将低类型对象转换为高类型对象：
>
> - **e → t提升**：x ↦ λP.P(x)
> - **IV提升**：P ↦ λx.P(x)

```
John walks

基础类型分析：
  John: e, walks: ⟨e,t⟩

类型提升后：
  John': λP.P(j)  （类型 ⟨⟨e,t⟩, t⟩）
  walks': walk'   （类型 ⟨e,t⟩）

问题：类型不匹配！

解决方案：将walks提升到期望的类型
  walk'' = λQ.Q(walk')  ? 不对

正确分析（PTQ方式）：
  John: T （通过规则S14，e → T）
  walks: IV

  S1规则组合：T IV → t
```

```haskell
-- 类型提升操作
typeRaiseEtoT :: ILTerm -> ILTerm
typeRaiseEtoT entity =
    Lam "P" (Arrow E T) (App (Var "P" (Arrow E T)) entity)

typeRaiseIVtoTV :: ILTerm -> ILTerm
typeRaiseIVtoTV iv =
    Lam "x" E (Lam "y" E (App iv (Var "x" E)))

-- Montague的S14规则：e → T
eToT :: ILTerm -> ILTerm
eToT entity = typeRaiseEtoT entity
```

### 14.2.1.4.2 函数组合

> **定义 14.2.1.6**（函数组合）
>
> - **前向组合**：(f ∘ g)(x) = f(g(x))
> - **类型组合**：λf.λg.λx.f(g(x))

```
语义组合示例：

John loves Mary

方法1（直接组合）：
  love'(John', Mary')

方法2（函数组合）：
  (love'(Mary'))(John')
  where love'(Mary') = λx.love(x, Mary')
```

```haskell
-- 函数组合
typeCompose :: MType -> MType -> MType -> ILTerm -> ILTerm -> ILTerm
typeCompose a b c f g =
    Lam "x" a (App f (App g (Var "x" a)))

-- 前向组合 >
forwardCompose :: ILTerm -> ILTerm -> Maybe ILTerm
forwardCompose f g = case (typeOf f, typeOf g) of
    (Arrow b c, Arrow a b') | b == b' ->
        Just (Lam "x" a (App f (App g (Var "x" a))))
    _ -> Nothing

-- 后向组合 <
backwardCompose :: ILTerm -> ILTerm -> Maybe ILTerm
backwardCompose g f = forwardCompose f g
```

## 14.2.1.5 内涵与外延

### 14.2.1.5.1 内涵对象

> **定义 14.2.1.7**（内涵）
> 表达式 α 的内涵 ^α 是从世界-时间对到 α 的外延的函数。
>
> ⟦^α⟧^{w,t} = λ⟨w',t'⟩.⟦α⟧^{w',t'}

```
必然性算子：
  □φ 翻译为: ^□(ˇφ)

  其中 □ 是类型 ⟨⟨s,t⟩, t⟩ 的算子

语义：□φ 在 w 为真当且仅当 φ 在所有与 w 相关的世界 w' 为真
```

```haskell
-- 内涵算子
intension :: ILTerm -> ILTerm
intension = Int

extension :: ILTerm -> ILTerm
extension = Ext

-- 必然性算子
necessarily :: ILTerm
necessarily = Lam "p" (Intension T)
    (App (Const "necessary" (Arrow (Intension T) T))
         (Var "p" (Intension T)))

-- 可能性算子
possibly :: ILTerm
possibly = Lam "p" (Intension T)
    (App (Const "possible" (Arrow (Intension T) T))
         (Var "p" (Intension T)))

-- 信念算子
believe :: ILTerm
believe = Lam "x" E
    (Lam "p" (Intension T)
        (App (App (Const "believe" (Arrow E (Arrow (Intension T) T)))
                  (Var "x" E))
             (Var "p" (Intension T))))
```

### 14.2.1.5.2 命题态度

```
John believes that Mary walks

翻译：
  believe'(John', ^walk'(Mary'))

类型检查：
  believe': ⟨e, ⟨⟨s,t⟩, t⟩⟩
  John': e
  ^walk'(Mary'): ⟨s, t⟩

组合：believe'(John')(Int(walk'(Mary')))
```

## 14.2.1.6 语义组合原则

### 14.2.1.6.1 函子原则

> **原则 14.2.1.1**（函子原则）
> 句法组合规则对应语义函数应用。

```
函子应用规则：

  X/Y  Y     语义: f(a)
      \  /
       X

  Y  X\Y     语义: f(a)
    \  /
       X
```

### 14.2.1.6.2 同构原则

> **原则 14.2.1.2**（句法-语义同构）
> 句法推导树与语义组合树同构。

```
句法树:                    语义树:
      S                         t
     / \                       / \
    NP  VP                    GQ  ⟨e,t⟩
    |    |                     \  /
   John  V'                    ∀x[...]
         |
        walks

每个句法节点对应一个语义类型
```

```haskell
-- 句法-语义同构映射
data SynSemTree = SST {
    ssSyntax :: String,
    ssSemantics :: ILTerm,
    ssChildren :: [SynSemTree]
} deriving (Show)

-- 构造同构树
buildSynSemTree :: String -> ILTerm -> [SynSemTree] -> SynSemTree
buildSynSemTree syn sem children = SST syn sem children

-- 语义组合
type ComposeRule = ILTerm -> ILTerm -> ILTerm

applyFunctor :: ComposeRule -> SynSemTree -> SynSemTree -> SynSemTree
applyFunctor rule (SST syn1 sem1 _) (SST syn2 sem2 _) =
    let combinedSem = rule sem1 sem2
        combinedSyn = syn1 ++ " " ++ syn2
    in SST combinedSyn combinedSem []

-- FA规则（函子应用）
faRule :: ComposeRule
faRule f x = App f x

-- BA规则（后向应用）
baRule :: ComposeRule
baRule x f = App f x
```

## 14.2.1.7 从PTQ到现代语义学

### 14.2.1.7.1 Montague遗产

| 贡献 | 影响 |
|------|------|
| 类型理论 | 现代形式语义学的基础框架 |
| λ演算 | 语义组合的核心工具 |
| 内涵逻辑 | 处理模态和命题态度 |
| 广义量词 | 限定词的自然分类 |
| 同构原则 | 句法-语义接口的范式 |

### 14.2.1.7.2 后续发展

1. **DRT**（Kamp, 1981）：处理无定名词的 discourse referents
2. **动态语义学**：将真值条件扩展为上下文更新函数
3. **类型语法**：句法范畴即语义类型
4. **博弈论语义学**：对话交互的解释模型

```haskell
-- 完整的Montague解释器
interpreter :: [PTQItem] -> Maybe ILTerm
interpreter items = case items of
    [item] -> Just (ptqMeaning item)
    (item1:item2:rest) -> do
        combined <- combineItems item1 item2
        interpreter (combined : rest)
    _ -> Nothing

combineItems :: PTQItem -> PTQItem -> Maybe PTQItem
combineItems item1 item2 =
    case (ptqCat item1, ptqCat item2) of
        (T_t', T_iv) -> Just $ PTQItem
            (ptqForm item1 ++ " " ++ ptqForm item2)
            T_t
            (App (ptqMeaning item2) (ptqMeaning item1))
        (T_det, T_cn) -> Just $
            let detMeaning = ptqMeaning item1
                cnMeaning = ptqMeaning item2
            in PTQItem
                (ptqForm item1 ++ " " ++ ptqForm item2)
                T_t'
                (App detMeaning cnMeaning)
        _ -> Nothing
```
