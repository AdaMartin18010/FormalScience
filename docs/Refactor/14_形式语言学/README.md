## 1. 模块概述

### 1.1 学科定位

形式语言学是应用数学和逻辑学方法研究自然语言的学科，旨在建立语言结构的严格形式化理论。
它将语言视为形式系统，通过精确的数学工具描述语言的句法、语义和语用特性。

### 1.2 核心研究问题

| 研究层次 | 核心问题 | 形式工具 |
|---------|---------|---------|
| 句法学 | 合法的句子结构是什么？ | 形式文法、自动机 |
| 语义学 | 句子如何表达意义？ | 逻辑、类型论 |
| 语用学 | 语境如何影响理解？ | 博弈论、信息论 |
| 心理语言学 | 语言如何在大脑中处理？ | 计算模型、统计学习 |

### 1.3 学科边界

```
形式语言学
    ├── 形式语法 ────┬──► 理论计算机科学 (自动机理论)
    │                └──► 数学 (形式语言理论)
    ├── 形式语义学 ──┬──► 数理逻辑 (模型论语义)
    │                └──► 哲学逻辑 (内涵逻辑)
    ├── 计算语言学 ──┬──► 人工智能 (NLP)
    │                └──► 认知科学 (语言处理)
    └── 语用学 ──────┬──► 博弈论 (语义博弈)
                     └──► 信息论 (信息结构)
```

## 3. 核心概念速览

### 3.1 Chomsky 层次

| 文法类型 | 语言类 | 自动机 | 生成能力 | 自然语言例子 |
|---------|--------|--------|---------|-------------|
| 0型 (无限制) | 递归可枚举 | 图灵机 | 最强 | 所有可计算问题 |
| 1型 (上下文有关) | 上下文有关语言 | 线性有界自动机 | 强 | 某些形态学现象 |
| 2型 (上下文无关) | 上下文无关语言 | 下推自动机 | 中等 | 大部分句法结构 |
| 3型 (正则) | 正则语言 | 有限自动机 | 最弱 | 形态词法、音系规则 |

### 3.2 语义形式化方法对比

| 方法 | 核心思想 | 形式工具 | 优势 | 局限 |
|------|---------|---------|------|------|
| Montague语法 | 自然语言是形式语言 | 类型论、λ演算 | 高度精确 | 复杂度高 |
| DRT | 动态更新话语表征 |  discourse表示结构 | 处理回指、预设 | 形式复杂 |
| 博弈论语义学 | 意义即验证博弈 | 博弈论 | 直观、交互式 | 计算复杂 |

## 4. 学习路径

### 4.1 初学者路径 (约40小时)

```
Week 1: 形式语法基础
    ├── 正则语言与有限自动机
    ├── 上下文无关文法
    └── 句法树与结构分析

Week 2: 语义学入门
    ├── 命题逻辑与一阶逻辑
    ├── λ演算基础
    └── Montague语法初步

Week 3: 计算实现
    ├── 句法分析算法 (CYK、Earley)
    └── 语义解析实现

Week 4: 应用专题
    ├── 语用学形式模型
    └── 心理语言学模型
```

### 4.2 进阶路径 (约80小时)

```
深入形式语法
    ├── 转换语法 (GB/MP)
    ├── 范畴语法 (AB/Lambek)
    └── 树邻接语法 (TAG)

深入形式语义学
    ├── 类型逻辑语法
    ├── 话语表示理论 (DRT)
    └── 博弈论语义学

高级计算语言学
    ├── 概率上下文无关文法 (PCFG)
    ├── 组合范畴语法 (CCG) 分析
    └── 神经网络句法分析
```
