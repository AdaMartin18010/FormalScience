# 14 形式语言学 - 目录与导航

---

📌 **内容摘要**

本文档深入探讨形式语言学 - 目录与导航的核心原理和关键方法。内容涵盖形式语言学领域的主要知识点，包括Chomsky, Montague, 形式语法, 形式文法等关键主题。适合有一定基础的学习者系统学习。

**关键词**: 形式语言学, Chomsky, Montague, 形式语法, 形式文法, Chomsky层次

📚 **学习目标**
- 掌握形式语言学 - 目录与导航的核心概念和主要方法
- 理解相关理论的应用场景
- 建立该领域的系统性知识框架

🎯 **难度级别**: 中级

⏱️ **预计阅读时间**: 15分钟

**前置知识**: 相关领域的基础概念, 离散数学
