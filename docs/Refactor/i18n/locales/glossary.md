## 使用说明

### 翻译优先级

1. **P0 (最高)** - 核心概念和系统组件术语
2. **P1 (高)** - 数学术语和计算机术语
3. **P2 (中)** - 网络术语和界面元素
4. **P3 (低)** - 补充说明和文档内容

### 术语选择原则

1. **准确性** - 确保术语翻译准确传达原意
2. **一致性** - 同一术语在全文中保持一致
3. **可读性** - 翻译应符合目标语言的表达习惯
4. **专业性** - 使用学科领域内公认的标准译法

### 更新维护

- 新术语添加时需要更新所有语言列
- 术语变更需要同步更新相关文档
- 定期审查术语翻译的准确性
