## 🤝 贡献翻译

我们欢迎社区成员参与翻译工作！

### 参与方式

1. **Fork 仓库** - 从主仓库创建你的分支
2. **选择任务** - 查看 [翻译任务看板](https://github.com/FormalScience/docs/projects/1)
3. **阅读指南** - 仔细阅读 [TRANSLATION_GUIDE.md](./TRANSLATION_GUIDE.md)
4. **开始翻译** - 按照规范进行翻译
5. **提交 PR** - 创建 Pull Request 并等待审核

### 翻译流程

```
选择文档 → 阅读指南 → 翻译内容 → 自我检查 → 提交 PR → 审核修改 → 合并发布
```

### 联系方式

- 📧 邮箱: i18n@formalscience.org
- 💬 Discord: #translation-channel
- 🐛 Issue: 使用标签 `i18n` 创建 Issue

## 📁 目录结构

```
i18n/
├── README.md                 # 本文件
├── TRANSLATION_GUIDE.md      # 翻译指南
├── terms.json                # 术语数据库
├── en/                       # 英文翻译目录
│   ├── README.md             # 英文入口
│   └── docs/                 # 英文文档
├── ja/                       # 日文翻译目录 (计划中)
├── ko/                       # 韩文翻译目录 (计划中)
└── locales/                  # 本地化文件
    ├── common.json           # 通用短语
    └── glossary.md           # 术语对照表
```

## 📝 翻译规范

- 所有翻译文件使用 **UTF-8** 编码
- 文档使用 **Markdown** 格式
- 遵循 [TRANSLATION_GUIDE.md](./TRANSLATION_GUIDE.md) 中的规范
- 保持与原文档的结构一致
