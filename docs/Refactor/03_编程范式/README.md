## 内容概要

### 01_编程语言理论

探讨编程语言的核心概念和理论基础：

- 四种主要编程范式的对比与形式化定义
- 编译器前端与后端的实现原理
- 内存管理的三种模型及其形式化分析
- 并发编程的四种模型：共享内存、消息传递、CSP、Actor

### 02_Rust语言深入

深入 Rust 语言的核心机制：

- 所有权系统的三条规则及其形式化模型
- Trait 系统与类型级别的编程
- Result/Option 单子与错误处理模式
- 声明宏与过程宏的元编程能力

### 03_异步编程模型

异步编程的理论与实践：

- Future trait 与 async/await 语法糖
- Tokio 运行时的调度器与执行器
- 流处理、背压控制与取消传播模式
- 状态机转换与 CPS 形式化分析

### 04_函数式编程

函数式编程的理论与实践：

- 纯函数、不可变性与高阶函数
- 函子、应用函子与单子的范畴论基础
- Thunk 机制与惰性流处理
- 组合子、透镜与箭头设计模式
