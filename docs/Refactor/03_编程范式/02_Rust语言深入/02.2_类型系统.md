## 02.2.1 概述

Rust的类型系统是**静态、强类型、带类型推断**的系统，它融合了多种类型理论：

- **代数数据类型 (ADT)**：来自ML家族
- **Traits**：类似Haskell的Type Classes
- **生命周期**：区域类型的变体
- **关联类型**：类型族的高级特性

### 02.2.1.1 类型系统特性

| 特性 | 说明 | 示例 |
|------|------|------|
| 静态类型 | 编译期类型检查 | `let x: i32 = 5;` |
| 类型推断 | 自动推导类型 | `let x = 5;` 推断为 `i32` |
| 名义子类型 | 基于名称的子类型 | `trait` 实现 |
| 结构化类型 | 基于结构的类型 | 元组、数组 |

## 02.2.2 Trait系统

### 02.2.2.1 基本Trait定义

**定义 02.2.1 (Trait)**

Trait定义了一组类型必须实现的方法接口：

```rust
// 定义Trait
pub trait Summary {
    // 必需方法
    fn summarize(&self) -> String;

    // 默认实现
    fn summarize_author(&self) -> String {
        String::from("(Read more...)")
    }
}

// 实现Trait
pub struct NewsArticle {
    pub headline: String,
    pub location: String,
}

impl Summary for NewsArticle {
    fn summarize(&self) -> String {
        format!("{}, 来自 {}", self.headline, self.location)
    }
}
```

### 02.2.2.2 Trait作为类型

```rust
// Trait对象 (动态分发)
pub fn notify(item: &dyn Summary) {
    println!("最新消息！{}", item.summarize());
}

// 泛型约束 (静态分发)
pub fn notify<T: Summary>(item: &T) {
    println!("最新消息！{}", item.summarize());
}

// 多重约束
pub fn notify<T: Summary + Display>(item: &T) { }

// where子句
fn some_function<T, U>(t: &T, u: &U) -> i32
where
    T: Display + Clone,
    U: Clone + Debug,
{ 0 }
```

### 02.2.2.3 形式化语义

**定义 02.2.2 (Trait字典)**

Trait实现可建模为记录类型（方法字典）：

```haskell
-- Haskell风格的形式化
data SummaryDict a = SummaryDict {
    summarize :: a -> String,
    summarize_author :: a -> String
}

-- 默认实现
mkSummaryDict :: (a -> String) -> SummaryDict a
mkSummaryDict summarize_impl = SummaryDict {
    summarize = summarize_impl,
    summarize_author = \_ -> "(Read more...)"
}
```

### 02.2.2.4 关联类型

**定义 02.2.3 (关联类型)**

关联类型允许Trait定义输出类型：

```rust
pub trait Iterator {
    type Item;  // 关联类型

    fn next(&mut self) -> Option<Self::Item>;
}

impl Iterator for Counter {
    type Item = u32;

    fn next(&mut self) -> Option<Self::Item> {
        self.count += 1;
        if self.count < 5 {
            Some(self.count)
        } else {
            None
        }
    }
}

// 使用
fn process<I: Iterator>(iter: I)
where
    I::Item: Display,
{ }
```

对比泛型参数：

```rust
// 泛型参数：允许多个实现
trait IteratorGeneric<Item> {
    fn next(&mut self) -> Option<Item>;
}
struct Counter { count: i32 }
impl IteratorGeneric<i32> for Counter {
    fn next(&mut self) -> Option<i32> { Some(self.count) }
}
impl IteratorGeneric<u32> for Counter {
    fn next(&mut self) -> Option<u32> { Some(self.count as u32) }
}

// 关联类型：每个类型只有一个实现
trait Iterator {
    type Item;
    fn next(&mut self) -> Option<Self::Item>;
}
impl Iterator for Counter {
    type Item = u32;
    fn next(&mut self) -> Option<Self::Item> { Some(self.count as u32) }
}
// impl Iterator for Counter { type Item = i32; ... }  // 错误：重复实现！
```

## 02.2.3 泛型

### 02.2.3.1 泛型基础

**定义 02.2.4 (参数多态)**

泛型提供**参数多态**，允许代码在多种类型上工作：

```rust
// 泛型函数
fn largest<T: PartialOrd>(list: &[T]) -> &T {
    let mut largest = &list[0];
    for item in list {
        if item > largest {
            largest = item;
        }
    }
    largest
}

// 泛型结构体
struct Point<T> {
    x: T,
    y: T,
}

// 多参数泛型
struct Point2<T, U> {
    x: T,
    y: U,
}
```

### 02.2.3.2 单态化

**定理 02.2.1 (单态化)**

Rust通过**单态化 (Monomorphization)** 将泛型代码特化为具体类型：

```rust
// 源码
fn identity<T>(x: T) -> T { x }

fn main() {
    let a = identity(5i32);   // 生成 identity_i32
    let b = identity('a');     // 生成 identity_char
}
```

编译后等价于：

```rust
fn identity_i32(x: i32) -> i32 { x }
fn identity_char(x: char) -> char { x }
```

### 02.2.3.3 泛型与零成本抽象

```rust
// 抽象代码
fn process_vector<T: Add<Output=T>>(v: Vec<T>) -> T {
    v.into_iter().fold(T::default(), |acc, x| acc + x)
}

// 编译器优化后（针对i32）相当于：
fn process_vector_i32(v: Vec<i32>) -> i32 {
    let mut sum = 0;
    for x in v {
        sum += x;
    }
    sum
}
```

## 02.2.4 高级类型特性

### 02.2.4.1 默认泛型参数

```rust
trait Add<Rhs=Self> {
    type Output;
    fn add(self, rhs: Rhs) -> Self::Output;
}

// 使用默认参数
struct Point { x: f64, y: f64 }
impl Add for Point {
    type Output = Self;
    fn add(self, rhs: Self) -> Self::Output {
        Point { x: self.x + rhs.x, y: self.y + rhs.y }
    }
}

// 覆盖默认参数
struct Velocity { dx: f64, dy: f64 }
impl Add<Velocity> for Point {
    type Output = Self;
    fn add(self, v: Velocity) -> Self::Output {
        Point { x: self.x + v.dx, y: self.y + v.dy }
    }
}
```

### 02.2.4.2 完全限定语法

```rust
trait Animal {
    fn baby_name() -> String;
}

struct Dog;
impl Dog {
    fn baby_name() -> String {
        String::from("Spot")
    }
}
impl Animal for Dog {
    fn baby_name() -> String {
        String::from("puppy")
    }
}

fn main() {
    println!("{}", Dog::baby_name());  // "Spot"
    println!("{}", <Dog as Animal>::baby_name());  // "puppy"
}
```

### 02.2.4.3 超级Trait

```rust
trait OutlinePrint: Display {  // OutlinePrint 要求实现 Display
    fn outline_print(&self) {
        let output = self.to_string();
        let len = output.len();
        println!("{}", "*".repeat(len + 4));
        println!("*{}*", " ".repeat(len + 2));
        println!("* {} *", output);
        println!("*{}*", " ".repeat(len + 2));
        println!("{}", "*".repeat(len + 4));
    }
}
```

## 02.2.5 类型推断与类型擦除

### 02.2.5.1 Hindley-Milner推断

Rust使用HM推断的扩展版本：

```rust
let v = Vec::new();  // Vec<_>
v.push(5);           // 推断为 Vec<i32>

// 显式标注
let v: Vec<i32> = Vec::new();
```

### 02.2.5.2 impl Trait

```rust
// 返回位置：隐藏具体类型
fn returns_iterator() -> impl Iterator<Item = i32> {
    (0..10).map(|x| x * 2)
}

// 参数位置：匿名类型参数
fn process(iter: impl Iterator<Item = i32>) {
    for x in iter {
        println!("{}", x);
    }
}

// 与泛型的区别：impl Trait是具体类型的语法糖
fn foo<T: Iterator>(iter: T) { }     // 调用者可选择类型
fn foo(iter: impl Iterator) { }       // 编译器选择类型
```

## 02.2.6 形式化类型规则

### 02.2.6.1 子类型与变型

**定义 02.2.5 (变型)**

| 变型 | 含义 | 示例 |
|------|------|------|
| 协变 (Covariant) | `T <: U` ⇒ `F<T> <: F<U>` | `&'a T`, `Box<T>` |
| 逆变 (Contravariant) | `T <: U` ⇒ `F<U> <: F<T>` | `fn(T) -> ()` |
| 不变 (Invariant) | `T = U` ⇒ `F<T> = F<U>` | `&mut T`, `Cell<T>` |

```rust
// 协变示例
fn foo(x: &'static str) { }
fn bar(x: &'a str) { foo(x); }  // 'a <: 'static

// 不变示例
fn bad(x: &mut &'static str) {
    let local = String::from("局部");
    *x = &local;  // 错误！会创建悬垂引用
}
```

### 02.2.6.2 类型规则形式化

```lean4
-- Lean4风格的形式化
inductive Ty : Type
  | int : Ty
  | bool : Ty
  | arrow : Ty → Ty → Ty
  | generic : String → Ty

inductive HasType : Expr → Ty → Prop
  | t_int (n : ℕ) : HasType (.int n) .int
  | t_bool (b : Bool) : HasType (.bool b) .bool
  | t_var {x τ} (h : ctx x = some τ) : HasType (.var x) τ
  | t_lam {x e τ1 τ2}
      (h : HasType (ctx.insert x τ1) e τ2) :
      HasType (.lam x e) (.arrow τ1 τ2)
  | t_app {e1 e2 τ1 τ2}
      (h1 : HasType ctx e1 (.arrow τ1 τ2))
      (h2 : HasType ctx e2 τ1) :
      HasType ctx (.app e1 e2) τ2
```

## 02.2.7 模式与最佳实践

### 02.2.7.1 构造者模式

```rust
#[derive(Default)]
struct HttpRequestBuilder {
    url: Option<String>,
    method: Option<String>,
    headers: Vec<(String, String)>,
}

impl HttpRequestBuilder {
    fn new() -> Self {
        Self::default()
    }

    fn url(mut self, url: impl Into<String>) -> Self {
        self.url = Some(url.into());
        self
    }

    fn method(mut self, method: impl Into<String>) -> Self {
        self.method = Some(method.into());
        self
    }

    fn build(self) -> Result<HttpRequest, BuilderError> {
        Ok(HttpRequest {
            url: self.url.ok_or(BuilderError::MissingUrl)?,
            method: self.method.unwrap_or_else(|| "GET".into()),
            headers: self.headers,
        })
    }
}
```

### 02.2.7.2 类型状态模式

```rust
struct Unverified;
struct Verified;

struct Email<State = Unverified> {
    address: String,
    _state: PhantomData<State>,
}

impl Email<Unverified> {
    fn new(address: String) -> Self {
        Self { address, _state: PhantomData }
    }

    fn verify(self) -> Result<Email<Verified>, Error> {
        if self.address.contains('@') {
            Ok(Email {
                address: self.address,
                _state: PhantomData
            })
        } else {
            Err(Error::Invalid)
        }
    }
}

impl Email<Verified> {
    fn send(&self, message: &str) {
        // 只能对验证过的邮箱发送
    }
}
```
